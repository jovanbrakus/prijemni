Formula za harmonijsko sabiranje

Izraz oblika \(a\sin x + b\cos x\) mo≈æe se zapisati kao:

\[a\sin x + b\cos x = R\sin(x + \varphi)\]

gde je:

\[R = \sqrt{a^2 + b^2}, \qquad \cos\varphi = \frac{a}{R},\quad \sin\varphi = \frac{b}{R}\]

Ova transformacija svodi zbir trigonometrijskih funkcija na jednu sinusnu funkciju sa pomerenom fazom.

Re≈°avanje jednaƒçine \(\sin\theta = c\)

\[\theta = \alpha + 2k\pi \quad \text{ili} \quad \theta = \pi - \alpha + 2k\pi, \quad k \in \mathbb{Z}\]

gde je \(\alpha = \arcsin c\).

Korak 1 ‚Äî Harmonijsko sabiranje

Zapi≈°imo levi deo jednaƒçine u obliku \(R\sin(x + \varphi)\).

Raƒçunamo amplitudu:

\[R = \sqrt{a^2 + b^2} = \sqrt{(\sqrt{3})^2 + 1^2} = \sqrt{3 + 1} = \sqrt{4} = 2\]

Odreƒëujemo fazni ugao \(\varphi\):

\[\cos\varphi = \frac{a}{R} = \frac{\sqrt{3}}{2}, \qquad \sin\varphi = \frac{b}{R} = \frac{1}{2}\]

Odavde je \(\varphi = \dfrac{\pi}{6}\) (jer su i sinus i kosinus pozitivni, ugao je u prvom kvadrantu).

Dakle:

\[\sqrt{3}\sin x + \cos x = 2\sin\!\left(x + \frac{\pi}{6}\right)\]

Razvij \(2\sin\!\left(x + \frac{\pi}{6}\right)\) koristeƒái adicione formule. Da li dobija≈° \(\sqrt{3}\sin x + \cos x\)?

\[2\sin\!\left(x + \frac{\pi}{6}\right) = 2\left[\sin x\cdot\cos\frac{\pi}{6} + \cos x\cdot\sin\frac{\pi}{6}\right] = 2\left[\sin x\cdot\frac{\sqrt{3}}{2} + \cos x\cdot\frac{1}{2}\right] = \sqrt{3}\sin x + \cos x\;\; ‚úì\]

\[\sin\theta = \frac{\sqrt{3}}{2}\]

Korak 4 ‚Äî Vraƒáanje na promenljivu \(x\)

\[x_1 = \frac{\pi}{3} - \frac{\pi}{6} + 2k\pi = \frac{2\pi - \pi}{6} + 2k\pi = \frac{\pi}{6} + 2k\pi\] \[x_2 = \frac{2\pi}{3} - \frac{\pi}{6} + 2k\pi = \frac{4\pi - \pi}{6} + 2k\pi = \frac{3\pi}{6} + 2k\pi = \frac{\pi}{2} + 2k\pi\]
