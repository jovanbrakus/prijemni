Plan rešavanja

  1. Primenimo formule za dvostruki ugao da bismo sve sveli na \(\sin x\) i \(\cos x\).
  2. Izlučimo zajednički faktor \(\cos x\) i rastavimo jednačinu na dva slučaja.
  3. Rešimo svaki slučaj posebno koristeći metod pomoćnog ugla.
  4. Saberemo sva rešenja iz intervala \((0,\, 2\pi)\).
Podsetnik: ključne formule i identiteti
  • Formule za dvostruki ugao: \(\sin 2x = 2\sin x \cos x\),   \(\cos 2x = 2\cos^2 x - 1\)
  • Pomoćni ugao: \(a\sin x + b\cos x = \sqrt{a^2 + b^2}\,\sin\!\left(x + \varphi\right)\), gde je \(\tan\varphi = \dfrac{b}{a}\).
    Posebno: \(\sin x - \cos x = \sqrt{2}\,\sin\!\left(x - \dfrac{\pi}{4}\right)\).
  • \(\cos x = 0\): \(x = \dfrac{\pi}{2} + k\pi,\ k \in \mathbb{Z}\).
  • \(\sin t = \dfrac{1}{2}\): \(t = \dfrac{\pi}{6} + 2k\pi\) ili \(t = \dfrac{5\pi}{6} + 2k\pi,\ k \in \mathbb{Z}\).

Grafički prikaz: rešenja na jediničnoj kružnici

Rešenje korak po korak

1Primena formula za dvostruki ugao

Polazna jednačina glasi:

\[\sin 2x = 1 + \sqrt{2}\,\cos x + \cos 2x\]

Zamenjujemo \(\sin 2x = 2\sin x\cos x\) i \(\cos 2x = 2\cos^2 x - 1\):

\[2\sin x\cos x = 1 + \sqrt{2}\,\cos x + 2\cos^2 x - 1\]

Sa desne strane se \(1\) i \(-1\) pokrate:

\[2\sin x\cos x = \sqrt{2}\,\cos x + 2\cos^2 x\]
2Prebacivanje na levu stranu i faktorisanje

Prebacimo sve na levu stranu:

\[2\sin x\cos x - \sqrt{2}\,\cos x - 2\cos^2 x = 0\]

Uočavamo da svaki sabirak sadrži faktor \(\cos x\). Izlučimo ga:

\[\cos x\,\bigl(2\sin x - \sqrt{2} - 2\cos x\bigr) = 0\]

Proizvod je nula ako i samo ako je bar jedan činilac nula. Dobijamo dva slučaja.

3Slučaj 1: \(\cos x = 0\)
Slučaj 1

Jednačina \(\cos x = 0\) na intervalu \((0,\, 2\pi)\) ima rešenja:

\[x_1 = \frac{\pi}{2}, \qquad x_2 = \frac{3\pi}{2}\]
Proveri se: da li \(x = \pi/2\) zadovoljava polaznu jednačinu?

Leva strana: \(\sin(2 \cdot \frac{\pi}{2}) = \sin\pi = 0\).
Desna strana: \(1 + \sqrt{2}\cos\frac{\pi}{2} + \cos\pi = 1 + 0 + (-1) = 0\).
Važi \(0 = 0\). Potvrdjeno!

Proveri se: da li \(x = 3\pi/2\) zadovoljava polaznu jednačinu?

Leva strana: \(\sin(2 \cdot \frac{3\pi}{2}) = \sin 3\pi = 0\).
Desna strana: \(1 + \sqrt{2}\cos\frac{3\pi}{2} + \cos 3\pi = 1 + 0 + (-1) = 0\).
Važi \(0 = 0\). Potvrdjeno!

4Slučaj 2: \(2\sin x - \sqrt{2} - 2\cos x = 0\)
Slučaj 2

Preuredimo jednačinu:

\[2\sin x - 2\cos x = \sqrt{2}\]

Podelimo obe strane sa \(2\):

\[\sin x - \cos x = \frac{\sqrt{2}}{2}\]
5Metod pomoćnog ugla

Koristimo poznati identitet:

\[\sin x - \cos x = \sqrt{2}\,\sin\!\left(x - \frac{\pi}{4}\right)\]

Objašnjenje: \(\sin x - \cos x = \sqrt{1^2 + (-1)^2}\,\sin(x + \varphi)\), gde je \(\tan\varphi = \frac{-1}{1} = -1\), dakle \(\varphi = -\frac{\pi}{4}\).

Zamenom u jednačinu:

\[\sqrt{2}\,\sin\!\left(x - \frac{\pi}{4}\right) = \frac{\sqrt{2}}{2}\]

Delimo obe strane sa \(\sqrt{2}\):

\[\sin\!\left(x - \frac{\pi}{4}\right) = \frac{1}{2}\]
6Rešavanje jednačine \(\sin t = \frac{1}{2}\)

Uvedimo smenu \(t = x - \frac{\pi}{4}\). Jednačina \(\sin t = \frac{1}{2}\) ima opšta rešenja:

\[t = \frac{\pi}{6} + 2k\pi \qquad \text{ili} \qquad t = \pi - \frac{\pi}{6} + 2k\pi = \frac{5\pi}{6} + 2k\pi, \quad k \in \mathbb{Z}\]

Vračamo smenu \(x = t + \frac{\pi}{4}\):

\[x = \frac{\pi}{6} + \frac{\pi}{4} + 2k\pi = \frac{2\pi + 3\pi}{12} + 2k\pi = \frac{5\pi}{12} + 2k\pi\]
\[x = \frac{5\pi}{6} + \frac{\pi}{4} + 2k\pi = \frac{10\pi + 3\pi}{12} + 2k\pi = \frac{13\pi}{12} + 2k\pi\]

Na intervalu \((0,\, 2\pi)\), za \(k = 0\) dobijamo:

\[x_3 = \frac{5\pi}{12}, \qquad x_4 = \frac{13\pi}{12}\]

Provera opsega: \(\frac{5\pi}{12} \approx 1{,}31 \in (0, 2\pi)\) i \(\frac{13\pi}{12} \approx 3{,}40 \in (0, 2\pi)\). Oba su validna.

7Zbir svih rešenja

Sakupimo sva četiri rešenja iz oba slučaja:

\(x_1\) (slučaj 1) \(\dfrac{\pi}{2}\) \(\dfrac{6\pi}{12}\)
\(x_2\) (slučaj 1) \(\dfrac{3\pi}{2}\) \(\dfrac{18\pi}{12}\)
\(x_3\) (slučaj 2) \(\dfrac{5\pi}{12}\) \(\dfrac{5\pi}{12}\)

\[\boxed{x_1 + x_2 + x_3 + x_4 = \frac{\pi}{2} + \frac{3\pi}{2} + \frac{5\pi}{12} + \frac{13\pi}{12} = \frac{42\pi}{12} = \frac{7\pi}{2}}\]

Ključni uvid

Srž ovog zadatka je u prepoznavanju zajedničkog faktora \(\cos x\) nakon primene formula za dvostruki ugao. Ovo razbija jednačinu na dva nezavisna, mnogo prostija problema. Drugi ključni korak je upotreba metoda pomoćnog ugla za svođenje izraza \(\sin x - \cos x\) na oblik \(\sqrt{2}\sin(x - \pi/4)\), čime se složena jednačina pretvara u elementarnu jednačinu oblika \(\sin t = \frac{1}{2}\).

Zapamtite ovu korisnu formulu:

\[\sin x - \cos x = \sqrt{2}\,\sin\!\left(x - \frac{\pi}{4}\right)\]

Česte greške

  • Deljenje sa \(\cos x\) umesto faktorisanja: Ako obe strane podelimo sa \(\cos x\), gubimo rešenja za koja važi \(\cos x = 0\) (tj. \(x = \pi/2\) i \(x = 3\pi/2\)). Uvek radije faktorišite nego delite!
  • Pogrešna primena formule za \(\cos 2x\): Postoje tri oblika: \(\cos^2 x - \sin^2 x\), \(2\cos^2 x - 1\) i \(1 - 2\sin^2 x\). Ovde je \(2\cos^2 x - 1\) najkorisniji jer omogućava kraćenje sa jedinicom na desnoj strani.
  • Greška u pomoćnom uglu: Identitet \(\sin x - \cos x = \sqrt{2}\sin(x - \pi/4)\) zahteva tačnu fazu \(-\pi/4\). Česta greška je pisanje \(\sqrt{2}\sin(x + \pi/4)\) ili \(\sqrt{2}\cos(x - \pi/4)\).
  • Propuštena rešenja sinusne jednačine: Jednačina \(\sin t = 1/2\) ima dva rešenja u svakom periodu: \(t = \pi/6\) i \(t = 5\pi/6\). Uzimanje samo jednog gubi rešenje i vodi na pogrešan zbir.
  • Računska greška pri sabiranju razlomaka: Pažljivo svedite sve na zajednički imenilac 12 pre sabiranja.
