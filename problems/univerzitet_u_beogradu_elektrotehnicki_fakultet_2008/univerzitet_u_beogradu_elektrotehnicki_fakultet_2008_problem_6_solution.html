Poredenje vrednosti izraza sa brojem 0,08

Odredivanje odnosa velicina \(x\), \(y\) i \(z\) definisanih pomocu kvadriranja i reciprocne vrednosti

Postavka zadatka

Ako je \(x = (0{,}08)^2\), \(y = \dfrac{1}{(0{,}08)^2}\) i \(z = (1 - 0{,}08)^2 - 1\), koji od sledecih iskaza je tacan?

\(x\)
\((0{,}08)^2\)
\(y\)
\(\dfrac{1}{(0{,}08)^2}\)
\(z\)
\((1 - 0{,}08)^2 - 1\)
(A) \(x = y = z\)
(B) \(y < z < x\)
(C) \(z < x < y\)
(D) \(y < x = z\)
(E) \(x + z = y\)
(N) Ne znam

Plan resavanja

Oznacicemo \(a = 0{,}08\) i izracunati numericke vrednosti sva tri izraza \(x\), \(y\) i \(z\). Zatim cemo ih uporediti i odrediti koji iskaz je tacan. Dodatno cemo proveriti i preko razlomaka (\(a = \tfrac{2}{25}\)) za potpunu preciznost.

Podsetnik: kljucne formule
  • Kvadrat decimale: \((0{,}ab)^2\) — pomnoži se broj sam sa sobom, paziti na broj decimala
  • Reciprocna vrednost: \(\dfrac{1}{a^2}\) — ako je \(0 < a < 1\), tada je \(\dfrac{1}{a^2} > 1\) (vece od 1)
  • Kvadrat razlike: \((1 - a)^2 = 1 - 2a + a^2\), pa je \((1-a)^2 - 1 = a^2 - 2a\)
  • Znak izraza: Ako je \(0 < a < 1\), tada \(a^2 < 2a\), pa je \(a^2 - 2a < 0\)
  • Razlomak: \(0{,}08 = \dfrac{8}{100} = \dfrac{2}{25}\)

Vizuelni prikaz

Brojevna prava sa pozicijama \(z\), \(x\) i \(y\) (y je van opsega, prikazan strelicom):

Resenje korak po korak

1

Uvedimo oznaku i zapišimo \(a\) kao razlomak

Neka je \(a = 0{,}08\). Pretvorimo u razlomak:

\[ a = 0{,}08 = \frac{8}{100} = \frac{2}{25} \]

Ovo ce nam olaksati racunanje bez kalkulatora.

2

Izracunajmo \(x = a^2\)

Kvadriramo \(a\):

\[ x = a^2 = \left(\frac{2}{25}\right)^2 = \frac{4}{625} \]

U decimalnom zapisu:

\[ x = (0{,}08)^2 = 0{,}0064 \]

Dakle, \(x\) je veoma mali pozitivan broj, blizu nule.

Proveri se: Zasto je \(0{,}08^2 = 0{,}0064\)?

Racunamo \(8 \times 8 = 64\). Svaki faktor ima 2 decimale, pa proizvod ima \(2 + 2 = 4\) decimale: \(0{,}0064\). Alternativno: \(\frac{2}{25} \times \frac{2}{25} = \frac{4}{625}\), a \(\frac{4}{625} = 0{,}0064\) jer \(625 \times 0{,}0064 = 4\).

3

Izracunajmo \(y = \dfrac{1}{a^2}\)

Reciprocna vrednost kvadrata:

\[ y = \frac{1}{a^2} = \frac{1}{\frac{4}{625}} = \frac{625}{4} = 156{,}25 \]

Dakle, \(y\) je veoma veliki broj, veci od 156!

4

Izracunajmo \(z = (1-a)^2 - 1\)

Prvo izracunamo \(1 - a = 1 - 0{,}08 = 0{,}92\), zatim kvadriramo:

\[ (0{,}92)^2 = 0{,}8464 \]

Oduzimamo 1:

\[ z = 0{,}8464 - 1 = -0{,}1536 \]

Dakle, \(z\) je negativan broj!

Proveri se: Algebarski pristup za \(z\)

Mozemo i bez decimalnog racunanja, koristeci formulu za kvadrat razlike:

\[ z = (1 - a)^2 - 1 = 1 - 2a + a^2 - 1 = a^2 - 2a = a(a - 2) \]

Posto je \(a = 0{,}08 > 0\) i \(a - 2 = -1{,}92 < 0\), dobijamo \(z < 0\). Konkretno:

\[ z = 0{,}08 \cdot (-1{,}92) = -0{,}1536 \]

Ili preko razlomaka: \(z = \frac{2}{25} \cdot \left(\frac{2}{25} - 2\right) = \frac{2}{25} \cdot \left(-\frac{48}{25}\right) = -\frac{96}{625}\).

5

Uporedimo \(x\), \(y\) i \(z\)

Sada imamo numericke vrednosti:

\[ z = -0{,}1536, \qquad x = 0{,}0064, \qquad y = 156{,}25 \]

Jasno je da vazi:

\[ \underbrace{z}_{-0{,}1536} \;<\; \underbrace{x}_{0{,}0064} \;<\; \underbrace{y}_{156{,}25} \]

Ovaj poredak odgovara opciji (C): \(z < x < y\).

6

Eliminisimo ostale opcije

Proverimo svaku opciju:

\[ \text{(A) } x = y = z \quad \text{--- Netacno, svi su razliciti} \] \[ \text{(B) } y < z < x \quad \text{--- Netacno, } y = 156{,}25 \text{ je najveci} \] \[ \text{(C) } z < x < y \quad \text{--- \textbf{Tacno!}} \] \[ \text{(D) } y < x = z \quad \text{--- Netacno, } x \neq z \text{ i } y \text{ je najveci} \] \[ \text{(E) } x + z = y \quad \text{--- } 0{,}0064 + (-0{,}1536) = -0{,}1472 \neq 156{,}25 \]
Proveri se: Moze li se resiti bez racunanja tacnih vrednosti?

Da! Dovoljno je odrediti znakove i grube velicine:

  • \(x = a^2 > 0\) (kvadrat je uvek pozitivan) i veoma mali jer je \(a\) mali
  • \(y = 1/a^2 > 0\) i veoma veliki jer je \(a\) mali (deli se 1 malim brojem)
  • \(z = a(a-2) < 0\) jer \(a > 0\) i \(a - 2 < 0\) (negativan!)

Vec samo iz znakova: \(z < 0 < x < y\), sto direktno daje odgovor (C).

Kljucni uvid

Kada je \(0 < a < 1\), kvadriranje ga smanjuje (priblizava nuli), dok reciprocna vrednost ga povecava (udaljava od nule). Zato je \(x = a^2\) veoma mali pozitivan broj, a \(y = 1/a^2\) veoma veliki. Sa druge strane, \(z = (1-a)^2 - 1\) se algebarski svodi na \(a^2 - 2a = a(a-2)\), sto je negativno za svako \(a \in (0, 2)\). Samim tim, bez ikakvog racunanja, iz znakova vec znamo: \(z < 0 < x < y\).

Konacan odgovor

(A) \(x = y = z\)
(B) \(y < z < x\)
(C) \(z < x < y\)
(D) \(y < x = z\)
(E) \(x + z = y\)
Odgovor je (C) \(z < x < y\)
Provera: Sa tacnim vrednostima: \(z = -0{,}1536\), \(x = 0{,}0064\), \(y = 156{,}25\). Zaista vazi \(-0{,}1536 < 0{,}0064 < 156{,}25\). Takodje, iz razlomaka: \(-\frac{96}{625} < \frac{4}{625} < \frac{625}{4}\).

Ceste greske

  • Pogresan znak za \(z\): Mnogi zaborave da je \((0{,}92)^2 = 0{,}8464 < 1\), pa kada se oduzme 1, rezultat je negativan. Ko previdi da je \(z < 0\), moze pogresno zakljuciti da je npr. \(x = z\).
  • Zamena \(x\) i \(y\): Vazno je razlikovati \(a^2\) (kvadrat malog broja — jos manji) od \(1/a^2\) (reciprocna vrednost kvadrata malog broja — veoma veliki). Za \(a < 1\) uvek vazi \(a^2 < 1 < 1/a^2\).
  • Greska pri racunanju \((0{,}92)^2\): Treba pazljivo pomnoziti: \(92 \times 92 = 8464\), pa sa cetiri decimale: \(0{,}8464\). Greska u ovoj cifri moze navesti na pogresan zakljucak o znaku \(z\).

Bonus izazov

Za koje vrednosti \(a \in (0, 1)\) vazi \(x + z = 0\), tj. \(a^2 + (1-a)^2 - 1 = 0\)?

\[ a^2 + (1-a)^2 - 1 = a^2 + 1 - 2a + a^2 - 1 = 2a^2 - 2a = 2a(a-1) = 0 \]
Pogledaj resenje

Iz \(2a(a-1) = 0\) dobijamo \(a = 0\) ili \(a = 1\). Posto trazimo \(a \in (0,1)\), ne postoji nijedno \(a\) u tom intervalu za koje je \(x + z = 0\). Na krajevima: za \(a = 0\) i \(a = 1\) izraz je nula, ali to su granicni slucajevi. Za svako \(a \in (0, 1)\) vazi \(x + z < 0\) (jer je \(z\) negativniji nego sto je \(x\) pozitivan).
