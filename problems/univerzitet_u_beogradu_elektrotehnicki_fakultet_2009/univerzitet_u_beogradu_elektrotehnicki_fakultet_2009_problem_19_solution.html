T Teorijski podsetnik
▼

Formule dvostrukog ugla za kosinus:

\[\cos 2x = \cos^2 x - \sin^2 x = 2\cos^2 x - 1 = 1 - 2\sin^2 x\]

Koristimo oblik \(\cos 2x = 1 - 2\sin^2 x\) jer želimo da izrazimo sve preko \(\sin x\).

Vrednosti \(\sin x = \frac{1}{2}\) na \([0, 2\pi]\):

  • \(x = \dfrac{\pi}{6}\) (prvi kvadrant)
  • \(x = \pi - \dfrac{\pi}{6} = \dfrac{5\pi}{6}\) (drugi kvadrant)

Vrednost \(\sin x = -1\) na \([0, 2\pi]\):

  • \(x = \dfrac{3\pi}{2}\)

Analiza znaka proizvoda:

Proizvod \(A \cdot B < 0\) znači da faktori imaju različit znak. Ako je jedan faktor uvek nenegativan, dovoljno je ispitati kada je drugi faktor negativan, uz uslov da nenegativni faktor nije jednak nuli.

Presečne tačke
R Rešavanje korak po korak
Korak 1 — Svođenje na jednu trigonometrijsku funkciju

Polazna nejednačina glasi:

\[\sin x < \cos 2x\]

Primenjujemo identitet \(\cos 2x = 1 - 2\sin^2 x\):

\[\sin x < 1 - 2\sin^2 x\]

Prebacujemo sve na levu stranu:

\[2\sin^2 x + \sin x - 1 < 0\]
Kontrolna tačka
Dobili smo kvadratnu nejednačinu po \(\sin x\). Smena \(t = \sin x\) daje \(2t^2 + t - 1 < 0\).
Korak 2 — Faktorizacija

Faktorizujemo kvadratni trinom \(2t^2 + t - 1\):

Tražimo nule: \(t = \dfrac{-1 \pm \sqrt{1 + 8}}{4} = \dfrac{-1 \pm 3}{4}\)

\[t_1 = \frac{-1 + 3}{4} = \frac{1}{2}, \qquad t_2 = \frac{-1 - 3}{4} = -1\]

Dakle:

\[2t^2 + t - 1 = 2(t - \tfrac{1}{2})(t + 1) = (2t - 1)(t + 1)\]

Vraćajući smenu \(t = \sin x\):

\[\boxed{(2\sin x - 1)(\sin x + 1) < 0}\]
Korak 3 — Analiza znaka faktora

Faktor \(\sin x + 1\):

Pošto je \(\sin x \geq -1\) za svako \(x\), imamo:

\[\sin x + 1 \geq 0 \quad \text{za svako } x\]

Jednakost \(\sin x + 1 = 0\) važi samo kad je \(\sin x = -1\), tj. za \(x = \dfrac{3\pi}{2}\).

Faktor \(2\sin x - 1\):

\(2\sin x - 1 < 0 \iff \sin x < \dfrac{1}{2}\)

\(2\sin x - 1 = 0 \iff \sin x = \dfrac{1}{2} \iff x = \dfrac{\pi}{6}\) ili \(x = \dfrac{5\pi}{6}\)

Kontrolna tačka
Proizvod dva faktora je negativan samo kada faktori imaju različit znak. Pošto je \(\sin x + 1 \geq 0\) uvek, proizvod je negativan samo kad je \(\sin x + 1 > 0\) i \(2\sin x - 1 < 0\).
Korak 4 — Rešavanje sistema uslova

Potrebno je da istovremeno važi:

\[\begin{cases} 2\sin x - 1 < 0 &\Longrightarrow \sin x < \dfrac{1}{2} \\ \sin x + 1 > 0 &\Longrightarrow x \neq \dfrac{3\pi}{2} \end{cases}\]

Rešavamo \(\sin x < \frac{1}{2}\) na \([0,\, 2\pi]\):

Funkcija \(\sin x\) je manja od \(\frac{1}{2}\) na intervalima:

\[\sin x < \frac{1}{2} \quad \Longleftrightarrow \quad x \in \left[0,\, \frac{\pi}{6}\right) \cup \left(\frac{5\pi}{6},\, 2\pi\right]\]

Isključujemo \(x = \frac{3\pi}{2}\):

U tački \(x = \frac{3\pi}{2}\) važi \(\sin x = -1\), pa je \(\sin x + 1 = 0\). To znači da je proizvod \((2\sin x - 1)(\sin x + 1) = 0\), a ne strogo negativan. Dakle, \(x = \frac{3\pi}{2}\) nije rešenje.

Konačno rešenje:

\[\boxed{x \in \left[0,\, \frac{\pi}{6}\right) \cup \left(\frac{5\pi}{6},\, \frac{3\pi}{2}\right) \cup \left(\frac{3\pi}{2},\, 2\pi\right]}\]

Kontrolna tačka
Sve provere se poklapaju. Na granicama \(x = \frac{\pi}{6}\) i \(x = \frac{5\pi}{6}\) važi \(\sin x = \frac{1}{2} = \cos 2x\) (jednakost, ne stroga nejednačina). U tački \(x = \frac{3\pi}{2}\) važi \(\sin x = \cos 2x = -1\) (takođe jednakost).
Ključni uvid

Faktor \(\sin x + 1\) je uvek nenegativan, ali u jednoj tački iznosi nula — tačno u \(x = \frac{3\pi}{2}\). To ne daje strogo negativan proizvod, pa se ta tačka mora isključiti. Mnoge ponuđene opcije razlikuju se baš po ovom detalju: da li je \(\frac{3\pi}{2}\) uključeno ili ne. Pažljiva analiza granične tačke razdvaja opcije (B) i (E).

Konačan odgovor
(E) \(\left[0,\, \dfrac{\pi}{6}\right) \cup \left(\dfrac{5\pi}{6},\, \dfrac{3\pi}{2}\right) \cup \left(\dfrac{3\pi}{2},\, 2\pi\right]\)
Nejednačina \(\sin x < \cos 2x\) važi za sve \(x \in [0, 2\pi]\) gde je \(\sin x < \frac{1}{2}\), osim u tački \(x = \frac{3\pi}{2}\) gde važi jednakost.
! Najčešće greške
Zaboraviti da isključimo \(x = \frac{3\pi}{2}\)

Opcija (B) daje \([0, \frac{\pi}{6}) \cup (\frac{5\pi}{6}, 2\pi]\) — ne isključuje \(\frac{3\pi}{2}\). Međutim, u toj tački je \(\sin x = \cos 2x = -1\) (jednakost), pa nejednačina nije zadovoljena.

Pogrešan izbor identiteta za \(\cos 2x\)

Ako koristimo \(\cos 2x = \cos^2 x - \sin^2 x\), dobijamo izraz sa i \(\sin x\) i \(\cos x\), što je komplikovanije za faktorizaciju. Oblik \(1 - 2\sin^2 x\) je optimalan za ovu nejednačinu.

Mešanje granica \(\sin x < \frac{1}{2}\) i \(\sin x < 1\)

Opcije (A) i (D) koriste granice \(\frac{\pi}{3}\) i \(\frac{2\pi}{3}\), što odgovara rešenju \(\sin x < \frac{\sqrt{3}}{2}\), a ne \(\sin x < \frac{1}{2}\). Pažljivo proverite koji je uslov dobijen faktorizacijom.

* Bonus izazov

Pitanje: Na segmentu \([0,\, 2\pi]\), odredi skup svih rešenja nejednačine \(\sin x \geq \cos 2x\).

Savet: Komplement traženog skupa je upravo skup koji smo već odredili.

Prikaži odgovor ▼

Komplement skupa rešenja \(\sin x < \cos 2x\) na \([0, 2\pi]\) daje:

\[\sin x \geq \cos 2x \iff x \in \left[\frac{\pi}{6},\, \frac{5\pi}{6}\right] \cup \left\{\frac{3\pi}{2}\right\}\]

Na intervalu \(\left[\frac{\pi}{6},\, \frac{5\pi}{6}\right]\) važi \(\sin x \geq \frac{1}{2}\), pa je \(\sin x \geq \cos 2x\).

U izolovanoj tački \(x = \frac{3\pi}{2}\) važi jednakost: \(\sin\frac{3\pi}{2} = \cos 3\pi = -1\).
