Trigonometrijska jednacina na segmentu

Nalazenje broja resenja jednacine \(\sin^4 x - \cos^4 x = \cos 4x\) na \([0,\, 2\pi]\)

Plan resavanja

Resavamo u cetiri glavna koraka:

  1. Transformisemo levu stranu koristeci razliku kvadrata: \(\sin^4 x - \cos^4 x = -\cos 2x\).
  2. Jednacinu svedemo na oblik \(\cos 4x + \cos 2x = 0\).
  3. Primenimo formulu za zbir kosinusa da faktorizujemo u \(2\cos 3x \cos x = 0\).
  4. Resimo svaki faktor posebno i prebrojimo razlicita resenja na \([0, 2\pi]\).
Teorija: Trigonometrijski identiteti i formule transformacije
  • Razlika kvadrata: \(a^2 - b^2 = (a-b)(a+b)\)
  • Pitagorin identitet: \(\sin^2 x + \cos^2 x = 1\)
  • Dupli ugao: \(\cos 2x = \cos^2 x - \sin^2 x\)
  • Zbir kosinusa: \(\cos A + \cos B = 2\cos\!\dfrac{A+B}{2}\cos\!\dfrac{A-B}{2}\)
  • Nule kosinusa: \(\cos \theta = 0 \;\Longleftrightarrow\; \theta = \dfrac{\pi}{2} + k\pi,\; k \in \mathbb{Z}\)

Vizuelni prikaz

Grafik prikazuje levu stranu \(\sin^4 x - \cos^4 x\) (plavo) i desnu stranu \(\cos 4x\) (roze). Presecne tacke su resenja jednacine. Resenja su oznacena zelenim tackama.

Resenje korak po korak

1

Transformisemo levu stranu jednacine

Izraz \(\sin^4 x - \cos^4 x\) je razlika kvadrata oblika \(a^2 - b^2\), gde je \(a = \sin^2 x\) i \(b = \cos^2 x\):

\[ \sin^4 x - \cos^4 x = (\sin^2 x)^2 - (\cos^2 x)^2 = (\sin^2 x - \cos^2 x)(\sin^2 x + \cos^2 x) \]

Posto je \(\sin^2 x + \cos^2 x = 1\) (Pitagorin identitet), imamo:

\[ \sin^4 x - \cos^4 x = \sin^2 x - \cos^2 x = -(\cos^2 x - \sin^2 x) = -\cos 2x \]

Dakle, leva strana jednacine se svodi na \(-\cos 2x\).

Proveri se: Zasto je \(\sin^2 x - \cos^2 x = -\cos 2x\)?

Po definiciji kosinusa dvostrukog ugla:

\[ \cos 2x = \cos^2 x - \sin^2 x \]

Mnozeci obe strane sa \(-1\):

\[ -\cos 2x = \sin^2 x - \cos^2 x \]

Ovo je upravo nas izraz sa leve strane.

2

Postavimo jednacinu u standardni oblik

Uvrstavanjem rezultata iz koraka 1, jednacina postaje:

\[ -\cos 2x = \cos 4x \]

Prebacimo sve na jednu stranu:

\[ \cos 4x + \cos 2x = 0 \]
3

Primenimo formulu za zbir kosinusa

Koristimo formulu transformacije zbira u proizvod:

\[ \cos A + \cos B = 2\cos\frac{A+B}{2}\cos\frac{A-B}{2} \]

Za \(A = 4x\) i \(B = 2x\):

\[ \cos 4x + \cos 2x = 2\cos\frac{4x+2x}{2}\cos\frac{4x-2x}{2} = 2\cos 3x \cdot \cos x \]

Jednacina sada glasi:

\[ 2\cos 3x \cdot \cos x = 0 \]

Proizvod je jednak nuli ako je bar jedan od faktora jednak nuli, dakle:

\[ \cos 3x = 0 \quad \text{ili} \quad \cos x = 0 \]
Proveri se: Zasto mozemo da podelimo na dva slucaja?

Kljucno svojstvo realnih brojeva: ako je \(a \cdot b = 0\), onda je \(a = 0\) ili \(b = 0\) (ili oba). Faktor 2 je razlicit od nule, pa preostaje da je \(\cos 3x = 0\) ili \(\cos x = 0\). Ovo nam omogucava da resimo svaki slucaj nezavisno, a zatim uzmemo uniju svih resenja.

4

Resimo jednacinu \(\cos x = 0\)

Opste resenje jednacine \(\cos x = 0\) je:

\[ x = \frac{\pi}{2} + k\pi, \quad k \in \mathbb{Z} \]

Na segmentu \([0, 2\pi]\):

\[ k = 0: \quad x = \frac{\pi}{2} \] \[ k = 1: \quad x = \frac{3\pi}{2} \]

Iz ovog slucaja dobijamo 2 resenja: \(\dfrac{\pi}{2}\) i \(\dfrac{3\pi}{2}\).

5

Resimo jednacinu \(\cos 3x = 0\)

Opste resenje jednacine \(\cos 3x = 0\) je:

\[ 3x = \frac{\pi}{2} + k\pi, \quad k \in \mathbb{Z} \]

Deljenjem sa 3:

\[ x = \frac{\pi}{6} + \frac{k\pi}{3}, \quad k \in \mathbb{Z} \]

Na segmentu \([0, 2\pi]\) trazimo sve celobrojne vrednosti \(k\) za koje \(0 \leq \dfrac{\pi}{6} + \dfrac{k\pi}{3} \leq 2\pi\):

6

Prebrojimo razlicita resenja

Uocimo da su resenja iz slucaja \(\cos x = 0\) (tj. \(\frac{\pi}{2}\) i \(\frac{3\pi}{2}\)) vec sadrzana u resenjima iz slucaja \(\cos 3x = 0\). Unija oba skupa daje:

\[ x \in \left\{\frac{\pi}{6},\; \frac{\pi}{2},\; \frac{5\pi}{6},\; \frac{7\pi}{6},\; \frac{3\pi}{2},\; \frac{11\pi}{6}\right\} \]

To je ukupno 6 razlicitih resenja na segmentu \([0, 2\pi]\).

Proveri se: Da li sva resenja zaista zadovoljavaju polaznu jednacinu?

Proverimo za svako resenje da li vazi \(\sin^4 x - \cos^4 x = \cos 4x\):

\(x = \frac{\pi}{6}\):

\[ \text{Leva: } \sin^4\!\frac{\pi}{6} - \cos^4\!\frac{\pi}{6} = \left(\frac{1}{2}\right)^{\!4} - \left(\frac{\sqrt{3}}{2}\right)^{\!4} = \frac{1}{16} - \frac{9}{16} = -\frac{1}{2} \] \[ \text{Desna: } \cos\frac{4\pi}{6} = \cos\frac{2\pi}{3} = -\frac{1}{2} \quad \checkmark \]

\(x = \frac{\pi}{2}\):

\[ \text{Leva: } 1^4 - 0^4 = 1 \qquad \text{Desna: } \cos 2\pi = 1 \quad \checkmark \]

\(x = \frac{5\pi}{6}\):

\[ \text{Leva: } \sin^4\!\frac{5\pi}{6} - \cos^4\!\frac{5\pi}{6} = \frac{1}{16} - \frac{9}{16} = -\frac{1}{2} \] \[ \text{Desna: } \cos\frac{20\pi}{6} = \cos\frac{10\pi}{3} = \cos\frac{4\pi}{3} = -\frac{1}{2} \quad \checkmark \]

Ostala tri resenja se proveravaju analogno -- sva zadovoljavaju polaznu jednacinu.

Kljucni uvid

Izraz \(\sin^4 x - \cos^4 x\) izgleda komplikovano, ali je u sustini razlika kvadrata koja se svodi na \(-\cos 2x\). Ovo je klasican primer gde prepoznavanje algebarskog identiteta drasticno pojednostavljuje trigonometrijski problem. Potom, formula za zbir kosinusa \(\cos A + \cos B = 2\cos\frac{A+B}{2}\cos\frac{A-B}{2}\) pretvara jednacinu u proizvod dva faktora, sto omogucava jednostavno resavanje.

Takodje, vazno je uociti da su resenja iz \(\cos x = 0\) podskup resenja iz \(\cos 3x = 0\) -- zato sto je \(\cos 3x = \cos x(4\cos^2 x - 3)\), pa kad je \(\cos x = 0\), automatski je i \(\cos 3x = 0\). Zbog toga ne moramo posebno dodavati ta resenja.

Provera: Sva resenja zadovoljavaju \(2\cos 3x \cdot \cos x = 0\), sto je ekvivalentno polaznoj jednacini. Sest resenja jednacine \(\cos 3x = 0\) na \([0, 2\pi]\) obuhvataju i oba resenja jednacine \(\cos x = 0\), tako da je konacno 6 razlicitih resenja.

Ceste greske

  • Nepotpuno pojednostavljenje leve strane: Studenti cesto pokusavaju da razviju \(\sin^4 x - \cos^4 x\) koristeci formule za cetvrte stepene, umesto da prepoznaju razliku kvadrata. To vodi ka komplikovanim izrazima i cestim greskim.
  • Duplo brojanje resenja: Posto su resenja \(\cos x = 0\) (tj. \(\frac{\pi}{2}\) i \(\frac{3\pi}{2}\)) vec sadrzana medju resenjima \(\cos 3x = 0\), lako je pogresno izbrojati 8 umesto 6 resenja. Uvek je bitno napraviti uniju i eliminisati duplikate.
  • Zaboravljena resenja: Kod jednacine \(\cos 3x = 0\), period argumenta \(3x\) je \(\frac{2\pi}{3}\), sto daje vise resenja na \([0, 2\pi]\) nego sto studenti ocekuju. Vazno je sistematski nabrojati sva \(k\).
  • Pogresan identitet za \(\cos 4x\): Neki studenti koriste \(\cos 4x = 2\cos^2 2x - 1\) i pokusavaju da rese kvadratnu jednacinu po \(\cos 2x\). Ovo funkcionise, ali je slozenije i podloznije greskim pri nalazenju svih resenja.

Bonus izazov

Pokusaj da resis slicnu jednacinu:

\[ \sin^4 x + \cos^4 x = \cos 4x + \frac{1}{2} \]

Koliko resenja ima na segmentu \([0, 2\pi]\)?

Pogledaj nagovestaj

Koristi identitet:

\[ \sin^4 x + \cos^4 x = (\sin^2 x + \cos^2 x)^2 - 2\sin^2 x\cos^2 x = 1 - \frac{1}{2}\sin^2 2x = \frac{3}{4} + \frac{1}{4}\cos 4x \]

Uvrstavanje u jednacinu daje \(\frac{3}{4} + \frac{1}{4}\cos 4x = \cos 4x + \frac{1}{2}\), tj. \(\frac{3}{4}\cos 4x = \frac{1}{4}\), odakle \(\cos 4x = \frac{1}{3}\). Ova jednacina ima 8 resenja na \([0, 2\pi]\).
