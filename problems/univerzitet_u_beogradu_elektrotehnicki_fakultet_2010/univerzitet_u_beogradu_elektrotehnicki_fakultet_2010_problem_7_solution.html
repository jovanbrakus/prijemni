Ostatak pri deljenju polinoma binomom

Primena teoreme o ostatku za nalazenje ostatka deljenja polinoma \(3x^5 + 2x^4 + 3\) binomom \(x + 1\)

Postavka zadatka

Ostatak pri deljenju polinoma \(3x^5 + 2x^4 + 3\) binomom \(x + 1\) jeste:

Odgovor (A)
\(x + 2\)
Odgovor (B)
\(-3\)
Odgovor (C)
\(-2\)
Odgovor (D)
\(2\)
Odgovor (E)
\(3x + 1\)
Odgovor (N)
Ne znam

Plan resavanja

Koristicemo teoremu o ostatku (Bezuova teorema): kada delimo polinom \(P(x)\) binomom \((x - a)\), ostatak je jednak \(P(a)\). Posto je nas binom \(x + 1 = x - (-1)\), potrebno je samo da izracunamo \(P(-1)\). Dodatno cemo prikazati i Hornerovu shemu kao verifikaciju.

Teorija: Teorema o ostatku (Bezuova teorema)
  • Teorema o ostatku: Ako se polinom \(P(x)\) deli binomom \((x - a)\), onda je ostatak jednak \(P(a)\).
  • Formalno: \(P(x) = (x - a) \cdot Q(x) + R\), gde je \(R = P(a)\) konstantan broj.
  • Zapazanje: Ostatak pri deljenju polinoma linearnim binomom je uvek konstanta (stepen 0), jer je stepen ostatka manji od stepena delioca.
  • Veza sa nulama: Ako je \(P(a) = 0\), tada je \((x - a)\) faktor polinoma \(P(x)\).
  • Hornerova shema: Efikasan nacin za izracunavanje vrednosti polinoma u tacki i za deljenje polinoma linearnim binomom.

Resenje korak po korak

1

Identifikujmo polinom i delilac

Dati su nam:

\[ P(x) = 3x^5 + 2x^4 + 3 \]

i delilac:

\[ d(x) = x + 1 \]

Zapazimo da je \(x + 1 = x - (-1)\), dakle u notaciji teoreme o ostatku imamo \(a = -1\).

2

Primenimo teoremu o ostatku

Prema teoremi o ostatku, ostatak \(R\) pri deljenju polinoma \(P(x)\) binomom \((x - a)\) je:

\[ R = P(a) \]

Posto je \(a = -1\), treba da izracunamo:

\[ R = P(-1) = 3 \cdot (-1)^5 + 2 \cdot (-1)^4 + 3 \]
Proveri se: Zasto koristimo \(a = -1\)?

Delilac je \(x + 1\). Teorema o ostatku zahteva oblik \((x - a)\). Zapisujemo:

\[ x + 1 = x - (-1) \]

Dakle, poredimo \(x - a = x - (-1)\) i zakljucujemo \(a = -1\). Zato uvrstimo \(-1\) u polinom.

3

Izracunajmo stepene broja \(-1\)

Pre nego sto uvrstimo, podsetimo se pravila za stepene negativnog broja:

\[ (-1)^n = \begin{cases} 1, & \text{ako je } n \text{ paran} \\ -1, & \text{ako je } n \text{ neparan} \end{cases} \]

Primenimo:

\[ (-1)^5 = -1 \quad \text{(5 je neparan)} \] \[ (-1)^4 = 1 \quad \text{(4 je paran)} \]
4

Uvrstimo i izracunajmo \(P(-1)\)

Zamenjujemo izracunate stepene u izraz za \(P(-1)\):

\[ \begin{aligned} P(-1) &= 3 \cdot (-1)^5 + 2 \cdot (-1)^4 + 3 \\[6pt] &= 3 \cdot (-1) + 2 \cdot 1 + 3 \\[6pt] &= -3 + 2 + 3 \\[6pt] &= \boxed{2} \end{aligned} \]

Dakle, ostatak pri deljenju je \(R = 2\).

Proveri se: Verifikacija Hornerovom shemom

Hornerova shema (sinteticko deljenje) za \(P(x) = 3x^5 + 2x^4 + 0x^3 + 0x^2 + 0x + 3\) sa \(c = -1\):

Koeficijenti polinoma su: \(3, \; 2, \; 0, \; 0, \; 0, \; 3\)

\(x^5\) \(x^4\) \(x^3\) \(x^2\) \(x^1\) \(x^0\)
Koef. 3 2 0 0 0 3
\(c = -1\) \(-3\) \(1\) \(-1\) \(1\) \(-1\)
Rez. 3 \(-1\) 1 \(-1\) 1 2

Postupak: svaki rezultat pomnozimo sa \(c = -1\) i dodamo sledecem koeficijentu.

\[ \begin{aligned} &3 \\ &3 \cdot (-1) + 2 = -3 + 2 = -1 \\ &(-1) \cdot (-1) + 0 = 1 + 0 = 1 \\ &1 \cdot (-1) + 0 = -1 + 0 = -1 \\ &(-1) \cdot (-1) + 0 = 1 + 0 = 1 \\ &1 \cdot (-1) + 3 = -1 + 3 = \mathbf{2} \end{aligned} \]

Poslednji broj u nizu je ostatak: \(R = 2\). Potvrdjeno!

Usput, dobijamo i kolicnik: \(Q(x) = 3x^4 - x^3 + x^2 - x + 1\), tj.:

\[ 3x^5 + 2x^4 + 3 = (x + 1)(3x^4 - x^3 + x^2 - x + 1) + 2 \]
5

Eliminacija ponudjenih odgovora

Pogledajmo ponudjene odgovore:

  • (A) \(x + 2\) i (E) \(3x + 1\) — Ovo su polinomi stepena 1. Medjutim, ostatak pri deljenju linearnim binomom je uvek konstanta (stepen 0). Ovi odgovori odmah otpadaju.
  • (B) \(-3\) — Ovo bi bio rezultat da smo samo izracunali \(3 \cdot (-1)^5 = -3\), zanemarujuci ostale sabirke.
  • (C) \(-2\) — Pogresno sabiranje: \(-3 + 2 + 3 \neq -2\).
  • (D) \(2\) — Tacan odgovor: \(-3 + 2 + 3 = 2\).
Proveri se: Dodatna verifikacija mnozeniem

Ako je nas rezultat tacan, onda mora da vazi:

\[ 3x^5 + 2x^4 + 3 = (x + 1)(3x^4 - x^3 + x^2 - x + 1) + 2 \]

Proverimo tako sto cemo pomnoziti desnu stranu:

\[ \begin{aligned} (x + 1)(3x^4 - x^3 + x^2 - x + 1) &= x \cdot (3x^4 - x^3 + x^2 - x + 1) \\ &\quad + 1 \cdot (3x^4 - x^3 + x^2 - x + 1) \\[6pt] &= 3x^5 - x^4 + x^3 - x^2 + x \\ &\quad + 3x^4 - x^3 + x^2 - x + 1 \\[6pt] &= 3x^5 + 2x^4 + 0x^3 + 0x^2 + 0x + 1 \end{aligned} \]

Dodamo ostatak 2:

\[ 3x^5 + 2x^4 + 1 + 2 = 3x^5 + 2x^4 + 3 \quad \checkmark \]

Sve se poklapa!

Kljucni uvid

Teorema o ostatku je izuzetno mocan alat: umesto dugog postupka deljenja polinoma, dovoljno je samo uvrstiti odgovarajucu vrednost u polinom. Kada delimo sa \(x + 1\), to je isto sto i \(x - (-1)\), pa uvrstimo \(x = -1\). Za polinome sa nedostajucim clanovima (kao ovde, gde nema \(x^3\), \(x^2\) ni \(x\)), ti clanovi prosto nestaju jer imaju koeficijent 0.

Konacan odgovor

(A) \(x + 2\)
(B) \(-3\)
(C) \(-2\)
(D) \(2\)
(E) \(3x + 1\)
(N) Ne znam
\(P(-1) = -3 + 2 + 3 = 2\)
Provera: Hornerova shema potvrdjuje: \(3x^5 + 2x^4 + 3 = (x+1)(3x^4 - x^3 + x^2 - x + 1) + 2\).

Ceste greske

  • Pogresan znak pri uvrstavanju: Mnogi studenti zamene \(x + 1\) sa \(x - 1\) i uvrstite \(x = 1\) umesto \(x = -1\). U tom slucaju bi dobili \(P(1) = 3 + 2 + 3 = 8\), sto nije medju ponudjenim odgovorima.
  • Greska sa parnoscu stepena: Ako zaboravite da je \((-1)^5 = -1\) (neparan stepen!) i stavite \((-1)^5 = 1\), dobicete \(3 + 2 + 3 = 8\) ili neku drugu pogresnu vrednost.
  • Zaboravljanje nedostajucih clanova: Polinom \(3x^5 + 2x^4 + 3\) nema clanove sa \(x^3\), \(x^2\) i \(x\). U Hornerovoj shemi je vazno staviti koeficijent 0 za svaki nedostajuci clan.
  • Ostatak viseg stepena: Odgovori (A) \(x + 2\) i (E) \(3x + 1\) su mamci. Ostatak pri deljenju polinomom prvog stepena je uvek konstanta, nikad polinom!

Bonus izazov

Nadji ostatak pri deljenju polinoma \(P(x) = x^{100} + x^{99} + 1\) binomom \(x + 1\).

Pogledaj nagovestaj

Primeni teoremu o ostatku: treba naci \(P(-1)\).

\[ P(-1) = (-1)^{100} + (-1)^{99} + 1 \]

Setimo se: \((-1)^{100} = 1\) (paran stepen) i \((-1)^{99} = -1\) (neparan stepen).

\[ P(-1) = 1 + (-1) + 1 = 1 \]

Ostatak je \(1\). Bez obzira koliko su veliki stepeni, pravilo parnosti sve resava!
