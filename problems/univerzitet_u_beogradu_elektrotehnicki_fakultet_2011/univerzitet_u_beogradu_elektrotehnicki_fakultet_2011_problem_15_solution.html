Iracionalna jednačina sa trigonometrijskim funkcijama

Koši-Švarcova nejednakost i analiza jednakosti za zbir korenova

Postavka zadatka

Zbir svih realnih rešenja jednačine

\[ \sqrt{\sin^2 x + \frac{1}{2}} + \sqrt{\cos^2 x + \frac{1}{2}} = 2 \]

Nijedan od ponuđenih odgovora

Plan rešavanja

  1. Uvedimo smenu \(a = \sin^2 x\), \(b = \cos^2 x\) i iskoristimo \(a + b = 1\).
  2. Primenom Koši-Švarcove nejednakosti pokažimo da je leva strana uvek \(\leq 2\).
  3. Nađimo kada važi jednakost: to zahteva \(\sin^2 x = \cos^2 x = \frac{1}{2}\).
  4. Odredimo sva rešenja na \([0, 2\pi]\) i izračunajmo njihov zbir.
Podsetnik: Koši-Švarcova nejednakost i konkavnost
  • Koši-Švarcova nejednakost (oblik za zbir korenova): \(\sqrt{u} + \sqrt{v} \leq \sqrt{2(u + v)}\) za \(u, v \geq 0\). Jednakost važi ako i samo ako je \(u = v\).
  • Pitagorin trigonometrijski identitet: \(\sin^2 x + \cos^2 x = 1\) za svako realno \(x\).
  • Konkavnost korena: Funkcija \(f(t) = \sqrt{t}\) je strogo konkavna na \((0, +\infty)\). Zbir dve konkavne funkcije je konkavan, pa \(g(a) = \sqrt{a + \tfrac{1}{2}} + \sqrt{\tfrac{3}{2} - a}\) ima jedinstven maksimum.
  • Rešenja \(\sin^2 x = \frac{1}{2}\): \(\sin x = \pm\frac{\sqrt{2}}{2}\), tj. \(x = \frac{\pi}{4} + k\cdot\frac{\pi}{2}\) za \(k \in \mathbb{Z}\).

Vizuelni prikaz

Graf funkcije \(f(x) = \sqrt{\sin^2 x + \frac{1}{2}} + \sqrt{\cos^2 x + \frac{1}{2}}\) na intervalu \([0, 2\pi]\). Horizontalna zelena linija označava nivo \(y = 2\). Pomerajte klizač da istražite vrednosti.

Rešenje korak po korak

1

Smena i pojednostavljenje

Neka je \(a = \sin^2 x\) i \(b = \cos^2 x\). Iz osnovnog trigonometrijskog identiteta:

\[ a + b = \sin^2 x + \cos^2 x = 1, \quad a \in [0,\,1],\; b \in [0,\,1] \]

Jednačina postaje:

\[ \sqrt{a + \tfrac{1}{2}} + \sqrt{b + \tfrac{1}{2}} = 2, \quad \text{gde je } b = 1 - a \]
2

Gornja granica: Koši-Švarcova nejednakost

Primenimo Koši-Švarcovu nejednakost na dva broja. Za \(u, v \geq 0\) važi:

\[ (\sqrt{u} + \sqrt{v})^2 \leq 2(u + v) \]

Ovo je specijalan slučaj nejednakosti \((\sqrt{u}\cdot 1 + \sqrt{v}\cdot 1)^2 \leq (u + v)(1 + 1)\).

Stavimo \(u = a + \frac{1}{2}\) i \(v = b + \frac{1}{2}\):

\[ \left(\sqrt{a + \tfrac{1}{2}} + \sqrt{b + \tfrac{1}{2}}\right)^2 \leq 2\!\left(\left(a + \tfrac{1}{2}\right) + \left(b + \tfrac{1}{2}\right)\right) = 2(a + b + 1) = 2 \cdot 2 = 4 \]

Dakle:

\[ \sqrt{a + \tfrac{1}{2}} + \sqrt{b + \tfrac{1}{2}} \leq \sqrt{4} = 2 \]

Leva strana jednačine je uvek manja ili jednaka 2, pa je vrednost 2 zapravo maksimum!

Proveri se: Zašto važi \((\sqrt{u} + \sqrt{v})^2 \leq 2(u+v)\)?

Razvijemo levu stranu: \((\sqrt{u} + \sqrt{v})^2 = u + 2\sqrt{uv} + v\).

Desna strana: \(2(u+v) = 2u + 2v\).

Nejednakost postaje: \(u + 2\sqrt{uv} + v \leq 2u + 2v\), tj. \(2\sqrt{uv} \leq u + v\).

Ovo je upravo AG-nejednakost (nejednakost aritmetičke i geometrijske sredine): \(\sqrt{uv} \leq \frac{u+v}{2}\), koja važi za sve \(u, v \geq 0\), a jednakost važi ako i samo ako je \(u = v\).

3

Uslov jednakosti

Jednakost u Koši-Švarcovoj nejednakosti (tj. u AG-nejednakosti) važi ako i samo ako je \(u = v\), tj.:

\[ a + \tfrac{1}{2} = b + \tfrac{1}{2} \quad \Longrightarrow \quad a = b \]

Pošto je \(a + b = 1\) i \(a = b\), sledi:

\[ 2a = 1 \quad \Longrightarrow \quad a = \frac{1}{2}, \quad b = \frac{1}{2} \]

Dakle, jednačina ima rešenje samo kada je \(\sin^2 x = \cos^2 x = \frac{1}{2}\).

4

Verifikacija: da li se zaista postiže vrednost 2?

Za \(a = b = \frac{1}{2}\):

\[ \sqrt{\tfrac{1}{2} + \tfrac{1}{2}} + \sqrt{\tfrac{1}{2} + \tfrac{1}{2}} = \sqrt{1} + \sqrt{1} = 1 + 1 = 2 \quad \checkmark \]

Maksimum se zaista dostiže, pa jednačina ima rešenja.

Proveri se: Kolika je vrednost \(f(x)\) kada \(\sin^2 x = 0\) (npr. za \(x = 0\))?

Za \(x = 0\): \(\sin^2 0 = 0\), \(\cos^2 0 = 1\).

\[ f(0) = \sqrt{0 + \tfrac{1}{2}} + \sqrt{1 + \tfrac{1}{2}} = \sqrt{\tfrac{1}{2}} + \sqrt{\tfrac{3}{2}} = \frac{1}{\sqrt{2}} + \frac{\sqrt{3}}{\sqrt{2}} = \frac{1 + \sqrt{3}}{\sqrt{2}} \approx \frac{2{,}732}{1{,}414} \approx 1{,}932 \]

Zaista manje od 2! Ovo potvrđuje da se maksimum ne postiže u krajnjim tačkama intervala za \(a\).

5

Nalaženje rešenja na \([0, 2\pi]\)

Potrebno je naći sva \(x \in [0, 2\pi]\) za koja je \(\sin^2 x = \frac{1}{2}\), tj.:

\[ |\sin x| = \frac{\sqrt{2}}{2} \quad \Longleftrightarrow \quad \sin x = \pm\frac{\sqrt{2}}{2} \]

Rešenja za \(\sin x = \frac{\sqrt{2}}{2}\):

\[ x_1 = \frac{\pi}{4}, \qquad x_2 = \pi - \frac{\pi}{4} = \frac{3\pi}{4} \]

Rešenja za \(\sin x = -\frac{\sqrt{2}}{2}\):

\[ x_3 = \pi + \frac{\pi}{4} = \frac{5\pi}{4}, \qquad x_4 = 2\pi - \frac{\pi}{4} = \frac{7\pi}{4} \]

Proverimo da li za svako od ovih rešenja važi i \(\cos^2 x = \frac{1}{2}\):

\[ \cos^2\!\left(\frac{\pi}{4}\right) = \left(\frac{\sqrt{2}}{2}\right)^2 = \frac{1}{2} \;\checkmark \qquad \cos^2\!\left(\frac{3\pi}{4}\right) = \left(-\frac{\sqrt{2}}{2}\right)^2 = \frac{1}{2} \;\checkmark \]

Analogno za \(\frac{5\pi}{4}\) i \(\frac{7\pi}{4}\). Sva četiri rešenja zadovoljavaju oba uslova.

6

Zbir svih rešenja

Sabiramo sva četiri rešenja:

\[ S = \frac{\pi}{4} + \frac{3\pi}{4} + \frac{5\pi}{4} + \frac{7\pi}{4} = \frac{(1 + 3 + 5 + 7)\,\pi}{4} = \frac{16\pi}{4} = 4\pi \]
Proveri se: Brza provera zbira 1 + 3 + 5 + 7

Zbir prvih \(n\) neparnih brojeva je \(n^2\). Ovde imamo 4 neparna broja: \(1 + 3 + 5 + 7 = 4^2 = 16\). Alternativno: \((1+7) + (3+5) = 8 + 8 = 16\).

Ključni uvid

Jednačina je oblika \(\sqrt{u} + \sqrt{v} = \sqrt{2(u+v)}\), što je gornja granica koja se dostiže samo kada su \(u\) i \(v\) jednaki. Umesto da pokušavamo komplikovane algebarske transformacije (kvadriranje, itd.), prepoznajemo da je desna strana upravo maksimum leve strane, pa rešenja nalazimo iz uslova jednakosti.

\[ \underbrace{\sqrt{a + \tfrac{1}{2}} + \sqrt{b + \tfrac{1}{2}}}_{\text{uvek } \leq\, 2} = 2 \quad \Longrightarrow \quad a = b = \frac{1}{2} \quad \Longrightarrow \quad \sin^2 x = \cos^2 x = \frac{1}{2} \]

Konačan odgovor

Rešenja na \([0, 2\pi]\) su \(x \in \left\{\frac{\pi}{4},\; \frac{3\pi}{4},\; \frac{5\pi}{4},\; \frac{7\pi}{4}\right\}\).

(A) \(\pi\)
(B) \(2\pi\)
(C) \(3\pi\)
(D) \(4\pi\)
(E) Nijedan
Odgovor je (D) \(4\pi\)

Zbir \(\frac{\pi}{4} + \frac{3\pi}{4} + \frac{5\pi}{4} + \frac{7\pi}{4} = \frac{16\pi}{4} = 4\pi\).

Provera: Numerički: \(f(\pi/4) = \sqrt{1} + \sqrt{1} = 2\;\checkmark\). Za bilo koje drugo \(x\), npr. \(x = 0\): \(f(0) \approx 1{,}932 < 2\). Zaista, samo tačke gde je \(\sin^2 x = \cos^2 x = \frac{1}{2}\) zadovoljavaju jednačinu.

Česte greške

  • Kvadriranje bez provere: Mnogi učenici pokušavaju da kvadriraju obe strane jednačine. To uvodi složene izraze i lažna rešenja. Elegantnije je prepoznati da je 2 maksimalna vrednost leve strane.
  • Zaboravljanje negativnih vrednosti sinusa: Uslov je \(\sin^2 x = \frac{1}{2}\), što daje i pozitivne i negativne vrednosti sinusa. To znači 4 rešenja, ne samo 2!
  • Segment ili interval: Jednačina je na segmentu \([0, 2\pi]\) (zatvoreni interval). Vrednosti \(x = 0\) i \(x = 2\pi\) ne zadovoljavaju jednačinu, pa to ne utiče na zbir, ali treba proveriti.
  • Zbir umesto broja rešenja: Pitanje traži zbir rešenja, ne njihov broj. Broj rešenja je 4, ali zbir je \(4\pi\).

Bonus izazov

Nađi zbir svih rešenja jednačine \(\sqrt{\sin^2 x + \frac{1}{2}} + \sqrt{\cos^2 x + \frac{1}{2}} = c\) na segmentu \([0, 2\pi]\) u zavisnosti od parametra \(c\). Za koje vrednosti \(c\) jednačina ima rešenja?

Pogledaj nagoveštaj

Definišimo \(g(a) = \sqrt{a + \frac{1}{2}} + \sqrt{\frac{3}{2} - a}\) za \(a \in [0, 1]\).

Minimum: za \(a = 0\) ili \(a = 1\), \(g = \frac{1+\sqrt{3}}{\sqrt{2}} \approx 1{,}932\).

Maksimum: za \(a = \frac{1}{2}\), \(g = 2\).

Dakle, jednačina ima rešenja samo za \(c \in \left[\frac{1+\sqrt{3}}{\sqrt{2}},\, 2\right]\). Za \(c = 2\) zbir je \(4\pi\). Za \(c < 2\) postoji 8 rešenja (jer funkcija \(g(\sin^2 x)\) dostiže svaku vrednost iz unutrašnjosti opsega na 8 tačaka), a zbir je uvek \(8\pi\) (po simetriji).
