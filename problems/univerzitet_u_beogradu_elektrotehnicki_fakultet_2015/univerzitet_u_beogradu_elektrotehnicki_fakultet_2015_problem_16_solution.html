Broj parova (p, q) za deljivost polinoma

Algebra polinoma, deljivost, sistem jednacina

Zadatak

Broj parova \((p, q)\), \(p, q \in \mathbb{R}\) takvih da je polinom \(x^4 + px^2 + q\) deljiv polinomom \(x^2 + px + q\), jednak je:

Deljenik
\(x^4 + px^2 + q\)
Delilac
\(x^2 + px + q\)
(A)
\(0\)
(B)
\(2\)
(C)
\(1\)
(D)
\(4\)
(E)
\(5\)

Plan resavanja

Ako je \(x^4 + px^2 + q\) deljiv sa \(x^2 + px + q\), tada postoji polinom drugog stepena \(x^2 + bx + c\) takav da vazi:

\[ x^4 + px^2 + q = (x^2 + px + q)(x^2 + bx + c) \]

Raspisacemo desnu stranu, uporediti koeficijente uz iste stepene promenljive \(x\) i dobiti sistem jednacina po \(p\), \(q\), \(b\) i \(c\). Sistematskom analizom slucajeva naci cemo sve parove \((p, q)\).

Kljucne formule i teoreme
  • Deljivost polinoma: Polinom \(f(x)\) je deljiv polinomom \(g(x)\) ako postoji polinom \(h(x)\) takav da \(f(x) = g(x) \cdot h(x)\).
  • Metod neodredjenih koeficijenata: Ako znamo stepene deljivca i kolicnika, mozemo zapisati kolicnik sa nepoznatim koeficijentima, raspisati proizvod i uporediti koeficijente uz iste stepene.
  • Jednakost polinoma: Dva polinoma su jednaka ako i samo ako su im jednaki svi koeficijenti uz iste stepene promenljive.
  • Sistematska analiza slucajeva: Kada sistem jednacina sadrzi proizvod faktora jednak nuli, razdvajamo na slucajeve i resavamo svaki posebno.

Vizuelni prikaz: Svih 5 parova

Koordinatna ravan sa tackama \((p,q)\) koje zadovoljavaju uslov deljivosti.

Prevucite misem za pomeranje prikaza

Resenje korak po korak

Korak 1
Postavljanje jednakosti

Ako je \(x^4 + px^2 + q\) deljiv sa \(x^2 + px + q\), tada je kolicnik polinom drugog stepena. Pisemo:

\[ x^4 + px^2 + q = (x^2 + px + q)(x^2 + bx + c) \]

gde su \(b, c \in \mathbb{R}\) nepoznati koeficijenti kolicnika.

Korak 2
Raspisivanje desne strane

Mnozimo polinome na desnoj strani:

\[ (x^2 + px + q)(x^2 + bx + c) \]
\[ = x^4 + bx^3 + cx^2 + px^3 + pbx^2 + pcx + qx^2 + qbx + qc \]

Grupisanjem po stepenima dobijamo:

\[ = x^4 + (p+b)x^3 + (q + c + pb)x^2 + (pc + bq)x + qc \]
Korak 3
Izjednacavanje koeficijenata

Leva strana je \(x^4 + 0 \cdot x^3 + p \cdot x^2 + 0 \cdot x + q\). Izjednacavamo koeficijente:

\[ \begin{aligned} x^3: & \quad p + b = 0 & \Rightarrow \quad & b = -p \\[6pt] x^2: & \quad q + c + pb = p & \Rightarrow \quad & q + c - p^2 = p \\[6pt] x^1: & \quad pc + bq = 0 & \Rightarrow \quad & pc - pq = 0 \\[6pt] x^0: & \quad qc = q & & \end{aligned} \]

Iz jednacine (1) odmah imamo \(b = -p\). Jednacine (3) i (4) daju kljucna razgranicenja.

Proveri se: Zasto nema koeficijenta uz \(x^3\) i \(x\) na levoj strani?

Polinom \(x^4 + px^2 + q\) sadrzi samo parne stepene promenljive \(x\) (plus slobodan clan). Koeficijenti uz \(x^3\) i \(x\) su jednaki nuli, sto nam daje dve jednacine: \(p + b = 0\) i \(pc + bq = 0\).

Korak 4
Analiza jednacine (4): \(qc = q\)

Iz \(qc = q\) sledi:

\[ q(c - 1) = 0 \quad \Rightarrow \quad q = 0 \;\text{ ili }\; c = 1 \]

Ovo nam daje dva glavna slucaja za dalju analizu.

Korak 5
Slucaj 1: \(q = 0\)

Iz jednacine (3): \(pc - pq = p(c - q) = p \cdot c = 0\), pa je \(p = 0\) ili \(c = 0\).

Slucaj 1a: \(q = 0\), \(p = 0\)

Iz jednacine (2): \(0 + c - 0 = 0 \Rightarrow c = 0\). Tada je \(b = 0\).

\[ x^4 = x^2 \cdot x^2 \quad \checkmark \]

(p, q) = (0, 0)

Slucaj 1b: \(q = 0\), \(c = 0\), \(p \neq 0\)

Iz jednacine (2): \(0 + 0 - p^2 = p \Rightarrow p^2 + p = 0 \Rightarrow p(p + 1) = 0\).

Posto je \(p \neq 0\), sledi \(p = -1\). Tada je \(b = 1\).

\[ x^4 - x^2 = x^2(x^2 - 1) = x^2(x-1)(x+1) \]
\[ x^2 - x = x(x-1) \]
\[ \frac{x^2(x-1)(x+1)}{x(x-1)} = x(x+1) = x^2 + x \quad \checkmark \]

Proverimo: \((x^2 - x)(x^2 + x) = x^4 + x^3 - x^3 - x^2 = x^4 - x^2\). Tacno!

(p, q) = (-1, 0)

Proveri se: Zasto (x^2 - x) deli (x^4 - x^2) kao polinom?

Deljivost polinoma znaci da ostatak pri deljenju mora biti nula-polinom. Zaista: \(x^4 - x^2 = (x^2 - x)(x^2 + x) + 0\). Kolicnik \(x^2 + x\) je polinom, pa je deljivost zadovoljena.

Korak 6
Slucaj 2: \(c = 1\)

Iz jednacine (3): \(p(c - q) = p(1 - q) = 0\), pa je \(p = 0\) ili \(q = 1\).

Slucaj 2a: \(c = 1\), \(p = 0\)

Tada je \(b = 0\). Iz jednacine (2): \(q + 1 - 0 = 0 \Rightarrow q = -1\).

\[ x^4 - 1 = (x^2 - 1)(x^2 + 1) \quad \checkmark \]

(p, q) = (0, -1)

Slucaj 2b: \(c = 1\), \(q = 1\)

Tada je \(b = -p\). Iz jednacine (2): \(1 + 1 - p^2 = p \Rightarrow p^2 + p - 2 = 0\).

\[ (p + 2)(p - 1) = 0 \quad \Rightarrow \quad p = -2 \;\text{ ili }\; p = 1 \]

Slucaj 2b-i: \(p = -2\), \(q = 1\)

\(b = 2\), \(c = 1\).

\[ x^4 - 2x^2 + 1 = (x^2 - 1)^2 = (x-1)^2(x+1)^2 \]
\[ x^2 - 2x + 1 = (x - 1)^2 \]
\[ \frac{(x-1)^2(x+1)^2}{(x-1)^2} = (x+1)^2 = x^2 + 2x + 1 \quad \checkmark \]

(p, q) = (-2, 1)

Slucaj 2b-ii: \(p = 1\), \(q = 1\)

\(b = -1\), \(c = 1\).

\[ x^4 + x^2 + 1 = (x^2 + x + 1)(x^2 - x + 1) \quad \checkmark \]

Verifikacija: \((x^2 + x + 1)(x^2 - x + 1) = x^4 - x^3 + x^2 + x^3 - x^2 + x + x^2 - x + 1 = x^4 + x^2 + 1\).

(p, q) = (1, 1)

Proveri se: Da li smo pokrili sve slucajeve?

Iz jednacine (4) imamo \(q = 0\) ili \(c = 1\) — to su jedine dve mogucnosti.
Za \(q = 0\): iz jednacine (3) imamo \(p = 0\) ili \(c = 0\) — oba analizirana.
Za \(c = 1\): iz jednacine (3) imamo \(p = 0\) ili \(q = 1\) — oba analizirana.
Za \(q = 1\): resenje kvadratne jednacine daje dva korena — oba ukljucena.
Dakle, svi slucajevi su potpuno pokriveni.

Korak 7
Pregled svih resenja

Svih 5 parova \((p, q)\) koji zadovoljavaju uslov su:

(0, 0) (-1, 0) (0, -1) (-2, 1) (1, 1)
\((p, q)\) Deljenik Delilac Kolicnik
\((0,\;0)\) \(x^4\) \(x^2\) \(x^2\)
\((-1,\;0)\) \(x^4 - x^2\) \(x^2 - x\) \(x^2 + x\)
\((0,\;-1)\) \(x^4 - 1\) \(x^2 - 1\) \(x^2 + 1\)
\((-2,\;1)\) \(x^4 - 2x^2 + 1\) \(x^2 - 2x + 1\) \(x^2 + 2x + 1\)
\((1,\;1)\) \(x^4 + x^2 + 1\) \(x^2 + x + 1\) \(x^2 - x + 1\)

Kljucni uvid

Kljuc resenja je u sistematskoj analizi slucajeva koja proistice iz jednacina \(qc = q\) i \(p(c - q) = 0\). Jednacina \(q(c-1) = 0\) razdvaja problem na dva glavna slucaja (\(q = 0\) i \(c = 1\)), a unutar svakog slucaja jednacina \(p(c-q) = 0\) daje dalja grananja. Svaka grana se na kraju svodi na prosto resavanje linearne ili kvadratne jednacine po \(p\).

\[ q(c-1) = 0 \;\text{ i }\; p(c-q) = 0 \quad \longrightarrow \quad 5 \text{ parova } (p,q) \]

Konacan odgovor

Broj parova \((p, q)\) koji zadovoljavaju uslov:

\(\displaystyle \text{(E)}\ 5\)
(0, 0) (-1, 0) (0, -1) (-2, 1) (1, 1)
(A) 0
(B) 2
(C) 1
(D) 4
(E) 5

Provera: Svaki od 5 parova je pojedinacno verifikovan mnozenjem delioca i kolicnika, pri cemu se u svakom slucaju dobija polazni deljenik.

Ceste greske

  • Zaboravljanje trivijalnog resenja \((0, 0)\): posto je \(x^4 \div x^2 = x^2\), par \((0, 0)\) je validan, ali se cesto previdi.
  • Pogresno tumacenje "deljivosti" kao deljenja brojeva umesto polinoma: deljivost polinoma znaci da kolicnik mora biti polinom (bez ostatka), ne samo da izraz ima smisla za neke vrednosti \(x\).
  • Nepotpuna analiza slucajeva: propustanje neke grane u razgranavanju, npr. kada \(q = 1\) daje kvadratnu jednacinu sa dva resenja, a ne samo jedno.
  • Greska u raspisivanju proizvoda polinoma: treba pazljivo sastaviti sve sabirke pri mnozenju dva polinoma drugog stepena.

Dodatni izazov

Za koliko parova \((p, q) \in \mathbb{R}^2\) je polinom \(x^4 + px^2 + q\) deljiv polinomom \(x^2 + px + q\), pri cemu delilac \(x^2 + px + q\) nema realnih korena?

Pomoc

Proverimo diskriminantu delioca \(D = p^2 - 4q\) za svaki od 5 parova:
\((0, 0)\): \(D = 0\) — ima dvostruki koren \(x = 0\). Ima realne korene.
\((-1, 0)\): \(D = 1\) — ima dva realna korena. Ima realne korene.
\((0, -1)\): \(D = 4\) — ima dva realna korena. Ima realne korene.
\((-2, 1)\): \(D = 0\) — ima dvostruki koren \(x = 1\). Ima realne korene.
\((1, 1)\): \(D = 1 - 4 = -3 < 0\) — nema realnih korena!
Dakle, tacno 1 par: \((1, 1)\).
