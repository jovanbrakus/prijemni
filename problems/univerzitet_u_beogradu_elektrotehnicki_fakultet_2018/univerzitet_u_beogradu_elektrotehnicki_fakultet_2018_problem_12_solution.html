Nejednacina sa korenom: \(x + 2 < \sqrt{x + 44}\)

Odredjivanje oblika skupa resenja iracionalne nejednacine

📜 Tekst zadatka
Skup resenja nejednacine \(x + 2 < \sqrt{x + 44}\) je oblika (za neke realne brojeve \(a, b, c, d\) takve da je \(-\infty < a < b < c < d < +\infty\)):
(A) \([a, b)\)
(B) \((a, b) \cup (c, d)\)
(C) \((a, b) \cup [c, +\infty)\)
(D) \((-\infty, b] \cup [c, +\infty)\)
(E) \([d, +\infty)\)
📌 Plan resavanja

1. Odrediti domen nejednacine (uslov pod korenom).
2. Razdvojiti na dva slucaja prema znaku leve strane: kada je \(x + 2 < 0\) (nejednacina vazi automatski) i kada je \(x + 2 \ge 0\) (mozemo kvadrirati).
3. U drugom slucaju resiti kvadratnu nejednacinu nastalu kvadriranjem.
4. Ujediniti resenja oba slucaja i prepoznati oblik intervala.

📚 Podsetnik iz teorije
Iracionalne nejednacine oblika \(f(x) < \sqrt{g(x)}\)

Da bismo resili nejednacinu \(f(x) < \sqrt{g(x)}\), moramo razlikovati dva slucaja:

  • Ako je \(f(x) < 0\): leva strana je negativna, a \(\sqrt{g(x)} \ge 0\), pa nejednacina vazi automatski (uz uslov da je \(g(x) \ge 0\)).
  • Ako je \(f(x) \ge 0\): obe strane su nenegativne, pa mozemo kvadrirati bez promene smera nejednakosti: \(f(x)^2 < g(x)\).
Kvadratna nejednacina \(ax^2 + bx + c < 0\)

Za \(a > 0\), nejednacina \(ax^2 + bx + c < 0\) ima resenja izmedju nula polinoma (ako postoje): \[x_1 < x < x_2\] gde su \(x_1 < x_2\) koreni jednacine \(ax^2 + bx + c = 0\).

Domen funkcije \(\sqrt{g(x)}\)

Kvadratni koren \(\sqrt{g(x)}\) je definisan samo za \(g(x) \ge 0\). To ogranicenje odredjuje domen cele nejednacine.

🛠 Resenje korak po korak
1
Odredjivanje domena

Kvadratni koren zahteva nenegativan argument:

\[x + 44 \ge 0 \quad \Rightarrow \quad x \ge -44\]

Domen nejednacine je \(D = [-44, +\infty)\).

2
Slucaj 1: \(x + 2 < 0\), tj. \(x < -2\)

Kada je \(x < -2\), leva strana nejednacine je negativna:

\[x + 2 < 0 \le \sqrt{x + 44}\]

Posto je kvadratni koren uvek nenegativan, nejednacina \(x + 2 < \sqrt{x + 44}\) vazi automatski za svako \(x\) iz domena gde je \(x < -2\).

Presek sa domenom:

\[x \ge -44 \;\;\text{i}\;\; x < -2 \quad \Rightarrow \quad x \in [-44, -2)\]
✅ Proveri sebe: Zasto je \(-44\) ukljuceno?

Za \(x = -44\): leva strana je \(-44 + 2 = -42 < 0\), a desna je \(\sqrt{-44 + 44} = 0\). Vazi \(-42 < 0\), dakle \(x = -44\) jeste resenje i pripada skupu.

3
Slucaj 2: \(x + 2 \ge 0\), tj. \(x \ge -2\)

Obe strane su nenegativne, pa smemo kvadrirati (cuvamo smer nejednakosti):

\[(x + 2)^2 < x + 44\]

Razvijamo levu stranu:

\[x^2 + 4x + 4 < x + 44\]

Prenosimo sve na levu stranu:

\[x^2 + 3x - 40 < 0\]

Resavamo kvadratnu jednacinu \(x^2 + 3x - 40 = 0\):

\[D = 9 + 160 = 169, \quad \sqrt{D} = 13\]
\[x_1 = \frac{-3 - 13}{2} = -8, \qquad x_2 = \frac{-3 + 13}{2} = 5\]

Faktorizacija: \(x^2 + 3x - 40 = (x + 8)(x - 5)\).

Nejednacina \((x + 8)(x - 5) < 0\) vazi za:

\[x \in (-8, 5)\]

Presek sa uslovom \(x \ge -2\):

\[(-8, 5) \cap [-2, +\infty) = [-2, 5)\]
✅ Proveri sebe: Zasto je \(x = 5\) iskljuceno?

Za \(x = 5\): leva strana je \(5 + 2 = 7\), desna je \(\sqrt{5 + 44} = \sqrt{49} = 7\). Vazi \(7 < 7\)? Ne! Jednakost, ne stroga nejednacina. Zato \(x = 5\) ne pripada skupu resenja.

4
Unija oba slucaja

Spajamo resenja iz oba slucaja:

\[[-44, -2) \cup [-2, 5)\]

Posto se ovi intervali dodiruju u tacki \(-2\) (koja pripada drugom intervalu), unija daje jedan interval:

\[[-44, 5)\]

Ovo je interval oblika \([a, b)\) gde je \(a = -44\) i \(b = 5\).

✅ Proveri sebe: Verifikacija na granicnim tackama

\(x = -44\): \(-42 < \sqrt{0} = 0\) ✔ (jeste resenje)
\(x = -2\): \(0 < \sqrt{42} \approx 6{,}48\) ✔ (jeste resenje)
\(x = 0\): \(2 < \sqrt{44} \approx 6{,}63\) ✔ (jeste resenje)
\(x = 5\): \(7 < \sqrt{49} = 7\) ✘ (nije resenje — jednakost)
\(x = 10\): \(12 < \sqrt{54} \approx 7{,}35\) ✘ (nije resenje)

💡 Kljucni uvid

Kljucna tehnika kod iracionalnih nejednacina je razdvajanje na slucajeve prema znaku strane koja ne sadrzi koren. Kada je \(x + 2 < 0\), ne trebamo nista racunati — nejednacina vazi automatski jer je negativan broj uvek manji od nenegativnog korena. Ovaj "besplatan" deo resenja cini veliki interval \([-44, -2)\), koji se zatim spaja sa intervalom \([-2, 5)\) dobijenim kvadriranjem.

🎯 Konacan odgovor
\([-44, 5)\) — oblik \([a, b)\)
(A) \([a, b)\)
(B) \((a, b) \cup (c, d)\)
(C) \((a, b) \cup [c, +\infty)\)
(D) \((-\infty, b] \cup [c, +\infty)\)
(E) \([d, +\infty)\)

Verifikacija: \(a = -44,\; b = 5\), skup resenja je \([-44, 5)\) ✔

⚠️ Ceste greske
❌
Kvadriranje bez provere znaka: Ako odmah kvadrirate obe strane bez razdvajanja na slucajeve, dobicete samo interval \((-8, 5)\) — cime gubite veliki deo resenja \([-44, -8]\). Kvadriranje je dozvoljeno samo kada su obe strane nenegativne!
❌
Zaboravljanje domena: Bez uslova \(x \ge -44\) bismo mogli pomisliti da nejednacina vazi za sve \(x < -2\), sto bi dalo \((-\infty, 5)\) umesto tacnog \([-44, 5)\). Domen je ogranicen uslovom pod korenom.
❌
Ukljucivanje granicne tacke \(x = 5\): Za \(x = 5\) vazi jednakost (\(7 = 7\)), a nejednacina zahteva strogu nejednakost (\( < \)). Zato je \(5\) iskljuceno i interval je poluotvoren: \([-44, 5)\), ne \([-44, 5]\).
🏋 Dodatni izazov

Kako bi se promenio skup resenja da nejednacina glasi \(x + 2 \le \sqrt{x + 44}\) (sa znakom \(\le\) umesto \( < \))?

Nagoveštaj

Jedina razlika je u granicnoj tacki \(x = 5\), gde vazi jednakost \(7 = 7\). Sa znakom \(\le\), tacka \(x = 5\) bi bila ukljucena, pa bi skup resenja bio \([-44, 5]\) — zatvoren interval. To bi i dalje bio oblik (A), samo sa drugacijim tipom intervala.
