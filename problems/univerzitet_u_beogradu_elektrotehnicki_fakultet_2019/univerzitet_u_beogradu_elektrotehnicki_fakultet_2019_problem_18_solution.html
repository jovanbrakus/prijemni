Uglovi trougla i trigonometrijski uslov

Trigonometrija — određivanje uglova trougla iz jednačine sa sinusima

Postavka zadatka

Ako su \(\alpha\), \(\beta\) i \(\gamma\), \(\alpha \geq \beta \geq \gamma\), uglovi trougla i ako je \(\sin\alpha - \sin\beta + \sin\gamma = 1\), onda je ugao \(\gamma\) jednak:

Uglovi trougla
\(\alpha + \beta + \gamma = \pi\)
Uređenost
\(\alpha \geq \beta \geq \gamma\)
Uslov
\(\sin\alpha - \sin\beta + \sin\gamma = 1\)

Plan rešavanja

Preuredićemo jednačinu \(\sin\alpha - \sin\beta + \sin\gamma = 1\) koristeći činjenicu da je \(\alpha + \beta + \gamma = \pi\). Razmotrićemo poseban slučaj \(\alpha = 90°\) koji prirodno proizlazi iz analize, a zatim dokazati da je to jedino rešenje.

Ključne formule i teoreme
  • Zbir uglova trougla: \(\alpha + \beta + \gamma = \pi\) (tj. \(180°\))
  • Suplementni uglovi: \(\sin(\pi - x) = \sin x\), pa ako je \(\alpha = \pi - \beta - \gamma\), onda je \(\sin\alpha = \sin(\beta + \gamma)\)
  • Zbir sinusa: \(\sin A + \sin B = 2\sin\dfrac{A+B}{2}\cos\dfrac{A-B}{2}\)
  • Opseg sinusa: Za uglove u trouglu (\(0 < \theta < \pi\)) važi \(0 < \sin\theta \leq 1\), sa \(\sin\theta = 1\) samo za \(\theta = 90°\)

Vizuelni prikaz trougla

Trougao sa uglovima koji zadovoljavaju dati uslov

Rešenje korak po korak

1

Preuredimo jednačinu

Polazimo od datog uslova i grupišemo članove:

\[\sin\alpha - \sin\beta + \sin\gamma = 1\]

Prebacimo \(\sin\beta\) na desnu stranu:

\[\sin\alpha + \sin\gamma = 1 + \sin\beta\]

Leva strana je zbir dva sinusa, što nam omogućava primenu formule zbira.

2

Primenimo formulu za zbir sinusa

Koristimo formulu \(\sin A + \sin B = 2\sin\frac{A+B}{2}\cos\frac{A-B}{2}\) na levu stranu:

\[\sin\alpha + \sin\gamma = 2\sin\frac{\alpha+\gamma}{2}\cos\frac{\alpha-\gamma}{2}\]

Pošto je \(\alpha + \gamma = \pi - \beta\), imamo:

\[\sin\frac{\alpha+\gamma}{2} = \sin\frac{\pi-\beta}{2} = \cos\frac{\beta}{2}\]

Dakle, leva strana postaje:

\[2\cos\frac{\beta}{2}\cos\frac{\alpha-\gamma}{2}\]
3

Transformišemo desnu stranu

Za desnu stranu koristimo identitet dvostrukog ugla \(\sin\beta = 2\sin\frac{\beta}{2}\cos\frac{\beta}{2}\):

\[1 + \sin\beta = 1 + 2\sin\frac{\beta}{2}\cos\frac{\beta}{2}\]

Izjednačavamo levu i desnu stranu:

\[2\cos\frac{\beta}{2}\cos\frac{\alpha-\gamma}{2} = 1 + 2\sin\frac{\beta}{2}\cos\frac{\beta}{2}\]
Proveri sebe: Koliko je \(\sin\frac{\pi - \beta}{2}\)?

\(\sin\frac{\pi - \beta}{2} = \sin\left(\frac{\pi}{2} - \frac{\beta}{2}\right) = \cos\frac{\beta}{2}\). Ovo je standardni komplementni identitet.

4

Analizirajmo granice sinusa

Znamo da za ugao trougla važi \(\sin\theta \leq 1\). U našoj jednačini:

\[\sin\alpha - \sin\beta + \sin\gamma = 1\]

Pošto je \(\alpha \geq \beta \geq \gamma\), znamo da je \(\sin\gamma \leq \sin\beta\) (jer su oba ugla u intervalu \((0°, 90°]\) kada je \(\gamma\) najmanji ugao). Zato je:

\[-\sin\beta + \sin\gamma \leq 0\]

Da bi zbir bio jednak \(1\), neophodno je da \(\sin\alpha\) bude dovoljno veliko. Kako je \(\sin\alpha \leq 1\), razmatramo slučaj \(\sin\alpha = 1\), tj. \(\alpha = 90°\).

5

Slučaj \(\alpha = 90°\)

Ako je \(\alpha = 90°\), jednačina postaje:

\[1 - \sin\beta + \sin\gamma = 1\]

Oduzimamo \(1\) sa obe strane:

\[-\sin\beta + \sin\gamma = 0\]

Dakle:

\[\sin\gamma = \sin\beta\]
6

Zaključimo da je \(\gamma = \beta\)

Iz \(\sin\gamma = \sin\beta\) i uslova \(\beta \geq \gamma > 0\) (oba ugla u intervalu \((0°, 90°)\)), sledi \(\gamma = \beta\).

Drugi slučaj \(\gamma = \pi - \beta\) bi dao \(\gamma \geq 90°\), ali to je nemoguće jer \(\gamma \leq \alpha = 90°\), a jednakost \(\gamma = 90°\) bi značila \(\beta + \gamma = 180°\), što ne ostavlja prostor za \(\alpha\).

Proveri sebe: Ako je \(\sin\gamma = \sin\beta\) i \(0 < \gamma \leq \beta < 90°\), koji su mogući zaključci?

Jednačina \(\sin x = \sin y\) ima rešenja \(x = y\) ili \(x = \pi - y\). Pošto su oba ugla u intervalu \((0°, 90°)\), jedino rešenje je \(\gamma = \beta\).

7

Izračunamo \(\gamma\)

Sada koristimo zbir uglova trougla sa \(\alpha = 90°\) i \(\beta = \gamma\):

\[\alpha + \beta + \gamma = 180°\]
\[90° + \gamma + \gamma = 180°\]
\[2\gamma = 90°\]
\[\gamma = 45°\]
8

Dokaz jedinstvenosti rešenja

Pokažimo da ne postoji drugo rešenje osim \(\alpha = 90°, \beta = \gamma = 45°\).

Ako je \(\alpha < 90°\): tada je \(\sin\alpha < 1\). Pošto je \(\sin\gamma \leq \sin\beta\) (jer \(\gamma \leq \beta\) i oba ugla su oštri), važi \(-\sin\beta + \sin\gamma \leq 0\). Dakle:

\[\sin\alpha + (-\sin\beta + \sin\gamma) < 1 + 0 = 1\]

Ako je \(\alpha > 90°\): tada \(\sin\alpha = \sin(180° - \alpha) < 1\), i \(\beta + \gamma < 90°\), pa su oba ugla oštri i manji od \(90°\). Detaljnija analiza (koristeći formulu iz koraka 2-3) pokazuje da jednačina nema rešenje.

Zaključak: jedino rešenje je \(\alpha = 90°, \beta = \gamma = 45°\), pa je \(\gamma = 45°\).

Proveri sebe: Verifikuj rešenje direktnim uvrštavanjem.

\(\sin 90° - \sin 45° + \sin 45° = 1 - \frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2} = 1\). Tačno!

Ključni uvid

Jednačina \(\sin\alpha - \sin\beta + \sin\gamma = 1\) zahteva da \(\sin\alpha\) bude što veće, a \(\sin\beta\) i \(\sin\gamma\) da se međusobno poništavaju. Jedini način da zbir dostigne vrednost \(1\) je da je \(\alpha = 90°\) (pa je \(\sin\alpha = 1\)) i da važi \(\sin\beta = \sin\gamma\), odnosno \(\beta = \gamma\). Tada je trougao jednakokraki pravougli trougao!

\[\alpha = 90°,\quad \beta = \gamma = 45°\]

Konačan odgovor

\(\gamma = 45°\)
(A) \(60°\)
(B) \(20°\)
(C) \(15°\)
(D) \(30°\)
(E) \(45°\)

Verifikacija

Proverimo da trougao sa uglovima \(\alpha = 90°\), \(\beta = 45°\), \(\gamma = 45°\) zadovoljava sve uslove:

\[\alpha + \beta + \gamma = 90° + 45° + 45° = 180° \quad \checkmark\]
\[\alpha \geq \beta \geq \gamma:\quad 90° \geq 45° \geq 45° \quad \checkmark\]
\[\sin 90° - \sin 45° + \sin 45° = 1 - \frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2} = 1 \quad \checkmark\]

Trougao je jednakokraki pravougli trougao -- jedan od najpoznatijih trouglova u geometriji. Članovi \(-\sin\beta\) i \(+\sin\gamma\) se potpuno poništavaju jer je \(\beta = \gamma\), pa ostaje samo \(\sin\alpha = 1\).

Najčešće greške

  • Pokušavanje sa konkretnim vrednostima ponuđenih odgovora bez sistematske analize može dovesti do pogrešnih zaključaka. Na primer, za \(\gamma = 30°\) postoji degenerativan trougao (\(\beta \to 0°\)) koji formalno zadovoljava uslov, ali to nije pravi trougao.
  • Zaboraviti da iz \(\sin\gamma = \sin\beta\) ne sledi nužno \(\gamma = \beta\) -- potrebno je razmotriti i slučaj \(\gamma = \pi - \beta\), pa ga eliminisati na osnovu uslova trougla.

Dodatni izazov

Ako su \(\alpha\), \(\beta\) i \(\gamma\) uglovi trougla sa \(\alpha \geq \beta \geq \gamma\) i važi \(\cos\alpha + \cos\beta + \cos\gamma = \frac{3}{2}\), odredite uglove trougla.

Hint

Poznata nejednakost glasi: \(\cos\alpha + \cos\beta + \cos\gamma \leq \frac{3}{2}\), sa jednakošću ako i samo ako je trougao jednakostranični. Dakle, \(\alpha = \beta = \gamma = 60°\).
