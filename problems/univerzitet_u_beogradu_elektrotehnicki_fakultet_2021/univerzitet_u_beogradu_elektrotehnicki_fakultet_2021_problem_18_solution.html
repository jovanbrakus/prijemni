Plan rešavanja

Pošto je \(Q(x)\) polinom 3. stepena, ostatak \(R(x)\) je polinom stepena najviše 2, tj. \(R(x) = ax^2 + bx + c\). Ključna ideja: na korenima polinoma \(Q(x)\) važi \(P(\text{koren}) = R(\text{koren})\). Faktorizujemo \(Q(x)\), nađemo njegove korene, izračunamo vrednosti \(P(x)\) u tim korenima, i rešimo sistem jednačina za koeficijente \(a\), \(b\), \(c\).

Teorijska podloga — deljenje polinoma i ostatak
  • Deljenje polinoma: Za polinome \(P(x)\) i \(Q(x)\) (\(Q \neq 0\)) postoje jedinstveni polinomi \(D(x)\) (količnik) i \(R(x)\) (ostatak) takvi da \(P(x) = Q(x) \cdot D(x) + R(x)\), gde je \(\deg R < \deg Q\).
  • Teorem o ostatku: Ako je \(\alpha\) koren polinoma \(Q(x)\), tj. \(Q(\alpha) = 0\), tada \(P(\alpha) = R(\alpha)\). Ovo nam omogućava da odredimo \(R(x)\) bez eksplicitnog deljenja.
  • Cikličnost stepena od \(i\): \(i^0 = 1\), \(i^1 = i\), \(i^2 = -1\), \(i^3 = -i\), \(i^4 = 1\), i ciklus se ponavlja. Za \(i^n\) računamo \(n \bmod 4\).
  • Cikličnost stepena od \(-1\): \((-1)^n = 1\) za parno \(n\), \((-1)^n = -1\) za neparno \(n\).
  • Faktorizacija: \(x^3 + x^2 + x + 1 = x^2(x + 1) + (x + 1) = (x + 1)(x^2 + 1)\).

Vizuelni prikaz

Dijagram prikazuje korene polinoma \(Q(x)\) u kompleksnoj ravni, kao i vrednosti \(P(x)\) u svakom korenu. Sva tri korena daju istu vrednost — to je ključ za određivanje ostatka.

Koreni polinoma Q(x) = (x+1)(x²+1) u kompleksnoj ravni — svi daju P(koren) = 2

Rešenje korak po korak

1

Faktorizujemo \(Q(x)\) i nađemo korene

Grupišemo članove polinoma \(Q(x)\):

\[ Q(x) = x^3 + x^2 + x + 1 = x^2(x + 1) + 1 \cdot (x + 1) = (x + 1)(x^2 + 1) \]

Koreni polinoma \(Q(x)\):

\[ x + 1 = 0 \implies x_1 = -1 \] \[ x^2 + 1 = 0 \implies x_2 = i, \quad x_3 = -i \]

Polinom \(Q(x)\) ima tri korena: jedan realan (\(-1\)) i dva kompleksno konjugovana (\(i\) i \(-i\)).

2

Postavimo oblik ostatka

Pošto je \(\deg Q = 3\), ostatak ima oblik:

\[ R(x) = ax^2 + bx + c \]

Važi: \(P(x) = Q(x) \cdot D(x) + R(x)\). Na korenima \(Q(x)\) prvi sabirak nestaje, pa:

\[ P(\alpha) = R(\alpha) \quad \text{za svaki koren } \alpha \text{ od } Q(x) \]
3

Izračunamo \(P(-1)\)

Koristimo pravilo da je \((-1)^n = 1\) za parno \(n\) i \((-1)^n = -1\) za neparno \(n\):

\[ P(-1) = \underbrace{(-1)^{2024}}_{1} + \underbrace{(-1)^{2023}}_{-1} + \underbrace{(-1)^{2022}}_{1} + \underbrace{(-1)^{2021}}_{-1} + \underbrace{(-1)^{20}}_{1} + 1 \]
\[ P(-1) = 1 - 1 + 1 - 1 + 1 + 1 = \boxed{2} \]

Parni eksponenti (2024, 2022, 20) daju +1, neparni (2023, 2021) daju -1. Parovi se poništavaju, ostaje 1 + 1 = 2.

Proveri se: Zašto se poništavaju parovi?
Članovi \((-1)^{2024} + (-1)^{2023} = 1 + (-1) = 0\) i \((-1)^{2022} + (-1)^{2021} = 1 + (-1) = 0\). Ostaju \((-1)^{20} + 1 = 1 + 1 = 2\).
4

Izračunamo \(P(i)\)

Koristimo cikličnost: \(i^n\) zavisi od \(n \bmod 4\). Odredimo ostatke pri deljenju sa 4:

Eksponent mod 4 Vrednost \(i^n\)
2024 0 \(1\)
2023 3 \(-i\)
2022 2 \(-1\)
2021 1 \(i\)
20 0 \(1\)
\[ P(i) = 1 + (-i) + (-1) + i + 1 + 1 \]

Realni deo: \(1 + (-1) + 1 + 1 = 2\). Imaginarni deo: \(-i + i = 0\).

\[ P(i) = \boxed{2} \]
5

Izračunamo \(P(-i)\)

Za \((-i)^n\) koristimo \((-i)^n = (-1)^n \cdot i^n\):

Eksponent \(n\) \((-1)^n\) \(i^n\) \((-i)^n\)
2024 \(1\) \(1\) \(1\)
2023 \(-1\) \(-i\) \(i\)
2022 \(1\) \(-1\) \(-1\)
2021 \(-1\) \(i\) \(-i\)
20 \(1\) \(1\) \(1\)
\[ P(-i) = 1 + i + (-1) + (-i) + 1 + 1 \]

Realni deo: \(1 + (-1) + 1 + 1 = 2\). Imaginarni deo: \(i + (-i) = 0\).

\[ P(-i) = \boxed{2} \]

Zapažamo lep obrazac: \(P(-1) = P(i) = P(-i) = 2\). Svi koreni od \(Q(x)\) daju istu vrednost!

Proveri se: Zašto su vrednosti P(i) i P(-i) konjugovane?
Polinom \(P(x)\) ima realne koeficijente, pa za konjugovane argumente daje konjugovane vrednosti: \(\overline{P(i)} = P(\overline{i}) = P(-i)\). Pošto je \(P(i) = 2\) (realan broj), i \(P(-i) = 2\) — konjugat realnog broja je on sam.
6

Rešimo sistem jednačina za \(R(x) = ax^2 + bx + c\)

Iz uslova \(P(\alpha) = R(\alpha)\) za svaki koren \(\alpha\) dobijamo:

\[ R(-1) = a \cdot 1 - b + c = a - b + c = 2 \quad \ldots (1) \] \[ R(i) = a \cdot (-1) + b \cdot i + c = -a + bi + c = 2 \quad \ldots (2) \] \[ R(-i) = a \cdot (-1) - b \cdot i + c = -a - bi + c = 2 \quad \ldots (3) \]

Iz (2) i (3): Oduzmemo (3) od (2):

\[ (-a + bi + c) - (-a - bi + c) = 0 \implies 2bi = 0 \implies \boxed{b = 0} \]

Sa \(b = 0\): Jednačine (1) i (2) postaju:

\[ a + c = 2 \quad \ldots (1') \] \[ -a + c = 2 \quad \ldots (2') \]

Sabiranjem: \(2c = 4\), dakle \(c = 2\). Iz (1'): \(a = 2 - c = 0\), dakle \(a = 0\).

\[ R(x) = 0 \cdot x^2 + 0 \cdot x + 2 = \boxed{2} \]

Ključno zapažanje

Polinom \(P(x)\) ima istu vrednost u sva tri korena polinoma \(Q(x)\). Pošto je ostatak polinom stepena \(\leq 2\) koji prolazi kroz tri tačke sa istom vrednošću, on mora biti konstanta:

\[ P(-1) = P(i) = P(-i) = 2 \implies R(x) = 2 \]

Tri različite tačke jednoznačno određuju polinom 2. stepena. Ako sve tri daju istu vrednost, jedini takav polinom stepena \(\leq 2\) je konstanta.

Konačan odgovor

\(R(x) = 2\)
(A) \(x^2 + 1\)
(B) \(2\)
(C) \(0\)
(D) \(1\)
(E) \(x^2 - 1\)

Verifikacija: Ostatak \(R(x) = 2\) je konstantan polinom stepena 0, što je manje od \(\deg Q = 3\). Provera na korenima: \(R(-1) = 2 = P(-1)\), \(R(i) = 2 = P(i)\), \(R(-i) = 2 = P(-i)\). Sve tri jednačine su zadovoljene. Tačan odgovor je (B) 2.

Česte greške

  • Greška 1: Pogrešno čitanje eksponenata. U polinomu \(P(x)\) stoji \(x^{20}\), ne \(x^{2020}\). To je namerna zamka — eksponenti nisu u savršenom nizu. Pažljivo prepisati zadatak!
  • Greška 2: Greška u cikličnosti stepena od \(i\). Česta greška je \(i^3 = i\) umesto \(i^3 = -i\). Uvek proveriti: \(i^3 = i^2 \cdot i = (-1) \cdot i = -i\).
  • Greška 3: Pokušaj direktnog deljenja polinoma stepena 2024. To je nepraktično — metod korena je daleko elegantniji i brži. Treba prepoznati da je evaluacija u korenima pravi put.
  • Greška 4: Zaboraviti da \((-i)^n \neq -(i^n)\) generalno. Tačno je \((-i)^n = (-1)^n \cdot i^n\), što zahteva poseban račun za svaki eksponent.

Dodatni izazov

Neka je \(P(x) = x^{2024} + x^{2023} + x^{2022} + x^{2021} + x^{20} + 1\). Odredi ostatak pri deljenju polinoma \(P(x)\) polinomom \(x^4 - 1\).

Nagoveštaj

Polinom \(x^4 - 1\) ima korene \(1, -1, i, -i\). Već znamo \(P(-1) = P(i) = P(-i) = 2\). Ostaje da izračunamo \(P(1) = 1 + 1 + 1 + 1 + 1 + 1 = 6\). Ostatak je polinom stepena \(\leq 3\): \(R(x) = ax^3 + bx^2 + cx + d\). Iz četiri jednačine (Vandermondov sistem) dobijamo: \(R(1) = 6\), \(R(-1) = 2\), \(R(i) = 2\), \(R(-i) = 2\). Rešenje: \(a = 1\), \(b = 0\), \(c = 1\), \(d = 2\), dakle \(R(x) = x^3 + x + 2\).
