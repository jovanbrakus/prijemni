Logaritamska nejednacina sa bazom \(\tfrac{1}{2}\)

Odredjivanje skupa resenja i razlike ekstrema

📝 Tekst zadatka
Razlika najveceg i najmanjeg realnog resenja nejednacine
\[ \log_{1/2}\!\Big(\sqrt{x+1} - x\Big) \leq 2 \]
iznosi:
(A) \(\dfrac{9}{4}\)
(B) nijedan od ponudjenih
(C) \(\dfrac{1}{2}\)
(D) \(2\)
(E) \(1\)
🎯 Plan resavanja

1. Odrediti domen — uslovi za koren i argument logaritma. → 2. Transformisati nejednacinu koristeci svojstvo logaritma sa bazom manjom od 1. → 3. Resiti rezultujucu algebarsku nejednacinu analizom slucajeva. → 4. Preseci sa domenom i izracunati razliku ekstrema.

📚 Podsecanje na teoriju
Logaritamska nejednacina sa bazom \(0 < a < 1\)
\[ \log_a(t) \leq b \;\;\Longleftrightarrow\;\; t \geq a^b \quad (\text{obrcanje znaka jer je } 0 < a < 1) \]

Kada je baza logaritma izmedju 0 i 1, logaritamska funkcija je opadajuca. To znaci da se pri prelazu sa logaritamskog na eksponencijalni oblik obrcemo smer nejednacine.

Uslov za postojanje logaritma
\[ \log_a(t) \text{ postoji samo ako je } t > 0 \]

Argument logaritma mora biti strogo pozitivan. U nasem slucaju: \(\sqrt{x+1} - x > 0\).

Kvadriranje nejednacine

Nejednacinu \(\sqrt{f(x)} \geq g(x)\) kvadriramo samo kada je \(g(x) \geq 0\). Ako je \(g(x) < 0\), nejednacina automatski vazi jer je koren nenegativan.

🎨 Vizuelni prikaz — graficki pregled
🛠 Resenje korak po korak
1 Odredi domen

Da bi logaritam postojao, potrebna su dva uslova:

\[ \text{(i)}\;\; x + 1 \geq 0 \;\Longrightarrow\; x \geq -1 \] \[ \text{(ii)}\;\; \sqrt{x+1} - x > 0 \;\Longrightarrow\; \sqrt{x+1} > x \]

Uslov (ii) analiziramo: za \(x \leq 0\), koren je nenegativan a \(x\) je nepozitivan, pa uslov automatski vazi. Za \(x > 0\), kvadriramo obe strane (obe pozitivne):

\[ x + 1 > x^2 \;\;\Longrightarrow\;\; x^2 - x - 1 < 0 \;\;\Longrightarrow\;\; x < \frac{1+\sqrt{5}}{2} \approx 1{,}618 \]

Dakle, domen je: \(\displaystyle x \in \Big[-1,\;\frac{1+\sqrt{5}}{2}\Big)\).

✓ Kontrolna tacka: Domen je otvoren sa desne strane jer za \(x = \frac{1+\sqrt{5}}{2}\) imamo \(\sqrt{x+1} = x\), pa bi argument logaritma bio 0.
2 Transformisi nejednacinu

Baza logaritma je \(\frac{1}{2}\), sto je manje od 1. Zato je logaritamska funkcija opadajuca i pri prelasku na eksponencijalni oblik obrcemo smer nejednacine:

\[ \log_{1/2}\!\Big(\sqrt{x+1} - x\Big) \leq 2 \;\;\Longleftrightarrow\;\; \sqrt{x+1} - x \geq \left(\frac{1}{2}\right)^{\!2} = \frac{1}{4} \]

Prema tome, nasa nejednacina postaje:

\[ \sqrt{x+1} \geq x + \frac{1}{4} \]
Zasto se obrcemo smer?

Funkcija \(f(t) = \log_{1/2}(t)\) je opadajuca. Kad je \(f(t) \leq 2\), to znaci da je \(t\) dovoljno veliko (a ne malo!) — tacnije \(t \geq (1/2)^2 = 1/4\). Zamislite grafik: vece \(t\) daje manji logaritam.

3 Analiza slucajeva za kvadriranje

Nejednacinu \(\sqrt{x+1} \geq x + \frac{1}{4}\) resavamo u dva slucaja, u zavisnosti od znaka desne strane.

Slucaj A: \(x + \frac{1}{4} < 0\), tj. \(x < -\frac{1}{4}\)

Desna strana je negativna, a \(\sqrt{x+1} \geq 0\), pa nejednacina automatski vazi za sve \(x\) iz domena:

\[ x \in \Big[-1,\; -\tfrac{1}{4}\Big) \quad \checkmark \]

Slucaj B: \(x + \frac{1}{4} \geq 0\), tj. \(x \geq -\frac{1}{4}\)

Obe strane su nenegativne, pa mozemo kvadrirati:

\[ x + 1 \geq \Big(x + \tfrac{1}{4}\Big)^2 = x^2 + \frac{x}{2} + \frac{1}{16} \]

Sredimo:

\[ x + 1 - x^2 - \frac{x}{2} - \frac{1}{16} \geq 0 \] \[ -x^2 + \frac{x}{2} + \frac{15}{16} \geq 0 \] \[ x^2 - \frac{x}{2} - \frac{15}{16} \leq 0 \]
4 Resi kvadratnu nejednacinu

Resavamo jednacinu \(x^2 - \frac{x}{2} - \frac{15}{16} = 0\):

\[ x = \frac{\frac{1}{2} \pm \sqrt{\frac{1}{4} + \frac{15}{4}}}{2} = \frac{\frac{1}{2} \pm \sqrt{\frac{16}{4}}}{2} = \frac{\frac{1}{2} \pm 2}{2} \]
\[ x_1 = \frac{\frac{1}{2} + 2}{2} = \frac{\frac{5}{2}}{2} = \frac{5}{4}, \qquad x_2 = \frac{\frac{1}{2} - 2}{2} = \frac{-\frac{3}{2}}{2} = -\frac{3}{4} \]

Kvadratna funkcija ima pozitivan koeficijent uz \(x^2\), pa je \(\leq 0\) izmedju nula:

\[ -\frac{3}{4} \leq x \leq \frac{5}{4} \]

Presecemo sa uslovom \(x \geq -\frac{1}{4}\) iz Slucaja B:

\[ x \in \Big[-\tfrac{1}{4},\; \tfrac{5}{4}\Big] \]
Proveri sebe: Da li \(x = \frac{5}{4}\) pripada domenu?

Domen zahteva \(x < \frac{1+\sqrt{5}}{2} \approx 1{,}618\). Posto je \(\frac{5}{4} = 1{,}25 < 1{,}618\), jeste u domenu. Takodje, \(\sqrt{\frac{5}{4}+1} - \frac{5}{4} = \sqrt{\frac{9}{4}} - \frac{5}{4} = \frac{3}{2} - \frac{5}{4} = \frac{1}{4} > 0\). ✓

5 Ujedini resenja

Kombinujemo oba slucaja:

\[ \text{Slucaj A:}\;\; x \in \Big[-1,\; -\tfrac{1}{4}\Big) \qquad \cup \qquad \text{Slucaj B:}\;\; x \in \Big[-\tfrac{1}{4},\; \tfrac{5}{4}\Big] \]
\[ \boxed{x \in \Big[-1,\; \frac{5}{4}\Big]} \]
✓ Kontrolna tacka: Intervali se spajaju u jedinstven segment jer se dodiruju u tacki \(x = -\frac{1}{4}\).
6 Izracunaj razliku ekstrema

Najvece resenje je \(x_{\max} = \frac{5}{4}\), a najmanje \(x_{\min} = -1\):

\[ x_{\max} - x_{\min} = \frac{5}{4} - (-1) = \frac{5}{4} + 1 = \frac{5}{4} + \frac{4}{4} = \boxed{\frac{9}{4}} \]
Kljucni uvid

Kod logaritamskih nejednacina sa bazom manjom od 1, kljucno je pravilno obrnuti smer nejednacine pri prelasku na eksponencijalni oblik. Mnogi ucenici pogrese upravo na ovom koraku. Takodje, analiza slucajeva pri kvadriranju je neophodna — ne mozemo kvadrirati nejednacinu ako desna strana moze biti negativna.

Konacni odgovor
\(\dfrac{9}{4}\)
(A) \(\dfrac{9}{4}\)
(B) nijedan od ponudjenih
(C) \(\dfrac{1}{2}\)
(D) \(2\)
(E) \(1\)

Verifikacija: Skup resenja je \([-1,\; \frac{5}{4}]\), razlika ekstrema: \(\frac{5}{4} - (-1) = \frac{9}{4}\) ✔

⚠️ Ceste greske
Pogresan smer nejednacine

Najcesca greska je zaboraviti da baza \(\frac{1}{2} < 1\) obrcemo smer. Ako napisete \(\sqrt{x+1} - x \leq \frac{1}{4}\) umesto \(\geq \frac{1}{4}\), dobicete potpuno pogresan skup resenja.

Kvadriranje bez analize znaka

Ako kvadrirate nejednacinu \(\sqrt{x+1} \geq x + \frac{1}{4}\) bez provere da li je desna strana nenegativna, propustate ceo interval \([-1, -\frac{1}{4})\) gde resenje automatski vazi.

Zaboravljen uslov domena

Argument logaritma mora biti strogo pozitivan: \(\sqrt{x+1} - x > 0\). Za \(x = \frac{5}{4}\) proverimo: \(\sqrt{\frac{9}{4}} - \frac{5}{4} = \frac{3}{2} - \frac{5}{4} = \frac{1}{4} > 0\). Vazi!

🏆 Dodatni izazov

Resiti nejednacinu i odrediti razliku najveceg i najmanjeg resenja:

\[ \log_{1/3}\!\Big(\sqrt{x+4} - x\Big) \leq 1 \]
Nagovestaj

Postupak je analogan: baza \(\frac{1}{3} < 1\) pa obrcemo smer. Dobijamo \(\sqrt{x+4} - x \geq \frac{1}{3}\), sto vodi na \(\sqrt{x+4} \geq x + \frac{1}{3}\). Razdvojite na dva slucaja prema znaku desne strane i resavajte kvadratnu nejednacinu.
