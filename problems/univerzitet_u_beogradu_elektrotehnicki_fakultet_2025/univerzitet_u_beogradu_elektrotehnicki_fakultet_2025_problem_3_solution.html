Logaritamski izrazi

Tekst zadatka

Ako je \(\log_a b = 6\), za \(a,\,b \in (0,1)\cup(1,+\infty)\), onda je

\[\log_b\!\left(a^2\cdot b\right) + \log_{\sqrt{a}}\!\left(\frac{a}{\sqrt[3]{b}}\right)\]

jednako:

(A) \(-\dfrac{2}{3}\)
(B) \(1\)
(C) \(11\)
(D) \(\dfrac{2}{3}\)
(E) \(\dfrac{29}{9}\)

Plan rešavanja

  1. Iz uslova \(\log_a b = 6\) zaključiti da je \(b = a^6\).
  2. Izračunati prvi sabirak \(\log_b(a^2 \cdot b)\) koristeći osobine logaritama.
  3. Izračunati drugi sabirak \(\log_{\sqrt{a}}\!\!\left(\dfrac{a}{\sqrt[3]{b}}\right)\) zamenom \(b = a^6\).
  4. Sabrati oba rezultata i odabrati tačan odgovor.

Osvrt na teoriju

Ključne osobine logaritama

  • Definicija: \(\log_a b = c \;\Longleftrightarrow\; a^c = b\)
  • Logaritam proizvoda: \(\log_a(xy) = \log_a x + \log_a y\)
  • Logaritam količnika: \(\log_a\!\left(\dfrac{x}{y}\right) = \log_a x - \log_a y\)
  • Logaritam stepena: \(\log_a(x^n) = n\cdot\log_a x\)
  • Promena baze: \(\log_a b = \dfrac{\ln b}{\ln a}\), odakle \(\log_b a = \dfrac{1}{\log_a b}\)
  • Promena baze (stepen u osnovi): \(\log_{a^k} x = \dfrac{1}{k}\cdot\log_a x\)

Ove osobine važe za \(a > 0,\; a\neq 1,\; x,y > 0\).

Rešenje korak po korak

1 Izrazimo \(b\) preko \(a\)

Iz uslova \(\log_a b = 6\) direktno sledi definicija logaritma:

\[b = a^6\]

Ovo ćemo koristiti za zamenu svugde gde se pojavljuje \(b\).

2 Računamo prvi sabirak: \(\log_b(a^2 \cdot b)\)

Koristimo osobinu logaritma proizvoda:

\[\log_b(a^2 \cdot b) = \log_b(a^2) + \log_b(b)\]

Drugi član je trivijalan: \(\log_b b = 1\).

Za prvi član koristimo vezu \(\log_b a = \dfrac{1}{\log_a b} = \dfrac{1}{6}\):

\[\log_b(a^2) = 2\cdot\log_b a = 2 \cdot \frac{1}{6} = \frac{1}{3}\]

Dakle:

\[\log_b(a^2 \cdot b) = \frac{1}{3} + 1 = \frac{4}{3}\]
Proveri se: Koliko iznosi \(\log_b(a^2)\) ako je \(\log_a b = 6\)?
\(\log_b(a^2) = 2\cdot\log_b a = 2\cdot\dfrac{1}{6} = \dfrac{1}{3}\). Tačno!
3 Pojednostavljujemo argument drugog logaritma

Zamenjujemo \(b = a^6\) u \(\dfrac{a}{\sqrt[3]{b}}\):

\[\sqrt[3]{b} = \sqrt[3]{a^6} = a^{6/3} = a^2\]

Prema tome:

\[\frac{a}{\sqrt[3]{b}} = \frac{a}{a^2} = a^{1-2} = a^{-1}\]
4 Računamo drugi sabirak: \(\log_{\sqrt{a}}(a^{-1})\)

Osnova je \(\sqrt{a} = a^{1/2}\). Koristimo osobinu promene baze sa stepenom u osnovi:

\[\log_{a^{1/2}}(a^{-1}) = \frac{\log_a(a^{-1})}{\log_a(a^{1/2})} = \frac{-1}{\;\tfrac{1}{2}\;} = -2\]

Alternativno, koristimo formulu \(\log_{a^k} x = \dfrac{1}{k}\cdot\log_a x\):

\[\log_{a^{1/2}}(a^{-1}) = \frac{1}{\;\tfrac{1}{2}\;}\cdot\log_a(a^{-1}) = 2\cdot(-1) = -2\]
Proveri se: Koliko iznosi \(\log_{\sqrt{a}}(a^{-1})\)?
\(\log_{\sqrt{a}}(a^{-1}) = -2\). Ključna ideja: ako je osnova \(a^{1/2}\), onda eksponent delimo sa \(\tfrac{1}{2}\), pa \(-1 \div \tfrac{1}{2} = -2\).
5 Sabiramo rezultate

Konačno:

\[\log_b(a^2\cdot b) + \log_{\sqrt{a}}\!\left(\frac{a}{\sqrt[3]{b}}\right) = \frac{4}{3} + (-2) = \frac{4}{3} - \frac{6}{3} = -\frac{2}{3}\]

Ključni uvid

Ceo zadatak se svodi na jednu zamenu: \(b = a^6\). Kada se to uradi, svi logaritmi postaju logaritmi stepena broja \(a\) sa različitim osnovama koje su takođe stepeni broja \(a\). Tada se koristi opšta formula:

\[\log_{a^k}(a^m) = \frac{m}{k}\]

Ova formula direktno daje: \(\log_b(a^2) = \dfrac{2}{6} = \dfrac{1}{3}\) i \(\log_{\sqrt{a}}(a^{-1}) = \dfrac{-1}{1/2} = -2\).

Konačan odgovor

\(\displaystyle -\frac{2}{3}\)

Tačan odgovor je (A).

Česte greške

Zaboraviti da je \(\log_b a = \frac{1}{\log_a b}\), pa pogrešno napisati \(\log_b(a^2) = 2\cdot 6 = 12\) umesto \(2\cdot\frac{1}{6} = \frac{1}{3}\).
Pogrešno izračunati \(\sqrt[3]{a^6}\) — rezultat je \(a^2\), ne \(a^3\) niti \(a^{1/2}\).
Prevideti negativan predznak: \(a/a^2 = a^{-1}\), ne \(a^{1}\).
Pomešati formulu za logaritam sa stepenom u osnovi: \(\log_{a^k}(x) = \frac{1}{k}\log_a x\), ne \(k\cdot\log_a x\).

Dodatni izazov

Ako je \(\log_a b = p\), izrazi u opštem obliku:

\[\log_b(a^2 \cdot b) + \log_{\sqrt{a}}\!\left(\frac{a}{\sqrt[3]{b}}\right)\]

Pokušaj sam, pa proveri odgovor ispod.

Zamenom \(b = a^p\):

\[\log_b(a^2\cdot b) = \frac{2}{p} + 1 = \frac{p+2}{p}\] \[\frac{a}{\sqrt[3]{b}} = a^{1-p/3} \implies \log_{\sqrt{a}}(a^{1-p/3}) = 2\!\left(1 - \frac{p}{3}\right) = 2 - \frac{2p}{3}\] \[\text{Ukupno} = \frac{p+2}{p} + 2 - \frac{2p}{3} = \frac{p+2}{p} + \frac{6-2p}{3}\]

Za \(p = 6\): \(\dfrac{8}{6} + \dfrac{6-12}{3} = \dfrac{4}{3} - 2 = -\dfrac{2}{3}\). Potvrđeno!
