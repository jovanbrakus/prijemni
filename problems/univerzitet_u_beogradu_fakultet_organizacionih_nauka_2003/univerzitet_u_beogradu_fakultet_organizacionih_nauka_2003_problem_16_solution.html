Logaritamska nejednačina sa složenim logaritmima

Logaritmi — nejednačine sa kompozicijom logaritamskih funkcija

Zadatak

Skup svih rešenja nejednačine \(\log_2(\log_2(x + 3)) < 0\) je:

(A)
\((-3,\;-1)\)
(B)
\((-3,\;+\infty)\)
(C)
\((-\infty,\;-1)\)
(D)
\((-2,\;-1)\)
(E)
\((-1,\;+\infty)\)

Plan rešavanja

Rešićemo nejednačinu „iznutra ka spolja" — prvo ćemo odrediti uslove definisanosti (domen) složene logaritamske funkcije, a zatim primeniti svojstvo da je \(\log_2(y) < 0\) ekvivalentno sa \(0 < y < 1\), kako bismo sveli problem na jednostavnu nejednačinu.

Osvežimo teoriju — Svojstva logaritamske funkcije

Definisanost logaritma: Funkcija \(\log_a(x)\) je definisana samo za \(x > 0\) i \(a > 0,\; a \neq 1\).

Znak logaritma: Za osnovu \(a > 1\):

\[ \log_a(x) < 0 \iff 0 < x < 1 \] \[ \log_a(x) = 0 \iff x = 1 \] \[ \log_a(x) > 0 \iff x > 1 \]

Monotonija: Za \(a > 1\), funkcija \(\log_a(x)\) je strogo rastuća. To znači:

\[ \log_a(x) < b \iff x < a^b \quad \text{(za } a > 1\text{)} \]

Vizuelni prikaz — Grafik i oblast rešenja

Grafik funkcije \(\log_2(\log_2(x + 3))\) — oblast ispod \(x\)-ose je rešenje nejednačine.

Rešenje korak po korak

1

Odredimo domen — unutrašnji logaritam

Da bi \(\log_2(x + 3)\) bio definisan, argument mora biti pozitivan:

\[ x + 3 > 0 \implies x > -3 \]
2

Odredimo domen — spoljašnji logaritam

Da bi \(\log_2(\log_2(x + 3))\) bio definisan, argument spoljašnjeg logaritma mora biti pozitivan:

\[ \log_2(x + 3) > 0 \]

Pošto je osnova \(2 > 1\), logaritam je pozitivan kada je argument veći od 1:

\[ x + 3 > 1 \implies x > -2 \]

Dakle, domen cele funkcije je \(x > -2\), tj. interval \((-2, +\infty)\).

Proveri sebe: Zašto je uslov \(\log_2(x+3) > 0\) a ne \(\log_2(x+3) \geq 0\)?

Zato što je argument logaritma \(\log_2(\cdot)\) mora biti strogo pozitivan (veći od nule). Vrednost \(\log_2(x+3) = 0\) nije dozvoljena jer bi tada imali \(\log_2(0)\), što nije definisano.

3

Rešimo nejednačinu

Sada rešavamo:

\[ \log_2(\log_2(x + 3)) < 0 \]

Primenimo svojstvo: za osnovu \(2 > 1\), važi \(\log_2(y) < 0 \iff 0 < y < 1\).

Označimo \(y = \log_2(x + 3)\). Tada:

\[ \log_2(y) < 0 \iff 0 < y < 1 \]
4

Vratimo se na promenljivu \(x\)

Zamenom \(y = \log_2(x + 3)\) dobijamo dvostruku nejednačinu:

\[ 0 < \log_2(x + 3) < 1 \]

Levi deo:

\[ \log_2(x + 3) > 0 \implies x + 3 > 2^0 = 1 \implies x > -2 \]

Desni deo:

\[ \log_2(x + 3) < 1 \implies x + 3 < 2^1 = 2 \implies x < -1 \]
Proveri sebe: Koliko je \(\log_2(x+3)\) kada je \(x = -1{,}5\)?

\(x + 3 = 1{,}5\), pa \(\log_2(1{,}5) \approx 0{,}585\). Ova vrednost je između 0 i 1, što znači da \(x = -1{,}5\) pripada skupu rešenja.

5

Zapišimo presek uslova

Kombinujemo oba uslova:

\[ x > -2 \quad \text{i} \quad x < -1 \] \[ \implies x \in (-2,\;-1) \]
Proveri sebe: Da li granične tačke pripadaju rešenju?

Za \(x = -2\): \(x + 3 = 1\), pa \(\log_2(1) = 0\). Tada \(\log_2(0)\) nije definisan — ne pripada.
Za \(x = -1\): \(x + 3 = 2\), pa \(\log_2(2) = 1\). Tada \(\log_2(1) = 0\), što nije strogo manje od 0 — ne pripada.
Dakle, obe granične tačke su isključene, interval je otvoren: \((-2,\;-1)\).

Ključni uvid

Složene logaritamske nejednačine rešavamo „ljuštenjem slojeva" — od spoljašnjeg logaritma ka unutrašnjem. Ključno je prepoznati da \(\log_a(y) < 0\) za \(a > 1\) znači da je \(y\) između 0 i 1, čime problem svodimo na jednostavnu dvostruku nejednačinu.

Konačan odgovor

\[ x \in (-2,\;-1) \]
(A) \((-3,\;-1)\) (B) \((-3,\;+\infty)\) (C) \((-\infty,\;-1)\) (D) \((-2,\;-1)\) (E) \((-1,\;+\infty)\)
Verifikacija: Za \(x = -1{,}5\) (sredina intervala): \(x + 3 = 1{,}5\), \(\log_2(1{,}5) \approx 0{,}585\), \(\log_2(0{,}585) \approx -0{,}773 < 0\). Rezultat je negativan — tačno! Za graničnu tačku \(x = -2\): izraz nije definisan. Za \(x = -1\): \(\log_2(\log_2(2)) = \log_2(1) = 0\), nije strogo manje od 0.

Česte greške

  • Zaboravljanje da argument spoljašnjeg logaritma mora biti strogo pozitivan, što daje stroži uslov od samog \(x + 3 > 0\). Mnogi uzmu \(x > -3\) umesto \(x > -2\) kao donju granicu.
  • Pogrešno tumačenje \(\log_2(y) < 0\) kao \(y < 0\) umesto \(0 < y < 1\). Negativan logaritam ne znači negativan argument — argument logaritma je uvek pozitivan!

Dodatni izazov

Rešiti nejednačinu \(\log_3(\log_3(x - 1)) \geq 1\).

Nagoveštaj

Primenite isti pristup: \(\log_3(y) \geq 1\) znači \(y \geq 3\). Dakle, treba rešiti \(\log_3(x - 1) \geq 3\), što daje \(x - 1 \geq 27\), tj. \(x \geq 28\).
