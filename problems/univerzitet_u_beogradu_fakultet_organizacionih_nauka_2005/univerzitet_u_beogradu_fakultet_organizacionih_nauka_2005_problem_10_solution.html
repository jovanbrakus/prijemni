Binomni koeficijenti u razvoju \((1 + x)^n\)

Kombinatorika i algebra — određivanje prirodnog broja \(n\) iz uslova za binomne koeficijente

Tekst zadatka

Neki članovi u razvoju \((1 + x)^n\), gde je \(n \in \mathbb{N}\), su oblika \(ax\), \(bx^2\) i \(cx^8\) (\(a, b, c \in \mathbb{R}\)).

Plan rešavanja

Izrazićemo koeficijente \(a\), \(b\) i \(c\) pomoću binomnih koeficijenata \(\binom{n}{k}\), zatim iz uslova \(a + b = 55\) formirati kvadratnu jednačinu po \(n\) i naći vrednost \(n\). Na kraju izračunamo \(c = \binom{n}{8}\).

📐 Ključne formule i teoreme
  • Binomna teorema: \[\displaystyle (1 + x)^n = \sum_{k=0}^{n} \binom{n}{k} x^k\]
  • Binomni koeficijent: \[\binom{n}{k} = \frac{n!}{k!(n-k)!}\]
  • Simetrija binomnih koeficijenata: \[\binom{n}{k} = \binom{n}{n-k}\]
  • Posebni slučajevi: \(\binom{n}{0} = 1\), \(\binom{n}{1} = n\), \(\binom{n}{2} = \frac{n(n-1)}{2}\)

Vizuelni prikaz — Paskalov trougao

Paskalov trougao prikazuje binomne koeficijente. Red \(n = 10\) sadrži koeficijente iz našeg zadatka. Pomeri klizač da istražiš različite redove.

Rešenje korak po korak

1

Opšti član binomnog razvoja

Prema binomnoj teoremi, razvoj \((1 + x)^n\) glasi:

\[(1 + x)^n = \sum_{k=0}^{n} \binom{n}{k} x^k = \binom{n}{0} + \binom{n}{1}x + \binom{n}{2}x^2 + \cdots + \binom{n}{n}x^n\]

Koeficijent uz \(x^k\) u ovom razvoju je \(\binom{n}{k}\).

2

Identifikacija koeficijenata \(a\), \(b\) i \(c\)

Iz razvoja čitamo koeficijente uz odgovarajuće stepene:

\[a = \binom{n}{1} = n\]
\[b = \binom{n}{2} = \frac{n(n-1)}{2}\]
\[c = \binom{n}{8} = \frac{n!}{8!(n-8)!}\]

Napomena: da bi član \(cx^8\) postojao, mora biti \(n \geq 8\).

🔍 Proveri sebe: Koliko iznosi \(\binom{n}{1}\)?

\(\binom{n}{1} = \frac{n!}{1!(n-1)!} = n\). Dakle, koeficijent uz \(x\) je upravo \(n\).

3

Postavljanje jednačine iz uslova \(a + b = 55\)

Zamenjujemo izraze za \(a\) i \(b\):

\[n + \frac{n(n-1)}{2} = 55\]

Množimo celu jednačinu sa 2:

\[2n + n(n-1) = 110\]

Razvijamo zagradu:

\[2n + n^2 - n = 110\]

Sređujemo:

\[n^2 + n - 110 = 0\]
4

Rešavanje kvadratne jednačine

Primenjujemo formulu za rešenja kvadratne jednačine \(n^2 + n - 110 = 0\):

\[D = 1^2 + 4 \cdot 110 = 1 + 440 = 441 = 21^2\]
\[n_{1,2} = \frac{-1 \pm 21}{2}\]

Dva rešenja:

\[n_1 = \frac{-1 + 21}{2} = \frac{20}{2} = 10\]
\[n_2 = \frac{-1 - 21}{2} = \frac{-22}{2} = -11\]

Pošto \(n \in \mathbb{N}\), odbacujemo negativno rešenje. Dakle, \(n = 10\).

🔍 Proveri sebe: Da li se slaže \(a + b = 55\) za \(n = 10\)?

\(a = \binom{10}{1} = 10\), \(b = \binom{10}{2} = \frac{10 \cdot 9}{2} = 45\). Dakle \(a + b = 10 + 45 = 55\). Tačno! ✓

5

Izračunavanje koeficijenta \(c\)

Sada računamo \(c = \binom{10}{8}\). Koristimo simetriju binomnih koeficijenata:

\[\binom{10}{8} = \binom{10}{10-8} = \binom{10}{2}\]

Ovo smo već izračunali:

\[c = \binom{10}{2} = \frac{10!}{2! \cdot 8!} = \frac{10 \cdot 9}{2 \cdot 1} = \frac{90}{2} = 45\]
🔍 Proveri sebe: Zašto je \(\binom{10}{8} = \binom{10}{2}\)?

Simetrija binomnih koeficijenata kaže da je \(\binom{n}{k} = \binom{n}{n-k}\). Za \(n = 10\) i \(k = 8\): \(\binom{10}{8} = \binom{10}{10-8} = \binom{10}{2}\). Intuitivno, biranje 8 elemenata od 10 je isto kao biranje 2 elementa koje nećemo uzeti.

Ključni uvid

Zahvaljujući simetriji \(\binom{n}{k} = \binom{n}{n-k}\), koeficijent \(c\) uz \(x^8\) je jednak koeficijentu \(b\) uz \(x^2\), jer je \(n - 8 = 10 - 8 = 2\).

\[c = \binom{10}{8} = \binom{10}{2} = b = 45\]

Konačan odgovor

\(c = 45\)

Verifikacija: Za \(n = 10\): \(a = \binom{10}{1} = 10\), \(b = \binom{10}{2} = 45\), pa je \(a + b = 55\) ✓. Koeficijent \(c = \binom{10}{8} = 45\), što se slaže sa odgovorom (B). Zanimljivo je da je \(b = c = 45\) — posledica simetrije Paskalovog trougla.

Česte greške

  • Zaboraviti simetriju: Mnogi pokušavaju da direktno računaju \(\binom{10}{8} = \frac{10!}{8! \cdot 2!}\) razlaganjem celog faktorijela, što je komplikovano i podložno greškama. Uvek koristite \(\binom{n}{k} = \binom{n}{n-k}\) da svedete na manji \(k\).
  • Pogrešno postaviti jednačinu: Neki učenici pišu \(a \cdot b = 55\) umesto \(a + b = 55\), ili zamene koeficijente (npr. \(a = \binom{n}{2}\)). Pažljivo čitajte: \(a\) je koeficijent uz \(x\), a \(b\) uz \(x^2\).

Dodatni izazov

Ako u razvoju \((1 + x)^n\) koeficijenti uz \(x^3\) i \(x^{n-3}\) imaju zbir 240, odrediti \(n\).

💡 Nagovještaj

Koristite istu simetriju: \(\binom{n}{3} = \binom{n}{n-3}\), pa je zbir \(2 \cdot \binom{n}{3} = 240\), tj. \(\binom{n}{3} = 120\). Rešite \(\frac{n(n-1)(n-2)}{6} = 120\).
