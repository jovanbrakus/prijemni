Izražavanje kosinusa preko parametra a

Trigonometrijski identiteti i algebarska manipulacija u drugom kvadrantu

Zadatak

Ako je

\[\frac{1 - \sin x}{1 + \sin x} = a \qquad \left(\frac{\pi}{2} < x < \pi\right)\]

tada je \(\cos x\) jednak:

(A)
\(a - 1\)
(B)
\(-\sqrt{1 - a^2}\)
(C)
\(-\dfrac{2\sqrt{a}}{1 + a}\)
(D)
\(-\dfrac{2a}{1 + a^2}\)
(E)
\(-\sqrt{a}\)
Plan rešavanja

Iz date jednačine ćemo algebarski izraziti \(\sin x\) preko \(a\). Zatim ćemo pomoću Pitagorinog identiteta \(\cos^2 x = 1 - \sin^2 x\) naći \(\cos^2 x\), pa odrediti znak kosinusa na osnovu kvadranta u kome se ugao nalazi.

Podsetnik iz teorije
Trigonometrijski identiteti i svojstva kvadranata -- klikni da pogledaš

Ključne formule i činjenice za ovaj zadatak:

Pitagorin identitet: \(\;\sin^2 x + \cos^2 x = 1\)

Drugi kvadrant \(\left(\frac{\pi}{2} < x < \pi\right)\): \(\sin x > 0\), \(\cos x < 0\)

Razlika kvadrata: \(\;a^2 - b^2 = (a-b)(a+b)\)

Binomna formula: \(\;(a \pm b)^2 = a^2 \pm 2ab + b^2\)

Vizuelni prikaz -- Jedinični krug
Rešenje korak po korak
1
Analiza uslova zadatka

Ugao \(x\) pripada drugom kvadrantu: \(\frac{\pi}{2} < x < \pi\). To znači:

\(\sin x > 0 \quad\) i \(\quad \cos x < 0\)

Pošto je \(0 < \sin x < 1\), oba člana \(1 - \sin x\) i \(1 + \sin x\) su pozitivna, pa važi \(a > 0\).

2
Izrazimo \(\sin x\) preko \(a\)

Polazimo od date jednačine i rešavamo je po \(\sin x\):

\[\frac{1 - \sin x}{1 + \sin x} = a\]

Pomnožimo obe strane sa \((1 + \sin x)\):

\[1 - \sin x = a(1 + \sin x)\]

Raspodelimo desnu stranu:

\[1 - \sin x = a + a\sin x\]

Grupišemo članove sa \(\sin x\) na jednu stranu:

\[1 - a = \sin x + a\sin x = \sin x\,(1 + a)\]

Odakle dobijamo:

\[\sin x = \frac{1 - a}{1 + a}\]
Proveri se: da li ovo zadovoljava polaznu jednačinu?

Zamena: \(\frac{1 - \frac{1-a}{1+a}}{1 + \frac{1-a}{1+a}} = \frac{\frac{(1+a)-(1-a)}{1+a}}{\frac{(1+a)+(1-a)}{1+a}} = \frac{\frac{2a}{1+a}}{\frac{2}{1+a}} = \frac{2a}{2} = a\). Tačno!

3
Nađimo \(\cos^2 x\) pomoću Pitagorinog identiteta

Koristimo identitet \(\cos^2 x = 1 - \sin^2 x\):

\[\cos^2 x = 1 - \left(\frac{1-a}{1+a}\right)^2 = \frac{(1+a)^2 - (1-a)^2}{(1+a)^2}\]

Razvijamo kvadrate u brojiocu:

\[(1+a)^2 = 1 + 2a + a^2\] \[(1-a)^2 = 1 - 2a + a^2\]

Njihova razlika:

\[(1+a)^2 - (1-a)^2 = (1 + 2a + a^2) - (1 - 2a + a^2) = 4a\]

Dakle:

\[\cos^2 x = \frac{4a}{(1+a)^2}\]
Proveri se: alternativni račun pomoću razlike kvadrata

Možemo i direktno primeniti formulu \(A^2 - B^2 = (A-B)(A+B)\):

\((1+a)^2 - (1-a)^2 = [(1+a)-(1-a)][(1+a)+(1-a)] = 2a \cdot 2 = 4a\). Isti rezultat!

4
Odredimo \(\cos x\) sa pravim znakom

Korenujemo obe strane:

\[\cos x = \pm\sqrt{\frac{4a}{(1+a)^2}} = \pm\frac{2\sqrt{a}}{1+a}\]

Pošto je \(a > 0\), važi \(1 + a > 0\), pa apsolutna vrednost nije potrebna u imeniocu.

Iz analize u koraku 1 znamo da je \(\cos x < 0\), pa biramo negativan znak:

\[\cos x = -\frac{2\sqrt{a}}{1 + a}\]
5
Verifikacija na konkretnom primeru

Proverimo za \(x = \frac{2\pi}{3}\) (tj. \(120°\)), gde je \(\sin x = \frac{\sqrt{3}}{2}\) i \(\cos x = -\frac{1}{2}\).

Računamo \(a\):

\[a = \frac{1 - \frac{\sqrt{3}}{2}}{1 + \frac{\sqrt{3}}{2}} = \frac{\frac{2-\sqrt{3}}{2}}{\frac{2+\sqrt{3}}{2}} = \frac{2 - \sqrt{3}}{2 + \sqrt{3}} = \frac{(2-\sqrt{3})^2}{(2+\sqrt{3})(2-\sqrt{3})} = \frac{4 - 4\sqrt{3} + 3}{4 - 3} = 7 - 4\sqrt{3}\]

Sada proverimo formulu \(-\frac{2\sqrt{a}}{1+a}\):

\[\sqrt{a} = \sqrt{7 - 4\sqrt{3}} = 2 - \sqrt{3}\] \[-\frac{2(2-\sqrt{3})}{1 + 7 - 4\sqrt{3}} = -\frac{4 - 2\sqrt{3}}{8 - 4\sqrt{3}} = -\frac{2(2-\sqrt{3})}{4(2-\sqrt{3})} = -\frac{2}{4} = -\frac{1}{2} = \cos\frac{2\pi}{3}\;\checkmark\]
Proveri se: zašto je \(\sqrt{7 - 4\sqrt{3}} = 2 - \sqrt{3}\)?

Kvadrirajmo: \((2 - \sqrt{3})^2 = 4 - 4\sqrt{3} + 3 = 7 - 4\sqrt{3}\). Pošto je \(2 > \sqrt{3} \approx 1{,}73\), koren je pozitivan. Tačno!

Ključni uvid

Ključ zadatka je u tome da se iz jedne trigonometrijske jednačine prvo algebarski izrazi \(\sin x\), a zatim koristi Pitagorin identitet za prelaz na \(\cos x\). Podatak o kvadrantu nije samo ukras -- on određuje znak kosinusa i time selektuje tačan odgovor među ponuđenima.

Konačan odgovor
\[\cos x = \boxed{-\frac{2\sqrt{a}}{1 + a}}\]
(A) \(a - 1\) (B) \(-\sqrt{1 - a^2}\) (C) \(-\dfrac{2\sqrt{a}}{1 + a}\) (D) \(-\dfrac{2a}{1 + a^2}\) (E) \(-\sqrt{a}\)
Česte greške

⚠ Zaboraviti da je \(\cos x < 0\) u drugom kvadrantu i uzeti pozitivan koren. To bi dalo \(+\frac{2\sqrt{a}}{1+a}\), što nije među ponuđenim odgovorima.

⚠ Pogrešno razvijanje razlike kvadrata u brojiocu: \((1+a)^2 - (1-a)^2 = 4a\), a ne \(4a^2\). Greška u znaku ili koeficijentu menja ceo rezultat.

⚠ Brkanje \(\sqrt{a}\) i \(a\) u konačnom izrazu. Odgovor D) ima \(a\) umesto \(\sqrt{a}\) u brojiocu i \(a^2\) umesto \(a\) u imeniocu -- to je distraktora za nepažljive.

Dodatni izazov

Ako je \(\dfrac{1 - \cos x}{1 + \cos x} = b\), gde je \(0 < x < \pi\), izrazi \(\sin x\) preko \(b\).

Prikaži nagoveštaj

Postupak je analogan. Iz jednačine: \(\cos x = \frac{1-b}{1+b}\). Zatim: \(\sin^2 x = 1 - \cos^2 x = \frac{4b}{(1+b)^2}\). Pošto je \(\sin x > 0\) za \(0 < x < \pi\), dobijamo \(\sin x = \frac{2\sqrt{b}}{1+b}\).
