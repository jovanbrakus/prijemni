Trigonometrijska jednačina sa zbirom kosinusa

📋 Tekst zadatka

Zbir kvadrata najmanjeg pozitivnog i najvećeg negativnog rešenja jednačine:

\[\cos\!\left(2x - \frac{\pi}{4}\right) + \sin\!\left(2x + \frac{3\pi}{4}\right) = \sqrt{2}\]

je:

(A)
\(\dfrac{2\pi^2}{9}\)
(B)
\(\dfrac{5\pi^2}{36}\)
(C)
\(\dfrac{\pi^2}{2}\)
(D)
\(2\pi^2\)
(E)
\(\dfrac{\pi^2}{8}\)

🗺️ Plan rešavanja

  1. Transformisati \(\sin\!\left(2x + \frac{3\pi}{4}\right)\) koristeći vezu sinusa i kosinusa
  2. Primeniti formulu za zbir dva kosinusa (sum-to-product)
  3. Rešiti dobijenu elementarnu trigonometrijsku jednačinu
  4. Identifikovati najmanje pozitivno i najveće negativno rešenje
  5. Izračunati zbir njihovih kvadrata

📖 Teorijsko podsetnik

Veze između sinusa i kosinusa — pomak za π/2

Osnovni identitet:

\[\sin\!\left(\alpha + \frac{\pi}{2}\right) = \cos \alpha\]

Ovo sledi iz adicione formule: \(\sin(\alpha + \frac{\pi}{2}) = \sin\alpha\cos\frac{\pi}{2} + \cos\alpha\sin\frac{\pi}{2} = \cos\alpha\).

Formula za zbir kosinusa:

\[\cos A + \cos B = 2\cos\!\frac{A+B}{2}\cos\!\frac{A-B}{2}\]

Rešenja jednačine \(\cos\theta = 1\):

\[\theta = 2k\pi, \quad k \in \mathbb{Z}\]

📊 Vizuelni prikaz

Grafik funkcije \(\cos(2x)\) sa označenim rešenjima \(x = k\pi\)

📝 Rešenje korak po korak

1 Transformacija sinusnog člana

Posmatramo drugi sabirak:

\[\sin\!\left(2x + \frac{3\pi}{4}\right)\]

Zapažamo da je \(\frac{3\pi}{4} = \frac{\pi}{4} + \frac{\pi}{2}\), pa možemo pisati:

\[\sin\!\left(2x + \frac{3\pi}{4}\right) = \sin\!\left(\left(2x + \frac{\pi}{4}\right) + \frac{\pi}{2}\right)\]

Primenom identiteta \(\sin(\alpha + \frac{\pi}{2}) = \cos\alpha\), za \(\alpha = 2x + \frac{\pi}{4}\):

\[\sin\!\left(2x + \frac{3\pi}{4}\right) = \cos\!\left(2x + \frac{\pi}{4}\right)\]
Kontrolna tačka

Proverimo za \(x = 0\): \(\sin\frac{3\pi}{4} = \sin\frac{\pi}{4} = \frac{\sqrt{2}}{2}\) i \(\cos\frac{\pi}{4} = \frac{\sqrt{2}}{2}\). Jednako. ✓

2 Zamena u jednačinu

Jednačina sada postaje:

\[\cos\!\left(2x - \frac{\pi}{4}\right) + \cos\!\left(2x + \frac{\pi}{4}\right) = \sqrt{2}\]

Dobili smo zbir dva kosinusa sa istim argumentom \(2x\), ali sa pomeranjem \(\pm\frac{\pi}{4}\).

3 Primena formule za zbir kosinusa

Koristimo formulu:

\[\cos A + \cos B = 2\cos\!\frac{A+B}{2}\cos\!\frac{A-B}{2}\]

gde je \(A = 2x - \frac{\pi}{4}\) i \(B = 2x + \frac{\pi}{4}\).

Izračunavamo:

\[\frac{A+B}{2} = \frac{(2x - \frac{\pi}{4}) + (2x + \frac{\pi}{4})}{2} = \frac{4x}{2} = 2x\] \[\frac{A-B}{2} = \frac{(2x - \frac{\pi}{4}) - (2x + \frac{\pi}{4})}{2} = \frac{-\frac{\pi}{2}}{2} = -\frac{\pi}{4}\]

Zamenom dobijamo:

\[2\cos(2x)\cos\!\left(-\frac{\pi}{4}\right) = \sqrt{2}\]

Pošto je \(\cos(-\frac{\pi}{4}) = \cos\frac{\pi}{4} = \frac{\sqrt{2}}{2}\):

\[2 \cdot \cos(2x) \cdot \frac{\sqrt{2}}{2} = \sqrt{2}\] \[\sqrt{2}\cdot\cos(2x) = \sqrt{2}\]
Kontrolna tačka — detalji računanja

Proveravamo: \(2 \cdot \frac{\sqrt{2}}{2} = \sqrt{2}\). Dakle, leva strana je \(\sqrt{2}\cos(2x)\), a desna je \(\sqrt{2}\). Deljenjem sa \(\sqrt{2} \neq 0\) dobijamo \(\cos(2x) = 1\). ✓

4 Rešavanje jednačine cos(2x) = 1

Deljenjem obe strane sa \(\sqrt{2}\) dobijamo:

\[\cos(2x) = 1\]

Jednačina \(\cos\theta = 1\) ima rešenja \(\theta = 2k\pi\), \(k \in \mathbb{Z}\).

Dakle:

\[2x = 2k\pi \implies x = k\pi, \quad k \in \mathbb{Z}\]

Skup rešenja: \(\{\ldots, -2\pi, -\pi, 0, \pi, 2\pi, \ldots\}\)

5 Identifikacija traženih rešenja

Iz skupa rešenja \(x = k\pi\), \(k \in \mathbb{Z}\):

  • Najmanje pozitivno rešenje: za \(k = 1\) dobijamo \(x_1 = \pi\)
  • Najveće negativno rešenje: za \(k = -1\) dobijamo \(x_2 = -\pi\)

Napomena: za \(k = 0\), \(x = 0\) nije ni pozitivno ni negativno, pa se ne uzima u obzir.

Kontrolna tačka — verifikacija rešenja

Za \(x = \pi\): \(\cos(2\pi - \frac{\pi}{4}) + \sin(2\pi + \frac{3\pi}{4}) = \cos(-\frac{\pi}{4}) + \sin(\frac{3\pi}{4}) = \frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2} = \sqrt{2}\). ✓

Za \(x = -\pi\): \(\cos(-2\pi - \frac{\pi}{4}) + \sin(-2\pi + \frac{3\pi}{4}) = \cos(\frac{\pi}{4}) + \sin(\frac{3\pi}{4}) = \frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2} = \sqrt{2}\). ✓

6 Izračunavanje zbira kvadrata

Zbir kvadrata najmanjeg pozitivnog i najvećeg negativnog rešenja:

\[x_1^2 + x_2^2 = \pi^2 + (-\pi)^2 = \pi^2 + \pi^2 = 2\pi^2\]

💡 Ključni uvid

Ključ ovog zadatka je prepoznavanje da se \(\sin(2x + \frac{3\pi}{4})\) može zapisati kao \(\cos(2x + \frac{\pi}{4})\) pomoću identiteta \(\sin(\alpha + \frac{\pi}{2}) = \cos\alpha\). Time se jednačina svodi na zbir dva kosinusa sa simetričnim pomerajima \(\pm\frac{\pi}{4}\), koji se formulom zbir-u-proizvod elegantno transformiše u \(\sqrt{2}\cos(2x) = \sqrt{2}\), odnosno \(\cos(2x) = 1\).

✅ Konačan odgovor

\(2\pi^2\)

Najmanje pozitivno rešenje je \(x_1 = \pi\), a najveće negativno \(x_2 = -\pi\).

Zbir kvadrata: \(\pi^2 + \pi^2 = 2\pi^2\).

A) \(\dfrac{2\pi^2}{9}\)
B) \(\dfrac{5\pi^2}{36}\)
C) \(\dfrac{\pi^2}{2}\)
D) \(2\pi^2\)
E) \(\dfrac{\pi^2}{8}\)

⚠️ Česte greške

Greška 1: Pogrešna transformacija sinusa u kosinus. Česta zabuna je pisanje \(\sin(\alpha + \frac{\pi}{2}) = -\cos\alpha\) ili \(\sin(\alpha + \frac{\pi}{2}) = \sin\alpha\). Ispravno je \(\sin(\alpha + \frac{\pi}{2}) = \cos\alpha\).
Greška 2: Uzimanje \(x = 0\) kao najmanjeg pozitivnog rešenja. Nula nije ni pozitivna ni negativna, pa se \(x = 0\) ne uzima u obzir. Najmanje pozitivno rešenje je \(x = \pi\).
Greška 3: Pogrešna primena formule za zbir kosinusa — zamena mesta zbira i razlike argumenata, što dovodi do pogrešnog rezultata.
Greška 4: Zaboravljanje da je \(\cos(-\alpha) = \cos\alpha\), pa se uvodi nepotreban minus koji kvari račun.

🎯 Dodatni izazov

Koliki je zbir svih rešenja jednačine \(\cos(2x - \frac{\pi}{4}) + \sin(2x + \frac{3\pi}{4}) = \sqrt{2}\) na intervalu \([-3\pi, 3\pi]\)?

Savet: Rešenja su \(x = k\pi\), \(k \in \mathbb{Z}\). Na intervalu \([-3\pi, 3\pi]\) to su \(-3\pi, -2\pi, -\pi, 0, \pi, 2\pi, 3\pi\). Zbir je \((-3\pi) + (-2\pi) + (-\pi) + 0 + \pi + 2\pi + 3\pi = 0\).
