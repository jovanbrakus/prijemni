Suma stepena imaginarne jedinice

Algebra — kompleksni brojevi, ciklična svojstva imaginarne jedinice

Postavka zadatka

Vrednost izraza \(1 + i + i^2 + \cdots + i^{2011}\), gde je \(i\) imaginarna jedinica, je:

Izraz
\(\displaystyle\sum_{k=0}^{2011} i^k\)
Imaginarna jedinica
\(i^2 = -1\)
(A)
\(1\)
(B)
\(1 + i\)
(C)
\(i\)
(D)
\(0\)
(E)
\(1 - i\)

Plan rešavanja

Iskoristićemo ciklično ponašanje stepena imaginarne jedinice \(i\), čiji se stepeni ponavljaju sa periodom 4. Grupisaćemo sabirke u grupe po 4 i pokazati da se svaka grupa poništava, čime dobijamo konačan rezultat.

Ključne formule i pojmovi
  • Stepeni imaginarne jedinice (ciklus perioda 4): \[ i^0 = 1, \quad i^1 = i, \quad i^2 = -1, \quad i^3 = -i \] Zatim se ciklus ponavlja: \(i^4 = 1,\; i^5 = i,\; i^6 = -1, \ldots\)
  • Opšte pravilo: Za bilo koji ceo broj \(n\), važi \(i^n = i^{n \bmod 4}\), gde \(\bmod\) označava ostatak pri deljenju sa 4.
  • Suma jednog punog ciklusa: \[ i^0 + i^1 + i^2 + i^3 = 1 + i + (-1) + (-i) = 0 \]
  • Formula za zbir geometrijskog reda: \[ \sum_{k=0}^{n-1} q^k = \frac{1 - q^n}{1 - q}, \quad q \neq 1 \]

Rešenje korak po korak

1

Zapišimo stepene imaginarne jedinice

Imaginarna jedinica \(i\) ima posebno svojstvo: njeni stepeni se ciklično ponavljaju sa periodom 4.

\[ i^0 = 1, \quad i^1 = i, \quad i^2 = -1, \quad i^3 = -i \]

Posle toga, ciklus se ponavlja: \(i^4 = (i^2)^2 = (-1)^2 = 1\), pa opet \(i^5 = i\), itd.

2

Suma jednog ciklusa je nula

Saberemo li četiri uzastopna stepena koji čine jedan pun ciklus, dobijamo:

\[ i^0 + i^1 + i^2 + i^3 = 1 + i + (-1) + (-i) = 0 \]

Realni delovi se poništavaju (\(1 + (-1) = 0\)), a imaginarni delovi se takođe poništavaju (\(i + (-i) = 0\)).

Proveri se: Koliko iznosi \(i^4 + i^5 + i^6 + i^7\)?
\(i^4 + i^5 + i^6 + i^7 = 1 + i + (-1) + (-i) = 0\). Isti rezultat kao i prvi ciklus!
3

Odredimo broj sabiraka

Naš izraz glasi \(1 + i + i^2 + \cdots + i^{2011}\), što je isto kao \(i^0 + i^1 + i^2 + \cdots + i^{2011}\).

Sabirci idu od \(i^0\) do \(i^{2011}\), što je ukupno:

\[ 2011 - 0 + 1 = 2012 \text{ sabiraka} \]
4

Grupisanje u cikluse po 4

Pošto je svaki ciklus dužine 4, proverimo da li je 2012 deljivo sa 4:

\[ 2012 \div 4 = 503 \]

Broj 2012 je tačno deljiv sa 4, bez ostatka. Dakle, imamo tačno 503 kompletna ciklusa.

Proveri se: Zašto je bitno da je broj sabiraka deljiv sa 4?
Ako je broj sabiraka deljiv sa 4, svi ciklusi su kompletni i svaki daje sumu 0. Da nije deljivo, imali bismo „višak“ sabiraka na kraju čija suma ne bi bila 0.
5

Izračunamo ukupnu sumu

Svaki od 503 ciklusa ima sumu 0, pa je ukupna suma:

\[ S = \underbrace{(1 + i - 1 - i)}_{\text{1. ciklus} = 0} + \underbrace{(1 + i - 1 - i)}_{\text{2. ciklus} = 0} + \cdots + \underbrace{(1 + i - 1 - i)}_{\text{503. ciklus} = 0} \]
\[ S = 503 \times 0 = 0 \]
6

Provera formulom za geometrijski red

Naš izraz je geometrijski red sa prvim članom \(a_1 = 1\), količnikom \(q = i\) i brojem članova \(n = 2012\):

\[ S = \frac{1 - i^{2012}}{1 - i} \]

Izračunajmo \(i^{2012}\):

\[ i^{2012} = (i^4)^{503} = 1^{503} = 1 \]

Uvrstimo nazad:

\[ S = \frac{1 - 1}{1 - i} = \frac{0}{1 - i} = 0 \]

Oba pristupa daju isti rezultat, što potvrđuje tačnost.

Proveri se: Koja bi bila suma da je poslednji stepen \(i^{2012}\) umesto \(i^{2011}\)?
Tada bismo imali 2013 sabiraka. \(2013 = 4 \times 503 + 1\), dakle 503 kompletna ciklusa (suma 0) plus jedan preostali član \(i^{2012} = 1\). Suma bi bila \(0 + 1 = 1\).

Ključno zapažanje

Stepeni imaginarne jedinice se ponavljaju sa periodom 4, a suma svakog kompletnog ciklusa je nula. Kad god je broj sabiraka deljiv sa 4, celokupna suma je 0 — bez obzira koliko ima članova.

\[ \underbrace{i^0 + i^1 + i^2 + i^3}_{=\,0} + \cdots + \underbrace{i^{2008} + i^{2009} + i^{2010} + i^{2011}}_{=\,0} = 0 \]

Konačan odgovor

\(1 + i + i^2 + \cdots + i^{2011} = 0\)

Tačan odgovor je D)

(A) \(1\)
(B) \(1 + i\)
(C) \(i\)
(D) \(0\)
(E) \(1 - i\)

Verifikacija: Izraz ima 2012 sabiraka (od \(i^0\) do \(i^{2011}\)). Pošto je \(2012 = 4 \times 503\), imamo tačno 503 kompletna ciklusa. Svaki ciklus \(1 + i + (-1) + (-i) = 0\), pa je ukupna suma \(503 \times 0 = 0\). Alternativno, formulom za geometrijski red: \(S = \frac{1 - i^{2012}}{1 - i} = \frac{1 - 1}{1 - i} = 0\).

Česte greške

  • Greška 1: Pogrešno brojanje sabiraka — od \(i^0\) do \(i^{2011}\) ima 2012 sabiraka, ne 2011. Mnogi učenici zaborave da se broji i nulti stepen. Ako pogrešno kažete da ima 2011 sabiraka, dobili biste nepotpun poslednji ciklus (3 ostatka) i pogrešan rezultat \(-i\).
  • Greška 2: Zaboravljanje da je \(i^2 = -1\), ne \(+1\). Ovo je osnovno svojstvo imaginarne jedinice koje je ključno za celokupno rešenje. Ako pomešate \(i^2 = 1\) i \(i^2 = -1\), ciklus neće raditi kako treba.

Dodatni izazov

Kolika je vrednost izraza \(1 + i + i^2 + \cdots + i^{2023}\)?

Nagoveštaj

Odredite koliko sabiraka ima (2024) i proverite deljivost sa 4. \(2024 = 4 \times 506\), dakle opet je tačno deljivo — suma je ponovo 0. A šta ako bi izraz bio \(1 + i + i^2 + \cdots + i^{2024}\)? Tada je 2025 sabiraka, \(2025 = 4 \times 506 + 1\), pa preostaje jedan član \(i^{2024} = (i^4)^{506} = 1\). Rezultat bi bio 1.
