Polinom drugog stepena i logaritamski izraz

Određivanje koeficijenata polinoma iz zadatih vrednosti i izračunavanje izraza sa logaritmom i korenom

📝 Postavka zadatka

Dat je polinom \(P(x) = ax^2 + bx + c\). Ako je \(P(0) = 4\), \(P(1) = 5\) i \(P(-1) = 9\), onda je vrednost izraza

\[ \log(c - a) + \sqrt{b^2} \]

🧭 Plan rešavanja

Uvrstićemo zadate vrednosti \(x = 0\), \(x = 1\) i \(x = -1\) u polinom da dobijemo sistem tri jednačine sa tri nepoznate (\(a\), \(b\), \(c\)). Nakon što odredimo koeficijente, izračunaćemo traženi izraz \(\log(c - a) + \sqrt{b^2}\).

📐 Ključne formule i teoreme
  • Polinom drugog stepena: \(P(x) = ax^2 + bx + c\), gde su \(a\), \(b\) i \(c\) realni koeficijenti, a \(a \neq 0\).
  • Vrednost polinoma u tački: Da bismo izračunali \(P(x_0)\), zamenjujemo \(x = x_0\) u izraz za polinom.
  • Logaritam jedinice: \(\log(1) = 0\) za bilo koju osnovu logaritma, jer je \(b^0 = 1\) za svako \(b > 0\), \(b \neq 1\).
  • Koren kvadrata: \(\sqrt{b^2} = |b|\) za svako realno \(b\). Rezultat je uvek nenegativan.

📊 Grafički prikaz

Grafik parabole \(P(x) = 3x^2 - 2x + 4\) sa označenim tačkama \(P(0) = 4\), \(P(1) = 5\) i \(P(-1) = 9\).

📋 Rešenje korak po korak

1

Određivanje koeficijenta \(c\) iz uslova \(P(0) = 4\)

Uvrstimo \(x = 0\) u polinom \(P(x) = ax^2 + bx + c\):

\[ P(0) = a \cdot 0^2 + b \cdot 0 + c = c \]

Pošto je dato da je \(P(0) = 4\), odmah dobijamo:

\[ c = 4 \]
2

Jednačina iz uslova \(P(1) = 5\)

Uvrstimo \(x = 1\):

\[ P(1) = a \cdot 1^2 + b \cdot 1 + c = a + b + c \]

Pošto je \(P(1) = 5\) i \(c = 4\):

\[ a + b + 4 = 5 \]

Odavde:

\[ a + b = 1 \quad \ldots \text{(I)} \]
3

Jednačina iz uslova \(P(-1) = 9\)

Uvrstimo \(x = -1\):

\[ P(-1) = a \cdot (-1)^2 + b \cdot (-1) + c = a - b + c \]

Pošto je \(P(-1) = 9\) i \(c = 4\):

\[ a - b + 4 = 9 \]

Odavde:

\[ a - b = 5 \quad \ldots \text{(II)} \]
🔍 Proveri sebe: Koji sistem jednačina treba rešiti?
Sistem je: \(a + b = 1\) i \(a - b = 5\). Sabiranjem ove dve jednačine eliminišemo \(b\).
4

Rešavanje sistema jednačina

Saberemo jednačine (I) i (II):

\[ (a + b) + (a - b) = 1 + 5 \]
\[ 2a = 6 \]
\[ a = 3 \]

Uvrstimo \(a = 3\) u jednačinu (I):

\[ 3 + b = 1 \]
\[ b = -2 \]

Dakle, polinom je \(P(x) = 3x^2 - 2x + 4\).

🔍 Proveri sebe: Koliko iznosi \(P(-1)\) za dobijene koeficijente?
\(P(-1) = 3 \cdot 1 - (-2) \cdot 1 + 4 = 3 + 2 + 4 = 9\). Tačno se poklapa sa zadatim uslovom!
5

Izračunavanje \(\log(c - a)\)

Izračunamo razliku \(c - a\):

\[ c - a = 4 - 3 = 1 \]

Logaritam jedinice je uvek nula, bez obzira na osnovu:

\[ \log(1) = 0 \]

Podsetimo se: \(\log_b(1) = 0\) za svako \(b > 0\), \(b \neq 1\), jer je \(b^0 = 1\).

6

Izračunavanje \(\sqrt{b^2}\)

Koristimo osobinu da je \(\sqrt{b^2} = |b|\):

\[ \sqrt{b^2} = \sqrt{(-2)^2} = \sqrt{4} = 2 \]

Pažnja: \(\sqrt{b^2} \neq b\) u opštem slučaju! Koren kvadrata daje apsolutnu vrednost, pa je \(\sqrt{(-2)^2} = |-2| = 2\), a ne \(-2\).

7

Konačno izračunavanje traženog izraza

Saberemo dobijene vrednosti:

\[ \log(c - a) + \sqrt{b^2} = 0 + 2 = 2 \]
🔍 Proveri sebe: Zašto je \(\sqrt{b^2} = 2\), a ne \(-2\)?
Zato što kvadratni koren po definiciji daje nenegativnu vrednost. Važi \(\sqrt{b^2} = |b|\), pa bez obzira na znak broja \(b\), rezultat je uvek nenegativan. Ovde: \(\sqrt{(-2)^2} = |-2| = 2\).
Ključni uvid

Izraz \(\sqrt{b^2}\) je zapravo \(|b|\), a \(\log(1) = 0\). Ova dva matematička fakta su srž zadatka. Razlika \(c - a\) "savršeno" ispada \(1\), čime se logaritam poništava i ostaje samo apsolutna vrednost od \(b\).

\[ \log(c - a) + \sqrt{b^2} = \log(1) + |{-2}| = 0 + 2 = 2 \]
Konačan odgovor
\(2\)
(A) \(4\) (B) \(-1\) (C) \(2\) (D) \(-2\) (E) \(3\)

Verifikacija: Koeficijenti \(a = 3\), \(b = -2\), \(c = 4\) zadovoljavaju sve tri jednačine: \(P(0) = 4\), \(P(1) = 3 - 2 + 4 = 5\), \(P(-1) = 3 + 2 + 4 = 9\). Traženi izraz: \(\log(4 - 3) + \sqrt{(-2)^2} = \log(1) + 2 = 0 + 2 = 2\).

⚠️ Česte greške

  • Zamena \(\sqrt{b^2}\) sa \(b\) umesto \(|b|\). Ako se zaboravi apsolutna vrednost, dobija se \(\sqrt{b^2} = b = -2\), pa bi odgovor bio \(0 + (-2) = -2\), što je pogrešan odgovor (D). Uvek važi \(\sqrt{b^2} = |b| \geq 0\).
  • Greška pri izračunavanju \(P(-1)\). Treba paziti na predznak: \(b \cdot (-1) = -b\), pa je \(P(-1) = a - b + c\), a ne \(a + b + c\). Pogrešan predznak kod \(b\) vodi ka pogrešnim vrednostima koeficijenata.
Dodatni izazov

Ako je dat polinom \(Q(x) = ax^3 + bx^2 + cx + d\) sa uslovima \(Q(0) = 1\), \(Q(1) = 3\), \(Q(-1) = -1\) i \(Q(2) = 15\), koliko iznosi \(a + b + c + d\)?

Pogledaj nagoveštaj

Iz \(Q(0) = 1\) odmah sledi \(d = 1\). Iz \(Q(1) = 3\) dobijamo \(a + b + c + d = 3\), pa je odmah \(a + b + c + d = 3\). Ali probajte odrediti i sve pojedinačne koeficijente koristeći preostala dva uslova!
