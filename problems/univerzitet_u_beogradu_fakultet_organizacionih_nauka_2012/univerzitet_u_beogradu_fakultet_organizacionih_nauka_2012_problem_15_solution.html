Polinom drugog stepena i logaritamski izraz

Algebra — Sistem linearnih jednačina, logaritmi, apsolutna vrednost

📋 Tekst zadatka

\[ \log(c - a) + \sqrt{b^2} \]

🎯 Plan rešavanja

Uvrstimo redom \(x = 0\), \(x = 1\) i \(x = -1\) u polinom da dobijemo tri jednačine. Iz prve odmah nalazimo \(c\), a preostale dve čine sistem od dve jednačine sa dve nepoznate (\(a\) i \(b\)). Rešavanjem sistema i uvršćivanjem u traženi izraz, koristimo činjenice da je \(\log(1) = 0\) i \(\sqrt{b^2} = |b|\).

📚 Podsetnik iz teorije
📐 Ključne formule i svojstva

Vrednost polinoma u tačci:

Za \(P(x) = ax^2 + bx + c\), uvršćivanje konkretne vrednosti \(x_0\) daje jednačinu po koeficijentima.

\[ P(0) = c, \quad P(1) = a + b + c, \quad P(-1) = a - b + c \]

Koren kvadrata:

\[ \sqrt{b^2} = |b| \quad \text{za svako realno } b \]

Rezultat je uvek nenegativan. Ovo je česta „zamka" na testovima.

Logaritam jedinice:

\[ \log_a(1) = 0 \quad \text{za svaku osnovu } a > 0,\ a \neq 1 \]

Jer je \(a^0 = 1\). Važi za dekadni logaritam, prirodni, bilo koji.

📈 Vizuelni prikaz — Grafik parabole i zadate tačke

Parabola \(P(x) = 3x^2 - 2x + 4\) sa tri zadate tačke koje jednoznačno određuju koeficijente

Prevucite miš preko grafika da vidite vrednosti funkcije u svakoj tačci

🔢 Rešenje korak po korak
1
Odredimo \(c\) iz uslova \(P(0) = 4\)

Uvrstimo \(x = 0\) u polinom \(P(x) = ax^2 + bx + c\):

\[ P(0) = a \cdot 0^2 + b \cdot 0 + c = c \]

Pošto je \(P(0) = 4\), odmah sledi:

\[ \boxed{c = 4} \]

Kada uvrstimo \(x = 0\), svi članovi osim slobodnog nestaju — zato je \(P(0)\) uvek jednak koeficijentu \(c\).

2
Jednačina iz uslova \(P(1) = 5\)

Uvrstimo \(x = 1\) i \(c = 4\):

\[ P(1) = a + b + c = a + b + 4 = 5 \]

Odatle:

\[ a + b = 1 \qquad \ldots \text{(I)} \]
3
Jednačina iz uslova \(P(-1) = 9\)

Uvrstimo \(x = -1\). Pažnja: \((-1)^2 = 1\), a \(b \cdot (-1) = -b\):

\[ P(-1) = a \cdot (-1)^2 + b \cdot (-1) + c = a - b + 4 = 9 \]

Odatle:

\[ a - b = 5 \qquad \ldots \text{(II)} \]
✅ Proveri sebe: Zašto je \((-1)^2 = 1\), a ne \(-1\)?

Jer se kvadrira ceo broj \(-1\), pa je \((-1)^2 = (-1) \cdot (-1) = +1\). Negativan puta negativan daje pozitivan. To znači da je koeficijent uz \(a\) pozitivan, tj. \(P(-1) = a - b + c\).

4
Rešimo sistem sabiranjem jednačina

Saberemo jednačine (I) i (II) da eliminišemo \(b\):

\[ \underbrace{(a + b)}_{=\,1} + \underbrace{(a - b)}_{=\,5} = 6 \quad \Rightarrow \quad 2a = 6 \quad \Rightarrow \quad \boxed{a = 3} \]

Uvrstimo \(a = 3\) nazad u jednačinu (I):

\[ 3 + b = 1 \quad \Rightarrow \quad \boxed{b = -2} \]

Dobili smo polinom: \(P(x) = 3x^2 - 2x + 4\).

✅ Proveri sebe: Verifikuj sa sva tri uslova

\(P(0) = 3 \cdot 0 - 2 \cdot 0 + 4 = 4\) ✓

\(P(1) = 3 \cdot 1 - 2 \cdot 1 + 4 = 3 - 2 + 4 = 5\) ✓

\(P(-1) = 3 \cdot 1 - 2 \cdot (-1) + 4 = 3 + 2 + 4 = 9\) ✓

Sva tri uslova su zadovoljena.

5
Izračunamo delove traženog izraza

Sa \(a = 3\), \(b = -2\), \(c = 4\) računamo:

\[ c - a = 4 - 3 = 1 \]
\[ \log(c - a) = \log(1) = 0 \]

jer je \(10^0 = 1\) (logaritam jedinice je uvek nula).

\[ \sqrt{b^2} = \sqrt{(-2)^2} = \sqrt{4} = 2 \]

Ključno: \(\sqrt{b^2} = |b| = |-2| = 2\), a ne \(-2\)! Koren daje nenegativnu vrednost.

6
Konačan rezultat

Saberemo:

\[ \log(c - a) + \sqrt{b^2} = 0 + 2 = \boxed{2} \]
✅ Proveri sebe: Zašto je \(\sqrt{b^2} = 2\), a ne \(-2\)?

Po definiciji, \(\sqrt{x}\) za \(x \geq 0\) uvek daje nenegativan rezultat. Važi identitet \(\sqrt{b^2} = |b|\). Za \(b = -2\): \(\sqrt{(-2)^2} = \sqrt{4} = 2\). Kada bi odgovor bio \(-2\), to bi značilo da je \(\sqrt{4} = -2\), što je pogrešno.

💡 Ključni uvid

Zadatak kombinuje dve oblasti: sisteme jednačina (za koeficijente) i algebarske identitete (za konačni izraz). Razlika \(c - a = 4 - 3 = 1\) daje \(\log(1) = 0\), čime se ceo prvi sabirak poništava. Drugi sabirak \(\sqrt{b^2} = |b| = 2\) zahteva razumevanje da koren kvadrata daje apsolutnu vrednost. Odgovor (D) \(-2\) je najčešća zamka — dobija se ako se pogrešno uzme \(\sqrt{b^2} = b\) umesto \(|b|\).

⚠️ Česte greške
✗

Zamena \(\sqrt{b^2}\) sa \(b\) umesto \(|b|\): Ovo je najčešća greška i vodi ka odgovoru (D) \(-2\). Ispravno: \(\sqrt{b^2} = |b| \geq 0\), pa je \(\sqrt{(-2)^2} = 2\), ne \(-2\). Na ispitu, odgovor \(-2\) je namerno ponuđen kao zamka.

✗

Pogreška u znaku pri \(P(-1)\): Pažljivo: \(a \cdot (-1)^2 = a \cdot 1 = a\) (pozitivno!), a \(b \cdot (-1) = -b\). Dakle \(P(-1) = a - b + c\). Greška \(P(-1) = -a - b + c\) daje pogrešne koeficijente.

✗

Brkanje osnove logaritma: Bez obzira da li je \(\log\) dekadni, prirodni ili sa bilo kojom osnovom, važi \(\log(1) = 0\). Ovde osnova uopšte ne utiče na rezultat jer je argument tačno \(1\).

🏆 Dodatni izazov

Neka je \(Q(x) = ax^2 + bx + c\) polinom za koji važi \(Q(0) = 2\), \(Q(2) = 14\) i \(Q(-2) = 22\). Odredi vrednost izraza \(\log_2(c - a) + \sqrt[3]{b^3}\).

Pogledaj nagoveštaj

Iz \(Q(0) = 2\) sledi \(c = 2\).

Iz \(Q(2) = 4a + 2b + 2 = 14\) sledi \(4a + 2b = 12\), tj. \(2a + b = 6\).

Iz \(Q(-2) = 4a - 2b + 2 = 22\) sledi \(4a - 2b = 20\), tj. \(2a - b = 10\).

Sabiranjem: \(4a = 16\), pa \(a = 4\), \(b = -2\).

Izraz: \(\log_2(2 - 4) + \sqrt[3]{(-2)^3} = \log_2(-2) + (-2)\). Ali \(\log_2(-2)\) ne postoji! Izraz nije definisan. Razmisli o tome šta ovo znači za postavljanje zadatka.
