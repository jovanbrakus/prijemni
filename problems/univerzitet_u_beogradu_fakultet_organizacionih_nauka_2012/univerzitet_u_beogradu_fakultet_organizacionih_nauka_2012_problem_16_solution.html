Logaritamska nejednačina sa osnovom 1/2

Algebra — logaritamske nejednačine, domen, kvadratne nejednačine

Z Postavka zadatka

Skup realnih rešenja nejednačine \(\log_{\frac{1}{2}}(x^2 - 4x + 3) \geq -3\) je:

(A)
\([-1,\, 5]\)
(B)
\([-1,\, 1) \cup (3,\, 5]\)
(C)
\([-1,\, 1)\)
(D)
\((3,\, 5]\)
(E)
\((-\infty,\, -1] \cup [5,\, +\infty)\)
P Plan rešavanja

Prvo ćemo odrediti domen (argument logaritma mora biti strogo pozitivan), zatim ćemo iskoristiti činjenicu da logaritam sa osnovom između 0 i 1 obrće smer nejednakosti. Na kraju ćemo naći presek dobijenih uslova.

Ključne formule i teoreme
Logaritamska nejednačina (osnova 0 < a < 1)
\[\log_a(u) \geq c \iff 0 < u \leq a^c\] Kada je osnova između 0 i 1, logaritamska funkcija je opadajuća, pa se smer nejednakosti menja.
Kvadratna nejednačina
\[(x - x_1)(x - x_2) \leq 0 \iff x_1 \leq x \leq x_2\] za \(x_1 < x_2\). Analogno za strogi znak.
Domen logaritma
\[\log_a(u) \text{ definisan } \iff u > 0\] Argument logaritma mora biti strogo pozitivan, bez obzira na osnovu.
Stepen sa negativnim eksponentom
\[\left(\frac{1}{2}\right)^{-3} = 2^3 = 8\]
D Grafički prikaz

Grafik funkcije \(f(x) = x^2 - 4x + 3\) sa označenim oblastima rešenja. Prevucite klizač ispod za promenu gornje granice.

Gornja granica: 8
K Rešenje korak po korak
1
Određivanje domena

Da bi logaritam bio definisan, argument mora biti strogo pozitivan:

\[x^2 - 4x + 3 > 0\]

Faktorizujemo kvadratni trinom:

\[x^2 - 4x + 3 = (x - 1)(x - 3)\]

Proizvod dva faktora je pozitivan kada su oba pozitivna ili oba negativna. Nule su \(x = 1\) i \(x = 3\).

\[(x - 1)(x - 3) > 0 \iff x \in (-\infty, 1) \cup (3, +\infty)\]

Tačke \(x = 1\) i \(x = 3\) ne pripadaju domenu jer u njima argument logaritma iznosi 0.

2
Preformulisanje nejednačine

Osnova logaritma je \(\frac{1}{2}\), što je broj između 0 i 1. Za takvu osnovu, logaritamska funkcija je opadajuća. To znači da važi pravilo:

\[\log_a(u) \geq c \iff 0 < u \leq a^c \quad \text{(za } 0 < a < 1\text{)}\]

Primenjujemo ovo na našu nejednačinu:

\[\log_{\frac{1}{2}}(x^2 - 4x + 3) \geq -3\]

postaje sistem:

\[0 < x^2 - 4x + 3 \leq \left(\frac{1}{2}\right)^{-3}\]
Proveri sebe: Koliko je \(\left(\frac{1}{2}\right)^{-3}\)?
\(\left(\frac{1}{2}\right)^{-3} = 2^3 = 8\). Negativan eksponent „obrće" razlomak, pa \(\frac{1}{2}\) postaje 2, a zatim stepenujemo sa 3.
3
Računanje gornje granice

Izračunajmo vrednost \(\left(\frac{1}{2}\right)^{-3}\):

\[\left(\frac{1}{2}\right)^{-3} = 2^3 = 8\]

Dakle, naš sistem glasi:

\[0 < x^2 - 4x + 3 \leq 8\]
4
Rešavanje nejednačine \(x^2 - 4x + 3 \leq 8\)

Preuredimo nejednačinu:

\[x^2 - 4x + 3 \leq 8\]
\[x^2 - 4x - 5 \leq 0\]

Faktorizujemo levi deo. Tražimo dva broja čiji je proizvod \(-5\) a zbir \(-4\). To su \(-5\) i \(1\):

\[x^2 - 4x - 5 = (x - 5)(x + 1)\]

Proizvod \((x - 5)(x + 1) \leq 0\) je zadovoljen kada su faktori različitog znaka (ili jedan od njih je nula). Nule su \(x = -1\) i \(x = 5\):

\[(x - 5)(x + 1) \leq 0 \iff x \in [-1, 5]\]
Proveri sebe: Da li \(x = -1\) i \(x = 5\) zadovoljavaju nejednačinu?
Za \(x = -1\): \((-1)^2 - 4 \cdot (-1) + 3 = 1 + 4 + 3 = 8 \leq 8\). Da!
Za \(x = 5\): \(5^2 - 4 \cdot 5 + 3 = 25 - 20 + 3 = 8 \leq 8\). Da!
Obe granične tačke zadovoljavaju nejednačinu (jednakost važi).
5
Presek svih uslova

Sada kombinujemo oba uslova:

\[\text{Uslov 1 (domen): } x \in (-\infty, 1) \cup (3, +\infty)\]
\[\text{Uslov 2 (nejednačina): } x \in [-1, 5]\]

Tražimo presek ova dva skupa:

\[[-1, 5] \cap \big((-\infty, 1) \cup (3, +\infty)\big)\]

Računamo presek sa svakim delom unije posebno:

\[[-1, 5] \cap (-\infty, 1) = [-1, 1)\]
\[[-1, 5] \cap (3, +\infty) = (3, 5]\]

Unija ova dva dela daje konačno rešenje:

\[x \in [-1, 1) \cup (3, 5]\]
6
Zaključak i verifikacija

Skup realnih rešenja nejednačine je:

\[\boxed{[-1, 1) \cup (3, 5]}\]

Proverimo rubne tačke:

  • \(x = -1\): argument = 8, \(\log_{1/2}(8) = -3 \geq -3\) — zadovoljeno
  • \(x = 5\): argument = 8, \(\log_{1/2}(8) = -3 \geq -3\) — zadovoljeno
  • \(x = 1\): argument = 0 — logaritam nije definisan, NE pripada
  • \(x = 3\): argument = 0 — logaritam nije definisan, NE pripada
Proveri sebe: Zašto je \(\log_{1/2}(8) = -3\)?
Tražimo \(n\) tako da \(\left(\frac{1}{2}\right)^n = 8\). Pošto je \(8 = 2^3\), a \(\left(\frac{1}{2}\right)^{-3} = 2^3 = 8\), zaključujemo da je \(n = -3\). Dakle \(\log_{1/2}(8) = -3\).
Ključni uvid

Kada je osnova logaritma između 0 i 1, logaritamska funkcija je opadajuća. To znači da se pri prelazu sa logaritamskog oblika na eksponencijalni obrće smer nejednakosti. Nejednačina \(\log_{1/2}(u) \geq -3\) postaje \(u \leq 8\), a ne \(u \geq 8\).

Nikada ne zaboravite na uslov \(u > 0\) za domen logaritma — upravo taj uslov „izbacuje" interval \([1, 3]\) iz konačnog rešenja!

Konačan odgovor
\([-1, 1) \cup (3, 5]\)
(A) \([-1, 5]\)
(B) \([-1, 1) \cup (3, 5]\)
(C) \([-1, 1)\)
(D) \((3, 5]\)
(E) \((-\infty, -1] \cup [5, +\infty)\)
Provera: Za \(x = 0\) (pripada \([-1, 1)\)): argument = 3, \(\log_{1/2}(3) \approx -1{,}585 \geq -3\). Za \(x = 4\) (pripada \((3, 5]\)): argument = 3, \(\log_{1/2}(3) \approx -1{,}585 \geq -3\). Za \(x = 2\) (ne pripada rešenju): argument = \(-1 < 0\), logaritam nije definisan. Sve se poklapa sa odgovorom B.
! Česte greške
! Zaboravljanje uslova domena: Ako se ne uzme u obzir da mora biti \(x^2 - 4x + 3 > 0\), dobija se \([-1, 5]\) (opcija A) umesto tačnog rešenja. Interval \([1, 3]\) mora biti isključen jer u njemu argument logaritma nije pozitivan.
! Pogrešan smer nejednakosti: Osnova \(\frac{1}{2}\) je manja od 1, pa logaritamska funkcija opada. Česta greška je da se nejednačina pretvori u \(x^2 - 4x + 3 \geq 8\) umesto \(x^2 - 4x + 3 \leq 8\), što vodi na opciju E.
Dodatni izazov

Kako bi se promenilo rešenje da je nejednačina bila \(\log_{2}(x^2 - 4x + 3) \geq -3\) (osnova 2 umesto \(\frac{1}{2}\))?

Pogledaj uput
Za osnovu 2 (koja je veća od 1), logaritamska funkcija je rastuća. Tada bi nejednačina postala \(x^2 - 4x + 3 \geq 2^{-3} = \frac{1}{8}\), što je sasvim drugačiji problem. Razmisli o tome kako se rešenje menja kada se promeni smer nejednakosti.
