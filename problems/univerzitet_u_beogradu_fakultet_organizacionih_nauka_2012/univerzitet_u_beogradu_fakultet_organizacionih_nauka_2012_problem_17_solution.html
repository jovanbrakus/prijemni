P Plan rešavanja

Primenićemo identitet za dvostruki ugao \(\cos 2x = 1 - 2\sin^2 x\) da bismo sveli jednačinu na jednu trigonometrijsku funkciju. Potom ćemo je faktorizovati i naći sva rešenja na datom intervalu.

Ključne formule i teoreme
Dvostruki ugao — kosinus
\[\cos 2x = 1 - 2\sin^2 x\] Jedan od tri oblika formule za dvostruki ugao kosinusa. Koristan kada želimo da izrazimo sve preko sinusa.
Rešenja \(\sin x = 0\)
\[\sin x = 0 \iff x = k\pi, \quad k \in \mathbb{Z}\] Na intervalu \((0, 2\pi)\): jedino rešenje je \(x = \pi\).
Rešenja \(\sin x = -\frac{1}{2}\)
\[\sin x = -\frac{1}{2} \iff x = \frac{7\pi}{6} + 2k\pi \text{ ili } x = \frac{11\pi}{6} + 2k\pi\] Sinus je \(-\frac{1}{2}\) u trećem i četvrtom kvadrantu.
Faktorizacija
\[ab = 0 \iff a = 0 \text{ ili } b = 0\] Proizvod je jednak nuli ako i samo ako je bar jedan činilac jednak nuli.
D Grafički prikaz

Grafik funkcije \(f(x) = \cos 2x - \sin x\) na intervalu \((0, 2\pi)\). Horizontalna isprekidana linija predstavlja \(y = 1\). Presečne tačke su rešenja jednačine.

K Rešenje korak po korak
1
Primena identiteta za dvostruki ugao

Koristimo identitet \(\cos 2x = 1 - 2\sin^2 x\) da bismo celu jednačinu izrazili preko \(\sin x\):

\[\cos 2x - \sin x = 1\] \[(1 - 2\sin^2 x) - \sin x = 1\]
2
Sređivanje jednačine

Prebacujemo sve na jednu stranu i pojednostavljujemo:

\[1 - 2\sin^2 x - \sin x = 1\]
\[- 2\sin^2 x - \sin x = 0\]

Množimo obe strane sa \(-1\):

\[2\sin^2 x + \sin x = 0\]
3
Faktorizacija

Izvlačimo zajednički činilac \(\sin x\):

\[\sin x \cdot (2\sin x + 1) = 0\]

Proizvod je jednak nuli ako je bar jedan činilac nula, pa dobijamo dva slučaja:

\[\sin x = 0 \qquad \text{ili} \qquad 2\sin x + 1 = 0 \;\Rightarrow\; \sin x = -\frac{1}{2}\]
Proveri sebe: Zašto smo mogli da razdvojimo jednačinu na dva slučaja?
Zato što smo dobili proizvod dva izraza jednak nuli. Po osobini realnih brojeva, \(ab = 0\) važi ako i samo ako je \(a = 0\) ili \(b = 0\). Dakle, dovoljno je rešiti svaki činilac posebno.
4
Slučaj 1: \(\sin x = 0\)

Opšte rešenje jednačine \(\sin x = 0\) je \(x = k\pi\), gde je \(k \in \mathbb{Z}\). Na intervalu \((0, 2\pi)\) tražimo vrednosti:

  • \(k = 0\): \(x = 0\) — ne pripada otvorenom intervalu \((0, 2\pi)\)
  • \(k = 1\): \(x = \pi\) — pripada!
  • \(k = 2\): \(x = 2\pi\) — ne pripada otvorenom intervalu \((0, 2\pi)\)
\[\text{Iz ovog slučaja: } x_1 = \pi\]
5
Slučaj 2: \(\sin x = -\frac{1}{2}\)

Sinus je negativan u trećem i četvrtom kvadrantu. Referentni ugao za koji je sinus jednak \(\frac{1}{2}\) je \(\frac{\pi}{6}\). Dakle:

\[\sin x = -\frac{1}{2} \implies x = \pi + \frac{\pi}{6} + 2k\pi \quad \text{ili} \quad x = 2\pi - \frac{\pi}{6} + 2k\pi\]

Za \(k = 0\) na intervalu \((0, 2\pi)\):

\[x_2 = \frac{7\pi}{6} \qquad \text{i} \qquad x_3 = \frac{11\pi}{6}\]
Proveri sebe: Koliko je \(\frac{7\pi}{6}\) u stepenima?
\(\frac{7\pi}{6} = \frac{7 \cdot 180°}{6} = 210°\). To je ugao u trećem kvadrantu (između 180° i 270°), gde je sinus zaista negativan. Analogno, \(\frac{11\pi}{6} = 330°\) je u četvrtom kvadrantu.
6
Prebrojavanje i verifikacija

Dobili smo ukupno 3 rešenja na intervalu \((0, 2\pi)\):

\[x_1 = \pi, \quad x_2 = \frac{7\pi}{6}, \quad x_3 = \frac{11\pi}{6}\]

Proverimo svako rešenje zamenom u polaznu jednačinu \(\cos 2x - \sin x = 1\):

  • \(x = \pi\): \(\cos 2\pi - \sin \pi = 1 - 0 = 1\) — tačno
  • \(x = \frac{7\pi}{6}\): \(\cos \frac{7\pi}{3} - \sin \frac{7\pi}{6} = \frac{1}{2} - \left(-\frac{1}{2}\right) = \frac{1}{2} + \frac{1}{2} = 1\) — tačno
  • \(x = \frac{11\pi}{6}\): \(\cos \frac{11\pi}{3} - \sin \frac{11\pi}{6} = \frac{1}{2} - \left(-\frac{1}{2}\right) = \frac{1}{2} + \frac{1}{2} = 1\) — tačno
Proveri sebe: Kako smo izračunali \(\cos \frac{7\pi}{3}\)?
Koristimo periodičnost kosinusa: \(\cos \frac{7\pi}{3} = \cos\left(\frac{7\pi}{3} - 2\pi\right) = \cos \frac{\pi}{3} = \frac{1}{2}\). Oduzeli smo pun krug (\(2\pi = \frac{6\pi}{3}\)) da bismo sveli ugao u osnovni interval.
Ključni uvid

Kada se u trigonometrijskoj jednačini pojavi \(\cos 2x\) zajedno sa \(\sin x\), ključna ideja je upotrebiti identitet \(\cos 2x = 1 - 2\sin^2 x\) čime se cela jednačina svodi na kvadratnu jednačinu po \(\sin x\). Nakon faktorizacije, problem se razbija na jednostavnije jednačine.

Posebno obratite pažnju na to da li su krajnje tačke intervala uključene ili ne — ovde je interval otvoren \((0, 2\pi)\), pa rešenja \(x = 0\) i \(x = 2\pi\) ne ulaze u konačan odgovor.

Konačan odgovor
3

Provera: Jednačina \(\sin x(2\sin x + 1) = 0\) daje \(\sin x = 0\) (rešenje \(x = \pi\)) i \(\sin x = -\frac{1}{2}\) (rešenja \(x = \frac{7\pi}{6}\) i \(x = \frac{11\pi}{6}\)). Sva tri rešenja su verifikovana zamenom u polaznu jednačinu. Odgovor je D) 3.
! Česte greške
! Uključivanje krajnjih tačaka: Ako uzmemo da \(x = 0\) i \(x = 2\pi\) pripadaju rešenjima (jer je \(\sin 0 = 0\) i \(\sin 2\pi = 0\)), dobijamo 5 umesto 3 rešenja. Ali interval je otvoren: \((0, 2\pi)\), pa ove tačke ne ulaze.
! Pogrešan identitet: Korišćenje \(\cos 2x = 2\cos^2 x - 1\) umesto \(\cos 2x = 1 - 2\sin^2 x\) ne dovodi do greške, ali komplikuje dalji rad jer se dobija jednačina sa i sinusom i kosinusom. Biramo identitet koji svodi sve na jednu funkciju.
! Deljenje sa \(\sin x\): Ako podelimo jednačinu sa \(\sin x\), gubimo rešenja gde je \(\sin x = 0\) (tj. \(x = \pi\)). Uvek je bezbednije koristiti faktorizaciju nego deljenje.
Dodatni izazov

Koliko rešenja ima jednačina \(\cos 2x + \sin x = 1\) na intervalu \((0, 2\pi)\)? Obratite pažnju na promenu znaka ispred \(\sin x\).

Pogledaj uputstvo
Zamenom \(\cos 2x = 1 - 2\sin^2 x\) dobijamo \(-2\sin^2 x + \sin x = 0\), tj. \(\sin x(1 - 2\sin x) = 0\). Slučajevi su \(\sin x = 0\) (rešenje \(x = \pi\)) i \(\sin x = \frac{1}{2}\) (rešenja \(x = \frac{\pi}{6}\) i \(x = \frac{5\pi}{6}\)). Opet 3 rešenja, ali u potpuno drugačijim tačkama!
