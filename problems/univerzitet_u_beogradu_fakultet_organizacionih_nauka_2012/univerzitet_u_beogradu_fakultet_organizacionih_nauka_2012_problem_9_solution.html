Logaritamsko-eksponencijalni izraz

Primena svojstava logaritama i stepenovanja

Postavka zadatka

Vrednost izraza

\[3^{\,\log_{\sqrt[3]{9}}\, 4} \;+\; 2^{\,\frac{1}{\log_{49}\, 4}}\]

jednaka je:

(A)
\(15\)
(B)
\(12\)
(C)
\(14\)
(D)
\(11\)
(E)
\(13\)

Plan rešavanja

Izraz sadrži dva sabirka. Svaki ćemo pojednostaviti zasebno koristeći osnovna svojstva logaritama: zamenu baze, recipročnu vezu \(\frac{1}{\log_b a} = \log_a b\), i ključnu osobinu \(a^{\log_a x} = x\). Na kraju sabiramo rezultate.

Ključne formule i teoreme
  • Promena baze: \(\log_{a^k} b = \frac{1}{k}\,\log_a b\), što sledi iz \(\log_{a^k} b = \frac{\ln b}{k\,\ln a}\).
  • Recipročna veza: \(\frac{1}{\log_b a} = \log_a b\) — zamena baze i argumenta.
  • Fundamentalna osobina: \(a^{\log_a x} = x\) za \(a > 0,\; a \neq 1,\; x > 0\).
  • Stepenovanje sa logaritmom: \(a^{c\,\log_a x} = \bigl(a^{\log_a x}\bigr)^c = x^c\).
  • Kubni koren: \(\sqrt[3]{9} = 9^{1/3} = (3^2)^{1/3} = 3^{2/3}\).

Vizuelni prikaz transformacija

Dijagram toka: kako se svaki sabirak pojednostavljuje do konačnog broja

Rešenje korak po korak

1

Pojednostavljivanje baze prvog logaritma

Baza prvog logaritma je \(\sqrt[3]{9}\). Izrazimo je kao stepen trojke:

\[\sqrt[3]{9} = 9^{1/3} = \bigl(3^2\bigr)^{1/3} = 3^{2/3}\]

Dakle, radimo sa logaritmom po bazi \(3^{2/3}\).

2

Transformacija \(\log_{3^{2/3}} 4\)

Koristimo pravilo za logaritam čija je baza stepen nekog broja:

\[\log_{a^k} b = \frac{1}{k}\,\log_a b\]

Ovde je \(a = 3\), \(k = \frac{2}{3}\), \(b = 4\):

\[\log_{3^{2/3}} 4 = \frac{1}{\;\frac{2}{3}\;}\,\log_3 4 = \frac{3}{2}\,\log_3 4\]
3

Izračunavanje prvog sabirka

Uvrštavamo u eksponent:

\[3^{\,\log_{\sqrt[3]{9}}\, 4} = 3^{\,\frac{3}{2}\,\log_3 4}\]

Koristimo pravilo stepenovanja \(a^{c\,\log_a x} = x^c\):

\[3^{\,\frac{3}{2}\,\log_3 4} = \bigl(3^{\log_3 4}\bigr)^{3/2} = 4^{3/2}\]

Izračunavamo \(4^{3/2}\):

\[4^{3/2} = \bigl(\sqrt{4}\bigr)^3 = 2^3 = \boxed{8}\]
Proveri se: Koliko je \(4^{3/2}\)?
\(4^{3/2} = (\sqrt{4})^3 = 2^3 = 8\). Alternativno: \(4^{3/2} = (4^3)^{1/2} = \sqrt{64} = 8\).
4

Transformacija eksponenta drugog sabirka

U drugom sabirku imamo \(\frac{1}{\log_{49} 4}\). Koristimo recipročnu vezu logaritama:

\[\frac{1}{\log_{49} 4} = \log_4 49\]

Ovo sledi direktno iz identiteta \(\frac{1}{\log_b a} = \log_a b\), koji važi jer je \(\log_b a = \frac{\ln a}{\ln b}\), pa je recipročna vrednost \(\frac{\ln b}{\ln a} = \log_a b\).

5

Izračunavanje drugog sabirka

Sada imamo:

\[2^{\,\log_4 49}\]

Zapišemo \(\log_4 49\) preko baze 2. Pošto je \(4 = 2^2\):

\[\log_4 49 = \frac{\log_2 49}{\log_2 4} = \frac{\log_2 49}{2}\]

Uvrštavamo:

\[2^{\,\log_4 49} = 2^{\,\frac{\log_2 49}{2}} = \bigl(2^{\log_2 49}\bigr)^{1/2} = 49^{1/2} = \sqrt{49} = \boxed{7}\]
Proveri se: Zašto važi \(2^{\log_2 49} = 49\)?
To je fundamentalna osobina logaritma: \(a^{\log_a x} = x\) za svako \(x > 0\). Ovde je \(a = 2\) i \(x = 49\).
6

Konačno sabiranje

Sabiramo dobijene vrednosti:

\[3^{\,\log_{\sqrt[3]{9}}\, 4} + 2^{\,\frac{1}{\log_{49}\, 4}} = 8 + 7 = \boxed{15}\]

Ključni uvid

Obe transformacije se svode na istu ideju: prepisati eksponent tako da sadrži \(\log_a\) gde je \(a\) baza stepena, pa primeniti \(a^{\log_a x} = x\).

Za prvi sabirak: \(3^{\frac{3}{2}\log_3 4} = 4^{3/2} = 8\). Za drugi: \(2^{\frac{1}{2}\log_2 49} = 49^{1/2} = 7\).

Konačan odgovor

\(3^{\,\log_{\sqrt[3]{9}}\, 4} + 2^{\,\frac{1}{\log_{49}\, 4}} = 8 + 7 = 15\)
(A) \(15\)
(B) \(12\)
(C) \(14\)
(D) \(11\)
(E) \(13\)
Provera: \(4^{3/2} = (\sqrt{4})^3 = 2^3 = 8\) i \(\sqrt{49} = 7\). Zbir: \(8 + 7 = 15\). Odgovor je (A).

Najčešće greške

  • Pogrešna interpretacija baze: \(\sqrt[3]{9}\) je kubni koren od 9, tj. \(3^{2/3}\), a ne \(\sqrt{9} = 3\) niti \(9^3\). Pažljivo čitajte zapis korena i stepena.
  • Zamena formule za promenu baze: \(\log_{a^k} b = \frac{1}{k}\log_a b\) — deli se sa \(k\), ne množi. Ovde \(k = 2/3\), pa delimo sa \(2/3\) i dobijamo množenje sa \(3/2\).
  • Recipročna veza: \(\frac{1}{\log_{49} 4} = \log_4 49\), ne \(\log_4 \frac{1}{49}\). Zamenjuju se baza i argument, a ne uzima recipročna vrednost argumenta.
  • Greška u stepenima: \(4^{3/2} = 8\), ne \(6\). Uvek rastavite: \(4^{3/2} = (\sqrt{4})^3 = 2^3 = 8\).

Izazov za vežbu

Izračunaj: \(\;5^{\,\log_{\sqrt[4]{25}}\, 3} + 3^{\,\frac{1}{\log_{27}\, 9}}\)

Prvi sabirak: \(\sqrt[4]{25} = 25^{1/4} = (5^2)^{1/4} = 5^{1/2}\)

\(\log_{5^{1/2}} 3 = \frac{1}{1/2}\,\log_5 3 = 2\log_5 3\)

\(5^{2\log_5 3} = (5^{\log_5 3})^2 = 3^2 = 9\)

Drugi sabirak: \(\frac{1}{\log_{27} 9} = \log_9 27 = \frac{\log_3 27}{\log_3 9} = \frac{3}{2}\)

\(3^{3/2} = 3\sqrt{3}\)

Ukupno: \(9 + 3\sqrt{3}\)
