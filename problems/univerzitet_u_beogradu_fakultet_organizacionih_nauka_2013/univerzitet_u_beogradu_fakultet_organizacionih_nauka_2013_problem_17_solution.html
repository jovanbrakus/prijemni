Trigonometrijska jednačina sa kosinusima

Zadatak 17

Zbir svih rešenja jednačine

\[\cos^2\frac{x}{2} + \cos^2 x = \frac{1}{2}\]

koja pripadaju intervalu \((\pi,\, 2\pi)\) jednak je:

(A)
\(3\pi\)
(B)
\(\dfrac{9\pi}{2}\)
(C)
\(\dfrac{17\pi}{6}\)
(D)
\(\dfrac{11\pi}{4}\)
(E)
\(\dfrac{13\pi}{3}\)

📋 Plan rešavanja

  1. Primeniti identitete za \(\cos^2\frac{x}{2}\) i \(\cos^2 x\) da svedemo jednačinu na izraz sa \(\cos x\).
  2. Dobiti algebarsku jednačinu po \(\cos x\) i faktorizovati.
  3. Naći sva rešenja u intervalu \((\pi, 2\pi)\).
  4. Sabrati rešenja.
📖 Podsetnik iz teorije — trigonometrijski identiteti

Identitet poluugla:

\[\cos^2\frac{x}{2} = \frac{1 + \cos x}{2}\]

Identitet dvostrukog ugla:

\[\cos^2 x = \frac{1 + \cos 2x}{2}\]

Oba proizlaze iz formule \(\cos 2\alpha = 2\cos^2\alpha - 1\), odnosno:

\[\cos^2\alpha = \frac{1 + \cos 2\alpha}{2}\]

Važne vrednosti:

  • \(\cos x = 0 \;\Rightarrow\; x = \frac{\pi}{2} + k\pi,\; k \in \mathbb{Z}\)
  • \(\cos x = -\frac{1}{2} \;\Rightarrow\; x = \pm\frac{2\pi}{3} + 2k\pi,\; k \in \mathbb{Z}\)

📐 Vizuelni prikaz — jedinični krug

Rešenja jednačine na jediničnom krugu u intervalu \((\pi, 2\pi)\)

🔢 Rešenje korak po korak

Primenjujemo identitete za \(\cos^2\frac{x}{2}\) i \(\cos^2 x\):

\[\frac{1 + \cos x}{2} + \frac{1 + \cos 2x}{2} = \frac{1}{2}\]

Množimo obe strane sa \(2\):

\[1 + \cos x + 1 + \cos 2x = 1\] \[\cos x + \cos 2x = -1\]
Jednačina je svedena na dva kosinusa: \(\cos x + \cos 2x = -1\)

Koristimo identitet \(\cos 2x = 2\cos^2 x - 1\):

\[\cos x + 2\cos^2 x - 1 = -1\] \[2\cos^2 x + \cos x = 0\]

Iznosimo \(\cos x\) ispred zagrade:

\[\cos x\,(2\cos x + 1) = 0\]

Jednačina se raspada na dva slučaja:

SlučajJednačinaOpšte rešenje
I\(\cos x = 0\)\(x = \dfrac{\pi}{2} + k\pi,\; k \in \mathbb{Z}\)
II\(\cos x = -\dfrac{1}{2}\)\(x = \pm\dfrac{2\pi}{3} + 2k\pi,\; k \in \mathbb{Z}\)
Faktorizacija daje dva uslova: \(\cos x = 0\) ili \(\cos x = -\frac{1}{2}\)

Slučaj I: \(\cos x = 0\)

  • \(k = 0\): \(x = \frac{\pi}{2}\) — nije u \((\pi, 2\pi)\)
  • \(k = 1\): \(x = \frac{3\pi}{2}\) — jeste u \((\pi, 2\pi)\) ✓
  • \(k = 2\): \(x = \frac{5\pi}{2}\) — nije u \((\pi, 2\pi)\)

Slučaj II: \(\cos x = -\frac{1}{2}\)

  • \(x = \frac{2\pi}{3} + 2k\pi\): za \(k=0\) dobijamo \(\frac{2\pi}{3}\) — nije u \((\pi, 2\pi)\); za \(k=1\) dobijamo \(\frac{8\pi}{3}\) — nije u \((\pi, 2\pi)\)
  • \(x = -\frac{2\pi}{3} + 2k\pi\): za \(k=1\) dobijamo \(\frac{4\pi}{3}\) — jeste u \((\pi, 2\pi)\) ✓; za \(k=0\) dobijamo \(-\frac{2\pi}{3}\) — nije
Pronađena su tačno dva rešenja u datom intervalu

Rešenja u intervalu \((\pi, 2\pi)\) su:

\[x_1 = \frac{4\pi}{3} \quad \text{i} \quad x_2 = \frac{3\pi}{2}\]

Njihov zbir:

\[x_1 + x_2 = \frac{4\pi}{3} + \frac{3\pi}{2} = \frac{8\pi}{6} + \frac{9\pi}{6} = \frac{17\pi}{6}\]

💡 Ključni uvid

Identiteti za \(\cos^2\frac{x}{2}\) i \(\cos^2 x\) oba svode izraz na \(\cos x\), što omogućava da se trigonometrijska jednačina transformiše u kvadratnu jednačinu po \(\cos x\). Faktorizacija \(\cos x(2\cos x + 1) = 0\) daje dva jednostavna uslova, a pažljivim pregledom rešenja biramo samo ona iz traženog intervala.

Konačan odgovor

(C) \[\displaystyle \frac{17\pi}{6}\]

Odgovor je (C) \(\dfrac{17\pi}{6}\)

⚠️ Česte greške

Zaboraviti da \(\cos x = -\frac{1}{2}\) daje dva skupa rešenja: \(x = \frac{2\pi}{3} + 2k\pi\) i \(x = \frac{4\pi}{3} + 2k\pi\). Samo jedan pripada intervalu \((\pi, 2\pi)\).
Uključiti granice intervala: zadatak traži otvoreni interval \((\pi, 2\pi)\), pa \(x = \pi\) i \(x = 2\pi\) nisu validna rešenja.
Koristiti pogrešan identitet — npr. \(\cos^2\frac{x}{2} = \frac{1 - \cos x}{2}\) umesto ispravnog \(\frac{1 + \cos x}{2}\). To je identitet za \(\sin^2\frac{x}{2}\).
Greška pri sabiranju razlomaka: \(\frac{4\pi}{3} + \frac{3\pi}{2}\) zahteva svođenje na zajednički imenilac \(6\).

🏆 Dodatni izazov

Nađi zbir svih rešenja jednačine \(\cos^2\frac{x}{2} + \cos^2 x = \frac{1}{2}\) na intervalu \((0, 4\pi)\).

Prikaži rešenje

Na \((0, 4\pi)\) za \(\cos x = 0\): \(x = \frac{\pi}{2},\; \frac{3\pi}{2},\; \frac{5\pi}{2},\; \frac{7\pi}{2}\)

Za \(\cos x = -\frac{1}{2}\): \(x = \frac{2\pi}{3},\; \frac{4\pi}{3},\; \frac{8\pi}{3},\; \frac{10\pi}{3}\)

Zbir:

\[\frac{\pi}{2} + \frac{3\pi}{2} + \frac{5\pi}{2} + \frac{7\pi}{2} + \frac{2\pi}{3} + \frac{4\pi}{3} + \frac{8\pi}{3} + \frac{10\pi}{3} = 8\pi + 8\pi = 16\pi\]
