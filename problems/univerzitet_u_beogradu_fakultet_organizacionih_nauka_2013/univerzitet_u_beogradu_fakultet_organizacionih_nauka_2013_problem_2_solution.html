Kompleksni broj iz dva uslova

Jednakost modula i geometrijsko tumačenje u kompleksnoj ravni

? Zadatak

Ako za kompleksan broj \(z\) važi \(|z - 3| = |z - 3 + 2i|\) i \(|z - 2i| = |z + 4 - 2i|\), gde je \(i^2 = -1\), tada je:

(A)
\(|z| = 2\)
(B)
\(|z| = 2\sqrt{5}\)
(C)
\(|z| = 5\)
(D)
\(|z| = 3\)
(E)
\(|z| = \sqrt{5}\)
⚙ Plan rešavanja

Postavićemo \(z = x + yi\) i svaki uslov prevesti u jednačinu sa realnim promenljivama \(x\) i \(y\). Jednakost modula \(|z - a| = |z - b|\) znači da je tačka \(z\) jednako udaljena od tačaka \(a\) i \(b\) u kompleksnoj ravni — to je simetrala duži \(ab\). Presek dve simetrale daje tačku \(z\), a zatim računamo \(|z|\).

Teorijski podsetnik

Modul kompleksnog broja: Za \(z = x + yi\) važi \(|z| = \sqrt{x^2 + y^2}\).

Geometrijsko značenje modula razlike: \(|z - w|\) predstavlja rastojanje između tačaka \(z\) i \(w\) u kompleksnoj ravni.

Geometrijsko mesto tačaka: Uslov \(|z - a| = |z - b|\) znači da je tačka \(z\) jednako udaljena od tačaka \(a\) i \(b\). To je simetrala (simetral) duži \(\overline{ab}\) — prava normalna na duž \(ab\) koja prolazi kroz njenu sredinu.

Algebarski pristup: Kvadriranjem obe strane jednakosti \(|z - a| = |z - b|\) i razvojem dobijamo linearnu jednačinu po \(x\) i \(y\).

📊 Vizuelni prikaz

Kompleksna ravan sa ključnim tačkama, simetralama i rešenjem \(z = -2 - i\).

1 Zapišimo \(z = x + yi\)

Neka je \(z = x + yi\), gde su \(x, y \in \mathbb{R}\). Sada ćemo svaki uslov prevesti u jednačinu po \(x\) i \(y\).

2 Obradimo prvi uslov: \(|z - 3| = |z - 3 + 2i|\)

Uvrstimo \(z = x + yi\):

\[ |z - 3| = |(x - 3) + yi| \] \[ |z - 3 + 2i| = |z - (3 - 2i)| = |(x - 3) + (y + 2)i| \]

Kvadriramo obe strane jednakosti:

\[ (x - 3)^2 + y^2 = (x - 3)^2 + (y + 2)^2 \]

Skraćujemo \((x - 3)^2\) sa obe strane:

\[ y^2 = y^2 + 4y + 4 \]
\[ 0 = 4y + 4 \quad \Rightarrow \quad \boxed{y = -1} \]
✓ Geometrijsko tumačenje: Tačke \(3\) i \(3 - 2i\) odgovaraju koordinatama \((3{,}0)\) i \((3{,}{-2})\). Obe imaju istu \(x\)-koordinatu, pa je njihova simetrala horizontalna prava \(y = -1\) (sredina ordinata).
3 Obradimo drugi uslov: \(|z - 2i| = |z + 4 - 2i|\)

Uvrstimo \(z = x + yi\):

\[ |z - 2i| = |x + (y - 2)i| \] \[ |z + 4 - 2i| = |z - (-4 + 2i)| = |(x + 4) + (y - 2)i| \]

Kvadriramo obe strane:

\[ x^2 + (y - 2)^2 = (x + 4)^2 + (y - 2)^2 \]

Skraćujemo \((y - 2)^2\) sa obe strane:

\[ x^2 = x^2 + 8x + 16 \]
\[ 0 = 8x + 16 \quad \Rightarrow \quad \boxed{x = -2} \]
✓ Geometrijsko tumačenje: Tačke \(2i\) i \(-4 + 2i\) odgovaraju koordinatama \((0{,}2)\) i \((-4{,}2)\). Obe imaju istu \(y\)-koordinatu, pa je njihova simetrala vertikalna prava \(x = -2\) (sredina apscisa).
4 Odredimo \(z\) i izračunajmo \(|z|\)

Iz koraka 2 i 3 imamo:

\[ x = -2, \quad y = -1 \]

Dakle:

\[ z = -2 - i \]

Sada računamo modul:

\[ |z| = \sqrt{x^2 + y^2} = \sqrt{(-2)^2 + (-1)^2} = \sqrt{4 + 1} = \sqrt{5} \]
💡 Ključni uvid

Svaki uslov oblika \(|z - a| = |z - b|\) definiše pravu — simetralu duži koja spaja tačke \(a\) i \(b\). Dva takva uslova daju dve prave čiji presek jednoznačno određuje kompleksan broj \(z\). Umesto čistog računa, ovaj geometrijski uvid pomaže da brzo prepoznate šta tražite: presek dve simetrale.

Konačan odgovor
E) \(|z| = \sqrt{5}\)

Kompleksan broj je \(z = -2 - i\), pa je \(|z| = \sqrt{(-2)^2 + (-1)^2} = \sqrt{5}\).

⚠ Česte greške
  • Pogrešno tumačenje \(z - 3 + 2i\): Ovaj izraz znači \(z - (3 - 2i)\), odnosno rastojanje od tačke \((3{,}{-2})\). Česta greška je protumačiti ga kao \((z - 3) + 2i\), ali to daje isti rezultat jer \(|z - 3 + 2i| = |(z-3) + 2i|\) i \(z - (3-2i) = (z-3) + 2i\).
  • Greška u znaku: Kod drugog uslova, \(z + 4 - 2i = z - (-4 + 2i)\), pa je druga tačka \((-4{,}2)\), ne \((4{,}{-2})\).
  • Računska greška u modulu: \(|z| = \sqrt{4 + 1} = \sqrt{5}\), ne \(\sqrt{4 - 1} = \sqrt{3}\) niti \(\sqrt{4 \cdot 1} = 2\).
