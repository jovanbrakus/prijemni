Deljivost polinoma i odredjivanje koeficijenata

Nalazenje vrednosti \(a - b\) iz uslova deljivosti dva polinoma

📜 Zadatak

Ako je polinom \(P(x) = x^3 + 3x^2 + 2bx + a\) deljiv polinomom \(Q(x) = x^2 + x + a\), tada je vrednost izraza \(a - b\) jednaka:

(A)
\(-3\)
(B)
\(3\)
(C)
\(5\)
(D)
\(-1\)
(E)
\(1\)
📌 Plan resavanja
  1. Postaviti jednakost \(P(x) = Q(x) \cdot R(x)\), gde je \(R(x)\) linearan polinom
  2. Razmnoziti desnu stranu i izjednaciti koeficijente uz odgovarajuce stepene od \(x\)
  3. Resiti dobijeni sistem jednacina za \(a\), \(b\) i koeficijent linearnog polinoma
  4. Izracunati vrednost \(a - b\)
Podsetnik iz teorije

Deljivost polinoma:

Kazemo da je polinom \(P(x)\) deljiv polinomom \(Q(x)\) ako postoji polinom \(R(x)\) takav da:

\[P(x) = Q(x) \cdot R(x)\]

Ovo znaci da pri deljenju \(P(x)\) sa \(Q(x)\) nema ostatka.

Stepen rezultata:

\[\deg(P) = \deg(Q) + \deg(R)\]

Ako je \(\deg(P) = 3\) i \(\deg(Q) = 2\), tada je \(\deg(R) = 1\), tj. \(R(x)\) je linearan polinom oblika \(x + c\).

Metod neodredjenih koeficijenata:

Kada razmnozimo \(Q(x) \cdot R(x)\) i izjednacimo sa \(P(x)\), dobijamo sistem jednacina uporedjivanjem koeficijenata uz iste stepene od \(x\).

✏️ Resenje korak po korak
Korak 1: Postavljamo jednakost polinoma

Ako je \(P(x)\) deljiv sa \(Q(x)\), postoji polinom \(R(x)\) takav da:

\[P(x) = Q(x) \cdot R(x)\]

Posto je \(\deg(P) = 3\) i \(\deg(Q) = 2\), sledi \(\deg(R) = 1\), pa:

\[R(x) = x + c\]

Vodeci koeficijent mora biti 1, jer je vodeci koeficijent od \(P(x)\) jednak 1, a vodeci koeficijent od \(Q(x)\) je takodje 1.

Korak 2: Razmnozavamo \(Q(x) \cdot R(x)\)

Razmnozimo \((x^2 + x + a)(x + c)\):

\[(x^2 + x + a)(x + c) = x^3 + cx^2 + x^2 + cx + ax + ac\]

Sredimo po stepenima od \(x\):

\[= x^3 + (c + 1)x^2 + (c + a)x + ac\]
Korak 3: Izjednacavamo koeficijente

Uporedjujemo koeficijente sa \(P(x) = x^3 + 3x^2 + 2bx + a\):

\[\begin{aligned} x^3&: \quad 1 = 1 \quad \checkmark \\[4pt] x^2&: \quad c + 1 = 3 \\[4pt] x^1&: \quad c + a = 2b \\[4pt] x^0&: \quad ac = a \end{aligned}\]
Kontrolna tacka

Dobili smo sistem od tri jednacine sa tri nepoznate (\(a\), \(b\), \(c\)). Sistem je resiv jer jedna jednacina daje \(c\) direktno.

Korak 4: Resavamo sistem jednacina

Iz jednacine za \(x^2\):

\[c + 1 = 3 \quad \Rightarrow \quad c = 2\]

Iz jednacine za slobodni clan:

\[ac = a \quad \Rightarrow \quad a \cdot 2 = a \quad \Rightarrow \quad 2a = a \quad \Rightarrow \quad a = 0\]

Iz jednacine za \(x\):

\[c + a = 2b \quad \Rightarrow \quad 2 + 0 = 2b \quad \Rightarrow \quad b = 1\]
Korak 5: Racunamo trazenu vrednost
\[a - b = 0 - 1 = -1\]
Proveri se: Verifikacija deljivosti

Sa \(a = 0\) i \(b = 1\) imamo:

\[P(x) = x^3 + 3x^2 + 2x = x(x^2 + 3x + 2) = x(x + 1)(x + 2)\] \[Q(x) = x^2 + x = x(x + 1)\]

Delimo:

\[\frac{P(x)}{Q(x)} = \frac{x(x + 1)(x + 2)}{x(x + 1)} = x + 2\]

Dobili smo polinom bez ostatka, sto potvrdjuje da \(R(x) = x + 2\), tj. \(c = 2\). \(\checkmark\)

Proveri se: Verifikacija razmnozavanjem

Proverimo da li vazi \(Q(x) \cdot R(x) = P(x)\):

\[(x^2 + x)(x + 2) = x^3 + 2x^2 + x^2 + 2x = x^3 + 3x^2 + 2x\]

A \(P(x) = x^3 + 3x^2 + 2 \cdot 1 \cdot x + 0 = x^3 + 3x^2 + 2x\).

Jednakost vazi! \(\checkmark\)

💡 Kljucni uvid

Jednacina slobodnog clana \(ac = a\) se moze zapisati kao \(a(c - 1) = 0\). Posto smo iz koeficijenta uz \(x^2\) vec odredili da je \(c = 2 \neq 1\), sledi da mora biti \(a = 0\). Ovo je kljucni momenat resenja — slobodni clan polaznog polinoma je nula, sto znacajno pojednostavljuje ceo problem.

\[ac = a \;\;\wedge\;\; c = 2 \quad \Longrightarrow \quad a = 0\]
Konacan odgovor
(D) \(a - b = -1\)
(A) \(-3\)
(B) \(3\)
(C) \(5\)
(D) \(-1\)
(E) \(1\)

Verifikacija: \(a = 0,\; b = 1,\; c = 2\). Polinom \(P(x) = x^3 + 3x^2 + 2x = x(x+1)(x+2)\) je deljiv sa \(Q(x) = x^2 + x = x(x+1)\), kolicnik je \(x + 2\). Vrednost \(a - b = 0 - 1 = -1\).

⚠️ Ceste greske
Pogresno resavanje jednacine \(ac = a\): Mnogi ucenici dele obe strane sa \(a\), dobijaju \(c = 1\), i zanemaruju mogucnost da je \(a = 0\). Deljenje nulom nije dozvoljeno! Ispravno je prebaciti sve na jednu stranu: \(a(c - 1) = 0\), pa razmotriti oba slucaja.
Zaboravljanje da vodeci koeficijent \(R(x)\) mora biti 1: Posto je vodeci koeficijent i kod \(P(x)\) i kod \(Q(x)\) jednak 1, kolicnik \(R(x)\) mora biti oblika \(x + c\), a ne \(kx + c\) sa proizvoljnim \(k\).
Mesanje koeficijenata pri izjednacavanju: Pazljivo raspisati proizvod \((x^2 + x + a)(x + c)\) i grupisati clanove po stepenima. Cesta greska je propustiti jedan od clanova pri mnozenju.
🏆 Dodatni izazov

Polinom \(P(x) = x^4 + 2x^3 + kx^2 + mx + 6\) je deljiv polinomom \(Q(x) = x^2 + x + 2\). Odredite vrednost \(k + m\).

Savet

Koristite isti pristup: \(R(x) = x^2 + cx + d\). Razmnozite \(Q(x) \cdot R(x)\) i izjednacite koeficijente. Iz koeficijenta uz \(x^3\): \(c + 1 = 2 \Rightarrow c = 1\). Iz slobodnog clana: \(2d = 6 \Rightarrow d = 3\). Iz koeficijenta uz \(x^2\): \(k = d + c + 2 = 3 + 1 + 2 = 6\). Iz koeficijenta uz \(x\): \(m = 2c + d = 2 + 3 = 5\). Dakle \(k + m = 11\).
