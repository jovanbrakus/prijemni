Trigonometrijska jednačina — zbir rešenja

Faktorizacija trigonometrijske jednačine i određivanje rešenja na datom intervalu

Postavka zadatka

Zbir svih rešenja jednačine \( \sin x + \sin 2x + 1 = \cos x + 2\cos^2 x \) koja pripadaju intervalu \( (-\pi, \pi) \) jednak je:

Jednačina
\( \sin x + \sin 2x + 1 = \cos x + 2\cos^2 x \)
Interval
\( (-\pi, \pi) \)

Plan rešavanja

Prebacićemo sve na jednu stranu, iskoristiti trigonometrijske identitete \( \sin 2x = 2\sin x \cos x \) i \( \sin^2 x + \cos^2 x = 1 \) da transformišemo izraz, a zatim ga faktorizovati. Rešićemo svaki faktor posebno i naći sva rešenja na intervalu \( (-\pi, \pi) \).

Ključne formule i teoreme
  • Dupli ugao za sinus: \( \sin 2x = 2\sin x \cos x \)
  • Pitagorin identitet: \( \sin^2 x + \cos^2 x = 1 \)
  • Pomoćni ugao: \( \sin x + \cos x = \sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right) \)
  • Kvadrat zbira: \( (\sin x + \cos x)^2 = 1 + 2\sin x \cos x \), odakle \( 2\sin x \cos x = (\sin x + \cos x)^2 - 1 \)

Rešenje korak po korak

1

Prebacivanje na jednu stranu

Polazna jednačina glasi:

\[ \sin x + \sin 2x + 1 = \cos x + 2\cos^2 x \]

Prebacimo sve na levu stranu:

\[ \sin x + \sin 2x + 1 - \cos x - 2\cos^2 x = 0 \]
2

Primena identiteta

Zamenimo \( \sin 2x = 2\sin x \cos x \) i \( 1 = \sin^2 x + \cos^2 x \):

\[ \sin x + 2\sin x \cos x + \sin^2 x + \cos^2 x - \cos x - 2\cos^2 x = 0 \]

Sredimo \( \cos^2 x - 2\cos^2 x = -\cos^2 x \):

\[ \sin^2 x - \cos^2 x + \sin x - \cos x + 2\sin x \cos x = 0 \]
3

Grupisanje i faktorizacija — prvi deo

Primetimo da važi razlika kvadrata: \( \sin^2 x - \cos^2 x = (\sin x - \cos x)(\sin x + \cos x) \). Grupišemo:

\[ \underbrace{(\sin x - \cos x)(\sin x + \cos x)}_{\sin^2 x - \cos^2 x} + \underbrace{(\sin x - \cos x)}_{\sin x - \cos x} + 2\sin x\cos x = 0 \]

Iz prva dva sabirka izdvajamo zajednički faktor \( (\sin x - \cos x) \):

\[ (\sin x - \cos x)(\sin x + \cos x + 1) + 2\sin x\cos x = 0 \]
4

Faktorizacija — drugi deo

Sada za treći sabirak koristimo identitet:

\[ 2\sin x\cos x = (\sin x + \cos x)^2 - 1 \]

A razliku kvadrata rastavimo kao:

\[ (\sin x + \cos x)^2 - 1 = (\sin x + \cos x - 1)(\sin x + \cos x + 1) \]

Zamenom u izraz dobijamo:

\[ (\sin x - \cos x)(\sin x + \cos x + 1) + (\sin x + \cos x - 1)(\sin x + \cos x + 1) = 0 \]
Proveri se: Koji je zajednički faktor oba sabirka?

Zajednički faktor je \( (\sin x + \cos x + 1) \). Oba sabirka ga sadrže.

5

Izvlačenje zajedničkog faktora

Izdvajamo \( (\sin x + \cos x + 1) \) ispred zagrade:

\[ (\sin x + \cos x + 1)\Big[(\sin x - \cos x) + (\sin x + \cos x - 1)\Big] = 0 \]

Pojednostavimo izraz u uglastoj zagradi:

\[ (\sin x - \cos x) + (\sin x + \cos x - 1) = 2\sin x - 1 \]

Konačno, jednačina je faktorizovana:

\[ (\sin x + \cos x + 1)(2\sin x - 1) = 0 \]
6

Slučaj 1: \( 2\sin x - 1 = 0 \)

Rešavamo:

\[ \sin x = \frac{1}{2} \]

Na intervalu \( (-\pi, \pi) \) ova jednačina ima dva rešenja:

\[ x_1 = \frac{\pi}{6} \quad \text{i} \quad x_2 = \pi - \frac{\pi}{6} = \frac{5\pi}{6} \]
Proveri se: Koliko je \( \sin\frac{5\pi}{6} \)?

\( \sin\frac{5\pi}{6} = \sin\left(\pi - \frac{\pi}{6}\right) = \sin\frac{\pi}{6} = \frac{1}{2} \). Tačno!

7

Slučaj 2: \( \sin x + \cos x + 1 = 0 \)

Rešavamo:

\[ \sin x + \cos x = -1 \]

Koristimo formulu za pomoćni ugao: \( \sin x + \cos x = \sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right) \):

\[ \sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right) = -1 \]
\[ \sin\!\left(x + \frac{\pi}{4}\right) = -\frac{1}{\sqrt{2}} = -\frac{\sqrt{2}}{2} \]
8

Rešenja drugog slučaja

Jednačina \( \sin\alpha = -\frac{\sqrt{2}}{2} \) ima opšta rešenja:

\[ \alpha = -\frac{\pi}{4} + 2k\pi \quad \text{ili} \quad \alpha = \pi + \frac{\pi}{4} + 2k\pi \]

Zamenom \( \alpha = x + \frac{\pi}{4} \):

\[ x + \frac{\pi}{4} = -\frac{\pi}{4} + 2k\pi \;\;\Rightarrow\;\; x = -\frac{\pi}{2} + 2k\pi \]
\[ x + \frac{\pi}{4} = \frac{5\pi}{4} + 2k\pi \;\;\Rightarrow\;\; x = \pi + 2k\pi \]

Na intervalu \( (-\pi, \pi) \):

  • Iz prvog: za \( k = 0 \) dobijamo \( x_3 = -\frac{\pi}{2} \) — pripada intervalu.
  • Iz drugog: za \( k = 0 \) dobijamo \( x = \pi \), ali \( \pi \notin (-\pi, \pi) \) jer je interval otvoren.
  • Za \( k = -1 \) dobijamo \( x = -\pi \), ali ni \( -\pi \notin (-\pi, \pi) \).

Dakle, iz drugog slučaja dobijamo samo jedno rešenje: \( x_3 = -\frac{\pi}{2} \).

9

Provera svih rešenja

Proverimo da sva tri rešenja zadovoljavaju polaznu jednačinu \( \sin x + \sin 2x + 1 = \cos x + 2\cos^2 x \):

\( x_1 = \frac{\pi}{6} \):

\[ L = \frac{1}{2} + \frac{\sqrt{3}}{2} + 1 = \frac{3 + \sqrt{3}}{2}, \quad D = \frac{\sqrt{3}}{2} + \frac{3}{2} = \frac{3 + \sqrt{3}}{2} \;\;\checkmark \]

\( x_2 = \frac{5\pi}{6} \):

\[ L = \frac{1}{2} - \frac{\sqrt{3}}{2} + 1 = \frac{3 - \sqrt{3}}{2}, \quad D = -\frac{\sqrt{3}}{2} + \frac{3}{2} = \frac{3 - \sqrt{3}}{2} \;\;\checkmark \]

\( x_3 = -\frac{\pi}{2} \):

\[ L = -1 + 0 + 1 = 0, \quad D = 0 + 0 = 0 \;\;\checkmark \]
Proveri se: Koliko rešenja ukupno imamo?

Ukupno 3 rešenja: \( x_1 = \frac{\pi}{6} \), \( x_2 = \frac{5\pi}{6} \), \( x_3 = -\frac{\pi}{2} \).

10

Zbir svih rešenja

Sabiramo sva tri rešenja:

\[ S = \frac{\pi}{6} + \frac{5\pi}{6} + \left(-\frac{\pi}{2}\right) \]

Zbir prva dva:

\[ \frac{\pi}{6} + \frac{5\pi}{6} = \frac{6\pi}{6} = \pi \]

Konačno:

\[ S = \pi - \frac{\pi}{2} = \frac{\pi}{2} \]

Ključni uvid

Ključ rešavanja je transformacija izraza \( 2\sin x\cos x \) pomoću identiteta \( 2\sin x\cos x = (\sin x + \cos x)^2 - 1 \), što omogućava faktorizaciju cele jednačine na proizvod \( (\sin x + \cos x + 1)(2\sin x - 1) = 0 \).

\[ x_1 + x_2 + x_3 = \frac{\pi}{6} + \frac{5\pi}{6} - \frac{\pi}{2} = \frac{\pi}{2} \]
Konačan odgovor
\( \displaystyle\frac{\pi}{2} \)

Verifikacija: Sva tri rešenja su proverena direktnom zamenom u polaznu jednačinu. Zbir \( \frac{\pi}{6} + \frac{5\pi}{6} = \pi \), a zatim \( \pi - \frac{\pi}{2} = \frac{\pi}{2} \). Rezultat je pozitivan i manji od \( \pi \), što je u skladu sa raspodelom rešenja na intervalu. Tačan odgovor je B).

Tipične greške

  • Zaboravljanje da je interval otvoren: Vrednost \( x = \pi \) (i \( x = -\pi \)) je rešenje jednačine \( \sin x + \cos x = -1 \), ali ne pripada otvorenom intervalu \( (-\pi, \pi) \). Da je interval bio zatvoren, zbir bi bio drugačiji.
  • Propuštanje rešenja iz jednog faktora: Potrebno je rešiti oba faktora: \( 2\sin x - 1 = 0 \) daje dva rešenja, a \( \sin x + \cos x + 1 = 0 \) daje još jedno. Ako se zaboravi bilo koji faktor, zbir neće biti tačan.

Dodatni izazov

Pronađi zbir svih rešenja jednačine \( \sin x + \sin 2x + \sin 3x = \cos x + \cos 2x + \cos 3x \) na intervalu \( [0, 2\pi) \).

Hint

Koristi formule za zbir sinusa i kosinusa: \( \sin x + \sin 3x = 2\sin 2x \cos x \) i analogno za kosinuse. Svedeš na proizvod faktora koji daju poznate jednačine.
