Trigonometrijska jednačina - Zbir rešenja

Trigonometrija / Rešavanje jednačina svodjenjem na kvadratnu

Postavka zadatka

Zbir svih rešenja jednačine

\[ 3(1 - \sin x) + \sin^4 x = 1 + \cos^4 x \]

koja pripadaju intervalu \( (-\pi,\; \pi) \) jednak je:

Plan rešavanja

Koristićemo identitet \(\cos^2 x = 1 - \sin^2 x\) da sve izrazimo preko \(\sin x\), a zatim ćemo svesti jednačinu na kvadratnu po \(t = \sin x\). Na kraju, sabiraćemo sva rešenja iz datog intervala.

Ključne formule i teoreme
  • Pitagorin identitet: \(\sin^2 x + \cos^2 x = 1\), odakle \(\cos^2 x = 1 - \sin^2 x\)
  • Kvadrat razlike: \((1 - \sin^2 x)^2 = 1 - 2\sin^2 x + \sin^4 x\), koristi se za razvoj \(\cos^4 x\)
  • Kvadratna jednačina: \(at^2 + bt + c = 0 \Rightarrow t = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)
  • Rešenja \(\sin x = \tfrac{1}{2}\): na intervalu \((-\pi, \pi)\) to su \(x = \dfrac{\pi}{6}\) i \(x = \dfrac{5\pi}{6}\)
  • Rešenje \(\sin x = 1\): na intervalu \((-\pi, \pi)\) to je \(x = \dfrac{\pi}{2}\)

Rešenje korak po korak

1

Raspisivanje leve strane

Prvo raspisujemo levu stranu jednačine:

\[ 3(1 - \sin x) + \sin^4 x = 3 - 3\sin x + \sin^4 x \]

Jednačina sada glasi:

\[ 3 - 3\sin x + \sin^4 x = 1 + \cos^4 x \]
2

Razvoj \(\cos^4 x\) preko \(\sin x\)

Koristimo identitet \(\cos^2 x = 1 - \sin^2 x\) da izrazimo \(\cos^4 x\):

\[ \cos^4 x = (\cos^2 x)^2 = (1 - \sin^2 x)^2 \]

Primenjujemo formulu za kvadrat razlike:

\[ \cos^4 x = 1 - 2\sin^2 x + \sin^4 x \]
3

Zamena u jednačini

Zamenjujemo izraz za \(\cos^4 x\) na desnoj strani:

\[ 3 - 3\sin x + \sin^4 x = 1 + 1 - 2\sin^2 x + \sin^4 x \]

Desna strana se pojednostavljuje:

\[ 3 - 3\sin x + \sin^4 x = 2 - 2\sin^2 x + \sin^4 x \]
Proveri sebe: Šta se dešava sa \(\sin^4 x\)?
\(\sin^4 x\) se pojavljuje na obe strane jednačine, pa se skraćuje!
4

Skraćivanje \(\sin^4 x\)

Član \(\sin^4 x\) se pojavljuje na obe strane, pa se skraćuje:

\[ 3 - 3\sin x = 2 - 2\sin^2 x \]

Prebacujemo sve na jednu stranu:

\[ 2\sin^2 x - 3\sin x + 3 - 2 = 0 \]
\[ 2\sin^2 x - 3\sin x + 1 = 0 \]
5

Smena \(t = \sin x\)

Uvodimo smenu \(t = \sin x\) i dobijamo kvadratnu jednačinu:

\[ 2t^2 - 3t + 1 = 0 \]

Rešavamo pomoću formule ili faktorizacije. Diskriminanta:

\[ D = (-3)^2 - 4 \cdot 2 \cdot 1 = 9 - 8 = 1 \]

Rešenja:

\[ t = \frac{3 \pm 1}{4} \implies t_1 = 1, \quad t_2 = \frac{1}{2} \]

Alternativno, faktorizacijom: \(2t^2 - 3t + 1 = (2t - 1)(t - 1) = 0\)

Proveri sebe: Da li su oba rešenja prihvatljiva za sinus?
Da! Oba rešenja \(t_1 = 1\) i \(t_2 = \frac{1}{2}\) pripadaju intervalu \([-1, 1]\), pa su oba prihvatljiva.
6

Rešenja za \(\sin x = 1\)

Jednačina \(\sin x = 1\) na intervalu \((-\pi,\;\pi)\) ima jedno rešenje:

\[ x_1 = \frac{\pi}{2} \]
7

Rešenja za \(\sin x = \frac{1}{2}\)

Jednačina \(\sin x = \frac{1}{2}\) na intervalu \((-\pi,\;\pi)\) ima dva rešenja:

\[ x_2 = \frac{\pi}{6}, \qquad x_3 = \pi - \frac{\pi}{6} = \frac{5\pi}{6} \]

Oba ugla su u prvom i drugom kvadrantu, gde je sinus pozitivan.

8

Zbir svih rešenja

Sabiramo sva tri rešenja:

\[ S = \frac{\pi}{6} + \frac{5\pi}{6} + \frac{\pi}{2} \]

Svodimo na zajednički imenilac (6):

\[ S = \frac{\pi}{6} + \frac{5\pi}{6} + \frac{3\pi}{6} = \frac{9\pi}{6} = \frac{3\pi}{2} \]
Proveri sebe: Provera za \(x = \frac{\pi}{6}\)
Za \(x = \frac{\pi}{6}\): \(\sin\frac{\pi}{6} = \frac{1}{2}\), \(\cos\frac{\pi}{6} = \frac{\sqrt{3}}{2}\).
LHS = \(3 \cdot \frac{1}{2} + \frac{1}{16} = \frac{24}{16} + \frac{1}{16} = \frac{25}{16}\).
RHS = \(1 + \frac{9}{16} = \frac{25}{16}\). Jednako!

Ključni uvid

Korišćenjem identiteta \(\cos^2 x = 1 - \sin^2 x\), članovi \(\sin^4 x\) se skraćuju i jednačina se svodi na prostu kvadratnu jednačinu po \(\sin x\).

\[ 2\sin^2 x - 3\sin x + 1 = 0 \implies \sin x = 1 \;\text{ ili }\; \sin x = \tfrac{1}{2} \]

Verifikacija: Proverili smo sva tri rešenja direktnom zamenom u originalnu jednačinu:
  • \(x = \frac{\pi}{6}\): LHS = RHS = \(\frac{25}{16}\)
  • \(x = \frac{5\pi}{6}\): LHS = RHS = \(\frac{25}{16}\)
  • \(x = \frac{\pi}{2}\): LHS = RHS = \(1\)

Zbir: \(\frac{\pi}{6} + \frac{5\pi}{6} + \frac{\pi}{2} = \frac{9\pi}{6} = \frac{3\pi}{2}\). Rezultat se poklapa sa opcijom (C).

Česte greške

  • Zaboravljanje da se \(\cos^4 x\) može izraziti preko \(\sin x\) koristeći Pitagorin identitet. Bez ovog koraka, jednačina izgleda mnogo komplikovanije nego što zaista jeste.
  • Propuštanje jednog od rešenja za \(\sin x = \frac{1}{2}\). Na intervalu \((-\pi, \pi)\) postoje dva rešenja: \(\frac{\pi}{6}\) i \(\frac{5\pi}{6}\), a ne samo jedno.
  • Dodavanje rešenja iz \((-\pi, 0)\) za \(\sin x = \frac{1}{2}\). Pošto je \(\frac{1}{2} > 0\), rešenja su samo u prvom i drugom kvadrantu, tj. oba su pozitivna.

Dodatni izazov

Pronađi zbir svih rešenja jednačine \(3(1 - \cos x) + \cos^4 x = 1 + \sin^4 x\) na intervalu \((-\pi, \pi)\).

Prikaži nagoveštaj

Postupak je analogan: zameni \(\sin^4 x = (1 - \cos^2 x)^2\), svedi na kvadratnu jednačinu po \(\cos x\), i pronađi rešenja na datom intervalu. Obrati pažnju na to koji uglovi zadovoljavaju \(\cos x = 1\) i \(\cos x = \frac{1}{2}\).
