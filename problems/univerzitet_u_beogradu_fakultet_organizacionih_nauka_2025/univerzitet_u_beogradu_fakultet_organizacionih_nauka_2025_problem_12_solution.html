Ostatak pri deljenju polinoma

Algebra — deljenje polinoma, koreni polinoma, kompleksni brojevi

Postavka zadatka

Ostatak koji se dobija deljenjem polinoma

\[ P(x) = x^{2025} + x^{2024} - x^{2023} + 1 \]

sa polinomom

\[ Q(x) = x^3 + x^2 + x + 1 \]

je:

(A)
\(x^2 + 2x + 3\)
(B)
\(2x + 1\)
(C)
\(3x^2 + 2x + 1\)
(D)
\(4x^2 + 2x - 1\)
(E)
\(x^2 + 3x + 2\)

Plan rešavanja

Faktorizovaćemo delilac \(Q(x)\) i naći njegove korene (uključujući kompleksne). Zatim ćemo izračunati vrednost \(P(x)\) u svakom korenu, čime dobijamo sistem jednačina za koeficijente ostatka \(R(x) = ax^2 + bx + c\).

Ključne formule i teoreme
  • Teorema o ostatku pri deljenju polinoma: Ako je \(P(x) = Q(x) \cdot S(x) + R(x)\), gde je \(\deg R < \deg Q\), onda za svaki koren \(\alpha\) polinoma \(Q(x)\) važi \(P(\alpha) = R(\alpha)\).
  • Faktorizacija: \(x^3 + x^2 + x + 1 = x^2(x+1) + (x+1) = (x+1)(x^2+1)\). Koreni su \(x = -1\), \(x = i\), \(x = -i\).
  • Stepeni imaginarne jedinice: \(i^0 = 1\), \(i^1 = i\), \(i^2 = -1\), \(i^3 = -i\), i ciklus se ponavlja sa periodom 4. Za proizvoljno \(n\): \(i^n = i^{n \bmod 4}\).
  • Stepeni broja \(-i\): \((-i)^n = (-1)^n \cdot i^n\). Ovo kombinuje ciklus stepena \(i\) sa naizmeničnom promenom znaka.

Vizuelni prikaz

Koreni polinoma \(Q(x)\) u kompleksnoj ravni i vrednosti \(P(x)\) u njima.

Koreni polinoma Q(x) i odgovarajuće vrednosti P(x) u kompleksnoj ravni

Rešenje korak po korak

1

Faktorizujmo delilac \(Q(x)\)

Grupisanjem članova faktorizujemo polinom \(Q(x)\):

\[ Q(x) = x^3 + x^2 + x + 1 = x^2(x + 1) + 1 \cdot (x + 1) = (x + 1)(x^2 + 1) \]

Koristili smo grupisanje: izdvojili \(x^2\) iz prva dva člana i \(1\) iz poslednja dva.

2

Odredimo korene polinoma \(Q(x)\)

Iz faktorizacije \(Q(x) = (x+1)(x^2+1)\) nalazimo korene:

\[ x + 1 = 0 \implies x_1 = -1 \] \[ x^2 + 1 = 0 \implies x_{2,3} = \pm i \]

Dakle, \(Q(x)\) ima tri korena: \(-1\), \(i\) i \(-i\).

3

Postavimo oblik ostatka

Pošto je \(\deg Q = 3\), ostatak je polinom stepena najviše 2:

\[ R(x) = ax^2 + bx + c \]

Važi \(P(x) = Q(x) \cdot S(x) + R(x)\), pa za svaki koren \(\alpha\) polinoma \(Q(x)\):

\[ P(\alpha) = R(\alpha) \]

Ovo nam daje tri jednačine (za \(\alpha = -1, i, -i\)) sa tri nepoznate (\(a, b, c\)).

4

Pripremimo stepene: \(2025 \bmod 4\)

Za izračunavanje stepena \(i\) i \(-i\), koristimo cikličnost sa periodom 4. Odredimo ostatke pri deljenju eksponenata sa 4:

\[ 2025 = 4 \cdot 506 + 1 \implies 2025 \bmod 4 = 1 \] \[ 2024 = 4 \cdot 506 \implies 2024 \bmod 4 = 0 \] \[ 2023 = 4 \cdot 505 + 3 \implies 2023 \bmod 4 = 3 \]

Prema tome:

\[ i^{2025} = i^1 = i, \quad i^{2024} = i^0 = 1, \quad i^{2023} = i^3 = -i \]
Proveri se: Koliko je \((-1)^{2025} + (-1)^{2024} - (-1)^{2023}\)?
\((-1)^{2025} + (-1)^{2024} - (-1)^{2023} = -1 + 1 - (-1) = -1 + 1 + 1 = 1\)
5

Izračunajmo \(P(-1)\)

Zamenjujemo \(x = -1\) u \(P(x) = x^{2025} + x^{2024} - x^{2023} + 1\):

\[ P(-1) = (-1)^{2025} + (-1)^{2024} - (-1)^{2023} + 1 \]

Pošto je 2025 neparno: \((-1)^{2025} = -1\). Pošto je 2024 parno: \((-1)^{2024} = 1\). Pošto je 2023 neparno: \((-1)^{2023} = -1\).

\[ P(-1) = -1 + 1 - (-1) + 1 = -1 + 1 + 1 + 1 = \color{#4ade80}{2} \]
6

Izračunajmo \(P(i)\)

Zamenjujemo \(x = i\) i koristimo stepene izračunate u koraku 4:

\[ P(i) = i^{2025} + i^{2024} - i^{2023} + 1 \] \[ = i + 1 - (-i) + 1 \] \[ = i + 1 + i + 1 = \color{#4ade80}{2 + 2i} \]
7

Izračunajmo \(P(-i)\)

Za stepene broja \(-i\) koristimo \((-i)^n = (-1)^n \cdot i^n\):

\[ (-i)^{2025} = (-1)^{2025} \cdot i^{2025} = (-1) \cdot i = -i \] \[ (-i)^{2024} = (-1)^{2024} \cdot i^{2024} = 1 \cdot 1 = 1 \] \[ (-i)^{2023} = (-1)^{2023} \cdot i^{2023} = (-1) \cdot (-i) = i \]

Zamenjujemo:

\[ P(-i) = -i + 1 - i + 1 = \color{#4ade80}{2 - 2i} \]

Primetimo da je \(P(-i) = \overline{P(i)}\) — konjugovane vrednosti, što je očekivano jer je \(P(x)\) polinom sa realnim koeficijentima.

Proveri se: Ako je \(R(x) = ax^2 + bx + c\), koliko je \(R(i)\)?
\(R(i) = a \cdot i^2 + b \cdot i + c = -a + bi + c\). Realni deo je \(c - a\), imaginarni deo je \(b\).
8

Postavimo sistem jednačina

Iz uslova \(P(\alpha) = R(\alpha)\) za svaki koren \(\alpha\) dobijamo:

\[ R(-1) = a - b + c = 2 \quad \ldots (1) \] \[ R(i) = -a + bi + c = 2 + 2i \quad \ldots (2) \] \[ R(-i) = -a - bi + c = 2 - 2i \quad \ldots (3) \]

Iz jednačine (2), razdvajamo realni i imaginarni deo:

\[ \text{Realni deo: } c - a = 2 \quad \ldots (2a) \] \[ \text{Imaginarni deo: } b = 2 \quad \ldots (2b) \]

Jednačina (3) daje iste informacije (jer su \(i\) i \(-i\) konjugovani), pa je konzistentna.

9

Rešimo sistem

Iz jednačine (2b) odmah znamo: \(b = 2\).

Zamenjujemo \(b = 2\) u jednačinu (1):

\[ a - 2 + c = 2 \implies a + c = 4 \quad \ldots (1a) \]

Sada imamo sistem dve jednačine sa dve nepoznate:

\[ \begin{cases} c - a = 2 & \ldots (2a) \\ a + c = 4 & \ldots (1a) \end{cases} \]

Sabiranjem: \(2c = 6\), pa \(c = 3\).

Oduzimanjem: \(2a = 2\), pa \(a = 1\).

10

Zapišimo ostatak

Sa koeficijentima \(a = 1\), \(b = 2\), \(c = 3\) dobijamo:

\[ R(x) = x^2 + 2x + 3 \]
Proveri se: Da li \(R(-1) = 2\)?
\(R(-1) = (-1)^2 + 2 \cdot (-1) + 3 = 1 - 2 + 3 = 2\). Da, tačno!

Ključno zapažanje

Umesto dugog direktnog deljenja polinoma stepena 2025, korišćenjem korenova delioca (uključujući kompleksne!) sveli smo problem na rešavanje prostog sistema od tri linearne jednačine. Suština je u tome da za svaki koren \(\alpha\) polinoma \(Q(x)\) važi \(P(\alpha) = R(\alpha)\).

\[ R(x) = x^2 + 2x + 3 \]

Konačan odgovor

\( R(x) = x^2 + 2x + 3 \)
(A) \(x^2 + 2x + 3\)
(B) \(2x + 1\)
(C) \(3x^2 + 2x + 1\)
(D) \(4x^2 + 2x - 1\)
(E) \(x^2 + 3x + 2\)

Verifikacija: Proveravamo ostatak u sva tri korena: \(R(-1) = 1 - 2 + 3 = 2 = P(-1)\) ✓, \(R(i) = -1 + 2i + 3 = 2 + 2i = P(i)\) ✓, \(R(-i) = -1 - 2i + 3 = 2 - 2i = P(-i)\) ✓. Sve tri vrednosti se poklapaju, pa je ostatak zaista \(x^2 + 2x + 3\), što odgovara opciji (A).

Česte greške

  • Greška 1: Pogrešno izračunavanje stepena \(i\). Na primer, ako se pogreši da je \(i^{2023} = i\) umesto \(-i\), dobija se \(P(i) = i + 1 - i + 1 = 2\) (realan broj), što vodi ka pogrešnom sistemu jednačina i pogrešnom odgovoru.
  • Greška 2: Pokušaj direktnog deljenja polinoma stepena 2025 sa polinomom stepena 3. Ovaj pristup je tehnički moguć, ali zahteva ogromno mnogo koraka. Metod korenova je daleko efikasniji.

Dodatni izazov

Nađite ostatak pri deljenju polinoma \(x^{100}\) sa \(x^2 - 1\).

Nagoveštaj

Polinom \(x^2 - 1 = (x-1)(x+1)\) ima korene \(x = 1\) i \(x = -1\). Ostatak je oblika \(R(x) = ax + b\). Izračunajte \(P(1)\) i \(P(-1)\) i postavite sistem dve jednačine sa dve nepoznate.
