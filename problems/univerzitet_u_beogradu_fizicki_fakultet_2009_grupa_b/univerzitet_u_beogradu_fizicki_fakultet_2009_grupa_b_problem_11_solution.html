Iracionalna jednačina sa korenom

Algebra — iracionalne jednačine, kvadriranje, провера решења

📋 Текст задатка

Rešenje iracionalne jednačine \(\sqrt{x + 9} = \sqrt{x} + 1\) je:

Jednačina
\(\sqrt{x + 9} = \sqrt{x} + 1\)
Traži se
\(x = \, ?\)
(а)
\(x = 9\)
(б)
\(x = 16\)
(в)
\(x = 25\)

🗺️ План решавања

Квадрираћемо обе стране једначине да уклонимо коренове, средићемо израз до облика \(\sqrt{x} = k\), поново квадрирамо и добијамо \(x\). На крају, обавезно проверавамо решење уврштавањем у полазну једначину.

📐 Подсетник из теорије

Кључни концепти за решавање иррационалних једначина:

\[\sqrt{a} = b \implies a = b^2 \quad (b \geq 0)\]

Квадрирање обе стране једначине може увести лажна решења (она која не задовољавају полазну једначину). Зато је провера обавезна!

\[(a + b)^2 = a^2 + 2ab + b^2\]

Формула за квадрат збира — користимо је при квадрирању десне стране \((\sqrt{x} + 1)^2\).

\[\text{Домен: } \sqrt{f(x)} \text{ постоји} \iff f(x) \geq 0\]

Пре решавања увек одредимо домен — вредности \(x\) за које су сви коренови дефинисани.

📊 Графички приказ

Графици функција \(y = \sqrt{x + 9}\) (лева страна) и \(y = \sqrt{x} + 1\) (десна страна). Пресечна тачка је решење једначине.

\(y = \sqrt{x + 9}\)
\(y = \sqrt{x} + 1\)
Пресечна тачка (16, 5)

Пређите мишем преко графика да видите вредности функција

📝 Решење корак по корак

1

Одређивање домена

Да би коренови били дефинисани, потребно је:

\[\sqrt{x + 9}: \quad x + 9 \geq 0 \implies x \geq -9\]
\[\sqrt{x}: \quad x \geq 0\]

Пресек услова даје домен: \(x \geq 0\).

2

Квадрирање обе стране

Полазна једначина:

\[\sqrt{x + 9} = \sqrt{x} + 1\]

Квадрирамо обе стране:

\[\left(\sqrt{x + 9}\right)^2 = \left(\sqrt{x} + 1\right)^2\]

Лева страна се поједностави у \(x + 9\). Десну страну развијамо по формули за квадрат збира \((a+b)^2 = a^2 + 2ab + b^2\):

\[x + 9 = \left(\sqrt{x}\right)^2 + 2 \cdot \sqrt{x} \cdot 1 + 1^2\]
\[x + 9 = x + 2\sqrt{x} + 1\]
🧠 Провери себе: Шта добијамо када скратимо \(x\) са обе стране?

Скраћивањем \(x\) добијамо: \(9 = 2\sqrt{x} + 1\), односно \(8 = 2\sqrt{x}\).

3

Поједностављивање

Одузмемо \(x\) и \(1\) са обе стране:

\[x + 9 - x - 1 = 2\sqrt{x}\]
\[8 = 2\sqrt{x}\]

Поделимо обе стране са 2:

\[\sqrt{x} = 4\]
4

Друго квадрирање

Квадрирамо обе стране једначине \(\sqrt{x} = 4\):

\[\left(\sqrt{x}\right)^2 = 4^2\]
\[x = 16\]

Добили смо кандидата за решење: \(x = 16\).

🧠 Провери себе: Зашто морамо да проверимо решење?

Квадрирање може увести лажна решења. На пример, ако бисмо имали \(\sqrt{x} = -3\), квадрирањем бисмо добили \(x = 9\), али \(\sqrt{9} = 3 \neq -3\). Зато увек проверавамо уврштавањем у полазну једначину!

5

Провера решења

Уврштавамо \(x = 16\) у полазну једначину \(\sqrt{x + 9} = \sqrt{x} + 1\):

\[\text{Лева страна: } \sqrt{16 + 9} = \sqrt{25} = 5\]
\[\text{Десна страна: } \sqrt{16} + 1 = 4 + 1 = 5\]

ЛС = ДС = 5 — решење је тачно!

Вредност \(x = 16\) припада домену \(x \geq 0\), тако да је ово ваљано решење.

6

Провера осталих опција

За потпуност, проверимо и остале понуђене одговоре:

\(x = 9\):

\[\sqrt{9 + 9} = \sqrt{18} \approx 4{,}24 \quad \text{и} \quad \sqrt{9} + 1 = 3 + 1 = 4\]

\(4{,}24 \neq 4\) — није решење.

\(x = 25\):

\[\sqrt{25 + 9} = \sqrt{34} \approx 5{,}83 \quad \text{и} \quad \sqrt{25} + 1 = 5 + 1 = 6\]

\(5{,}83 \neq 6\) — није решење.

🧠 Провери себе: Како можемо брзо да проверимо да \(x = 25\) није решење без калкулатора?

Ако \(x = 25\), тада би \(\sqrt{x} = 5\) и \(\sqrt{x+9} = \sqrt{34}\). Пошто \(34\) није потпуни квадрат (5² = 25, 6² = 36), \(\sqrt{34}\) је ирационалан број, а десна страна \(5 + 1 = 6\) је цео број. Ирационалан број не може бити једнак целом, тако да \(x = 25\) сигурно није решење.

💡 Кључни увид

Код иррационалних једначина, квадрирање уклања коренове, али може увести лажна решења. Зато је провера уврштавањем обавезан корак, а не само формалност!

\[\sqrt{x + 9} = \sqrt{x} + 1 \quad \xRightarrow{\text{квадрирање}} \quad 8 = 2\sqrt{x} \quad \Rightarrow \quad x = 16\]

🏆 Коначан одговор

\(x = 16\)
(а) \(x = 9\)
(б) \(x = 16\)
(в) \(x = 25\)
Верификација: \(\sqrt{16 + 9} = \sqrt{25} = 5\) и \(\sqrt{16} + 1 = 4 + 1 = 5\). Обе стране су једнаке, решење је потврђено.

⚠️ Честе грешке

  • Заборављена провера: Квадрирање може увести лажна решења. Многи ученици пресдоче овај корак и изгубе бодове.
  • Погрешно развијање квадрата збира: Честа грешка је \((\sqrt{x} + 1)^2 = x + 1\) уместо \(x + 2\sqrt{x} + 1\). Не заборавите двоструки производ \(2 \cdot \sqrt{x} \cdot 1\)!

🚀 Додатни изазов

Реши иррационалну једначину: \(\sqrt{x + 5} = \sqrt{x - 3} + 2\)

💡 Наговештај

Примени исти поступак: квадрирај обе стране, изолуј преостали корен, поново квадрирај. Обрати пажњу на домен: \(x \geq 3\). Решење је \(x = 4\) — провери!
