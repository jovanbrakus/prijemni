Rešavanje kvadratne jednačine

Algebra — kvadratne jednačine, razvoj kvadrata binoma, faktorizacija

Postavka zadatka

Rešenja jednačine \( (x + 2)^2 = 2x^2 + x + 4 \) su:

Jednačina
\( (x+2)^2 = 2x^2 + x + 4 \)
Тражимо
\( x_1, \; x_2 \)
(а)
\( x_1 = x_2 = 1 \)
(б)
\( x_1 = 0, \; x_2 = -1 \)
(в)
\( x_1 = 0, \; x_2 = 3 \)

План решавања

Развићемо леву страну једначине користећи формулу за квадрат бинома, пребацити све на једну страну, средити и факторисати добијени израз, а затим наћи решења.

Подсетник из теорије

Кључне формуле и теореме

Квадрат бинома:

\[ (a + b)^2 = a^2 + 2ab + b^2 \]

Ово је основна алгебарска идентичност коју користимо за развијање леве стране једначине.

Факторизација — изношење заједничког чиниоца:

\[ ax^2 + bx = x(ax + b) \]

Ако сваки члан израза садржи фактор \( x \), можемо га изнети испред заграде.

Производ једнак нули:

\[ A \cdot B = 0 \;\Longrightarrow\; A = 0 \;\text{ или }\; B = 0 \]

Производ два фактора је нула ако и само ако је бар један од њих нула. Ово нам даје решења факторисане једначине.

Визуелни приказ

Графици функција \( y = (x+2)^2 \) (плава парабола) и \( y = 2x^2 + x + 4 \) (розе парабола). Тачке пресека су решења једначине. Превлачите тачке за истраживање.

Решење корак по корак

1
Развијамо леву страну

Применимо формулу за квадрат бинома \( (a+b)^2 = a^2 + 2ab + b^2 \) на леву страну једначине, где је \( a = x \) и \( b = 2 \):

\[ (x + 2)^2 = x^2 + 2 \cdot x \cdot 2 + 2^2 \]

Израчунамо:

\[ (x + 2)^2 = x^2 + 4x + 4 \]
2
Записујемо једначину са развијеном левом страном

Замењујемо развијени облик у полазну једначину:

\[ x^2 + 4x + 4 = 2x^2 + x + 4 \]
Провери себе: Колико је \( (x+2)^2 \) за \( x = 1 \)?

\( (1+2)^2 = 3^2 = 9 \). А десна страна: \( 2 \cdot 1^2 + 1 + 4 = 2 + 1 + 4 = 7 \). Пошто је \( 9 \neq 7 \), вредност \( x = 1 \) није решење!

3
Пребацујемо све на леву страну

Одузимамо \( 2x^2 + x + 4 \) са обе стране:

\[ x^2 + 4x + 4 - 2x^2 - x - 4 = 0 \]

Сређујемо — групишемо сличне чланове:

\[ (x^2 - 2x^2) + (4x - x) + (4 - 4) = 0 \]
\[ -x^2 + 3x = 0 \]
4
Множимо једначину са \( -1 \)

Множење целе једначине са \( -1 \) даје нам једначину са позитивним водећим коефицијентом, што је прегледније:

\[ x^2 - 3x = 0 \]
Провери себе: Зашто смемо помножити целу једначину са \( -1 \)?

Множење обе стране једначине истим ненултим бројем не мења скуп решења. Пошто је \( -1 \neq 0 \), једначина \( x^2 - 3x = 0 \) има иста решења као \( -x^2 + 3x = 0 \).

5
Факторишемо

Оба члана садрже \( x \) као заједнички фактор. Износимо га испред заграде:

\[ x(x - 3) = 0 \]
6
Налазимо решења

Производ два фактора је нула ако је бар један од њих нула:

\[ x = 0 \quad \text{или} \quad x - 3 = 0 \]

Из другог услова:

\[ x - 3 = 0 \;\Longrightarrow\; x = 3 \]

Дакле, решења су:

\[ x_1 = 0, \quad x_2 = 3 \]
7
Верификација

Проверимо оба решења уврштавањем у полазну једначину \( (x+2)^2 = 2x^2 + x + 4 \):

За \( x_1 = 0 \):

\[ \text{Лева страна: } (0 + 2)^2 = 4 \]
\[ \text{Десна страна: } 2 \cdot 0^2 + 0 + 4 = 4 \quad \checkmark \]

За \( x_2 = 3 \):

\[ \text{Лева страна: } (3 + 2)^2 = 25 \]
\[ \text{Десна страна: } 2 \cdot 9 + 3 + 4 = 18 + 3 + 4 = 25 \quad \checkmark \]

Оба решења задовољавају једначину.

Провери себе: Зашто \( x = -1 \) није решење?

За \( x = -1 \): лева страна \( (-1+2)^2 = 1 \), десна страна \( 2 \cdot 1 + (-1) + 4 = 5 \). Пошто је \( 1 \neq 5 \), вредност \( x = -1 \) не задовољава једначину.

Кључни увид

Кључ овог задатка је препознати да се после развијања и сређивања добија непотпуна квадратна једначина (без слободног члана). Такве једначине се увек решавају изношењем заједничког фактора, а не квадратном формулом. Фактор \( x \) нам одмах даје једно решење \( x = 0 \):

\[ x^2 - 3x = 0 \;\Longrightarrow\; x(x - 3) = 0 \;\Longrightarrow\; x_1 = 0, \; x_2 = 3 \]

Коначан одговор

\( x_1 = 0, \quad x_2 = 3 \)
(а) \( x_1 = x_2 = 1 \)
(б) \( x_1 = 0, \; x_2 = -1 \)
(в) \( x_1 = 0, \; x_2 = 3 \)

Верификација: За \( x = 0 \): обе стране дају \( 4 \). За \( x = 3 \): обе стране дају \( 25 \). Такође, Вијетове формуле: збир решења \( 0 + 3 = 3 \) (коефицијент уз \( x \) у \( x^2 - 3x = 0 \)) и производ \( 0 \cdot 3 = 0 \) (слободан члан), што се подудара.

Честе грешке

✗

Дељење једначине са \( x \): Ученици често поделе обе стране једначине \( x^2 - 3x = 0 \) са \( x \) и добију \( x - 3 = 0 \), односно само \( x = 3 \). Тиме се губи решење \( x = 0 \)! Никада не делити једначину непознатом, већ факторисати.

✗

Погрешно развијање квадрата бинома: Честа грешка је \( (x+2)^2 = x^2 + 4 \), где се изостави средишњи члан \( 2 \cdot x \cdot 2 = 4x \). Увек се сетите формуле: \( (a+b)^2 = a^2 + \mathbf{2ab} + b^2 \).

Додатни изазов

Решите једначину \( (x + 3)^2 = 3x^2 + x + 9 \). Да ли и ова једначина има решење \( x = 0 \)?

Погледај наговештај

Развијте леву страну: \( x^2 + 6x + 9 = 3x^2 + x + 9 \). Пребаците на једну страну: \( -2x^2 + 5x = 0 \), односно \( x(2x - 5) = 0 \). Решења су \( x = 0 \) и \( x = \frac{5}{2} = 2{,}5 \). Да, и ова једначина има решење \( x = 0 \)!
