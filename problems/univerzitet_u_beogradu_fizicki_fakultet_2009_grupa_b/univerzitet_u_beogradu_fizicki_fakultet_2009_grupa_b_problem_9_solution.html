Рационална неједначина са количником линеарних израза

Алгебра — анализа знака рационалног израза, критичне тачке, бројевна права

Текст задатка

Решити неједначину:

\[ \frac{x + 1}{x - 3} \geq 0 \]
(а)
\(x \in [-1,\; 3)\)
(б)
\(x \in (-\infty,\; -1] \cup (3,\; +\infty)\)
(в)
\(x \in [3,\; +\infty)\)

План решавања

Наћи ћемо нуле бројиоца и именилоца (критичне тачке), затим анализирати знак израза на сваком интервалу бројевне праве. На крају ћемо формирати скуп решења водећи рачуна о томе где израз има вредност нула (укључујемо), а где није дефинисан (искључујемо).

Подсетник из теорије

Кључне формуле и правила

Знак количника: Количник два реална броја \( \frac{a}{b} \) је:

  • позитиван ако су \( a \) и \( b \) истог знака,
  • негативан ако су \( a \) и \( b \) различитог знака,
  • једнак нули ако је \( a = 0 \) и \( b \neq 0 \).

Рационална неједначина \( \frac{f(x)}{g(x)} \geq 0 \):

  • Наћи нуле бројиоца \( f(x) = 0 \) и именилоца \( g(x) = 0 \) — то су критичне тачке.
  • Нуле именилоца увек искључујемо из решења (израз није дефинисан).
  • Нуле бројиоца укључујемо ако неједначина садржи знак \( \geq \) или \( \leq \) (јер је \( 0 \geq 0 \) тачно).

Метод интервала: Критичне тачке деле бројевну праву на интервале. На сваком интервалу знак израза је константан — довољно је проверити једну тест-вредност.

Бројевна права и анализа знака

Бројевна права са критичним тачкама, знаковима бројиоца, именилоца и количника. Скуп решења је означен зеленом бојом.

Решење корак по корак

1
Одредимо критичне тачке

Критичне тачке добијамо тако што изједначимо бројилац и именилац са нулом.

Нула бројиоца:

\[ x + 1 = 0 \quad \Rightarrow \quad x = -1 \]

Нула именилоца:

\[ x - 3 = 0 \quad \Rightarrow \quad x = 3 \]

Имамо две критичне тачке: \( x = -1 \) и \( x = 3 \). Оне деле бројевну праву на три интервала.

2
Поделимо бројевну праву на интервале

Критичне тачке \( x = -1 \) и \( x = 3 \) деле бројевну праву на три интервала:

\[ (-\infty,\; -1), \qquad (-1,\; 3), \qquad (3,\; +\infty) \]

На сваком од ових интервала, знак израза \( \frac{x+1}{x-3} \) је константан. Треба да проверимо знак на сваком интервалу.

3
Анализа знака на интервалу \( (-\infty,\; -1) \)

Бирамо тест-вредност \( x = -2 \):

\[ x + 1 = -2 + 1 = -1 \;<\; 0 \qquad (\text{негативно}) \]
\[ x - 3 = -2 - 3 = -5 \;<\; 0 \qquad (\text{негативно}) \]
\[ \frac{x+1}{x-3} = \frac{(-)}{(-)} = (+) \;\geq\; 0 \quad \checkmark \]

Количник два негативна броја је позитиван. Неједначина је задовољена.

4
Анализа знака на интервалу \( (-1,\; 3) \)

Бирамо тест-вредност \( x = 0 \):

\[ x + 1 = 0 + 1 = 1 \;>\; 0 \qquad (\text{позитивно}) \]
\[ x - 3 = 0 - 3 = -3 \;<\; 0 \qquad (\text{негативно}) \]
\[ \frac{x+1}{x-3} = \frac{(+)}{(-)} = (-) \;<\; 0 \quad \times \]

Количник позитивног и негативног броја је негативан. Неједначина није задовољена.

Провери себе: Шта је знак израза за \( x = 1 \)?
За \( x = 1 \): бројилац \( 1 + 1 = 2 > 0 \), именилац \( 1 - 3 = -2 < 0 \). Количник: \( \frac{2}{-2} = -1 < 0 \). Дакле, израз је негативан на интервалу \( (-1, 3) \), што потврђује наш резултат.
5
Анализа знака на интервалу \( (3,\; +\infty) \)

Бирамо тест-вредност \( x = 4 \):

\[ x + 1 = 4 + 1 = 5 \;>\; 0 \qquad (\text{позитивно}) \]
\[ x - 3 = 4 - 3 = 1 \;>\; 0 \qquad (\text{позитивно}) \]
\[ \frac{x+1}{x-3} = \frac{(+)}{(+)} = (+) \;\geq\; 0 \quad \checkmark \]

Количник два позитивна броја је позитиван. Неједначина је задовољена.

6
Табела знакова

Сумирајмо анализу знака у прегледну табелу:

Интервал \( (-\infty,\; -1) \) \( x = -1 \) \( (-1,\; 3) \) \( x = 3 \) \( (3,\; +\infty) \)
\( x + 1 \) − 0 + + +
\( x - 3 \) − − − 0 +
\( \dfrac{x+1}{x-3} \) + 0 − н.д. +

н.д. = није дефинисано (дељење нулом)

Провери себе: Зашто тачка \( x = -1 \) улази у решење, а \( x = 3 \) не?
За \( x = -1 \): израз \( \frac{-1+1}{-1-3} = \frac{0}{-4} = 0 \), а \( 0 \geq 0 \) је тачно — зато укључујемо \( x = -1 \). За \( x = 3 \): именилац \( 3 - 3 = 0 \), па израз уопште није дефинисан — зато искључујемо \( x = 3 \).
7
Проверимо граничне тачке

Тачка \( x = -1 \) (нула бројиоца):

\[ \frac{-1 + 1}{-1 - 3} = \frac{0}{-4} = 0 \;\geq\; 0 \quad \checkmark \]

Вредност израза је нула, а \( 0 \geq 0 \) јесте тачно. Укључујемо \( x = -1 \) (затворена заграда).

Тачка \( x = 3 \) (нула именилоца):

\[ \frac{3 + 1}{3 - 3} = \frac{4}{0} \quad \text{— није дефинисано!} \]

Израз није дефинисан за \( x = 3 \). Искључујемо \( x = 3 \) (отворена заграда).

8
Формирамо скуп решења

Неједначина \( \frac{x+1}{x-3} \geq 0 \) важи на интервалима где је израз позитиван или једнак нули:

  • На \( (-\infty, -1) \) — израз је позитиван
  • У тачки \( x = -1 \) — израз је једнак нули
  • На \( (3, +\infty) \) — израз је позитиван

Спајамо интервал \( (-\infty, -1) \) са тачком \( x = -1 \) и добијамо \( (-\infty, -1] \).

\[ x \in (-\infty,\; -1] \cup (3,\; +\infty) \]
Провери себе: Колико је \( \frac{x+1}{x-3} \) за \( x = -100 \) и \( x = 100 \)?
За \( x = -100 \): \( \frac{-100+1}{-100-3} = \frac{-99}{-103} \approx 0{,}96 > 0 \). За \( x = 100 \): \( \frac{100+1}{100-3} = \frac{101}{97} \approx 1{,}04 > 0 \). Обе вредности су позитивне, што потврђује да ова решења припадају скупу решења.

Кључни увид

Количник два линеарна израза мења знак само у критичним тачкама (нулама бројиоца и именилоца). Довољно је проверити знак на једном од интервала, а затим се знак наизменично мења преласком сваке критичне тачке. Код неједначине \( \geq 0 \), нуле бројиоца укључујемо (јер је \( 0 \geq 0 \) тачно), док нуле именилоца увек искључујемо (дељење нулом).

\[ \frac{x + 1}{x - 3} \geq 0 \quad \Longleftrightarrow \quad x \in (-\infty,\; -1] \cup (3,\; +\infty) \]
Коначан одговор
\( x \in (-\infty,\; -1] \cup (3,\; +\infty) \)
(а) \(x \in [-1,\; 3)\)
(б) \(x \in (-\infty,\; -1] \cup (3,\; +\infty)\)
(в) \(x \in [3,\; +\infty)\)
Верификација: Проверимо по једну вредност из сваког дела решења. За \( x = -5 \): \( \frac{-5+1}{-5-3} = \frac{-4}{-8} = 0{,}5 > 0 \) — тачно. За \( x = -1 \): \( \frac{0}{-4} = 0 \geq 0 \) — тачно. За \( x = 10 \): \( \frac{11}{7} \approx 1{,}57 > 0 \) — тачно. А за \( x = 0 \) (не припада решењу): \( \frac{1}{-3} \approx -0{,}33 < 0 \) — заиста не задовољава. Одговор је (б).

Честе грешке

✗

Заборавити да искључимо нулу именилоца: Многи ученици напишу \( (-\infty, -1] \cup [3, +\infty) \) уместо \( (-\infty, -1] \cup (3, +\infty) \). Тачка \( x = 3 \) се мора искључити јер је \( \frac{4}{0} \) недефинисано. Увек проверите: именилац = 0 значи отворена заграда!

✗

Множити обе стране именилоцем без поделе на случајеве: Ако неједначину помножимо са \( (x-3) \), морамо разликовати случај \( x - 3 > 0 \) (знак остаје) и \( x - 3 < 0 \) (знак се мења). Метод интервала је сигурнији и бржи приступ.

Додатни изазов

Решити неједначину \( \displaystyle \frac{(x+1)(x-5)}{(x-3)^2} \leq 0 \).

Погледај наговештај

Критичне тачке су \( x = -1 \), \( x = 3 \) и \( x = 5 \). Обратите пажњу да \( (x-3)^2 \geq 0 \) увек, па именилац не мења знак — он је увек ненегативан (нула у тачки \( x = 3 \), коју искључујемо). Знак целог израза зависи само од бројиоца \( (x+1)(x-5) \). Решење: \( x \in [-1, 3) \cup (3, 5] \).
