Racionalna nejednačina

Analiza znaka racionalnog izraza primenom metode intervala

Postavka zadatka

Rešiti nejednačinu:

\[\frac{(x-1)(x+2)}{x+1} < 0\]
(a)
\(x \in (-\infty,\, 1)\)
(b)
\(x \in (-\infty,\, -2) \cup (-1,\, 1)\)
(v)
\(x \in (-2,\, -1) \cup (1,\, +\infty)\)

Plan rešavanja

Odredićemo kritične tačke (nule brojioca i imenioca), zatim ćemo metodom intervala ispitati znak izraza u svakom od dobijenih podintervala i odabrati one gde je izraz negativan.

Ključne formule i teoreme
  • Metoda intervala: Za rešavanje nejednačine oblika \(\frac{P(x)}{Q(x)} < 0\), nalazimo nule brojioca i imenioca, pa ispitujemo znak izraza u svakom intervalu.
  • Nule brojioca: Tačke u kojima je \(P(x) = 0\) — izraz menja znak i jednak je nuli.
  • Nule imenioca: Tačke u kojima je \(Q(x) = 0\) — izraz nije definisan, te se te tačke isključuju iz domena.
  • Pravilo znaka: Proizvod/količnik dva broja istog znaka je pozitivan, a različitog znaka je negativan.

Rešenje — korak po korak

1

Određivanje kritičnih tačaka

Nalazimo vrednosti \(x\) za koje je izraz jednak nuli ili nije definisan.

\[\text{Brojilac: } (x-1)(x+2) = 0 \implies x = 1 \text{ ili } x = -2\] \[\text{Imenilac: } x+1 = 0 \implies x = -1\]

Kritične tačke su: \(x = -2\), \(x = -1\), \(x = 1\). Tačka \(x = -1\) se isključuje iz domena jer imenilac ne sme biti nula.

2

Podela realne prave na intervale

Kritične tačke dele realnu pravu na četiri intervala:

\[(-\infty,\, -2), \quad (-2,\, -1), \quad (-1,\, 1), \quad (1,\, +\infty)\]
Proveri sebe: Zašto tačka \(x = -1\) nije u rešenju?

Za \(x = -1\) imenilac \(x + 1 = 0\), pa je izraz nedefinisan. Vrednosti u kojima je imenilac jednak nuli uvek se isključuju iz domena.

3

Analiza znaka u svakom intervalu

U svakom intervalu biramo probnu tačku i određujemo znak svakog činioca:

Interval \(x - 1\) \(x + 2\) \(x + 1\) Izraz
\(x < -2\) \(-\) \(-\) \(-\) \(-\) < 0 ✓
\(-2 < x < -1\) \(-\) \(+\) \(-\) \(+\) > 0 ✗
\(-1 < x < 1\) \(-\) \(+\) \(+\) \(-\) < 0 ✓
\(x > 1\) \(+\) \(+\) \(+\) \(+\) > 0 ✗
4

Detaljno objašnjenje za svaki interval

Interval \((-\infty, -2)\): Uzmimo \(x = -3\).

\[\frac{(-3-1)(-3+2)}{-3+1} = \frac{(-4)(-1)}{-2} = \frac{4}{-2} = -2 < 0 \quad \checkmark\]

Interval \((-2, -1)\): Uzmimo \(x = -1{,}5\).

\[\frac{(-1{,}5-1)(-1{,}5+2)}{-1{,}5+1} = \frac{(-2{,}5)(0{,}5)}{-0{,}5} = \frac{-1{,}25}{-0{,}5} = 2{,}5 > 0 \quad \times\]

Interval \((-1, 1)\): Uzmimo \(x = 0\).

\[\frac{(0-1)(0+2)}{0+1} = \frac{(-1)(2)}{1} = -2 < 0 \quad \checkmark\]

Interval \((1, +\infty)\): Uzmimo \(x = 2\).

\[\frac{(2-1)(2+2)}{2+1} = \frac{(1)(4)}{3} = \frac{4}{3} > 0 \quad \times\]
Proveri sebe: Zašto su tačke \(x = -2\) i \(x = 1\) isključene iz rešenja?

Nejednačina je stroga (\( < 0\)), pa vrednosti za koje je izraz jednak nuli (\(x = -2\) i \(x = 1\)) nisu deo rešenja. Da je nejednačina bila \(\leq 0\), ove tačke bi bile uključene.

5

Formiranje konačnog rešenja

Izraz je negativan u intervalima \((-\infty, -2)\) i \((-1, 1)\). Spajamo ih u uniju:

\[x \in (-\infty,\, -2) \cup (-1,\, 1)\]

Ključni uvid

Znak racionalnog izraza menja se u svakoj kritičnoj tački (nuli brojioca ili imenioca). Kada prolazimo kroz kritičnu tačku, znak se menja jer tačno jedan činilac menja znak. Zato se znaci u susednim intervalima naizmenično smenjuju — pod uslovom da su sve nule jednostruke.

Konačan odgovor

\[x \in (-\infty,\, -2) \cup (-1,\, 1)\]
(a) \(x \in (-\infty,\, 1)\)
(b) \(x \in (-\infty,\, -2) \cup (-1,\, 1)\)
(v) \(x \in (-2,\, -1) \cup (1,\, +\infty)\)

Verifikacija: Za \(x = -3\), izraz daje \(-2 < 0\) ✓. Za \(x = 0\), izraz daje \(-2 < 0\) ✓. Za \(x = -1{,}5\), izraz daje \(2{,}5 > 0\) ✗ — ispravno isključen.

Česte greške

⚠ Zaboravljanje da se \(x = -1\) isključi iz domena. Imenilac ne sme biti nula, pa \(x = -1\) nikada nije deo rešenja.

⚠ Brkanje stroge i nestroge nejednačine. Kod stroge nejednačine (\( < 0\)), nule brojioca (\(x = -2\) i \(x = 1\)) se ne uključuju u rešenje.

Dodatni izazov

Kako bi se promenilo rešenje da je nejednačina bila \(\frac{(x-1)(x+2)}{x+1} \leq 0\)?

Pogledaj odgovor

Kod nestroge nejednačine (\(\leq 0\)), uključujemo i tačke gde je izraz jednak nuli, tj. nule brojioca \(x = -2\) i \(x = 1\). Tačka \(x = -1\) i dalje ostaje isključena jer izraz nije definisan. Rešenje bi glasilo:

\[x \in (-\infty,\, -2] \cup (-1,\, 1]\]
