Izracunavanje vrednosti \(x\) iz slozenog razlomka

Aritmetika | Razlomci i decimalni brojevi

Zadatak

Ako je

\[\frac{3{,}3375 : 0{,}445 + 0{,}5}{3\dfrac{17}{35} : 8\dfrac{5}{7} + 0{,}6} = \frac{15}{x}\]

onda je:

\(\text{a) } x = -2\)
\(\text{b) } x = 2\)
\(\text{v) } x = 0{,}533\)
\(\text{g) } x = 1{,}875\)

Plan resavanja

  • Izracunati brojilac: \(3{,}3375 : 0{,}445 + 0{,}5\)
  • Izracunati imenilac: \(3\tfrac{17}{35} : 8\tfrac{5}{7} + 0{,}6\)
  • Podeliti brojilac sa imeniocem i resiti jednacinu za \(x\)
Podsetnik iz teorije

Deljenje decimalnih brojeva: Pri deljenju \(a : b\), mnozimo i brojilac i imenilac sa \(10^n\) da uklonimo decimale, cime pretvaramo deljenje u deljenje celih brojeva.

Mesoviti brojevi: Mesoviti broj \(a\tfrac{b}{c}\) se pretvara u nepravi razlomak: \(\frac{a \cdot c + b}{c}\).

Deljenje razlomaka: \(\frac{a}{b} : \frac{c}{d} = \frac{a}{b} \cdot \frac{d}{c}\) — deljenje razlomkom se svodi na mnozenje reciprocnom vrednoscu.

Prioritet operacija: Prvo mnozenje/deljenje, pa sabiranje/oduzimanje.

Vizuelni prikaz

Resenje korak po korak

1 Izracunavanje \(3{,}3375 : 0{,}445\)

Mnozimo brojilac i imenilac sa \(10\,000\) da uklonimo decimale:

\[3{,}3375 : 0{,}445 = \frac{33375}{4450}\]

Skracujemo razlomak. Delimo oba broja sa \(5\) dva puta:

\[\frac{33375}{4450} = \frac{6675}{890} = \frac{1335}{178}\]

Primetimo da je \(1335 = 7{,}5 \times 178\). Proverimo:

\[7{,}5 \times 0{,}445 = 7 \times 0{,}445 + 0{,}5 \times 0{,}445 = 3{,}115 + 0{,}2225 = 3{,}3375 \;\checkmark\]
Kontrolna tacka
\(3{,}3375 : 0{,}445 = 7{,}5\)
2 Izracunavanje brojioca: \(7{,}5 + 0{,}5\)

Dodajemo \(0{,}5\) na rezultat iz koraka 1:

\[7{,}5 + 0{,}5 = 8\]
Kontrolna tacka
Brojilac = \(8\)
3 Pretvaranje mesovitih brojeva u neprave razlomke

Pretvaramo \(3\tfrac{17}{35}\) i \(8\tfrac{5}{7}\):

\[3\tfrac{17}{35} = \frac{3 \cdot 35 + 17}{35} = \frac{105 + 17}{35} = \frac{122}{35}\] \[8\tfrac{5}{7} = \frac{8 \cdot 7 + 5}{7} = \frac{56 + 5}{7} = \frac{61}{7}\]
Kontrolna tacka
Mesoviti brojevi pretvoreni u neprave razlomke
4 Izracunavanje \(\frac{122}{35} : \frac{61}{7}\)

Deljenje razlomaka — mnozimo reciprocnom vrednoscu:

\[\frac{122}{35} : \frac{61}{7} = \frac{122}{35} \cdot \frac{7}{61}\]

Primetimo da je \(122 = 2 \times 61\), pa skracujemo \(61\):

\[= \frac{2 \cdot \cancel{61}}{35} \cdot \frac{7}{\cancel{61}} = \frac{2 \cdot 7}{35} = \frac{14}{35} = \frac{2}{5}\]
Kontrolna tacka
Provera: \(\frac{2}{5} \times \frac{61}{7} = \frac{122}{35} = 3\tfrac{17}{35}\) ✓
5 Izracunavanje imenioca: \(\frac{2}{5} + 0{,}6\)

Pretvaramo \(0{,}6 = \frac{3}{5}\) i sabiramo:

\[\frac{2}{5} + \frac{3}{5} = \frac{2 + 3}{5} = \frac{5}{5} = 1\]
Kontrolna tacka
Imenilac = \(1\)
6 Nalazenje vrednosti \(x\)

Sada imamo:

\[\frac{8}{1} = \frac{15}{x}\]

Resavamo po \(x\):

\[8 = \frac{15}{x} \implies x = \frac{15}{8} = 1{,}875\]
Kontrolna tacka
\(x = \frac{15}{8} = 1{,}875\)

Kljucni uvid

Zadatak deluje slozeno, ali kada se sistematski izracunaju brojilac i imenilac, oba daju lepe vrednosti (\(8\) i \(1\)). Kljuc je pazljivo deljenje decimalnih brojeva i skracivanje razlomaka — posebno uocavanje da je \(122 = 2 \times 61\) sto omogucava elegantno skracivanje. Na prijemnim ispitima, kada dobijete "lep" medjurezultat, to je signal da ste na pravom putu.

Konacan odgovor

\[x = \dfrac{15}{8} = 1{,}875\]
\(\text{a) } x = -2\)
\(\text{b) } x = 2\)
\(\text{v) } x = 0{,}533\)
\(\text{g) } x = 1{,}875\) ✓

Provera:

\[\frac{15}{x} = \frac{15}{1{,}875} = \frac{15}{\;\frac{15}{8}\;} = 15 \cdot \frac{8}{15} = 8 \;\checkmark\]

Brojilac naseg izraza je takodje \(8\), a imenilac \(1\), tako da \(\frac{8}{1} = 8\). Jednakost vazi.

Najcesce greske

Pogresan redosled operacija: Ako se prvo sabere \(0{,}445 + 0{,}5\) pa tek onda deli, dobija se pogresan rezultat. Uvek prvo deljenje, pa sabiranje!
Greska pri pretvaranju mesovitih brojeva: Cest propust je da se napise \(3\tfrac{17}{35} = \frac{17}{35}\) umesto \(\frac{122}{35}\). Ne zaboravite da pomnozite celi deo sa imeniocem i dodate brojilac.
Deljenje razlomaka: Deljenje razlomkom znaci mnozenje reciprocnom vrednoscu — ne mnozenje istim razlomkom.
Greska u decimalnom deljenju: Pri deljenju \(3{,}3375 : 0{,}445\) treba pazljivo pomeriti decimalni zarez i proveriti rezultat.

Dodatni izazov

Pokusajte da resite slican zadatak:

\[\frac{2{,}275 : 0{,}35 + 0{,}5}{4\tfrac{3}{8} : 1\tfrac{3}{4} + 0{,}5} = \frac{21}{y}\]
Prikazi odgovor

Brojilac: \(2{,}275 : 0{,}35 = 6{,}5\), pa \(6{,}5 + 0{,}5 = 7\).

Imenilac: \(\frac{35}{8} : \frac{7}{4} = \frac{35}{8} \times \frac{4}{7} = \frac{5}{2} = 2{,}5\), pa \(2{,}5 + 0{,}5 = 3\).

Dakle \(\frac{7}{3} = \frac{21}{y}\), odakle \(y = 9\).
