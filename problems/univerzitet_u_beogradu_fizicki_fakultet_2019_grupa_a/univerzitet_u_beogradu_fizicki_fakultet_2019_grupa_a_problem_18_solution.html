Zbir rešenja jednačine |sin x| = sin x + 2cos x

Trigonometrijska jednačina sa apsolutnom vrednošću

📝 Tekst zadatka

Koliki je zbir svih realnih rešenja jednačine \( |\sin x| = \sin x + 2\cos x \), koja pripadaju intervalu \( (0,\, 3\pi) \)?

(A) \( \dfrac{3\pi}{4} \)
(Б) \( \dfrac{9\pi}{4} \)
(В) \( \dfrac{19\pi}{4} \)
(Г) \( \dfrac{25\pi}{4} \)
💡 Plan rešavanja

Raščlanićemo jednačinu na dva slučaja prema znaku \(\sin x\): kada je \(\sin x \geq 0\) i kada je \(\sin x < 0\). U svakom slučaju dobijamo standardnu trigonometrijsku jednačinu. Potom biramo rešenja iz intervala \((0, 3\pi)\) koja zadovoljavaju odgovarajući uslov za znak sinusa.

📚 Podsetnik iz teorije
Apsolutna vrednost i trigonometrijske jednačine ▼

Definicija apsolutne vrednosti:

\[ |a| = \begin{cases} a, & a \geq 0 \\ -a, & a < 0 \end{cases} \]

Ključne vrednosti trigonometrijskih funkcija:

\[ \cos x = 0 \iff x = \frac{\pi}{2} + k\pi, \quad k \in \mathbb{Z} \]
\[ \tan x = -1 \iff x = -\frac{\pi}{4} + k\pi = \frac{3\pi}{4} + k\pi, \quad k \in \mathbb{Z} \]

Znakovi trigonometrijskih funkcija po kvadrantima:

\[ \begin{array}{c|c|c} \text{Kvadrant} & \sin x & \cos x \\ \hline \text{I } (0, \pi/2) & + & + \\ \text{II } (\pi/2, \pi) & + & - \\ \text{III } (\pi, 3\pi/2) & - & - \\ \text{IV } (3\pi/2, 2\pi) & - & + \end{array} \]

✏️ Rešenje korak po korak
1
Postavljamo dva slučaja prema znaku \(\sin x\)

Jednačina glasi \( |\sin x| = \sin x + 2\cos x \). Apsolutna vrednost se uklanja razmatranjem dva slučaja:

Slučaj 1 — Kada je \(\sin x \geq 0\), važi \(|\sin x| = \sin x\).

Slučaj 2 — Kada je \(\sin x < 0\), važi \(|\sin x| = -\sin x\).

2
Slučaj 1: \(\sin x \geq 0\)

Zamenom \(|\sin x| = \sin x\) u jednačinu dobijamo:

\[ \sin x = \sin x + 2\cos x \]

Oduzimanjem \(\sin x\) sa obe strane:

\[ 0 = 2\cos x \implies \cos x = 0 \]

Opšte rešenje jednačine \(\cos x = 0\) je:

\[ x = \frac{\pi}{2} + k\pi, \quad k \in \mathbb{Z} \]

Rešenja u intervalu \((0, 3\pi)\):

\[ x = \frac{\pi}{2},\quad x = \frac{3\pi}{2},\quad x = \frac{5\pi}{2} \]

Sada proveravamo uslov \(\sin x \geq 0\):

  • \( x = \dfrac{\pi}{2} \): \(\sin\dfrac{\pi}{2} = 1 > 0\) — zadovoljava ✔
  • \( x = \dfrac{3\pi}{2} \): \(\sin\dfrac{3\pi}{2} = -1 < 0\) — ne zadovoljava ✘
  • \( x = \dfrac{5\pi}{2} \): \(\sin\dfrac{5\pi}{2} = 1 > 0\) — zadovoljava ✔

Rešenja iz Slučaja 1: \( x = \dfrac{\pi}{2} \) i \( x = \dfrac{5\pi}{2} \).

✓ Proveri sebe: Zašto \( x = 3\pi/2 \) ne zadovoljava?

Jer smo u Slučaju 1 pretpostavili \(\sin x \geq 0\), a \(\sin(3\pi/2) = -1 < 0\). Dakle, za tu vrednost \(x\) smo u pogrešnom slučaju — moramo je razmatrati u Slučaju 2.

3
Slučaj 2: \(\sin x < 0\)

Zamenom \(|\sin x| = -\sin x\) u jednačinu dobijamo:

\[ -\sin x = \sin x + 2\cos x \]

Prebacivanjem \(\sin x\) na levu stranu:

\[ -2\sin x = 2\cos x \implies -\sin x = \cos x \implies \tan x = -1 \]

Opšte rešenje jednačine \(\tan x = -1\) je:

\[ x = \frac{3\pi}{4} + k\pi, \quad k \in \mathbb{Z} \]

Rešenja u intervalu \((0, 3\pi)\):

\[ x = \frac{3\pi}{4},\quad x = \frac{7\pi}{4},\quad x = \frac{11\pi}{4} \]

Proveravamo uslov \(\sin x < 0\):

  • \( x = \dfrac{3\pi}{4} \): \(\sin\dfrac{3\pi}{4} = \dfrac{\sqrt{2}}{2} > 0\) — ne zadovoljava ✘
  • \( x = \dfrac{7\pi}{4} \): \(\sin\dfrac{7\pi}{4} = -\dfrac{\sqrt{2}}{2} < 0\) — zadovoljava ✔
  • \( x = \dfrac{11\pi}{4} \): \(\sin\dfrac{11\pi}{4} = \sin\dfrac{3\pi}{4} = \dfrac{\sqrt{2}}{2} > 0\) — ne zadovoljava ✘

Rešenje iz Slučaja 2: \( x = \dfrac{7\pi}{4} \).

✓ Proveri sebe: Zašto tačno \(\sin(11\pi/4) = \sin(3\pi/4)\)?

Jer je \(\dfrac{11\pi}{4} = \dfrac{3\pi}{4} + 2\pi\), a funkcija sinus je periodična sa periodom \(2\pi\), pa važi \(\sin\dfrac{11\pi}{4} = \sin\dfrac{3\pi}{4}\).

4
Verifikacija svih rešenja

Proverimo da sva tri rešenja zaista zadovoljavaju polaznu jednačinu \(|\sin x| = \sin x + 2\cos x\):

\[ x = \frac{\pi}{2}: \quad |\sin\frac{\pi}{2}| = |1| = 1, \quad \sin\frac{\pi}{2} + 2\cos\frac{\pi}{2} = 1 + 0 = 1 \quad \checkmark \]
\[ x = \frac{7\pi}{4}: \quad \left|\sin\frac{7\pi}{4}\right| = \frac{\sqrt{2}}{2}, \quad \sin\frac{7\pi}{4} + 2\cos\frac{7\pi}{4} = -\frac{\sqrt{2}}{2} + 2 \cdot \frac{\sqrt{2}}{2} = \frac{\sqrt{2}}{2} \quad \checkmark \]
\[ x = \frac{5\pi}{2}: \quad \left|\sin\frac{5\pi}{2}\right| = |1| = 1, \quad \sin\frac{5\pi}{2} + 2\cos\frac{5\pi}{2} = 1 + 0 = 1 \quad \checkmark \]
5
Računamo zbir rešenja

Sva rešenja iz intervala \((0, 3\pi)\) su: \( x_1 = \dfrac{\pi}{2} \), \( x_2 = \dfrac{7\pi}{4} \), \( x_3 = \dfrac{5\pi}{2} \).

\[ S = \frac{\pi}{2} + \frac{7\pi}{4} + \frac{5\pi}{2} \]

Svodimo na zajednički imenilac 4:

\[ S = \frac{2\pi}{4} + \frac{7\pi}{4} + \frac{10\pi}{4} = \frac{2\pi + 7\pi + 10\pi}{4} = \frac{19\pi}{4} \]
🌟 Ključni uvid

Apsolutna vrednost zahteva razdvajanje na dva slučaja, ali ključno je ne zaboraviti proveriti uslove za svaki slučaj. Od šest kandidata (tri iz svakog slučaja), samo tri zaista zadovoljavaju polaznu jednačinu. Česta greška je uključiti \(x = 3\pi/2\) (gde \(\cos x = 0\), ali \(\sin x < 0\)) ili \(x = 3\pi/4\) (gde \(\tan x = -1\), ali \(\sin x > 0\)).

🏆 Konačan odgovor
\[ S = \frac{19\pi}{4} \]
(A) \( \dfrac{3\pi}{4} \)
(Б) \( \dfrac{9\pi}{4} \)
(В) \( \dfrac{19\pi}{4} \)
(Г) \( \dfrac{25\pi}{4} \)

Tačan odgovor je (В). Sva tri rešenja su verifikovana direktnom zamenom u polaznu jednačinu.

⚠️ Česte greške
❌

Izostavljanje provere znaka: Kandidat \( x = 3\pi/2 \) zadovoljava \(\cos x = 0\), ali ne zadovoljava \(\sin x \geq 0\). Uključivanje ovog lažnog rešenja daje zbir \(\dfrac{25\pi}{4}\) — pogrešan odgovor (Г).

❌

Propuštanje rešenja \( x = 5\pi/2 \): Interval \((0, 3\pi)\) prelazi dve pune periode kosinusa. Ako se razmatraju samo rešenja do \(2\pi\), gubi se rešenje \(x = 5\pi/2\), što daje pogrešan zbir \(\dfrac{9\pi}{4}\) — odgovor (Б).

❌

Deljenje nulom pri izvlačenju tangensa: Jednačina \(-\sin x = \cos x\) se deli sa \(\cos x\) da bi se dobilo \(\tan x = -1\). Treba proveriti da \(\cos x \neq 0\) za ta rešenja (jeste \(\neq 0\) za sva rešenja oblika \(x = 3\pi/4 + k\pi\)).

Nagoveštaj ▼

Primenite isti princip: razdvojte na slučajeve \(\cos x \geq 0\) i \(\cos x < 0\). U prvom slučaju dobijate \(\sin x = 0\) (uz uslov \(\cos x \geq 0\)), a u drugom \(\tan x = -1\) (uz uslov \(\cos x < 0\)). Pažljivo proverite uslove za svako rešenje-kandidat na datom intervalu.
