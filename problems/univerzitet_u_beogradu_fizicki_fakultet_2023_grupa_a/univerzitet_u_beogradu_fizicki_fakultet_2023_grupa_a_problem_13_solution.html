Kompozicija funkcija — f(g(x))

Izracunavanje vrednosti slozene funkcije u datoj tacki

📋 Tekst zadatka

Date su dve funkcije:

\[f(x) = x + 3 \qquad \text{i} \qquad g(x) = \frac{1 - f(x)}{1 + f(x)}\]

Kolika je vrednost funkcije \(f(g(x))\) u tacki \(x = 1\)?

(a)
\(4\)
(b)
\(-\dfrac{3}{5}\)
(v)
\(\dfrac{3}{5}\)
(g)
\(\dfrac{12}{5}\)

🗺️ Plan resavanja

Prvo cemo razviti funkciju \(g(x)\) tako sto cemo u njen izraz uvrstiti \(f(x) = x + 3\). Zatim cemo izracunati vrednost \(g(1)\), i na kraju tu vrednost uvrstiti u \(f\) da bismo dobili \(f(g(1))\).

📐 Podsetnik iz teorije

Kljucni koncepti za ovaj zadatak:

\[\text{Kompozicija funkcija: } (f \circ g)(x) = f(g(x))\]

Kompozicija funkcija znaci da izlaz jedne funkcije postaje ulaz druge. Da bismo izracunali \(f(g(x))\), prvo izracunamo \(g(x)\), a zatim taj rezultat uvrstimo u \(f\).

\[\text{Zamena: ako je } g(x) = \frac{1 - f(x)}{1 + f(x)}, \text{ uvrstimo } f(x) = x + 3\]

Kada je \(g(x)\) definisana preko \(f(x)\), potrebno je prvo eksplicitno izraziti \(g(x)\) samo preko \(x\).

\[\text{Sabiranje razlomaka: } \frac{a}{b} + c = \frac{a + bc}{b}\]

Ovo pravilo koristimo kada na kraju sabiramo razlomak sa celim brojem.

📊 Vizuelni prikaz

Prevucite tacku po x-osi da vidite kako se menjaju vrednosti \(g(x)\) i \(f(g(x))\)

📝 Resenje korak po korak

1 Zapisujemo poznate funkcije

Imamo dve funkcije:

\[f(x) = x + 3\]
\[g(x) = \frac{1 - f(x)}{1 + f(x)}\]

Primetimo da je \(g(x)\) definisana preko funkcije \(f(x)\), pa moramo prvo uvrstiti izraz za \(f(x)\) da bismo dobili eksplicitan oblik za \(g(x)\).

2 Razvijamo \(g(x)\) — uvrstavanje \(f(x) = x + 3\)

Zamenjujemo svako pojavljivanje \(f(x)\) u izrazu za \(g(x)\) sa \(x + 3\):

\[g(x) = \frac{1 - (x + 3)}{1 + (x + 3)}\]

Sredujemo brojilac:

\[1 - (x + 3) = 1 - x - 3 = -2 - x\]

Sredujemo imenilac:

\[1 + (x + 3) = 1 + x + 3 = 4 + x\]

Dakle, eksplicitan oblik funkcije \(g(x)\) je:

\[g(x) = \frac{-2 - x}{4 + x}\]
Proveri sebe: Koliko je \(g(0)\)?

Uvrstimo \(x = 0\): \(g(0) = \frac{-2 - 0}{4 + 0} = \frac{-2}{4} = -\frac{1}{2}\). Proverimo i preko originalne definicije: \(f(0) = 3\), pa \(g(0) = \frac{1 - 3}{1 + 3} = \frac{-2}{4} = -\frac{1}{2}\). Poklapa se!

3 Racunamo \(g(1)\)

Uvrstimo \(x = 1\) u izraz za \(g(x)\):

\[g(1) = \frac{-2 - 1}{4 + 1} = \frac{-3}{5}\]

Dakle, \(g(1) = -\dfrac{3}{5}\).

Napomena: Ovo je vrednost opcije (b), ali to nije konacan odgovor! Zadatak trazi \(f(g(1))\), ne samo \(g(1)\).

4 Racunamo \(f(g(1))\) — uvrstavanje u \(f\)

Sada uvrstimo vrednost \(g(1) = -\frac{3}{5}\) u funkciju \(f\):

\[f\!\left(g(1)\right) = f\!\left(-\frac{3}{5}\right) = -\frac{3}{5} + 3\]

Broj \(3\) zapisujemo kao razlomak sa imeniocem \(5\):

\[-\frac{3}{5} + 3 = -\frac{3}{5} + \frac{15}{5}\]

Sabiramo razlomke sa istim imeniocem:

\[-\frac{3}{5} + \frac{15}{5} = \frac{-3 + 15}{5} = \frac{12}{5}\]
Proveri sebe: Koliko je \(\frac{12}{5}\) u decimalnom zapisu?

\(\frac{12}{5} = 2{,}4\). To je pozitivan broj, veci od 2 i manji od 3.

5 Verifikacija — provera svim putem

Proverimo ceo postupak od pocetka do kraja:

\[\begin{align} f(1) &= 1 + 3 = 4 \\[6pt] g(1) &= \frac{1 - f(1)}{1 + f(1)} = \frac{1 - 4}{1 + 4} = \frac{-3}{5} \\[6pt] f(g(1)) &= f\!\left(-\frac{3}{5}\right) = -\frac{3}{5} + 3 = \frac{12}{5} \quad \checkmark \end{align}\]

Sve se poklapa. Rezultat je \(\dfrac{12}{5}\).

Proveri sebe: Da li je bilo koja druga ponudjena opcija jednaka \(f(g(1))\)?

Ne. Opcija (a) = 4 je zapravo \(f(1)\), a opcija (b) = \(-\frac{3}{5}\) je \(g(1)\). Obe su medjurezultati, ali ne i konacan odgovor. Opcija (v) = \(\frac{3}{5}\) bi bila greska u znaku. Jedini tacan odgovor je (g) = \(\frac{12}{5}\).

💡 Kljucni uvid

Kod kompozicije funkcija, kljucno je postovati redosled operacija: pri racunanju \(f(g(x))\), prvo se izracunava unutrasnja funkcija \(g(x)\), a zatim se njen rezultat uvrstava u spoljasnju funkciju \(f\). Cesta zamka je da se zaustavi posle prvog koraka — izracunavanja \(g(1) = -\frac{3}{5}\) — i proglasi to za odgovor. Uvek proverite sta tacno zadatak trazi!

\[f(g(1)) = f\!\left(-\frac{3}{5}\right) = -\frac{3}{5} + 3 = \frac{12}{5}\]

Konacan odgovor

\[f(g(1)) = \frac{12}{5}\]
(a) \(4\)
(b) \(-\dfrac{3}{5}\)
(v) \(\dfrac{3}{5}\)
(g) \(\dfrac{12}{5}\)

Odgovor je (g)

Provera: \(g(1) = \frac{-3}{5}\), pa \(f\!\left(-\frac{3}{5}\right) = -\frac{3}{5} + 3 = \frac{12}{5} = 2{,}4\). Vrednost je pozitivna i veca od 2 — sto ima smisla jer \(f\) dodaje 3 na ulaz koji je blizak nuli.

⚠️ Ceste greske

  • Zaustavljanje na \(g(1)\) umesto \(f(g(1))\): Mnogi ucenici izracunaju \(g(1) = -\frac{3}{5}\) i odmah zaokruze opciju (b), zaboravljajuci da zadatak trazi kompoziciju \(f(g(1))\). Uvek proverite sta je tacno trazeno — \(g(x)\) ili \(f(g(x))\).
  • Greska u znaku pri svodjenju brojioca: Pri razvijanju \(1 - (x + 3)\), cesta greska je da se napise \(1 - x + 3 = 4 - x\) umesto tacnog \(1 - x - 3 = -2 - x\). Minus ispred zagrade menja znak svih clanova unutar zagrade!

🚀 Izazov za tebe

Ako su iste funkcije \(f(x) = x + 3\) i \(g(x) = \frac{1 - f(x)}{1 + f(x)}\), kolika je vrednost \(g(f(1))\)?

Savet

Ovde je redosled obrnut: prvo izracunaj \(f(1) = 4\), a zatim uvrsti u \(g\). Seti se da je \(g(x) = \frac{-2 - x}{4 + x}\). Koliko je \(g(4)\)?
