P Plan rešavanja

Levу stranu jednačine \(\sin x - \sqrt{3}\cos x\) svešćemo na oblik \(R\sin(x - \varphi)\) koristeći formulu za linearnu kombinaciju sinusa i kosinusa. Time dobijamo jednostavnu sinusnu jednačinu čija rešenja lako nalazimo, a zatim biramo ona koja pripadaju intervalu \((0, 2\pi)\).

T Teorijska podloga

Prikaži ključne formule i činjenice
  • Linearna kombinacija: \(a\sin x + b\cos x = R\sin(x + \varphi)\), gde je \(R = \sqrt{a^2 + b^2}\), \(\cos\varphi = \dfrac{a}{R}\), \(\sin\varphi = \dfrac{b}{R}\).
  • Adiciona formula: \(\sin(x - \alpha) = \sin x \cos\alpha - \cos x \sin\alpha\).
  • Osnovna sinusna jednačina: Ako je \(\sin\theta = c\) i \(|c| \leq 1\), tada: \(\theta = \arcsin c + 2k\pi\) ili \(\theta = \pi - \arcsin c + 2k\pi\), \(k \in \mathbb{Z}\).
  • Posebna vrednost: \(\sin\dfrac{\pi}{6} = \dfrac{1}{2}\), \(\cos\dfrac{\pi}{3} = \dfrac{1}{2}\), \(\sin\dfrac{\pi}{3} = \dfrac{\sqrt{3}}{2}\).

1 Korak 1: Određivanje amplitude \(R\)

Izračunavanje \(R\)

Imamo izraz oblika \(a\sin x + b\cos x\), gde je \(a = 1\) i \(b = -\sqrt{3}\).

Amplituda je:

\[ R = \sqrt{a^2 + b^2} = \sqrt{1^2 + (-\sqrt{3})^2} = \sqrt{1 + 3} = \sqrt{4} = 2 \]

2 Korak 2: Svođenje na oblik \(R\sin(x - \varphi)\)

Rastavljanje na sinusni oblik

Delimo celу jednačinu sa \(R = 2\):

\[ \sin x - \sqrt{3}\cos x = 2\left(\frac{1}{2}\sin x - \frac{\sqrt{3}}{2}\cos x\right) \]

Prepoznajemo poznate vrednosti trigonometrijskih funkcija:

\[ \frac{1}{2} = \cos\frac{\pi}{3}, \qquad \frac{\sqrt{3}}{2} = \sin\frac{\pi}{3} \]

Primenom adicione formule \(\sin(x - \alpha) = \sin x\cos\alpha - \cos x\sin\alpha\) dobijamo:

\[ \sin x - \sqrt{3}\cos x = 2\sin\!\left(x - \frac{\pi}{3}\right) \]
✓ Kontrolna tačka: Proverimo za \(x = \frac{\pi}{3}\): leva strana = \(\sin\frac{\pi}{3} - \sqrt{3}\cos\frac{\pi}{3} = \frac{\sqrt{3}}{2} - \frac{\sqrt{3}}{2} = 0\). Desna strana = \(2\sin 0 = 0\). Tačno!

3 Korak 3: Rešavanje sinusne jednačine

Svođenje na osnovni oblik

Jednačina sada glasi:

\[ 2\sin\!\left(x - \frac{\pi}{3}\right) = 1 \]

Delimo obe strane sa 2:

\[ \sin\!\left(x - \frac{\pi}{3}\right) = \frac{1}{2} \]

Opšte rešenje

Jednačina \(\sin\theta = \frac{1}{2}\) ima rešenja:

\[ \theta = \frac{\pi}{6} + 2k\pi \quad \text{ili} \quad \theta = \frac{5\pi}{6} + 2k\pi, \quad k \in \mathbb{Z} \]

Zamenom \(\theta = x - \frac{\pi}{3}\) dobijamo:

\[ x - \frac{\pi}{3} = \frac{\pi}{6} + 2k\pi \quad \Rightarrow \quad x = \frac{\pi}{3} + \frac{\pi}{6} + 2k\pi = \frac{\pi}{2} + 2k\pi \]
\[ x - \frac{\pi}{3} = \frac{5\pi}{6} + 2k\pi \quad \Rightarrow \quad x = \frac{\pi}{3} + \frac{5\pi}{6} + 2k\pi = \frac{2\pi}{6} + \frac{5\pi}{6} + 2k\pi = \frac{7\pi}{6} + 2k\pi \]
Proveri se: Koliko je \(\frac{\pi}{3} + \frac{\pi}{6}\)?

\(\dfrac{\pi}{3} + \dfrac{\pi}{6} = \dfrac{2\pi}{6} + \dfrac{\pi}{6} = \dfrac{3\pi}{6} = \dfrac{\pi}{2}\). Tačno!

4 Korak 4: Izbor rešenja iz intervala \((0, 2\pi)\)

Prva familija: \(x = \frac{\pi}{2} + 2k\pi\)

Za \(k = 0\): \(x = \frac{\pi}{2} \approx 1{,}571\). Pošto je \(0 < \frac{\pi}{2} < 2\pi\), ovo rešenje pripada intervalu.

Za \(k = 1\): \(x = \frac{\pi}{2} + 2\pi = \frac{5\pi}{2} > 2\pi\). Ne pripada.

Za \(k = -1\): \(x = \frac{\pi}{2} - 2\pi = -\frac{3\pi}{2} < 0\). Ne pripada.

Druga familija: \(x = \frac{7\pi}{6} + 2k\pi\)

Za \(k = 0\): \(x = \frac{7\pi}{6} \approx 3{,}665\). Pošto je \(0 < \frac{7\pi}{6} < 2\pi\), ovo rešenje pripada intervalu.

Za \(k = 1\): \(x = \frac{7\pi}{6} + 2\pi = \frac{19\pi}{6} > 2\pi\). Ne pripada.

Za \(k = -1\): \(x = \frac{7\pi}{6} - 2\pi = -\frac{5\pi}{6} < 0\). Ne pripada.

✓ Kontrolna tačka: Ukupno 2 rešenja u intervalu \((0, 2\pi)\): \(x_1 = \frac{\pi}{2}\) i \(x_2 = \frac{7\pi}{6}\).

5 Korak 5: Verifikacija rešenja

Provera \(x_1 = \frac{\pi}{2}\)

\[ \sin\frac{\pi}{2} - \sqrt{3}\cos\frac{\pi}{2} = 1 - \sqrt{3} \cdot 0 = 1 \quad \checkmark \]

Provera \(x_2 = \frac{7\pi}{6}\)

Koristimo činjenice: \(\sin\frac{7\pi}{6} = -\frac{1}{2}\), \(\cos\frac{7\pi}{6} = -\frac{\sqrt{3}}{2}\).

\[ \sin\frac{7\pi}{6} - \sqrt{3}\cos\frac{7\pi}{6} = -\frac{1}{2} - \sqrt{3}\cdot\!\left(-\frac{\sqrt{3}}{2}\right) = -\frac{1}{2} + \frac{3}{2} = \frac{2}{2} = 1 \quad \checkmark \]
Proveri se: Gde se nalaze ova rešenja na jediničnom krugu?

\(x_1 = \frac{\pi}{2}\) odgovara tački \((0, 1)\) — vrh kruga (pozitivni deo \(y\)-ose).

\(x_2 = \frac{7\pi}{6}\) odgovara tački \(\left(-\frac{\sqrt{3}}{2}, -\frac{1}{2}\right)\) — treći kvadrant, ugao od \(210°\).

! Ključni uvid

Svaki izraz oblika \(a\sin x + b\cos x\) može se zapisati kao jedan sinusni izraz sa amplitudom \(R = \sqrt{a^2 + b^2}\) i faznim pomakom. U ovom slučaju:

\(\displaystyle \sin x - \sqrt{3}\cos x = 2\sin\!\left(x - \frac{\pi}{3}\right)\)

Ova transformacija je univerzalan alat za rešavanje trigonometrijskih jednačina sa linearnom kombinacijom sinusa i kosinusa.

✓ Konačan odgovor

\(\displaystyle \text{Broj rešenja} = 2\)

Rešenja su: \(x_1 = \dfrac{\pi}{2}\) i \(x_2 = \dfrac{7\pi}{6}\)

Verifikacija

Proverimo oba rešenja uvrštavanjem u originalnu jednačinu \(\sin x - \sqrt{3}\cos x = 1\):

  • \(x = \frac{\pi}{2}\): \(\sin\frac{\pi}{2} = 1\), \(\cos\frac{\pi}{2} = 0\). Dakle \(1 - 0 = 1\). ✓
  • \(x = \frac{7\pi}{6}\): \(\sin\frac{7\pi}{6} = -\frac{1}{2}\), \(\cos\frac{7\pi}{6} = -\frac{\sqrt{3}}{2}\). Dakle \(-\frac{1}{2} + \frac{3}{2} = 1\). ✓

Oba rešenja zadovoljavaju jednačinu i nalaze se u intervalu \((0, 2\pi)\). Odgovor (G) 2 je tačan.

⚠ Česte greške

Pogrešan fazni pomak: Česta greška je pisanje \(\cos\frac{\pi}{3}\) umesto \(\sin\frac{\pi}{3}\) (ili obrnuto) prilikom identifikacije ugla \(\varphi\). Treba pažljivo uporediti koeficijente sa adicionom formulom.
Zaboravljanje drugog rešenja: Jednačina \(\sin\theta = \frac{1}{2}\) ima dva rešenja u jednom periodu: \(\theta = \frac{\pi}{6}\) i \(\theta = \frac{5\pi}{6}\). Ako se uzme samo prvo, dobija se pogrešan odgovor (A) 1.

★ Izazov za razmišljanje

Proširen zadatak: Za koje vrednosti parametra \(c\) jednačina \(\sin x - \sqrt{3}\cos x = c\) ima tačno jedno rešenje u intervalu \((0, 2\pi)\)?

Nagovještaj

Pošto je \(\sin x - \sqrt{3}\cos x = 2\sin(x - \frac{\pi}{3})\), jednačina postaje \(\sin(x - \frac{\pi}{3}) = \frac{c}{2}\). Razmislite kada jednačina \(\sin\theta = k\) ima tačno jedno rešenje u intervalu koji odgovara \((0, 2\pi)\) nakon pomaka. To se dešava kada je \(|k| = 1\), tj. \(c = \pm 2\).
