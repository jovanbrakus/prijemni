Kompozicija funkcije — nalaženje \(f(f(\tfrac{1}{2}))\)

Funkcionalne jednačine i supstitucija argumenata

Zadatak

Ako je

\[ f\!\left(\frac{x-1}{x+1}\right) = x \]

onda je \(f(f(\tfrac{1}{2}))\) jednako:

\(2\)

(В)

План решавања

Прво ћемо из дате функционалне једначине извести експлицитну формулу за \(f(t)\). Затим ћемо два пута применити ту формулу: најпре израчунати \(f(\tfrac{1}{2})\), а потом и \(f\) од тог резултата.

📐 Кључне формуле и концепти
  • Функционална једначина: Ако имамо \(f(g(x)) = x\), функција \(f\) је инверзна функцији \(g\). Тада је \(f = g^{-1}\).
  • Инверзна супституција: Ако је \(t = g(x) = \dfrac{x-1}{x+1}\), онда решавањем по \(x\) добијамо \(x = g^{-1}(t)\), што је управо \(f(t)\).
  • Композиција функција: \((f \circ f)(a) = f(f(a))\) значи „примени \(f\) два пута узастопно".
  • Мебијусова (Мöбиус) трансформација: Функција облика \(\dfrac{at+b}{ct+d}\) има занимљиве особине при итерацији — двоструко примењена често даје изненађујуће једноставну формулу.

Графички приказ

Функција \(f(t) = \dfrac{1+t}{1-t}\) и путања композиције: \(\tfrac{1}{2} \to 3 \to -2\)

График функције \(f(t) = \frac{1+t}{1-t}\) са означеним тачкама

Решење — корак по корак

1

Увођење смене

Означимо аргумент функције \(f\) са \(t\):

\[ t = \frac{x-1}{x+1} \]

Тада је по задатку \(f(t) = x\). Потребно је изразити \(x\) преко \(t\).

2

Решавање по \(x\)

Полазимо од \(t = \dfrac{x-1}{x+1}\) и множимо обе стране са \((x+1)\):

\[ t(x+1) = x - 1 \]

Раскланпамо леву страну:

\[ tx + t = x - 1 \]

Пребацимо чланове са \(x\) на једну страну:

\[ tx - x = -1 - t \]

Изнесемо \(x\) испред заграде:

\[ x(t - 1) = -(1 + t) \]

Поделимо са \((t-1)\), уз услов \(t \neq 1\):

\[ x = \frac{-(1+t)}{t-1} = \frac{1+t}{1-t} \]
3

Експлицитна формула за \(f\)

Пошто је \(f(t) = x\), добијамо:

\[ \boxed{f(t) = \frac{1+t}{1-t}}, \quad t \neq 1 \]

Ово је Мебијусова трансформација са вертикалном асимптотом у \(t = 1\).

🧠 Провери себе — Колико је \(f(0)\)?
\(f(0) = \dfrac{1+0}{1-0} = 1\). Ово можемо и проверити: ако ставимо \(x = 1\) у полазну формулу, добијамо \(\dfrac{1-1}{1+1} = 0\), па \(f(0) = 1\). ✓
4

Рачунамо \(f\!\left(\tfrac{1}{2}\right)\)

Заменимо \(t = \dfrac{1}{2}\) у формулу:

\[ f\!\left(\frac{1}{2}\right) = \frac{1 + \frac{1}{2}}{1 - \frac{1}{2}} = \frac{\frac{3}{2}}{\frac{1}{2}} = 3 \]

Дакле, \(f\!\left(\tfrac{1}{2}\right) = 3\).

5

Рачунамо \(f(3)\)

Сада заменимо \(t = 3\) у исту формулу:

\[ f(3) = \frac{1 + 3}{1 - 3} = \frac{4}{-2} = -2 \]

Дакле, \(f(3) = -2\).

🧠 Провери себе — Да ли се слаже са полазном дефиницијом?

Проверимо \(f(3) = -2\) помоћу оригиналне формуле. Ставимо \(x = -2\):

\(\dfrac{x-1}{x+1} = \dfrac{-2-1}{-2+1} = \dfrac{-3}{-1} = 3\) ✓

И заиста, \(f(3) = x = -2\). Све се слаже!

6

Коначан резултат

Спајамо оба корака:

\[ f\!\left(f\!\left(\frac{1}{2}\right)\right) = f(3) = -2 \]

Кључни увид

Занимљиво својство ове функције: за двоструку композицију важи

\[ f(f(t)) = -\frac{1}{t} \]

Зато је \(f\!\left(f\!\left(\tfrac{1}{2}\right)\right) = -\dfrac{1}{\,\frac{1}{2}\,} = -2\). Нема потребе за рачунањем међурезултата!

Коначан одговор

\( f\!\left(f\!\left(\tfrac{1}{2}\right)\right) = -2 \)
(А) \(2\)
(Б) \(1\)
(В) \(0\)
(Г) \(-1\)
(Д) \(-2\)
Верификација: Проверимо користећи оригиналну дефиницију. За \(x = -2\): \(\dfrac{-2-1}{-2+1} = \dfrac{-3}{-1} = 3\), па \(f(3) = -2\). За \(x = 3\): \(\dfrac{3-1}{3+1} = \dfrac{2}{4} = \dfrac{1}{2}\), па \(f\!\left(\tfrac{1}{2}\right) = 3\). Комбинујући: \(f(f(\tfrac{1}{2})) = f(3) = -2\). ✓

Честе грешке

  • Замена \(f\!\left(\dfrac{x-1}{x+1}\right) = x\) директно са \(f(x) = x\). Аргумент функције \(f\) није \(x\), већ \(\dfrac{x-1}{x+1}\). Потребно је инвертовати трансформацију.
  • Грешка у алгебри приликом решавања по \(x\) — нарочито погрешан знак. Пазите: \(x(t-1) = -(1+t)\), делимо са \((t-1)\) и добијамо \(\dfrac{1+t}{1-t}\), не \(\dfrac{1+t}{t-1}\).

Додатни изазов

Докажите да за ову функцију \(f\) важи \(f(f(f(f(t)))) = t\), тј. четвороструком применом враћамо се на полазну вредност. Који је \(f(f(f(\tfrac{1}{2})))\)?

💡 Наговештај

Искористите да је \(f(f(t)) = -\dfrac{1}{t}\). Шта се дешава кад то примените још једном?

\(f(f(f(f(t)))) = f(f(-\tfrac{1}{t})) = -\dfrac{1}{\,-\frac{1}{t}\,} = t\). Дакле \(f(f(f(\tfrac{1}{2}))) = f(-2) = \dfrac{1+(-2)}{1-(-2)} = \dfrac{-1}{3} = -\dfrac{1}{3}\).
