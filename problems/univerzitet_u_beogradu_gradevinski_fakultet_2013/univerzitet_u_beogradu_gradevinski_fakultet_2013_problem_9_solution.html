Kompleksni brojevi: jednačina sa modulom i konjugatom

Algebra kompleksnih brojeva — razdvajanje na realni i imaginarni deo, analiza uslova postojanja rešenja

Postavka zadatka

План решавања

Записаћемо \(|z + 3|\) и \(\bar{z}\) преко \(x\) и \(y\), раздвојити једначину на реални и имагинарни део, а затим проверити да ли добијени кандидат за решење задовољава услов постојања (ненегативност корена). Ако не задовољава — решења нема.

Кључне формуле и теореме
  • Konjugat: Ako je \(z = x + iy\), tada je \(\bar{z} = x - iy\)
  • Modul: \(|z| = \sqrt{x^2 + y^2}\) — uvek je realan i nenegativan broj
  • Jednakost kompleksnih brojeva: \(a + bi = c + di \iff a = c \text{ i } b = d\)
  • Kvadriranje jednačina sa korenom: Ako je \(\sqrt{f(x)} = g(x)\), tada mora važiti \(g(x) \geq 0\) i \(f(x) = [g(x)]^2\). Obavezno proveriti dobijeno rešenje!

Визуелни приказ

Комплексна раван: приказ кандидат-решења и разлог зашто не задовољава једначину

Решење корак по корак

1

Записујемо све преко \(x\) и \(y\)

Нека је \(z = x + iy\), где су \(x, y \in \mathbb{R}\). Тада:

\[\bar{z} = x - iy\]
\[z + 3 = (x + 3) + iy\]
\[|z + 3| = \sqrt{(x+3)^2 + y^2}\]

Модул \(|z + 3|\) је реалан и ненегативан број — нема имагинарну компоненту.

2

Уврштавамо у једначину

Једначина \(|z + 3| - \bar{z} = 2 - i\) постаје:

\[\sqrt{(x+3)^2 + y^2} - (x - iy) = 2 - i\]

Лева страна, раздвојена на реални и имагинарни део:

\[\underbrace{\bigl(\sqrt{(x+3)^2 + y^2} - x\bigr)}_{\text{реални део}} + \underbrace{y}_{\text{имагинарни део}} \cdot i = 2 - i\]
3

Изједначавамо имагинарне делове

Имагинарни део леве стране мора бити једнак имагинарном делу десне стране:

\[y = -1\]

Овим смо одмах одредили да свако евентуално решење мора имати \(y = -1\).

Провери се: Зашто \(|z + 3|\) не доприноси имагинарном делу?

Модул комплексног броја је по дефиницији \(\sqrt{a^2 + b^2}\), што је увек реалан ненегативан број. Зато он учествује само у реалном делу леве стране.

4

Изједначавамо реалне делове

Замењујемо \(y = -1\) у реални део:

\[\sqrt{(x+3)^2 + (-1)^2} - x = 2\]
\[\sqrt{(x+3)^2 + 1} - x = 2\]

Изолујемо корен:

\[\sqrt{(x+3)^2 + 1} = x + 2\]
5

Анализирамо услов постојања

Левa страна, \(\sqrt{(x+3)^2 + 1}\), увек је ненегативна (корен). Да би једнакост важила, десна страна такође мора бити ненегативна:

\[x + 2 \geq 0 \implies x \geq -2\]

Ово је обавезан услов — ако га кандидат-решење не задовољава, оно се одбацује.

Провери се: Зашто мора \(x + 2 \geq 0\)?

Квадратни корен \(\sqrt{(x+3)^2 + 1}\) је увек \(\geq 0\). Ако би \(x + 2 < 0\), имали бисмо ненегативну леву страну једнаку негативној десној, што је немогуће.

6

Квадрирамо обе стране

Под претпоставком да је \(x \geq -2\), квадрирање је легитимно (обе стране ненегативне):

\[(x+3)^2 + 1 = (x+2)^2\]

Развијамо:

\[x^2 + 6x + 9 + 1 = x^2 + 4x + 4\]

\(x^2\) се скраћује, остаје:

\[6x + 10 = 4x + 4\]
\[2x = -6\]
\[x = -3\]
7

Провера услова — кључни тренутак!

Добили смо јединог кандидата \(x = -3\). Да ли задовољава услов \(x \geq -2\)?

❌

\(x = -3 < -2\) — услов НИЈЕ задовољен!

Дакле, \(x = -3\) се одбацује. Кандидат-решење не постоји.

8

Директна провера (\(x = -3, \, y = -1\))

За потпуност, уврстимо \(z = -3 - i\) у оригиналну једначину:

\[|z + 3| = |(-3 + 3) + (-1)i| = |{-i}| = 1\]
\[\bar{z} = -3 + i\]
\[|z + 3| - \bar{z} = 1 - (-3 + i) = 1 + 3 - i = 4 - i\]

Добијамо \(4 - i\), а тражимо \(2 - i\). Очигледно:

\[4 - i \neq 2 - i\]

Потврђено: \(z = -3 - i\) не задовољава полазну једначину.

Провери се: Зашто квадрирање може да „произведе" лажно решење?

Квадрирање није еквивалентна трансформација — оно може увести лажна (паразитна) решења. Зато увек морамо проверити свако решење у оригиналној једначини (или бар проверити услов ненегативности пре квадрирања).

Кључни увид

Иако алгебарски поступак даје кандидата \(x = -3\), он пада на провери услова \(x \geq -2\). Квадрирање једначине са кореном може увести паразитна решења — зато је провера обавезна. У овом случају, ниједан комплексан број не задовољава задату једначину.

\[\underbrace{x = -3}_{\text{кандидат}} \;\not\geq\; \underbrace{-2}_{\text{услов}} \implies \text{број решења} = 0\]

Верификација: Из имагинарног дела добијамо \(y = -1\). Из реалног дела добијамо \(\sqrt{(x+3)^2 + 1} = x + 2\), што захтева \(x \geq -2\). Квадрирањем добијамо \(x = -3\), али \(-3 < -2\), па кандидат не задовољава услов. Директна провера: \(|{-3-i+3}| - \overline{(-3-i)} = 1-(-3+i) = 4-i \neq 2-i\). Заиста нема решења.

Честе грешке

Заборавити проверу услова ненегативности: Многи ученици квадрирају обе стране, добију \(x = -3\), уврсте \(y = -1\) и закључе да постоји једно решење. Али квадрирање може увести паразитна решења — обавезно проверити!
Погрешно раздвајање на реални и имагинарни део: Модул \(|z + 3|\) је реалан број и не доприноси имагинарном делу. Грешка \(-iy\) у знаку (заборавити минус испред \(\bar{z}\)) такође може довести до погрешног \(y\).

Додатни изазов

Шта ако би једначина гласила \(|z + 3| - \bar{z} = 4 - i\)? Колико решења има тада?

Наговештај

Поступак је исти: из имагинарног дела добијамо \(y = -1\), а из реалног дела \(\sqrt{(x+3)^2 + 1} = x + 4\), што захтева \(x \geq -4\). Квадрирањем добијамо \(x = -3\). Овог пута \(-3 \geq -4\), па кандидат пролази проверу. Проверите и у оригиналној једначини!
