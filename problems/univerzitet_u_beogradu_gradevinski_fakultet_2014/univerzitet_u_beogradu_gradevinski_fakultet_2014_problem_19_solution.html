Analiza jednačine koja kombinuje trigonometrijsku funkciju, apsolutnu vrednost i linearnu funkciju

Plan rešavanja

Razdvojićemo jednačinu na dva slučaja prema znaku \(\cos x\): kada je \(\cos x \geq 0\) i kada je \(\cos x < 0\). U svakom slučaju dobijamo jednostavniju jednačinu čiji broj rešenja možemo grafički i analitički odrediti.

📐 Ključne formule i pojmovi
  • Apsolutna vrednost: \( |a| = \begin{cases} a, & a \geq 0 \\ -a, & a < 0 \end{cases} \)
  • Osobina: \( a + |a| = \begin{cases} 2a, & a \geq 0 \\ 0, & a < 0 \end{cases} \)
  • Grafičko rešavanje: Broj rešenja jednačine \(f(x) = g(x)\) jednak je broju presečnih tačaka grafika funkcija \(f\) i \(g\).
  • Kosinus funkcija: \(\cos x \geq 0\) za \(x \in \left[-\frac{\pi}{2} + 2k\pi,\; \frac{\pi}{2} + 2k\pi\right]\), gde je \(k \in \mathbb{Z}\).

Grafički prikaz

Grafik prikazuje levu stranu (plava) i desnu stranu (zelena) jednačine. Presečne tačke su rešenja.

Rešenje korak po korak

1

Razmatramo levu stranu jednačine

Leva strana je \(\cos x + |\cos x|\). Koristimo osobinu \(a + |a|\):

\[ \cos x + |\cos x| = \begin{cases} 2\cos x, & \cos x \geq 0 \\ 0, & \cos x < 0 \end{cases} \]

Ovo je ključna osobina: zbir broja i njegove apsolutne vrednosti daje ili dvostruku vrednost (ako je nenegativan) ili nulu (ako je negativan).

2

Slučaj 1: \(\cos x < 0\)

Kada je \(\cos x < 0\), leva strana je jednaka \(0\), pa jednačina postaje:

\[ 0 = 2 - \frac{2}{\pi}x \]

Rešavamo po \(x\):

\[ \frac{2}{\pi}x = 2 \implies x = \pi \]

Proveravamo uslov: \(\cos \pi = -1 < 0\) — uslov je ispunjen.

Verifikacija: LHS \(= \cos\pi + |\cos\pi| = -1 + 1 = 0\), RHS \(= 2 - \frac{2}{\pi}\cdot\pi = 0\). ✓

Iz ovog slučaja dobijamo tačno jedno rešenje: \(x = \pi\).

🔍 Proveri sebe: Koliko rešenja daje slučaj \(\cos x < 0\)?

Tačno jedno: \(x = \pi\). Jednačina \(0 = 2 - \frac{2}{\pi}x\) ima jedinstveno rešenje \(x = \pi\), i ono zadovoljava uslov \(\cos\pi < 0\).

3

Slučaj 2: \(\cos x \geq 0\)

Kada je \(\cos x \geq 0\), leva strana je \(2\cos x\), pa jednačina postaje:

\[ 2\cos x = 2 - \frac{2}{\pi}x \]

Delimo obe strane sa \(2\):

\[ \cos x = 1 - \frac{x}{\pi} \]

Tražimo presečne tačke grafika funkcije \(y = \cos x\) i prave \(y = 1 - \frac{x}{\pi}\), uz dodatni uslov \(\cos x \geq 0\).

4

Analiza prave \(y = 1 - \frac{x}{\pi}\)

Ova prava prolazi kroz tačke:

\[ (0,\;1) \quad \text{i} \quad (\pi,\;0) \]

Za \(x < 0\): prava daje \(y > 1\), ali \(\cos x \leq 1\) sa jednakošću samo u \(x = 2k\pi\). Za \(x < 0\) najbliža tačka gde \(\cos x = 1\) je \(x = -2\pi\), ali \(1 - \frac{-2\pi}{\pi} = 3 \neq 1\). Dakle, za \(x < 0\) prava je strogo iznad kosinusa — nema rešenja.

Za \(x > \pi\): prava daje \(y < 0\), ali uslov zahteva \(\cos x \geq 0\), što bi značilo \(\cos x = 1 - \frac{x}{\pi} < 0\) — kontradikcija, pa nema rešenja.

Rešenja u ovom slučaju mogu postojati samo za \(x \in [0, \pi]\).

5

Presečne tačke na \([0, \pi]\)

Definišimo \(h(x) = \cos x - 1 + \frac{x}{\pi}\). Tražimo nule ove funkcije.

Tačka \(x = 0\):

\[ h(0) = \cos 0 - 1 + 0 = 1 - 1 = 0 \]

Dakle, \(x = 0\) je rešenje. Proveravamo: \(\cos 0 = 1 \geq 0\) ✓

6

Ponašanje funkcije \(h(x)\) na \((0, \pi)\)

Ispitujemo izvod:

\[ h'(x) = -\sin x + \frac{1}{\pi} \]

Za malo \(x > 0\): \(\sin x \approx 0\), pa je \(h'(0) = \frac{1}{\pi} > 0\). Funkcija \(h\) raste od nule — dakle \(h(x) > 0\) blizu nule.

Proverimo vrednost u tački \(x = \frac{\pi}{2}\):

\[ h\!\left(\frac{\pi}{2}\right) = \cos\frac{\pi}{2} - 1 + \frac{1}{2} = 0 - 1 + 0{,}5 = -0{,}5 < 0 \]

Pošto je \(h(0) = 0\), \(h\) raste za malo \(x > 0\) (postaje pozitivna), a zatim u \(x = \frac{\pi}{2}\) postaje negativna, po teoremi o međuvrednosti postoji tačno jedna nula \(x_1 \in \left(0, \frac{\pi}{2}\right)\).

🔍 Proveri sebe: Zašto nema više nula na \(\left(\frac{\pi}{2}, \pi\right]\)?

Na intervalu \(\left(\frac{\pi}{2}, \pi\right)\) važi \(\cos x < 0\), što znači da uslov \(\cos x \geq 0\) nije ispunjen. Dakle, čak i ako bi se grafici ponovo presekli, ta presečna tačka ne bi pripadala ovom slučaju.

7

Jedinstvenost nule na \(\left(0, \frac{\pi}{2}\right)\)

Da pokažemo da je nula jedinstvena, posmatramo drugi izvod:

\[ h''(x) = -\cos x \]

Na intervalu \(\left(0, \frac{\pi}{2}\right)\) važi \(\cos x > 0\), pa je \(h''(x) < 0\) — funkcija \(h\) je konkavna. Konkavna funkcija koja kreće od pozitivne vrednosti i pada na negativnu može imati najviše jednu nulu na tom intervalu.

Slučaj 2 daje tačno dva rešenja: \(x = 0\) i \(x_1 \approx 0{,}66\).

8

Ukupan broj rešenja

Sabiramo rešenja iz oba slučaja:

\[ \underbrace{x = 0, \; x_1 \approx 0{,}66}_{\text{Slučaj 2: } \cos x \geq 0} + \underbrace{x = \pi}_{\text{Slučaj 1: } \cos x < 0} = \mathbf{3} \text{ rešenja} \]
🔍 Proveri sebe: Verifikuj rešenje \(x = 0\)

LHS: \(\cos 0 + |\cos 0| = 1 + 1 = 2\). RHS: \(2 - \frac{2}{\pi}\cdot 0 = 2\). Jednako! ✓

Ključni uvid

Osobina \(a + |a| = \begin{cases} 2a, & a \geq 0 \\ 0, & a < 0 \end{cases}\) razbija jednačinu na dva potpuno različita problema: linearnu jednačinu (trivijalno rešivo) i transcendentnu jednačinu (grafička analiza).

Konačan odgovor

\( \text{Broj rešenja} = 3 \)
(A) 0
(B) 1
(V) 2
(G) 3
(D) 5
Verifikacija: Tri rešenja su \(x = 0\), \(x_1 \approx 0{,}66\) i \(x = \pi\). Svako možemo numerički proveriti:
  • \(x = 0\): LHS \(= 1 + 1 = 2\), RHS \(= 2 - 0 = 2\) ✓
  • \(x = \pi\): LHS \(= -1 + 1 = 0\), RHS \(= 2 - 2 = 0\) ✓
  • \(x_1\): grafički potvrđeno presecanje između \(0\) i \(\frac{\pi}{2}\) ✓

Česte greške

  • Zaboraviti \(x = 0\): Mnogi studenti previde da je \(x = 0\) rešenje jer se ono nalazi na granici intervala. Obavezno proverite rubne tačke!
  • Ne razdvojiti slučajeve: Bez razdvajanja po znaku \(\cos x\), jednačina sa apsolutnom vrednošću je teško rešiva. Uvek razbijte apsolutnu vrednost na slučajeve.
