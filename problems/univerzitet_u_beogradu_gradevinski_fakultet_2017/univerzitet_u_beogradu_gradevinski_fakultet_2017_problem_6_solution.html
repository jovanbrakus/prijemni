📌 Plan resavanja
  1. Izraziti svaki clan niza pomocu prvog clana \(a_1\) i razlike \(d\), pa formirati sistem od dve jednacine
  2. Resiti sistem, odrediti \(a_1\) i \(d\), pa izracunati \(a_4 = a_1 + 3d\)
Kljucne formule i teoreme

Opsti clan aritmetickog niza:

\[a_n = a_1 + (n - 1) \cdot d\]

gde je \(a_1\) prvi clan, a \(d\) razlika niza (konstantan korak izmedju uzastopnih clanova).

  • \(a_2 = a_1 + d\), \(\;a_3 = a_1 + 2d\), \(\;a_4 = a_1 + 3d\), \(\;\ldots\)
  • Razlika niza: \(d = a_{n+1} - a_n\) za svako \(n\)
  • Srednji clan: \(a_n = \dfrac{a_{n-1} + a_{n+1}}{2}\) (svaki clan je aritmeticka sredina svojih suseda)

Korisno svojstvo: Za clanove aritmetickog niza ciji su indeksi simetricni u odnosu na neki srednji indeks, zbir je isti. Na primer, \(a_2 + a_6 = a_3 + a_5 = 2a_4\).

✏️ Resenje korak po korak
Korak 1: Zapisati opsti clan niza

Opsti clan aritmetickog niza je:

\[a_n = a_1 + (n - 1) \cdot d\]

gde je \(a_1\) prvi clan, a \(d\) razlika niza.

Korak 2: Izraziti konkretne clanove

Primenom formule za svaki clan koji se pojavljuje u uslovima:

\[a_2 = a_1 + d\] \[a_3 = a_1 + 2d\] \[a_5 = a_1 + 4d\] \[a_7 = a_1 + 6d\]
Korak 3: Formirati sistem jednacina

Zamenimo u date uslove:

Iz uslova \(a_2 + a_5 = 8\):

\[(a_1 + d) + (a_1 + 4d) = 8\] \[2a_1 + 5d = 8 \quad \ldots (1)\]

Iz uslova \(a_3 + a_7 = 32\):

\[(a_1 + 2d) + (a_1 + 6d) = 32\] \[2a_1 + 8d = 32 \quad \ldots (2)\]
Proveri se: Koliko jednacina imamo i koliko nepoznatih?

Imamo 2 jednacine sa 2 nepoznate (\(a_1\) i \(d\)). To je dovoljno za jednoznacno resenje sistema!

Korak 4: Odrediti razliku \(d\)

Oduzmimo jednacinu (1) od jednacine (2):

\[(2a_1 + 8d) - (2a_1 + 5d) = 32 - 8\]

Clanovi sa \(a_1\) se poniste:

\[3d = 24\] \[d = 8\]
Korak 5: Odrediti prvi clan \(a_1\)

Zamenimo \(d = 8\) u jednacinu (1):

\[2a_1 + 5 \cdot 8 = 8\] \[2a_1 + 40 = 8\] \[2a_1 = 8 - 40 = -32\] \[a_1 = -16\]
Proveri se: Koliko iznosi \(a_1\)?

\(a_1 = -16\). Ako si dobio/la drugi rezultat, proveri da li si korektno oduzeo/la jednacine i zamenio/la \(d = 8\).

Korak 6: Izracunati \(a_4\)

Sada imamo sve sto nam treba:

\[a_4 = a_1 + 3d = -16 + 3 \cdot 8 = -16 + 24 = 8\]
Korak 7: Verifikacija

Proverimo oba polazna uslova sa \(a_1 = -16\) i \(d = 8\):

\[a_2 = -16 + 8 = -8, \quad a_5 = -16 + 32 = 16\] \[a_2 + a_5 = -8 + 16 = 8 \;\; \checkmark\]
\[a_3 = -16 + 16 = 0, \quad a_7 = -16 + 48 = 32\] \[a_3 + a_7 = 0 + 32 = 32 \;\; \checkmark\]

Oba uslova su zadovoljena.

Proveri se: Koji je peti clan niza?

\(a_5 = a_1 + 4d = -16 + 32 = 16\). Primetimo da je \(a_5\) tacno duplo vece od \(a_4 = 8\), sto ima smisla jer \(a_5 = a_4 + d = 8 + 8 = 16\).

💡 Kljucni uvid

Kada imate dva uslova oblika "zbir dva clana = nesto", uvek zapisite svaki clan pomocu \(a_1\) i \(d\). Dobicete sistem sa dve nepoznate koji se resava oduzimanjem jednacina — clanovi sa \(a_1\) se poniste, a ostaje samo \(d\).

U ovom zadatku, kljucan korak je bio oduzimanje jednacina koje eliminise \(a_1\) i direktno daje:

\[3d = 24 \quad \Rightarrow \quad d = 8\]

Verifikacija: Niz glasi \(-16, -8, 0, \mathbf{8}, 16, 24, \ldots\) sa razlikom \(d = 8\). Uslovi \(a_2 + a_5 = -8 + 16 = 8\) i \(a_3 + a_7 = 0 + 32 = 32\) su oba zadovoljena.

⚠️ Ceste greske
Pogresan indeks u formuli: Cesta greska je pisati \(a_5 = a_1 + 5d\) umesto \(a_5 = a_1 + 4d\). Zapamtite: \(a_n = a_1 + (n-1)d\), pa za \(n = 5\) ide \((5-1)d = 4d\), ne \(5d\).
Greska u znaku pri oduzimanju: Kada oduzimate jednacine, pazite na znakove. Oduzimanje \((2a_1 + 5d)\) od \((2a_1 + 8d)\) daje \(3d\), a ne \(-3d\). Takodje, na desnoj strani: \(32 - 8 = 24\), ne obrnuto.
🏆 Dodatni izazov

Za isti aritmeticki niz, izracunaj zbir prvih 7 clanova, \(S_7 = a_1 + a_2 + \ldots + a_7\).

Prikazi nagovestaj

Koristi formulu \(S_n = \dfrac{n}{2}(2a_1 + (n-1)d)\). Za \(n = 7\), \(a_1 = -16\), \(d = 8\):

\(S_7 = \dfrac{7}{2}(2 \cdot (-16) + 6 \cdot 8) = \dfrac{7}{2}(-32 + 48) = \dfrac{7}{2} \cdot 16 = 56\).
