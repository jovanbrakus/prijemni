Algebra — Polinomi, deljivost i uslovi za vishestruke korene

Postavka zadatka

Plan reshavanja

Primetimo da je \(Q(x) = (x+1)^2\), pa je \(x = -1\) dvostruki koren delioca. Ako \(P(x)\) deli \(Q(x)\), onda mora da vazhi \(P(-1) = 0\) i \(P'(-1) = 0\). Iz ova dva uslova odredichemo \(a\) i \(b\), a zatim izrachunati \(4a + b\).

Kljuchne formule i teoreme
  • Potpun kvadrat: Kvadratni trinom \(x^2 + 2x + 1\) je potpun kvadrat binoma: \[x^2 + 2x + 1 = (x + 1)^2\]
  • Uslov deljivosti polinoma: Ako je polinom \(P(x)\) deljiv sa \((x - c)\), onda je \(P(c) = 0\). Ovo je posledica Bezuove teoreme (teorema o ostatku): \[P(x) = (x - c) \cdot Q_1(x) \implies P(c) = 0\]
  • Dvostruki koren i izvod: Ako je \(x = c\) dvostruki koren polinoma \(P(x)\), tj. \(P(x) = (x - c)^2 \cdot R(x)\), onda vazhi: \[P(c) = 0 \quad \text{i} \quad P'(c) = 0\] Ovo sledi jer je \(P'(x) = 2(x - c) R(x) + (x - c)^2 R'(x)\), pa \(P'(c) = 0\).
  • Opshte pravilo: Ako je \(x = c\) koren kratnosti \(k\) polinoma \(P(x)\), onda: \[P(c) = P'(c) = P''(c) = \cdots = P^{(k-1)}(c) = 0\]

Reshavanje korak po korak

1

Faktorizacija delioca \(Q(x)\)

Prepoznajemo da je \(Q(x)\) potpun kvadrat:

\[Q(x) = x^2 + 2x + 1 = (x + 1)^2\]

Dakle, \(x = -1\) je dvostruki koren polinoma \(Q(x)\).

2

Shta znachi deljivost?

Ako je \(P(x)\) deljiv sa \(Q(x) = (x + 1)^2\), to znachi da postoji polinom \(R(x)\) takav da:

\[P(x) = (x + 1)^2 \cdot R(x)\]

Poshto je \(x = -1\) dvostruki koren faktora \((x+1)^2\), on mora biti i dvostruki koren polinoma \(P(x)\). To nam daje dva uslova:

\[P(-1) = 0 \quad \text{i} \quad P'(-1) = 0\]
Proveri sebe: Zashto iz \(P(x) = (x+1)^2 R(x)\) sledi \(P'(-1) = 0\)?
Diferenciranjem: \(P'(x) = 2(x+1)R(x) + (x+1)^2 R'(x)\). Za \(x = -1\): oba sabirka sadrzhre faktor \((x+1)\), pa je \(P'(-1) = 0\).
3

Uslov \(P(-1) = 0\)

Uvrshtavamo \(x = -1\) u \(P(x) = x^4 + ax^2 + 4x + b\):

\[P(-1) = (-1)^4 + a \cdot (-1)^2 + 4 \cdot (-1) + b\]

Izrachunajmo svaki clan:

\[P(-1) = 1 + a - 4 + b = a + b - 3\]

Iz uslova \(P(-1) = 0\):

\[a + b - 3 = 0 \implies a + b = 3 \quad \cdots (1)\]
4

Nalazimo izvod \(P'(x)\)

Diferenciramo polinom \(P(x) = x^4 + ax^2 + 4x + b\):

\[P'(x) = 4x^3 + 2ax + 4\]

Primetite: \(b\) je konstanta, pa nestaje pri diferenciranju, a \(ax^2\) daje \(2ax\).

5

Uslov \(P'(-1) = 0\)

Uvrshtavamo \(x = -1\) u izvod:

\[P'(-1) = 4 \cdot (-1)^3 + 2a \cdot (-1) + 4\]

Izrachunajmo:

\[P'(-1) = -4 - 2a + 4 = -2a\]

Iz uslova \(P'(-1) = 0\):

\[-2a = 0 \implies a = 0 \quad \cdots (2)\]
Proveri sebe: Ako je \(a = 0\), koliko je \(b\) iz jednachine (1)?
Iz \(a + b = 3\) i \(a = 0\) sledi: \(0 + b = 3\), dakle \(b = 3\).
6

Odredjivanje parametara

Iz jednachine (2) imamo \(a = 0\). Uvrshtavamo u jednachinu (1):

\[0 + b = 3 \implies b = 3\]

Sada rachunamo trazhenu vrednost:

\[4a + b = 4 \cdot 0 + 3 = 3\]
7

Verifikacija: deljivost polinoma

Sa \(a = 0\) i \(b = 3\) nashrecimo polinom je \(P(x) = x^4 + 4x + 3\). Proverimo da je zaista deljiv sa \((x+1)^2\) deljenjem:

\[x^4 + 4x + 3 = (x^2 + 2x + 1)(x^2 - 2x + 3)\]

Proverimo mnozhrenjem:

\[\begin{align} &(x^2 + 2x + 1)(x^2 - 2x + 3) \\ &= x^4 - 2x^3 + 3x^2 + 2x^3 - 4x^2 + 6x + x^2 - 2x + 3 \\ &= x^4 + 0 \cdot x^3 + 0 \cdot x^2 + 4x + 3 \quad \checkmark \end{align}\]

Deljivost je potvrdjena. Polinom \(R(x) = x^2 - 2x + 3\) ima diskriminantu \(D = 4 - 12 = -8 < 0\), pa nema realnih korena.

Proveri sebe: Koliko realnih korena ima polinom \(P(x) = x^4 + 4x + 3\)?
Polinom \(P(x) = (x+1)^2(x^2 - 2x + 3)\). Faktor \((x+1)^2\) daje dvostruki koren \(x = -1\). Faktor \(x^2 - 2x + 3\) nema realnih korena (diskriminanta \(D = -8 < 0\)). Dakle, \(P(x)\) ima samo jedan realan koren: \(x = -1\) (kratnosti 2).

Kljuchni uvid

Dvostruki koren polinoma ponishrtava i sam polinom i njegov prvi izvod. Ova dva uslova daju sistem od dve jednachine sa dve nepoznate:

\[P(-1) = 0 \implies a + b = 3\]
\[P'(-1) = 0 \implies a = 0\]

Cheste greshke

  • Zaboravljanje uslova za izvod: mnogi studenti koriste samo \(P(-1) = 0\), chime dobijaju jednu jednachinu sa dve nepoznate i ne mogu jednoznachno odrediti \(a\) i \(b\). Kljuchno je prepoznati dvostruki koren i primeniti i uslov \(P'(-1) = 0\).
  • Greshka u predznaku pri izrachrunavanju \((-1)^3 = -1\) ili \((-1)^4 = 1\). Uvek pazhljivo pratite parnost eksponenta!
  • Neshto finiji problem: zadatak trazhi \(4a + b\), a ne \(a + b\). Ako pozhrurite i odgovorite \(a + b = 3\), to bi bio tachan odgovor samo sticajem okolnosti (jer je \(a = 0\), pa je \(4a + b = b = 3 = a + b\)).

Dodatni izazov

Neka je \(P(x) = x^4 + cx^3 + ax^2 + 4x + b\) deljiv sa \((x - 2)^2\). Odredite \(a\), \(b\) i \(c\).

Nagovestaj
Primenite isti metod: iz \(P(2) = 0\) i \(P'(2) = 0\) dobijate dve jednachine. Ali sada imate tri nepoznate — trebache vam josh jedan uslov. Razmosite i drugi izvod ili pokushajte konkretno deljenje.
