Kompleksni brojevi — Razdvajanje na realni i imaginarni deo

Postavka zadatka

Ako je \(z = x + iy\) kompleksan broj takav da je \(|z + 1| - \bar{z} = 2 - i\), onda je \(x + y\) jednako:

Plan rešavanja

Izrazićemo \(|z + 1|\) i \(\bar{z}\) preko \(x\) i \(y\), a zatim ćemo razdvojiti jednačinu na realni i imaginarni deo. Iz imaginarnog dela direktno dobijamo \(y\), a iz realnog dela nalazimo \(x\).

📐 Ključne formule i teoreme
  • Kompleksan broj: \(z = x + iy\), gde su \(x = \operatorname{Re}(z)\) i \(y = \operatorname{Im}(z)\) realni brojevi, a \(i^2 = -1\).
  • Konjugovano kompleksan broj: \(\bar{z} = x - iy\). Menja se samo predznak imaginarnog dela.
  • Modul kompleksnog broja: \(|z| = \sqrt{x^2 + y^2}\). Modul je uvek realan nenegativan broj.
  • Jednakost kompleksnih brojeva: Dva kompleksna broja su jednaka ako i samo ako su im jednaki i realni i imaginarni delovi: \[a + bi = c + di \iff a = c \text{ i } b = d\]

Vizuelni prikaz

Kompleksna ravan sa tačkom \(z = -1 - i\) i modulom \(|z + 1|\)

Rešenje korak po korak

1

Zapišimo sve komponente

Neka je \(z = x + iy\). Tada su:

\[\bar{z} = x - iy\]
\[z + 1 = (x + 1) + iy\]
\[|z + 1| = \sqrt{(x+1)^2 + y^2}\]

Ključna opservacija: modul \(|z + 1|\) je realan broj (nema imaginarni deo).

2

Uvrstimo u jednačinu

Polazna jednačina je \(|z + 1| - \bar{z} = 2 - i\). Uvrštavamo:

\[\sqrt{(x+1)^2 + y^2} - (x - iy) = 2 - i\]

Rasporedimo članove:

\[\bigl(\sqrt{(x+1)^2 + y^2} - x\bigr) + iy = 2 - i\]

Na levoj strani imamo: realni deo \(\sqrt{(x+1)^2 + y^2} - x\) i imaginarni deo \(y\).

🔍 Proveri sebe: Zašto je \(-(-iy) = +iy\)?
Jer \(\bar{z} = x - iy\), pa \(-\bar{z} = -x + iy\). Kada oduzmemo konjugat, imaginarni deo menja predznak u plus.
3

Razdvojimo na realni i imaginarni deo

Dva kompleksna broja su jednaka ako i samo ako su im jednaki realni i imaginarni delovi. Poredimo levu i desnu stranu:

\[\text{Im: } \quad y = -1\]
\[\text{Re: } \quad \sqrt{(x+1)^2 + y^2} - x = 2\]

Iz imaginarnog dela odmah dobijamo \(y = -1\).

4

Uvrstimo \(y = -1\) u realni deo

Zamenjujemo \(y = -1\) u jednačinu za realni deo:

\[\sqrt{(x+1)^2 + (-1)^2} - x = 2\]
\[\sqrt{(x+1)^2 + 1} = x + 2\]

Da bi ova jednačina imala smisla, desna strana mora biti nenegativna: \(x + 2 \geq 0\), tj. \(x \geq -2\).

🔍 Proveri sebe: Zašto moramo proveriti uslov \(x + 2 \geq 0\)?
Jer leva strana \(\sqrt{(x+1)^2 + 1}\) je uvek nenegativna (koren je nenegativan), pa i desna strana \(x + 2\) mora biti nenegativna. Inače jednačina nema rešenje.
5

Kvadriramo obe strane

Pošto su obe strane nenegativne, kvadriranje je ekvivalentna operacija:

\[(x+1)^2 + 1 = (x+2)^2\]

Raspisujemo obe strane:

\[x^2 + 2x + 1 + 1 = x^2 + 4x + 4\]
\[x^2 + 2x + 2 = x^2 + 4x + 4\]
6

Rešimo po \(x\)

Oduzimamo \(x^2\) sa obe strane (poništavaju se):

\[2x + 2 = 4x + 4\]

Prebacimo \(4x\) na levu, a \(2\) na desnu stranu:

\[2x - 4x = 4 - 2\]
\[-2x = 2\]
\[x = -1\]

Proveravamo uslov: \(x = -1 \geq -2\) — zadovoljeno.

7

Izračunajmo \(x + y\)

Sada znamo da je \(x = -1\) i \(y = -1\), pa je:

\[x + y = -1 + (-1) = -2\]
🔍 Proveri sebe: Da li se \(z = -1 - i\) slaže sa polaznom jednačinom?
\(z + 1 = 0 - i = -i\), pa \(|z+1| = 1\).
\(\bar{z} = -1 + i\).
\(|z+1| - \bar{z} = 1 - (-1 + i) = 1 + 1 - i = 2 - i\). Tačno!

Ključni uvid

Modul kompleksnog broja \(|z + 1|\) je uvek realan broj. Zato u jednačini \(|z + 1| - \bar{z} = 2 - i\) sav imaginarni deo dolazi isključivo od \(-\bar{z} = -x + iy\). To nam odmah daje \(y = -1\), a ostatak se svodi na obično kvadriranje iracionalne jednačine.

\[x + y = -1 + (-1) = -2\]

Konačan odgovor

\(x + y = -2\)
(A) \(-2\)
(B) \(2\)
(V) \(-4\)
(G) \(4\)
(D) \(8\)

Provera: Za \(z = -1 - i\): \(|z + 1| = |-i| = 1\), \(\bar{z} = -1 + i\). Tada je \(|z+1| - \bar{z} = 1 - (-1 + i) = 2 - i\), što je tačno jednako desnoj strani jednačine.

Česte greške

  • Zaboraviti da je modul \(|z + 1|\) realan broj i pokušati mu dodeliti imaginarni deo. Ovo je ključ za razdvajanje jednačine — imaginarni deo jednačine zavisi samo od \(\bar{z}\).
  • Pogrešan predznak kod \(-\bar{z}\): Studenti često napišu \(-\bar{z} = -(x + iy) = -x - iy\), ali konjugat je \(\bar{z} = x - iy\), pa je \(-\bar{z} = -x + iy\). Imaginarni deo je \(+y\), ne \(-y\).

Dodatni izazov

Za kompleksan broj \(z = x + iy\) važi \(|z - 2| + \bar{z} = 4 + 3i\). Odrediti \(z\).

💡 Nagovještaj

Isti pristup: modul je realan, pa iz imaginarnog dela sledi \(-y = 3\), tj. \(y = -3\). Iz realnog dela: \(\sqrt{(x-2)^2 + 9} + x = 4\), pa \(\sqrt{(x-2)^2 + 9} = 4 - x\). Kvadriraj i reši po \(x\).
