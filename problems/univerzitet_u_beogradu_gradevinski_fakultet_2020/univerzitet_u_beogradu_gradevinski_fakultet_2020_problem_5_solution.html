Zbir članova aritmetičkog niza

Algebra — Aritmetički nizovi i sistemi linearnih jednačina

Postavka zadatka

Ako je \((a_n)\) aritmetički niz takav da je:

\[a_5 + a_{10} + a_{12} = 3 \quad \text{i} \quad a_{11} + a_6 = 0\]

onda je zbir \(a_1 + a_2 + a_3\) jednak:

Plan rešavanja

Svaki član aritmetičkog niza izrazićemo pomoću prvog člana \(a_1\) i razlike niza \(d\). Iz dva data uslova dobićemo sistem od dve jednačine sa dve nepoznate, iz koga ćemo odrediti \(a_1\) i \(d\), a zatim izračunati traženi zbir.

📐 Ključne formule i teoreme
  • Opšti član aritmetičkog niza: \(a_n = a_1 + (n-1)d\), gde je \(a_1\) prvi član, a \(d\) razlika niza (konstantna razlika između uzastopnih članova).
  • Zbir prvih \(n\) članova: \(S_n = \frac{n}{2}(2a_1 + (n-1)d)\), ili ekvivalentno \(S_n = \frac{n(a_1 + a_n)}{2}\).
  • Korisno svojstvo: Zbir dva člana aritmetičkog niza na jednakoj udaljenosti od nekog člana zavisi samo od tog srednjeg člana. Na primer: \(a_k + a_m = 2a_1 + (k+m-2)d\).
  • Rešavanje sistema linearnih jednačina: Metod supstitucije — iz jedne jednačine izrazimo jednu nepoznatu i zamenimo u drugu.

Rešenje korak po korak

1

Izrazimo članove niza pomoću \(a_1\) i \(d\)

Opšti član aritmetičkog niza glasi \(a_n = a_1 + (n-1)d\). Izrazimo članove koji se pojavljuju u zadatku:

\[\begin{align} a_5 &= a_1 + 4d \\ a_6 &= a_1 + 5d \\ a_{10} &= a_1 + 9d \\ a_{11} &= a_1 + 10d \\ a_{12} &= a_1 + 11d \end{align}\]
2

Postavimo prvu jednačinu iz uslova \(a_{11} + a_6 = 0\)

Zamenimo izraze za \(a_{11}\) i \(a_6\):

\[(a_1 + 10d) + (a_1 + 5d) = 0\]

Saberemo slične članove:

\[2a_1 + 15d = 0 \quad \text{...(I)}\]
3

Postavimo drugu jednačinu iz uslova \(a_5 + a_{10} + a_{12} = 3\)

Zamenimo izraze za svaki član:

\[(a_1 + 4d) + (a_1 + 9d) + (a_1 + 11d) = 3\]

Saberemo slične članove:

\[3a_1 + 24d = 3\]

Podelimo obe strane sa 3 da bismo uprositili:

\[a_1 + 8d = 1 \quad \text{...(II)}\]
🔍 Proveri sebe: Koliko iznosi zbir indeksa u uslovu \(a_5 + a_{10} + a_{12} = 3\)?
Zbir koeficijenata uz \(d\) je \(4 + 9 + 11 = 24\), a uz \(a_1\) je \(1 + 1 + 1 = 3\). Dakle dobijamo \(3a_1 + 24d = 3\).
4

Rešimo sistem jednačina

Imamo sistem:

\[\begin{cases} 2a_1 + 15d = 0 & \text{...(I)} \\ a_1 + 8d = 1 & \text{...(II)} \end{cases}\]

Iz jednačine (II) izrazimo \(a_1\):

\[a_1 = 1 - 8d\]

Zamenimo u jednačinu (I):

\[2(1 - 8d) + 15d = 0\]

Razvijemo zagradu:

\[2 - 16d + 15d = 0\]
\[2 - d = 0\]
\[d = 2\]
5

Odredimo \(a_1\)

Vratimo \(d = 2\) u izraz za \(a_1\):

\[a_1 = 1 - 8 \cdot 2 = 1 - 16 = -15\]

Dakle, prvi član niza je \(a_1 = -15\), a razlika niza je \(d = 2\).

🔍 Proveri sebe: Da li važe polazni uslovi za \(a_1 = -15\) i \(d = 2\)?
\(a_{11} + a_6 = (-15 + 20) + (-15 + 10) = 5 + (-5) = 0\) ✓
\(a_5 + a_{10} + a_{12} = (-15 + 8) + (-15 + 18) + (-15 + 22) = -7 + 3 + 7 = 3\) ✓
6

Izračunamo traženi zbir \(a_1 + a_2 + a_3\)

Odredimo prva tri člana niza:

\[\begin{align} a_1 &= -15 \\ a_2 &= a_1 + d = -15 + 2 = -13 \\ a_3 &= a_1 + 2d = -15 + 4 = -11 \end{align}\]

Saberemo ih:

\[a_1 + a_2 + a_3 = -15 + (-13) + (-11) = -39\]

Alternativno, koristeći formulu za zbir: \(S_3 = 3a_1 + 3d = 3 \cdot (-15) + 3 \cdot 2 = -45 + 6 = -39\).

🔍 Brza provera: Koliko je \(3a_1 + 3d\) za \(a_1 = -15\) i \(d = 2\)?
\(3 \cdot (-15) + 3 \cdot 2 = -45 + 6 = -39\) ✓

Ključni uvid

Svaki uslov koji uključuje zbir članova aritmetičkog niza može se svesti na linearnu jednačinu po \(a_1\) i \(d\). Dva uslova daju sistem od dve jednačine — dovoljno za jednoznačno određivanje oba parametra niza.

\[a_1 = -15, \quad d = 2 \quad \Rightarrow \quad a_1 + a_2 + a_3 = -39\]

Konačan odgovor

\(a_1 + a_2 + a_3 = -39\)
(A) \(0\)
(B) \(-9\)
(V) \(-17\)
(G) \(-11\)
(D) \(-39\)

Provera: Za \(a_1 = -15\) i \(d = 2\), niz počinje: \(-15, -13, -11, -9, -7, -5, -3, -1, 1, 3, 5, 7, \ldots\) Proverimo uslove: \(a_5 + a_{10} + a_{12} = -7 + 3 + 7 = 3\) ✓ i \(a_{11} + a_6 = 5 + (-5) = 0\) ✓. Zbir \(-15 + (-13) + (-11) = -39\) ✓.

Česte greške

  • Pogrešno primenjivanje formule za opšti član: \(a_n = a_1 + (n-1)d\), ne \(a_n = a_1 + nd\). Na primer, \(a_5 = a_1 + 4d\), ne \(a_1 + 5d\). Greška u indeksu dovodi do pogrešnog sistema jednačina.
  • Greška pri deljenju jednačine \(3a_1 + 24d = 3\). Ako ne podelimo sa 3, sistem postaje komplikovaniji i povećava se mogućnost računske greške. Uvek uprošćavajte jednačine pre rešavanja sistema.

Dodatni izazov

Za isti aritmetički niz \((a_n)\) gde je \(a_1 = -15\) i \(d = 2\), odredite vrednost \(n\) za koju je parcijalni zbir \(S_n = a_1 + a_2 + \ldots + a_n\) minimalan.

💡 Nagovještaj

Parcijalni zbir \(S_n\) je minimalan kada poslednji sabrani član \(a_n\) prelazi iz negativne u nenegativnu vrednost. Odredite za koje \(n\) važi \(a_n < 0\), a \(a_{n+1} \geq 0\). Koristite \(a_n = -15 + 2(n-1) \geq 0\), odakle \(n \geq 8{,}5\), pa je \(a_8 = -1 < 0\) i \(a_9 = 1 > 0\). Minimum se postiže za \(n = 8\).
