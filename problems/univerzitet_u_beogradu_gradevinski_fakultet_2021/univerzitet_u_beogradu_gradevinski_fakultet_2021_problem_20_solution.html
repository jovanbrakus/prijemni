Trigonometrija — Jednačine sa sin⁴x i cos⁴x, dvostruki uglovi

Postavka zadatka

\[\sin^4 x + \cos^4 x = \frac{3}{4}\]

koja pripadaju intervalu \((0,\, 2\pi)\) jednak je:

\(2\pi\)

Plan rešavanja

Izraz \(\sin^4 x + \cos^4 x\) transformisaćemo koristeći algebarski identitet za zbir kvadrata i formulu za dvostruki ugao. Time ćemo svesti jednačinu na oblik \(\sin^2(2x) = \frac{1}{2}\), odrediti sva rešenja na intervalu \((0, 2\pi)\) i izračunati njihov zbir.

📐 Ključne formule i teoreme
  • Algebarski identitet za zbir kvadrata: \[a^2 + b^2 = (a + b)^2 - 2ab\] Primenjeno na \(\sin^2 x\) i \(\cos^2 x\): \[\sin^4 x + \cos^4 x = (\sin^2 x + \cos^2 x)^2 - 2\sin^2 x \cos^2 x\]
  • Osnovni trigonometrijski identitet: \[\sin^2 x + \cos^2 x = 1\]
  • Formula za dvostruki ugao (sinus): \[\sin 2x = 2\sin x \cos x\] Odatle sledi: \[\sin^2 x \cos^2 x = \frac{1}{4}\sin^2 2x\]
  • Rešenja jednačine \(\sin t = c\): \[t = \arcsin c + 2k\pi \quad \text{ili} \quad t = \pi - \arcsin c + 2k\pi, \quad k \in \mathbb{Z}\]

Vizuelni prikaz rešenja na jediničnom krugu

8 rešenja jednačine na intervalu (0, 2π) prikazana na jediničnom krugu

Rešenje korak po korak

1

Primenimo algebarski identitet

Izraz \(\sin^4 x + \cos^4 x\) možemo zapisati kao zbir kvadrata \((\sin^2 x)^2 + (\cos^2 x)^2\). Koristimo identitet \(a^2 + b^2 = (a+b)^2 - 2ab\):

\[\sin^4 x + \cos^4 x = (\sin^2 x + \cos^2 x)^2 - 2\sin^2 x \cos^2 x\]
2

Uprostimo koristeći osnovni identitet

Pošto je \(\sin^2 x + \cos^2 x = 1\), dobijamo:

\[\sin^4 x + \cos^4 x = 1 - 2\sin^2 x \cos^2 x\]
3

Uvedemo dvostruki ugao

Iz formule \(\sin 2x = 2\sin x \cos x\) sledi da je \(\sin^2 x \cos^2 x = \frac{1}{4}\sin^2 2x\). Zamenjujemo:

\[1 - 2 \cdot \frac{1}{4}\sin^2 2x = 1 - \frac{1}{2}\sin^2 2x\]
🔍 Proveri sebe: Koliko je \(2\sin^2 x \cos^2 x\) izraženo preko \(\sin 2x\)?
Pošto je \(\sin 2x = 2\sin x\cos x\), sledi \(\sin^2 x\cos^2 x = \frac{\sin^2 2x}{4}\), pa je \(2\sin^2 x \cos^2 x = \frac{\sin^2 2x}{2}\).
4

Postavimo jednačinu

Sada naša jednačina glasi:

\[1 - \frac{1}{2}\sin^2 2x = \frac{3}{4}\]

Prebacimo sve na jednu stranu:

\[\frac{1}{2}\sin^2 2x = 1 - \frac{3}{4} = \frac{1}{4}\]

Pomnožimo obe strane sa 2:

\[\sin^2 2x = \frac{1}{2}\]
5

Rešimo po \(\sin 2x\)

Korenujemo obe strane:

\[\sin 2x = \pm \frac{\sqrt{2}}{2}\]

Obe vrednosti (+) i (−) su moguće jer je kvadriran sinus.

🔍 Proveri sebe: Za koje uglove \(\alpha\) važi \(\sin\alpha = \frac{\sqrt{2}}{2}\)?
\(\alpha = \frac{\pi}{4} + 2k\pi\) ili \(\alpha = \frac{3\pi}{4} + 2k\pi\), za \(k \in \mathbb{Z}\).
6

Rešimo \(\sin 2x = \frac{\sqrt{2}}{2}\)

Opšta rešenja su:

\[2x = \frac{\pi}{4} + 2k\pi \quad \Rightarrow \quad x = \frac{\pi}{8} + k\pi\]
\[2x = \frac{3\pi}{4} + 2k\pi \quad \Rightarrow \quad x = \frac{3\pi}{8} + k\pi\]
7

Rešimo \(\sin 2x = -\frac{\sqrt{2}}{2}\)

Opšta rešenja su:

\[2x = \frac{5\pi}{4} + 2k\pi \quad \Rightarrow \quad x = \frac{5\pi}{8} + k\pi\]
\[2x = \frac{7\pi}{4} + 2k\pi \quad \Rightarrow \quad x = \frac{7\pi}{8} + k\pi\]
8

Odredimo rešenja na intervalu \((0, 2\pi)\)

Za svaku familiju rešenja, uzimamo \(k = 0\) i \(k = 1\):

\[\begin{align} x_1 &= \frac{\pi}{8}, & x_2 &= \frac{\pi}{8} + \pi = \frac{9\pi}{8} \\[6pt] x_3 &= \frac{3\pi}{8}, & x_4 &= \frac{3\pi}{8} + \pi = \frac{11\pi}{8} \\[6pt] x_5 &= \frac{5\pi}{8}, & x_6 &= \frac{5\pi}{8} + \pi = \frac{13\pi}{8} \\[6pt] x_7 &= \frac{7\pi}{8}, & x_8 &= \frac{7\pi}{8} + \pi = \frac{15\pi}{8} \end{align}\]

Za \(k = 2\) dobijamo vrednosti veće od \(2\pi\), a za \(k = -1\) dobijamo negativne vrednosti, pa je \(k \in \{0, 1\}\) jedini izbor.

9

Izračunajmo zbir svih rešenja

Sabiramo svih 8 rešenja. Grupisaćemo ih u parove \((x_i,\, x_i + \pi)\):

\[S = \left(\frac{\pi}{8} + \frac{9\pi}{8}\right) + \left(\frac{3\pi}{8} + \frac{11\pi}{8}\right) + \left(\frac{5\pi}{8} + \frac{13\pi}{8}\right) + \left(\frac{7\pi}{8} + \frac{15\pi}{8}\right)\]

Svaki par daje:

\[S = \frac{10\pi}{8} + \frac{14\pi}{8} + \frac{18\pi}{8} + \frac{22\pi}{8}\]

Saberemo brojioce:

\[S = \frac{(10 + 14 + 18 + 22)\pi}{8} = \frac{64\pi}{8} = 8\pi\]
🔍 Proveri sebe: Koliko ukupno rešenja ima jednačina na intervalu \((0, 2\pi)\)?
Jednačina ima tačno 8 rešenja: \(\frac{\pi}{8}, \frac{3\pi}{8}, \frac{5\pi}{8}, \frac{7\pi}{8}, \frac{9\pi}{8}, \frac{11\pi}{8}, \frac{13\pi}{8}, \frac{15\pi}{8}\).

Ključni uvid

Svih 8 rešenja su ravnomerno raspoređena po krugu, na razmaku od \(\frac{\pi}{4}\). Svako rešenje \(x_i\) ima svoj "par" \(x_i + \pi\) dijametralno suprotan na krugu. Četiri takva para daju zbir:

\[\frac{\pi}{8} + \frac{3\pi}{8} + \frac{5\pi}{8} + \frac{7\pi}{8} + \frac{9\pi}{8} + \frac{11\pi}{8} + \frac{13\pi}{8} + \frac{15\pi}{8} = 8\pi\]

Konačan odgovor

\(8\pi\)
(A) \(2\pi\)

Provera: Rešenja su simetrična — prosečna vrednost 8 jednako raspoređenih rešenja je \(\pi\) (sredina intervala \((0, 2\pi)\)), pa je zbir \(8 \times \pi = 8\pi\). Takođe, numerički: \(\frac{64\pi}{8} = 8\pi \approx 25{,}13\). ✓

Česte greške

  • Zaboravljanje negativnog rešenja: iz \(\sin^2 2x = \frac{1}{2}\) sledi \(\sin 2x = \pm\frac{\sqrt{2}}{2}\), ne samo pozitivna vrednost. Bez negativnog rešenja, nalazimo samo 4 od 8 rešenja i dobijamo pogrešan zbir \(4\pi\).
  • Propuštanje rešenja za \(k = 1\): svaka familija daje po dva rešenja na \((0, 2\pi)\) — jedno za \(k = 0\) i jedno za \(k = 1\). Ako uzmemo samo \(k = 0\), nalazimo 4 rešenja umesto 8.

Dodatni izazov

Odredi zbir rešenja jednačine na intervalu \((0, 2\pi)\):

\[\sin^6 x + \cos^6 x = \frac{5}{8}\]
💡 Nagovještaj

Koristi formulu za zbir kubova: \(a^3 + b^3 = (a + b)(a^2 - ab + b^2)\) sa \(a = \sin^2 x\), \(b = \cos^2 x\). Dobija se \(\sin^6 x + \cos^6 x = 1 - \frac{3}{4}\sin^2 2x\). Jednačina \(\sin^2 2x = \frac{1}{2}\) je ista kao u ovom zadatku!
