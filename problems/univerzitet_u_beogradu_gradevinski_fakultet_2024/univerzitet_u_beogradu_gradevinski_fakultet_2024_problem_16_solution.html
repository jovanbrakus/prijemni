Trigonometrijski izraz sa poluugaonom tangentom

Trigonometrija -- izracunavanje vrednosti izraza pomocu formule za tangent poluugla

Zadatak

Ako je \(\operatorname{tg}\dfrac{x}{2} = \sqrt{3}\), onda je vrednost izraza \(\sqrt{3}\cdot\sin x + \cos x\) jednaka:

(A)
\(1\)
(B)
\(\dfrac{4}{5}\)
(V)
\(\dfrac{3}{5}\)
(G)
\(0\)
(D)
\(-1\)

Plan resavanja

Iz uslova \(\operatorname{tg}\frac{x}{2} = \sqrt{3}\) odredimo ugao \(\frac{x}{2}\), pa zatim izracunamo \(\sin x\) i \(\cos x\). Alternativno, koristicemo univerzalnu trigonometrijsku smenu \(t = \operatorname{tg}\frac{x}{2}\) koja direktno daje \(\sin x\) i \(\cos x\) preko \(t\).

Podsetnik iz teorije

Kljucne formule -- tangent poluugla

Tangent poluugla i poznati uglovi:

\[\operatorname{tg}\frac{\pi}{3} = \sqrt{3}\]

Univerzalna trigonometrijska smena: Ako je \(t = \operatorname{tg}\frac{x}{2}\), onda:

\[\sin x = \frac{2t}{1 + t^2}, \qquad \cos x = \frac{1 - t^2}{1 + t^2}\]

Obrazlozenje: Ove formule proizilaze iz identiteta \(\sin x = 2\sin\frac{x}{2}\cos\frac{x}{2}\) i \(\cos x = \cos^2\frac{x}{2} - \sin^2\frac{x}{2}\), deljenjem sa \(\cos^2\frac{x}{2}\) i koriscenjem \(1 + \operatorname{tg}^2\frac{x}{2} = \frac{1}{\cos^2\frac{x}{2}}\).

Vizuelni prikaz -- jedinicna kruznica

Dijagram prikazuje ugao \(x = \frac{2\pi}{3}\) na jedinicnoj kruznici sa vrednostima sinusa i kosinusa.

Resenje korak po korak

1

Odredjivanje ugla \(\frac{x}{2}\)

Iz uslova \(\operatorname{tg}\frac{x}{2} = \sqrt{3}\), znamo da je:

\[\frac{x}{2} = \frac{\pi}{3} + n\pi, \quad n \in \mathbb{Z}\]

jer je \(\operatorname{tg}\frac{\pi}{3} = \sqrt{3}\), a tangent ima period \(\pi\).

2

Odredjivanje ugla \(x\)

Mnozimo obe strane sa 2:

\[x = \frac{2\pi}{3} + 2n\pi, \quad n \in \mathbb{Z}\]

Posto su \(\sin\) i \(\cos\) periodicne funkcije sa periodom \(2\pi\), dovoljno je uzeti \(x = \frac{2\pi}{3}\).

Proveri sebe: Koliko je \(\operatorname{tg}\frac{\pi}{3}\)?

\(\operatorname{tg}\frac{\pi}{3} = \frac{\sin\frac{\pi}{3}}{\cos\frac{\pi}{3}} = \frac{\frac{\sqrt{3}}{2}}{\frac{1}{2}} = \sqrt{3}\). Tacno!

3

Izracunavanje \(\sin x\) i \(\cos x\)

Za \(x = \frac{2\pi}{3}\) koristimo cinjenicu da je ovaj ugao u drugom kvadrantu (\(\frac{2\pi}{3} = \pi - \frac{\pi}{3}\)):

\[\sin\frac{2\pi}{3} = \sin\!\left(\pi - \frac{\pi}{3}\right) = \sin\frac{\pi}{3} = \frac{\sqrt{3}}{2}\]
\[\cos\frac{2\pi}{3} = \cos\!\left(\pi - \frac{\pi}{3}\right) = -\cos\frac{\pi}{3} = -\frac{1}{2}\]

U drugom kvadrantu sinus je pozitivan, a kosinus negativan.

4

Zamena u izraz

Uvrstimo dobijene vrednosti u trazeni izraz:

\[\sqrt{3}\cdot\sin x + \cos x = \sqrt{3}\cdot\frac{\sqrt{3}}{2} + \left(-\frac{1}{2}\right)\]

Izracunamo svaki sabirak:

\[\sqrt{3}\cdot\frac{\sqrt{3}}{2} = \frac{\sqrt{3}\cdot\sqrt{3}}{2} = \frac{3}{2}\]
Proveri sebe: Koliko je \(\sqrt{3}\cdot\sqrt{3}\)?

\(\sqrt{3}\cdot\sqrt{3} = 3\), po definiciji kvadratnog korena. Dakle \(\frac{\sqrt{3}\cdot\sqrt{3}}{2} = \frac{3}{2}\). Tacno!

5

Konacno izracunavanje

Saberemo dva sabirka:

\[\frac{3}{2} + \left(-\frac{1}{2}\right) = \frac{3}{2} - \frac{1}{2} = \frac{2}{2} = 1\]
6

Alternativni pristup -- univerzalna smena

Mogli smo i bez odredjivanja ugla, koristeci formule sa \(t = \operatorname{tg}\frac{x}{2} = \sqrt{3}\):

\[\sin x = \frac{2t}{1 + t^2} = \frac{2\sqrt{3}}{1 + 3} = \frac{2\sqrt{3}}{4} = \frac{\sqrt{3}}{2}\]
\[\cos x = \frac{1 - t^2}{1 + t^2} = \frac{1 - 3}{1 + 3} = \frac{-2}{4} = -\frac{1}{2}\]

Rezultat je isti: \(\sqrt{3}\cdot\frac{\sqrt{3}}{2} + \left(-\frac{1}{2}\right) = \frac{3}{2} - \frac{1}{2} = 1\).

Proveri sebe: Primeni formulu za \(\cos x\) sa \(t = \sqrt{3}\).

\(t^2 = (\sqrt{3})^2 = 3\). Dakle: \(\cos x = \frac{1 - 3}{1 + 3} = \frac{-2}{4} = -\frac{1}{2}\). Tacno!

Kljucni uvid

Kada je dat \(\operatorname{tg}\frac{x}{2}\), imamo dva mocna puta: ili odredimo sam ugao \(x\) (ako je tangent tablicna vrednost), ili koristimo univerzalnu trigonometrijsku smenu koja direktno izrazava \(\sin x\) i \(\cos x\) preko \(t = \operatorname{tg}\frac{x}{2}\). Oba pristupa ovde vode ka istom rezultatu, ali univerzalna smena je korisna i kada ugao nije tablicna vrednost.

Konacan odgovor

\(\displaystyle \sqrt{3}\cdot\sin x + \cos x = 1\)
(A) \(1\)
(B) \(\frac{4}{5}\)
(V) \(\frac{3}{5}\)
(G) \(0\)
(D) \(-1\)
Verifikacija: Proverimo obrnuto -- ako je izraz jednak 1, onda \(\sqrt{3}\sin x + \cos x = 1\). Za \(\sin x = \frac{\sqrt{3}}{2}\) i \(\cos x = -\frac{1}{2}\): izraz daje \(\frac{3}{2} - \frac{1}{2} = 1\). Takodje, \(\sin^2 x + \cos^2 x = \frac{3}{4} + \frac{1}{4} = 1\), sto potvrdjuje korektnost vrednosti sinusa i kosinusa.

Ceste greske

Zaboravljanje znaka kosinusa: Ugao \(\frac{2\pi}{3}\) je u drugom kvadrantu, gde je kosinus negativan. Cesta greska je uzeti \(\cos\frac{2\pi}{3} = \frac{1}{2}\) umesto \(-\frac{1}{2}\), sto bi dalo pogresan rezultat \(\frac{3}{2} + \frac{1}{2} = 2\).
Greska u univerzalnoj smeni: U formuli \(\cos x = \frac{1-t^2}{1+t^2}\) treba paziti na redosled: u brojiocu je \(1-t^2\), a ne \(t^2-1\). Zamena znaka dovodi do \(\cos x = \frac{1}{2}\) umesto \(-\frac{1}{2}\).

Dodatni izazov

Ako je \(\operatorname{tg}\frac{x}{2} = 2\), kolika je vrednost izraza \(\frac{\sin x}{1 + \cos x}\)?

Hint

Koristite identitet: \(\frac{\sin x}{1 + \cos x} = \operatorname{tg}\frac{x}{2}\). Dakle, odgovor je prosto... 2! Ovaj identitet mozete dokazati koriscenjem formula za dvostruki ugao.
