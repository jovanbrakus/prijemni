Kompozicija funkcija - sabiranje vrednosti

Kompozicija racionalne i linearne funkcije, izračunavanje konkretnih vrednosti

Zadatak

Ako je \(\displaystyle f(x) = \frac{3-x}{3+x}\) i \(g(x) = 3 + x\), onda je \(g(f(3)) + f(g(-3))\) jednako:

Plan rešavanja

Izraz \(g(f(3)) + f(g(-3))\) sadrži dva člana. Svaki ćemo izračunati zasebno, radeći "iznutra ka spolja": prvo izračunamo unutrašnju funkciju, pa njen rezultat uvrstimo u spoljašnju. Na kraju saberemo oba rezultata.

Podsetnik iz teorije

Kompozicija funkcija i pravila izračunavanja

Kompozicija funkcija: Kompozicija \((g \circ f)(x) = g(f(x))\) znači da se prvo primeni funkcija \(f\), a zatim se njen rezultat prosledi funkciji \(g\).

Redosled izračunavanja: Uvek računamo "iznutra ka spolja" - najpre vrednost unutrašnje funkcije, pa tu vrednost uvrstimo u spoljašnju:

\[g(f(a)) \;:\quad \text{1. izračunaj } f(a) = b, \quad \text{2. izračunaj } g(b)\]

Racionalna funkcija: Za \(f(x) = \dfrac{3-x}{3+x}\), domen je \(x \neq -3\) (imenilac ne sme biti nula).

Vizuelni prikaz

Kliknite na dugmad ispod dijagrama da vidite svaki korak izračunavanja

Rešenje korak po korak

1

Računamo \(f(3)\)

Uvrstimo \(x = 3\) u funkciju \(f\):

\[f(3) = \frac{3 - 3}{3 + 3} = \frac{0}{6} = 0\]

Brojilac je nula, pa je ceo razlomak jednak nuli.

2

Računamo \(g(f(3)) = g(0)\)

Rezultat iz prethodnog koraka, \(f(3) = 0\), uvrstimo u funkciju \(g\):

\[g(0) = 3 + 0 = 3\]

Dakle, prvi član našeg zbira je:

\[g(f(3)) = 3\]
Proveri sebe: šta je \(f(3)\)?

Brojilac \(3 - 3 = 0\), imenilac \(3 + 3 = 6\). Razlomak \(\frac{0}{6} = 0\). Kad god je brojilac nula (a imenilac nije), razlomak je nula.

3

Računamo \(g(-3)\)

Sada prelazimo na drugi član. Uvrstimo \(x = -3\) u funkciju \(g\):

\[g(-3) = 3 + (-3) = 0\]
4

Računamo \(f(g(-3)) = f(0)\)

Rezultat iz prethodnog koraka, \(g(-3) = 0\), uvrstimo u funkciju \(f\):

\[f(0) = \frac{3 - 0}{3 + 0} = \frac{3}{3} = 1\]

Dakle, drugi član našeg zbira je:

\[f(g(-3)) = 1\]
Proveri sebe: šta je \(f(0)\)?

Brojilac: \(3 - 0 = 3\). Imenilac: \(3 + 0 = 3\). Razlomak: \(\frac{3}{3} = 1\). Kada su brojilac i imenilac jednaki (i različiti od nule), razlomak je \(1\).

5

Sabiramo rezultate

Konačno, sabiramo oba člana:

\[g(f(3)) + f(g(-3)) = 3 + 1 = 4\]

Ključni uvid

Oba člana koriste isti "trik": ulazna vrednost (\(3\) ili \(-3\)) pri uvrštavanju daje međurezultat \(0\), koji se zatim prosleđuje drugoj funkciji. Kada primetimo da su međurezultati nule, računica postaje veoma jednostavna.

\[f(3) = 0 \;\xrightarrow{g}\; 3 \qquad\qquad g(-3) = 0 \;\xrightarrow{f}\; 1 \qquad\qquad 3 + 1 = 4\]

Konačan odgovor

\(\displaystyle g(f(3)) + f(g(-3)) = 4\)
(A) \(1\)
(B) \(0\)
(V) \(3\)
(G) \(4\)
(D) \(2\)
Verifikacija: Proverimo ponovo oba člana nezavisno. Za prvi: \(f(3) = \frac{0}{6} = 0\), pa \(g(0) = 3\). Za drugi: \(g(-3) = 0\), pa \(f(0) = \frac{3}{3} = 1\). Zbir: \(3 + 1 = 4\). Rezultat se poklapa sa odgovorom (G).

Česte greške

Pogrešan redosled kompozicije: Česta greška je zameniti redosled - npr. računati \(f(g(3))\) umesto \(g(f(3))\). Kod kompozicije, uvek se kreće iznutra ka spolja: u izrazu \(g(f(3))\), prvo se računa \(f(3)\), pa se rezultat uvrsti u \(g\).
Greška sa predznacima: Kod \(g(-3) = 3 + (-3)\), studenti ponekad zaborave da je argument negativan i zapišu \(g(-3) = 3 + 3 = 6\), što daje pogrešan rezultat. Obratite pažnju na znak argumenta pri uvrštavanju.

Dodatni izazov

Pokušajte da izračunate \(f(f(x))\) i pojednostavite izraz. Šta primećujete?

Nagoveštaj

Uvrstite \(f(x) = \frac{3-x}{3+x}\) umesto \(x\) u sam izraz za \(f\):

\(\displaystyle f(f(x)) = \frac{3 - \frac{3-x}{3+x}}{3 + \frac{3-x}{3+x}}\)

Svedite na zajednički imenilac i pojednostavite. Trebalo bi da dobijete \(f(f(x)) = x\) - funkcija \(f\) je sama sebi inverzna (involutivna funkcija)!
