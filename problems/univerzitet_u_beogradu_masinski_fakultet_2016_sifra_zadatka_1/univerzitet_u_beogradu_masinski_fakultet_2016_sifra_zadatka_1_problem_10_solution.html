Zbir trigonometrijskih razlomaka

Sabiranje razlomaka sa trigonometrijskim funkcijama i primena osnovnog trigonometrijskog identiteta

Tekst zadatka

Za \(0 < x < \dfrac{\pi}{2}\) izraz \(\dfrac{\sin x}{1 - \cos x} + \dfrac{\sin x}{1 + \cos x}\) je jednak:

(A)
\(\sin x\)
(B)
\(\dfrac{1}{\sin x}\)
(C)
\(\dfrac{1}{\cos \frac{x}{2}}\)
(D)
\(\dfrac{2}{\cos x}\)
(E)
\(\dfrac{2}{\sin x}\)

Plan rešavanja

Sabraćemo dva razlomka svođenjem na zajednički imenilac. Imenilac \((1 - \cos x)(1 + \cos x)\) upravo je razlika kvadrata, što daje \(\sin^2 x\). Zatim ćemo pojednostaviti brojilac i skratiti izraz.

Podsetnik iz teorije

Osnovni trigonometrijski identitet

Za svaki ugao \(x\) važi:

\[ \sin^2 x + \cos^2 x = 1 \]

Odavde sledi da je \(1 - \cos^2 x = \sin^2 x\).

Razlika kvadrata

Za bilo koje brojeve \(a\) i \(b\):

\[ (a - b)(a + b) = a^2 - b^2 \]

U našem slučaju: \((1 - \cos x)(1 + \cos x) = 1 - \cos^2 x\).

Sabiranje razlomaka

Za razlomke sa različitim imenilacima:

\[ \frac{a}{b} + \frac{a}{c} = \frac{a \cdot c + a \cdot b}{b \cdot c} = \frac{a(b + c)}{b \cdot c} \]

Vizuelni prikaz

Pomerite tačku duž kruga da vidite kako se vrednost izraza menja za različite uglove \(x\)

Rešenje korak po korak

1 Zapišemo izraz

Polazni izraz glasi:

\[ \frac{\sin x}{1 - \cos x} + \frac{\sin x}{1 + \cos x} \]

Ovo su dva razlomka sa različitim imenilacima. Treba ih sabrati.

2 Nađemo zajednički imenilac

Zajednički imenilac je proizvod dva imenioca:

\[ (1 - \cos x)(1 + \cos x) \]

Prepoznajemo razliku kvadrata:

\[ (1 - \cos x)(1 + \cos x) = 1 - \cos^2 x \]
Proveri sebe: Čemu je jednako \(1 - \cos^2 x\)?

Iz osnovnog trigonometrijskog identiteta \(\sin^2 x + \cos^2 x = 1\) sledi da je \(1 - \cos^2 x = \sin^2 x\).

3 Primenimo trigonometrijski identitet

Prema osnovnom trigonometrijskom identitetu:

\[ 1 - \cos^2 x = \sin^2 x \]

Dakle, zajednički imenilac je \(\sin^2 x\).

4 Saberemo razlomke

Sada sabiramo razlomke tako što svaki pomnožimo odgovarajućim faktorom:

\[ \frac{\sin x \cdot (1 + \cos x) + \sin x \cdot (1 - \cos x)}{\sin^2 x} \]
5 Razvijemo brojilac

Razvijemo svaki sabirak u brojiocu:

\[ \sin x \cdot (1 + \cos x) = \sin x + \sin x \cos x \]
\[ \sin x \cdot (1 - \cos x) = \sin x - \sin x \cos x \]

Kada ih saberemo, članovi sa \(\sin x \cos x\) se poništavaju:

\[ (\sin x + \sin x \cos x) + (\sin x - \sin x \cos x) = 2\sin x \]
Proveri sebe: Zašto se članovi sa \(\cos x\) poništavaju?

Zato što imamo \(+\sin x \cos x\) iz prvog razlomka i \(-\sin x \cos x\) iz drugog. Njihov zbir je nula: \(\sin x \cos x - \sin x \cos x = 0\).

6 Skratimo razlomak

Sada imamo:

\[ \frac{2\sin x}{\sin^2 x} \]

Skraćujemo sa \(\sin x\) (što možemo jer za \(0 < x < \frac{\pi}{2}\) važi \(\sin x \neq 0\)):

\[ \frac{2\sin x}{\sin^2 x} = \frac{2}{\sin x} \]

Ključni uvid

Celokupno pojednostavljenje se svodi na dva ključna koraka: prepoznavanje razlike kvadrata u imeniocu \((1 - \cos x)(1 + \cos x) = 1 - \cos^2 x = \sin^2 x\) i poništavanje mešovitih članova \(\pm \sin x \cos x\) u brojiocu. Oba koraka su veoma česta na prijemnim ispitima.

Konačan odgovor

\(\dfrac{\sin x}{1 - \cos x} + \dfrac{\sin x}{1 + \cos x} = \dfrac{2}{\sin x}\)
(A) \(\sin x\)
(B) \(\frac{1}{\sin x}\)
(C) \(\frac{1}{\cos \frac{x}{2}}\)
(D) \(\frac{2}{\cos x}\)
(E) \(\frac{2}{\sin x}\)

Verifikacija: za \(x = \frac{\pi}{6}\) imamo \(\sin \frac{\pi}{6} = \frac{1}{2}\), \(\cos \frac{\pi}{6} = \frac{\sqrt{3}}{2}\). Uvrštavanjem u polazni izraz dobijamo \(\frac{0{,}5}{1 - 0{,}866} + \frac{0{,}5}{1 + 0{,}866} = \frac{0{,}5}{0{,}134} + \frac{0{,}5}{1{,}866} \approx 3{,}732 + 0{,}268 = 4\). Proverimo: \(\frac{2}{\sin \frac{\pi}{6}} = \frac{2}{0{,}5} = 4\) ✓

Česte greške

Neprepoznavanje razlike kvadrata: Mnogi učenici ne vide da je \((1 - \cos x)(1 + \cos x) = 1 - \cos^2 x\). Umesto toga, množu „ukršteno" i prave računske greške. Kad god vidite oblik \((a - b)(a + b)\), odmah ga prepoznajte kao \(a^2 - b^2\).

Skraćivanje kad je \(\sin x = 0\): Skraćivanje sa \(\sin x\) je dozvoljeno samo ako je \(\sin x \neq 0\). U ovom zadatku je dato \(0 < x < \frac{\pi}{2}\), pa važi \(\sin x > 0\), ali na to treba obratiti pažnju u opštem slučaju.

Izazov za vežbu

Pojednostavite izraz \(\dfrac{\cos x}{1 - \sin x} + \dfrac{\cos x}{1 + \sin x}\) za \(0 < x < \dfrac{\pi}{2}\).

Uputstvo

Primenite potpuno isti postupak: zajednički imenilac je \((1 - \sin x)(1 + \sin x) = 1 - \sin^2 x = \cos^2 x\). U brojiocu se ponovo poništavaju mešoviti članovi, pa ostaje \(\frac{2\cos x}{\cos^2 x} = \frac{2}{\cos x}\).
