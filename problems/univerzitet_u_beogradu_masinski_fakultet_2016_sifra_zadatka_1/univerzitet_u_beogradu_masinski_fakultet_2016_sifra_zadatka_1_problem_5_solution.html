Kotangens iz datog sinusa u trećem krugu

Trigonometrija — osnovni trigonometrijski identitet, određivanje znaka funkcije prema kvadrantu, kotangens

Zadatak

Ako je \(\sin x = \dfrac{1}{3}\) i \(\dfrac{5\pi}{2} < x < 3\pi\), onda je \(\operatorname{ctg} x\) jednako:

(A)
\(-2\sqrt{2}\)
(B)
\(-3\)
(C)
\(-\dfrac{1}{4}\sqrt{2}\)
(D)
\(\dfrac{1}{4}\sqrt{2}\)
(E)
\(2\sqrt{2}\)

Plan rešavanja

Najpre ćemo odrediti u kom se kvadrantu nalazi ugao \(x\), zatim pomoću osnovnog trigonometrijskog identiteta \(\sin^2 x + \cos^2 x = 1\) naći \(\cos x\) sa ispravnim predznakom, i na kraju izračunati \(\operatorname{ctg} x = \dfrac{\cos x}{\sin x}\).

Podsetnik iz teorije

Ključne formule i teoreme

Osnovni trigonometrijski identitet:

\[\sin^2 x + \cos^2 x = 1\]

Kotangens:

\[\operatorname{ctg} x = \frac{\cos x}{\sin x}\]

Znakovi trigonometrijskih funkcija po kvadrantima:

  • I kvadrant \((0, \frac{\pi}{2})\): svi pozitivni
  • II kvadrant \((\frac{\pi}{2}, \pi)\): \(\sin > 0\), \(\cos < 0\), \(\operatorname{ctg} < 0\)
  • III kvadrant \((\pi, \frac{3\pi}{2})\): \(\sin < 0\), \(\cos < 0\), \(\operatorname{ctg} > 0\)
  • IV kvadrant \((\frac{3\pi}{2}, 2\pi)\): \(\sin < 0\), \(\cos > 0\), \(\operatorname{ctg} < 0\)

Periodičnost: \(\sin(x + 2\pi) = \sin x\), \(\cos(x + 2\pi) = \cos x\). Ugao se uvek može svesti na interval \([0, 2\pi)\).

Vizuelni prikaz — Jedinični krug

Dijagram prikazuje položaj ugla \(x\) na jediničnom krugu i odgovarajuće vrednosti \(\sin x\) i \(\cos x\).

Ugao u drugom kvadrantu: \(\sin x > 0\), \(\cos x < 0\)

Rešenje korak po korak

1

Određujemo kvadrant ugla \(x\)

Dato je da je \(\dfrac{5\pi}{2} < x < 3\pi\). Oduzimamo \(2\pi\) da svedemo ugao u osnovni interval:

\[\frac{5\pi}{2} - 2\pi = \frac{5\pi}{2} - \frac{4\pi}{2} = \frac{\pi}{2}\]
\[3\pi - 2\pi = \pi\]

Dakle, ugao \(x\) je ekvivalentan uglu u intervalu \(\left(\dfrac{\pi}{2},\, \pi\right)\), što je drugi kvadrant.

U drugom kvadrantu: \(\sin x > 0\) i \(\cos x < 0\).

2

Nalazimo \(\cos^2 x\) pomoću osnovnog identiteta

Iz identiteta \(\sin^2 x + \cos^2 x = 1\) izražavamo \(\cos^2 x\):

\[\cos^2 x = 1 - \sin^2 x\]

Zamenjujemo \(\sin x = \dfrac{1}{3}\):

\[\cos^2 x = 1 - \left(\frac{1}{3}\right)^2 = 1 - \frac{1}{9} = \frac{9}{9} - \frac{1}{9} = \frac{8}{9}\]
Proveri sebe: Koliko je \(\cos^2 x\)?

\(\cos^2 x = \dfrac{8}{9}\). Proveravamo: \(\dfrac{1}{9} + \dfrac{8}{9} = \dfrac{9}{9} = 1\) ✓

3

Određujemo \(\cos x\) sa ispravnim predznakom

Korenujemo obe strane:

\[\cos x = \pm\sqrt{\frac{8}{9}} = \pm\frac{\sqrt{8}}{3} = \pm\frac{2\sqrt{2}}{3}\]

Pošto je ugao u drugom kvadrantu, kosinus je negativan:

\[\cos x = -\frac{2\sqrt{2}}{3}\]
4

Računamo \(\operatorname{ctg} x\)

Kotangens je količnik kosinusa i sinusa:

\[\operatorname{ctg} x = \frac{\cos x}{\sin x} = \frac{-\dfrac{2\sqrt{2}}{3}}{\dfrac{1}{3}}\]

Delimo razlomke (množimo prvim i recipročnom vrednošću drugog):

\[\operatorname{ctg} x = -\frac{2\sqrt{2}}{3} \cdot \frac{3}{1} = -\frac{2\sqrt{2} \cdot 3}{3} = -2\sqrt{2}\]

Trojke u brojiocu i imeniocu se skrate.

Proveri sebe: Zašto je rezultat negativan?

U drugom kvadrantu, \(\cos x < 0\) a \(\sin x > 0\), pa je njihov količnik \(\operatorname{ctg} x = \dfrac{\cos x}{\sin x} < 0\). To se poklapa sa odgovorom \(-2\sqrt{2}\).

5

Identifikujemo tačan odgovor

Dobili smo:

\[\operatorname{ctg} x = -2\sqrt{2}\]

Ovo odgovara opciji (A).

Ključni uvid

Zadatak testira dve stvari: (1) sposobnost prepoznavanja kvadranta kada je ugao van osnovnog intervala \([0, 2\pi)\), i (2) pravilno određivanje predznaka kosinusa pre izračunavanja kotangensa. Čim odredimo da je ugao u drugom kvadrantu, znamo da je \(\cos x < 0\), i ostatak je rutinski račun:

\[\operatorname{ctg} x = \frac{\cos x}{\sin x} = \frac{-\frac{2\sqrt{2}}{3}}{\frac{1}{3}} = \color{#4ade80}{-2\sqrt{2}}\]

Konačan odgovor

\(\displaystyle \text{(A)}\ -2\sqrt{2}\)
(A) \(-2\sqrt{2}\)
(B) \(-3\)
(C) \(-\dfrac{1}{4}\sqrt{2}\)
(D) \(\dfrac{1}{4}\sqrt{2}\)
(E) \(2\sqrt{2}\)
Verifikacija: Proveravamo numerički. \(\sin x = \frac{1}{3} \approx 0{,}333\). Tada \(\cos x = -\frac{2\sqrt{2}}{3} \approx -\frac{2 \cdot 1{,}414}{3} \approx -0{,}943\). Proverimo: \(\sin^2 x + \cos^2 x = 0{,}111 + 0{,}889 = 1{,}000\) ✓. Kotangens: \(\frac{-0{,}943}{0{,}333} \approx -2{,}828 = -2\sqrt{2}\) ✓.

Česte greške

Pogrešan predznak kosinusa: Ako se ne odredi kvadrant ili se pogrešno zaključi da je ugao u prvom/četvrtom kvadrantu, dobija se \(\cos x = +\frac{2\sqrt{2}}{3}\) i odgovor \(+2\sqrt{2}\) — opcija (E), koja je netačna.
Pogrešno svođenje ugla: Interval \(\left(\frac{5\pi}{2},\, 3\pi\right)\) izgleda neobično. Ključ je oduzeti \(2\pi\) da se ugao svede na poznat interval. Neki učenici greše i oduzimaju \(\pi\) ili \(\frac{\pi}{2}\), što vodi pogrešnom kvadrantu.

Dodatni izazov

Ako je \(\cos x = -\dfrac{1}{4}\) i \(\pi < x < \dfrac{3\pi}{2}\), koliko je \(\operatorname{tg} x\)?

Nagoveštaj

Ugao je u trećem kvadrantu, gde su i sinus i kosinus negativni. Iz \(\sin^2 x = 1 - \frac{1}{16} = \frac{15}{16}\) sledi \(\sin x = -\frac{\sqrt{15}}{4}\). Dakle \(\operatorname{tg} x = \frac{\sin x}{\cos x} = \frac{-\sqrt{15}/4}{-1/4} = \sqrt{15}\).
