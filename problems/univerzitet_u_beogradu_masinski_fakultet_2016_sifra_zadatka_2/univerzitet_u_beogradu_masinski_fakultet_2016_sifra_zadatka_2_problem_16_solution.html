Deljivost polinoma četvrtog stepena

Algebra polinoma — faktorizacija i izjednačavanje koeficijenata

📝 Postavka zadatka

Ako je polinom \(x^4 + ax^2 + b\) (gde su \(a, b \in \mathbb{R}\)) deljiv polinomom \(x^2 + x + 1\), onda je \(a + b\) jednako:

(A)
\(2\)
(B)
\(-1\)
(C)
\(0\)
(D)
\(1\)
(E)
\(-2\)

🗺️ Plan rešavanja

Pošto je polinom četvrtog stepena deljiv polinomom drugog stepena, možemo ga zapisati kao proizvod \((x^2 + x + 1)(x^2 + cx + d)\). Razvićemo taj proizvod i izjednačiti koeficijente uz odgovarajuće stepene od \(x\) da nađemo \(a\) i \(b\).

📐 Ključne formule i teoreme
  • Deljivost polinoma: Ako polinom \(P(x)\) stepena \(n\) deli polinom \(Q(x)\) stepena \(m\) (\(m \geq n\)), onda postoji polinom \(R(x)\) stepena \(m - n\) takav da \(Q(x) = P(x) \cdot R(x)\).
  • Izjednačavanje koeficijenata: Ako su \(a_n x^n + \ldots + a_0 = b_n x^n + \ldots + b_0\) za sve vrednosti \(x\), onda \(a_k = b_k\) za svako \(k\).
  • Razlika kvadrata: \(A^2 - B^2 = (A - B)(A + B)\) — korisna identičnost za faktorizaciju izrazâ oblika \(x^4 + x^2 + 1\).
  • Koreni od \(x^2 + x + 1\): Ovaj polinom ima kompleksne korene \(\omega = \frac{-1 + i\sqrt{3}}{2}\) i \(\bar{\omega}\), koji su primitivni šesti koreni jedinice (\(\omega^6 = 1\)).

📊 Vizuelni prikaz

Grafik polinoma \(x^4 + x^2 + 1\) i njegovih faktora

📋 Rešenje korak po korak

1

Zapišemo polinom kao proizvod

Pošto \(x^2 + x + 1\) (stepen 2) deli \(x^4 + ax^2 + b\) (stepen 4), količnik mora biti polinom drugog stepena. Njegov vodeći koeficijent mora biti 1 (jer je vodeći koeficijent oba polinoma jednak 1).

\[x^4 + ax^2 + b = (x^2 + x + 1)(x^2 + cx + d)\]

Nepoznati koeficijenti \(c\) i \(d\) su realni brojevi koje treba odrediti.

2

Razvijemo desnu stranu

Množimo polinom po polinom, sabiramo sabirke uz iste stepene:

\[\begin{align} &(x^2 + x + 1)(x^2 + cx + d) \\ &= x^4 + cx^3 + dx^2 \\ &\quad + x^3 + cx^2 + dx \\ &\quad + x^2 + cx + d \end{align}\]
3

Grupišemo po stepenima

Saberemo sve sabirke koji imaju isti stepen od \(x\):

\[x^4 + (c + 1)x^3 + (d + c + 1)x^2 + (d + c)x + d\]
🧠 Proveri sebe: Koliko jednačina dobijamo izjednačavanjem koeficijenata?
Dobijamo 4 jednačine (za koeficijente uz \(x^3\), \(x^2\), \(x^1\) i \(x^0\)), a imamo 4 nepoznate (\(a\), \(b\), \(c\), \(d\)).
4

Izjednačimo koeficijente

Leva strana je \(x^4 + 0 \cdot x^3 + a \cdot x^2 + 0 \cdot x + b\). Izjednačavamo koeficijente uz svaki stepen:

\[\begin{align} x^3: \quad & c + 1 = 0 \\ x^2: \quad & d + c + 1 = a \\ x^1: \quad & d + c = 0 \\ x^0: \quad & d = b \end{align}\]

Ključna stvar: koeficijenti uz \(x^3\) i \(x\) na levoj strani su nula!

5

Odredimo \(c\) iz prve jednačine

Iz jednačine za \(x^3\):

\[c + 1 = 0 \implies c = -1\]
6

Odredimo \(d\) iz treće jednačine

Iz jednačine za \(x^1\):

\[d + c = 0 \implies d = -c = -(-1) = 1\]
🧠 Proveri sebe: Koliko iznosi \(a\)?
Uvrštavamo \(c = -1\) i \(d = 1\) u jednačinu za \(x^2\): \(a = d + c + 1 = 1 + (-1) + 1 = 1\). Dakle, \(a = 1\).
7

Izračunamo \(a\) i \(b\)

Iz jednačine za \(x^2\):

\[a = d + c + 1 = 1 + (-1) + 1 = 1\]

Iz jednačine za \(x^0\):

\[b = d = 1\]
8

Računamo \(a + b\)

\[a + b = 1 + 1 = 2\]
🧠 Proveri sebe: Kako glasi kompletna faktorizacija?
\(x^4 + x^2 + 1 = (x^2 + x + 1)(x^2 - x + 1)\). Proverite množeći!

💡 Ključni uvid

Izraz \(x^4 + x^2 + 1\) se može elegantno faktorizovati koristeći razliku kvadrata:

\[x^4 + x^2 + 1 = (x^2 + 1)^2 - x^2 = (x^2 + x + 1)(x^2 - x + 1)\]

Dodali smo i oduzeli \(x^2\) da bismo dobili potpuni kvadrat, a zatim primenili formulu \(A^2 - B^2 = (A-B)(A+B)\).

🎯 Konačan odgovor

\(a + b = 2\)
(A) \(2\)
(B) \(-1\)
(C) \(0\)
(D) \(1\)
(E) \(-2\)
Verifikacija: Proveravamo direktnim množenjem: \((x^2 + x + 1)(x^2 - x + 1) = x^4 - x^3 + x^2 + x^3 - x^2 + x + x^2 - x + 1 = x^4 + x^2 + 1\). Zaista, za \(a = 1\) i \(b = 1\) polinom \(x^4 + x^2 + 1\) je deljiv sa \(x^2 + x + 1\), pa je \(a + b = 1 + 1 = 2\).

⚠️ Česte greške

  • Zamena koeficijenata: Lako se pobrka koji koeficijent odgovara kom stepenu prilikom izjednačavanja. Posebno obratite pažnju da koeficijenti uz \(x^3\) i \(x\) moraju biti nula — to je ključ za određivanje \(c\) i \(d\).
  • Preskakanje provere: Uvek verifikujte rešenje direktnim množenjem. Čest propust je da se izračunaju \(c\) i \(d\), ali se ne proveri da li zaista daju odgovarajuće \(a\) i \(b\).

🚀 Dodatni izazov

Za koje vrednosti realnog parametra \(k\) je polinom \(x^4 + kx^2 + 4\) deljiv nekim polinomom oblika \(x^2 + px + 2\) (gde je \(p \in \mathbb{R}\))?

Prikaži nagoveštaj

Zapišite \(x^4 + kx^2 + 4 = (x^2 + px + 2)(x^2 + qx + 2)\) i izjednačite koeficijente. Obratite pažnju na to šta se dešava sa slobodnim članom — zašto je drugi faktor istog oblika?
