Zbir rešenja trigonometrijske jednačine

Trigonometrijske jednačine — zbir kosinusa, faktorizacija

📜 Zadatak

Zbir 4 najmanja pozitivna rešenja jednačine \(\cos x + \cos 2x + \cos 4x = 0\) je:

(A)
\(\dfrac{37}{18}\pi\)
(B)
\(\dfrac{23}{14}\pi\)
(C)
\(\dfrac{8}{3}\pi\)
(D)
\(\dfrac{35}{18}\pi\)
(E)
\(\dfrac{20}{9}\pi\)
📌 Plan rešavanja
  1. Pomnožiti jednačinu sa \(2\sin x\) i primeniti formulu za proizvod sinusa i kosinusa
  2. Teleskopskim sažimanjem svesti jednačinu na proizvod dva faktora
  3. Rešiti svaki faktor posebno i naći sva pozitivna rešenja
  4. Odabrati 4 najmanja i izračunati njihov zbir
Podsetnik iz teorije

Proizvod sinusa i kosinusa:

\[2\sin A \cos B = \sin(A + B) + \sin(A - B)\]

Razlika sinusa (teleskopsko sažimanje):

Kada primenimo gornju formulu na sabirke \(\cos x\), \(\cos 2x\), \(\cos 4x\), dobijamo teleskopski niz u kome se srednji članovi poništavaju.

Opšta rešenja elementarnih jednačina:

\[\sin \alpha = 0 \;\Rightarrow\; \alpha = n\pi, \quad n \in \mathbb{Z}\] \[\cos \alpha = -\tfrac{1}{2} \;\Rightarrow\; \alpha = \pm\tfrac{2\pi}{3} + 2k\pi, \quad k \in \mathbb{Z}\]
📊 Grafički prikaz

Graf funkcije \(f(x) = \cos x + \cos 2x + \cos 4x\) sa označenim nulama

Zelene tačke označavaju 4 najmanja pozitivna rešenja

✏️ Rešenje korak po korak
Korak 1: Množenje sa \(2\sin x\)

Polazimo od jednačine:

\[\cos x + \cos 2x + \cos 4x = 0\]

Pomnožimo obe strane sa \(2\sin x\) (uz uslov \(\sin x \neq 0\)):

\[2\sin x \cos x + 2\sin x \cos 2x + 2\sin x \cos 4x = 0\]
Korak 2: Primena formule na svaki sabirak

Koristimo identitet \(2\sin A \cos B = \sin(A+B) + \sin(A-B)\):

\[2\sin x \cos x = \sin 2x + \sin 0 = \sin 2x\] \[2\sin x \cos 2x = \sin 3x + \sin(-x) = \sin 3x - \sin x\] \[2\sin x \cos 4x = \sin 5x + \sin(-3x) = \sin 5x - \sin 3x\]
Korak 3: Teleskopsko sažimanje

Sabiramo sve tri jednakosti:

\[\sin 2x + (\sin 3x - \sin x) + (\sin 5x - \sin 3x) = 0\]

Članovi \(\sin 3x\) i \(-\sin 3x\) se poništavaju:

\[\sin 5x + \sin 2x - \sin x = 0\]
Korak 4: Faktorizacija razlike sinusa

Primenimo formulu za razliku \(\sin 5x - \sin x = 2\cos 3x \sin 2x\):

\[2\cos 3x \sin 2x + \sin 2x = 0\]

Iznosimo \(\sin 2x\) ispred zagrade:

\[\sin 2x \,(2\cos 3x + 1) = 0\]
Proveri se: Da li si dobio/la faktorizovani oblik \(\sin 2x \cdot (2\cos 3x + 1) = 0\)?
Korak 5: Provera uslova \(\sin x \neq 0\)

Množili smo sa \(2\sin x\), pa moramo proveriti slučaj \(\sin x = 0\), tj. \(x = n\pi\).

Za \(x = n\pi\):

\[\cos(n\pi) + \cos(2n\pi) + \cos(4n\pi) = (-1)^n + 1 + 1 = (-1)^n + 2\]

Za parno \(n\): \(1 + 2 = 3 \neq 0\). Za neparno \(n\): \(-1 + 2 = 1 \neq 0\).

Dakle, \(\sin x = 0\) ne daje rešenja originalne jednačine. Nismo izgubili ništa množeći.

Korak 6: Rešavanje prvog faktora: \(\sin 2x = 0\)

Jednačina \(\sin 2x = 0\) daje:

\[2x = n\pi \quad \Rightarrow \quad x = \frac{n\pi}{2}, \quad n \in \mathbb{Z}\]

Pošto \(\sin x \neq 0\), odbacujemo \(x = n\pi\) (tj. parne vrednosti \(n\)).

Ostaju rešenja sa neparnim \(n\):

\[x = \frac{\pi}{2},\; \frac{3\pi}{2},\; \frac{5\pi}{2},\; \ldots\]
Korak 7: Rešavanje drugog faktora: \(\cos 3x = -\tfrac{1}{2}\)

Jednačina \(\cos 3x = -\frac{1}{2}\) daje:

\[3x = \frac{2\pi}{3} + 2k\pi \quad \text{ili} \quad 3x = \frac{4\pi}{3} + 2k\pi\]

Delimo sa 3:

\[x = \frac{2\pi}{9} + \frac{2k\pi}{3} \quad \text{ili} \quad x = \frac{4\pi}{9} + \frac{2k\pi}{3}\]
Proveri se: Za \(x = \frac{2\pi}{9}\): izračunaj \(\cos\frac{2\pi}{9} + \cos\frac{4\pi}{9} + \cos\frac{8\pi}{9}\). Hint: ovo su kosinusi uglova \(40°, 80°, 160°\) koji se teleskopski sažimaju na 0.
Korak 8: Nabrajanje najmanjih pozitivnih rešenja

Iz \(\cos 3x = -\frac{1}{2}\) za \(k = 0, 1, 2, \ldots\):

\[x = \frac{2\pi}{9} \approx 0{,}698, \quad x = \frac{4\pi}{9} \approx 1{,}396, \quad x = \frac{8\pi}{9} \approx 2{,}793, \quad \ldots\]

Iz \(\sin 2x = 0\) (sa \(\sin x \neq 0\)):

\[x = \frac{\pi}{2} \approx 1{,}571, \quad x = \frac{3\pi}{2} \approx 4{,}712, \quad \ldots\]

Sortiramo po veličini. Četiri najmanja pozitivna rešenja su:

\[x_1 = \frac{2\pi}{9}, \quad x_2 = \frac{4\pi}{9}, \quad x_3 = \frac{\pi}{2}, \quad x_4 = \frac{8\pi}{9}\]
Korak 9: Izračunavanje zbira

Svodimo na zajednički imenilac 18:

\[\frac{2\pi}{9} + \frac{4\pi}{9} + \frac{\pi}{2} + \frac{8\pi}{9} = \frac{4\pi}{18} + \frac{8\pi}{18} + \frac{9\pi}{18} + \frac{16\pi}{18}\]

Sabiramo brojioce:

\[= \frac{(4 + 8 + 9 + 16)\pi}{18} = \frac{37\pi}{18}\]
Proveri se: \(4 + 8 + 9 + 16 = 37\). Tačno!
💡 Ključni uvid

Ključni trik je množenje sa \(2\sin x\) koje pretvara zbir kosinusa u teleskopski niz sinusa. Ovo je standardna tehnika za jednačine oblika \(\cos x + \cos 2x + \cdots + \cos nx = 0\). Nakon teleskopskog sažimanja, jednačina se faktoriše u jednostavne faktore: \(\sin 2x = 0\) ili \(\cos 3x = -\frac{1}{2}\).

Konačan odgovor
\(\dfrac{2\pi}{9} + \dfrac{4\pi}{9} + \dfrac{\pi}{2} + \dfrac{8\pi}{9} = \dfrac{37\pi}{18}\)
(A) \(\dfrac{37}{18}\pi\)
(B) \(\dfrac{23}{14}\pi\)
(C) \(\dfrac{8}{3}\pi\)
(D) \(\dfrac{35}{18}\pi\)
(E) \(\dfrac{20}{9}\pi\)
Verifikacija: Svako od 4 rešenja se može numerički proveriti uvrštavanjem u \(\cos x + \cos 2x + \cos 4x\). Na primer, za \(x = \frac{\pi}{2}\): \(\cos\frac{\pi}{2} + \cos\pi + \cos 2\pi = 0 + (-1) + 1 = 0\). Zbir \(\frac{37}{18}\pi \approx 6{,}458\) je blizu \(2\pi \approx 6{,}283\), što je razumno za četiri ugla raspoređena u prvom periodu.
⚠️ Česte greške
Zaboravljanje provere \(\sin x = 0\): Kad množimo jednačinu sa \(2\sin x\), uvodimo potencijalna lažna rešenja. Obavezno proveriti da \(x = n\pi\) ne zadovoljava originalnu jednačinu.
Propuštanje rešenja iz \(\sin 2x = 0\): Rešenje \(x = \frac{\pi}{2}\) dolazi iz prvog faktora i lako se previdi ako se fokusiramo samo na \(\cos 3x = -\frac{1}{2}\). Bez njega, zbir bi bio \(\frac{14\pi}{9}\) — nijedna od ponuđenih opcija.

Savet

Pomnoži sa \(2\sin x\) i primeni istu tehniku teleskopskog sažimanja. Dobićeš \(\sin 6x = 0\) (uz uslov \(\sin x \neq 0\)), pa rešenja su \(x = \frac{k\pi}{6}\) za \(k = 1, 2, 3, 4, 5, 7, \ldots\) (bez \(k = 6\)).
