Razvoj petog stepena sinusa po višestrukim uglovima

Trigonometrija — formule za redukciju stepena, linearizacija trigonometrijskih izraza

Zadatak

Ako je izraz \(A\sin x + B\sin 3x + C\sin 5x\) identički jednak \(\sin^5 x\), pri čemu su \(A\), \(B\) i \(C\) konstante, koliko je \(A\)?

(A)
\(\dfrac{3}{4}\)
(B)
\(\dfrac{1}{8}\)
(C)
\(\dfrac{5}{8}\)
(D)
\(\dfrac{3}{8}\)
(E)
\(\dfrac{1}{16}\)

Plan rešavanja

Izrazićemo \(\sin^5 x\) kao linearnu kombinaciju sinusa višestrukih uglova koristeći formule za redukciju stepena. Uporedićemo koeficijente uz \(\sin x\), \(\sin 3x\) i \(\sin 5x\) sa leve i desne strane identiteta da odredimo \(A\).

Podsetnik iz teorije

Formule za redukciju stepena i linearizaciju

Ojlerova formula: \(\sin x = \dfrac{e^{ix} - e^{-ix}}{2i}\), odakle stepenovanje i primena binomne formule daje razvoj po višestrukim uglovima.

Binomna formula: \((a+b)^n = \displaystyle\sum_{k=0}^{n}\binom{n}{k}a^{n-k}b^k\)

Ključni identitet: \(e^{inx} - e^{-inx} = 2i\sin nx\)

Korisna formula za sin²: \(\sin^2 x = \dfrac{1 - \cos 2x}{2}\)

Produkt sinusa: \(\sin x \cdot \sin^2 x = \sin x \cdot \dfrac{1-\cos 2x}{2}\)

Vizuelni prikaz

Graf funkcije \(\sin^5 x\) (zeleno) i njenog razvoja \(\frac{5}{8}\sin x - \frac{5}{16}\sin 3x + \frac{1}{16}\sin 5x\) (isprekidano plavo) — poklapaju se savršeno

Rešenje korak po korak

1
Zapišimo \(\sin^2 x\) pomoću formule za redukciju

Krenimo od osnovne formule za redukciju stepena:

\[\sin^2 x = \frac{1 - \cos 2x}{2}\]

Ovo ćemo koristiti kao gradivni element za veće stepene.

2
Izrazimo \(\sin^4 x\) koristeći \(\sin^2 x\)

Kvadriramo prethodni rezultat:

\[\sin^4 x = \left(\sin^2 x\right)^2 = \left(\frac{1 - \cos 2x}{2}\right)^2 = \frac{1 - 2\cos 2x + \cos^2 2x}{4}\]

Sada primenimo formulu redukcije na \(\cos^2 2x\):

\[\cos^2 2x = \frac{1 + \cos 4x}{2}\]

Uvrstimo:

\[\sin^4 x = \frac{1 - 2\cos 2x + \frac{1 + \cos 4x}{2}}{4} = \frac{3 - 4\cos 2x + \cos 4x}{8}\]
3
Izrazimo \(\sin^5 x = \sin^4 x \cdot \sin x\)

Pomnožimo obe strane sa \(\sin x\):

\[\sin^5 x = \frac{3\sin x - 4\sin x\cos 2x + \sin x\cos 4x}{8}\]

Sada treba da pretvorimo svaki član u sinuse višestrukih uglova.

Proveri sebe: Koji proizvod-u-zbir identitet koristimo?
Koristimo identitet: \(\sin \alpha \cos \beta = \frac{1}{2}[\sin(\alpha+\beta) + \sin(\alpha-\beta)]\). Ovo pretvaramo proizvode u zbir sinusa.
4
Pretvorimo \(\sin x \cos 2x\) u zbir

Primenimo identitet \(\sin\alpha\cos\beta = \frac{1}{2}[\sin(\alpha+\beta)+\sin(\alpha-\beta)]\):

\[\sin x \cos 2x = \frac{1}{2}[\sin(x+2x) + \sin(x-2x)] = \frac{1}{2}[\sin 3x + \sin(-x)]\]

Pošto je \(\sin(-x) = -\sin x\):

\[\sin x \cos 2x = \frac{1}{2}[\sin 3x - \sin x]\]
5
Pretvorimo \(\sin x \cos 4x\) u zbir

Isti identitet, ali sa \(\beta = 4x\):

\[\sin x \cos 4x = \frac{1}{2}[\sin(x+4x) + \sin(x-4x)] = \frac{1}{2}[\sin 5x + \sin(-3x)]\]

Pošto je \(\sin(-3x) = -\sin 3x\):

\[\sin x \cos 4x = \frac{1}{2}[\sin 5x - \sin 3x]\]
6
Uvrstimo sve u izraz za \(\sin^5 x\)

Zamenjujemo u izraz iz koraka 3:

\[\sin^5 x = \frac{1}{8}\left[3\sin x - 4 \cdot \frac{1}{2}(\sin 3x - \sin x) + \frac{1}{2}(\sin 5x - \sin 3x)\right]\]

Sredimo koeficijente:

\[\sin^5 x = \frac{1}{8}\left[3\sin x - 2\sin 3x + 2\sin x + \frac{1}{2}\sin 5x - \frac{1}{2}\sin 3x\right]\]

Grupišemo po sinusima:

\[\sin^5 x = \frac{1}{8}\left[(3+2)\sin x + \left(-2-\frac{1}{2}\right)\sin 3x + \frac{1}{2}\sin 5x\right]\]
Proveri sebe: Koliko je \(3+2\)?
Broj 3 dolazi od originalnog \(3\sin x\), a broj 2 od \(-4 \cdot (-\frac{1}{2})\sin x = +2\sin x\). Dakle, koeficijent uz \(\sin x\) je \(3+2=5\).
7
Završni oblik i očitavanje koeficijenata

Konačno dobijamo:

\[\sin^5 x = \frac{1}{8}\left[5\sin x - \frac{5}{2}\sin 3x + \frac{1}{2}\sin 5x\right]\]

Odnosno:

\[\boxed{\sin^5 x = \frac{5}{8}\sin x - \frac{5}{16}\sin 3x + \frac{1}{16}\sin 5x}\]

Poređenjem sa \(A\sin x + B\sin 3x + C\sin 5x\) očitavamo:

\[A = \frac{5}{8}, \qquad B = -\frac{5}{16}, \qquad C = \frac{1}{16}\]
Proveri sebe: Da li formula daje tačnu vrednost za \(x = \frac{\pi}{2}\)?

Za \(x = \frac{\pi}{2}\): \(\sin^5\frac{\pi}{2} = 1\).

\(\frac{5}{8}\sin\frac{\pi}{2} - \frac{5}{16}\sin\frac{3\pi}{2} + \frac{1}{16}\sin\frac{5\pi}{2}\)

\(= \frac{5}{8}\cdot 1 - \frac{5}{16}\cdot(-1) + \frac{1}{16}\cdot 1 = \frac{10}{16} + \frac{5}{16} + \frac{1}{16} = \frac{16}{16} = 1\) ✓

Ključni uvid

Svaki stepen sinusa (ili kosinusa) može se izraziti kao linearna kombinacija sinusa (ili kosinusa) višestrukih uglova. Za \(\sin^5 x\), koeficijent uz \(\sin x\) u tom razvoju iznosi \(\frac{10}{16} = \frac{5}{8}\). Ovaj rezultat proističe iz binomnih koeficijenata: \(\binom{5}{2} = 10\) daje brojilac, a \(2^4 = 16\) imenilac.

Konačan odgovor
\(A = \dfrac{5}{8}\)
Tačan odgovor je (C)
(A) \(\frac{3}{4}\)
(B) \(\frac{1}{8}\)
(C) \(\frac{5}{8}\)
(D) \(\frac{3}{8}\)
(E) \(\frac{1}{16}\)

Verifikacija: za \(x=\frac{\pi}{2}\), razvoj daje \(\frac{5}{8}+\frac{5}{16}+\frac{1}{16}=1=\sin^5\frac{\pi}{2}\) ✓

Česte greške

Pogrešan predznak u proizvod-u-zbir formuli: Kod \(\sin x \cos 2x\) lako se pogreši sa \(\sin(x-2x) = \sin(-x) = -\sin x\). Zaboraviti minus ovde dovodi do potpuno pogrešnih koeficijenata.
Mešanje \(\frac{1}{16}\) sa \(\frac{5}{8}\): Koeficijent \(C = \frac{1}{16}\) je ponuđen kao opcija (E). Pažljivo čitajte pitanje — traži se \(A\) (koeficijent uz \(\sin x\)), a ne \(C\) (koeficijent uz \(\sin 5x\)).

Dodatni izazov

Primenom istog postupka, izrazite \(\cos^4 x\) kao linearnu kombinaciju kosinusa višestrukih uglova. Koliki je koeficijent uz \(\cos 0x = 1\) (slobodan član)?

Nagoveštaj
Krenite od \(\cos^2 x = \frac{1+\cos 2x}{2}\), pa kvadrirajte. Koeficijent uz konstantu je \(\frac{3}{8}\).
