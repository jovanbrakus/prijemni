Trigonometrijski identitet za peti stepen sinusa

Trigonometrija -- razvoj stepena trigonometrijskih funkcija u zbir višestrukih uglova

Zadatak

Ako je izraz

\[a \cdot \sin x + b \cdot \sin 3x + c \cdot \sin 5x\]

identički jednak \(\sin^5 x\), koliko je \(a\)?

(A)
\(\dfrac{1}{8}\)
(B)
\(\dfrac{1}{16}\)
(C)
\(\dfrac{3}{4}\)
(D)
\(\dfrac{5}{8}\)
(E)
\(\dfrac{3}{8}\)

Plan rešavanja

Razvićemo \(\sin^5 x\) koristeći formule za snižavanje stepena i produkt-u-sumu identitete. Najpre ćemo \(\sin^4 x\) izraziti pomoću kosinusa dvostrukog i četvorostrukog ugla, pa pomnožiti sa \(\sin x\) i primeniti formulu \(\sin A \cos B = \tfrac{1}{2}[\sin(A+B) + \sin(A-B)]\) da bismo dobili linearnu kombinaciju \(\sin x\), \(\sin 3x\) i \(\sin 5x\).

Podsetnik iz teorije

Ključne formule za trigonometrijske stepene

Snižavanje stepena (poluugaone formule):

\[\sin^2 x = \frac{1 - \cos 2x}{2}, \qquad \cos^2 x = \frac{1 + \cos 2x}{2}\]

Produkt u zbir:

\[\sin A \cdot \cos B = \frac{1}{2}\bigl[\sin(A + B) + \sin(A - B)\bigr]\]

Poznati razvoj za sin⁵x:

\[\sin^5 x = \frac{1}{16}\bigl(10\sin x - 5\sin 3x + \sin 5x\bigr)\]

Ova formula se može izvesti iz opšte Čebiševljeve dekompozicije stepena trigonometrijskih funkcija. Ideja je da se \(\sin^n x\) za neparno \(n\) uvek može zapisati kao linearna kombinacija \(\sin kx\) za neparne vrednosti \(k\).

Vizuelni prikaz

Grafik prikazuje funkciju \(\sin^5 x\) (zelena) i njenu dekompoziciju na tri sinusne komponente. Koristite klizač da vidite doprinos svake komponente.

Sve komponente

0 = samo sin⁵x, 1 = + a·sin x, 2 = + b·sin 3x, 3 = sve tri komponente

Rešenje korak po korak

1

Zapisujemo sin⁴x pomoću formule za snižavanje stepena

Polazimo od \(\sin^5 x = \sin^4 x \cdot \sin x\). Najpre razvijamo \(\sin^4 x\):

\[\sin^2 x = \frac{1 - \cos 2x}{2}\]

Kvadriramo obe strane:

\[\sin^4 x = \left(\frac{1 - \cos 2x}{2}\right)^2 = \frac{1 - 2\cos 2x + \cos^2 2x}{4}\]
2

Primenjujemo formulu za cos²2x

Koristimo istu formulu za snižavanje stepena, ali za \(\cos^2 2x\):

\[\cos^2 2x = \frac{1 + \cos 4x}{2}\]

Zamenjujemo u izraz za \(\sin^4 x\):

\[\sin^4 x = \frac{1 - 2\cos 2x + \frac{1 + \cos 4x}{2}}{4} = \frac{2 - 4\cos 2x + 1 + \cos 4x}{8}\]
\[\sin^4 x = \frac{3 - 4\cos 2x + \cos 4x}{8}\]
Proveri sebe: Koliko je sin⁴(π/2)?

\(\sin^4(\pi/2) = 1^4 = 1\). Po formuli: \(\frac{3 - 4\cos\pi + \cos 2\pi}{8} = \frac{3 - 4 \cdot (-1) + 1}{8} = \frac{8}{8} = 1\). Tačno!

3

Množimo sa sin x

Sada dobijamo \(\sin^5 x\):

\[\sin^5 x = \frac{3 - 4\cos 2x + \cos 4x}{8} \cdot \sin x\]

Raspisujemo:

\[\sin^5 x = \frac{1}{8}\bigl(3\sin x - 4\sin x \cos 2x + \sin x \cos 4x\bigr)\]
4

Primenjujemo produkt-u-sumu formulu na sinx·cos2x

Koristimo \(\sin A \cos B = \frac{1}{2}[\sin(A+B) + \sin(A-B)]\):

\[\sin x \cos 2x = \frac{1}{2}\bigl[\sin(x + 2x) + \sin(x - 2x)\bigr] = \frac{1}{2}\bigl[\sin 3x + \sin(-x)\bigr]\]

Pošto je \(\sin(-x) = -\sin x\):

\[\sin x \cos 2x = \frac{1}{2}(\sin 3x - \sin x)\]

Dakle:

\[4\sin x \cos 2x = 2\sin 3x - 2\sin x\]
5

Primenjujemo produkt-u-sumu formulu na sinx·cos4x

Analogno:

\[\sin x \cos 4x = \frac{1}{2}\bigl[\sin(x + 4x) + \sin(x - 4x)\bigr] = \frac{1}{2}\bigl[\sin 5x + \sin(-3x)\bigr]\]

Pošto je \(\sin(-3x) = -\sin 3x\):

\[\sin x \cos 4x = \frac{1}{2}(\sin 5x - \sin 3x)\]
Proveri sebe: Za x = π/6, koliko je sin(π/6)·cos(4π/6)?

Leva strana: \(\sin\frac{\pi}{6} \cdot \cos\frac{2\pi}{3} = \frac{1}{2} \cdot \left(-\frac{1}{2}\right) = -\frac{1}{4}\).

Desna strana: \(\frac{1}{2}(\sin\frac{5\pi}{6} - \sin\frac{3\pi}{6}) = \frac{1}{2}\left(\frac{1}{2} - 1\right) = \frac{1}{2} \cdot \left(-\frac{1}{2}\right) = -\frac{1}{4}\). Slaže se!

6

Zamenjujemo sve nazad i sređujemo

Vraćamo u izraz iz koraka 3:

\[\sin^5 x = \frac{1}{8}\Bigl(3\sin x - (2\sin 3x - 2\sin x) + \frac{1}{2}(\sin 5x - \sin 3x)\Bigr)\]

Skupljamo članove uz svaki sinusni izraz:

\[\begin{align} \sin x: &\quad 3 + 2 = 5 \\ \sin 3x: &\quad -2 - \tfrac{1}{2} = -\tfrac{5}{2} \\ \sin 5x: &\quad \tfrac{1}{2} \end{align}\]

Dakle:

\[\sin^5 x = \frac{1}{8}\left(5\sin x - \frac{5}{2}\sin 3x + \frac{1}{2}\sin 5x\right)\]
7

Završna forma i čitanje koeficijenata

Množimo brojilac i imenilac da sredimo razlomke:

\[\sin^5 x = \frac{1}{16}\bigl(10\sin x - 5\sin 3x + \sin 5x\bigr)\]

Raspisujemo:

\[\sin^5 x = \frac{10}{16}\sin x - \frac{5}{16}\sin 3x + \frac{1}{16}\sin 5x\]

Poređenjem sa \(a\sin x + b\sin 3x + c\sin 5x\) čitamo:

\[a = \frac{10}{16} = \frac{5}{8}, \qquad b = -\frac{5}{16}, \qquad c = \frac{1}{16}\]
Proveri sebe: Da li a + b + c čini smisla? (Hint: uvrsti x = π/2)

Za \(x = \frac{\pi}{2}\): \(\sin^5\frac{\pi}{2} = 1\).

\(a\sin\frac{\pi}{2} + b\sin\frac{3\pi}{2} + c\sin\frac{5\pi}{2} = a - b + c = \frac{5}{8} + \frac{5}{16} + \frac{1}{16} = \frac{10 + 5 + 1}{16} = \frac{16}{16} = 1\). Potvrđeno!

Ključni uvid

Stepen trigonometrijske funkcije uvek se može zapisati kao linearna kombinacija funkcija višestrukih uglova. Za neparan stepen sinusa dobijamo samo sinusne članove. Ključna tehnika je snižavanje stepena pomoću \(\sin^2 x = \frac{1-\cos 2x}{2}\), praćeno produkt-u-sumu transformacijom. Koeficijent uz \(\sin x\) u razvoju \(\sin^5 x\) je:

\[a = \frac{10}{16} = \color{#4ade80}{\frac{5}{8}}\]

Konačan odgovor

\(\displaystyle \text{(D)}\ a = \frac{5}{8}\)
(A) \(\dfrac{1}{8}\)
(B) \(\dfrac{1}{16}\)
(C) \(\dfrac{3}{4}\)
(D) \(\dfrac{5}{8}\)
(E) \(\dfrac{3}{8}\)
Verifikacija: Uvrštavamo \(x = \frac{\pi}{2}\): leva strana je \(\sin^5\frac{\pi}{2} = 1\). Desna strana: \(\frac{5}{8} \cdot 1 + \left(-\frac{5}{16}\right) \cdot (-1) + \frac{1}{16} \cdot 1 = \frac{10}{16} + \frac{5}{16} + \frac{1}{16} = \frac{16}{16} = 1\). Identitet važi.

Česte greške

Mešanje sa koeficijentom c: Odgovor (B) \(\frac{1}{16}\) je zapravo vrednost koeficijenta \(c\) (uz \(\sin 5x\)), ne \(a\). Pažljivo pročitajte šta se traži!
Zaboravljanje minusa u sin(-x): Prilikom primene produkt-u-sumu formule, \(\sin(-x) = -\sin x\). Ako se ovo previdi, koeficijenti postaju pogrešni.

Dodatni izazov

Neka je \(\cos^5 x = p\cos x + q\cos 3x + r\cos 5x\). Koliko je \(p\)?

Nagoveštaj

Pristup je analogan: koristite \(\cos^2 x = \frac{1 + \cos 2x}{2}\) umesto sinusne verzije. Razvoj daje:

\(\cos^5 x = \frac{1}{16}(10\cos x + 5\cos 3x + \cos 5x)\)

Dakle \(p = \frac{10}{16} = \frac{5}{8}\) -- isti koeficijent kao za sinusni slučaj!
