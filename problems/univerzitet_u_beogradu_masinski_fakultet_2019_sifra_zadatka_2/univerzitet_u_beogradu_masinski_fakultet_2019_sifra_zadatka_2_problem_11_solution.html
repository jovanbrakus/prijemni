Trigonometrijski identitet -- uprošćavanje razlomka

Trigonometrija -- faktorizacija zbira kubova i primena osnovnih trigonometrijskih identiteta

Zadatak

Izraz

\[\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x}\]

je identički jednak:

(A)
\(\dfrac{\cos x - \sin x}{2}\)
(B)
\(1\)
(C)
\(\dfrac{\cos x}{2}\)
(D)
\(\dfrac{1 + \cos^2 x}{4}\)
(E)
\(\dfrac{\sin x + \cos x}{2}\)

Plan rešavanja

Brojilac ćemo faktorizovati pomoću formule za zbir kubova \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\). Imenilac ćemo transformisati koristeći formulu za dupli ugao \(\sin 2x = 2\sin x \cos x\). Nakon toga, skratićemo zajednički faktor i dobiti konačan rezultat.

Podsetnik iz teorije

Ključne formule i identiteti

Zbir kubova:

\[a^3 + b^3 = (a + b)(a^2 - ab + b^2)\]

Osnovni trigonometrijski identitet:

\[\sin^2 x + \cos^2 x = 1\]

Formula za dupli ugao:

\[\sin 2x = 2 \sin x \cos x\]

Kombinacija ovih tri formule je dovoljna da se izraz potpuno uprosti.

Vizuelni prikaz

Grafik pokazuje da se polazni izraz (plava linija) savršeno poklapa sa \((\sin x + \cos x)/2\) (zelena isprekidana linija). Pomerajte klizač da istaknete konkretnu vrednost ugla \(x\).

Rešenje korak po korak

1

Faktorizujemo brojilac

Brojilac ima oblik \(a^3 + b^3\), gde je \(a = \sin x\) i \(b = \cos x\). Primenjujemo formulu za zbir kubova:

\[\sin^3 x + \cos^3 x = (\sin x + \cos x)(\sin^2 x - \sin x \cos x + \cos^2 x)\]
2

Primenjujemo osnovni trigonometrijski identitet

U drugom faktoru imamo \(\sin^2 x + \cos^2 x\), što je jednako \(1\):

\[\sin^2 x - \sin x \cos x + \cos^2 x = (\sin^2 x + \cos^2 x) - \sin x \cos x\]
\[= 1 - \sin x \cos x\]

Dakle, brojilac postaje:

\[\sin^3 x + \cos^3 x = (\sin x + \cos x)(1 - \sin x \cos x)\]
Proveri sebe: Koliko iznosi \(\sin^2 x + \cos^2 x\) za bilo koje \(x\)?

Uvek iznosi \(1\). To je fundamentalni Pitagorin identitet u trigonometriji.

3

Transformišemo imenilac

Koristimo formulu za dupli ugao \(\sin 2x = 2\sin x \cos x\):

\[2 - \sin 2x = 2 - 2\sin x \cos x\]

Iznosimo zajednički faktor \(2\):

\[= 2(1 - \sin x \cos x)\]
Proveri sebe: Koji faktor se javlja i u brojiocu i u imeniocu?

Faktor \((1 - \sin x \cos x)\) se pojavljuje na oba mesta, pa će se skratiti.

4

Kraćenje zajedničkog faktora

Sada zapisujemo ceo razlomak sa faktorizovanim brojiocem i imeniocem:

\[\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \frac{(\sin x + \cos x)\cancel{(1 - \sin x \cos x)}}{2\cancel{(1 - \sin x \cos x)}}\]

Faktor \((1 - \sin x \cos x)\) možemo skratiti jer je uvek različit od nule. Naime, \(\sin x \cos x = \frac{1}{2}\sin 2x\), a \(\sin 2x \in [-1, 1]\), pa \(\sin x \cos x \in \left[-\frac{1}{2}, \frac{1}{2}\right]\), što znači \(1 - \sin x \cos x \geq \frac{1}{2} > 0\).

\[= \frac{\sin x + \cos x}{2}\]
5

Verifikacija za konkretnu vrednost

Proverimo rezultat za \(x = 0\):

\[\text{Brojilac: } \sin^3 0 + \cos^3 0 = 0 + 1 = 1\]
\[\text{Imenilac: } 2 - \sin 0 = 2 - 0 = 2\]
\[\text{Razlomak: } \frac{1}{2}\]

Kontrola sa odgovorom (E):

\[\frac{\sin 0 + \cos 0}{2} = \frac{0 + 1}{2} = \frac{1}{2} \quad \checkmark\]
Proveri sebe: Koliko iznosi polazni izraz za \(x = \frac{\pi}{4}\)?

Za \(x = \frac{\pi}{4}\): \(\sin x = \cos x = \frac{\sqrt{2}}{2}\).

Brojilac: \(2 \cdot \left(\frac{\sqrt{2}}{2}\right)^3 = 2 \cdot \frac{2\sqrt{2}}{8} = \frac{\sqrt{2}}{2}\).

Imenilac: \(2 - \sin\frac{\pi}{2} = 2 - 1 = 1\).

Rezultat: \(\frac{\sqrt{2}}{2}\). Kontrola: \(\frac{\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}}{2} = \frac{\sqrt{2}}{2}\). Tačno!

Ključni uvid

Prepoznavanje obrasca zbira kubova u brojiocu je suština ovog zadatka. Nakon faktorizacije \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\) i zamene \(\sin^2 x + \cos^2 x = 1\), pojavljuje se faktor \((1 - \sin x \cos x)\) koji se krati sa imeniocem:

\[\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \color{#4ade80}{\frac{\sin x + \cos x}{2}}\]

Konačan odgovor

\(\displaystyle \text{(E)}\ \frac{\sin x + \cos x}{2}\)
(A) \(\frac{\cos x - \sin x}{2}\)
(B) \(1\)
(C) \(\frac{\cos x}{2}\)
(D) \(\frac{1 + \cos^2 x}{4}\)
(E) \(\frac{\sin x + \cos x}{2}\)
Verifikacija: Za \(x = 0\): polazni izraz = \(\frac{0+1}{2-0} = \frac{1}{2}\), a \(\frac{\sin 0 + \cos 0}{2} = \frac{1}{2}\). Za \(x = \frac{\pi}{2}\): polazni izraz = \(\frac{1+0}{2-0} = \frac{1}{2}\), a \(\frac{1+0}{2} = \frac{1}{2}\). Za \(x = \frac{\pi}{4}\): polazni izraz = \(\frac{\sqrt{2}/2}{1} = \frac{\sqrt{2}}{2}\), a \(\frac{\sqrt{2}}{2}\). Rezultat se poklapa u svim slučajevima.

Česte greške

Neprepoznavanje obrasca zbira kubova: Mnogi učenici pokušavaju da razviju \(\sin^3 x + \cos^3 x\) koristeći trigonometrijske formule za višestruke uglove, što vodi u nepotrebno komplikovano računanje. Ključ je prepoznati algebarski obrazac \(a^3 + b^3\).
Greška u znaku pri faktorizaciji: Formula za zbir kubova sadrži \(a^2 - ab + b^2\) (sa minusom ispred \(ab\)). Česta greška je staviti plus umesto minusa, što dovodi do pogrešnog rezultata.

Dodatni izazov

Koliko iznosi izraz \(\dfrac{\sin^3 x - \cos^3 x}{2 + \sin 2x}\)?

Nagoveštaj

Koristite formulu za razliku kubova: \(a^3 - b^3 = (a - b)(a^2 + ab + b^2)\). Postupak je analogan -- u imeniocu ćete dobiti \(2(1 + \sin x \cos x)\), a u drugom faktoru brojioca \(\sin^2 x + \sin x \cos x + \cos^2 x = 1 + \sin x \cos x\).

Odgovor: \(\dfrac{\sin x - \cos x}{2}\).
