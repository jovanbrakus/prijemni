Trigonometrijski identitet -- zbir kubova sinusa i kosinusa

Trigonometrija -- faktorizacija i uprošćavanje trigonometrijskih izraza

Zadatak

Izraz

\[\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x}\]

je identički jednak:

(A)
\(\dfrac{\cos x - \sin x}{2}\)
(B)
\(1\)
(C)
\(\dfrac{\cos x}{2}\)
(D)
\(\dfrac{1 + \cos^2 x}{4}\)
(E)
\(\dfrac{\sin x + \cos x}{2}\)

Plan rešavanja

Prepoznaćemo brojilac kao zbir kubova i faktorizovati ga formulom \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\). Zatim ćemo imenilac prepisati koristeći formulu za dupli ugao \(\sin 2x = 2\sin x \cos x\). Na kraju ćemo skratiti zajednički faktor i dobiti jednostavan izraz.

Podsetnik iz teorije

Ključne formule i identiteti

Zbir kubova:

\[a^3 + b^3 = (a + b)(a^2 - ab + b^2)\]

Pitagorin trigonometrijski identitet:

\[\sin^2 x + \cos^2 x = 1\]

Formula za dupli ugao:

\[\sin 2x = 2 \sin x \cos x\]

Napomena o domenu: Izraz \(1 - \sin x \cos x\) je uvek pozitivan jer je \(\sin x \cos x = \frac{1}{2}\sin 2x \leq \frac{1}{2} < 1\), tako da imenilac nikad nije nula.

Vizuelna verifikacija identiteta

Grafici originalne funkcije \(f(x) = \frac{\sin^3 x + \cos^3 x}{2 - \sin 2x}\) i uprošćene funkcije \(g(x) = \frac{\sin x + \cos x}{2}\) se potpuno preklapaju, što potvrđuje da je identitet tačan.

▬▬ f(x) = (sin³x + cos³x) / (2 − sin 2x) - - - g(x) = (sin x + cos x) / 2

Obe krive se savršeno poklapaju -- identitet važi za svako x

Rešenje korak po korak

1

Prepoznajemo oblik brojioca

Brojilac \(\sin^3 x + \cos^3 x\) je zbir kubova. Primenićemo formulu:

\[a^3 + b^3 = (a + b)(a^2 - ab + b^2)\]

gde je \(a = \sin x\) i \(b = \cos x\).

2

Faktorizujemo brojilac

Primenom formule za zbir kubova:

\[\sin^3 x + \cos^3 x = (\sin x + \cos x)(\sin^2 x - \sin x \cos x + \cos^2 x)\]

Sada koristimo Pitagorin identitet \(\sin^2 x + \cos^2 x = 1\):

\[\sin^2 x - \sin x \cos x + \cos^2 x = 1 - \sin x \cos x\]

Dakle, brojilac postaje:

\[\sin^3 x + \cos^3 x = (\sin x + \cos x)(1 - \sin x \cos x)\]
Proveri sebe: Za \(x = 0\), koliko iznosi \(\sin^3 0 + \cos^3 0\)?

\(\sin^3 0 + \cos^3 0 = 0 + 1 = 1\). Proveravamo faktorizaciju: \((\sin 0 + \cos 0)(1 - \sin 0 \cos 0) = (0 + 1)(1 - 0) = 1\). Tačno!

3

Transformišemo imenilac

Primenjujemo formulu za dupli ugao \(\sin 2x = 2\sin x \cos x\):

\[2 - \sin 2x = 2 - 2\sin x \cos x\]

Iznosimo faktor 2:

\[2 - 2\sin x \cos x = 2(1 - \sin x \cos x)\]
4

Zapisujemo ceo razlomak

Zamenjujemo faktorizovane oblike u polazni izraz:

\[\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \frac{(\sin x + \cos x)(1 - \sin x \cos x)}{2(1 - \sin x \cos x)}\]
5

Skraćujemo zajednički faktor

Izraz \((1 - \sin x \cos x)\) se pojavljuje i u brojiocu i u imeniocu. Pošto je \(\sin x \cos x = \frac{1}{2}\sin 2x \leq \frac{1}{2}\), važi \(1 - \sin x \cos x \geq \frac{1}{2} > 0\), pa možemo slobodno da skratimo:

\[\frac{\cancel{(1 - \sin x \cos x)} \cdot (\sin x + \cos x)}{2 \cdot \cancel{(1 - \sin x \cos x)}} = \frac{\sin x + \cos x}{2}\]
Proveri sebe: Za \(x = \frac{\pi}{4}\), koliko iznosi originalni izraz?

Brojilac: \(\sin^3\frac{\pi}{4} + \cos^3\frac{\pi}{4} = 2 \cdot \left(\frac{\sqrt{2}}{2}\right)^3 = 2 \cdot \frac{2\sqrt{2}}{8} = \frac{\sqrt{2}}{2}\).

Imenilac: \(2 - \sin\frac{\pi}{2} = 2 - 1 = 1\).

Izraz = \(\frac{\sqrt{2}/2}{1} = \frac{\sqrt{2}}{2}\).

Proveravamo: \(\frac{\sin\frac{\pi}{4} + \cos\frac{\pi}{4}}{2} = \frac{\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}}{2} = \frac{\sqrt{2}}{2}\). Tačno!

6

Zaključak

Dati izraz je identički jednak:

\[\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \frac{\sin x + \cos x}{2}\]

Ovo odgovara opciji (E).

Ključni uvid

Celokupno rešenje se svodi na prepoznavanje zbira kubova u brojiocu i formule za dupli ugao u imeniocu. Obe transformacije dovode do zajedničkog faktora \((1 - \sin x \cos x)\), koji se skraćuje i ostavlja elegantan rezultat:

\[\frac{(\sin x + \cos x)\cancel{(1 - \sin x \cos x)}}{2\cancel{(1 - \sin x \cos x)}} = \color{#4ade80}{\frac{\sin x + \cos x}{2}}\]

Konačan odgovor

\(\displaystyle \text{(E)}\ \frac{\sin x + \cos x}{2}\)
(A) \(\dfrac{\cos x - \sin x}{2}\)
(B) \(1\)
(C) \(\dfrac{\cos x}{2}\)
(D) \(\dfrac{1 + \cos^2 x}{4}\)
(E) \(\dfrac{\sin x + \cos x}{2}\)
Verifikacija: Za \(x = 0\): originalni izraz = \(\frac{0 + 1}{2 - 0} = \frac{1}{2}\), a \(\frac{\sin 0 + \cos 0}{2} = \frac{1}{2}\). Za \(x = \frac{\pi}{4}\): originalni izraz = \(\frac{\sqrt{2}/2}{1} = \frac{\sqrt{2}}{2}\), a \(\frac{\sqrt{2}/2 + \sqrt{2}/2}{2} = \frac{\sqrt{2}}{2}\). Obe provere se slažu.

Česte greške

Neprepoznavanje zbira kubova: Bez formule \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\), nemoguće je faktorizovati brojilac. Mnogi učenici pokušavaju razne trigonometrijske supstitucije umesto da primene čistu algebarsku faktorizaciju.
Zaboravljanje da je \(\sin^2 x + \cos^2 x = 1\): Posle faktorizacije, u zagradi ostaje \(\sin^2 x - \sin x \cos x + \cos^2 x\). Ako se ne primeti da je \(\sin^2 x + \cos^2 x = 1\), izraz se ne može dalje uprostiti.

Dodatni izazov

Uprosti izraz \(\dfrac{\sin^3 x - \cos^3 x}{1 + \sin 2x}\). Koja formula za faktorizaciju je sada potrebna?

Nagoveštaj

Koristite formulu za razliku kubova: \(a^3 - b^3 = (a - b)(a^2 + ab + b^2)\).

Za imenilac: \(1 + \sin 2x = 1 + 2\sin x \cos x = \sin^2 x + 2\sin x \cos x + \cos^2 x = (\sin x + \cos x)^2\).

Rezultat: \(\dfrac{\sin x - \cos x}{(\sin x + \cos x)^2} \cdot (1 + \sin x \cos x)\). Pokušajte da završite!
