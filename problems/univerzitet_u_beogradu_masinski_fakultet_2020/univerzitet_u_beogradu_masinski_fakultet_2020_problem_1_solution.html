Plan rešavanja

Najpre ćemo decimalne brojeve pretvoriti u razlomke: \(x = -\frac{1}{8}\), \(y = \frac{9}{8}\). Zatim ćemo primeniti identitet za zbir kubova \(x^3 + y^3 = (x+y)(x^2 - xy + y^2)\), izračunati međuvrednosti \(x+y\) i \(xy\), i sve uvrstiti u izraz. Grupisanje članova \(3x + 3y = 3(x+y)\) dodatno pojednostavljuje račun.

Podsetnik iz teorije

Ključne formule i identiteti

Zbir kubova:

\[a^3 + b^3 = (a + b)(a^2 - ab + b^2)\]

Kvadrat zbira (za izračunavanje \(x^2 + y^2\)):

\[(a + b)^2 = a^2 + 2ab + b^2 \quad \Rightarrow \quad a^2 + b^2 = (a+b)^2 - 2ab\]

Prevođenje decimalnog zapisa u razlomak:

\[0{,}125 = \frac{125}{1000} = \frac{1}{8}, \qquad 1{,}125 = \frac{1125}{1000} = \frac{9}{8}\]

Deljenje razlomaka:

\[\frac{a/b}{c/d} = \frac{a}{b} \cdot \frac{d}{c}\]

Vizuelni prikaz

Dijagram prikazuje identitet za zbir kubova \(x^3 + y^3 = (x+y)(x^2 - xy + y^2)\) i tok pojednostavljivanja izraza. Koristite dugmad da se prebacujete između prikaza identiteta i toka računanja.

Pritisnite dugmad da promenite prikaz

Rešenje korak po korak

1

Pretvaramo decimalne brojeve u razlomke

Zadati decimalni brojevi imaju prepoznatljive razlomačke oblike:

\[x = -0{,}125 = -\frac{125}{1000} = -\frac{1}{8}\]
\[y = 1{,}125 = \frac{1125}{1000} = \frac{9}{8}\]

Zapamtite: \(0{,}125 = \frac{1}{8}\) je korisna jednakost koju vredi znati napamet.

2

Računamo zbir i proizvod

Izračunavamo \(x + y\) i \(xy\) jer će nam obe vrednosti trebati:

\[x + y = -\frac{1}{8} + \frac{9}{8} = \frac{-1 + 9}{8} = \frac{8}{8} = 1\]
\[xy = \left(-\frac{1}{8}\right) \cdot \frac{9}{8} = -\frac{9}{64}\]

Primetimo da je \(x + y = 1\) -- veoma zgodna vrednost koja će nam značajno olakšati račun!

Proveri sebe: Koliko iznosi \(3(x + y)\)?

Pošto je \(x + y = 1\), to je \(3(x + y) = 3 \cdot 1 = 3\).

3

Grupišemo članove u izrazu

Primetimo da se \(3x + 3y\) može faktorisati:

\[\frac{x^3 + y^3}{xy} + 3x + 3y = \frac{x^3 + y^3}{xy} + 3(x + y)\]

Pošto je \(x + y = 1\), drugi sabirak je jednostavno:

\[3(x + y) = 3 \cdot 1 = 3\]

Preostaje nam da izračunamo \(\dfrac{x^3 + y^3}{xy}\).

4

Primenjujemo identitet za zbir kubova

Koristimo formulu:

\[x^3 + y^3 = (x + y)(x^2 - xy + y^2)\]

Već znamo da je \(x + y = 1\), pa nam treba samo \(x^2 - xy + y^2\). Izračunajmo najpre \(x^2 + y^2\):

\[x^2 + y^2 = (x + y)^2 - 2xy = 1^2 - 2 \cdot \left(-\frac{9}{64}\right)\]
\[= 1 + \frac{9}{32} = \frac{32}{32} + \frac{9}{32} = \frac{41}{32}\]
5

Računamo \(x^2 - xy + y^2\)

Sada kombinujemo:

\[x^2 - xy + y^2 = (x^2 + y^2) - xy = \frac{41}{32} - \left(-\frac{9}{64}\right)\]
\[= \frac{41}{32} + \frac{9}{64} = \frac{82}{64} + \frac{9}{64} = \frac{91}{64}\]
Proveri sebe: Koliko onda iznosi \(x^3 + y^3\)?

Pošto je \(x + y = 1\), imamo \(x^3 + y^3 = 1 \cdot \frac{91}{64} = \frac{91}{64}\).

6

Delimo sa \(xy\)

Sada računamo prvi deo izraza:

\[\frac{x^3 + y^3}{xy} = \frac{\;\dfrac{91}{64}\;}{-\dfrac{9}{64}} = \frac{91}{64} \cdot \frac{64}{-9}\]

Šezdesetčetvorke se skrate:

\[= \frac{91}{-9} = -\frac{91}{9}\]
7

Sabiramo oba dela

Konačno, vraćamo se na ceo izraz:

\[\frac{x^3 + y^3}{xy} + 3(x+y) = -\frac{91}{9} + 3\]

Pretvaramo \(3\) u razlomak sa imeniocem \(9\):

\[= -\frac{91}{9} + \frac{27}{9} = \frac{-91 + 27}{9} = -\frac{64}{9}\]
Proveri sebe: Koliko je \(-64/9\) približno?

\(-\frac{64}{9} \approx -7{,}11\). To je negativan broj, što ima smisla jer je \(-\frac{91}{9} \approx -10{,}11\), a dodajemo samo \(3\).

Ključni uvid

Umesto da direktno računamo kubove razlomaka (što bi bilo zamorno), koristimo dva moćna alata: identitet za zbir kubova i činjenicu da je \(x + y = 1\). Zahvaljujući tome, \(x^3 + y^3\) se svodi na \(x^2 - xy + y^2\), a članovi \(3x + 3y\) postaju prosto \(3\):

\[\frac{x^3 + y^3}{xy} + 3(x+y) = -\frac{91}{9} + 3 = \color{#4ade80}{-\frac{64}{9}}\]

Konačan odgovor

\(\displaystyle \text{(E)}\ -\frac{64}{9}\)
(A) \(-\dfrac{125}{9}\)
(B) \(\dfrac{125}{9}\)
(C) \(\dfrac{9}{64}\)
(D) \(\dfrac{64}{9}\)
(E) \(-\dfrac{64}{9}\)
Verifikacija: Proveravamo direktnim računom. \(x^3 = \left(-\frac{1}{8}\right)^3 = -\frac{1}{512}\), \(y^3 = \left(\frac{9}{8}\right)^3 = \frac{729}{512}\). Dakle \(x^3 + y^3 = \frac{728}{512} = \frac{91}{64}\). Zatim: \(\frac{91/64}{-9/64} + 3 = -\frac{91}{9} + \frac{27}{9} = -\frac{64}{9}\). Rezultat se slaže.

Česte greške

Pogrešan predznak: Lako je zaboraviti da je \(x\) negativan. Ako se previdi minus pri računanju \(xy\), proizvod postaje pozitivan \(\frac{9}{64}\) umesto \(-\frac{9}{64}\), što vodi do odgovora (D) \(\frac{64}{9}\) umesto tačnog (E) \(-\frac{64}{9}\).
Zamena identiteta za zbir i razliku kubova: Studenti ponekad pomešaju \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\) sa \(a^3 - b^3 = (a-b)(a^2 + ab + b^2)\). Obratite pažnju na predznake u drugom faktoru!

Dodatni izazov

Za iste vrednosti \(x = -\frac{1}{8}\) i \(y = \frac{9}{8}\), koliko iznosi \(\dfrac{x^3 - y^3}{x - y}\)?

Nagoveštaj

Koristite identitet za razliku kubova: \(x^3 - y^3 = (x - y)(x^2 + xy + y^2)\). Šta se desi kad podelite obe strane sa \((x - y)\)?

Ostaje \(x^2 + xy + y^2 = \frac{41}{32} + \left(-\frac{9}{64}\right) = \frac{82 - 9}{64} = \frac{73}{64}\).
