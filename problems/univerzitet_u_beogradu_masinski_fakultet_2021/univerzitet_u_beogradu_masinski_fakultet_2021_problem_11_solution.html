Određivanje kosinusa pomoću tangensa

Trigonometrija — veza između trigonometrijskih funkcija i određivanje znaka u zadatom kvadrantu

Postavka zadatka

Ako je \(\operatorname{tg} x = 4\) i \(\pi < x < 2\pi\), koliko je \(\cos x\)?

Dato
\(\operatorname{tg} x = 4\)
Opseg
\(\pi < x < 2\pi\)

Ponuđeni odgovori

(A)
\(\dfrac{1}{17}\)
(B)
\(-\dfrac{1}{\sqrt{17}}\)
(C)
\(-\dfrac{4}{\sqrt{17}}\)
(D)
\(\dfrac{1}{\sqrt{17}}\)
(E)
\(\dfrac{4}{\sqrt{17}}\)

Plan rešavanja

Prvo ćemo odrediti u kom kvadrantu se nalazi ugao \(x\) na osnovu uslova \(\pi < x < 2\pi\) i \(\operatorname{tg} x > 0\). Zatim ćemo primeniti fundamentalni trigonometrijski identitet \(1 + \operatorname{tg}^2 x = \dfrac{1}{\cos^2 x}\) da izračunamo \(\cos x\), koristeći pravilan znak za odgovarajući kvadrant.

📐 Ključne formule i teoreme
  • Osnovni trigonometrijski identitet: \(\sin^2 x + \cos^2 x = 1\)
  • Veza tangensa i kosinusa: Deljenjem osnovnog identiteta sa \(\cos^2 x\) dobijamo: \[1 + \operatorname{tg}^2 x = \frac{1}{\cos^2 x}\]
  • Znakovi u kvadrantima:
    I kvadrant: svi pozitivni
    II kvadrant: \(\sin > 0\), ostali negativni
    III kvadrant: \(\operatorname{tg} > 0\), \(\sin < 0\), \(\cos < 0\)
    IV kvadrant: \(\cos > 0\), \(\sin < 0\), \(\operatorname{tg} < 0\)
  • Definicija tangensa: \(\operatorname{tg} x = \dfrac{\sin x}{\cos x}\)

Jedinični krug — vizuelni prikaz

Prikaz ugla \(x\) u III kvadrantu gde je \(\operatorname{tg} x = 4\)

Rešenje korak po korak

1

Određivanje kvadranta

Dato nam je da je \(\pi < x < 2\pi\), što znači da je \(x\) u III ili IV kvadrantu.

Takođe znamo da je \(\operatorname{tg} x = 4 > 0\).

Tangens je pozitivan u I i III kvadrantu. Pošto je \(x\) između \(\pi\) i \(2\pi\), jedina mogućnost je:

\[\pi < x < \frac{3\pi}{2} \quad \text{(III kvadrant)}\]

U III kvadrantu i sinus i kosinus su negativni, pa je njihov količnik (tangens) pozitivan.

2

Primena trigonometrijskog identiteta

Koristimo identitet koji povezuje tangens i kosinus:

\[1 + \operatorname{tg}^2 x = \frac{1}{\cos^2 x}\]

Zamenjujemo \(\operatorname{tg} x = 4\):

\[1 + 4^2 = \frac{1}{\cos^2 x}\]
\[1 + 16 = \frac{1}{\cos^2 x}\]
\[17 = \frac{1}{\cos^2 x}\]
🔍 Proveri se: Koliko je \(\cos^2 x\)?
\(\cos^2 x = \dfrac{1}{17}\)
3

Izračunavanje \(\cos^2 x\)

Iz prethodnog koraka dobijamo:

\[\cos^2 x = \frac{1}{17}\]

Vađenjem korena:

\[\cos x = \pm \frac{1}{\sqrt{17}}\]
4

Određivanje znaka kosinusa

U koraku 1 smo ustanovili da je \(x\) u III kvadrantu.

U III kvadrantu kosinus je negativan, pa biramo negativnu vrednost:

\[\cos x = -\frac{1}{\sqrt{17}}\]
🔍 Proveri se: Koliko je \(\sin x\) ako je \(\cos x = -\frac{1}{\sqrt{17}}\)?
\(\sin x = \operatorname{tg} x \cdot \cos x = 4 \cdot \left(-\dfrac{1}{\sqrt{17}}\right) = -\dfrac{4}{\sqrt{17}}\)
Negativan sinus — to je konzistentno sa III kvadrantom!
5

Verifikacija rešenja

Proverimo da li su vrednosti konzistentne koristeći \(\sin^2 x + \cos^2 x = 1\):

\[\sin^2 x + \cos^2 x = \left(-\frac{4}{\sqrt{17}}\right)^2 + \left(-\frac{1}{\sqrt{17}}\right)^2 = \frac{16}{17} + \frac{1}{17} = \frac{17}{17} = 1 \; \checkmark\]

Proverimo i tangens:

\[\operatorname{tg} x = \frac{\sin x}{\cos x} = \frac{-\frac{4}{\sqrt{17}}}{-\frac{1}{\sqrt{17}}} = \frac{4}{1} = 4 \; \checkmark\]

Obe provere potvrđuju tačnost rešenja.

Ključni uvid

Identitet \(1 + \operatorname{tg}^2 x = \dfrac{1}{\cos^2 x}\) daje nam \(\cos^2 x\), ali ne i znak. Znak kosinusa određujemo iz kvadranta u kom se ugao nalazi!

\[\cos x = -\frac{1}{\sqrt{17}}\]

Konačan odgovor

\(\cos x = -\dfrac{1}{\sqrt{17}}\)
(A) \(\dfrac{1}{17}\)
(B) \(-\dfrac{1}{\sqrt{17}}\)
(C) \(-\dfrac{4}{\sqrt{17}}\)
(D) \(\dfrac{1}{\sqrt{17}}\)
(E) \(\dfrac{4}{\sqrt{17}}\)
Provera: Vrednost \(-\frac{1}{\sqrt{17}} \approx -0{,}243\) je negativna (III kvadrant) i ima apsolutnu vrednost manju od 1 (kao što se očekuje za kosinus). Sa \(\operatorname{tg} x = 4\), sinus mora biti \(-\frac{4}{\sqrt{17}}\), i zaista: \(\sin^2 x + \cos^2 x = \frac{16}{17} + \frac{1}{17} = 1\).

Česte greške

  • Zaboravljen znak: Mnogi učenici izračunaju \(\cos^2 x = \frac{1}{17}\), izvade koren i dobiju \(\frac{1}{\sqrt{17}}\) (odgovor D), zaboravljajući da je kosinus negativan u III kvadrantu. Uvek proverite u kom ste kvadrantu!
  • Mešanje sinusa i kosinusa: Odgovor (C) je zapravo \(\sin x = -\frac{4}{\sqrt{17}}\), ne \(\cos x\). Pažljivo čitajte šta se traži.

Dodatni izazov

Ako je \(\operatorname{tg} x = 4\) i \(\pi < x < 2\pi\), koliko je \(\sin x + \cos x\)?

Prikaži nagoveštaj
Već znamo da je \(\cos x = -\frac{1}{\sqrt{17}}\) i \(\sin x = -\frac{4}{\sqrt{17}}\). Saberite ove dve vrednosti i racionalizujte imenilac.
