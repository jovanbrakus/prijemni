Aritmetički i geometrijski niz

Algebra — svojstva aritmetičkog i geometrijskog niza, sistemi jednačina

Postavka zadatka

Brojevi \(a\), \(b\), \(c\) su uzastopni članovi rastućeg aritmetičkog niza, a brojevi \(a\), \(b\), \(c + 3\) su uzastopni članovi geometrijskog niza.

Ako je \(a + b + c = 10\), onda je \(a^2 + b^2 + c^2\) jednako:

\(a + c = 2b\)
Uslov GP
\(b^2 = a(c+3)\)

\(a + b + c = 10\)

Plan rešavanja

Iz osobine aritmetičkog niza i datog zbira odredićemo \(b\), a zatim članove \(a\) i \(c\) izraziti pomoću razlike \(d\). Uslov geometrijskog niza daće nam kvadratnu jednačinu po \(d\), iz koje nalazimo sve članove i traženi zbir kvadrata.

📐 Ključne formule i pojmovi
  • Aritmetički niz: Tri uzastopna člana \(a, b, c\) zadovoljavaju \(b - a = c - b\), tj. \(a + c = 2b\). Srednji član je aritmetička sredina krajnjih.
  • Geometrijski niz: Tri uzastopna člana \(p, q, r\) zadovoljavaju \(q^2 = p \cdot r\). Srednji član je geometrijska sredina krajnjih.
  • Kvadratna jednačina: \(ax^2 + bx + c = 0\) ima rešenja \(x = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\), ili se može razložiti na \((x - x_1)(x - x_2) = 0\).

Vizuelni prikaz

Pomeranjem klizača menjajte razliku \(d\) i posmatrajte kako se nizovi ponašaju

Rešenje korak po korak

1

Nalaženje vrednosti \(b\)

Pošto su \(a, b, c\) uzastopni članovi aritmetičkog niza, važi:

\[a + c = 2b\]

Zamenom u uslov za zbir:

\[a + b + c = 2b + b = 3b = 10\]
\[b = \frac{10}{3}\]
2

Izražavanje \(a\) i \(c\) preko razlike \(d\)

Neka je \(d\) razlika aritmetičkog niza (\(d > 0\), jer je niz rastući). Tada:

\[a = b - d = \frac{10}{3} - d, \qquad c = b + d = \frac{10}{3} + d\]
🔍 Proveri sebe: Kolika je vrednost \(b\)?

\(b = \dfrac{10}{3} \approx 3{,}33\)

3

Primena uslova geometrijskog niza

Za geometrijski niz \(a, b, c + 3\), važi \(b^2 = a \cdot (c + 3)\):

\[\left(\frac{10}{3}\right)^2 = \left(\frac{10}{3} - d\right)\left(\frac{10}{3} + d + 3\right)\]

Sredićemo desnu stranu. Primetimo da je \(\frac{10}{3} + 3 = \frac{19}{3}\):

\[\frac{100}{9} = \left(\frac{10}{3} - d\right)\left(\frac{19}{3} + d\right)\]
4

Razvijanje proizvoda

Množimo zagrade:

\[\frac{10}{3} \cdot \frac{19}{3} + \frac{10}{3} \cdot d - d \cdot \frac{19}{3} - d^2 = \frac{100}{9}\]
\[\frac{190}{9} + \frac{10d}{3} - \frac{19d}{3} - d^2 = \frac{100}{9}\]

Sređujemo koeficijente uz \(d\):

\[\frac{190}{9} - 3d - d^2 = \frac{100}{9}\]
5

Rešavanje kvadratne jednačine

Prebacujemo sve na jednu stranu:

\[-d^2 - 3d + \frac{190}{9} - \frac{100}{9} = 0\]
\[-d^2 - 3d + \frac{90}{9} = 0 \implies -d^2 - 3d + 10 = 0\]

Množimo sa \(-1\):

\[d^2 + 3d - 10 = 0\]

Rastavimo na činioce:

\[(d + 5)(d - 2) = 0\]
\[d = -5 \quad \text{ili} \quad d = 2\]

Pošto je niz rastući, mora biti \(d > 0\), pa je \(d = 2\).

🔍 Proveri sebe: Zašto odbacujemo \(d = -5\)?

Zato što bi za \(d = -5\) niz bio opadajući (\(a > b > c\)), a u zadatku je rečeno da je niz rastući.

6

Nalaženje članova niza

Za \(d = 2\):

\[a = \frac{10}{3} - 2 = \frac{4}{3}, \qquad b = \frac{10}{3}, \qquad c = \frac{10}{3} + 2 = \frac{16}{3}\]
7

Izračunavanje \(a^2 + b^2 + c^2\)

\[a^2 + b^2 + c^2 = \left(\frac{4}{3}\right)^2 + \left(\frac{10}{3}\right)^2 + \left(\frac{16}{3}\right)^2\]
\[= \frac{16}{9} + \frac{100}{9} + \frac{256}{9}\]
\[= \frac{16 + 100 + 256}{9} = \frac{372}{9} = \frac{124}{3}\]
🔍 Proveri sebe: Koliko je \(\frac{124}{3}\) u decimalnom obliku?

\(\frac{124}{3} \approx 41{,}33\)

Ključni uvid

Srednji član aritmetičkog niza tri uzastopna člana jednak je trećini njihovog zbira. To nam odmah daje \(b\), a preostala dva člana izražavamo preko jednog nepoznatog parametra \(d\).

\[a + b + c = 3b = 10 \implies b = \frac{10}{3}\]

Verifikacija:
Aritmetički niz: \(\frac{4}{3},\; \frac{10}{3},\; \frac{16}{3}\) — razlika je \(d = 2\), niz je rastući. ✓
Geometrijski niz: \(\frac{4}{3},\; \frac{10}{3},\; \frac{25}{3}\) — količnik je \(q = \frac{10/3}{4/3} = \frac{10}{4} = \frac{5}{2}\), a \(\frac{25/3}{10/3} = \frac{5}{2}\). ✓
Zbir: \(\frac{4}{3} + \frac{10}{3} + \frac{16}{3} = \frac{30}{3} = 10\). ✓

Česte greške

  • Zaboraviti uslov da je niz rastući i prihvatiti \(d = -5\), što daje potpuno drugačije vrednosti (\(a = \frac{25}{3}\), \(c = \frac{-5}{3}\)) i pogrešan zbir kvadrata.
  • Pomešati uslove: \(c + 3\) je treći član geometrijskog niza, ne \(c\) sam. Ako se zanemari dodatak \(+3\), dobija se pogrešna jednačina.

Dodatni izazov

Šta ako je umesto \(c + 3\) u geometrijskom nizu dato \(c + k\) za proizvoljno \(k > 0\)? Za koje vrednosti \(k\) rešenje \(d\) ostaje pozitivan ceo broj?

Nagoveštaj

Ponovite postupak ali sa \(k\) umesto \(3\). Dobićete kvadratnu jednačinu čiji diskriminant zavisi od \(k\). Ispitajte za koje \(k\) je rešenje \(d\) pozitivan ceo broj.
