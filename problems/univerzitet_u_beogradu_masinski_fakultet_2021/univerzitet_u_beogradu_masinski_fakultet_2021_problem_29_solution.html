Logaritamska jednačina sa različitim osnovama

Algebra – Logaritamske jednačine, promena osnove logaritma

Postavka zadatka

Rešenje jednačine

\[\log_4(x - 2) + \log_{16}(x - 2) + \log_2(x - 2) = 7\]

pripada intervalu:

(A)
\([0, 5]\)
(B)
\([5, 10]\)
(C)
\([10, 15]\)
(D)
\([15, 20]\)
(E)
\([20, 25]\)

Plan rešavanja

Sve tri logaritamske funkcije imaju isti argument \((x-2)\), ali različite osnove: \(4\), \(16\) i \(2\). Koristeći formulu za promenu osnove, svešćemo sve logaritme na osnovu \(2\), čime dobijamo prostu linearnu jednačinu.

📐 Ključne formule i teoreme
  • Formula za promenu osnove: \(\log_a b = \dfrac{\log_c b}{\log_c a}\) — omogućava prevođenje logaritma iz jedne osnove u drugu.
  • Poseban slučaj: Ako je \(a = c^k\), tada \(\log_{c^k} b = \dfrac{1}{k} \log_c b\). Na primer, \(\log_4 b = \frac{1}{2}\log_2 b\) jer je \(4 = 2^2\).
  • Definicija logaritma: \(\log_a b = c \iff a^c = b\), pri čemu je \(a > 0\), \(a \neq 1\), \(b > 0\).
  • Uslov definisanosti: Argument logaritma mora biti strogo pozitivan, tj. \(x - 2 > 0 \Rightarrow x > 2\).

Vizuelni prikaz

Grafik funkcije leve strane jednačine — tražimo gde preseca \(y = 7\)

Rešenje korak po korak

1

Odredimo uslov definisanosti

Argument svih logaritama je \((x - 2)\). Da bi logaritam bio definisan, argument mora biti strogo pozitivan:

\[x - 2 > 0 \implies x > 2\]
2

Uvedemo smenu \(t = \log_2(x-2)\)

Pošto su sve osnove stepeni dvojke (\(2 = 2^1\), \(4 = 2^2\), \(16 = 2^4\)), svaki logaritam možemo izraziti preko \(\log_2\):

\[\log_4(x-2) = \frac{\log_2(x-2)}{\log_2 4} = \frac{t}{2}\]
\[\log_{16}(x-2) = \frac{\log_2(x-2)}{\log_2 16} = \frac{t}{4}\]
\[\log_2(x-2) = t\]

Koristili smo formulu \(\log_{a^k} b = \frac{1}{k}\log_a b\).

🧠 Proveri sebe: Koliko iznosi \(\log_4 16\)?

\(\log_4 16 = \frac{\log_2 16}{\log_2 4} = \frac{4}{2} = 2\). Ili direktno: \(4^2 = 16\), dakle \(\log_4 16 = 2\). ✓

3

Zapišemo jednačinu u zavisnosti od \(t\)

Zamenjujemo u polaznu jednačinu:

\[\frac{t}{2} + \frac{t}{4} + t = 7\]
4

Rešimo linearnu jednačinu

Pomnožimo obe strane sa \(4\) da eliminišemo razlomke:

\[2t + t + 4t = 28\]

Saberemo koeficijente:

\[7t = 28\]

Podelimo obe strane sa \(7\):

\[t = 4\]
🧠 Proveri sebe: Koliki je zbir koeficijenata \(\frac{1}{2} + \frac{1}{4} + 1\)?

\(\frac{1}{2} + \frac{1}{4} + 1 = \frac{2}{4} + \frac{1}{4} + \frac{4}{4} = \frac{7}{4}\). Jednačina \(\frac{7}{4}t = 7\) daje \(t = 4\). ✓

5

Vratimo se na promenljivu \(x\)

Iz smene \(t = \log_2(x - 2) = 4\) sledi:

\[x - 2 = 2^4 = 16\]
\[x = 18\]

Proverimo uslov: \(x - 2 = 16 > 0\) ✓ — rešenje zadovoljava uslov definisanosti.

6

Verifikacija

Proverimo ubacivanjem \(x = 18\) u polaznu jednačinu:

\[\log_4 16 + \log_{16} 16 + \log_2 16 = 2 + 1 + 4 = 7 \; \checkmark\]

\(4^2 = 16\), \(16^1 = 16\), \(2^4 = 16\) — svaki sabirak tačan.

🧠 Proveri sebe: Kom intervalu pripada broj 18?

\(15 \leq 18 \leq 20\), dakle \(18 \in [15, 20]\). Tačan odgovor je (D). ✓

Ključni uvid

Kad su osnove logaritama stepeni istog broja, sve se svodi na jedan logaritam! Osnove \(2\), \(4 = 2^2\) i \(16 = 2^4\) daju koeficijente \(1\), \(\frac{1}{2}\) i \(\frac{1}{4}\):

\[\frac{1}{2}t + \frac{1}{4}t + t = \frac{7}{4}t = 7 \implies t = 4\]

Konačan odgovor

\(x = 18 \in [15, 20]\)
(A) \([0, 5]\)
(B) \([5, 10]\)
(C) \([10, 15]\)
(D) \([15, 20]\)
(E) \([20, 25]\)
Provera: \(\log_4 16 + \log_{16} 16 + \log_2 16 = 2 + 1 + 4 = 7\) ✓. Rešenje \(x = 18\) je u intervalu \([15, 20]\).

Česte greške

  • Pogrešno svođenje osnova: Često se meša \(\log_4(x-2) = \frac{1}{2}\log_2(x-2)\) sa \(\log_4(x-2) = 2\log_2(x-2)\). Zapamtite: veća osnova daje manji logaritam, pa je koeficijent \(\frac{1}{2}\), a ne \(2\).
  • Zaboravljen uslov definisanosti: Logaritam zahteva \(x - 2 > 0\), tj. \(x > 2\). Za ovu jednačinu to ne eliminiše rešenje, ali uvek treba proveriti.

Dodatni izazov

Reši jednačinu: \(\log_3(x+1) + \log_9(x+1) + \log_{27}(x+1) = \dfrac{11}{6}\)

Hint

Primeni istu ideju — sve osnove su stepeni trojke: \(3 = 3^1\), \(9 = 3^2\), \(27 = 3^3\). Uvedi smenu \(t = \log_3(x+1)\) i dobij jednačinu \(t + \frac{t}{2} + \frac{t}{3} = \frac{11}{6}\), tj. \(\frac{11}{6}t = \frac{11}{6}\), odakle \(t = 1\).
