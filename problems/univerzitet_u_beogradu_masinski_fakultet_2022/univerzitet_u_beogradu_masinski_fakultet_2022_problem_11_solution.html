Trigonometrijski identitet: zbir kubova

Uprošćavanje razlomka sa kubovima sinusa i kosinusa pomoću formule za zbir kubova

Postavka zadatka

Izraz

\[\dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x}\]

je identički jednak:

(A)
\(\dfrac{\cos x - \sin x}{2}\)
(B)
\(1\)
(C)
\(\dfrac{\cos x}{2}\)
(D)
\(\dfrac{1 + \cos^2 x}{4}\)
(E)
\(\dfrac{\sin x + \cos x}{2}\)

Plan rešavanja

Brojilac ćemo faktorizovati pomoću formule za zbir kubova, a imenilac ćemo preurediti pomoću formule za dupli ugao. Potom ćemo skratiti zajednički faktor.

Ključne formule i teoreme
  • Zbir kubova: \(a^3 + b^3 = (a + b)(a^2 - ab + b^2)\)
  • Osnovni trigonometrijski identitet: \(\sin^2 x + \cos^2 x = 1\)
  • Formula za dupli ugao: \(\sin 2x = 2\sin x \cos x\)
  • Skraćivanje: Ako je \(f(x) \neq 0\) za sve \(x\), onda je \(\dfrac{g(x) \cdot f(x)}{h(x) \cdot f(x)} = \dfrac{g(x)}{h(x)}\)

Vizuelni prikaz

Grafički poredimo početni izraz (plava) sa odgovorom \(\frac{\sin x + \cos x}{2}\) (zelena). Pomerите клизач да видите да се графici poklapaju.

[-2π, 2π]

Rešenje korak po korak

1

Faktorizujemo brojilac

Koristimo formulu za zbir kubova: \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\).

Neka je \(a = \sin x\), \(b = \cos x\):

\[\sin^3 x + \cos^3 x = (\sin x + \cos x)(\sin^2 x - \sin x \cos x + \cos^2 x)\]
2

Primenimo Pitagorin identitet

U drugom faktoru se pojavljuje \(\sin^2 x + \cos^2 x\), što je uvek jednako \(1\):

\[\sin^2 x - \sin x \cos x + \cos^2 x = (\sin^2 x + \cos^2 x) - \sin x \cos x = 1 - \sin x \cos x\]

Dakle, brojilac postaje:

\[\sin^3 x + \cos^3 x = (\sin x + \cos x)(1 - \sin x \cos x)\]
Proveri sebe: Koliko iznosi \(\sin^2\!\left(\frac{\pi}{6}\right) + \cos^2\!\left(\frac{\pi}{6}\right)\)?
Odgovor: \(1\). Pitagorin identitet važi za svaki ugao!
3

Transformišemo imenilac

Koristimo formulu za dupli ugao: \(\sin 2x = 2\sin x \cos x\):

\[2 - \sin 2x = 2 - 2\sin x \cos x = 2(1 - \sin x \cos x)\]
4

Formiramo razlomak

Zamenjujemo faktorizovane oblike u originalni izraz:

\[\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \frac{(\sin x + \cos x)(1 - \sin x \cos x)}{2(1 - \sin x \cos x)}\]
Proveri sebe: Da li je \(1 - \sin x \cos x\) uvek različito od nule?
Da! Pošto je \(\sin x \cos x = \frac{\sin 2x}{2}\), a \(|\sin 2x| \leq 1\), sledi \(-\frac{1}{2} \leq \sin x \cos x \leq \frac{1}{2}\). Dakle \(1 - \sin x \cos x \geq \frac{1}{2} > 0\). Možemo slobodno da skraćujemo!
5

Skraćujemo zajednički faktor

Faktor \((1 - \sin x \cos x)\) se javlja i u brojiocu i u imeniocu. Pošto je uvek pozitivan, smemo da ga skratimo:

\[\frac{(\sin x + \cos x)\cancel{(1 - \sin x \cos x)}}{2\cancel{(1 - \sin x \cos x)}} = \frac{\sin x + \cos x}{2}\]

Identitet važi za svako realno \(x\) jer imenilac nikad nije nula.

Proveri sebe: Koliko iznosi izraz za \(x = 0\)?
Za \(x = 0\): \(\frac{\sin^3 0 + \cos^3 0}{2 - \sin 0} = \frac{0 + 1}{2 - 0} = \frac{1}{2}\). A \(\frac{\sin 0 + \cos 0}{2} = \frac{0 + 1}{2} = \frac{1}{2}\). Poklapa se!

Ključni uvid

Trik je u prepoznavanju zbira kubova u brojiocu. Jednom kad faktorizujemo, pojavljuje se isti faktor \((1 - \sin x \cos x)\) i gore i dole, što omogućava elegantno skraćivanje.

\[\frac{a^3 + b^3}{2 - 2ab} = \frac{(a+b)(1-ab)}{2(1-ab)} = \frac{a+b}{2} \quad \text{za } a^2 + b^2 = 1\]

Konačan odgovor

\[\dfrac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \dfrac{\sin x + \cos x}{2}\]
(A) \(\dfrac{\cos x - \sin x}{2}\)
(B) \(1\)
(C) \(\dfrac{\cos x}{2}\)
(D) \(\dfrac{1 + \cos^2 x}{4}\)
(E) \(\dfrac{\sin x + \cos x}{2}\)

Tačan odgovor: (E)

Verifikacija: Proverimo za \(x = \frac{\pi}{4}\):
Brojilac: \(\sin^3\!\frac{\pi}{4} + \cos^3\!\frac{\pi}{4} = 2 \cdot \left(\frac{\sqrt{2}}{2}\right)^3 = 2 \cdot \frac{2\sqrt{2}}{8} = \frac{\sqrt{2}}{2}\).
Imenilac: \(2 - \sin\frac{\pi}{2} = 2 - 1 = 1\).
Rezultat: \(\frac{\sqrt{2}/2}{1} = \frac{\sqrt{2}}{2}\).
Odgovor (E): \(\frac{\sin\frac{\pi}{4} + \cos\frac{\pi}{4}}{2} = \frac{\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}}{2} = \frac{\sqrt{2}}{2}\). Poklapa se!

Česte greške

  • Zaboravljanje formule za zbir kubova: Mnogi studenti pokušavaju da "razviju" \(\sin^3 x + \cos^3 x\) koristeći formule za trostruki ugao, što dovodi do nepotrebno komplikovanog računa. Formula \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\) je daleko elegantnija.
  • Brkanje zbira i razlike kubova: Obratite pažnju da je \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\), dok je \(a^3 - b^3 = (a-b)(a^2 + ab + b^2)\). Zamena znakova daje pogrešan rezultat.

Dodatni izazov

Pokušajte da uprošćujete sličan izraz:

\[\frac{\sin^3 x - \cos^3 x}{2 + \sin 2x}\]
Hint

Koristite formulu za razliku kubova: \(a^3 - b^3 = (a-b)(a^2 + ab + b^2)\). Imenilac \(2 + \sin 2x = 2 + 2\sin x \cos x = 2(1 + \sin x \cos x)\). Vidite li zajednički faktor?
