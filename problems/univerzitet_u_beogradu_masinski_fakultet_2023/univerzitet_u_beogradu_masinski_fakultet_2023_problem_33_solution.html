Trigonometrijski identitet: cos²(2x) preko razlike sinusa i kosinusa

Trigonometrija -- primena osnovnih identiteta i formule za dvostruki ugao

Zadatak

Ako je \(\sin x - \cos x = a\), onda je \(\cos^2(2x)\) jednako:

(A)
\(\dfrac{1}{4}(2 - a^2)^2\)
(B)
\(a^2 - a^4\)
(C)
\((1 - a^2)^2\)
(D)
\(4a - a^2\)
(E)
\(2a^2 - a^4\)

Plan rešavanja

Kvadriraćemo datu jednakost \(\sin x - \cos x = a\) da bismo dobili izraz za \(\sin(2x)\) pomoću Pitagorinog identiteta. Zatim ćemo iskoristiti vezu \(\cos^2(2x) = 1 - \sin^2(2x)\) da izrazimo traženi izraz preko \(a\).

Podsetnik iz teorije

Ključne formule i identiteti

Pitagorin trigonometrijski identitet:

\[\sin^2 x + \cos^2 x = 1\]

Formula za dvostruki ugao (sinus):

\[\sin(2x) = 2\sin x \cos x\]

Pitagorin identitet za dvostruki ugao:

\[\sin^2(2x) + \cos^2(2x) = 1 \quad \Rightarrow \quad \cos^2(2x) = 1 - \sin^2(2x)\]

Kvadrat razlike:

\[(a - b)^2 = a^2 - 2ab + b^2\]

Vizuelni prikaz

Dijagram pokazuje kako se vrednosti \(\sin x\), \(\cos x\), njihova razlika \(a\), i rezultujuće \(\cos^2(2x)\) menjaju sa uglom \(x\). Koristite klizač da istražite različite vrednosti ugla.


Pomerajte klizač da vidite kako se menjaju vrednosti za različite uglove

Rešenje korak po korak

1

Kvadriramo polaznu jednakost

Dato nam je da je \(\sin x - \cos x = a\). Kvadriramo obe strane:

\[(\sin x - \cos x)^2 = a^2\]

Primenjujemo formulu za kvadrat razlike \((A - B)^2 = A^2 - 2AB + B^2\):

\[\sin^2 x - 2\sin x \cos x + \cos^2 x = a^2\]
2

Primenjujemo Pitagorin identitet

Znamo da je \(\sin^2 x + \cos^2 x = 1\), pa zamenjujemo:

\[1 - 2\sin x \cos x = a^2\]

Ovo je ključni korak -- Pitagorin identitet nam drastično pojednostavljuje izraz.

3

Izražavamo sin(2x)

Prepoznajemo da je \(2\sin x \cos x = \sin(2x)\), pa iz prethodnog koraka:

\[1 - \sin(2x) = a^2\]

Izražavamo \(\sin(2x)\):

\[\sin(2x) = 1 - a^2\]
Proveri sebe: Ako je \(a = 0\), koliko je \(\sin(2x)\)?

Kada je \(a = 0\), imamo \(\sin x = \cos x\), što znači \(x = \frac{\pi}{4}\). Tada je \(\sin(2x) = \sin\!\left(\frac{\pi}{2}\right) = 1\). Formula daje: \(1 - 0^2 = 1\). Tačno!

4

Računamo cos²(2x)

Koristimo osnovni Pitagorin identitet za ugao \(2x\):

\[\cos^2(2x) = 1 - \sin^2(2x)\]

Zamenjujemo \(\sin(2x) = 1 - a^2\):

\[\cos^2(2x) = 1 - (1 - a^2)^2\]
5

Razvijamo kvadrat binoma

Razvijamo \((1 - a^2)^2\) pomoću formule \((A - B)^2 = A^2 - 2AB + B^2\):

\[(1 - a^2)^2 = 1 - 2a^2 + a^4\]

Zamenjujemo nazad:

\[\cos^2(2x) = 1 - (1 - 2a^2 + a^4)\]
6

Pojednostavljujemo

Uklanjamo zagrade (menjamo znakove):

\[\cos^2(2x) = 1 - 1 + 2a^2 - a^4\]

Jedinice se poništavaju:

\[\cos^2(2x) = 2a^2 - a^4\]
Proveri sebe: Ako je \(a = 1\) (tj. \(x = \frac{\pi}{2}\)), koliko je \(\cos^2(2x)\)?

Za \(x = \frac{\pi}{2}\): \(\sin\frac{\pi}{2} - \cos\frac{\pi}{2} = 1 - 0 = 1\), dakle \(a = 1\).

Direktno: \(\cos^2(\pi) = (-1)^2 = 1\).

Formula: \(2 \cdot 1^2 - 1^4 = 2 - 1 = 1\). Tačno!

7

Identifikujemo tačan odgovor

Dobijeni izraz \(2a^2 - a^4\) odgovara opciji (E).

Napomena: Može se zapisati i kao \(a^2(2 - a^2)\) faktorisanjem, ali to ne menja rezultat.

Proveri sebe: Eliminacija pogrešnih odgovora za \(a = 1\)

Za \(a = 1\), \(\cos^2(2x)\) mora biti \(1\). Proverimo opcije:

(A) \(\frac{1}{4}(2 - 1)^2 = \frac{1}{4}\) -- nije 1

(B) \(1 - 1 = 0\) -- nije 1

(C) \((1 - 1)^2 = 0\) -- nije 1

(D) \(4 - 1 = 3\) -- nije 1

(E) \(2 - 1 = 1\) -- tačno!

Ključni uvid

Čest trik u trigonometriji: kada vam je data razlika ili zbir sinusa i kosinusa, kvadrirajte obe strane. Dobićete Pitagorin identitet (\(\sin^2 x + \cos^2 x = 1\)) plus proizvod \(\sin x \cos x\), koji je zapravo pola sinusa dvostrukog ugla:

\[(\sin x - \cos x)^2 = 1 - \sin(2x) \quad \Rightarrow \quad \color{#4ade80}{\cos^2(2x) = 2a^2 - a^4}\]

Konačan odgovor

\(\displaystyle \text{(E)}\ \ 2a^2 - a^4\)
(A) \(\dfrac{1}{4}(2 - a^2)^2\)
(B) \(a^2 - a^4\)
(C) \((1 - a^2)^2\)
(D) \(4a - a^2\)
(E) \(2a^2 - a^4\)
Verifikacija: Za \(x = \frac{\pi}{2}\): \(a = \sin\frac{\pi}{2} - \cos\frac{\pi}{2} = 1\). Direktno: \(\cos^2(\pi) = 1\). Formula: \(2 \cdot 1 - 1 = 1\). Za \(x = \frac{\pi}{4}\): \(a = 0\). Direktno: \(\cos^2\!\left(\frac{\pi}{2}\right) = 0\). Formula: \(0 - 0 = 0\). Oba se slažu.

Česte greške

Zamena sin²(2x) sa cos²(2x): Pažljivo sa identitetom -- tražimo \(\cos^2(2x) = 1 - \sin^2(2x)\), a ne sam \(\sin^2(2x) = (1 - a^2)^2\). Odgovor (C) je upravo \(\sin^2(2x)\), ne \(\cos^2(2x)\)!
Pogrešno razvijanje \((1 - a^2)^2\): Česta greška je da se napiše \(1 - a^4\) umesto \(1 - 2a^2 + a^4\). To bi dovelo do pogrešnog odgovora \(a^2 - a^4\) (opcija B), koji je zamka za nepažljive.

Dodatni izazov

Ako je \(\sin x + \cos x = b\), koliko je \(\cos^2(2x)\)?

Nagoveštaj

Postupak je gotovo identičan! Kvadrirajte \(\sin x + \cos x = b\) i dobićete \(1 + \sin(2x) = b^2\), dakle \(\sin(2x) = b^2 - 1\). Dalje:

\(\cos^2(2x) = 1 - (b^2 - 1)^2 = 1 - b^4 + 2b^2 - 1 = 2b^2 - b^4\)

Rezultat je isti oblik: \(2b^2 - b^4\). Zanimljivo!
