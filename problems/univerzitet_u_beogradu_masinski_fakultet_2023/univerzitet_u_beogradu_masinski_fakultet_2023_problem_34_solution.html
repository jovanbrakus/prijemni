Maksimum trigonometrijske funkcije

Trigonometrija -- transformacija linearnog trigonometrijskog izraza u sinusoidu

Zadatak

Maksimum funkcije

\[f(x) = \sin x + \sqrt{3} \cdot \cos x\]

je jednak:

(A)
\(\sqrt{3}\)
(B)
\(\dfrac{1}{2}(\sqrt{2} + \sqrt{6})\)
(C)
\(1\)
(D)
\(1 + \sqrt{3}\)
(E)
\(2\)

Plan rešavanja

Transformisaćemo izraz \(\sin x + \sqrt{3}\cos x\) u oblik \(R\sin(x + \varphi)\) koristeći formulu za zbir trigonometrijskih funkcija. Amplituda \(R\) direktno daje maksimum funkcije.

Ključne formule i teoreme
  • Transformacija linearnog trigonometrijskog izraza:
    \[a\sin x + b\cos x = R\sin(x + \varphi)\] gde je \(R = \sqrt{a^2 + b^2}\), \(\cos\varphi = \dfrac{a}{R}\), \(\sin\varphi = \dfrac{b}{R}\).
  • Adiciona formula za sinus:
    \[\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta\]
  • Opseg sinusne funkcije:
    \[-1 \leq \sin\theta \leq 1\] za svako \(\theta \in \mathbb{R}\). Dakle, \(-R \leq R\sin(x+\varphi) \leq R\).
  • Poznate vrednosti:
    \(\sin\dfrac{\pi}{3} = \dfrac{\sqrt{3}}{2}\), \(\cos\dfrac{\pi}{3} = \dfrac{1}{2}\), \(\sin\dfrac{\pi}{6} = \dfrac{1}{2}\), \(\cos\dfrac{\pi}{6} = \dfrac{\sqrt{3}}{2}\).

Rešenje korak po korak

1

Prepoznajemo oblik izraza

Funkcija \(f(x) = \sin x + \sqrt{3}\cos x\) je linearna kombinacija sinusa i kosinusa. Ovaj tip izraza možemo transformisati u jednu sinusnu funkciju pomoću formule:

\[a\sin x + b\cos x = R\sin(x + \varphi)\]

Identifikujemo koeficijente: \(a = 1\) (uz \(\sin x\)) i \(b = \sqrt{3}\) (uz \(\cos x\)).

2

Računamo amplitudu \(R\)

Amplituda se računa kao:

\[R = \sqrt{a^2 + b^2}\]

Zamenjujemo vrednosti:

\[R = \sqrt{1^2 + (\sqrt{3})^2}\]
\[R = \sqrt{1 + 3}\]
\[R = \sqrt{4} = 2\]
Proveri se: Koliko iznosi \(\sqrt{1^2 + (\sqrt{3})^2}\)?

\(\sqrt{1 + 3} = \sqrt{4} = 2\). Amplituda je \(R = 2\).

3

Određujemo fazni ugao \(\varphi\)

Fazni ugao zadovoljava uslove:

\[\cos\varphi = \frac{a}{R} = \frac{1}{2}, \quad \sin\varphi = \frac{b}{R} = \frac{\sqrt{3}}{2}\]

Obe vrednosti su pozitivne, pa je \(\varphi\) u prvom kvadrantu.

Prepoznajemo poznate vrednosti:

\[\varphi = \frac{\pi}{3} \quad (60°)\]
4

Zapisujemo transformisanu funkciju

Sada možemo zapisati:

\[f(x) = \sin x + \sqrt{3}\cos x = 2\sin\!\left(x + \frac{\pi}{3}\right)\]

Proverimo razvojem adicione formule:

\[2\sin\!\left(x + \frac{\pi}{3}\right) = 2\left(\sin x \cdot \cos\frac{\pi}{3} + \cos x \cdot \sin\frac{\pi}{3}\right)\]
\[= 2\left(\sin x \cdot \frac{1}{2} + \cos x \cdot \frac{\sqrt{3}}{2}\right)\]
\[= \sin x + \sqrt{3}\cos x \quad \checkmark\]
Proveri se: Koja je amplituda funkcije \(2\sin(x + \frac{\pi}{3})\)?

Amplituda je koeficijent ispred sinusa, dakle \(R = 2\). Funkcija osciluje između \(-2\) i \(2\).

5

Nalazimo maksimum

Znamo da sinusna funkcija ima opseg \([-1, 1]\):

\[-1 \leq \sin\!\left(x + \frac{\pi}{3}\right) \leq 1\]

Množimo celokupnu nejednakost sa \(2\):

\[-2 \leq 2\sin\!\left(x + \frac{\pi}{3}\right) \leq 2\]

Dakle, maksimum funkcije je \(2\).

Maksimum se dostiže kada je \(\sin(x + \frac{\pi}{3}) = 1\), tj. za \(x + \frac{\pi}{3} = \frac{\pi}{2}\), odnosno \(x = \frac{\pi}{6}\).

6

Verifikacija direktnim uvrštavanjem

Proverimo izračunavanjem \(f\!\left(\frac{\pi}{6}\right)\):

\[f\!\left(\frac{\pi}{6}\right) = \sin\frac{\pi}{6} + \sqrt{3}\cos\frac{\pi}{6}\]
\[= \frac{1}{2} + \sqrt{3} \cdot \frac{\sqrt{3}}{2}\]
\[= \frac{1}{2} + \frac{3}{2}\]
\[= \frac{4}{2} = 2 \quad \checkmark\]
Proveri se: Da li \(f(x)\) može biti veće od 2?

Ne! Pošto je \(f(x) = 2\sin(x + \frac{\pi}{3})\) i \(\sin\) nikada ne prelazi 1, to je \(f(x) \leq 2 \cdot 1 = 2\). Vrednost 2 je apsolutni maksimum.

Ključni uvid

Svaki izraz oblika \(a\sin x + b\cos x\) se može zapisati kao jedna sinusoida sa amplitudom \(R = \sqrt{a^2 + b^2}\). Maksimum funkcije je upravo ta amplituda!

\[\max(\sin x + \sqrt{3}\cos x) = \sqrt{1^2 + (\sqrt{3})^2} = \sqrt{4} = \color{#4ade80}{2}\]

Konačan odgovor

\(\displaystyle \max f(x) = 2\)
(A)
\(\sqrt{3}\)
(B)
\(\frac{1}{2}(\sqrt{2} + \sqrt{6})\)
(C)
\(1\)
(D)
\(1 + \sqrt{3}\)
(E)
\(2\)
Verifikacija: Direktnim uvrštavanjem \(x = \frac{\pi}{6}\) u originalnu funkciju dobijamo \(f(\frac{\pi}{6}) = \frac{1}{2} + \frac{3}{2} = 2\). Pošto je amplituda sinusoide \(R = 2\), vrednost funkcije ne može preći 2. Odgovor ima smisla jer je \(R = \sqrt{1 + 3} = 2\), a opcija (D) \(1 + \sqrt{3} \approx 2{,}73\) bi bila nemoguća za sinusoidu sa amplitudom 2.

Česte greške

Sabiranje koeficijenata umesto korišćenja formule: Mnogi studenti pogrešno računaju \(1 + \sqrt{3}\) (opcija D) umesto \(\sqrt{1^2 + (\sqrt{3})^2}\). Koeficijenti se ne sabiraju direktno -- koristi se Pitagorina teorema (zbir kvadrata pod korenom).
Zaboravljanje korena: Neki studenti izračunaju \(a^2 + b^2 = 4\) ali zaborave da izvuku koren, pa misle da je maksimum 4. Amplituda je \(R = \sqrt{a^2 + b^2}\), ne \(a^2 + b^2\).

Dodatni izazov

Za koju vrednost \(x\) iz intervala \([0, 2\pi]\) funkcija \(f(x) = \sin x + \sqrt{3}\cos x\) dostiže svoj minimum? Kolika je ta minimalna vrednost?

Hint

Minimum sinusoide \(2\sin(x + \frac{\pi}{3})\) se dostiže kada je \(\sin(x + \frac{\pi}{3}) = -1\). Za koji argument sinusna funkcija ima vrednost \(-1\)?
