Uprošćavanje trigonometrijskog izraza sa trostrukim uglovima

Zadatak

Čemu je jednak izraz

\[\frac{\sin 3x - \cos 3y}{\sin x + \cos y}\]
(A) \(\cos 2x - \cos 2y + 4\cos x \cdot \cos y + 1\)
(B) \(2\cos 2x + 2\cos 2y - 4\sin x \cdot \cos y + 1\)
(C) \(2\sin 2x + 2\cos 2y + 4\sin x \cdot \cos y - 1\)
(D) \(2\cos 2x - 2\cos 2y + 4\sin x \cdot \cos y - 1\)
(E) ničemu od navedenog

Plan rešavanja

Razvijamo \(\sin 3x\) i \(\cos 3y\) pomoću formula za trostruki ugao, grupišemo članove tako da iz brojioca izvučemo faktor \((\sin x + \cos y)\), skratimo sa imeniocem, a zatim preostali izraz prevedemo u dvostruke uglove.

Podsetnik iz teorije

Formule za trostruki ugao i zbir kubova

Trostruki ugao:

\[\sin 3\alpha = 3\sin\alpha - 4\sin^3\alpha\] \[\cos 3\alpha = 4\cos^3\alpha - 3\cos\alpha\]

Zbir kubova:

\[a^3 + b^3 = (a + b)(a^2 - ab + b^2)\]

Veza sa dvostrukim uglom:

\[\sin^2\alpha = \frac{1 - \cos 2\alpha}{2}, \qquad \cos^2\alpha = \frac{1 + \cos 2\alpha}{2}\]

Vizuelni prikaz

Dijagram prikazuje korake transformacije izraza. Prevucite mišem (ili prstom) za pomicanje.

Rešenje korak po korak

1

Razvijanje trostrukih uglova

Primenjujemo formule za trostruki ugao na brojilac:

\[\sin 3x = 3\sin x - 4\sin^3 x\] \[\cos 3y = 4\cos^3 y - 3\cos y\]

Uvrštavamo u brojilac:

\[\sin 3x - \cos 3y = (3\sin x - 4\sin^3 x) - (4\cos^3 y - 3\cos y)\]
\[= 3\sin x - 4\sin^3 x - 4\cos^3 y + 3\cos y\]
2

Grupisanje članova

Preuređujemo tako da izdvojimo članove sa faktorom 3 i članove sa faktorom 4:

\[= 3(\sin x + \cos y) - 4(\sin^3 x + \cos^3 y)\]

Primetimo da se pojavio izraz \((\sin x + \cos y)\) - isti kao imenilac!

Proveri sebe

Provera grupisanja: \(3\sin x + 3\cos y = 3(\sin x + \cos y)\). Za drugi deo: \(-4\sin^3 x - 4\cos^3 y = -4(\sin^3 x + \cos^3 y)\). Tačno!

3

Faktorizacija zbira kubova

Primenjujemo formulu \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\) gde je \(a = \sin x\) i \(b = \cos y\):

\[\sin^3 x + \cos^3 y = (\sin x + \cos y)(\sin^2 x - \sin x \cdot \cos y + \cos^2 y)\]

Sada ceo brojilac postaje:

\[= 3(\sin x + \cos y) - 4(\sin x + \cos y)(\sin^2 x - \sin x \cdot \cos y + \cos^2 y)\]
\[= (\sin x + \cos y)\Big[3 - 4(\sin^2 x - \sin x \cdot \cos y + \cos^2 y)\Big]\]
4

Skraćivanje sa imeniocem

Delimo brojilac sa imeniocem \((\sin x + \cos y)\):

\[\frac{\sin 3x - \cos 3y}{\sin x + \cos y} = 3 - 4\sin^2 x + 4\sin x \cdot \cos y - 4\cos^2 y\]
Proveri sebe

Razvijamo zagradu: \(3 - 4\sin^2 x + 4\sin x\cos y - 4\cos^2 y\). Provera za \(x = 0,\, y = 0\): \(3 - 0 + 0 - 4 = -1\). Originalni izraz: \(\frac{0 - 1}{0 + 1} = -1\). Tačno!

5

Prevođenje u dvostruke uglove

Koristimo formule za prelaz sa kvadrata na dvostruki ugao:

\[-4\sin^2 x = -4 \cdot \frac{1 - \cos 2x}{2} = -2 + 2\cos 2x\]
\[-4\cos^2 y = -4 \cdot \frac{1 + \cos 2y}{2} = -2 - 2\cos 2y\]

Sabiramo sve:

\[3 + (-2 + 2\cos 2x) + 4\sin x \cdot \cos y + (-2 - 2\cos 2y)\]
\[= (3 - 2 - 2) + 2\cos 2x - 2\cos 2y + 4\sin x \cdot \cos y\]
\[= \boxed{2\cos 2x - 2\cos 2y + 4\sin x \cdot \cos y - 1}\]
Proveri sebe

Konstantni članovi: \(3 - 2 - 2 = -1\). Tačno!

6

Verifikacija numeričkim vrednostima

Test 1: \(x = 0,\; y = 0\)

\[\text{Leva strana: } \frac{\sin 0 - \cos 0}{\sin 0 + \cos 0} = \frac{0 - 1}{0 + 1} = -1\]
\[\text{(D): } 2\cos 0 - 2\cos 0 + 4\sin 0 \cdot \cos 0 - 1 = 2 - 2 + 0 - 1 = -1 \quad \checkmark\]

Test 2: \(x = \frac{\pi}{2},\; y = 0\)

\[\text{Leva strana: } \frac{\sin\frac{3\pi}{2} - \cos 0}{\sin\frac{\pi}{2} + \cos 0} = \frac{-1 - 1}{1 + 1} = -1\]
\[\text{(D): } 2\cos\pi - 2\cos 0 + 4\sin\frac{\pi}{2}\cdot\cos 0 - 1 = -2 - 2 + 4 - 1 = -1 \quad \checkmark\]

Ključni uvid

Suština ovog zadatka je prepoznati da se formule za trostruki ugao (\(\sin 3x = 3\sin x - 4\sin^3 x\) i \(\cos 3y = 4\cos^3 y - 3\cos y\)) mogu tako preurediti da se u brojiocu pojavi faktor \((\sin x + \cos y)\), koji se zatim skrati sa imeniocem. Ključ je grupisanje: koeficijent 3 daje linearni član, a koeficijent 4 daje kubni član koji se faktoriše formulom za zbir kubova \(a^3 + b^3 = (a+b)(a^2-ab+b^2)\).

Konačan odgovor

\(\displaystyle \text{(D)}\ 2\cos 2x - 2\cos 2y + 4\sin x \cdot \cos y - 1\)

Česte greške

Zamena formula za trostruki ugao: Studenti često mešaju formule \(\sin 3\alpha = 3\sin\alpha - 4\sin^3\alpha\) i \(\cos 3\alpha = 4\cos^3\alpha - 3\cos\alpha\). Obratite pažnju na znakove i koeficijente - kod sinusa je "3 minus 4", kod kosinusa je "4 minus 3".
Pogrešan predznak kod cos 3y: Oduzimamo \(\cos 3y = 4\cos^3 y - 3\cos y\), pa se predznak svakog člana menja: \(-4\cos^3 y + 3\cos y\). Česta greška je zaboraviti promeniti predznak drugog člana.
Greška pri prevođenju u dvostruke uglove: Formule \(\sin^2\alpha = \frac{1-\cos 2\alpha}{2}\) i \(\cos^2\alpha = \frac{1+\cos 2\alpha}{2}\) imaju različite predznake pred \(\cos 2\alpha\). Zamena ovih formula dovodi do pogrešnog rezultata.

Dodatni izazov

Pokušajte da dokažete da za \(y = x\) izraz \(\frac{\sin 3x - \cos 3x}{\sin x + \cos x}\) ima oblik koji se može izraziti preko samo jedne trigonometrijske funkcije od \(2x\). Koji je to izraz?

Odgovor

Za \(y = x\) dobijamo:

\[2\cos 2x - 2\cos 2x + 4\sin x\cos x - 1 = 4\sin x\cos x - 1 = 2\sin 2x - 1\]

Dakle, za \(y = x\) izraz se svodi na \(2\sin 2x - 1\).
