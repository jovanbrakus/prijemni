Deljivost polinoma i odredjivanje koeficijenata

Polinomi — deljivost i identifikacija koeficijenata

📜 Zadatak

Ako je polinom \(x^4 + ax^2 + b\) \((a, b \in \mathbb{R})\) deljiv polinomom \(x^2 + x + 1\), onda je \(a + b\) jednako:

A) \(-1\)
B) \(2\)
C) \(0\)
D) \(1\)
E) \(-2\)
📌 Plan resavanja
  1. Zapisati uslov deljivosti: \(x^4 + ax^2 + b = (x^2 + x + 1)(x^2 + cx + d)\)
  2. Razviti desnu stranu i izjednaciti koeficijente uz iste stepene
  3. Resiti sistem jednacina za \(a\), \(b\), \(c\), \(d\)
  4. Izracunati \(a + b\) i verifikovati rezultat
Podsetnik iz teorije

Deljivost polinoma:

Polinom \(P(x)\) je deljiv polinomom \(Q(x)\) ako postoji polinom \(R(x)\) takav da je \(P(x) = Q(x) \cdot R(x)\).

Izjednacavanje koeficijenata:

Dva polinoma su jednaka ako i samo ako su im jednaki koeficijenti uz svaki stepen promenljive.

Koreni polinoma \(x^2 + x + 1\):

Polinom \(x^2 + x + 1\) je ciklotomicki polinom treceg reda. Njegovi koreni su primitivni treci koreni jedinice:

\[\omega = \frac{-1 + i\sqrt{3}}{2}, \qquad \overline{\omega} = \frac{-1 - i\sqrt{3}}{2}\]

Za njih vazi: \(\omega^3 = 1\) i \(\omega^2 + \omega + 1 = 0\), tj. \(\omega^2 = -\omega - 1\).

📊 Graficki prikaz

Faktorizacija polinoma: \(x^4 + x^2 + 1 = (x^2 + x + 1)(x^2 - x + 1)\). Prikazan je grafik polinoma \(P(x) = x^4 + x^2 + 1\) koji je uvek pozitivan (nema realnih korena).

✏️ Resenje korak po korak
Korak 1: Postavljanje uslova deljivosti

Ako je polinom \(x^4 + ax^2 + b\) deljiv polinomom \(x^2 + x + 1\), tada postoji polinom drugog stepena \(x^2 + cx + d\) takav da:

\[x^4 + ax^2 + b = (x^2 + x + 1)(x^2 + cx + d)\]

Kolicnik mora biti drugog stepena jer je stepen deljenika 4, a stepen delioca 2.

Korak 2: Razvijanje desne strane

Mnozimo polinom po polinom:

\[(x^2 + x + 1)(x^2 + cx + d)\] \[= x^4 + cx^3 + dx^2 + x^3 + cx^2 + dx + x^2 + cx + d\]

Grupisemo po stepenima:

\[= x^4 + (c + 1)x^3 + (d + c + 1)x^2 + (d + c)x + d\]
Proveri se: Leva strana \(x^4 + ax^2 + b\) nema clanove uz \(x^3\) i \(x^1\), sto znaci da su koeficijenti uz te stepene jednaki nuli.
Korak 3: Izjednacavanje koeficijenata

Uporedimo koeficijente leve i desne strane:

\(x^4\):   \(1 = 1\)   ✓

\(x^3\):   \(c + 1 = 0\)   \(\Rightarrow\)   \(c = -1\)

\(x^2\):   \(d + c + 1 = a\)

\(x^1\):   \(d + c = 0\)   \(\Rightarrow\)   \(d = -c = 1\)

\(x^0\):   \(d = b\)

Korak 4: Resavanje sistema

Iz jednacine za \(x^3\): \(c = -1\).

Iz jednacine za \(x^1\): \(d + c = 0\), dakle \(d = -c = -(-1) = 1\).

Iz jednacine za \(x^0\): \(b = d = 1\).

Iz jednacine za \(x^2\): \(a = d + c + 1 = 1 + (-1) + 1 = 1\).

Rezultat

\(a = 1\), \(b = 1\), \(c = -1\), \(d = 1\)

Korak 5: Verifikacija

Proveravamo mnozenje:

\[(x^2 + x + 1)(x^2 - x + 1)\] \[= x^4 - x^3 + x^2 + x^3 - x^2 + x + x^2 - x + 1\] \[= x^4 + x^2 + 1\]

Zaista dobijamo \(x^4 + 1 \cdot x^2 + 1\), sto potvrdjuje \(a = 1\), \(b = 1\).

Korak 6: Zakljucak

Trazena vrednost:

\[a + b = 1 + 1 = \boxed{2}\]
Alternativni pristup: koreni polinoma

Koreni polinoma \(x^2 + x + 1\) su \(\omega\) i \(\overline{\omega}\), gde je \(\omega = \frac{-1 + i\sqrt{3}}{2}\) — primitivni treci koren jedinice.

Kljucno svojstvo: \(\omega^2 = -\omega - 1\) i \(\omega^3 = 1\), pa:

\[\omega^4 = \omega^3 \cdot \omega = 1 \cdot \omega = \omega\]

Uslov deljivosti: \(P(\omega) = 0\):

\[P(\omega) = \omega^4 + a\omega^2 + b = \omega + a(-\omega - 1) + b\] \[= \omega(1 - a) + (b - a) = 0\]

Posto je \(\omega\) kompleksan broj koji nije realan, a \(1 - a\) i \(b - a\) su realni koeficijenti, oba moraju biti nula:

\[1 - a = 0 \;\Rightarrow\; a = 1\] \[b - a = 0 \;\Rightarrow\; b = a = 1\]

Dakle \(a + b = 2\).

💡 Kljucni uvid

Identitet \(x^4 + x^2 + 1 = (x^2 + x + 1)(x^2 - x + 1)\) je poznata faktorizacija. Moze se izvesti dodavanjem i oduzimanjem \(x^2\):

\[x^4 + x^2 + 1 = x^4 + 2x^2 + 1 - x^2 = (x^2 + 1)^2 - x^2 = (x^2 + x + 1)(x^2 - x + 1)\]

Ovo je primena formule razlike kvadrata: \(A^2 - B^2 = (A - B)(A + B)\) gde je \(A = x^2 + 1\) i \(B = x\).

Konacan odgovor
B) \(\;a + b = 2\)
⚠️ Ceste greske
Pogresan oblik kolicnika: Neki ucenici pretpostave da je kolicnik oblika \(x^2 + d\) (bez clana \(cx\)), sto ne mora biti tacno. Kolicnik mora biti opsti polinom drugog stepena \(x^2 + cx + d\).
Greska pri mnozenju: Pri razvijanju proizvoda \((x^2 + x + 1)(x^2 + cx + d)\) lako se previdi neki clan. Sistematicno mnoziti svaki clan prvog polinoma sa svakim clanom drugog.
Zaboravljeni uslovi: Koeficijenti uz \(x^3\) i \(x^1\) na levoj strani su nula. Oba ova uslova daju po jednu jednacinu — ne smemo preskociti nijedan.
