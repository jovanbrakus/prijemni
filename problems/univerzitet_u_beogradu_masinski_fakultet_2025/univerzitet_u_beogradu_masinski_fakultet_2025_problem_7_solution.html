Deljivost polinoma i razlika kvadrata

Polinomi — nule i deljivost

📜 Zadatak

Ako je polinom \(x^4 + ax^3 + bx^2 + 3x + 2\) (\(a, b \in \mathbb{R}\)) deljiv polinomom \(x^2 + 3x + 2\), onda je vrednost izraza \(a^2 - b^2\) jednaka:

(A) \(18\)
(B) \(6\)
(C) \(3\)
(D) \(0\)
(E) \(9\)
📌 Plan resavanja
  1. Faktorizovati delilac \(x^2 + 3x + 2\) na linearne cinioce
  2. Iskoristiti cinjenicu da su nule delioca ujedno i nule polinoma \(P(x)\)
  3. Uvrstiti nule u \(P(x)\) i dobiti sistem jednacina po \(a\) i \(b\)
  4. Resiti sistem i izracunati \(a^2 - b^2\)
Podsetnik iz teorije

Deljivost polinoma:

Ako je polinom \(P(x)\) deljiv polinomom \(D(x)\), tada svaka nula od \(D(x)\) je ujedno i nula od \(P(x)\). Formalno:

\[D(x_0) = 0 \;\Rightarrow\; P(x_0) = 0\]

Faktorizacija kvadratnog trinoma:

\[x^2 + 3x + 2 = (x + 1)(x + 2)\]

Nule su \(x_1 = -1\) i \(x_2 = -2\).

Razlika kvadrata:

\[a^2 - b^2 = (a - b)(a + b)\]

Ako je \(a = b\), odmah sledi \(a^2 - b^2 = 0\), bez obzira na konkretnu vrednost.

📊 Graficki prikaz

Grafik polinoma \(P(x) = x^4 + 3x^3 + 3x^2 + 3x + 2\) sa oznacenim nulama \(x = -1\) i \(x = -2\).

✏️ Resenje korak po korak
Korak 1: Faktorizacija delioca

Razlozimo kvadratni trinom \(x^2 + 3x + 2\) na linearne cinioce:

\[x^2 + 3x + 2 = (x + 1)(x + 2)\]

Nule delioca su \(x_1 = -1\) i \(x_2 = -2\).

Korak 2: Uslov deljivosti

Neka je \(P(x) = x^4 + ax^3 + bx^2 + 3x + 2\). Posto je \(P(x)\) deljiv sa \((x+1)(x+2)\), vazi:

\[P(-1) = 0 \quad \text{i} \quad P(-2) = 0\]
Korak 3: Jednacina iz \(P(-1) = 0\)

Uvrstimo \(x = -1\) u polinom:

\[(-1)^4 + a \cdot (-1)^3 + b \cdot (-1)^2 + 3 \cdot (-1) + 2 = 0\] \[1 - a + b - 3 + 2 = 0\] \[-a + b = 0\]
Prva jednacina

\(a = b\)   … (I)

Kljucno zapazanje: Vec iz prve jednacine dobijamo \(a = b\), sto znaci da je \(a^2 - b^2 = (a - b)(a + b) = 0 \cdot (a + b) = 0\). Mozemo odmah zakljuciti odgovor! Ipak, nastavimo da odredimo konkretne vrednosti.
Korak 4: Jednacina iz \(P(-2) = 0\)

Uvrstimo \(x = -2\) u polinom:

\[(-2)^4 + a \cdot (-2)^3 + b \cdot (-2)^2 + 3 \cdot (-2) + 2 = 0\] \[16 - 8a + 4b - 6 + 2 = 0\] \[12 - 8a + 4b = 0\]

Delimo celу jednacinu sa \(4\):

\[-2a + b = -3 \quad \ldots \text{(II)}\]
Korak 5: Resavanje sistema jednacina

Iz jednacine (I) imamo \(a = b\). Zamenom u (II):

\[-2a + a = -3\] \[-a = -3\] \[a = 3\]

Posto je \(a = b\), sledi \(b = 3\).

Korak 6: Izracunavanje trazenog izraza
\[a^2 - b^2 = 3^2 - 3^2 = 9 - 9 = 0\]
Korak 7: Verifikacija

Za \(a = 3\) i \(b = 3\), polinom glasi:

\[P(x) = x^4 + 3x^3 + 3x^2 + 3x + 2\]

Proverimo deljenje:

\[x^4 + 3x^3 + 3x^2 + 3x + 2 = (x^2 + 3x + 2)(x^2 + 1)\]

Razvijmo desnu stranu:

\[(x^2 + 3x + 2)(x^2 + 1) = x^4 + x^2 + 3x^3 + 3x + 2x^2 + 2\] \[= x^4 + 3x^3 + 3x^2 + 3x + 2 \;\checkmark\]
Potvrdeno

Polinom je zaista deljiv deliocerm \(x^2 + 3x + 2\), a kolicnik je \(x^2 + 1\).

💡 Kljucni uvid

Cim iz uslova \(P(-1) = 0\) dobijemo da je \(a = b\), mozemo odmah zakljuciti da je \(a^2 - b^2 = 0\), jer vazi identitet \(a^2 - b^2 = (a-b)(a+b)\), a \(a - b = 0\). Druga jednacina je potrebna samo ako zelimo da odredimo konkretne vrednosti \(a\) i \(b\), ali za trazeni izraz nije neophodna. Ovo je elegantna precica koja stedi vreme na ispitu.

Konacan odgovor
(D) \(\;a^2 - b^2 = 0\)
⚠️ Ceste greske
Greska u znaku pri stepenima: Cesta greska je pogresno racunanje \((-2)^3 = 8\) umesto \((-2)^3 = -8\), ili \((-1)^4 = -1\) umesto \((-1)^4 = 1\). Uvek pazite na parnost eksponenta kod negativnih baza.
Neprepoznavanje razlike kvadrata: Neki ucenici izracunaju \(a = 3\) i \(b = 3\), pa racunaju \(a^2 - b^2 = 9 - 9 = 0\), ali im promakne da je rezultat mogao da se zakljuci i bez nalazenja konkretnih vrednosti — dovoljno je bilo uociti \(a = b\).
Zamena pogresnе nule: Ponekad ucenici pogresno uzmu \(x = 1\) ili \(x = 2\) umesto \(x = -1\) i \(x = -2\). Nule polinoma \(x^2 + 3x + 2\) su negativni brojevi!
🏆 Dodatni izazov

Polinom \(x^4 + cx^3 + dx^2 - 7x - 6\) je deljiv sa \(x^2 - 1\). Odredite vrednosti \(c\) i \(d\), a zatim potpuno faktorizujte polinom.

Savet: \(x^2 - 1 = (x-1)(x+1)\), pa uvrstite \(x = 1\) i \(x = -1\). Dobicete sistem dve jednacine sa dve nepoznate.
