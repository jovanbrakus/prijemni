Kotangens ugla pri poznatom sinusu

Trigonometrija -- određivanje vrednosti kotangensa na osnovu sinusa i intervala ugla

Zadatak

Ako je \(\sin x = \dfrac{1}{3}\) i \(\dfrac{5\pi}{2} < x < 3\pi\), onda je \(\operatorname{ctg} x\) jednako:

(A)
\(-3\)
(B)
\(-\dfrac{1}{4}\sqrt{2}\)
(C)
\(\dfrac{1}{4}\sqrt{2}\)
(D)
\(-2\sqrt{2}\)
(E)
\(2\sqrt{2}\)

Plan rešavanja

Prvo ćemo odrediti u kom se kvadrantu nalazi ugao \(x\) (svođenjem intervala oduzimanjem \(2\pi\)), zatim iz Pitagorine trigonometrijske identičnosti naći \(\cos x\), i konačno izračunati \(\operatorname{ctg} x = \frac{\cos x}{\sin x}\).

Teorijski podsetnik

Ključne formule i teoreme
  • Osnovni trigonometrijski identitet: \(\sin^2 x + \cos^2 x = 1\)
  • Kotangens: \(\operatorname{ctg} x = \dfrac{\cos x}{\sin x}\)
  • Periodičnost: \(\sin(x + 2\pi) = \sin x\), \(\cos(x + 2\pi) = \cos x\) -- trigonometrijske funkcije su periodične sa periodom \(2\pi\)
  • Znakovi u II kvadrantu (\(\pi/2 < \alpha < \pi\)): \(\sin \alpha > 0\), \(\cos \alpha < 0\), \(\operatorname{ctg} \alpha < 0\)

Vizuelni prikaz -- jedinična kružnica

Položaj ugla \(x\) na jediničnoj kružnici (II kvadrant nakon svođenja)

Rešenje korak po korak

1

Određivanje kvadranta

Dato je da je \(\dfrac{5\pi}{2} < x < 3\pi\). Pošto su trigonometrijske funkcije periodične sa periodom \(2\pi\), možemo oduzeti \(2\pi\) da bismo odredili položaj ugla:

\[\frac{5\pi}{2} - 2\pi < x - 2\pi < 3\pi - 2\pi\]
\[\frac{5\pi - 4\pi}{2} < x - 2\pi < \pi\]
\[\frac{\pi}{2} < x - 2\pi < \pi\]

To znači da je ekvivalentni ugao u intervalu \(\left(\dfrac{\pi}{2},\, \pi\right)\), što odgovara drugom kvadrantu.

U drugom kvadrantu važi: \(\sin > 0\), \(\cos < 0\), pa je i \(\operatorname{ctg} < 0\).

Proveri se: U kom kvadrantu je ugao?

Ugao \(x\), nakon svođenja za \(2\pi\), pripada drugom kvadrantu (\(\pi/2 < x - 2\pi < \pi\)). U drugom kvadrantu sinus je pozitivan, a kosinus negativan.

2

Računanje kosinusa

Koristimo osnovni trigonometrijski identitet:

\[\sin^2 x + \cos^2 x = 1\]

Zamenjujemo \(\sin x = \dfrac{1}{3}\):

\[\left(\frac{1}{3}\right)^2 + \cos^2 x = 1\]
\[\frac{1}{9} + \cos^2 x = 1\]
\[\cos^2 x = 1 - \frac{1}{9} = \frac{9 - 1}{9} = \frac{8}{9}\]
\[\cos x = \pm\sqrt{\frac{8}{9}} = \pm\frac{2\sqrt{2}}{3}\]
3

Izbor znaka kosinusa

Pošto smo u koraku 1 utvrdili da je ugao u drugom kvadrantu, kosinus mora biti negativan:

\[\cos x = -\frac{2\sqrt{2}}{3}\]
Proveri se: Koliko je \(\cos^2 x + \sin^2 x\)?

\(\cos^2 x + \sin^2 x = \dfrac{8}{9} + \dfrac{1}{9} = \dfrac{9}{9} = 1\) -- identitet je zadovoljen.

4

Računanje kotangensa

Primenjujemo formulu za kotangens:

\[\operatorname{ctg} x = \frac{\cos x}{\sin x}\]

Zamenjujemo poznate vrednosti:

\[\operatorname{ctg} x = \frac{-\dfrac{2\sqrt{2}}{3}}{\dfrac{1}{3}}\]

Delimo razlomke (množimo prvim i recipročnom vrednošću drugog):

\[\operatorname{ctg} x = -\frac{2\sqrt{2}}{3} \cdot \frac{3}{1} = -2\sqrt{2}\]
Proveri se: Da li je znak rezultata očekivan?

Da! U drugom kvadrantu \(\cos < 0\) i \(\sin > 0\), pa je \(\operatorname{ctg} = \cos/\sin < 0\). Naš rezultat \(-2\sqrt{2} \approx -2{,}83\) je zaista negativan.

Ključni uvid

Ključ ovog zadatka je pravilno svođenje intervala \(\left(\dfrac{5\pi}{2},\, 3\pi\right)\) na standardni interval oduzimanjem pune periode \(2\pi\), čime dobijamo drugi kvadrant. Tek nakon toga možemo ispravno odrediti znak kosinusa i izračunati kotangens.

\(\operatorname{ctg} x = \dfrac{\cos x}{\sin x} = \dfrac{-\frac{2\sqrt{2}}{3}}{\frac{1}{3}} = -2\sqrt{2}\)

Konačan odgovor

\(\operatorname{ctg} x = -2\sqrt{2}\)
(A)
\(-3\)
(B)
\(-\dfrac{1}{4}\sqrt{2}\)
(C)
\(\dfrac{1}{4}\sqrt{2}\)
(D)
\(-2\sqrt{2}\)
(E)
\(2\sqrt{2}\)

Verifikacija: Proverimo da se sve slaže: \(\sin x = \frac{1}{3}\), \(\cos x = -\frac{2\sqrt{2}}{3}\). Tada je \(\sin^2 x + \cos^2 x = \frac{1}{9} + \frac{8}{9} = 1\) i \(\operatorname{ctg} x = \frac{\cos x}{\sin x} = -2\sqrt{2}\). Znak je negativan, što odgovara drugom kvadrantu.

Česte greške

Pogrešan kvadrant: Ako se ne oduzme \(2\pi\) od datog intervala, može se pogrešno zaključiti da je ugao u nekom drugom kvadrantu i odabrati pogrešan znak za kosinus. Na primer, ako se uzme \(\cos x > 0\), dobija se \(\operatorname{ctg} x = +2\sqrt{2}\), što je odgovor (E) -- pogrešno!
Greška pri deljenju razlomaka: Pri računanju \(\dfrac{-2\sqrt{2}/3}{1/3}\) učenici ponekad zaborave da se trojke pokrate, pa dobiju \(-\dfrac{2\sqrt{2}}{9}\) ili sličan pogrešan rezultat.

Dodatni izazov

Ako je \(\cos x = -\dfrac{3}{5}\) i \(\pi < x < \dfrac{3\pi}{2}\), koliko je \(\operatorname{tg} x + \operatorname{ctg} x\)?

Nagovještaj

U trećem kvadrantu obe funkcije \(\sin\) i \(\cos\) su negativne. Nađi \(\sin x\) koristeći Pitagorin identitet, a zatim izračunaj \(\operatorname{tg} x = \sin x / \cos x\) i \(\operatorname{ctg} x = \cos x / \sin x\).
