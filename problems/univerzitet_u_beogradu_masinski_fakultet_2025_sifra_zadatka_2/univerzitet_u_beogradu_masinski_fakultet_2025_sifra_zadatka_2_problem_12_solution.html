Zbir kubova sinusa i kosinusa

Trigonometrijski identiteti — faktorizacija i pojednostavljivanje razlomka

Postavka zadatka

Izraz

\[\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x}\]

je identički jednak:

(A)
\(\dfrac{\cos x - \sin x}{2}\)
(B)
\(1\)
(C)
\(\dfrac{\cos x}{2}\)
(D)
\(\dfrac{1 + \cos^2 x}{4}\)
(E)
\(\dfrac{\sin x + \cos x}{2}\)

Plan rešavanja

Brojilac ćemo faktorizovati korišćenjem formule za zbir kubova, a imenilac prepisati pomoću formule za dupli ugao. Nakon toga, skratićemo zajednički faktor i dobiti konačan rezultat.

Ključne formule i teoreme
  • Zbir kubova: \(a^3 + b^3 = (a + b)(a^2 - ab + b^2)\)
  • Osnovni trigonometrijski identitet: \(\sin^2 x + \cos^2 x = 1\)
  • Dupli ugao (sinus): \(\sin 2x = 2\sin x \cos x\)
  • Skraćivanje: Ako je \(f(x) \neq 0\), tada \(\dfrac{g(x) \cdot f(x)}{h(x) \cdot f(x)} = \dfrac{g(x)}{h(x)}\)

Grafički prikaz

Pomerajte klizač ispod da promenite vrednost ugla \(x\) i uporedite originalni izraz sa odgovorom (E)


Rešenje korak po korak

1

Faktorizujemo brojilac — formula za zbir kubova

Primenimo formulu \(a^3 + b^3 = (a + b)(a^2 - ab + b^2)\) gde je \(a = \sin x\) i \(b = \cos x\):

\[\sin^3 x + \cos^3 x = (\sin x + \cos x)(\sin^2 x - \sin x \cos x + \cos^2 x)\]
2

Pojednostavimo drugi faktor u brojilocu

Grupisanjem \(\sin^2 x + \cos^2 x\) i primenom osnovnog trigonometrijskog identiteta:

\[\sin^2 x - \sin x \cos x + \cos^2 x = \underbrace{(\sin^2 x + \cos^2 x)}_{= 1} - \sin x \cos x = 1 - \sin x \cos x\]

Dakle, brojilac postaje:

\[\sin^3 x + \cos^3 x = (\sin x + \cos x)(1 - \sin x \cos x)\]
Proveri sebe: Koliko je \(\sin^2 x + \cos^2 x\)?

Uvek je \(\sin^2 x + \cos^2 x = 1\), za svako \(x\). To je osnovni Pitagorin identitet u trigonometriji.

3

Transformišemo imenilac pomoću duplih uglova

Koristimo formulu \(\sin 2x = 2\sin x \cos x\) da prepišemo imenilac:

\[2 - \sin 2x = 2 - 2\sin x \cos x\]

Iznosimo zajednički faktor \(2\):

\[2 - 2\sin x \cos x = 2(1 - \sin x \cos x)\]
4

Skraćujemo zajednički faktor

Sada sastavimo razlomak i uočimo da se \((1 - \sin x \cos x)\) javlja i u brojilocu i u imeniocu:

\[\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \frac{(\sin x + \cos x)\cancel{(1 - \sin x \cos x)}}{2\cancel{(1 - \sin x \cos x)}}\]

Napomena: Skraćivanje je validno jer je \(1 - \sin x \cos x > 0\) za svako \(x\). Naime, \(\sin x \cos x = \frac{1}{2}\sin 2x\), pa je \(|\sin x \cos x| \leq \frac{1}{2} < 1\).

Proveri sebe: Zašto smemo da skratimo \(1 - \sin x \cos x\)?

Jer je \(\sin x \cos x = \frac{1}{2}\sin 2x\), a \(|\sin 2x| \leq 1\), pa je \(|\sin x \cos x| \leq \frac{1}{2}\). Zato je \(1 - \sin x \cos x \geq \frac{1}{2} > 0\) za svako \(x\), što znači da nikad ne delimo nulom.

5

Konačan rezultat

Posle skraćivanja ostaje:

\[\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \frac{\sin x + \cos x}{2}\]
Ključni uvid

Ceo trik leži u prepoznavanju formule za zbir kubova u brojilocu i formule za dupli ugao u imeniocu. Oba izraza sadrže faktor \(1 - \sin x \cos x\), koji se skrati!

\[\frac{(\sin x + \cos x)\cancel{(1 - \sin x \cos x)}}{2\cancel{(1 - \sin x \cos x)}} = \frac{\sin x + \cos x}{2}\]
Konačan odgovor
\(\dfrac{\sin x + \cos x}{2}\)
(A)
\(\frac{\cos x - \sin x}{2}\)
(B)
\(1\)
(C)
\(\frac{\cos x}{2}\)
(D)
\(\frac{1 + \cos^2 x}{4}\)
(E)
\(\frac{\sin x + \cos x}{2}\)
Provera: Za \(x = \frac{\pi}{4}\): Brojilac = \(2 \cdot \left(\frac{\sqrt{2}}{2}\right)^3 = \frac{\sqrt{2}}{2}\), Imenilac = \(2 - 1 = 1\), pa razlomak = \(\frac{\sqrt{2}}{2}\). Odgovor (E): \(\frac{\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}}{2} = \frac{\sqrt{2}}{2}\). Vrednosti se poklapaju.
Česte greške
  • Zaboravljanje formule za zbir kubova — mnogi pokušavaju da razvijaju kubove ili koriste neke druge transformacije. Uvek proverite da li se može primeniti \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\).
  • Biranje odgovora (A) umesto (E) — razlika kubova (\(a^3 - b^3\)) daje faktor \((a - b)\), ali ovde imamo zbir kubova, pa je faktor \((a + b)\), odnosno \((\sin x + \cos x)\), a ne \((\cos x - \sin x)\).
Dodatni izazov

Pojednostavite izraz \(\dfrac{\sin^3 x - \cos^3 x}{2 + \sin 2x}\). Kakav rezultat očekujete na osnovu analogije sa rešenim zadatkom?

Nagoveštaj

Koristite formulu za razliku kubova: \(a^3 - b^3 = (a - b)(a^2 + ab + b^2)\). U imeniocu ćete dobiti \(2 + 2\sin x \cos x = 2(1 + \sin x \cos x)\). Zajednički faktor za skraćivanje biće \((1 + \sin x \cos x)\).
