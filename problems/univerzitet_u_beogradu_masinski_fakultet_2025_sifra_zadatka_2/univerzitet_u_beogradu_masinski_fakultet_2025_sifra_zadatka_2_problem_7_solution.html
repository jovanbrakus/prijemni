Deljivost polinoma trećeg stepena

Algebra — izjednačavanje koeficijenata pri deljenju polinoma

Postavka zadatka

Ako je polinom \(x^3 + 2x^2 + ax + b\) deljiv polinomom \(x^2 + x + ab\) \((a, b \in \mathbb{R})\), onda je \(a + b\) jednako:

(A)
\(-1\)
(B)
\(\dfrac{1}{3}\)
(C)
\(1\)
(D)
\(3\)
(E)
\(-3\)

Plan rešavanja

Pošto je polinom trećeg stepena deljiv polinomom drugog stepena, količnik mora biti linearan polinom \(x + k\). Pomnožićemo delilac sa količnikom, razviti proizvod, pa izjednačiti koeficijente uz iste stepene od \(x\) — to daje sistem jednačina po \(a\), \(b\) i \(k\).

📐 Ključne formule i teoreme
  • Deljivost polinoma: Ako je \(P(x)\) deljiv sa \(Q(x)\), onda postoji polinom \(R(x)\) takav da je \(P(x) = Q(x) \cdot R(x)\), pri čemu je \(\deg P = \deg Q + \deg R\).
  • Izjednačavanje koeficijenata: Dva polinoma su jednaka ako i samo ako su im koeficijenti uz iste stepene promenljive jednaki.
  • Množenje polinoma: Svaki sabirak jednog polinoma se množi sa svakim sabirkom drugog, a zatim se sabiraju članovi istog stepena.

Vizuelni prikaz — proizvod polinoma

Dijagram prikazuje kako se koeficijenti iz množenja polinoma uparuju

Rešenje korak po korak

1

Odredimo oblik količnika

Označimo polinome:

\[P(x) = x^3 + 2x^2 + ax + b, \quad Q(x) = x^2 + x + ab\]

Stepen \(P(x)\) je 3, a stepen \(Q(x)\) je 2. Ako je \(P(x)\) deljiv sa \(Q(x)\), količnik mora biti polinom stepena \(3 - 2 = 1\), dakle oblika:

\[R(x) = x + k\]

Koeficijent uz \(x\) u količniku mora biti 1, jer su vodeći koeficijenti oba polinoma jednaki 1.

2

Pomnožimo delilac i količnik

Važi jednakost \(P(x) = Q(x) \cdot R(x)\), pa razvijamo desnu stranu:

\[\begin{align} (x^2 + x + ab)(x + k) &= x^3 + kx^2 + x^2 + kx + abx + abk \\[6pt] &= x^3 + (k+1)x^2 + (k + ab)x + abk \end{align}\]
3

Izjednačimo koeficijente

Poredimo koeficijente sa levom stranom \(x^3 + 2x^2 + ax + b\):

\[\begin{align} x^2: &\quad k + 1 = 2 \\[6pt] x^1: &\quad k + ab = a \\[6pt] x^0: &\quad abk = b \end{align}\]
🧠 Proveri sebe — koliko iznosi \(k\)?

Iz prve jednačine: \(k + 1 = 2\), dakle \(k = 1\).

4

Rešimo prvu jednačinu

Iz koeficijenta uz \(x^2\):

\[k + 1 = 2 \implies k = 1\]
5

Zamenimo \(k = 1\) u treću jednačinu

Iz koeficijenta uz slobodan član (\(x^0\)):

\[abk = b \implies ab \cdot 1 = b \implies ab = b\]

Prebacimo sve na jednu stranu:

\[ab - b = 0 \implies b(a - 1) = 0\]

Dakle, ili je \(b = 0\) ili je \(a = 1\).

6

Zamenimo \(k = 1\) u drugu jednačinu

Iz koeficijenta uz \(x\):

\[k + ab = a \implies 1 + ab = a \implies a - ab = 1 \implies a(1 - b) = 1\]
🧠 Proveri sebe — šta sledi ako je \(a = 1\)?

Ako je \(a = 1\), iz jednačine \(a(1 - b) = 1\) dobijamo \(1 \cdot (1 - b) = 1\), pa je \(b = 0\). Dakle \(a + b = 1\).

7

Razmotrimo slučajeve

Slučaj 1: \(b = 0\)

Iz \(a(1 - b) = 1\) dobijamo \(a \cdot 1 = 1\), dakle \(a = 1\).

\[a + b = 1 + 0 = 1\]

Slučaj 2: \(a = 1\)

Iz \(a(1 - b) = 1\) dobijamo \(1 \cdot (1 - b) = 1\), dakle \(b = 0\).

\[a + b = 1 + 0 = 1\]

Oba slučaja daju isti rezultat: \(a = 1\), \(b = 0\), pa je \(a + b = 1\).

8

Verifikacija — proverimo deljenje

Za \(a = 1\) i \(b = 0\):

\[P(x) = x^3 + 2x^2 + x = x(x^2 + 2x + 1) = x(x + 1)^2\]
\[Q(x) = x^2 + x + 0 = x(x + 1)\]

Delimo:

\[\frac{P(x)}{Q(x)} = \frac{x(x+1)^2}{x(x+1)} = x + 1\]

Količnik je \(x + 1\), što je polinom — deljenje je tačno, bez ostatka. ✓

🧠 Proveri sebe — da li se \(k = 1\) slaže sa količnikom?

Da! Utvrdili smo da je \(k = 1\), pa je količnik \(R(x) = x + 1\), što se slaže sa rezultatom deljenja.

Ključni uvid

Disjunkcija \(b(a-1) = 0\) nudi dva slučaja, ali jednačina \(a(1-b) = 1\) ih oba svodi na isti par vrednosti. Sistem je konzistentan i ima jedinstveno rešenje:

\[a = 1, \quad b = 0, \quad a + b = 1\]

Konačan odgovor

\(a + b = 1\)
(A) \(-1\)
(B) \(\dfrac{1}{3}\)
(C) \(1\)
(D) \(3\)
(E) \(-3\)

Verifikacija: Za \(a = 1\), \(b = 0\): polinom \(x^3 + 2x^2 + x = x(x+1)^2\) zaista je deljiv sa \(x^2 + x = x(x+1)\), jer je količnik \(x + 1\) — polinom bez ostatka.

Česte greške

⚠ Zaboraviti da \(k\) mora biti 1: Ako se ne odredi \(k\) iz koeficijenta uz \(x^2\) pre nego što se koriste ostale jednačine, sistem postaje previše komplikovan i lako se pogreši.
⚠ Odbaciti slučaj \(b = 0\): Neki učenici pretpostave da \(b \neq 0\) jer je deo zadatka parametar, ali \(b = 0\) je potpuno validno rešenje koje zadovoljava sve uslove.

Dodatni izazov

Šta ako umesto \(x^2 + x + ab\) delilac bude \(x^2 + x + a + b\)? Koje vrednosti \(a\) i \(b\) tada čine polinom \(x^3 + 2x^2 + ax + b\) deljivim?

💡 Nagoveštaj

Postupak je isti: množenje \((x^2 + x + a + b)(x + k)\), razvijanje i izjednačavanje koeficijenata. Iz koeficijenta uz \(x^2\) opet dobijate \(k = 1\). Zatim iz preostalih jednačina dobijate sistem po \(a\) i \(b\) — pokušajte da ga rešite!
