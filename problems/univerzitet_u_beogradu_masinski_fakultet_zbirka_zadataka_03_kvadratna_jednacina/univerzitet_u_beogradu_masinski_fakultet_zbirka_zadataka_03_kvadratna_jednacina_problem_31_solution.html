Zbir kubova korena kvadratne jednačine

Vijetove formule -- izražavanje simetričnih funkcija korena

Zadatak

Zbir kubova rešenja jednačine \(x^2 + ax + b = 0\) \((a, b \in \mathbb{R})\) je:

(A)
\(-a(a^2 - b)\)
(B)
\(a(3b - a^2)\)
(C)
\(-a(a^2 + b)\)
(D)
\(-a^3\)
(E)
\(a(a^2 - 3b)\)

Plan rešavanja

Primenićemo Vijetove formule da izrazimo zbir i proizvod korena preko koeficijenata \(a\) i \(b\). Zatim ćemo koristiti formulu za zbir kubova \(x_1^3 + x_2^3\) i sve izraziti preko \(a\) i \(b\).

Ključne formule i teoreme
  • Vijetove formule za \(x^2 + px + q = 0\): \(x_1 + x_2 = -p\), \(\;x_1 \cdot x_2 = q\)
  • Zbir kubova: \(x_1^3 + x_2^3 = (x_1 + x_2)(x_1^2 - x_1 x_2 + x_2^2)\)
  • Zbir kvadrata: \(x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1 x_2\)
  • Kombinovano: \(x_1^2 - x_1 x_2 + x_2^2 = (x_1 + x_2)^2 - 3x_1 x_2\)

Vizuelni prikaz

Primer: za \(a = -3\), \(b = 2\) jednačina \(x^2 - 3x + 2 = 0\) ima korene \(x_1 = 1\) i \(x_2 = 2\).

Parabola \(y = x^2 - 3x + 2\) sa obeleženim korenima

Rešenje korak po korak

1

Primenimo Vijetove formule

Za jednačinu \(x^2 + ax + b = 0\) sa korenima \(x_1\) i \(x_2\):

\[x_1 + x_2 = -a\]
\[x_1 \cdot x_2 = b\]
2

Zapišimo formulu za zbir kubova

Koristimo poznati identitet:

\[x_1^3 + x_2^3 = (x_1 + x_2)(x_1^2 - x_1 x_2 + x_2^2)\]
Proveri se: Kako se izražava \(x_1^2 + x_2^2\) pomoću zbira i proizvoda korena?

\(x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1 x_2\). Ovo dobijamo razvojem kvadrata zbira.

3

Transformišimo drugi faktor

Izraz \(x_1^2 - x_1 x_2 + x_2^2\) preuredimo:

\[x_1^2 - x_1 x_2 + x_2^2 = (x_1^2 + x_2^2) - x_1 x_2\]

Zamenimo \(x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1 x_2\):

\[= (x_1 + x_2)^2 - 2x_1 x_2 - x_1 x_2\]
\[= (x_1 + x_2)^2 - 3x_1 x_2\]
4

Zamenimo Vijetove izraze

Sada u formulu za zbir kubova uvrstimo \(x_1 + x_2 = -a\) i \(x_1 x_2 = b\):

\[x_1^3 + x_2^3 = (-a)\left((-a)^2 - 3b\right)\]
\[= (-a)(a^2 - 3b)\]
5

Pojednostavimo

Raspišemo proizvod:

\[= -a^3 + 3ab\]

Izvedemo \(a\) ispred zagrade:

\[= a(-a^2 + 3b) = a(3b - a^2)\]
Proveri se: Koliko je \(x_1^3 + x_2^3\) za jednačinu \(x^2 - 3x + 2 = 0\)?

Koreni su \(x_1 = 1\) i \(x_2 = 2\), pa je \(x_1^3 + x_2^3 = 1 + 8 = 9\).
Formulom: \(a = -3\), \(b = 2\), pa \(a(3b - a^2) = (-3)(6 - 9) = (-3)(-3) = 9\). Tačno!

Proveri se: A za jednačinu \(x^2 + x - 2 = 0\)?

Koreni su \(x_1 = 1\) i \(x_2 = -2\), pa je \(x_1^3 + x_2^3 = 1 - 8 = -7\).
Formulom: \(a = 1\), \(b = -2\), pa \(a(3b - a^2) = 1(-6 - 1) = -7\). Tačno!

Ključni uvid

Svaka simetrična funkcija korena kvadratne jednačine može se izraziti samo pomoću koeficijenata, bez poznavanja samih korena. Formula za zbir kubova kombinuje identitet \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\) sa Vijetovim formulama.

\[x_1^3 + x_2^3 = a(3b - a^2)\]

Konačan odgovor

\(x_1^3 + x_2^3 = a(3b - a^2)\)
(A) \(-a(a^2 - b)\) (B) \(a(3b - a^2)\) (C) \(-a(a^2 + b)\) (D) \(-a^3\) (E) \(a(a^2 - 3b)\)
Verifikacija: Za \(x^2 - 3x + 2 = 0\) (koreni 1 i 2): direktno \(1^3 + 2^3 = 9\). Formulom: \((-3)(6 - 9) = 9\). Rezultat se poklapa. Opcija (E) daje \((-3)(9 - 6) = -9\) -- pogrešan znak, pa se upravo po tome razlikuje od tačnog odgovora.

Česte greške

Zamena znaka: Pažljivo sa \(x_1 + x_2 = -a\), ne \(+a\)! Ako se pogreši znak, dobija se \(a(a^2 - 3b)\) umesto \(a(3b - a^2)\) -- to je opcija (E), koja je "zamka".
Pogrešan identitet: Neki učenici koriste \((x_1 + x_2)^3 = x_1^3 + x_2^3\), ali to nije tačno. Razlika je \((x_1+x_2)^3 = x_1^3 + 3x_1^2 x_2 + 3x_1 x_2^2 + x_2^3\).

Dodatni izazov

Izračunaj \(x_1^4 + x_2^4\) za istu jednačinu \(x^2 + ax + b = 0\), koristeći samo \(a\) i \(b\).

Savet

Koristi \(x_1^4 + x_2^4 = (x_1^2 + x_2^2)^2 - 2(x_1 x_2)^2\), gde je \(x_1^2 + x_2^2 = a^2 - 2b\). Rezultat: \((a^2 - 2b)^2 - 2b^2 = a^4 - 4a^2 b + 2b^2\).
