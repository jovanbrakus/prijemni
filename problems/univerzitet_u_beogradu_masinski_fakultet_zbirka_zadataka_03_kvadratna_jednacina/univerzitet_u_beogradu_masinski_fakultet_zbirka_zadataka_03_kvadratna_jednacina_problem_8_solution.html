Razlika korena kvadratne jednačine

Primena Vijetovih formula i osobina korena

Postavka zadatka

Koreni jednačine \(x^2 + (p+2)x + p + 1 = 0\) razlikuju se za 2 ako i samo ako je:

(A)
\(|p| = 1\)
(B)
\(|p| = 3\)
(C)
\(|p| = 2\)
(D)
\(|p| = 1{,}5\)
(E)
\(|p| = 4\)

Plan rešavanja

Upotrebićemo Vijetove formule za zbir i proizvod korena, zatim poznati identitet \((x_1 - x_2)^2 = (x_1 + x_2)^2 - 4x_1 x_2\) kako bismo iz uslova \(|x_1 - x_2| = 2\) odredili vrednost parametra \(p\).

Ključne formule i teoreme
  • Vijetove formule za jednačinu \(x^2 + bx + c = 0\): \(x_1 + x_2 = -b\) i \(x_1 \cdot x_2 = c\).
  • Identitet za razliku korena: \((x_1 - x_2)^2 = (x_1 + x_2)^2 - 4 \cdot x_1 x_2\). Ovo sledi iz razvoja \((x_1 - x_2)^2 = x_1^2 - 2x_1x_2 + x_2^2\) i dodavanja i oduzimanja \(2x_1x_2\).
  • Diskriminanta: \(D = b^2 - 4ac\). Jednačina ima dva realna korena ako je \(D \geq 0\). Važi i \((x_1 - x_2)^2 = \frac{D}{a^2}\).

Vizuelni prikaz

Pomerajte klizač da menjate vrednost parametra p i posmatrate kako se koreni menjaju

|x₁ - x₂| = 2

Rešenje korak po korak

1

Identifikujemo koeficijente

Data jednačina je \(x^2 + (p+2)x + (p+1) = 0\). Poredimo sa opštim oblikom \(x^2 + bx + c = 0\):

\[b = p + 2, \quad c = p + 1\]
2

Primenimo Vijetove formule

Ako su \(x_1\) i \(x_2\) koreni jednačine, tada:

\[x_1 + x_2 = -(p + 2)\]
\[x_1 \cdot x_2 = p + 1\]
3

Postavimo uslov za razliku korena

Koreni se razlikuju za 2, tj. \(|x_1 - x_2| = 2\). Kvadriramo obe strane:

\[(x_1 - x_2)^2 = 4\]
Proveri se: Kako se \((x_1 - x_2)^2\) izražava pomoću zbira i proizvoda korena?

\((x_1 - x_2)^2 = (x_1 + x_2)^2 - 4x_1 x_2\). Ovo je ključni identitet za ovaj zadatak.

4

Primenimo identitet za razliku korena

Koristimo poznati identitet:

\[(x_1 - x_2)^2 = (x_1 + x_2)^2 - 4 \cdot x_1 x_2\]

Zamenjujemo Vijetove formule:

\[(x_1 - x_2)^2 = (-(p+2))^2 - 4(p+1)\]
5

Razvijemo i uprostimo

Razvijemo kvadrat:

\[(p+2)^2 = p^2 + 4p + 4\]

Razvijemo drugi član:

\[4(p+1) = 4p + 4\]

Oduzmemo:

\[(x_1 - x_2)^2 = p^2 + 4p + 4 - 4p - 4 = p^2\]

Primetimo da su se članovi sa \(p\) i slobodni članovi poništili, ostaje samo \(p^2\).

Proveri se: Koliko je \((x_1 - x_2)^2\) u funkciji od \(p\)?

\((x_1 - x_2)^2 = p^2\). Ovo je iznenađujuće jednostavan rezultat!

6

Rešimo jednačinu

Iz uslova \((x_1 - x_2)^2 = 4\) i rezultata \((x_1 - x_2)^2 = p^2\) dobijamo:

\[p^2 = 4\]

Odavde sledi:

\[|p| = 2\]
7

Verifikacija za \(p = 2\)

Zamenjujemo \(p = 2\) u polaznu jednačinu:

\[x^2 + 4x + 3 = 0\]

Faktorizujemo:

\[(x + 1)(x + 3) = 0 \implies x_1 = -1, \quad x_2 = -3\]

Razlika korena: \(|x_1 - x_2| = |-1 - (-3)| = |2| = 2\) ✓

8

Verifikacija za \(p = -2\)

Zamenjujemo \(p = -2\) u polaznu jednačinu:

\[x^2 + 0 \cdot x + (-1) = 0 \implies x^2 - 1 = 0\]

Faktorizujemo:

\[(x - 1)(x + 1) = 0 \implies x_1 = 1, \quad x_2 = -1\]

Razlika korena: \(|x_1 - x_2| = |1 - (-1)| = |2| = 2\) ✓

Proveri se: Šta se dešava za \(p = 1\) (odgovor A iz ključa)?

Za \(p = 1\): \(x^2 + 3x + 2 = 0 \Rightarrow (x+1)(x+2) = 0\), koreni su \(-1\) i \(-2\), razlika je \(|{-1} - ({-2})| = 1 \neq 2\). Dakle, odgovor (A) ne zadovoljava uslov.

Napomena o ključu: Zvanični ključ navodi odgovor (A) \(|p| = 1\), ali numerička provera pokazuje da je to pogrešno. Za \(p = \pm 1\) razlika korena iznosi 1, a ne 2. Algebarski i numerički, tačan odgovor je (C) \(|p| = 2\).

Ključni uvid

Posebna struktura koeficijenata ove jednačine daje izuzetno elegantan rezultat:

\[(x_1 - x_2)^2 = p^2\]

Članovi sa \(p\) i slobodni članovi se potpuno poništavaju, pa je razlika korena direktno jednaka \(|p|\).

Konačan odgovor

\(|p| = 2\)
(A) \(|p| = 1\) (B) \(|p| = 3\) (C) \(|p| = 2\) (D) \(|p| = 1{,}5\) (E) \(|p| = 4\)
Provera: Za \(p = 2\) koreni su \(-1\) i \(-3\) (razlika 2). Za \(p = -2\) koreni su \(1\) i \(-1\) (razlika 2). Oba slučaja potvrđuju rezultat. Suprotno, za \(p = \pm 1\) (odgovor A) razlika korena iznosi 1, ne 2.

Česte greške

  • Zaboraviti apsolutnu vrednost: Iz \(p^2 = 4\) sledi \(p = 2\) ili \(p = -2\), tj. \(|p| = 2\). Ne treba uzeti samo pozitivnu vrednost.
  • Pogrešno primenjivanje identiteta: Česta greška je pisati \((x_1 - x_2)^2 = (x_1 + x_2)^2 - 2x_1x_2\) umesto ispravnog \((x_1 + x_2)^2 - 4x_1x_2\). Faktor 4 je ključan.

Dodatni izazov

Za koju vrednost parametra \(p\) koreni jednačine \(x^2 + (p+2)x + p + 1 = 0\) razlikuju se za 3?

Savet

Koristite isti identitet: \((x_1 - x_2)^2 = p^2 = 9\), pa je \(|p| = 3\). Proverite zamenom u polaznu jednačinu.
