Inverzna funkcija i kompozicija

Algebra — Nalazenje inverzne funkcije i izracunavanje vrednosti kompozicije

Postavka zadatka

Ako je \(f(x) = \dfrac{1 - x}{1 + x}\) i \(g(x) = \dfrac{1}{1 + x^2}\), onda je \(f^{-1}(g(0))\) jednako:

(A)
\(0\)
(B)
\(1\)
(C)
\(-1\)
(D)
\(2\)
(E)
\(-2\)

P Plan resavanja

Prvo cemo izracunati vrednost \(g(0)\), zatim naci inverznu funkciju \(f^{-1}(x)\) i na kraju izracunati \(f^{-1}(g(0))\).

T Teorijska podloga

Prikazi teoriju
  • Inverzna funkcija: Ako je \(y = f(x)\), onda je \(f^{-1}(y) = x\). Nalazimo je tako sto iz jednacine \(y = f(x)\) izrazimo \(x\) preko \(y\).
  • Vrednost funkcije u tacki: Da bismo izracunali \(g(a)\), zamenimo \(x = a\) u formulu za \(g(x)\).
  • Involucija: Funkcija \(f\) je involucija ako vazi \(f(f(x)) = x\), tj. ako je sama sebi inverzna: \(f^{-1} = f\).
  • Kompozicija: \(f^{-1}(g(0))\) znaci: prvo izracunaj \(g(0)\), pa taj rezultat uvrsti u \(f^{-1}\).

D Dijagram

Grafici funkcija f(x) i g(x) sa oznacenom kljucnom tackom

1 Korak 1: Izracunavanje \(g(0)\)

Zamena \(x = 0\) u \(g(x)\)

Funkcija \(g\) je definisana kao:

\[ g(x) = \frac{1}{1 + x^2} \]

Uvrstimo \(x = 0\):

\[ g(0) = \frac{1}{1 + 0^2} = \frac{1}{1} = 1 \]

2 Korak 2: Nalazenje inverzne funkcije \(f^{-1}(x)\)

Izrazavanje \(x\) iz \(y = f(x)\)

Postavimo \(y = f(x)\):

\[ y = \frac{1 - x}{1 + x} \]

Pomnozimo obe strane sa \((1 + x)\):

\[ y(1 + x) = 1 - x \]

Raspisemo levu stranu:

\[ y + yx = 1 - x \]

Prenesemo clanove sa \(x\) na jednu stranu:

\[ yx + x = 1 - y \]

Izlucimo \(x\):

\[ x(y + 1) = 1 - y \]

Konacno:

\[ x = \frac{1 - y}{1 + y} \]

Formula za inverznu funkciju

Dakle, inverzna funkcija glasi:

\[ f^{-1}(y) = \frac{1 - y}{1 + y} \]

Primetimo: \(f^{-1}\) ima istu formulu kao \(f\)! To znaci da je \(f\) involucija: \(f(f(x)) = x\).

Proveri se: Koja je vrednost \(f(f(2))\)?
\(f(2) = \frac{1-2}{1+2} = -\frac{1}{3}\), pa \(f\!\left(-\frac{1}{3}\right) = \frac{1+\frac{1}{3}}{1-\frac{1}{3}} = \frac{\frac{4}{3}}{\frac{2}{3}} = 2\). Dakle \(f(f(2)) = 2\). Potvrda involucije!

3 Korak 3: Izracunavanje \(f^{-1}(g(0))\)

Uvrstavanje \(g(0) = 1\)

Sada izracunamo:

\[ f^{-1}(g(0)) = f^{-1}(1) = \frac{1 - 1}{1 + 1} = \frac{0}{2} = 0 \]
Proveri se: Ako je \(f^{-1}(1) = 0\), sta je \(f(0)\)?
\(f(0) = \frac{1 - 0}{1 + 0} = 1\). Posto \(f(0) = 1\), sledi \(f^{-1}(1) = 0\). Tacno!

! Kljucni uvid

Funkcija \(f(x) = \frac{1-x}{1+x}\) je involucija -- ona je sama sebi inverzna. To znaci da je \(f^{-1} = f\), pa umesto da trazimo inverznu, mogli smo odmah da izracunamo \(f(g(0)) = f(1) = 0\).

Ovo je korisno prepoznati jer skracuje racun: ne moramo prolaziti kroz ceo postupak nalazenja inverzne funkcije.

✓ Konacan odgovor

\( f^{-1}(g(0)) = 0 \)
(A) \(0\) ✓ (B) \(1\) (C) \(-1\) (D) \(2\) (E) \(-2\)
Verifikacija

Proveravamo u dva koraka:

  • \(g(0) = \frac{1}{1 + 0} = 1\) ✓
  • \(f(0) = \frac{1 - 0}{1 + 0} = 1\), dakle \(f^{-1}(1) = 0\) ✓

Rezultat je potvrden. Odgovor je (A) 0.

⚠ Ceste greske

Zamena redosleda: Zadatak trazi \(f^{-1}(g(0))\), ne \(g(f^{-1}(0))\). Redosled je bitan: prvo racunamo \(g(0)\), pa rezultat uvrstimo u \(f^{-1}\).
Greska sa inverznom: Inverzna funkcija \(f^{-1}(x)\) nije isto sto i \(\frac{1}{f(x)}\). To je funkcija koja „ponistava" dejstvo \(f\), tj. \(f^{-1}(f(x)) = x\).

★ Izazov za razmisljanje

Prosiren zadatak: Izracunajte \(f^{-1}(g(1))\). Savet: Prvo odredite \(g(1)\), a zatim primenite inverznu funkciju.

Nagovest

\(g(1) = \frac{1}{1+1} = \frac{1}{2}\), pa \(f^{-1}\!\left(\frac{1}{2}\right) = \frac{1 - \frac{1}{2}}{1 + \frac{1}{2}} = \frac{\frac{1}{2}}{\frac{3}{2}} = \frac{1}{3}\).
