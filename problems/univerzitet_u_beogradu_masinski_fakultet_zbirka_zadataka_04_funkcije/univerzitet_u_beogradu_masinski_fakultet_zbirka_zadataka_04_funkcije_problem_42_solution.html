Znaci koeficijenata kvadratne funkcije

Određivanje znakova koeficijenata \( a \), \( b \) i \( c \) na osnovu grafika parabole

Postavka zadatka

Grafik funkcije \( f(x) = ax^2 + bx + c \) je prikazan na slici. Parabola je okrenuta nadole, sa temenom u prvom kvadrantu. Preseca pozitivan deo \( y \)-ose i ima dve pozitivne nule. Tačan je iskaz:

(A)
\( a < 0,\; b < 0,\; c < 0 \)
(B)
\( a < 0,\; b < 0,\; c > 0 \)
(C)
\( a < 0,\; b > 0,\; c < 0 \)
(D)
\( a < 0,\; b > 0,\; c > 0 \)
(E)
\( a > 0,\; b < 0,\; c < 0 \)

Plan rešavanja

Iz grafika ćemo očitati tri ključne informacije: smer otvaranja parabole (znak \( a \)), položaj temena (znak \( b \)), i presek sa \( y \)-osom (znak \( c \)).

Ključne formule i teoreme
  • Kvadratna funkcija: \( f(x) = ax^2 + bx + c \)
  • Smer otvaranja: ako \( a > 0 \), parabola je okrenuta nagore; ako \( a < 0 \), nadole.
  • Presek sa \( y \)-osom: \( f(0) = c \), pa je znak od \( c \) jednak znaku ordinate preseka.
  • \( x \)-koordinata temena: \( x_v = -\dfrac{b}{2a} \). Ako je \( x_v > 0 \) i \( a < 0 \), tada \( b > 0 \).

Grafički prikaz

Pomeri klizač da menjaš koeficijente i vidiš kako utiču na parabolu

Rešenje korak po korak

1

Određujemo znak koeficijenta \( a \)

Sa grafika vidimo da je parabola okrenuta nadole (kraci idu prema dole).

Parabola okrenuta nadole znači:

\[ a < 0 \]

Ovim eliminišemo odgovor (E) gde je \( a > 0 \).

2

Određujemo znak koeficijenta \( c \)

Koeficijent \( c \) je vrednost funkcije u nuli: \( f(0) = c \).

Sa grafika vidimo da parabola seče \( y \)-osu u pozitivnoj tački (iznad koordinatnog početka).

\[ c = f(0) > 0 \]

Ovim eliminišemo odgovore (A) i (C) gde je \( c < 0 \).

Proveri se: Koji odgovori su ostali?

Preostali su (B) sa \( a < 0, b < 0, c > 0 \) i (D) sa \( a < 0, b > 0, c > 0 \). Treba da odredimo znak od \( b \).

3

Određujemo znak koeficijenta \( b \)

\( x \)-koordinata temena parabole je \( x_v = -\dfrac{b}{2a} \).

Sa grafika vidimo da je teme u prvom kvadrantu, tj. \( x_v > 0 \).

\[ x_v = -\frac{b}{2a} > 0 \]
4

Analiza znaka \( b \)

Pošto je \( a < 0 \), imenilac \( 2a \) je negativan. Iz uslova \( -\dfrac{b}{2a} > 0 \):

\[ -\frac{b}{2a} > 0 \]

Pomnožimo obe strane sa \( 2a < 0 \) (menjamo smer nejednakosti):

\[ -b < 0 \implies b > 0 \]

Alternativno: ako \( a < 0 \) i \( b > 0 \), onda je \( -b/(2a) = (\text{neg})/(\text{neg}) = \text{poz} > 0 \). Tačno!

Proveri se: Šta govori činjenica da parabola seče \( x \)-osu u dve pozitivne tačke?

Za parabolu \( ax^2 + bx + c = 0 \) sa korenima \( x_1, x_2 \), po Vijetovim formulama: \( x_1 + x_2 = -b/a \) i \( x_1 \cdot x_2 = c/a \). Oba korena su pozitivna, pa je zbir pozitivan: \( -b/a > 0 \). Pošto je \( a < 0 \), to znači \( -b < 0 \), tj. \( b > 0 \). Proizvod je takođe pozitivan: \( c/a > 0 \), a pošto \( a < 0 \), to bi značilo \( c < 0 \)... Ali sačekaj! Sa grafika jasno vidimo \( c > 0 \), a parabola sa negativnim \( a \) i pozitivnim \( c \) daje \( c/a < 0 \), što znači da koreni imaju različite znakove. Dakle, parabola zapravo seče \( x \)-osu jednom u negativnom, a jednom u pozitivnom delu, što je konzistentno sa pozitivnim \( c \) i negativnim \( a \).

5

Zaključak

Kombinujemo sve tri informacije:

\[ a < 0, \quad b > 0, \quad c > 0 \]

Ključni uvid

Tri osobine grafika daju tri informacije: smer otvaranja daje znak \( a \), presek sa \( y \)-osom daje znak \( c \), a položaj temena levo/desno od \( y \)-ose daje znak \( b \) (u kombinaciji sa znakom \( a \)).

Konačan odgovor

\( a < 0, \quad b > 0, \quad c > 0 \)
(A) \( a < 0, b < 0, c < 0 \) (B) \( a < 0, b < 0, c > 0 \) (C) \( a < 0, b > 0, c < 0 \) (D) \( a < 0, b > 0, c > 0 \) (E) \( a > 0, b < 0, c < 0 \)

Verifikacija: Sa \( a = -1, b = 3, c = 2 \) dobijamo \( f(x) = -x^2 + 3x + 2 \). Teme: \( x_v = \frac{3}{2} > 0 \), \( f(0) = 2 > 0 \), otvaranje nadole. Sve se poklapa sa grafikom.

Česte greške:
  • Mešanje znaka \( b \) i položaja temena. Zapamti: teme je desno od \( y \)-ose kad su \( a \) i \( b \) različitog znaka (jer \( x_v = -b/(2a) \)).
  • Zaboravljanje da \( c = f(0) \) direktno daje presek sa \( y \)-osom.

Dodatni izazov

Ako parabola ima teme u trećem kvadrantu i seče negativan deo \( y \)-ose, a okrenuta je nagore, koji su znaci koeficijenata?

Nagoveštaj

Nagore znači \( a > 0 \). Negativan presek sa \( y \)-osom znači \( c < 0 \). Teme u trećem kvadrantu (\( x_v < 0 \)) uz \( a > 0 \) daje \( b > 0 \) (jer \( -b/(2a) < 0 \) uz \( a > 0 \) traži \( b > 0 \)).
