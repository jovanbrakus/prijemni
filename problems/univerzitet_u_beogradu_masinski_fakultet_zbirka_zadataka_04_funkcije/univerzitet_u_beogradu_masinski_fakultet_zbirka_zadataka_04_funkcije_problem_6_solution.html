Translacija argumenta funkcije

Složena funkcija -- nalaženje funkcije za pomereni argument pomoću smene

Zadatak

Ako je

\[f(x - 1) = x^2 + 2x - 3\]

onda je \(f(x + 1)\) jednako:

(A)
\(x^2 + 2x - 3\)
(B)
\(x^2 - 2x - 3\)
(C)
\(x^2 + 6x + 5\)
(D)
\(x^2 + 4x\)
(E)
\(x^2 - 4\)

Plan rešavanja

Prvo ćemo uvesti smenu \(t = x - 1\) da bismo dobili eksplicitnu formulu za \(f(t)\). Zatim ćemo u tu formulu uvrstiti \(t = x + 1\) da dobijemo traženo \(f(x + 1)\). Prikazaćemo i alternativni pristup -- direktnu smenu \(x \to x + 2\) u polaznoj jednačini.

Podsetnik iz teorije

Ključne formule i teoreme

Smena za nalaženje funkcije: Ako je dato \(f(g(x)) = h(x)\), stavljamo \(t = g(x)\), izrazimo \(x\) preko \(t\), pa uvrštavamo u \(h(x)\).

\[t = g(x) \implies x = g^{-1}(t) \implies f(t) = h(g^{-1}(t))\]

Kvadrat binoma:

\[(a + b)^2 = a^2 + 2ab + b^2\]

Translacija argumenta: Zamena \(x\) sa \(x + c\) u \(f(x - 1)\) daje \(f(x + c - 1)\). Za \(c = 2\) dobijamo \(f(x + 1)\).

Vizuelni prikaz

Grafici tri funkcije: \(f(x-1)\), \(f(x)\) i \(f(x+1)\) -- pomeranje argumenta pomera parabolu

Rešenje korak po korak

Metod 1: Smena promenljive

1

Uvodimo smenu

Neka je \(t = x - 1\). Tada je \(x = t + 1\).

Leva strana polazne jednačine postaje upravo \(f(t)\).

2

Uvrštavamo u desnu stranu

Zamenjujemo \(x = t + 1\) u izraz \(x^2 + 2x - 3\):

\[f(t) = (t + 1)^2 + 2(t + 1) - 3\]
3

Raspisujemo kvadrat binoma

Primenjujemo formulu \((t + 1)^2 = t^2 + 2t + 1\):

\[f(t) = t^2 + 2t + 1 + 2t + 2 - 3\]
Proveri se: Koliko je \((t+1)^2\) raspisano?

\((t+1)^2 = t^2 + 2 \cdot t \cdot 1 + 1^2 = t^2 + 2t + 1\)

4

Sređujemo

Skupljamo slične članove:

\[f(t) = t^2 + (2t + 2t) + (1 + 2 - 3)\]
\[f(t) = t^2 + 4t\]

Sada znamo eksplicitnu formulu za \(f\).

5

Nalazimo \(f(x + 1)\)

Uvrštavamo \(t = x + 1\) u formulu \(f(t) = t^2 + 4t\):

\[f(x + 1) = (x + 1)^2 + 4(x + 1)\]
6

Raspisujemo i sređujemo

Ponovo primenjujemo kvadrat binoma:

\[f(x + 1) = x^2 + 2x + 1 + 4x + 4\]
\[f(x + 1) = x^2 + 6x + 5\]
Proveri se: Da li se \(x^2 + 6x + 5\) može faktorizovati?

Da! \(x^2 + 6x + 5 = (x + 1)(x + 5)\), jer je \(1 \cdot 5 = 5\) i \(1 + 5 = 6\).

Metod 2: Direktna smena (brži način)

1

Zamenjujemo \(x\) sa \(x + 2\)

Da od \(f(x - 1)\) dobijemo \(f(x + 1)\), treba da argument poraste za \(2\). To postižemo zamenom \(x \to x + 2\) u celoj jednačini:

\[f((x+2) - 1) = (x+2)^2 + 2(x+2) - 3\]
\[f(x + 1) = x^2 + 4x + 4 + 2x + 4 - 3\]
\[f(x + 1) = x^2 + 6x + 5\]
Proveri se: Zašto baš \(x + 2\)?

Jer želimo da \(x - 1\) postane \(x + 1\), a razlika je \((x+1) - (x-1) = 2\). Dakle, svako \(x\) treba zameniti sa \(x + 2\).

Ključni uvid

Kada je \(f\) data u obliku \(f(x - 1) = \ldots\), najbrži način da nađemo \(f(x + 1)\) jeste da uočimo koliko treba pomeriti argument: od \(x - 1\) do \(x + 1\) razlika je \(2\). Zato svuda zamenjujemo \(x\) sa \(x + 2\).

\[f(t) = t^2 + 4t \implies f(x+1) = x^2 + 6x + 5\]

Konačan odgovor

\(f(x + 1) = x^2 + 6x + 5\)
(A) \(x^2 + 2x - 3\)
(B) \(x^2 - 2x - 3\)
(C) \(x^2 + 6x + 5\)
(D) \(x^2 + 4x\)
(E) \(x^2 - 4\)
Verifikacija: Proveravamo za \(x = 0\): Iz \(f(t) = t^2 + 4t\) imamo \(f(1) = 1 + 4 = 5\). Iz našeg odgovora: \(f(0 + 1) = 0 + 0 + 5 = 5\). Tačno! Proveravamo i polazni uslov: \(f(x-1) = (x-1)^2 + 4(x-1) = x^2 - 2x + 1 + 4x - 4 = x^2 + 2x - 3\). Tačno!

Česte greške

Odgovor (D) kao zamka: Ako se pogrešno zaključi da je \(f(x) = x^2 + 4x\) i da je to ujedno i \(f(x+1)\), dolazi se do pogrešnog odgovora \(x^2 + 4x\). Zapamtite: \(f(x)\) i \(f(x+1)\) nisu ista stvar!
Odgovor (A) kao zamka: Ako se uopšte ne uradi smena i samo prepiše desna strana, dobija se \(x^2 + 2x - 3\), što je zapravo \(f(x-1)\), a ne \(f(x+1)\).

Dodatni izazov

Koristeći istu funkciju \(f(t) = t^2 + 4t\), odredite za koje vrednosti \(x\) važi \(f(x + 1) = 0\).

Nagoveštaj

Rešite \(x^2 + 6x + 5 = 0\). Faktorizujte: \((x + 1)(x + 5) = 0\), pa je \(x = -1\) ili \(x = -5\).
