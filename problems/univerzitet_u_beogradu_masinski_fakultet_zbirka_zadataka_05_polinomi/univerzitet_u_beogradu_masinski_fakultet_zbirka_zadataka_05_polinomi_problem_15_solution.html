Algebra — polinomi, nule polinoma, Vijetove formule

Postavka zadatka

\[ a(1 + b + c) + b(1 + a + c) + c(1 + a + b) \]

Plan rešavanja

Razvićemo dati izraz algebarski, a zatim primeniti Vijetove formule za kubni polinom da bismo zamenili zbir korena i zbir proizvoda korena po dva njihovim vrednostima.

Ključne formule i teoreme
  • Vijetove formule za kubni polinom: Za polinom \(px^3 + qx^2 + rx + s\) sa nulama \(a, b, c\): \[ a + b + c = -\frac{q}{p}, \quad ab + ac + bc = \frac{r}{p}, \quad abc = -\frac{s}{p} \]
  • Razvijanje izraza: Kada imamo izraz oblika \(a(1+b+c) + b(1+a+c) + c(1+a+b)\), korisno je raspisati ga i grupisati po poznatim simetričnim funkcijama korena.
  • Simetrične funkcije korena: Zbir korena \(a+b+c\) i zbir proizvoda po dva \(ab+ac+bc\) direktno se čitaju iz koeficijenata polinoma pomoću Vijetovih formula.

Shematski prikaz razvijanja izraza i primene Vijetovih formula

Rešenje korak po korak

1

Razvijemo izraz

Pomnožimo svaki faktor sa sadržajem zagrade:

\[ a(1 + b + c) = a + ab + ac \]
\[ b(1 + a + c) = b + ba + bc \]
\[ c(1 + a + b) = c + ca + cb \]
2

Saberemo sve članove

Sabiramo sva tri razvijena izraza i grupišemo slične članove:

\[ (a + b + c) + (ab + ac + ba + bc + ca + cb) \]

Primetimo da se svaki par pojavljuje dva puta: \(ab\) i \(ba\), \(ac\) i \(ca\), \(bc\) i \(cb\). Pošto je množenje komutativno (\(ab = ba\)):

\[ (a + b + c) + 2(ab + ac + bc) \]
Proveri se: Na šta se svodi naš izraz?
Izraz se svodi na \((a + b + c) + 2(ab + ac + bc)\) — zbir korena plus dvostruki zbir proizvoda korena po dva.
3

Primenimo Vijetove formule

Za polinom \(5x^3 + 4x^2 - 8x + 6\) sa nulama \(a, b, c\), koeficijenti su: \(p = 5\), \(q = 4\), \(r = -8\), \(s = 6\).

Zbir korena:

\[ a + b + c = -\frac{q}{p} = -\frac{4}{5} \]

Zbir proizvoda korena po dva:

\[ ab + ac + bc = \frac{r}{p} = \frac{-8}{5} = -\frac{8}{5} \]
Proveri se: Koliko iznosi \(2(ab + ac + bc)\)?
\(2 \cdot \left(-\dfrac{8}{5}\right) = -\dfrac{16}{5}\)
4

Zamenimo vrednosti u izraz

Uvrštavamo izračunate vrednosti u naš uprošćeni izraz:

\[ (a + b + c) + 2(ab + ac + bc) = -\frac{4}{5} + 2 \cdot \left(-\frac{8}{5}\right) \]
\[ = -\frac{4}{5} - \frac{16}{5} = -\frac{20}{5} = -4 \]
Proveri se: Da li se rezultat poklapa sa nekim od ponuđenih odgovora?
Da! Rezultat \(-4\) odgovara opciji (E).

Ključno zapažanje

Ključ ovog zadatka je prepoznati da se zadati izraz može zapisati pomoću elementarnih simetričnih funkcija korena polinoma. Razvijanjem izraza dobijamo oblik koji direktno odgovara Vijetovim formulama — nije potrebno naći same korene!

\[ a(1+b+c) + b(1+a+c) + c(1+a+b) = (a+b+c) + 2(ab+ac+bc) = -4 \]

Konačan odgovor

\( -4 \)

Tačan odgovor je (E)

(A) \(\dfrac{12}{5}\)
(B) \(-\dfrac{12}{5}\)
(C) \(-\dfrac{22}{5}\)
(D) \(4\)
(E) \(-4\)

Verifikacija: Razvijanje: \(a(1+b+c)+b(1+a+c)+c(1+a+b) = (a+b+c)+2(ab+ac+bc)\). Vijetove formule za \(5x^3+4x^2-8x+6\): \(a+b+c = -\frac{4}{5}\), \(ab+ac+bc = -\frac{8}{5}\). Rezultat: \(-\frac{4}{5} + 2 \cdot (-\frac{8}{5}) = -\frac{4}{5} - \frac{16}{5} = -\frac{20}{5} = -4\).

Česte greške

  • Greška pri brojanju: Zaboraviti da se svaki proizvod pojavljuje dva puta (npr. \(ab\) iz prvog i \(ba\) iz drugog sabirka). Lako se može dobiti samo \((a+b+c) + (ab+ac+bc)\) umesto korektnog \((a+b+c) + 2(ab+ac+bc)\).
  • Pogrešan znak u Vijetovim formulama: Zbir korena je \(-\frac{q}{p}\), ne \(\frac{q}{p}\). Ako se zaboravi minus, dobija se \(\frac{4}{5}\) umesto \(-\frac{4}{5}\), što vodi do pogrešnog odgovora.

Dodatni izazov

Za isti polinom \(5x^3 + 4x^2 - 8x + 6\), izračunaj vrednost izraza \(\dfrac{1}{a} + \dfrac{1}{b} + \dfrac{1}{c}\).

Pogledaj nagoveštaj
Svedi na zajednički imenilac: \(\dfrac{1}{a} + \dfrac{1}{b} + \dfrac{1}{c} = \dfrac{bc + ac + ab}{abc}\). Koristi Vijetove formule za \(ab+ac+bc\) i \(abc\). Rezultat je \(\dfrac{-8/5}{-6/5} = \dfrac{8}{6} = \dfrac{4}{3}\).
