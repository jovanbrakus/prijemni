Deljivost polinoma četvrtog stepena

Polinomi, deljivost i koreni jedinice

📜 Zadatak

Ako je \(x^4 + ax^2 + b\) (\(a, b \in \mathbb{R}\)) deljiv polinomom \(x^2 + x + 1\), onda je \(a + b\) jednako:

(A)
\(-1\)
(B)
\(2\)
(C)
\(0\)
(D)
\(1\)
(E)
\(-2\)
📌 Plan rešavanja
  1. Iskoristiti činjenicu da su koreni polinoma \(x^2 + x + 1\) primitivni kubni koreni jedinice
  2. Uvrstiti koren u \(x^4 + ax^2 + b = 0\) i odrediti \(a\) i \(b\)
Podsetnik iz teorije

Kubni koreni jedinice: Jednačina \(x^3 = 1\) ima tri korena: \(1\), \(\omega\) i \(\bar{\omega}\), gde je

\[\omega = \frac{-1 + i\sqrt{3}}{2}, \quad \bar{\omega} = \frac{-1 - i\sqrt{3}}{2}\]

Minimalni polinom: Primitivni kubni koreni \(\omega\) i \(\bar{\omega}\) su koreni polinoma

\[x^2 + x + 1 = 0\]

Ključne osobine:

\[\omega^2 + \omega + 1 = 0 \;\Rightarrow\; \omega^2 = -\omega - 1\] \[\omega^3 = 1\]

Deljivost polinoma: Ako polinom \(p(x)\) deli polinom \(f(x)\), onda svaki koren od \(p(x)\) mora biti i koren od \(f(x)\).

📊 Vizuelni prikaz

Kubni koreni jedinice u kompleksnoj ravni i faktorizacija polinoma \(x^4 + x^2 + 1\).

✏️ Rešenje korak po korak
Korak 1: Identifikacija korena delioca

Polinom \(x^2 + x + 1\) ima korene \(\omega\) i \(\bar{\omega}\) — primitivne kubne korene jedinice.

Za njih važi:

\[\omega^3 = 1 \quad \text{i} \quad \omega^2 + \omega + 1 = 0\]
Korak 2: Uvrštavanje korena u polinom

Ako \(x^2 + x + 1\) deli \(x^4 + ax^2 + b\), onda \(\omega\) mora biti koren tog polinoma:

\[\omega^4 + a\omega^2 + b = 0\]
Korak 3: Pojednostavljivanje stepena

Koristimo \(\omega^3 = 1\):

\[\omega^4 = \omega^3 \cdot \omega = 1 \cdot \omega = \omega\]

Koristimo \(\omega^2 = -\omega - 1\) (iz jednačine \(\omega^2 + \omega + 1 = 0\)):

\[\omega + a(-\omega - 1) + b = 0\]
Proveri se: Koliko je \(\omega^4\) u terminima \(\omega\)?

Pošto je \(\omega^3 = 1\), imamo \(\omega^4 = \omega^3 \cdot \omega = \omega\). Tačno!

Korak 4: Razdvajanje na realni i imaginarni deo

Raspisujemo izraz:

\[\omega - a\omega - a + b = 0\]

Grupišemo po \(\omega\) i slobodnom članu:

\[\omega(1 - a) + (b - a) = 0\]

Pošto je \(\omega\) kompleksan broj (nije realan), oba koeficijenta moraju biti jednaka nuli:

\[1 - a = 0 \quad \Rightarrow \quad a = 1\] \[b - a = 0 \quad \Rightarrow \quad b = 1\]
Proveri se: Koliko je \(a + b\)?

\(a + b = 1 + 1 = 2\). Odgovor je \((B)\).

Korak 5: Verifikacija faktorizacijom

Za \(a = 1\) i \(b = 1\) polinom glasi \(x^4 + x^2 + 1\). Proveravamo:

\[x^4 + x^2 + 1 = x^4 + 2x^2 + 1 - x^2\]
\[= (x^2 + 1)^2 - x^2\]

Ovo je razlika kvadrata:

\[= (x^2 + 1 - x)(x^2 + 1 + x) = (x^2 - x + 1)(x^2 + x + 1)\]

Zaista, \(x^2 + x + 1\) je faktor!

💡 Ključni uvid

Polinom \(x^4 + x^2 + 1\) se može faktorizovati pomoću trika „dodaj i oduzmi \(x^2\)". Rezultat je elegantna faktorizacija na dva kvadratna trinoma koji su konjugovani po znaku linearnog člana:

\[x^4 + x^2 + 1 = (x^2 + x + 1)(x^2 - x + 1)\]
Konačan odgovor
\(a + b = 1 + 1 = 2\)
(A) \(-1\) (B) \(2\) (C) \(0\) (D) \(1\) (E) \(-2\)

Verifikacija: \((x^2 + x + 1)(x^2 - x + 1) = x^4 + x^2 + 1\), dakle \(a = 1\), \(b = 1\), \(a + b = 2\). Odgovor se poklapa.

⚠️ Česte greške
Pogrešna redukcija stepena: Neki učenici računaju \(\omega^4 = \omega^2\) umesto \(\omega^4 = \omega\). Zapamtite: \(\omega^3 = 1\), pa \(\omega^4 = \omega^{3+1} = \omega\).
Zaborav da je \(\omega\) kompleksan: Jednačina \(\omega(1-a) + (b-a) = 0\) daje dva uslova (koeficijent uz \(\omega\) i slobodni član), jer \(\omega\) nije realan broj.
🏆 Dodatni izazov

Ako je \(x^4 + ax^3 + bx^2 + ax + 1\) deljiv sa \(x^2 + x + 1\), koliko je \(a + b\)?

Savet

Ponovo uvrstite \(\omega\) i koristite \(\omega^3 = 1\). Recipročni polinom ima posebne osobine — razmislite o smeni \(t = x + \frac{1}{x}\).
