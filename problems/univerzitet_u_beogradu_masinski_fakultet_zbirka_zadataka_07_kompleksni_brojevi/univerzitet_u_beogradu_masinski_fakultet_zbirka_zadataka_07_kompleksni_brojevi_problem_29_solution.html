Realni deo zbira korena kubne jednacine

Kompleksni brojevi — koreni jednacine, Vijetove formule

📋 Tekst zadatka

Realni deo zbira svih resenja jednacine \( z^3 + 1 = 0 \) je:

(A)
\(1\)
(B)
\(0\)
(C)
\(-1\)
(D)
\(2\)
(E)
\(-2\)
🎯 Plan resavanja

Faktorizovacemo jednacinu \( z^3 + 1 = 0 \) pomocu formule za zbir kubova, naci sva tri korena, sabrati ih i odrediti realni deo tog zbira. Alternativno, koristicemo Vijetove formule za brzu proveru.

📚 Podsetnik iz teorije
Kljucne formule i teoreme (klikni da otvoris)

Zbir kubova:

\[ a^3 + b^3 = (a + b)(a^2 - ab + b^2) \]

Kvadratna formula:

\[ z = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \]

Vijetove formule za jednacinu \( z^3 + pz^2 + qz + r = 0 \):

\[ z_1 + z_2 + z_3 = -p \]

Zbir svih korena polinoma stepena \( n \) jednak je koeficijentu uz \( z^{n-1} \) sa promenjenim znakom, podeljen vodecim koeficijentom.

📊 Koreni na kompleksnoj ravni

Tri korena jednacine \( z^3 + 1 = 0 \) na jedinicnoj kruznici — ravnomerno rasporedjeni na 120°

🔢 Resenje korak po korak
1
Faktorizujemo jednacinu

Jednacina \( z^3 + 1 = 0 \) moze se zapisati kao \( z^3 + 1^3 = 0 \). Koristimo formulu za zbir kubova:

\[ z^3 + 1 = (z + 1)(z^2 - z + 1) = 0 \]

Proizvod dva faktora je nula, pa je bar jedan od njih jednak nuli.

2
Nalazimo prvi koren

Iz prvog faktora dobijamo:

\[ z + 1 = 0 \implies z_1 = -1 \]

Ovo je realan koren. Njegov realni deo je \( -1 \).

3
Resavamo kvadratnu jednacinu

Iz drugog faktora imamo \( z^2 - z + 1 = 0 \). Primenimo kvadratnu formulu:

\[ z = \frac{1 \pm \sqrt{1 - 4}}{2} = \frac{1 \pm \sqrt{-3}}{2} \]

Posto je diskriminanta negativna (\( \Delta = -3 \)), dobijamo dva konjugovano kompleksna korena:

\[ z_2 = \frac{1 + i\sqrt{3}}{2}, \qquad z_3 = \frac{1 - i\sqrt{3}}{2} \]
Proveri sebe: Koliki je realni deo svakog kompleksnog korena?
Oba kompleksna korena imaju realni deo \( \frac{1}{2} \). Imaginarni delovi su \( +\frac{\sqrt{3}}{2} \) i \( -\frac{\sqrt{3}}{2} \).
4
Sabiramo sve korene

Sada saberimo sva tri korena:

\[ z_1 + z_2 + z_3 = -1 + \frac{1 + i\sqrt{3}}{2} + \frac{1 - i\sqrt{3}}{2} \]

Grupisemo realne i imaginarne delove:

\[ = -1 + \frac{1}{2} + \frac{1}{2} + i\left(\frac{\sqrt{3}}{2} - \frac{\sqrt{3}}{2}\right) \]
\[ = -1 + 1 + 0 \cdot i = 0 \]
Proveri sebe: Zasto se imaginarni delovi poniste?
Kompleksni koreni \( z_2 \) i \( z_3 \) su konjugovano kompleksni parovi. Kada se saberu, imaginarni delovi se uvek poniste: \( \frac{\sqrt{3}}{2}i + \left(-\frac{\sqrt{3}}{2}i\right) = 0 \).
5
Odredjujemo realni deo zbira

Zbir svih korena je \( 0 \), sto je realan broj. Dakle:

\[ \text{Re}(z_1 + z_2 + z_3) = \text{Re}(0) = 0 \]
Proveri sebe: Moze li se ovo dobiti bez nalazenja korena?

Da! Pomocu Vijetovih formula: za jednacinu \( z^3 + 0 \cdot z^2 + 0 \cdot z + 1 = 0 \), zbir korena je \( -\frac{0}{1} = 0 \). Posto je zbir realan broj (0), realni deo je takodje 0.

💡 Kljucni uvid

Vijetove formule nam odmah daju odgovor bez ikakvog racunanja! Za jednacinu \( z^3 + 1 = 0 \), koeficijent uz \( z^2 \) je nula, pa je zbir svih korena jednak nuli. Koreni jednacine \( z^3 = -1 \) su tri tacke na jedinicnoj kruznici, ravnomerno rasporedjene na svakih 120°. Njihov zbir kao vektora u kompleksnoj ravni je uvek nula.

Konacan odgovor
\( \text{Re}(z_1 + z_2 + z_3) = 0 \)
(A) \(1\) (B) \(0\) (C) \(-1\) (D) \(2\) (E) \(-2\)

Verifikacija: Vijetove formule za \( z^3 + 0 \cdot z^2 + 0 \cdot z + 1 = 0 \) daju zbir korena \( = 0 \). Takodje, geometrijski, tri ravnomerno rasporedjena vektora na kruznici se poniste.

⚠️ Ceste greske
✗

Nalazenje samo realnog korena: Mnogi ucenici nadju samo \( z = -1 \) i zakljuce da je realni deo zbira \( -1 \). Jednacina treceg stepena ima tri korena (ukljucujuci kompleksne), i svi moraju biti ukljuceni u zbir.

✗

Mesanje "realnog dela zbira" i "zbira realnih delova": U ovom slucaju su isti (jer je Re linearan operator), ali vazno je razumeti razliku u opstem slucaju.

🏆 Dodatni izazov

Koliki je zbir kvadrata svih korena jednacine \( z^3 + 1 = 0 \)?

Pogledaj nagoveštaj

Koristite identitet \( z_1^2 + z_2^2 + z_3^2 = (z_1 + z_2 + z_3)^2 - 2(z_1 z_2 + z_1 z_3 + z_2 z_3) \). Iz Vijetovih formula, zbir korena je 0, a zbir proizvoda parova je 0 (koeficijent uz \( z \)). Dakle \( z_1^2 + z_2^2 + z_3^2 = 0 - 0 = 0 \).
