Kvadrat kosinusa dvostrukog ugla

Izražavanje trigonometrijskog izraza pomoću parametra \(a = \sin x - \cos x\)

📝 Tekst zadatka
Ako je \(\sin x - \cos x = a\), onda je \(\cos^2 2x\) jednako:
(A)
\(2a^2 - a^4\)
(B)
\(a^2 - a^4\)
(C)
\((1 - a^2)^2\)
(D)
\(4a - a^2\)
(E)
\(\frac{1}{4}(2 - a^2)^2\)
🎯 Plan rešavanja

Kvadriranjem uslova \(\sin x - \cos x = a\) izrazićemo \(\sin(2x)\) pomoću \(a\), a zatim ćemo iskoristiti Pitagorin identitet \(\cos^2(2x) = 1 - \sin^2(2x)\) da dobijemo konačan rezultat.

📚 Podsetnik iz teorije
Pitagorin trigonometrijski identitet
\[ \sin^2 x + \cos^2 x = 1 \]

Osnovni identitet koji važi za svako \(x\). Koristimo ga i za dvostruki ugao: \(\sin^2(2x) + \cos^2(2x) = 1\).

Sinus dvostrukog ugla
\[ \sin(2x) = 2\sin x \cos x \]

Ova formula se pojavljuje prirodno kada kvadriramo izraz \(\sin x - \cos x\).

Kvadrat razlike
\[ (p - q)^2 = p^2 - 2pq + q^2 \]

Algebarska formula koja nam omogućava da iz \((\sin x - \cos x)^2\) izvučemo \(\sin^2 x + \cos^2 x\) i \(2\sin x\cos x\).

🛠 Rešenje korak po korak
1 Kvadriramo obe strane uslova

Polazimo od datog uslova \(\sin x - \cos x = a\) i kvadriramo obe strane:

\[ (\sin x - \cos x)^2 = a^2 \]
2 Razvijamo kvadrat razlike

Primenjujemo formulu \((p-q)^2 = p^2 - 2pq + q^2\):

\[ \sin^2 x - 2\sin x \cos x + \cos^2 x = a^2 \]
3 Primenjujemo Pitagorin identitet

Znamo da je \(\sin^2 x + \cos^2 x = 1\), pa zamenjujemo:

\[ 1 - 2\sin x \cos x = a^2 \]
Proveri sebe: Koliko je \(2\sin x\cos x\) pomoću \(a\)?

Iz jednačine \(1 - 2\sin x\cos x = a^2\) sledi \(2\sin x\cos x = 1 - a^2\). Ovo je upravo \(\sin(2x)\)!

4 Prepoznajemo sinus dvostrukog ugla

Pošto je \(\sin(2x) = 2\sin x\cos x\), iz prethodnog koraka dobijamo:

\[ \sin(2x) = 1 - a^2 \]
5 Nalazimo \(\cos^2(2x)\) pomoću identiteta

Koristimo fundamentalni identitet \(\cos^2(2x) = 1 - \sin^2(2x)\):

\[ \cos^2(2x) = 1 - \sin^2(2x) = 1 - (1 - a^2)^2 \]
6 Razvijamo i pojednostavljujemo

Razvijamo \((1 - a^2)^2\):

\[ (1 - a^2)^2 = 1 - 2a^2 + a^4 \]

Zamenjujemo nazad:

\[ \cos^2(2x) = 1 - (1 - 2a^2 + a^4) \]

Skidamo zagradu:

\[ \cos^2(2x) = 1 - 1 + 2a^2 - a^4 = 2a^2 - a^4 \]
Proveri sebe: Ako je \(a = 1\), koliko je \(\cos^2(2x)\)?

Za \(a = 1\): \(\sin x - \cos x = 1\). Jedno rešenje je \(x = \frac{\pi}{2}\). Tada \(\cos^2(\pi) = (-1)^2 = 1\). Formula: \(2 \cdot 1 - 1 = 1\). Tačno!

7 Faktorišemo rezultat

Konačan izraz možemo i faktorisati:

\[ \cos^2(2x) = 2a^2 - a^4 = a^2(2 - a^2) \]
Proveri sebe: Zašto (C) \((1-a^2)^2\) nije tačan odgovor?

Odgovor (C) \((1-a^2)^2\) je zapravo \(\sin^2(2x)\), ne \(\cos^2(2x)\)! Pošto je \(\sin(2x) = 1 - a^2\), kvadriranjem dobijamo \(\sin^2(2x) = (1-a^2)^2\). Tražen je \(\cos^2(2x) = 1 - \sin^2(2x)\), što je \(1 - (1-a^2)^2 = 2a^2 - a^4\).

Ključni uvid

Tehnika „kvadriraj uslov" je izuzetno moćna u trigonometriji. Kada se pojavi izraz oblika \(\sin x \pm \cos x = a\), kvadriranjem dobijamo \(\sin(2x)\) jer se pojavljuju \(\sin^2 x + \cos^2 x = 1\) i \(2\sin x \cos x = \sin(2x)\) na prirodan način.

Konačni odgovor
\(2a^2 - a^4\)
(A) \(2a^2 - a^4\) (B) \(a^2 - a^4\) (C) \((1-a^2)^2\) (D) \(4a - a^2\) (E) \(\frac{1}{4}(2-a^2)^2\)

Verifikacija: Za \(x = \frac{\pi}{4}\): \(a = \sin\frac{\pi}{4} - \cos\frac{\pi}{4} = 0\), pa \(\cos^2\frac{\pi}{2} = 0\). Formula: \(2 \cdot 0 - 0 = 0\) ✔. Za \(x = \frac{\pi}{2}\): \(a = 1\), pa \(\cos^2\pi = 1\). Formula: \(2 - 1 = 1\) ✔

⚠️ Česte greške
Zamena \(\sin^2(2x)\) i \(\cos^2(2x)\)

Odgovor (C) \((1-a^2)^2\) je \(\sin^2(2x)\), ne \(\cos^2(2x)\). Obratite pažnju na to šta se tačno traži. Veza je: \(\cos^2(2x) = 1 - \sin^2(2x)\).

Pogrešno razvijanje \((1-a^2)^2\)

Česta računska greška je pisanje \((1-a^2)^2 = 1 - a^4\) umesto tačnog \(1 - 2a^2 + a^4\). Uvek primenite formulu kvadrata razlike pažljivo.

🏆 Dodatni izazov

Ako je \(\sin x + \cos x = b\), izrazite \(\cos^2(2x)\) pomoću \(b\).

Nagoveštaj

Kvadrirajte uslov: \((\sin x + \cos x)^2 = b^2\), pa \(1 + \sin(2x) = b^2\), dakle \(\sin(2x) = b^2 - 1\). Onda \(\cos^2(2x) = 1 - (b^2 - 1)^2 = 1 - b^4 + 2b^2 - 1 = 2b^2 - b^4\). Isti oblik formule!
