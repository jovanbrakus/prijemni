Pojednostavljivanje razlomka sa kubovima sinusa i kosinusa

Trigonometrija -- zbir kubova i identiteti sa dvostrukim uglom

📋 Tekst zadatka
Izraz \(\displaystyle\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x}\) je identički jednak:
(A)
\(\dfrac{\cos x - \sin x}{2}\)
(B)
\(1\)
(C)
\(\dfrac{\cos x}{2}\)
(D)
\(\dfrac{1 + \cos^2 x}{4}\)
(E)
\(\dfrac{\sin x + \cos x}{2}\)
🧭 Plan rešavanja

Rastavićemo brojilac koristeći formulu za zbir kubova \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\), a imenilac transformisati koristeći \(\sin 2x = 2\sin x\cos x\). Zatim ćemo skratiti zajednički faktor.

📐 Ključne formule i teoreme
  • Zbir kubova: \(a^3 + b^3 = (a + b)(a^2 - ab + b^2)\)
  • Pitagorin identitet: \(\sin^2 x + \cos^2 x = 1\)
  • Sinus dvostrukog ugla: \(\sin 2x = 2\sin x \cos x\)
  • Posledica: Kada u zbiru kubova stavimo \(a = \sin x\), \(b = \cos x\), dobijamo \(a^2 - ab + b^2 = 1 - \sin x\cos x\)

📝 Rešenje korak po korak
1

Rastavimo brojilac koristeći formulu za zbir kubova

Primenjujemo formulu \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\) gde je \(a = \sin x\) i \(b = \cos x\):

\[\sin^3 x + \cos^3 x = (\sin x + \cos x)(\sin^2 x - \sin x \cos x + \cos^2 x)\]
2

Pojednostavimo drugi faktor u brojiocu

Znamo da je \(\sin^2 x + \cos^2 x = 1\), pa drugi faktor postaje:

\[\sin^2 x - \sin x \cos x + \cos^2 x = 1 - \sin x \cos x\]

Dakle, brojilac glasi: \((\sin x + \cos x)(1 - \sin x \cos x)\).

🔍 Proveri se: Za \(x = 0\), koliko je \(\sin^3 0 + \cos^3 0\)?

\(\sin^3 0 + \cos^3 0 = 0 + 1 = 1\). Proverimo rastav: \((\sin 0 + \cos 0)(1 - \sin 0 \cdot \cos 0) = (0+1)(1-0) = 1\). Tačno!

3

Transformišemo imenilac

Koristimo identitet \(\sin 2x = 2\sin x \cos x\):

\[2 - \sin 2x = 2 - 2\sin x \cos x = 2(1 - \sin x \cos x)\]
4

Sastavimo razlomak sa rastavama

Unosimo dobijene oblike u polazni razlomak:

\[\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \frac{(\sin x + \cos x)(1 - \sin x \cos x)}{2(1 - \sin x \cos x)}\]
5

Skratimo zajednički faktor

Faktor \((1 - \sin x \cos x)\) se javlja i u brojiocu i u imeniocu.

Primetimo da je \(1 - \sin x \cos x = 1 - \frac{\sin 2x}{2} \geq 1 - \frac{1}{2} = \frac{1}{2} > 0\) za svako \(x\), pa nikada nije nula. Skraćivanje je uvek dozvoljeno.

\[\frac{(\sin x + \cos x)\cancel{(1 - \sin x \cos x)}}{2\cancel{(1 - \sin x \cos x)}} = \frac{\sin x + \cos x}{2}\]
🔍 Proveri se: Za \(x = \frac{\pi}{4}\), koliko iznosi rezultat?

Polazni izraz: \(\frac{(\frac{\sqrt{2}}{2})^3 + (\frac{\sqrt{2}}{2})^3}{2 - \sin 90°} = \frac{2 \cdot \frac{\sqrt{2}}{4}}{2 - 1} = \frac{\sqrt{2}}{2}\).

Rezultat formule: \(\frac{\sin 45° + \cos 45°}{2} = \frac{\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}}{2} = \frac{\sqrt{2}}{2}\). Poklapa se!

🔍 Proveri se: Zašto smemo da skratimo \((1 - \sin x \cos x)\)?

Zato što važi \(-1 \leq \sin x \cos x \leq \frac{1}{2}\), pa je \(1 - \sin x \cos x \geq \frac{1}{2} > 0\). Ovaj faktor nikada nije nula, tako da je skraćivanje dozvoljeno za svako \(x\).

Ključni uvid

Zbir kubova sinusa i kosinusa se savršeno faktoriše, a imenilac se pretvara u isti faktor pomnožen sa 2. Sve se lepo skrati!

\[\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \frac{\sin x + \cos x}{2}\]

Konačan odgovor

\(\displaystyle\frac{\sin^3 x + \cos^3 x}{2 - \sin 2x} = \frac{\sin x + \cos x}{2}\)
(A) \(\frac{\cos x - \sin x}{2}\) (B) \(1\) (C) \(\frac{\cos x}{2}\) (D) \(\frac{1 + \cos^2 x}{4}\) (E) \(\frac{\sin x + \cos x}{2}\)
Verifikacija: Za \(x = 0\): brojilac \(= 0 + 1 = 1\), imenilac \(= 2 - 0 = 2\), rezultat \(= \frac{1}{2}\). Formula daje \(\frac{0 + 1}{2} = \frac{1}{2}\). Za \(x = \frac{\pi}{2}\): brojilac \(= 1 + 0 = 1\), imenilac \(= 2 - 0 = 2\), rezultat \(= \frac{1}{2}\). Formula daje \(\frac{1 + 0}{2} = \frac{1}{2}\). Sve se poklapa.
⚠️ Najčešće greške
  • Pogrešna faktorizacija kubova: formula je \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\), ne \((a+b)(a^2 + ab + b^2)\). Minus ispred \(ab\) je ključan!
  • Zaboraviti da se \(\sin 2x\) zameni sa \(2\sin x \cos x\) -- bez ove zamene, imenilac se ne može dovesti u oblik koji se skraćuje sa brojiocem.
🏆 Dodatni izazov

Pojednostavite sličan izraz \(\displaystyle\frac{\sin^3 x - \cos^3 x}{2 + \sin 2x}\).

Prikaži nagoveštaj

Koristite formulu za razliku kubova: \(a^3 - b^3 = (a - b)(a^2 + ab + b^2)\). Imenilac se transformiše u \(2(1 + \sin x \cos x)\). Rezultat je \(\frac{\sin x - \cos x}{2}\).
