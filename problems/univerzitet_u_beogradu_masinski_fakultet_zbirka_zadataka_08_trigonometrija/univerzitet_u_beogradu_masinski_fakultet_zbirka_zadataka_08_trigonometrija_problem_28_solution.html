Zbir šestih stepena sinusa i kosinusa

Transformacija izraza \(\sin^6 x + \cos^6 x\) pomoću algebarskih i trigonometrijskih identiteta

📝 Tekst zadatka
Koji izraz je identički jednak izrazu:
\[ \sin^6 x + \cos^6 x \]
(A)
\(\dfrac{3+\cos 4x}{4}\)
(B)
\(\dfrac{5+\cos 4x}{6}\)
(C)
\(\dfrac{5+\cos 6x}{6}\)
(D)
\(\dfrac{5-3\cos 6x}{8}\)
(E)
\(\dfrac{5+3\cos 4x}{8}\)
🎯 Plan rešavanja

Primenićemo formulu za zbir kubova \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\) na \((\sin^2 x)^3 + (\cos^2 x)^3\), iskoristiti Pitagorin identitet \(\sin^2 x + \cos^2 x = 1\), a zatim pomoću formula za dvostruki i četvorostruki ugao svesti izraz na oblik sa \(\cos 4x\).

📚 Podsetnik iz teorije
Zbir kubova
\[ a^3 + b^3 = (a + b)(a^2 - ab + b^2) \]

Algebarski identitet koji ćemo primeniti na \(a = \sin^2 x\) i \(b = \cos^2 x\).

Pitagorin trigonometrijski identitet
\[ \sin^2 x + \cos^2 x = 1 \]

Osnovni identitet koji značajno pojednostavljuje račun jer je \(a + b = 1\).

Formula za dvostruki ugao
\[ \sin 2x = 2\sin x\cos x \]

Odavde sledi \(\sin^2 x\cos^2 x = \frac{\sin^2 2x}{4}\).

Formula za snižavanje stepena
\[ \sin^2\theta = \frac{1 - \cos 2\theta}{2} \]

Koristimo ovo za \(\sin^2 2x = \frac{1 - \cos 4x}{2}\).

🎨 Vizuelni prikaz

Koristite klizač da menjate ugao \(x\) i vidite kako se menjaju obe strane identiteta

🛠 Rešenje korak po korak
1 Zapišemo izraz kao zbir kubova

Primetimo da je \(\sin^6 x = (\sin^2 x)^3\) i \(\cos^6 x = (\cos^2 x)^3\). Uvedemo smenu:

\[ a = \sin^2 x, \quad b = \cos^2 x \]

pa je naš izraz:

\[ \sin^6 x + \cos^6 x = a^3 + b^3 \]

Pritom znamo da važi \(a + b = \sin^2 x + \cos^2 x = 1\).

2 Primenimo formulu za zbir kubova

Koristimo identitet \(a^3 + b^3 = (a + b)(a^2 - ab + b^2)\):

\[ a^3 + b^3 = (a + b)(a^2 - ab + b^2) = 1 \cdot (a^2 - ab + b^2) \]

Dakle:

\[ \sin^6 x + \cos^6 x = a^2 + b^2 - ab \]
3 Izrazimo \(a^2 + b^2\) preko \(ab\)

Iz \(a + b = 1\), kvadriranjem:

\[ (a + b)^2 = a^2 + 2ab + b^2 = 1 \]

Odatle:

\[ a^2 + b^2 = 1 - 2ab \]
Proveri sebe: Koliko je \(a^2 + b^2 - ab\) ako je \(a + b = 1\)?

Zamenom \(a^2 + b^2 = 1 - 2ab\) dobijamo: \(1 - 2ab - ab = 1 - 3ab\).

4 Svedemo na \(1 - 3ab\)

Uvrštavamo rezultat iz prethodnog koraka:

\[ \sin^6 x + \cos^6 x = (1 - 2ab) - ab = 1 - 3ab \]

Sada treba da izrazimo proizvod \(ab = \sin^2 x \cdot \cos^2 x\).

5 Izračunajmo \(\sin^2 x \cdot \cos^2 x\)

Koristimo formulu za dvostruki ugao \(\sin 2x = 2\sin x\cos x\), odakle je:

\[ \sin x\cos x = \frac{\sin 2x}{2} \]

Kvadriranjem obe strane:

\[ \sin^2 x\cos^2 x = \frac{\sin^2 2x}{4} \]
6 Uvrstimo u izraz

Uvrštavamo \(ab = \frac{\sin^2 2x}{4}\) u formulu iz koraka 4:

\[ \sin^6 x + \cos^6 x = 1 - 3 \cdot \frac{\sin^2 2x}{4} = 1 - \frac{3\sin^2 2x}{4} \]
Proveri sebe: Koliko je \(\sin^2 2x\) pomoću \(\cos 4x\)?

Po formuli za snižavanje stepena: \(\sin^2 2x = \frac{1 - \cos 4x}{2}\). Ovo je ključni korak za dobijanje konačnog oblika!

7 Primenimo formulu za snižavanje stepena

Koristimo \(\sin^2 2x = \frac{1 - \cos 4x}{2}\):

\[ \sin^6 x + \cos^6 x = 1 - \frac{3}{4} \cdot \frac{1 - \cos 4x}{2} \]

Raspisujemo:

\[ = 1 - \frac{3(1 - \cos 4x)}{8} = 1 - \frac{3}{8} + \frac{3\cos 4x}{8} \]
8 Sređujemo konačan rezultat

Svodimo na zajednički imenilac:

\[ = \frac{8}{8} - \frac{3}{8} + \frac{3\cos 4x}{8} = \frac{5 + 3\cos 4x}{8} \]
Proveri sebe: Proverimo za \(x = 0\)

Leva strana: \(\sin^6 0 + \cos^6 0 = 0 + 1 = 1\).

Desna strana: \(\frac{5 + 3\cos 0}{8} = \frac{5 + 3}{8} = \frac{8}{8} = 1\). Poklapaju se!

Ključni uvid

Svaki put kada vidite izraz oblika \(\sin^n x + \cos^n x\) za parno \(n\), razmislite o algebarskim identitetima (zbir kvadrata, kubova...) u kombinaciji sa \(\sin^2 x + \cos^2 x = 1\). Ova strategija drastično pojednostavljuje račun jer eliminišete jednu promenljivu odmah na početku.

\[ \sin^6 x + \cos^6 x = 1 - 3\sin^2 x\cos^2 x = \frac{5 + 3\cos 4x}{8} \]
Konačni odgovor
\(\dfrac{5 + 3\cos 4x}{8}\)
(A) \(\frac{3+\cos 4x}{4}\) (B) \(\frac{5+\cos 4x}{6}\) (C) \(\frac{5+\cos 6x}{6}\) (D) \(\frac{5-3\cos 6x}{8}\) (E) \(\frac{5+3\cos 4x}{8}\)

Verifikacija: Za \(x = \frac{\pi}{4}\): leva strana \(= 2 \cdot \left(\frac{\sqrt{2}}{2}\right)^6 = 2 \cdot \frac{1}{8} = \frac{1}{4}\). Desna strana: \(\frac{5 + 3\cos\pi}{8} = \frac{5 - 3}{8} = \frac{1}{4}\) ✔

⚠️ Česte greške
Mešanje \(\cos 4x\) i \(\cos 6x\)

Izraz \(\sin^6 x + \cos^6 x\) sadrži šeste stepene, ali se u rezultatu pojavljuje \(\cos 4x\), a ne \(\cos 6x\). To je zato što se snižavanje stepena radi u dva koraka: prvo do \(\sin^2 2x\), pa do \(\cos 4x\). Opcije (C) i (D) su mamci sa pogrešnim uglom.

Zaboravljanje faktora 3

Lako je izgubiti koeficijent 3 ispred \(\sin^2 x\cos^2 x\) koji dolazi iz formule za zbir kubova. Opcija (B) ima tačan imenilac 6 ali pogrešan brojilac upravo zbog ove greške.
