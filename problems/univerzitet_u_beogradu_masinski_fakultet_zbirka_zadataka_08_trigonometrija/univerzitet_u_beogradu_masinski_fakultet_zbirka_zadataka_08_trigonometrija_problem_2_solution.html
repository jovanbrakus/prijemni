Kotangens ugla za dati sinus

Trigonometrija -- određivanje kotangensa pomoću fundamentalnog trigonometrijskog identiteta

Zadatak

Ako je \(\sin x = \dfrac{1}{3}\) i \(\dfrac{5\pi}{2} < x < 3\pi\), onda je \(\operatorname{ctg} x\) jednako:

(A)
\(-3\)
(B)
\(-\dfrac{1}{4}\sqrt{2}\)
(C)
\(\dfrac{1}{4}\sqrt{2}\)
(D)
\(-2\sqrt{2}\)
(E)
\(2\sqrt{2}\)

Plan rešavanja

Prvo ćemo odrediti u kom se kvadrantu nalazi ugao \(x\), zatim pomoću identiteta \(\sin^2 x + \cos^2 x = 1\) naći \(\cos x\), i konačno izračunati \(\operatorname{ctg} x = \dfrac{\cos x}{\sin x}\).

Podsetnik iz teorije

Fundamentalni identitet i kotangens

Pitagorin trigonometrijski identitet:

\[\sin^2 x + \cos^2 x = 1\]

Kotangens preko sinusa i kosinusa:

\[\operatorname{ctg} x = \frac{\cos x}{\sin x}\]

Znakovi u kvadrantima:

  • I kvadrant: svi pozitivni
  • II kvadrant: sin > 0, cos < 0, ctg < 0
  • III kvadrant: sin < 0, cos < 0, ctg > 0
  • IV kvadrant: sin < 0, cos > 0, ctg < 0

Položaj ugla na jediničnom krugu

Ugao \(x\) u II kvadrantu (nakon oduzimanja punog obrta)

Rešenje korak po korak

1

Određivanje kvadranta

Dato je da je \(\dfrac{5\pi}{2} < x < 3\pi\).

Zapišimo ove granice preglednije:

\[\frac{5\pi}{2} = 2\pi + \frac{\pi}{2}, \quad 3\pi = 2\pi + \pi\]

Dakle, nakon oduzimanja jednog punog obrta \((2\pi)\), efektivni ugao je:

\[\frac{\pi}{2} < x - 2\pi < \pi\]

To znači da se ugao \(x\) efektivno nalazi u II kvadrantu.

Proveri se: Koji je znak kosinusa u II kvadrantu?

U II kvadrantu kosinus je negativan (\(\cos x < 0\)).

2

Nalaženje \(\cos^2 x\)

Koristimo fundamentalni trigonometrijski identitet:

\[\sin^2 x + \cos^2 x = 1\]

Zamenjujemo \(\sin x = \dfrac{1}{3}\):

\[\left(\frac{1}{3}\right)^2 + \cos^2 x = 1\]
\[\frac{1}{9} + \cos^2 x = 1\]
\[\cos^2 x = 1 - \frac{1}{9} = \frac{8}{9}\]
3

Određivanje \(\cos x\)

Iz \(\cos^2 x = \dfrac{8}{9}\) dobijamo:

\[\cos x = \pm\sqrt{\frac{8}{9}} = \pm\frac{2\sqrt{2}}{3}\]

Pošto je ugao \(x\) u II kvadrantu, kosinus je negativan:

\[\cos x = -\frac{2\sqrt{2}}{3}\]
Proveri se: Koliko je \(\sin^2 x + \cos^2 x\) sa ovim vrednostima?

\(\left(\dfrac{1}{3}\right)^2 + \left(-\dfrac{2\sqrt{2}}{3}\right)^2 = \dfrac{1}{9} + \dfrac{8}{9} = \dfrac{9}{9} = 1\) -- tačno!

4

Računanje kotangensa

Primenjujemo formulu za kotangens:

\[\operatorname{ctg} x = \frac{\cos x}{\sin x}\]

Zamenjujemo poznate vrednosti:

\[\operatorname{ctg} x = \frac{-\dfrac{2\sqrt{2}}{3}}{\dfrac{1}{3}}\]
5

Pojednostavljivanje

Deljenje razlomaka -- množimo prvim, delimo drugim:

\[\operatorname{ctg} x = -\frac{2\sqrt{2}}{3} \cdot \frac{3}{1} = -2\sqrt{2}\]

Trojke u brojiocu i imeniocu se skrate.

Ključni uvid

Ključ zadatka je pravilno odrediti kvadrant ugla \(x\). Interval \(\left(\dfrac{5\pi}{2},\, 3\pi\right)\) izgleda neobično, ali nakon oduzimanja punog obrta \(2\pi\) svodi se na poznati interval \(\left(\dfrac{\pi}{2},\, \pi\right)\) -- drugi kvadrant. Odatle sledi negativan predznak kosinusa, a samim tim i kotangensa.

\[\operatorname{ctg} x = -2\sqrt{2}\]

Konačan odgovor

\(\operatorname{ctg} x = -2\sqrt{2}\)
(A) \(-3\) (B) \(-\dfrac{1}{4}\sqrt{2}\) (C) \(\dfrac{1}{4}\sqrt{2}\) (D) \(-2\sqrt{2}\) (E) \(2\sqrt{2}\)
Verifikacija: Proverimo numerički: \(\sin x = \frac{1}{3} \approx 0{,}333\), \(\cos x = -\frac{2\sqrt{2}}{3} \approx -0{,}943\). Tada je \(\operatorname{ctg} x = \frac{-0{,}943}{0{,}333} \approx -2{,}828 = -2\sqrt{2}\). Rezultat je negativan što se slaže sa II kvadrantom. Sve je konzistentno.

Najčešće greške

Pogrešan kvadrant: Interval \(\left(\frac{5\pi}{2}, 3\pi\right)\) može da zbuni. Mnogi studenti pogrešno zaključe da je ugao u III kvadrantu. Uvek prvo oduzmite \(2\pi\) da biste sveli ugao na interval \((0, 2\pi)\).
Pogrešan znak kosinusa: Iz \(\cos^2 x = \frac{8}{9}\) slede dva rešenja (\(\pm\)). Izbor znaka zavisi od kvadranta -- u II kvadrantu kosinus je negativan.

Dodatni izazov

Ako je \(\cos x = -\dfrac{2}{5}\) i \(\pi < x < \dfrac{3\pi}{2}\), koliko je \(\operatorname{tg} x\)?

Savet

Isti postupak: nađi \(\sin x\) iz \(\sin^2 x + \cos^2 x = 1\), odredi znak iz kvadranta (III kvadrant -- sin < 0), pa izračunaj \(\operatorname{tg} x = \frac{\sin x}{\cos x}\).
