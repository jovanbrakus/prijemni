Maksimum trigonometrijske funkcije

Trigonometrija — transformacija oblika \(a\sin x + b\cos x\) u amplitudno-fazni oblik

📝 Postavka zadatka

Maksimum funkcije \(f(x) = \sin x + \sqrt{3}\cos x\) je jednak:

(A)
\(\sqrt{3}\)
(B)
\(\frac{1}{2}(\sqrt{2}+\sqrt{6})\)
(C)
\(1\)
(D)
\(1+\sqrt{3}\)
(E)
\(2\)

🗺️ Plan rešavanja

Transformisaćemo izraz \(\sin x + \sqrt{3}\cos x\) u oblik \(R\sin(x + \varphi)\) pomoću formule za amplitudu, a zatim očitati maksimalnu vrednost.

📐 Ključne formule i teoreme
  • Amplitudno-fazna transformacija: \(a\sin x + b\cos x = R\sin(x + \varphi)\) gde je \(R = \sqrt{a^2 + b^2}\), \(\cos\varphi = \frac{a}{R}\), \(\sin\varphi = \frac{b}{R}\).
  • Adiciona formula: \(\sin(x+\varphi) = \sin x\cos\varphi + \cos x \sin\varphi\)
  • Maksimum sinusa: \(\max\sin\alpha = 1\), pa je \(\max R\sin(x+\varphi) = R\).
  • Poznati uglovi: \(\cos\frac{\pi}{3} = \frac{1}{2}\), \(\sin\frac{\pi}{3} = \frac{\sqrt{3}}{2}\)

📚 Rešenje korak po korak

1

Identifikujemo koeficijente

Funkcija je oblika \(a\sin x + b\cos x\), gde je:

\[a = 1, \quad b = \sqrt{3}\]
2

Računamo amplitudu \(R\)

Amplituda se dobija iz:

\[R = \sqrt{a^2 + b^2} = \sqrt{1^2 + (\sqrt{3})^2} = \sqrt{1 + 3} = \sqrt{4} = 2\]
🔍 Proveri sebe: Kolika je amplituda \(R\)?
\(R = 2\). Ovo je ujedno i kandidat za maksimum!
3

Određujemo fazni pomeraj \(\varphi\)

Iz uslova \(\cos\varphi = \frac{a}{R}\) i \(\sin\varphi = \frac{b}{R}\):

\[\cos\varphi = \frac{1}{2}, \quad \sin\varphi = \frac{\sqrt{3}}{2}\]

Oba uslova daju:

\[\varphi = \frac{\pi}{3}\]
4

Zapisujemo funkciju u novom obliku

Sada možemo da zapišemo:

\[f(x) = 2\sin\!\left(x + \frac{\pi}{3}\right)\]
5

Provera adicione formule

Raspisujemo da potvrdimo:

\[2\sin\!\left(x + \frac{\pi}{3}\right) = 2\left(\sin x \cos\frac{\pi}{3} + \cos x \sin\frac{\pi}{3}\right)\]
\[= 2\left(\sin x \cdot \frac{1}{2} + \cos x \cdot \frac{\sqrt{3}}{2}\right) = \sin x + \sqrt{3}\cos x \;\checkmark\]
🔍 Proveri sebe: Za koje \(x\) funkcija \(f(x) = 2\sin(x + \frac{\pi}{3})\) dostiže maksimum?
Kada je \(\sin(x + \frac{\pi}{3}) = 1\), tj. kada je \(x + \frac{\pi}{3} = \frac{\pi}{2}\), što daje \(x = \frac{\pi}{6}\).
6

Određujemo maksimum

Pošto je maksimalna vrednost sinusne funkcije jednaka 1:

\[\max f(x) = 2 \cdot 1 = 2\]
7

Verifikacija uvrštavanjem

Proveravamo za \(x = \frac{\pi}{6}\):

\[f\!\left(\frac{\pi}{6}\right) = \sin\frac{\pi}{6} + \sqrt{3}\cos\frac{\pi}{6} = \frac{1}{2} + \sqrt{3} \cdot \frac{\sqrt{3}}{2} = \frac{1}{2} + \frac{3}{2} = 2 \;\checkmark\]
🔍 Proveri sebe: Zašto odgovor (D) \(1 + \sqrt{3} \approx 2{,}732\) ne može biti tačan?
Jer je amplituda \(R = 2\), a funkcija oblika \(R\sin(\cdots)\) nikada ne može preći vrednost \(R\). Pošto je \(1 + \sqrt{3} \approx 2{,}732 > 2\), ta vrednost je nedostižna.

💡 Ključni uvid

Svaka linearna kombinacija \(a\sin x + b\cos x\) je zapravo sinusoida sa amplitudom \(\sqrt{a^2 + b^2}\). Maksimum je uvek upravo ta amplituda!

\[\max(a\sin x + b\cos x) = \sqrt{a^2 + b^2}\]

🏆 Konačan odgovor

\[\max f(x) = \max(\sin x + \sqrt{3}\cos x) = 2\]
(A) \(\sqrt{3}\) (B) \(\frac{1}{2}(\sqrt{2}+\sqrt{6})\) (C) \(1\) (D) \(1+\sqrt{3}\) (E) \(2\)
Verifikacija: Za \(x = \frac{\pi}{6}\) direktno računanje daje \(f(\frac{\pi}{6}) = \frac{1}{2} + \frac{3}{2} = 2\), a amplituda \(R = \sqrt{1+3} = 2\) potvrđuje da je ovo zaista globalni maksimum.

⚠️ Česte greške

  • Pomisliti da je maksimum jednak zbiru apsolutnih vrednosti koeficijenata (\(1 + \sqrt{3}\)). Tačna formula je \(\sqrt{a^2 + b^2}\), ne \(|a| + |b|\).
  • Zaboraviti da \((\sqrt{3})^2 = 3\), a ne \(\sqrt{3}\), pri računanju \(R^2 = a^2 + b^2\).

🚀 Dodatni izazov

Nagoveštaj
Primenite istu formulu: \(R = \sqrt{3^2 + (-4)^2} = \sqrt{9 + 16} = 5\). Maksimum je 5, minimum je \(-5\). Funkcija se može zapisati kao \(5\sin(x + \varphi)\) gde je \(\varphi = \arctan\!\left(\frac{-4}{3}\right)\).
