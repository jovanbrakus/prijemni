Najveca vrednost inverznog kvadrata zbira tangensa i kotangensa

Trigonometrija -- ekstremne vrednosti trigonometrijskih funkcija, formula dvostrukog ugla

Zadatak

Najveca vrednost funkcije

\[f(x) = (\tg x + \ctg x)^{-2}\]

je:

(A)
\(\dfrac{1}{4}\)
(B)
\(\dfrac{1}{2}\)
(C)
\(1\)
(D)
\(2\)
(E)
\(4\)

Plan resavanja

Pojednostavicemo izraz \(\tg x + \ctg x\) svodenjem na sinus i kosinus, zatim primeniti formulu dvostrukog ugla da bismo dobili kompaktan oblik za \(f(x)\) i nasli njenu najvecu vrednost.

Kljucne formule i teoreme
  • Osnovna veza: \(\tg x = \dfrac{\sin x}{\cos x}\), \(\ctg x = \dfrac{\cos x}{\sin x}\)
  • Pitagorin trigonometrijski identitet: \(\sin^2 x + \cos^2 x = 1\)
  • Formula dvostrukog ugla: \(\sin(2x) = 2\sin x \cos x\)
  • Opseg: \(\sin^2(2x) \in [0, 1]\), maksimum kada \(\sin(2x) = \pm 1\)

Vizuelni prikaz

Grafik funkcije \(f(x) = (\tg x + \ctg x)^{-2}\) sa oznacenim maksimumom

Plava kriva: \(f(x)\). Zelena linija: maksimum \(y = \frac{1}{4}\)

Resenje korak po korak

1

Zapisujemo tangens i kotangens preko sinusa i kosinusa

Pocinjemo od zbira \(\tg x + \ctg x\):

\[\tg x + \ctg x = \frac{\sin x}{\cos x} + \frac{\cos x}{\sin x}\]
2

Svodimo na zajednicki imenilac

Zajednicki imenilac je \(\sin x \cdot \cos x\):

\[\tg x + \ctg x = \frac{\sin^2 x + \cos^2 x}{\sin x \cdot \cos x}\]

Pitagorinim identitetom, brojilac je 1:

\[\tg x + \ctg x = \frac{1}{\sin x \cdot \cos x}\]
Proveri se: Koliko je \(\tg 45° + \ctg 45°\)?

\(\tg 45° + \ctg 45° = 1 + 1 = 2\). Po nasoj formuli: \(\dfrac{1}{\sin 45° \cdot \cos 45°} = \dfrac{1}{\frac{\sqrt{2}}{2} \cdot \frac{\sqrt{2}}{2}} = \dfrac{1}{\frac{1}{2}} = 2\). ✓

3

Primena formule dvostrukog ugla

Koristimo \(\sin(2x) = 2\sin x \cos x\), odakle \(\sin x \cos x = \dfrac{\sin(2x)}{2}\):

\[\tg x + \ctg x = \frac{1}{\frac{\sin(2x)}{2}} = \frac{2}{\sin(2x)}\]
4

Racunamo \(f(x)\)

Stepenujemo na \(-2\):

\[f(x) = \left(\frac{2}{\sin(2x)}\right)^{-2} = \left(\frac{\sin(2x)}{2}\right)^{2} = \frac{\sin^2(2x)}{4}\]
5

Nalazimo maksimum

Funkcija \(\sin^2(2x)\) ima vrednosti u intervalu \([0, 1]\).

Maksimum se dostize kada \(\sin(2x) = \pm 1\), tj. \(2x = \dfrac{\pi}{2} + k\pi\), odakle \(x = \dfrac{\pi}{4} + \dfrac{k\pi}{2}\).

\[\max f(x) = \frac{1}{4}\]
Proveri se: Koliko je \(f(\pi/4)\)?

\(f(\pi/4) = (\tg(\pi/4) + \ctg(\pi/4))^{-2} = (1 + 1)^{-2} = 2^{-2} = \frac{1}{4}\). ✓

Kljucni uvid

Zbir tangensa i kotangensa se elegantno svodi na reciprocnu vrednost proizvoda sinusa i kosinusa:

\[\tg x + \ctg x = \frac{2}{\sin(2x)} \quad \Rightarrow \quad f(x) = \frac{\sin^2(2x)}{4}\]

Konacni odgovor

\(\displaystyle \max f(x) = \frac{1}{4}\)
(A) \(\dfrac{1}{4}\) (B) \(\dfrac{1}{2}\) (C) \(1\) (D) \(2\) (E) \(4\)
Verifikacija: Za \(x = \pi/4\): \(\tg(\pi/4) + \ctg(\pi/4) = 1 + 1 = 2\), pa \(f(\pi/4) = 2^{-2} = 1/4\). Za bilo koje drugo \(x\), zbir \(\tg x + \ctg x\) je po apsolutnoj vrednosti veci od 2 (jer AM-GM daje \(|\tg x + \ctg x| \geq 2\) za isti znak), pa \(f(x) \leq 1/4\). ✓

Ceste greske

Greska sa eksponentom: Obratite paznju da je eksponent \(-2\), sto znaci da trazimo \(\dfrac{1}{(\tg x + \ctg x)^2}\), a ne \((\tg x + \ctg x)^2\). Kada je imenilac minimalan, razlomak je maksimalan.

Zaboraviti domen: Funkcija nije definisana za \(x = k\pi/2\) (gde tangens ili kotangens nisu definisani), ali to ne utice na nalazenje maksimuma.

Dodatni izazov

Odredite najmanju vrednost funkcije \(g(x) = (\tg x + \ctg x)^{2}\).

Hint

Koristite istu transformaciju: \(g(x) = \dfrac{4}{\sin^2(2x)}\). Kako je \(\sin^2(2x) \leq 1\), sledi \(g(x) \geq 4\). Minimum je 4 i dostize se za \(x = \pi/4 + k\pi/2\).
