Maksimum linearne kombinacije sinusnih funkcija sa parametrom

Trigonometrija -- adiciona formula, amplituda linearne kombinacije, parametar k

Zadatak

Ako je \(k \in \mathbb{R}\) konstanta, maksimum funkcije

\[f(x) = k\sin(x + 60°) - \sin x\]

je:

(A)
\(\sqrt{k^2 + k + 1}\)
(B)
\(\sqrt{k^2 - k + 1}\)
(C)
\(k - \dfrac{1}{2}\)
(D)
\(k\)
(E)
\(\sqrt{k^2 - k + \dfrac{1}{2}}\)

Plan resavanja

Razvicemo \(\sin(x + 60°)\) adicionom formulom, srediti \(f(x)\) u oblik \(A\sin x + B\cos x\), a zatim primeniti cinjenicu da je maksimum takve funkcije jednak \(\sqrt{A^2 + B^2}\).

Kljucne formule i teoreme
  • Adiciona formula: \(\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta\)
  • Poznate vrednosti: \(\cos 60° = \dfrac{1}{2}\), \(\sin 60° = \dfrac{\sqrt{3}}{2}\)
  • Amplituda: Maksimum funkcije \(A\sin x + B\cos x\) je \(\sqrt{A^2 + B^2}\)
  • Dokaz amplitude: \(A\sin x + B\cos x = \sqrt{A^2+B^2}\sin(x+\varphi)\) gde je \(\tg\varphi = B/A\)

Vizuelni prikaz

Grafik \(f(x)\) za razlicite vrednosti parametra \(k\). Koristite klizac da menjate \(k\).



Maksimum: 1,00

Resenje korak po korak

1

Razvijamo \(\sin(x + 60°)\) adicionom formulom

Primenjujemo formulu \(\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta\):

\[\sin(x + 60°) = \sin x \cdot \cos 60° + \cos x \cdot \sin 60°\]
\[\sin(x + 60°) = \frac{1}{2}\sin x + \frac{\sqrt{3}}{2}\cos x\]
2

Zamena u \(f(x)\)

Zamenjujemo razvijeni izraz:

\[f(x) = k\left(\frac{1}{2}\sin x + \frac{\sqrt{3}}{2}\cos x\right) - \sin x\]

Mnozimo \(k\) i sređujemo:

\[f(x) = \frac{k}{2}\sin x + \frac{k\sqrt{3}}{2}\cos x - \sin x\]
3

Grupisemo po \(\sin x\) i \(\cos x\)

Sabiramo koeficijente uz \(\sin x\):

\[f(x) = \left(\frac{k}{2} - 1\right)\sin x + \frac{k\sqrt{3}}{2}\cos x\]

Dakle imamo oblik \(A\sin x + B\cos x\) gde je:

\[A = \frac{k}{2} - 1, \quad B = \frac{k\sqrt{3}}{2}\]
Proveri se: Za \(k = 2\), koliki su koeficijenti \(A\) i \(B\)?

\(A = 2/2 - 1 = 0\), \(B = 2\sqrt{3}/2 = \sqrt{3}\). Funkcija postaje \(f(x) = \sqrt{3}\cos x\), ciji je maksimum \(\sqrt{3}\). ✓

4

Primena formule za amplitudu

Maksimum funkcije \(A\sin x + B\cos x\) je \(\sqrt{A^2 + B^2}\). Racunamo:

\[A^2 = \left(\frac{k}{2} - 1\right)^2 = \frac{k^2}{4} - k + 1\]
\[B^2 = \left(\frac{k\sqrt{3}}{2}\right)^2 = \frac{3k^2}{4}\]
5

Sabiramo \(A^2 + B^2\)

\[A^2 + B^2 = \frac{k^2}{4} - k + 1 + \frac{3k^2}{4}\]

Sabiramo clanove sa \(k^2\):

\[\frac{k^2}{4} + \frac{3k^2}{4} = \frac{4k^2}{4} = k^2\]

Konacno:

\[A^2 + B^2 = k^2 - k + 1\]
6

Zapisujemo maksimum

\[\max f(x) = \sqrt{A^2 + B^2} = \sqrt{k^2 - k + 1}\]

Napomena: \(k^2 - k + 1 = (k - 1/2)^2 + 3/4 > 0\) za svako \(k \in \mathbb{R}\), pa je koren uvek definisan.

Proveri se: Za \(k = 0\), koliki je maksimum?

Za \(k = 0\): \(f(x) = -\sin x\), maksimum je \(1\). Formula: \(\sqrt{0 - 0 + 1} = 1\). ✓

Kljucni uvid

Svaka funkcija oblika \(A\sin x + B\cos x\) se moze zapisati kao \(\sqrt{A^2+B^2}\sin(x + \varphi)\), pa joj je amplituda (maksimum) upravo:

\[\max f(x) = \sqrt{A^2 + B^2} = \sqrt{k^2 - k + 1}\]

Konacni odgovor

\(\displaystyle \max f(x) = \sqrt{k^2 - k + 1}\)
(A) \(\sqrt{k^2 + k + 1}\) (B) \(\sqrt{k^2 - k + 1}\) (C) \(k - \dfrac{1}{2}\) (D) \(k\) (E) \(\sqrt{k^2 - k + \dfrac{1}{2}}\)
Verifikacija: Za \(k = 1\): \(f(x) = \sin(x+60°) - \sin x = -\frac{1}{2}\sin x + \frac{\sqrt{3}}{2}\cos x\). Maksimum: \(\sqrt{1/4 + 3/4} = 1\). Formula: \(\sqrt{1 - 1 + 1} = 1\). ✓ Za \(k = 2\): \(\sqrt{4 - 2 + 1} = \sqrt{3} \approx 1{,}73\). ✓

Ceste greske

Greska u znaku kod \(-k\): Pazite na razvoj \((k/2 - 1)^2 = k^2/4 - k + 1\). Cesta greska je pisati \(+k\) umesto \(-k\), sto daje pogresan odgovor \(\sqrt{k^2 + k + 1}\) (opcija A).

Zaboraviti oduzeti \(\sin x\): Obavezno oduzeti \(\sin x\) od razvijenog izraza. Koeficijent uz \(\sin x\) je \(k/2 - 1\), ne samo \(k/2\).

Dodatni izazov

Za koju vrednost parametra \(k\) je maksimum funkcije \(f(x)\) najmanji?

Hint

Trazimo minimum izraza \(k^2 - k + 1\). Ovo je kvadratna funkcija po \(k\) sa minimumom u temenu: \(k = \dfrac{1}{2}\). Tada je \(\max f = \sqrt{1/4 - 1/2 + 1} = \sqrt{3/4} = \dfrac{\sqrt{3}}{2}\).
