Tangens ugla iz dvostrukog ugla

Trigonometrija — formula za tangens dvostrukog ugla i kvadratna jednačina

📝 Tekst zadatka

Ako je \(0 < x < \frac{\pi}{2}\) i \(\tg 2x = -\frac{1}{3}\), onda je \(\tg x\) jednako:

(A)
\(3 + \sqrt{10}\)
(B)
\(\sqrt{10} - 3\)
(C)
\(3 + \sqrt{8}\)
(D)
\(3 - \sqrt{8}\)
(E)
\(-\frac{1}{6}\)
🗺️ Plan rešavanja

Primenićemo formulu za tangens dvostrukog ugla \(\tg 2x = \frac{2\tg x}{1 - \tg^2 x}\), uvesti smenu \(t = \tg x\), dobiti kvadratnu jednačinu, i izabrati rešenje koje odgovara intervalu \((0, \frac{\pi}{2})\).

📐 Ključne formule i teoreme
  • Tangens dvostrukog ugla: \(\tg 2\alpha = \dfrac{2\tg \alpha}{1 - \tg^2 \alpha}\)
  • Kvadratna formula: Za \(at^2 + bt + c = 0\), rešenja su \(t = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)
  • Znak tangensa: U prvom kvadrantu (\(0 < x < \frac{\pi}{2}\)) važi \(\tg x > 0\)
  • Analiza intervala: Ako je \(\tg 2x < 0\) i \(0 < 2x < \pi\), onda je \(2x\) u drugom kvadrantu, tj. \(\frac{\pi}{4} < x < \frac{\pi}{2}\), pa je \(\tg x > 1\)
📊 Vizuelni prikaz

Prikaz ugla x u prvom kvadrantu i odgovarajuće vrednosti tangensa

📋 Rešenje korak po korak
1

Analiza intervala

Dato je \(0 < x < \frac{\pi}{2}\), a \(\tg 2x = -\frac{1}{3} < 0\).

Pošto je \(0 < 2x < \pi\) i tangens je negativan, ugao \(2x\) leži u drugom kvadrantu:

\[\frac{\pi}{2} < 2x < \pi \implies \frac{\pi}{4} < x < \frac{\pi}{2}\]

Dakle, \(\tg x > 1\) jer je \(x > \frac{\pi}{4}\) u prvom kvadrantu.

2

Primena formule za dvostruki ugao

Koristimo formulu za tangens dvostrukog ugla:

\[\tg 2x = \frac{2\tg x}{1 - \tg^2 x}\]

Uvedimo smenu \(t = \tg x\) i zamenimo \(\tg 2x = -\frac{1}{3}\):

\[\frac{2t}{1 - t^2} = -\frac{1}{3}\]
3

Formiranje kvadratne jednačine

Množimo obe strane sa \(3(1 - t^2)\):

\[6t = -(1 - t^2)\]

Rastvaramo zagradu:

\[6t = -1 + t^2\]

Preuređujemo u standardni oblik:

\[t^2 - 6t - 1 = 0\]
🔍 Proveri se: Koji je diskriminant ove jednačine?
\(D = 36 + 4 = 40\)
4

Rešavanje kvadratne jednačine

Primenjujemo kvadratnu formulu sa \(a = 1\), \(b = -6\), \(c = -1\):

\[t = \frac{6 \pm \sqrt{36 + 4}}{2} = \frac{6 \pm \sqrt{40}}{2}\]

Pojednostavljujemo \(\sqrt{40} = 2\sqrt{10}\):

\[t = \frac{6 \pm 2\sqrt{10}}{2} = 3 \pm \sqrt{10}\]
5

Izbor odgovarajućeg rešenja

Dobili smo dva rešenja:

\[t_1 = 3 + \sqrt{10} \approx 6{,}162 \quad \text{i} \quad t_2 = 3 - \sqrt{10} \approx -0{,}162\]

Iz koraka 1 znamo da mora biti \(\tg x > 1\) (pošto je \(x \in (\frac{\pi}{4}, \frac{\pi}{2})\)).

Rešenje \(t_2 = 3 - \sqrt{10} < 0\) ne zadovoljava uslov \(\tg x > 0\).

Rešenje \(t_1 = 3 + \sqrt{10} > 1\) zadovoljava sve uslove.

🔍 Proveri se: Uvrstimo \(t = 3 + \sqrt{10}\) nazad — da li dobijamo \(\tg 2x = -\frac{1}{3}\)?
Da! \(\tg^2 x = 19 + 6\sqrt{10}\), pa je \(1 - \tg^2 x = -18 - 6\sqrt{10} = -6(3 + \sqrt{10})\). Tada: \(\frac{2(3+\sqrt{10})}{-6(3+\sqrt{10})} = \frac{2}{-6} = -\frac{1}{3}\) ✓
Ključni uvid

Formula za dvostruki ugao pretvara trigonometrijski problem u algebarski — kvadratnu jednačinu. Analiza intervala je ključna za eliminaciju lažnog rešenja!

\[\tg x = 3 + \sqrt{10}\]
Konačan odgovor
\(\tg x = 3 + \sqrt{10}\)
(A) \(3 + \sqrt{10}\) (B) \(\sqrt{10} - 3\) (C) \(3 + \sqrt{8}\) (D) \(3 - \sqrt{8}\) (E) \(-\frac{1}{6}\)
Verifikacija: Uvrštavanjem \(t = 3 + \sqrt{10}\) u formulu za dvostruki ugao dobijamo \(\tg 2x = \frac{2}{-6} = -\frac{1}{3}\), što se slaže sa zadatim uslovom.
⚠️ Česte greške
  • Uzimanje pogrešnog rešenja kvadratne jednačine — obavezno proverite znak tangensa na osnovu intervala u kome se x nalazi.
  • Mešanje opcije (B) \(\sqrt{10} - 3\) sa tačnim odgovorom — ovo je zapravo \(-t_2 = -(3 - \sqrt{10})\), ali to nije \(\tg x\).
🏆 Dodatni izazov

Koliko iznosi \(\sin x\) za isti ugao \(x\)?

Prikaži nagoveštaj
Iskoristi \(\sin^2 x = \frac{\tg^2 x}{1 + \tg^2 x}\) sa \(\tg x = 3 + \sqrt{10}\). Računaj \(\tg^2 x = 19 + 6\sqrt{10}\).
