Kosinus iz tangensa dvostrukog ugla

Trigonometrija — od tangensa dvostrukog ugla do kosinusa preko kvadratne jednačine

📝 Tekst zadatka

Ako je \(\frac{\pi}{2} < x < \pi\) i \(\tg 2x = \frac{4}{3}\), koliko je \(\cos x\)?

(A)
\(\frac{1}{\sqrt{5}}\)
(B)
\(\frac{2}{\sqrt{5}}\)
(C)
\(-\frac{1}{\sqrt{5}}\)
(D)
\(-\frac{2}{\sqrt{5}}\)
(E)
\(-\frac{3}{\sqrt{10}}\)
🗺️ Plan rešavanja

Primenićemo formulu za tangens dvostrukog ugla da nađemo \(\tg x\), pa ćemo iz veze \(1 + \tg^2 x = \frac{1}{\cos^2 x}\) izračunati \(\cos x\), uzimajući u obzir da je \(x\) u drugom kvadrantu.

📐 Ključne formule i teoreme
  • Tangens dvostrukog ugla: \(\tg 2\alpha = \dfrac{2\tg \alpha}{1 - \tg^2 \alpha}\)
  • Veza tangensa i kosinusa: \(1 + \tg^2 \alpha = \dfrac{1}{\cos^2 \alpha}\), odnosno \(\cos^2 \alpha = \dfrac{1}{1 + \tg^2 \alpha}\)
  • Drugi kvadrant: Za \(\frac{\pi}{2} < x < \pi\) važi \(\cos x < 0\), \(\sin x > 0\), \(\tg x < 0\)
  • Kvadratna formula: \(t = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\)
📊 Vizuelni prikaz

Ugao x u drugom kvadrantu sa odgovarajućim vrednostima trigonometrijskih funkcija

📋 Rešenje korak po korak
1

Analiza intervala

Dato je \(\frac{\pi}{2} < x < \pi\), što znači da je \(x\) u drugom kvadrantu.

Zato su:

\[\cos x < 0, \quad \sin x > 0, \quad \tg x < 0\]

Ugao \(2x\) pripada intervalu \((\pi, 2\pi)\). Pošto je \(\tg 2x = \frac{4}{3} > 0\), ugao \(2x\) je u trećem kvadrantu: \(\pi < 2x < \frac{3\pi}{2}\).

2

Primena formule za dvostruki ugao

Koristimo formulu za tangens dvostrukog ugla sa \(t = \tg x\):

\[\frac{2t}{1 - t^2} = \frac{4}{3}\]
3

Formiranje kvadratne jednačine

Množimo unakrsno:

\[6t = 4(1 - t^2)\]

Rastvaramo i preuređujemo:

\[6t = 4 - 4t^2\]
\[4t^2 + 6t - 4 = 0\]

Delimo celom jednačinom sa 2:

\[2t^2 + 3t - 2 = 0\]
🔍 Proveri se: Koliki je diskriminant jednačine \(2t^2 + 3t - 2 = 0\)?
\(D = 9 + 16 = 25\), dakle \(\sqrt{D} = 5\)
4

Rešavanje kvadratne jednačine

Primenjujemo formulu sa \(a = 2\), \(b = 3\), \(c = -2\):

\[t = \frac{-3 \pm \sqrt{9 + 16}}{4} = \frac{-3 \pm 5}{4}\]

Dva rešenja:

\[t_1 = \frac{-3 + 5}{4} = \frac{1}{2} \qquad t_2 = \frac{-3 - 5}{4} = -2\]
5

Izbor odgovarajućeg rešenja za tg x

Pošto je \(x\) u drugom kvadrantu, \(\tg x\) mora biti negativan.

Rešenje \(t_1 = \frac{1}{2} > 0\) ne zadovoljava uslov.

Rešenje \(t_2 = -2 < 0\) zadovoljava uslov.

\[\tg x = -2\]
🔍 Proveri se: Ako je tg x = -2, koliko je tg²x?
\(\tg^2 x = (-2)^2 = 4\)
6

Nalaženje cos x iz tg x

Koristimo fundamentalnu vezu:

\[1 + \tg^2 x = \frac{1}{\cos^2 x}\]

Uvrštavamo \(\tg x = -2\):

\[1 + 4 = \frac{1}{\cos^2 x}\]
\[\cos^2 x = \frac{1}{5}\]
7

Određivanje znaka kosinusa

Iz \(\cos^2 x = \frac{1}{5}\) dobijamo:

\[\cos x = \pm \frac{1}{\sqrt{5}}\]

Pošto je \(x\) u drugom kvadrantu, \(\cos x < 0\), pa uzimamo negativnu vrednost:

\[\cos x = -\frac{1}{\sqrt{5}}\]
🔍 Proveri se: Da li se slaže sa polaznim uslovom? Izračunaj tg(2x) koristeći cos x = -1/√5.
\(\sin x = \frac{2}{\sqrt{5}}\) (pozitivan u II kv.), \(\cos 2x = 2\cos^2 x - 1 = \frac{2}{5} - 1 = -\frac{3}{5}\), \(\sin 2x = 2 \cdot \frac{2}{\sqrt{5}} \cdot \left(-\frac{1}{\sqrt{5}}\right) = -\frac{4}{5}\), \(\tg 2x = \frac{-4/5}{-3/5} = \frac{4}{3}\) ✓
Ključni uvid

Zadatak zahteva dva koraka selekcije na osnovu kvadranta: prvo biramo \(\tg x < 0\), pa onda \(\cos x < 0\). Svaki korak koristi informaciju o položaju ugla \(x\)!

\[\cos x = -\frac{1}{\sqrt{5}}\]
Konačan odgovor
\(\cos x = -\dfrac{1}{\sqrt{5}}\)
(A) \(\frac{1}{\sqrt{5}}\) (B) \(\frac{2}{\sqrt{5}}\) (C) \(-\frac{1}{\sqrt{5}}\) (D) \(-\frac{2}{\sqrt{5}}\) (E) \(-\frac{3}{\sqrt{10}}\)
Verifikacija: Sa \(\cos x = -\frac{1}{\sqrt{5}}\) i \(\sin x = \frac{2}{\sqrt{5}}\), dobijamo \(\tg x = -2\), pa \(\tg 2x = \frac{2 \cdot (-2)}{1 - 4} = \frac{-4}{-3} = \frac{4}{3}\), što potvrđuje tačnost.
⚠️ Česte greške
  • Izbor \(\tg x = \frac{1}{2}\) umesto \(\tg x = -2\) — obavezno proverite znak tangensa prema kvadrantu.
  • Uzimanje pozitivne vrednosti kosinusa (\(\frac{1}{\sqrt{5}}\)) umesto negativne — u drugom kvadrantu kosinus je uvek negativan. Opcija (A) je česta zamka!
🏆 Dodatni izazov

Koliko iznosi \(\sin 2x\) za isti ugao?

Prikaži nagoveštaj
Koristite \(\sin 2x = 2\sin x \cos x\). Već znate da je \(\cos x = -\frac{1}{\sqrt{5}}\) i \(\sin x = \frac{2}{\sqrt{5}}\).
