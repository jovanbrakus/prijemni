Trigonometrijska jednačina sa tangensom

Primena adicione formule za tangens i izračunavanje vrednosti trigonometrijske funkcije

📋 Tekst zadatka

Ako je \(\operatorname{tg}\!\left(x + \dfrac{\pi}{4}\right) - 2\operatorname{tg} x = 2\) i \(0 < x < \dfrac{\pi}{2}\), naći \(\sin^2 x\).

(A)
\(0{,}44\)
(B)
\(0{,}2\)
(C)
\(0{,}56\)
(D)
\(0{,}36\)
(E)
\(0{,}6\)

🗺️ Plan rešavanja

Primenićemo adicionu formulu za \(\operatorname{tg}(x + \frac{\pi}{4})\) da bismo jednačinu sveli na algebarsku po \(t = \operatorname{tg} x\). Zatim ćemo iz poznate vrednosti tangensa odrediti \(\sin^2 x\) koristeći fundamentalni trigonometrijski identitet.

📐 Ključne formule i teoreme
  • Adiciona formula za tangens: \(\operatorname{tg}(\alpha + \beta) = \dfrac{\operatorname{tg}\alpha + \operatorname{tg}\beta}{1 - \operatorname{tg}\alpha \cdot \operatorname{tg}\beta}\)
  • Poznata vrednost: \(\operatorname{tg}\dfrac{\pi}{4} = 1\)
  • Veza tangensa i sinusa: \(\sin^2 x = \dfrac{\operatorname{tg}^2 x}{1 + \operatorname{tg}^2 x}\) (dobija se deljenjem identiteta \(\sin^2 x + \cos^2 x = 1\) sa \(\cos^2 x\))

📊 Vizuelni prikaz

Pravougli trougao sa \(\operatorname{tg} x = \frac{1}{2}\) i vrednosti stranica

📝 Rešenje korak po korak

1

Primenimo adicionu formulu

Koristimo formulu za tangens zbira uglova uz činjenicu da je \(\operatorname{tg}\frac{\pi}{4} = 1\):

\[\operatorname{tg}\!\left(x + \frac{\pi}{4}\right) = \frac{\operatorname{tg} x + \operatorname{tg}\frac{\pi}{4}}{1 - \operatorname{tg} x \cdot \operatorname{tg}\frac{\pi}{4}} = \frac{\operatorname{tg} x + 1}{1 - \operatorname{tg} x}\]
2

Uvedimo smenu \(t = \operatorname{tg} x\)

Jednačina postaje:

\[\frac{t + 1}{1 - t} - 2t = 2\]

Prebacimo \(2t\) na desnu stranu:

\[\frac{t + 1}{1 - t} = 2 + 2t = 2(1 + t)\]
🔎 Proveri se: Šta dobijamo ako je \(t + 1 = 0\)?

Ako je \(t = -1\), obe strane jednačine su jednake nuli: leva \(\frac{0}{2} = 0\) i desna \(2 \cdot 0 = 0\). Međutim, \(\operatorname{tg} x = -1\) daje \(x = -\frac{\pi}{4} + k\pi\), što ne pripada intervalu \((0, \frac{\pi}{2})\). Dakle, ovo rešenje odbacujemo.

3

Rešimo jednačinu

Pošto je \(0 < x < \frac{\pi}{2}\), važi \(\operatorname{tg} x > 0\), pa je \(1 + t > 0\). Delimo obe strane sa \((1 + t)\):

\[\frac{1}{1 - t} = 2\]

Odavde:

\[1 - t = \frac{1}{2}\]
\[t = \frac{1}{2}\]

Dakle, \(\operatorname{tg} x = \frac{1}{2}\). Ova vrednost je pozitivna, što je u skladu sa uslovom \(x \in (0, \frac{\pi}{2})\).

4

Izračunajmo \(\sin^2 x\) iz poznate vrednosti tangensa

Koristimo identitet koji povezuje \(\sin^2 x\) i \(\operatorname{tg} x\). Iz \(\sin^2 x + \cos^2 x = 1\), deljenjem sa \(\cos^2 x\), dobijamo \(\operatorname{tg}^2 x + 1 = \frac{1}{\cos^2 x}\), pa je:

\[\sin^2 x = \frac{\operatorname{tg}^2 x}{1 + \operatorname{tg}^2 x}\]

Zamenimo \(\operatorname{tg} x = \frac{1}{2}\):

\[\sin^2 x = \frac{\left(\frac{1}{2}\right)^2}{1 + \left(\frac{1}{2}\right)^2} = \frac{\frac{1}{4}}{\frac{5}{4}} = \frac{1}{5} = 0{,}2\]
🔎 Proveri se: Koliko je \(\cos^2 x\) za \(\operatorname{tg} x = \frac{1}{2}\)?

\(\cos^2 x = \frac{1}{1 + \operatorname{tg}^2 x} = \frac{1}{\frac{5}{4}} = \frac{4}{5} = 0{,}8\). Proverimo: \(\sin^2 x + \cos^2 x = 0{,}2 + 0{,}8 = 1\) ✓

5

Verifikacija u polaznoj jednačini

Proverimo rezultat direktnom zamenom u polaznu jednačinu:

\[\operatorname{tg}\!\left(x + \frac{\pi}{4}\right) = \frac{\frac{1}{2} + 1}{1 - \frac{1}{2}} = \frac{\frac{3}{2}}{\frac{1}{2}} = 3\]
\[\operatorname{tg}\!\left(x + \frac{\pi}{4}\right) - 2\operatorname{tg} x = 3 - 2 \cdot \frac{1}{2} = 3 - 1 = 2 \quad \checkmark\]

Jednačina je zadovoljena, što potvrđuje da je \(\operatorname{tg} x = \frac{1}{2}\) ispravno rešenje.

Ključni uvid

Izraz \(\frac{t+1}{1-t} = 2(1+t)\) sadrži zajednički faktor \((1+t)\). Deljenje njime (uz proveru da \(t \neq -1\)) momentalno daje \(t = \frac{1}{2}\).

Konačan odgovor
\(\sin^2 x = \dfrac{1}{5} = 0{,}2\)
(A) \(0{,}44\) (B) \(0{,}2\) (C) \(0{,}56\) (D) \(0{,}36\) (E) \(0{,}6\)
Provera: Za \(\operatorname{tg} x = \frac{1}{2}\) važi \(\sin x = \frac{1}{\sqrt{5}}\), \(\cos x = \frac{2}{\sqrt{5}}\). Direktnom proverom: \(\operatorname{tg}(x + \frac{\pi}{4}) - 2\operatorname{tg} x = 3 - 1 = 2\). Vrednost \(\sin^2 x = \frac{1}{5} = 0{,}2\) je razuman rezultat za ugao od približno \(26{,}6°\).

⚠️ Česte greške

  • Zaboravljanje smene \(t = -1\): Prilikom deljenja sa \((1+t)\) obavezno proveriti slučaj \(t = -1\). U ovom zadatku se ta vrednost odbacuje zbog ograničenja \(x \in (0, \frac{\pi}{2})\), ali u opštem slučaju bi mogla biti validno rešenje.
  • Pogrešna formula za \(\sin^2 x\): Studenti ponekad mešaju formulu sa \(\cos^2 x = \frac{\operatorname{tg}^2 x}{1 + \operatorname{tg}^2 x}\). Ispravno je: \(\sin^2 x = \frac{\operatorname{tg}^2 x}{1 + \operatorname{tg}^2 x}\), dok je \(\cos^2 x = \frac{1}{1 + \operatorname{tg}^2 x}\).
🚀 Dodatni izazov

Ako je \(\operatorname{tg}(x + \frac{\pi}{4}) - 3\operatorname{tg} x = 3\) i \(0 < x < \frac{\pi}{2}\), odrediti \(\cos 2x\).

Nagovještaj: Primeniti istu tehniku. Iz \(\operatorname{tg} x = t\) će se dobiti linearna jednačina. Zatim koristiti formulu \(\cos 2x = \frac{1 - \operatorname{tg}^2 x}{1 + \operatorname{tg}^2 x}\).
