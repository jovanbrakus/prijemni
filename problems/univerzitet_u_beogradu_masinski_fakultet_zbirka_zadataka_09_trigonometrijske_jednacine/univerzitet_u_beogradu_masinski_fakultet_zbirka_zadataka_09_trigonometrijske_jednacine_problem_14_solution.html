Zbir rešenja trigonometrijske jednačine

Nalaženje četiri najmanja pozitivna rešenja jednačine sa kosinusima višestrukih uglova

📝 Tekst zadatka
Zbir 4 najmanja pozitivna rešenja jednačine
\[ \cos x + \cos 2x + \cos 4x = 0 \]
je:
(A)
\(\dfrac{35}{18}\pi\)
(B)
\(\dfrac{23}{14}\pi\)
(C)
\(\dfrac{8}{3}\pi\)
(D)
\(\dfrac{37}{18}\pi\)
(E)
\(\dfrac{20}{9}\pi\)
🎯 Plan rešavanja

Pomnožićemo jednačinu sa \(2\sin x\) i primeniti formulu za proizvod sinusa i kosinusa da dobijemo teleskopski niz. To će svesti jednačinu na proizvod dva faktora, od kojih svaki daje familiju rešenja. Na kraju sortiramo rešenja i saberemo 4 najmanja.

📚 Podsetnik iz teorije
Formula za proizvod sinusa i kosinusa
\[ 2\sin\alpha\cos\beta = \sin(\alpha+\beta) + \sin(\alpha-\beta) \]

Ova formula nam omogućava da proizvod pretvorimo u zbir/razliku sinusa, što vodi ka teleskopiranju.

Razlika sinusa
\[ \sin A - \sin B = 2\cos\frac{A+B}{2}\sin\frac{A-B}{2} \]

Koristimo za faktorizaciju izraza oblika \(\sin 5x - \sin x\).

Rešenja jednačine \(\cos\theta = -\frac{1}{2}\)
\[ \theta = \frac{2\pi}{3} + 2k\pi \quad \text{ili} \quad \theta = \frac{4\pi}{3} + 2k\pi, \quad k \in \mathbb{Z} \]

Odnosno, \(\theta = \pm\frac{2\pi}{3} + 2k\pi\).

🎨 Vizuelni prikaz

Graf funkcije \(f(x) = \cos x + \cos 2x + \cos 4x\). Zelene tačke označavaju 4 najmanja pozitivna rešenja.

🛠 Rešenje korak po korak
1 Pomnoži jednačinu sa \(2\sin x\)

Polazna jednačina je:

\[ \cos x + \cos 2x + \cos 4x = 0 \]

Pomnožimo obe strane sa \(2\sin x\) (naknadno ćemo proveriti slučaj \(\sin x = 0\)):

\[ 2\sin x\cos x + 2\sin x\cos 2x + 2\sin x\cos 4x = 0 \]
2 Primeni formulu za proizvod

Koristimo \(2\sin\alpha\cos\beta = \sin(\alpha+\beta) + \sin(\alpha-\beta)\) za svaki sabirak:

\[ 2\sin x\cos x = \sin 2x \]
\[ 2\sin x\cos 2x = \sin 3x + \sin(-x) = \sin 3x - \sin x \]
\[ 2\sin x\cos 4x = \sin 5x + \sin(-3x) = \sin 5x - \sin 3x \]
Proveri sebe: Zašto je \(\sin(-x) = -\sin x\)?

Sinus je neparna funkcija, tj. \(\sin(-\alpha) = -\sin\alpha\) za svako \(\alpha\). Geometrijski, refleksija tačke na jediničnom krugu u odnosu na \(x\)-osu menja znak \(y\)-koordinate.

3 Saberi i pojednostavi (teleskopiranje)

Saberemo sva tri izraza. Primetimo da se \(\sin 3x\) i \(-\sin 3x\) poništavaju:

\[ \sin 2x + (\sin 3x - \sin x) + (\sin 5x - \sin 3x) = 0 \] \[ \sin 5x + \sin 2x - \sin x = 0 \]
4 Faktoriši \(\sin 5x - \sin x\)

Koristimo formulu za razliku sinusa:

\[ \sin 5x - \sin x = 2\cos\frac{5x+x}{2}\sin\frac{5x-x}{2} = 2\cos 3x\sin 2x \]

Zamenjujemo u jednačinu:

\[ 2\cos 3x\sin 2x + \sin 2x = 0 \]
5 Izvuci zajednički faktor

Izvlačimo \(\sin 2x\) ispred zagrade:

\[ \sin 2x\,(2\cos 3x + 1) = 0 \]

Jednačina je zadovoljena kada je bar jedan faktor jednak nuli.

Proveri sebe: Zašto \(\sin x = 0\) ne daje rešenja?

Ako je \(\sin x = 0\), onda je \(x = k\pi\). Za \(x = 0\): \(\cos 0 + \cos 0 + \cos 0 = 3 \neq 0\). Za \(x = \pi\): \(\cos\pi + \cos 2\pi + \cos 4\pi = -1 + 1 + 1 = 1 \neq 0\). Dakle, množenje sa \(2\sin x\) ne uvodi lažna rešenja.

6 Slučaj 1: \(\sin 2x = 0\)

Jednačina \(\sin 2x = 0\) daje \(2x = n\pi\), tj. \(x = \frac{n\pi}{2}\), gde je \(n \in \mathbb{Z}\).

Proveravamo koja pozitivna rešenja zadovoljavaju polaznu jednačinu:

\[ x = \frac{\pi}{2}: \quad \cos\frac{\pi}{2} + \cos\pi + \cos 2\pi = 0 + (-1) + 1 = 0 \;\checkmark \]
\[ x = \pi: \quad \cos\pi + \cos 2\pi + \cos 4\pi = -1 + 1 + 1 = 1 \neq 0 \;\text{(otpada)} \]

Dakle, iz ovog slučaja dobijamo rešenja oblika \(x = \frac{\pi}{2} + k\pi\), tj. \(x = \frac{\pi}{2},\; \frac{3\pi}{2},\; \frac{5\pi}{2},\ldots\)

7 Slučaj 2: \(\cos 3x = -\frac{1}{2}\)

Jednačina \(\cos 3x = -\frac{1}{2}\) ima opšta rešenja:

\[ 3x = \frac{2\pi}{3} + 2k\pi \quad \Rightarrow \quad x = \frac{2\pi}{9} + \frac{2k\pi}{3} \]
\[ 3x = \frac{4\pi}{3} + 2k\pi \quad \Rightarrow \quad x = \frac{4\pi}{9} + \frac{2k\pi}{3} \]

Najmanja pozitivna rešenja iz prve familije (za \(k = 0, 1, 2, \ldots\)):

\[ x = \frac{2\pi}{9},\; \frac{8\pi}{9},\; \frac{14\pi}{9},\; \ldots \]

Najmanja pozitivna rešenja iz druge familije:

\[ x = \frac{4\pi}{9},\; \frac{10\pi}{9},\; \frac{16\pi}{9},\; \ldots \]
8 Sortiraj i odaberi 4 najmanja

Poređajmo sva pronađena rešenja po veličini:

\[ \frac{2\pi}{9} \approx 0{,}698 \quad < \quad \frac{4\pi}{9} \approx 1{,}396 \quad < \quad \frac{\pi}{2} \approx 1{,}571 \quad < \quad \frac{8\pi}{9} \approx 2{,}793 \]

Dakle, četiri najmanja pozitivna rešenja su: \(\frac{2\pi}{9}\), \(\frac{4\pi}{9}\), \(\frac{\pi}{2}\) i \(\frac{8\pi}{9}\).

Proveri sebe: Zašto \(\frac{\pi}{2}\) dolazi posle \(\frac{4\pi}{9}\)?

Uporedimo: \(\frac{4\pi}{9} = \frac{8\pi}{18}\), a \(\frac{\pi}{2} = \frac{9\pi}{18}\). Pošto je \(\frac{8}{18} < \frac{9}{18}\), važi \(\frac{4\pi}{9} < \frac{\pi}{2}\).

9 Izračunaj zbir

Saberimo četiri najmanja rešenja. Prvo saberemo tri razlomka sa imeniocem 9:

\[ \frac{2\pi}{9} + \frac{4\pi}{9} + \frac{8\pi}{9} = \frac{14\pi}{9} \]

Sada dodamo \(\frac{\pi}{2}\), svodeći na zajednički imenilac 18:

\[ \frac{14\pi}{9} + \frac{\pi}{2} = \frac{28\pi}{18} + \frac{9\pi}{18} = \frac{37\pi}{18} \]
Ključni uvid

Tehnika množenja sa \(2\sin x\) i korišćenje formule \(2\sin\alpha\cos\beta = \sin(\alpha+\beta) + \sin(\alpha-\beta)\) dovodi do teleskopiranja — većina članova se poništi! Krajnji rezultat je elegantan proizvod \(\sin 2x(2\cos 3x + 1) = 0\).

Konačan odgovor
\(\dfrac{2\pi}{9} + \dfrac{4\pi}{9} + \dfrac{\pi}{2} + \dfrac{8\pi}{9} = \dfrac{37}{18}\pi\)
(A) \(\dfrac{35}{18}\pi\) (B) \(\dfrac{23}{14}\pi\) (C) \(\dfrac{8}{3}\pi\) (D) \(\dfrac{37}{18}\pi\) (E) \(\dfrac{20}{9}\pi\)

Verifikacija: \(\frac{37\pi}{18} \approx 6{,}458\). Numerička provera: \(0{,}698 + 1{,}396 + 1{,}571 + 2{,}793 = 6{,}458\). Vrednosti se poklapaju.

⚠ Česte greške
Propustiti rešenja iz \(\sin 2x = 0\)
Slučaj \(\sin 2x = 0\) daje \(x = \frac{\pi}{2}\), koje je jedno od 4 najmanja rešenja. Ako se fokusiramo samo na \(\cos 3x = -\frac{1}{2}\), propuštamo ovo rešenje i dobijamo pogrešan zbir \(\frac{14\pi}{9} + \frac{10\pi}{9} = \frac{24\pi}{9} = \frac{8\pi}{3}\), što je odgovor (C).
Ne proveriti lažna rešenja posle množenja
Množenje sa \(2\sin x\) može uvesti lažna rešenja (gde je \(\sin x = 0\)). Uvek treba proveriti da kandidati zaista zadovoljavaju polaznu jednačinu. U ovom slučaju, \(x = \pi\) zadovoljava \(\sin 2x = 0\) ali ne zadovoljava polaznu jednačinu.
🏆 Dodatni izazov

Koliki je zbir svih pozitivnih rešenja jednačine \(\cos x + \cos 2x + \cos 4x = 0\) na intervalu \([0, 2\pi)\)?

Nagovještaj

Na intervalu \([0, 2\pi)\) postoji ukupno 8 rešenja. Iz slučaja \(\sin 2x = 0\) imamo \(x = \frac{\pi}{2}\) i \(x = \frac{3\pi}{2}\). Iz slučaja \(\cos 3x = -\frac{1}{2}\) imamo 6 rešenja: \(\frac{2\pi}{9}, \frac{4\pi}{9}, \frac{8\pi}{9}, \frac{10\pi}{9}, \frac{14\pi}{9}, \frac{16\pi}{9}\). Saberi ih sve!
