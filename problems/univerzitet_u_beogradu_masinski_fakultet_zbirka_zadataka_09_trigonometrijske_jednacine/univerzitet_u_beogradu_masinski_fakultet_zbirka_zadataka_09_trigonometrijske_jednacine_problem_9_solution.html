Trigonometrijska jednačina sa kvadratima sinusa

Trigonometrijske jednačine -- korišćenje identiteta i rešavanje u zadatom intervalu

Zadatak

Koliko rešenja ima jednačina

\[\sin^2 2x + \sin^2 3x = 1\]

u intervalu \((0, \pi)\)?

(A)
\(1\)
(B)
\(2\)
(C)
\(4\)
(D)
\(5\)
(E)
\(6\)

Plan rešavanja

Iskoristićemo identitet \(\sin^2\alpha + \cos^2\alpha = 1\) da transformišemo jednačinu u oblik koji se može faktorisati. Zatim ćemo rešiti svaki faktor posebno i prebrojati rešenja u intervalu \((0, \pi)\).

Ključne formule i teoreme
  • Pitagorin identitet: \(\sin^2\alpha + \cos^2\alpha = 1\), pa je \(\cos^2\alpha = 1 - \sin^2\alpha\)
  • Razlika kvadrata: \(a^2 - b^2 = (a-b)(a+b)\)
  • Veza kosinus-sinus: \(\cos\alpha = \sin\!\left(\frac{\pi}{2} - \alpha\right)\)
  • Opšte rešenje za \(\sin\alpha = \sin\beta\): \(\alpha = \beta + 2k\pi\) ili \(\alpha = \pi - \beta + 2k\pi\), \(k \in \mathbb{Z}\)

Grafički prikaz

Grafici funkcija \(y = \sin^2 2x\) (plavo) i \(y = \cos^2 2x\) (zeleno) u intervalu \((0, \pi)\). Rešenja jednačine su tačke gde je \(\sin^2 3x = \cos^2 2x\).

Vertikalne linije označavaju svih 5 rešenja jednačine

Rešenje korak po korak

1

Transformacija jednačine

Polazimo od jednačine i koristimo identitet \(\cos^2 2x = 1 - \sin^2 2x\):

\[\sin^2 2x + \sin^2 3x = 1\]
\[\sin^2 3x = 1 - \sin^2 2x = \cos^2 2x\]
2

Faktorisanje

Prenosimo sve na jednu stranu i primenjujemo razliku kvadrata:

\[\sin^2 3x - \cos^2 2x = 0\]
\[(\sin 3x - \cos 2x)(\sin 3x + \cos 2x) = 0\]

Sada rešavamo svaki faktor posebno.

Proveri sebe: Koji identitet smo koristili za prelaz sa \(\sin^2 3x - \cos^2 2x\) na proizvod?

Razliku kvadrata: \(a^2 - b^2 = (a - b)(a + b)\), gde je \(a = \sin 3x\), \(b = \cos 2x\).

3

Slučaj 1: \(\sin 3x = \cos 2x\)

Koristimo vezu \(\cos 2x = \sin\!\left(\frac{\pi}{2} - 2x\right)\), pa jednačina postaje:

\[\sin 3x = \sin\!\left(\frac{\pi}{2} - 2x\right)\]

Podslučaj 1a:

\[3x = \frac{\pi}{2} - 2x + 2k\pi \;\Rightarrow\; 5x = \frac{\pi}{2} + 2k\pi \;\Rightarrow\; x = \frac{\pi}{10} + \frac{2k\pi}{5}\]

Podslučaj 1b:

\[3x = \pi - \left(\frac{\pi}{2} - 2x\right) + 2k\pi \;\Rightarrow\; 3x = \frac{\pi}{2} + 2x + 2k\pi \;\Rightarrow\; x = \frac{\pi}{2} + 2k\pi\]
4

Slučaj 2: \(\sin 3x = -\cos 2x\)

Koristimo \(-\cos 2x = \sin\!\left(2x - \frac{\pi}{2}\right)\), pa:

\[\sin 3x = \sin\!\left(2x - \frac{\pi}{2}\right)\]

Podslučaj 2a:

\[3x = 2x - \frac{\pi}{2} + 2k\pi \;\Rightarrow\; x = -\frac{\pi}{2} + 2k\pi\]

Podslučaj 2b:

\[3x = \pi - \left(2x - \frac{\pi}{2}\right) + 2k\pi \;\Rightarrow\; 5x = \frac{3\pi}{2} + 2k\pi \;\Rightarrow\; x = \frac{3\pi}{10} + \frac{2k\pi}{5}\]
Proveri sebe: Koliko familija rešenja smo dobili?

Četiri familije: \(x = \frac{\pi}{10} + \frac{2k\pi}{5}\), \(x = \frac{\pi}{2} + 2k\pi\), \(x = -\frac{\pi}{2} + 2k\pi\), i \(x = \frac{3\pi}{10} + \frac{2k\pi}{5}\).

5

Nalaženje rešenja u \((0, \pi)\)

Iz familije \(x = \frac{\pi}{10} + \frac{2k\pi}{5}\):

\[k = 0:\; x = \frac{\pi}{10} \approx 0{,}314 \quad \checkmark\] \[k = 1:\; x = \frac{\pi}{10} + \frac{2\pi}{5} = \frac{\pi}{2} \approx 1{,}571 \quad \checkmark\] \[k = 2:\; x = \frac{\pi}{10} + \frac{4\pi}{5} = \frac{9\pi}{10} \approx 2{,}827 \quad \checkmark\]

Iz familije \(x = \frac{\pi}{2} + 2k\pi\): za \(k=0\) dobijamo \(x = \frac{\pi}{2}\), koje već imamo.

Iz familije \(x = -\frac{\pi}{2} + 2k\pi\): za \(k=1\) dobijamo \(x = \frac{3\pi}{2} > \pi\). Nema novih rešenja.

Iz familije \(x = \frac{3\pi}{10} + \frac{2k\pi}{5}\):

\[k = 0:\; x = \frac{3\pi}{10} \approx 0{,}942 \quad \checkmark\] \[k = 1:\; x = \frac{3\pi}{10} + \frac{2\pi}{5} = \frac{7\pi}{10} \approx 2{,}199 \quad \checkmark\]
6

Pregled svih rešenja

Ukupan skup rešenja u intervalu \((0, \pi)\):

\[x \in \left\{\frac{\pi}{10},\; \frac{3\pi}{10},\; \frac{\pi}{2},\; \frac{7\pi}{10},\; \frac{9\pi}{10}\right\}\]

To je ukupno 5 rešenja.

Proveri sebe: Da li rešenje \(x = \frac{\pi}{2}\) zaista zadovoljava jednačinu?

\(\sin^2(2 \cdot \frac{\pi}{2}) + \sin^2(3 \cdot \frac{\pi}{2}) = \sin^2 \pi + \sin^2 \frac{3\pi}{2} = 0 + 1 = 1\). Da, zadovoljava!

Ključni uvid

Trik je u prepoznavanju da se \(1 - \sin^2 2x\) može zameniti sa \(\cos^2 2x\), čime se jednačina svodi na razliku kvadrata. To daje dva linearna trigonometrijska uslova umesto jednog kvadratnog.

\[\sin^2 3x = \cos^2 2x \;\Longleftrightarrow\; (\sin 3x - \cos 2x)(\sin 3x + \cos 2x) = 0\]

Konačan odgovor

\(\text{Jednačina ima } 5 \text{ rešenja u intervalu } (0, \pi)\)
(A) \(1\) (B) \(2\) (C) \(4\) (D) \(5\) (E) \(6\)
Verifikacija: Proverom za svaku vrednost: \(\frac{\pi}{10}\), \(\frac{3\pi}{10}\), \(\frac{\pi}{2}\), \(\frac{7\pi}{10}\), \(\frac{9\pi}{10}\) potvrđujemo da sve zadovoljavaju polaznu jednačinu. Na primer, za \(x = \frac{\pi}{10}\): \(\sin^2\frac{\pi}{5} + \sin^2\frac{3\pi}{10} = \sin^2 36° + \sin^2 54° = \sin^2 36° + \cos^2 36° = 1\). Simetrija rešenja oko \(\frac{\pi}{2}\) dodatno potvrđuje rezultat.

Česte greške

Propuštanje podslučajeva: Kada rešavamo \(\sin\alpha = \sin\beta\), obavezno razmatramo oba slučaja: \(\alpha = \beta + 2k\pi\) i \(\alpha = \pi - \beta + 2k\pi\). Propuštanje drugog slučaja vodi na samo 3 od 5 rešenja.
Dupla rešenja: Rešenje \(x = \frac{\pi}{2}\) se pojavljuje u dve različite familije. Treba ga brojati samo jednom!

Dodatni izazov

Koliko rešenja ima jednačina \(\sin^2 2x + \sin^2 3x = 1\) u intervalu \((0, 2\pi)\)?

Nagovještaj

Koristite iste familije rešenja, ali sada tražite vrednosti \(k\) za koje je \(0 < x < 2\pi\). Uočite simetriju — za svaku familiju oblika \(x = a + \frac{2k\pi}{5}\) dobijate dvostruko više rešenja.
