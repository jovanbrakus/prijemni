Obim pravouglog trougla sa upisanim krugom

Geometrija u ravni -- poluprečnik upisanog kruga i obim trougla 30-60-90

Zadatak

Zbir dužina stranica pravouglog trougla čiji je jedan ugao \(\dfrac{\pi}{3}\) i poluprečnik upisanog kruga \(\sqrt{3} - 1\) jednak je:

Plan rešavanja

Iz uglova trougla (30-60-90) izrazićemo sve stranice preko jednog parametra. Zatim ćemo primeniti formulu za poluprečnik upisanog kruga pravouglog trougla \(r = \frac{a + b - c}{2}\), odrediti parametar, i izračunati obim.

Podsetnik iz teorije

Ključne formule i teoreme

Trougao 30-60-90:

\[\text{Ako je } \alpha = 30°, \text{ tada su stranice u odnosu } 1 : \sqrt{3} : 2\]

Poluprečnik upisanog kruga trougla:

\[r = \frac{P}{s}\]

gde je \(P\) površina, a \(s = \frac{a+b+c}{2}\) poluobim.

Za pravougli trougao (specijalan slučaj):

\[r = \frac{a + b - c}{2}\]

gde su \(a\) i \(b\) katete, a \(c\) hipotenuza. Ova formula se izvodi iz opšte formule jer je \(P = \frac{ab}{2}\) i \(s = \frac{a+b+c}{2}\), a za pravougli trougao važi \(a + b - c = 2r\).

Dijagram

Pravougli trougao 30-60-90 sa upisanim krugom

Rešenje korak po korak

1

Određivanje uglova

Imamo pravougli trougao (jedan ugao je \(90°\)) i još jedan ugao je \(\dfrac{\pi}{3} = 60°\).

Treći ugao je:

\[180° - 90° - 60° = 30°\]

Dakle, radi se o trouglu sa uglovima \(30°\), \(60°\) i \(90°\).

2

Izražavanje stranica preko parametra

Neka je \(a\) kateta naspram ugla od \(30°\) (najkraća stranica).

Prema osobinama trougla 30-60-90:

\[a = a \quad \text{(kateta naspram } 30°\text{)}\]
\[b = a\sqrt{3} \quad \text{(kateta naspram } 60°\text{)}\]
\[c = 2a \quad \text{(hipotenuza naspram } 90°\text{)}\]
Proveri se: Koliki je odnos stranica u trouglu 30-60-90?

Odnos je \(1 : \sqrt{3} : 2\). Najkraća kateta je naspram ugla od \(30°\), najduža stranica (hipotenuza) je naspram pravog ugla.

3

Primena formule za upisani krug

Za pravougli trougao, poluprečnik upisanog kruga je:

\[r = \frac{a + b - c}{2}\]

Zamenjujemo izraze za stranice:

\[r = \frac{a + a\sqrt{3} - 2a}{2}\]
\[r = \frac{a(\sqrt{3} - 1)}{2}\]
4

Određivanje vrednosti \(a\)

Poznato je da je \(r = \sqrt{3} - 1\). Zamenjujemo:

\[\frac{a(\sqrt{3} - 1)}{2} = \sqrt{3} - 1\]

Delimo obe strane sa \((\sqrt{3} - 1)\), što je dozvoljeno jer je \(\sqrt{3} - 1 \neq 0\):

\[\frac{a}{2} = 1\]
\[a = 2\]
Proveri se: Kolika je kraća kateta \(a\)?

\(a = 2\). Iz ovoga sledi \(b = 2\sqrt{3}\) i \(c = 4\).

5

Izračunavanje obima

Sada nalazimo sve stranice i sabiramo:

\[a = 2, \quad b = 2\sqrt{3}, \quad c = 4\]
\[\text{Obim} = a + b + c = 2 + 2\sqrt{3} + 4 = 6 + 2\sqrt{3}\]

Ključni uvid

Zahvaljujući formuli \(r = \frac{a + b - c}{2}\) za pravougli trougao i poznatim odnosima stranica trougla 30-60-90, problem se svodi na jednu jednostavnu jednačinu po parametru \(a\):

\[\frac{a(\sqrt{3} - 1)}{2} = \sqrt{3} - 1 \implies a = 2\]

Konačan odgovor

\(\text{Obim} = 6 + 2\sqrt{3}\)
(A) \(4 + 2\sqrt{3}\) (B) \(6 + 4\sqrt{3}\) (C) \(3 + 3\sqrt{3}\) (D) \(3 + \sqrt{3}\) (E) \(6 + 2\sqrt{3}\)
Verifikacija: Proveravamo poluprečnik upisanog kruga: \(r = \frac{2 + 2\sqrt{3} - 4}{2} = \frac{2\sqrt{3} - 2}{2} = \sqrt{3} - 1\). Ovo se slaže sa zadatom vrednošću. Takođe, Pitagorina teorema: \(2^2 + (2\sqrt{3})^2 = 4 + 12 = 16 = 4^2\). Sve se proverava.

Česte greške

Zamena kateta: Učenici ponekad zamene koja kateta je naspram kog ugla. Kateta naspram \(30°\) je najkraća (\(a\)), a kateta naspram \(60°\) je duža (\(a\sqrt{3}\)). Hipotenuza je uvek \(2a\).
Pogrešna formula za \(r\): Formula \(r = \frac{a + b - c}{2}\) važi samo za pravougli trougao. Opšta formula je \(r = \frac{P}{s}\) gde je \(P\) površina, a \(s\) poluobim. Za pravougli trougao se obe svode na isto.

Dodatni izazov

Koliki je poluprečnik opisanog kruga ovog trougla? Koliki je odnos poluprečnika opisanog i upisanog kruga?

Nagoveštaj

Za pravougli trougao, poluprečnik opisanog kruga je \(R = \frac{c}{2}\). Izračunajte \(R\) i uporedite sa \(r = \sqrt{3} - 1\).
