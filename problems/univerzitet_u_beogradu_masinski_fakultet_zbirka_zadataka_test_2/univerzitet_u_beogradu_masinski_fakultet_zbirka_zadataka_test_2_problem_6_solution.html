Deljivost polinoma i određivanje koeficijenata

Algebra -- nalaženje nepoznatih koeficijenata polinoma na osnovu uslova deljivosti

Zadatak

Ako je polinom \(x^3 + ax^2 + b\) deljiv polinomom \(x^2 + 3x + 1\), onda je \(a + 2b\) jednako:

(A)
\(0\)
(B)
\(1\)
(C)
\(\frac{5}{3}\)
(D)
\(2\)
(E)
\(\frac{8}{3}\)

Plan rešavanja

Pošto je polinom trećeg stepena deljiv polinomom drugog stepena, količnik mora biti linearan: \(x + c\). Pomnožićemo i izjednačiti koeficijente.

Podsetnik iz teorije

Ključne formule

Deljivost polinoma: Ako je \(P(x)\) deljiv sa \(Q(x)\), tada postoji polinom \(R(x)\) takav da:

\[P(x) = Q(x) \cdot R(x)\]

Izjednačavanje koeficijenata: Dva polinoma su jednaka ako i samo ako su im svi odgovarajući koeficijenti jednaki.

Vizuelni prikaz

Šema izjednačavanja koeficijenata pri množenju polinoma.

Rešenje korak po korak

1

Postavljanje jednakosti

Količnik mora biti oblika \(x + c\) (jer je razlika stepena jednaka 1):

\[x^3 + ax^2 + b = (x^2 + 3x + 1)(x + c)\]
2

Razvijamo desnu stranu

Množimo:

\[(x^2 + 3x + 1)(x + c) = x^3 + cx^2 + 3x^2 + 3cx + x + c\]

Sređujemo po stepenima:

\[= x^3 + (c + 3)x^2 + (3c + 1)x + c\]
3

Izjednačavanje koeficijenata

Uporedimo sa \(x^3 + ax^2 + 0 \cdot x + b\):

\[\text{Uz } x^2: \quad a = c + 3\]
\[\text{Uz } x^1: \quad 0 = 3c + 1 \implies c = -\frac{1}{3}\]
\[\text{Uz } x^0: \quad b = c = -\frac{1}{3}\]
Proveri sebe: Koliko je \(a\)?

\(a = c + 3 = -\frac{1}{3} + 3 = \frac{8}{3}\)

4

Računanje \(a + 2b\)

Zamenjujemo nađene vrednosti:

\[a + 2b = \frac{8}{3} + 2 \cdot \left(-\frac{1}{3}\right) = \frac{8}{3} - \frac{2}{3} = \frac{6}{3} = 2\]

Ključni uvid

Uslov da koeficijent uz \(x\) bude nula (jer u polaznom polinomu nema člana sa \(x\)) direktno određuje \(c = -\frac{1}{3}\), a odatle i sve ostalo:

\[a + 2b = 2\]

Konačan odgovor

\[a + 2b = 2\]
(A) \(0\) (B) \(1\) (C) \(\frac{5}{3}\) (D) \(2\) (E) \(\frac{8}{3}\)
Verifikacija: \(\left(x^2 + 3x + 1\right)\left(x - \frac{1}{3}\right) = x^3 + 3x^2 + x - \frac{x^2}{3} - x - \frac{1}{3} = x^3 + \frac{8}{3}x^2 - \frac{1}{3}\). Koeficijent uz \(x\) je zaista \(0\).

Česte greške

Zaboravljanje da nema člana sa \(x\): U polinomu \(x^3 + ax^2 + b\) koeficijent uz \(x\) je 0, što daje ključnu jednačinu \(3c + 1 = 0\).
Mešanje \(a\) i \(b\): \(a = \frac{8}{3}\) (odgovor E) nije tražena veličina -- traži se \(a + 2b\).

Dodatni izazov

Za koje vrednosti \(p\) je polinom \(x^3 - px^2 + p\) deljiv sa \(x - 1\)?

Nagovještaj

Koristite Bezuovu teoremu: ako je \(x - 1\) delilac, onda je \(f(1) = 0\), tj. \(1 - p + p = 1 \neq 0\). Dakle, \(x - 1\) nikada nije delilac!
