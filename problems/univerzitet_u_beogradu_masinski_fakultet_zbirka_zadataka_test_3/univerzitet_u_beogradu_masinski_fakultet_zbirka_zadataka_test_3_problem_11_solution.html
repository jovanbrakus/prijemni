Nalaženje sinusa preko tangensa

Trigonometrija — veza između trigonometrijskih funkcija i znaci po kvadrantima

Zadatak

Ako je \(\operatorname{tg} x = -\sqrt{8}\) i \(\frac{\pi}{2} < x < \frac{3\pi}{2}\), koliko je \(\sin x\)?

(A)
\(-\frac{\sqrt{8}}{3}\)
(B)
\(\frac{\sqrt{8}}{3}\)
(C)
\(-\frac{1}{3}\)
(D)
\(\frac{1}{3}\)
(E)
\(\frac{\sqrt{8}}{8}\)

Plan rešavanja

Iz tangensa ćemo naći kosinus koristeći identitet \(1+\operatorname{tg}^2 x = \frac{1}{\cos^2 x}\), odrediti znak kosinusa na osnovu kvadranta, pa izračunati sinus kao \(\sin x = \operatorname{tg} x \cdot \cos x\).

Podsetnik iz teorije

Ključne formule i teoreme

Osnovni identitet:

\[1+\operatorname{tg}^2 x = \frac{1}{\cos^2 x}\]

Znaci u II kvadrantu (\(\pi/2 < x < \pi\)):

\(\sin x > 0\), \(\cos x < 0\), \(\operatorname{tg} x < 0\)

Vizuelni prikaz

Položaj ugla \(x\) na jediničnoj kružnici — II kvadrant sa \(\operatorname{tg} x < 0\).

Rešenje korak po korak

1

Nalazimo \(\cos^2 x\)

\[1+\operatorname{tg}^2 x = \frac{1}{\cos^2 x}\]
\[1 + 8 = \frac{1}{\cos^2 x}\]
\[\cos^2 x = \frac{1}{9}\]
2

Određujemo znak kosinusa

Ugao \(x\) je u oblasti \(\frac{\pi}{2} < x < \frac{3\pi}{2}\).

Pošto je \(\operatorname{tg} x = -\sqrt{8} < 0\), ugao mora biti u II kvadrantu (gde je \(\operatorname{tg} < 0\), za razliku od III gde je \(\operatorname{tg} > 0\)).

U II kvadrantu, \(\cos x < 0\):

\[\cos x = -\frac{1}{3}\]
Proveri sebe: U kom kvadrantu je tangens negativan?

Tangens je negativan u II i IV kvadrantu (gde sinus i kosinus imaju različite znake).

3

Nalazimo \(\sin x\)

\[\sin x = \operatorname{tg} x \cdot \cos x = (-\sqrt{8})\cdot\left(-\frac{1}{3}\right) = \frac{\sqrt{8}}{3}\]
4

Verifikacija znaka

U II kvadrantu: \(\sin x > 0\). Zaista, \(\frac{\sqrt{8}}{3} > 0\). ✓

Ključni uvid

Oblast \(\pi/2 < x < 3\pi/2\) obuhvata i II i III kvadrant, ali znak tangensa sužava izbor na II kvadrant, gde su i sinus pozitivan i kosinus negativan.

Konačan odgovor

\(\sin x = \dfrac{\sqrt{8}}{3} = \dfrac{2\sqrt{2}}{3}\)
(A) \(-\frac{\sqrt{8}}{3}\) (B) \(\frac{\sqrt{8}}{3}\) (C) \(-\frac{1}{3}\) (D) \(\frac{1}{3}\) (E) \(\frac{\sqrt{8}}{8}\)
Verifikacija: \(\sin^2 x + \cos^2 x = \frac{8}{9}+\frac{1}{9}=1\) ✓. Takođe, \(\operatorname{tg} x = \frac{\sqrt{8}/3}{-1/3} = -\sqrt{8}\) ✓.

Česte greške

Pogrešan znak: Bez analize kvadranta lako se dobije \(-\sqrt{8}/3\) umesto \(\sqrt{8}/3\).

Ignorisanje oblasti: Oblast \(\pi/2 < x < 3\pi/2\) je šira od jednog kvadranta — treba je presediti sa znakom tangensa.
