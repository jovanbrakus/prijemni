Trigonometrijski identitet sa razlikom petih stepena

Trigonometrija — uprošćavanje izraza koristeći razliku stepena i osnovne identitete

Zadatak

Koji izraz je identički jednak izrazu \(\dfrac{4(\sin^5 x - \cos^5 x)}{\sin x - \cos x}\)?

(A)
\(1 + \sin 2x - \sin^2 2x\)
(B)
\(4 + 2\sin 2x - \sin^2 2x\)
(C)
\(4 + 2\sin 2x + \sin^2 2x\)
(D)
\(1 + \sin 2x + \sin^2 2x\)
(E)
\(4 - 2\sin 2x + \sin^2 2x\)

Plan rešavanja

Primenićemo formulu za razliku petih stepena \(a^5 - b^5 = (a-b)(a^4+a^3b+a^2b^2+ab^3+b^4)\), skratimo \(\sin x - \cos x\), pa sredimo preostali izraz koristeći \(\sin^2 x + \cos^2 x = 1\) i \(\sin 2x = 2\sin x\cos x\).

Podsetnik iz teorije

Ključne formule

Razlika petih stepena:

\[a^5 - b^5 = (a-b)(a^4 + a^3b + a^2b^2 + ab^3 + b^4)\]

Pitagorin identitet: \(\sin^2 x + \cos^2 x = 1\)

Dupli ugao: \(\sin 2x = 2\sin x\cos x\)

Zbir četvrtih stepena: \(\sin^4 x + \cos^4 x = 1 - 2\sin^2 x\cos^2 x\)

Vizuelni prikaz

Grafici se potpuno poklapaju, potvrđujući identitet

Rešenje korak po korak

1

Primenimo formulu za razliku petih stepena

Sa \(a = \sin x\), \(b = \cos x\):

\[\frac{4(\sin^5 x - \cos^5 x)}{\sin x - \cos x} = 4(\sin^4 x + \sin^3 x\cos x + \sin^2 x\cos^2 x + \sin x\cos^3 x + \cos^4 x)\]
2

Grupišemo članove

\[\underbrace{\sin^4 x + \cos^4 x}_{A} + \underbrace{\sin^3 x\cos x + \sin x\cos^3 x}_{B} + \underbrace{\sin^2 x\cos^2 x}_{C}\]
3

Sredimo svaku grupu

Grupa A:

\[\sin^4 x + \cos^4 x = (\sin^2 x + \cos^2 x)^2 - 2\sin^2 x\cos^2 x = 1 - 2\sin^2 x\cos^2 x\]

Grupa B:

\[\sin^3 x\cos x + \sin x\cos^3 x = \sin x\cos x(\sin^2 x + \cos^2 x) = \sin x\cos x\]

Grupa C: \(\sin^2 x\cos^2 x\) ostaje kao jeste.

Proveri sebe: Koliko je A + B + C?

\(1 - 2\sin^2 x\cos^2 x + \sin x\cos x + \sin^2 x\cos^2 x = 1 - \sin^2 x\cos^2 x + \sin x\cos x\)

4

Saberemo

\[A + B + C = 1 - \sin^2 x\cos^2 x + \sin x\cos x\]
5

Zamenimo sa sin 2x

Koristimo \(\sin x\cos x = \frac{\sin 2x}{2}\) i \(\sin^2 x\cos^2 x = \frac{\sin^2 2x}{4}\):

\[A + B + C = 1 - \frac{\sin^2 2x}{4} + \frac{\sin 2x}{2}\]
6

Pomnožimo sa 4

\[4(A + B + C) = 4 - \sin^2 2x + 2\sin 2x = 4 + 2\sin 2x - \sin^2 2x\]

Ključni uvid

Formula za razliku petih stepena skraćuje imenilac, a zatim se sve svodi na osnovne trigonometrijske identitete i formulu za dvostruki ugao.

Konačan odgovor

\(\dfrac{4(\sin^5 x - \cos^5 x)}{\sin x - \cos x} = 4 + 2\sin 2x - \sin^2 2x\)
(A) \(1+\sin 2x-\sin^2 2x\) (B) \(4+2\sin 2x-\sin^2 2x\) (C) \(4+2\sin 2x+\sin^2 2x\) (D) \(1+\sin 2x+\sin^2 2x\) (E) \(4-2\sin 2x+\sin^2 2x\)
Verifikacija: Za \(x = \frac{\pi}{6}\): leva strana = \(\frac{4(1/32 - \sqrt{3}^5/32)}{1/2 - \sqrt{3}/2}\). Desna strana = \(4 + 2\cdot\frac{\sqrt{3}}{2} - \frac{3}{4} = 4 + \sqrt{3} - 0{,}75 \approx 4{,}98\). Obe strane daju isti rezultat. ✓

Česte greške

Zaboravljanje množenja sa 4: Bez faktora 4 ispred zagrade, dobija se pogrešan odgovor.
Greška u znaku: U odgovoru je \(-\sin^2 2x\), ne \(+\sin^2 2x\). Pažljivo sa grupom A.
