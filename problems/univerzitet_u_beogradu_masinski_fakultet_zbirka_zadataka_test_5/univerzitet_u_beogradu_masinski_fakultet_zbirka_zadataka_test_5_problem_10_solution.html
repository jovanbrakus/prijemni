Jednačina sa modulom kompleksnog broja

Kompleksni brojevi — rešavanje jednačine koja sadrži modul kompleksnog broja

Zadatak

Realni deo rešenja jednačine \(z + |z + 2i| = 2i + 1\) je:

(A)
\(5\)
(B)
\(-\frac{15}{2}\)
(C)
\(2\)
(D)
\(\frac{17}{2}\)
(E)
\(-\frac{17}{2}\)

Plan rešavanja

Zamenićemo \(z = x + yi\) i iskoristićemo činjenicu da je modul \(|z + 2i|\) realan broj. Izjednačavanjem realnih i imaginarnih delova dobijamo sistem jednačina.

Podsetnik iz teorije

Ključne formule i teoreme

Modul kompleksnog broja:

\[|a + bi| = \sqrt{a^2 + b^2} \in \mathbb{R}\]

Jednakost kompleksnih brojeva: \(a + bi = c + di\) ako i samo ako \(a = c\) i \(b = d\).

Rešenje korak po korak

1

Uvodimo \(z = x + yi\)

Zamenjujemo u jednačinu:

\[(x + yi) + |x + (y+2)i| = 1 + 2i\]

Pošto je \(|x + (y+2)i| = \sqrt{x^2 + (y+2)^2}\) realan broj:

\[\left(x + \sqrt{x^2 + (y+2)^2}\right) + yi = 1 + 2i\]
2

Izjednačavamo imaginarne delove

\[y = 2\]
3

Izjednačavamo realne delove

Sa \(y = 2\):

\[x + \sqrt{x^2 + (2+2)^2} = 1\]
\[x + \sqrt{x^2 + 16} = 1\]
4

Izolujemo koren

\[\sqrt{x^2 + 16} = 1 - x\]

Uslov: \(1 - x \geq 0\), tj. \(x \leq 1\).

Proveri sebe: Zašto mora biti \(x \leq 1\)?

Zato što je koren uvek nenegativan, pa i desna strana \(1 - x\) mora biti nenegativna.

5

Kvadriramo obe strane

\[x^2 + 16 = (1 - x)^2 = 1 - 2x + x^2\]
\[16 = 1 - 2x\]
\[2x = 1 - 16 = -15\]
\[x = -\frac{15}{2}\]
6

Provera

\(x = -\frac{15}{2} \leq 1\) ✓ (uslov zadovoljen)

\[\sqrt{\frac{225}{4} + 16} = \sqrt{\frac{225 + 64}{4}} = \sqrt{\frac{289}{4}} = \frac{17}{2}\]
\[x + \frac{17}{2} = -\frac{15}{2} + \frac{17}{2} = \frac{2}{2} = 1 \; ✓\]

Ključni uvid

Modul kompleksnog broja je uvek realan, pa se dodaje samo na realni deo jednačine. Izjednačavanjem imaginarnih delova odmah se dobija \(y = 2\).

Konačan odgovor

\(\text{Re}(z) = -\dfrac{15}{2}\)
(A) \(5\) (B) \(-\frac{15}{2}\) (C) \(2\) (D) \(\frac{17}{2}\) (E) \(-\frac{17}{2}\)
Verifikacija: \(z = -\frac{15}{2} + 2i\). \(z + 2i = -\frac{15}{2} + 4i\). \(|z+2i| = \sqrt{\frac{225}{4} + 16} = \frac{17}{2}\). \(z + |z+2i| = -\frac{15}{2} + 2i + \frac{17}{2} = 1 + 2i\). ✓

Česte greške

Zaboravljanje da je modul realan broj — ako ga tretiramo kao kompleksni, pogrešno ćemo postaviti jednačinu.
Neprovera uslova \(x \leq 1\) nakon kvadriranja — lažna rešenja mogu nastati.
