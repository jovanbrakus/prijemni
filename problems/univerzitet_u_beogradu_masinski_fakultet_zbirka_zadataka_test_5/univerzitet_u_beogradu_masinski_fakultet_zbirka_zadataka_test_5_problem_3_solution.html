Zbir apsolutnih vrednosti rešenja jednačine

Algebra — jednačina sa apsolutnom vrednošću i kvadratnom funkcijom

Zadatak

Zbir apsolutnih vrednosti rešenja jednačine \(x^2 = |5x - 6|\) je:

(A)
\(0\)
(B)
\(5\)
(C)
\(7\)
(D)
\(10\)
(E)
\(12\)

Plan rešavanja

Uklonićemo apsolutnu vrednost razmatranjem dva slučaja: kada je \(5x - 6 \geq 0\) i kada je \(5x - 6 < 0\). U svakom slučaju rešavamo kvadratnu jednačinu, proveravamo validnost rešenja, pa sabiramo apsolutne vrednosti.

Podsetnik iz teorije

Ključne formule i teoreme

Definicija apsolutne vrednosti:

\[|a| = \begin{cases} a, & a \geq 0 \\ -a, & a < 0 \end{cases}\]

Kvadratna jednačina: \(ax^2 + bx + c = 0\) ima rešenja \(x_{1,2} = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}\).

Vjetov teoreme: \(x_1 + x_2 = -\frac{b}{a}\), \(x_1 \cdot x_2 = \frac{c}{a}\).

Vizuelni prikaz

Grafik pokazuje preseke parabole \(y = x^2\) i funkcije \(y = |5x - 6|\).

Rešenje korak po korak

1

Slučaj 1: \(5x - 6 \geq 0\), tj. \(x \geq \frac{6}{5}\)

Apsolutna vrednost se sklanja bez promene znaka:

\[x^2 = 5x - 6\]
2

Rešavamo prvu kvadratnu jednačinu

\[x^2 - 5x + 6 = 0\]
\[(x - 2)(x - 3) = 0\]

\(x_1 = 2\) i \(x_2 = 3\). Oba zadovoljavaju uslov \(x \geq \frac{6}{5}\). ✓

Proveri sebe: Da li je \(x = 2\) zaista rešenje?

Leva strana: \(2^2 = 4\). Desna strana: \(|5 \cdot 2 - 6| = |4| = 4\). Da, jednako je! ✓

3

Slučaj 2: \(5x - 6 < 0\), tj. \(x < \frac{6}{5}\)

Apsolutna vrednost menja znak:

\[x^2 = -(5x - 6) = -5x + 6\]
4

Rešavamo drugu kvadratnu jednačinu

\[x^2 + 5x - 6 = 0\]
\[(x + 6)(x - 1) = 0\]

\(x_3 = -6\) i \(x_4 = 1\). Oba zadovoljavaju uslov \(x < \frac{6}{5}\). ✓

Proveri sebe: Da li je \(x = -6\) zaista rešenje?

Leva strana: \((-6)^2 = 36\). Desna strana: \(|5 \cdot (-6) - 6| = |-36| = 36\). Da! ✓

5

Zbir apsolutnih vrednosti

Sva četiri rešenja su: \(x \in \{-6,\; 1,\; 2,\; 3\}\).

\[|-6| + |1| + |2| + |3| = 6 + 1 + 2 + 3 = 12\]

Ključni uvid

Jednačina sa apsolutnom vrednošću se uvek razdvaja na dva slučaja. U ovom zadatku oba slučaja daju po dva validna rešenja, ukupno četiri.

Konačan odgovor

\(|-6| + |1| + |2| + |3| = 12\)
(A) \(0\) (B) \(5\) (C) \(7\) (D) \(10\) (E) \(12\)
Verifikacija: Proverimo svako rešenje: \((-6)^2 = 36 = |{-36}|\) ✓, \(1^2 = 1 = |{-1}|\) ✓, \(2^2 = 4 = |4|\) ✓, \(3^2 = 9 = |9|\) ✓. Zbir apsolutnih vrednosti: \(6 + 1 + 2 + 3 = 12\). ✓

Česte greške

Razmatranje samo jednog slučaja za apsolutnu vrednost — time se gube rešenja \(x = -6\) i \(x = 1\).
Sabiranje rešenja umesto apsolutnih vrednosti: \(-6 + 1 + 2 + 3 = 0\), što bi dalo pogrešan odgovor (A).

Dodatni izazov

Nađi zbir apsolutnih vrednosti rešenja jednačine \(x^2 = |3x - 2|\).

Hint

Slučaj 1: \(x^2 - 3x + 2 = 0 \Rightarrow x = 1, 2\). Slučaj 2: \(x^2 + 3x - 2 = 0 \Rightarrow x = \frac{-3 \pm \sqrt{17}}{2}\). Saberi apsolutne vrednosti svih validnih rešenja.
