Zbir kvadrata nula polinoma

Algebra — Vjetove formule za kubnu jednačinu

Zadatak

Ako su \(a\), \(b\) i \(c\) nule polinoma \(x^3 + x^2 - 4x + 1\), čemu je jednako \(a^2 + b^2 + c^2\)?

(A)
\(1\)
(B)
\(5\)
(C)
\(8\)
(D)
\(-7\)
(E)
\(9\)

Plan rešavanja

Koristićemo Vjetove formule za kubnu jednačinu da izrazimo \(a + b + c\) i \(ab + bc + ac\), a zatim identitet \(a^2 + b^2 + c^2 = (a+b+c)^2 - 2(ab+bc+ac)\).

Podsetnik iz teorije

Ključne formule i teoreme

Vjetove formule za kubnu jednačinu \(x^3 + px^2 + qx + r = 0\) sa korenima \(a, b, c\):

\[a + b + c = -p, \quad ab + bc + ac = q, \quad abc = -r\]

Korisni identitet:

\[a^2 + b^2 + c^2 = (a + b + c)^2 - 2(ab + bc + ac)\]

Vizuelni prikaz

Grafik polinoma \(P(x) = x^3 + x^2 - 4x + 1\) sa označenim nulama.

Rešenje korak po korak

1

Identifikujemo koeficijente

Polinom \(x^3 + x^2 - 4x + 1\) ima koeficijente:

\[p = 1, \quad q = -4, \quad r = 1\]
2

Primenjujemo Vjetove formule

\[a + b + c = -p = -1\]
\[ab + bc + ac = q = -4\]
Proveri sebe: Koliko je \(abc\)?

\(abc = -r = -1\). Ovo nam neće trebati za ovaj zadatak, ali je korisno znati.

3

Primenjujemo identitet

\[a^2 + b^2 + c^2 = (a + b + c)^2 - 2(ab + bc + ac)\]
4

Zamenjujemo vrednosti

\[a^2 + b^2 + c^2 = (-1)^2 - 2 \cdot (-4)\]
\[= 1 + 8 = 9\]

Ključni uvid

Ne moramo naći same korene kubne jednačine! Vjetove formule i algebarski identiteti omogućavaju izračunavanje simetričnih izraza korena direktno iz koeficijenata polinoma.

Konačan odgovor

\(a^2 + b^2 + c^2 = 9\)
(A) \(1\) (B) \(5\) (C) \(8\) (D) \(-7\) (E) \(9\)
Verifikacija: Numerički koreni su približno \(a \approx -2{,}57\), \(b \approx 0{,}27\), \(c \approx 1{,}30\). Tada \(a^2 + b^2 + c^2 \approx 6{,}60 + 0{,}07 + 1{,}69 \approx 8{,}36\)... Za tačniju proveru: \((-1)^2 - 2 \cdot (-4) = 1 + 8 = 9\). ✓

Česte greške

Pogrešan znak u Vjetovim formulama — za \(x^3 + px^2 + qx + r = 0\), zbir korena je \(-p\), ne \(p\).
Pokušaj nalaženja korena kubne jednačine — to je nepotrebno i komplikovano. Vjetove formule daju rezultat direktno.

Dodatni izazov

Koristeći iste Vjetove formule, izračunaj \(a^3 + b^3 + c^3\).

Hint

Koristi identitet \(a^3 + b^3 + c^3 - 3abc = (a+b+c)(a^2+b^2+c^2 - ab - bc - ac)\). Već znamo sve potrebne veličine: \(a+b+c = -1\), \(a^2+b^2+c^2 = 9\), \(ab+bc+ac = -4\), \(abc = -1\).
