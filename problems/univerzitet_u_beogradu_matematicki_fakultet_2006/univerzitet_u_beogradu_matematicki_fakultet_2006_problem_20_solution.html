Poređenje binomnih koeficijenata

📝 Tekst zadatka

Ako je \(a = \binom{2006}{1002}\), \(b = \binom{2006}{1003}\), \(c = \binom{2006}{1004}\), onda je:

Dato: \(a = \binom{2006}{1002}\)
Dato: \(b = \binom{2006}{1003}\)
Dato: \(c = \binom{2006}{1004}\)
(A) \(a < b < c\)
(B) \(c < b < a\)
(C) \(a = c < b\)
(D) \(a = b > c\)
(E) \(a = c > b\)
🗺️ Plan rešavanja
  1. Primeniti svojstvo simetrije binomnog koeficijenta da uporedimo \(a\) i \(c\)
  2. Koristiti formulu za odnos uzastopnih binomnih koeficijenata da uporedimo \(b\) sa \(a\)
  3. Odrediti položaj maksimuma binomnog koeficijenta \(\binom{2006}{k}\) u zavisnosti od \(k\)
  4. Zaključiti konačan odnos \(a\), \(b\) i \(c\)
📚 Teorijsko podsetnik — Binomni koeficijenti

Definicija:

\[ \binom{n}{k} = \frac{n!}{k!\,(n-k)!} \]

Svojstvo simetrije:

\[ \binom{n}{k} = \binom{n}{n-k} \]

Ovo važi jer zamena \(k\) sa \(n - k\) u formuli daje istu vrednost (zamena mesta imenilaca \(k!\) i \((n-k)!\)).


Odnos uzastopnih koeficijenata:

\[ \frac{\binom{n}{k}}{\binom{n}{k-1}} = \frac{n - k + 1}{k} \]

Ako je ovaj količnik veći od 1, tj. \(n - k + 1 > k\), odnosno \(k < \dfrac{n+1}{2}\), onda \(\binom{n}{k}\) raste sa porastom \(k\).


Maksimum: Za parno \(n\), binomni koeficijent \(\binom{n}{k}\) je maksimalan za \(k = \dfrac{n}{2}\). Za neparno \(n\), postoje dva jednaka maksimuma: \(k = \dfrac{n-1}{2}\) i \(k = \dfrac{n+1}{2}\).


Oblik grafika: Funkcija \(k \mapsto \binom{n}{k}\) je simetrična u odnosu na \(k = \dfrac{n}{2}\) i ima oblik zvona (unimodalna je).

📊 Vizuelni prikaz — Binomni koeficijenti oko maksimuma

Dijagram prikazuje relativne vrednosti \(\binom{2006}{k}\) za \(k\) oko 1003. Maksimum je u \(k = 1003\), a vrednosti u \(k = 1002\) i \(k = 1004\) su jednake (simetrija).

📐 Rešenje korak po korak
1

Upoređujemo \(a\) i \(c\) pomoću simetrije

Primenjujemo svojstvo simetrije \(\binom{n}{k} = \binom{n}{n - k}\):

\[ a = \binom{2006}{1002} = \binom{2006}{2006 - 1002} = \binom{2006}{1004} = c \]
✓ Kontrolna tačka: \(a = c\). Ovo odmah eliminiše opcije (A) i (B), gde su sva tri broja različita.
2

Upoređujemo \(b\) i \(a\) pomoću odnosa uzastopnih koeficijenata

Računamo količnik \(\dfrac{b}{a}\):

\[ \frac{b}{a} = \frac{\binom{2006}{1003}}{\binom{2006}{1002}} = \frac{2006 - 1003 + 1}{1003} = \frac{1004}{1003} \]

Pošto je \(\dfrac{1004}{1003} > 1\), zaključujemo:

\[ b > a \]
✓ Proveri sebe — izvođenje formule za odnos

Podsećanje na izvođenje:

\[ \frac{\binom{n}{k}}{\binom{n}{k-1}} = \frac{n!}{k!\,(n-k)!} \cdot \frac{(k-1)!\,(n-k+1)!}{n!} = \frac{(n-k+1)}{k} \]

Za \(n = 2006\), \(k = 1003\):

\[ \frac{\binom{2006}{1003}}{\binom{2006}{1002}} = \frac{2006 - 1003 + 1}{1003} = \frac{1004}{1003} \approx 1{,}001 \]
3

Razmatramo položaj maksimuma

Za \(n = 2006\) (paran broj), binomni koeficijent \(\binom{2006}{k}\) dostiže maksimum za:

\[ k = \frac{n}{2} = \frac{2006}{2} = 1003 \]

Dakle, \(b = \binom{2006}{1003}\) je upravo maksimalni binomni koeficijent u nizu \(\binom{2006}{0}, \binom{2006}{1}, \ldots, \binom{2006}{2006}\).

Vrednosti \(a = \binom{2006}{1002}\) i \(c = \binom{2006}{1004}\) su simetrične oko maksimuma, pa su jednake i manje od njega.

✓ Kontrolna tačka: \(a = c < b\), jer je \(b\) maksimum, a \(a\) i \(c\) su jednako udaljeni od njega.
4

Formiramo konačan zaključak

Kombinujemo rezultate:

  • \(a = c\) (iz koraka 1, svojstvo simetrije)
  • \(b > a\) (iz koraka 2, odnos \(\frac{1004}{1003} > 1\))

Dakle:

\[ a = c < b \]
5

Verifikacija — dodatna provera odnosa \(b\) i \(c\)

Da bismo bili potpuno sigurni, proveravamo i \(\dfrac{b}{c}\):

\[ \frac{b}{c} = \frac{\binom{2006}{1003}}{\binom{2006}{1004}} = \frac{1004}{2006 - 1004 + 1} \cdot \frac{1}{1} \]

Koristimo formulu u obrnutom smeru. Pošto je \(\dfrac{\binom{2006}{1004}}{\binom{2006}{1003}} = \dfrac{2006 - 1004 + 1}{1004} = \dfrac{1003}{1004} < 1\), sledi \(c < b\). Potvrđeno!

✓ Kompletna verifikacija: \(a = c\) i \(b > a = c\), tj. \(a = c < b\) ✓
💡 Ključni uvid

Binomni koeficijent \(\binom{n}{k}\) ima oblik simetričnog zvona oko srednje vrednosti \(k = \dfrac{n}{2}\). Pošto je \(n = 2006\) paran, maksimum je u tačno jednoj tački: \(k = 1003\). Vrednosti \(k = 1002\) i \(k = 1004\) su na jednakoj udaljenosti od centra, pa daju jednake koeficijente — a oboje su manje od centralnog maksimuma.

Generalno pravilo: \(\binom{n}{k} = \binom{n}{n-k}\) je jedno od najkorisnijih svojstava za brzo poređenje binomnih koeficijenata. Uvek proverite da li se tražene vrednosti nalaze simetrično oko \(\dfrac{n}{2}\).

✅ Konačan odgovor
\(a = c < b\)

\(\binom{2006}{1002} = \binom{2006}{1004} < \binom{2006}{1003}\), jer je \(\binom{2006}{1003}\) maksimalni binomni koeficijent.

(A) \(a < b < c\)
(B) \(c < b < a\)
(C) \(a = c < b\)
(D) \(a = b > c\)
(E) \(a = c > b\)
⚠️ Česte greške
Zaboravljanje svojstva simetrije: Mnogi studenti pokušavaju da direktno uporede \(\binom{2006}{1002}\) i \(\binom{2006}{1004}\) bez primene \(\binom{n}{k} = \binom{n}{n-k}\). Ovo svojstvo odmah daje \(a = c\) i eliminiše dve opcije.
Pogrešan smer nejednakosti: Opcija (E) kaže \(a = c > b\). Ovo bi značilo da su koeficijenti pored maksimuma veći od samog maksimuma — što je nemoguće. Binomni koeficijenti rastu do \(k = \lfloor n/2 \rfloor\), a zatim opadaju.
Mešanje \(n/2\) sa \((n-1)/2\): Za parno \(n = 2006\), maksimum je u jednoj tački \(k = 1003\). Za neparno \(n\), postoje dva jednaka maksimuma. Bitno je prepoznati da je \(2006\) paran.
Numerička računica umesto formule: Nije potrebno računati vrednosti ovih ogromnih brojeva. Dovoljne su osobine binomnog koeficijenta (simetrija + unimodalnost) za potpun odgovor.
🏆 Dodatni izazov

Koliko iznosi tačan odnos \(\dfrac{a}{b} = \dfrac{\binom{2006}{1002}}{\binom{2006}{1003}}\)?

Uputstvo: Koristi formulu za odnos uzastopnih koeficijenata. Rezultat je \(\dfrac{a}{b} = \dfrac{1003}{1004}\), što znači da je \(a\) samo za faktor \(\dfrac{1}{1004} \approx 0{,}001\) manji od \(b\). Razlika je izuzetno mala, ali ipak strogo važi \(a < b\).
