Jednačina sa apsolutnom vrednošću: \(x^2 + |x-1| = 1\)

Zadatak

Koliko različitih realnih rešenja ima jednačina \(x^2 + \sqrt{(x-1)^2} = 1\)?

(A) \(0\)
(B) \(1\)
(C) \(2\)
(D) \(3\)
(E) više od 3

Plan rešavanja

  1. Pojednostaviti izraz \(\sqrt{(x-1)^2}\) koristeći svojstvo apsolutne vrednosti
  2. Razdvojiti jednačinu na dva slučaja u zavisnosti od znaka izraza \(x - 1\)
  3. Rešiti svaku kvadratnu jednačinu i proveriti validnost rešenja
  4. Prebrojati ukupan broj različitih realnih rešenja

Teorijsko podsetnik

Apsolutna vrednost i kvadratni koren

Ključno svojstvo koje koristimo:

\[\sqrt{a^2} = |a| \quad \text{za svako realno } a\]

Ovo važi jer je kvadratni koren uvek nenegativan. Dakle:

\[\sqrt{(x-1)^2} = |x-1|\]

Apsolutna vrednost se razbija na slučajeve:

\[|x - 1| = \begin{cases} x - 1, & \text{ako } x \geq 1 \\ 1 - x, & \text{ako } x < 1 \end{cases}\]

Grafički prikaz

Grafik funkcija \(y = x^2 + |x-1|\) i \(y = 1\). Presečne tačke su rešenja jednačine.

Rešenje korak po korak

1

Pojednostavljivanje izraza

Primenjujemo svojstvo \(\sqrt{a^2} = |a|\):

\[x^2 + \sqrt{(x-1)^2} = 1 \quad \Longrightarrow \quad x^2 + |x-1| = 1\]
✅ Kontrolna tačka: Jednačina je sada \(x^2 + |x-1| = 1\). Treba razmotriti dva slučaja.
2

Slučaj 1: \(x \geq 1\)

Kada je \(x \geq 1\), važi \(|x-1| = x - 1\). Jednačina postaje:

\[x^2 + x - 1 = 1\] \[x^2 + x - 2 = 0\]

Faktorizujemo:

\[(x + 2)(x - 1) = 0\]

Rešenja su \(x = -2\) i \(x = 1\).

Provera uslova \(x \geq 1\):

  • \(x = -2\): Ne zadovoljava uslov \(x \geq 1\) — odbacujemo
  • \(x = 1\): Zadovoljava uslov — prihvatamo
✅ Kontrolna tačka: Iz prvog slučaja imamo jedno rešenje: \(x = 1\).
3

Slučaj 2: \(x < 1\)

Kada je \(x < 1\), važi \(|x-1| = 1 - x\). Jednačina postaje:

\[x^2 + 1 - x = 1\] \[x^2 - x = 0\]

Faktorizujemo:

\[x(x - 1) = 0\]

Rešenja su \(x = 0\) i \(x = 1\).

Provera uslova \(x < 1\):

  • \(x = 0\): Zadovoljava uslov \(x < 1\) — prihvatamo
  • \(x = 1\): Ne zadovoljava strogi uslov \(x < 1\) — odbacujemo (već pronađeno u slučaju 1)
✅ Kontrolna tačka: Iz drugog slučaja imamo jedno novo rešenje: \(x = 0\).
4

Provera rešenja

Za \(x = 1\):

\[1^2 + \sqrt{(1-1)^2} = 1 + \sqrt{0} = 1 + 0 = 1 \quad \checkmark\]

Za \(x = 0\):

\[0^2 + \sqrt{(0-1)^2} = 0 + \sqrt{1} = 0 + 1 = 1 \quad \checkmark\]

Oba rešenja zadovoljavaju polaznu jednačinu.

Ključni uvid

Izraz \(\sqrt{(x-1)^2}\) je zapravo \(|x-1|\) — apsolutna vrednost. Ovo je česta "zamka" na prijemnim ispitima. Kada se jednačina rastavi na slučajeve, u svakom slučaju dobijamo po dve kandidat-rešenja, ali se provera uslova svodi na po jedno validno rešenje iz svakog slučaja.

Konačan odgovor

Jednačina ima tačno dva različita realna rešenja:

\(x = 0 \quad \text{i} \quad x = 1\)
(A) 0 (B) 1 (C) 2 (D) 3 (E) više od 3

Najčešće greške

Greška 1: Pogrešno pojednostaviti \(\sqrt{(x-1)^2} = x - 1\) bez apsolutne vrednosti. Ovo važi samo za \(x \geq 1\), a za \(x < 1\) rezultat je \(1 - x\).
Greška 2: Zaboraviti proveriti uslov za svaki slučaj. Na primer, rešenje \(x = -2\) iz prvog slučaja ne zadovoljava uslov \(x \geq 1\) i mora se odbaciti.
Greška 3: Prebrojati \(x = 1\) dva puta jer se pojavljuje u oba slučaja. Pitanje traži broj različitih rešenja.

Dodatni izazov

Pokušaj da rešiš sličnu jednačinu: \(x^2 + \sqrt{(x-2)^2} = 4\). Koliko različitih realnih rešenja ima?

Pogledaj odgovor

Jednačina postaje \(x^2 + |x-2| = 4\).

Za \(x \geq 2\): \(x^2 + x - 6 = 0 \Rightarrow (x+3)(x-2) = 0\), pa je \(x = 2\) (validno).

Za \(x < 2\): \(x^2 - x - 2 = 0 \Rightarrow (x-2)(x+1) = 0\), pa je \(x = -1\) (validno).

Odgovor: 2 rešenja (\(x = -1\) i \(x = 2\)).
