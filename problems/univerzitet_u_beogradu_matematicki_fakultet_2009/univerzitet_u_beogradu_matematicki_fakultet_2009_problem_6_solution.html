Stepenovanje kompleksnog broja

Trigonometrijski oblik i De Moavrov obrazac — kada je stepen realnog tipa

✎ Postavka zadatka

Broj \((1 + i\sqrt{3})^n\) je realan ako i samo ako je (\(k\) je ceo broj):

(A)
\(n = 2k\)
(B)
\(n = 3k\)
(C)
\(n = 3k + 1\)
(D)
\(n = 3k + 2\)
(E)
\(n = 6k\)
⚙ Plan rešavanja
  1. Pretvoriti kompleksni broj \(1 + i\sqrt{3}\) u trigonometrijski (polarni) oblik
  2. Primeniti De Moavrov obrazac za \(n\)-ti stepen
  3. Odrediti uslov pod kojim je imaginarni deo jednak nuli
Ključne formule i teoreme

Trigonometrijski oblik kompleksnog broja: Svaki kompleksni broj \(z = a + bi\) se može zapisati kao:

\[z = r(\cos\varphi + i\sin\varphi)\]

gde je \(r = |z| = \sqrt{a^2 + b^2}\) modul, a \(\varphi = \arg(z) = \arctan\!\left(\frac{b}{a}\right)\) argument.

De Moavrov obrazac: Za bilo koji kompleksni broj u trigonometrijskom obliku i ceo broj \(n\):

\[z^n = r^n \bigl(\cos(n\varphi) + i\sin(n\varphi)\bigr)\]

Uslov realnosti: Kompleksni broj \(z = a + bi\) je realan ako i samo ako je \(b = 0\), odnosno \(\operatorname{Im}(z) = 0\).

Nule sinusne funkcije: \(\sin\theta = 0\) ako i samo ako je \(\theta = k\pi\), za \(k \in \mathbb{Z}\).

◆ Kompleksna ravan — interaktivni prikaz

Koristite klizač ispod da menjate eksponent \(n\) i vidite gde se \((1+i\sqrt{3})^n\) nalazi u kompleksnoj ravni.


📚 Rešenje korak po korak
1

Određivanje modula

Dati kompleksni broj je \(z = 1 + i\sqrt{3}\). Realni deo je \(a = 1\), a imaginarni deo je \(b = \sqrt{3}\).

Računamo modul:

\[r = |z| = \sqrt{a^2 + b^2} = \sqrt{1^2 + (\sqrt{3})^2} = \sqrt{1 + 3} = \sqrt{4} = 2\]

Modul nam govori koliko je broj udaljen od koordinatnog početka u kompleksnoj ravni.

2

Određivanje argumenta

Argument je ugao koji broj zaklapa sa pozitivnim delom realne ose:

\[\varphi = \arctan\!\left(\frac{b}{a}\right) = \arctan\!\left(\frac{\sqrt{3}}{1}\right) = \arctan(\sqrt{3}) = \frac{\pi}{3}\]

Oba dela (\(a > 0\), \(b > 0\)) su pozitivna, pa je broj u prvom kvadrantu, što potvrđuje \(\varphi = \frac{\pi}{3} = 60°\).

3

Trigonometrijski oblik

Sada zapisujemo broj u trigonometrijskom obliku:

\[1 + i\sqrt{3} = 2\left(\cos\frac{\pi}{3} + i\sin\frac{\pi}{3}\right)\]

Proverite: \(2\cos\frac{\pi}{3} = 2 \cdot \frac{1}{2} = 1\) i \(2\sin\frac{\pi}{3} = 2 \cdot \frac{\sqrt{3}}{2} = \sqrt{3}\). Sve se poklapa!

Proveri sebe: Koliki je modul broja \(1 + i\sqrt{3}\)?
\(|1 + i\sqrt{3}| = 2\), a argument je \(\frac{\pi}{3}\).
4

Primena De Moavrovog obrasca

Stepenujemo broj koristeći De Moavrov obrazac:

\[(1 + i\sqrt{3})^n = 2^n \left(\cos\frac{n\pi}{3} + i\sin\frac{n\pi}{3}\right)\]

Ovaj obrazac daje i modul (\(2^n\)) i argument (\(\frac{n\pi}{3}\)) rezultata.

5

Uslov realnosti

Broj \((1 + i\sqrt{3})^n\) je realan ako i samo ako mu je imaginarni deo jednak nuli:

\[\operatorname{Im}\!\bigl((1 + i\sqrt{3})^n\bigr) = 0\]

Imaginarni deo je \(2^n \sin\frac{n\pi}{3}\). Pošto je \(2^n > 0\) za svako \(n\), uslov se svodi na:

\[\sin\frac{n\pi}{3} = 0\]
Proveri sebe: Kada je \(\sin\theta = 0\)?
Kada je \(\theta = k\pi\), gde je \(k\) ceo broj. Drugim rečima, za \(\theta = 0, \pm\pi, \pm 2\pi, \ldots\)
6

Rešavanje uslova

Funkcija sinus je jednaka nuli kada je njen argument celobrojni umnožak broja \(\pi\):

\[\sin\frac{n\pi}{3} = 0 \quad \Longleftrightarrow \quad \frac{n\pi}{3} = k\pi, \quad k \in \mathbb{Z}\]

Podelimo obe strane sa \(\pi\):

\[\frac{n}{3} = k \quad \Longleftrightarrow \quad n = 3k\]

Dakle, \((1 + i\sqrt{3})^n\) je realan ako i samo ako je \(n\) deljivo sa 3.

7

Verifikacija za konkretne vrednosti

Proverimo za nekoliko vrednosti \(n\):

\(n\) \(\frac{n\pi}{3}\) \(\sin\frac{n\pi}{3}\) \((1+i\sqrt{3})^n\) Realan?
0 0 0 1 Da (3·0)
1 \(\frac{\pi}{3}\) \(\frac{\sqrt{3}}{2}\) \(1 + i\sqrt{3}\) Ne
2 \(\frac{2\pi}{3}\) \(\frac{\sqrt{3}}{2}\) \(-2 + 2i\sqrt{3}\) Ne
3 \(\pi\) 0 \(-8\) Da (3·1)
4 \(\frac{4\pi}{3}\) \(-\frac{\sqrt{3}}{2}\) \(-8 - 8i\sqrt{3}\) Ne
5 \(\frac{5\pi}{3}\) \(-\frac{\sqrt{3}}{2}\) \(16 - 16i\sqrt{3}\) Ne
6 \(2\pi\) 0 64 Da (3·2)

Vidimo jasan obrazac: broj je realan tačno za \(n = 0, 3, 6, 9, \ldots\), tj. za \(n = 3k\).

Proveri sebe: Zašto \(n = 6k\) nije tačan odgovor?
Zato što \(n = 3\) daje realan broj (\(-8\)), ali \(3 \neq 6k\) za bilo koji ceo broj \(k\). Uslov \(n = 6k\) je previše restriktivan — propušta svaki drugi realan rezultat.
★ Ključni uvid

Argument broja \(1 + i\sqrt{3}\) je \(\frac{\pi}{3}\), što je tačno jedna šestina punog kruga. Stepenovanje sa \(n\) rotira broj za \(\frac{n\pi}{3}\) u kompleksnoj ravni. Da bi rezultat pao na realnu osu, potrebna je rotacija za celobrojni umnožak \(\pi\) — a to se dešava svaki treći korak.

\[\sin\frac{n\pi}{3} = 0 \;\;\Longleftrightarrow\;\; n = 3k\]
Tačan odgovor
\(\text{B)}\quad n = 3k\)
(A) \(n = 2k\)
(B) \(n = 3k\)
(C) \(n = 3k + 1\)
(D) \(n = 3k + 2\)
(E) \(n = 6k\)
Verifikacija: Za \(n = 3\): \((1+i\sqrt{3})^3 = 8(\cos\pi + i\sin\pi) = -8\) — realan broj. Za \(n = 6\): \((1+i\sqrt{3})^6 = 64(\cos 2\pi + i\sin 2\pi) = 64\) — realan broj. Za \(n = 1, 2, 4, 5\): imaginarni deo \(\neq 0\) — nisu realni.
⚠ Česte greške
✗ Brkanje \(n = 3k\) i \(n = 6k\): Odgovor \(n = 6k\) bi značio da broj mora da napravi pun krug (ili višestruki) da bi bio realan. Ali on je realan i na pola kruga (ugao \(\pi\)), pa je dovoljno \(n = 3k\).
✗ Pogrešan argument: Ako se pogrešno odredi da je \(\varphi = \frac{\pi}{6}\) umesto \(\frac{\pi}{3}\) (jer se pomešaju uloge sinusa i kosinusa), dobija se pogrešan uslov za \(n\). Proverite: \(\cos\frac{\pi}{3} = \frac{1}{2}\), \(\sin\frac{\pi}{3} = \frac{\sqrt{3}}{2}\).
🎓 Izazov za razmišljanje

Za koje vrednosti \(n\) je broj \((1 + i\sqrt{3})^n\) čisto imaginaran (tj. realni deo jednak nuli)?

Prikaži nagoveštaj

Uslov je \(\cos\frac{n\pi}{3} = 0\), što znači \(\frac{n\pi}{3} = \frac{\pi}{2} + k\pi\). Rešite za \(n\) — da li uvek postoji celobrojno rešenje?
