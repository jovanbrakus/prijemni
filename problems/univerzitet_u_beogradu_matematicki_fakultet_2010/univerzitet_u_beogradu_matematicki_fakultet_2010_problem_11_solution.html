Znakovi koeficijenata kvadratne funkcije

Algebra — analiza grafika parabole i odredjivanje znakova koeficijenata \( a \), \( b \), \( c \)

Postavka zadatka

Grafik funkcije \( f(x) = ax^2 + bx + c \) prikazan je na slici. Na grafiku se vidi parabola koja je okrenuta nagore, ima teme u četvrtom kvadrantu (desno od \( y \)-ose i ispod \( x \)-ose), a \( y \)-osu seče ispod nule. Tačan je iskaz:

Plan rešavanja

Analiziraćemo grafik parabole korak po korak: smer otvaranja određuje znak koeficijenta \( a \), položaj temena na \( x \)-osi određuje znak koeficijenta \( b \), a presek sa \( y \)-osom daje znak koeficijenta \( c \).

Podsetnik iz teorije

Ključne formule i svojstva

Opšti oblik kvadratne funkcije:

\[ f(x) = ax^2 + bx + c \]

Smer otvaranja parabole:

  • Ako je \( a > 0 \), parabola je okrenuta nagore (ima minimum).
  • Ako je \( a < 0 \), parabola je okrenuta nadole (ima maksimum).

\( x \)-koordinata temena:

\[ x_T = -\frac{b}{2a} \]

Ako je \( x_T > 0 \), teme je desno od \( y \)-ose. Ako je \( x_T < 0 \), teme je levo od \( y \)-ose.

Presek sa \( y \)-osom:

\[ f(0) = c \]

Vrednost \( c \) direktno govori da li grafik seče \( y \)-osu iznad ili ispod \( x \)-ose.

Rešenje korak po korak

1
Određivanje znaka koeficijenta \( a \)

Posmatramo smer otvaranja parabole. Iz grafika vidimo da je parabola okrenuta nagore — ima oblik slova „U".

Ovo znači da funkcija ima minimum, što se dešava kada je vodeći koeficijent pozitivan:

\[ a > 0 \]

Ovo odmah eliminiše odgovor (E), jer u njemu stoji \( a < 0 \).

2
Određivanje znaka koeficijenta \( b \)

Teme parabole se nalazi desno od \( y \)-ose, tj. \( x \)-koordinata temena je pozitivna.

Formula za \( x \)-koordinatu temena je:

\[ x_T = -\frac{b}{2a} \]

Pošto je \( x_T > 0 \) i \( a > 0 \) (što smo već utvrdili), moramo imati:

\[ -\frac{b}{2a} > 0 \]

Množimo obe strane sa \( 2a > 0 \) (znak se ne menja jer je \( a > 0 \)):

\[ -b > 0 \quad \Longrightarrow \quad b < 0 \]

Ovo eliminiše odgovore (A) i (B), jer u njima stoji \( b > 0 \).

Proveri sebe: Ako je teme parabole levo od \( y \)-ose, a \( a > 0 \), koji je znak \( b \)?

Ako je \( x_T = -\frac{b}{2a} < 0 \) i \( a > 0 \), onda je \( -b < 0 \), tj. \( b > 0 \). Dakle, teme levo od \( y \)-ose uz \( a > 0 \) znači \( b > 0 \).

3
Određivanje znaka koeficijenta \( c \)

Koeficijent \( c \) je vrednost funkcije u nuli, jer:

\[ f(0) = a \cdot 0^2 + b \cdot 0 + c = c \]

Iz grafika vidimo da parabola seče \( y \)-osu ispod \( x \)-ose, što znači da je \( f(0) < 0 \):

\[ c < 0 \]

Ovo eliminiše odgovor (D), jer u njemu stoji \( c > 0 \).

Proveri sebe: Koji odgovor je preostao?

Eliminisali smo (E) u koraku 1, zatim (A) i (B) u koraku 2, i konačno (D) u koraku 3. Jedini preostali odgovor je (C): \( a > 0,\; b < 0,\; c < 0 \).

4
Verifikacija pomoću konkretnog primera

Proverimo odgovor na konkretnom primeru. Neka je \( f(x) = x^2 - 2x - 3 \). Tada:

\[ a = 1 > 0, \quad b = -2 < 0, \quad c = -3 < 0 \]

Teme parabole se nalazi u:

\[ x_T = -\frac{-2}{2 \cdot 1} = 1 > 0 \quad \checkmark \]

Vrednost u temenu je:

\[ f(1) = 1 - 2 - 3 = -4 < 0 \quad \checkmark \]

Presek sa \( y \)-osom:

\[ f(0) = -3 < 0 \quad \checkmark \]

Sva svojstva se poklapaju sa grafikom: parabola nagore, teme u IV kvadrantu, \( y \)-presek ispod nule.

Proveri sebe: Gde su nule funkcije \( f(x) = x^2 - 2x - 3 \)?

\( f(x) = (x - 3)(x + 1) = 0 \), pa su nule \( x_1 = 3 \) i \( x_2 = -1 \). Jedna nula je pozitivna, a druga negativna, što je u skladu sa činjenicom da je \( c < 0 \) i \( a > 0 \) (jer je proizvod nula \( \frac{c}{a} = -3 < 0 \)).

Ključni uvid

Svaki od koeficijenata \( a \), \( b \), \( c \) kontroliše jedno vizuelno svojstvo parabole:

  • \( a \) — smer otvaranja (nagore ili nadole)
  • \( b \) — u kombinaciji sa \( a \), položaj temena levo/desno od \( y \)-ose
  • \( c \) — visina preseka sa \( y \)-osom

Čitanje grafika „jednu osobinu po jednu" omogućava sistematsku eliminaciju pogrešnih odgovora.

\[ a > 0, \quad b < 0, \quad c < 0 \]

Konačan odgovor

\( a > 0, \quad b < 0, \quad c < 0 \)
(A) \( a > 0, b > 0, c < 0 \)
(B) \( a > 0, b > 0, c > 0 \)
(C) \( a > 0, b < 0, c < 0 \)
(D) \( a > 0, b < 0, c > 0 \)
(E) \( a < 0, b < 0, c < 0 \)

Verifikacija: Parabola okrenuta nagore (\( a > 0 \)), teme desno od \( y \)-ose (\( b < 0 \) jer \( -b/(2a) > 0 \)), presek sa \( y \)-osom ispod nule (\( c < 0 \)). Sve se poklapa sa opisom grafika.

Česte greške

✗

Mešanje znaka \( b \) i položaja temena: Česta greška je zaključiti da „teme desno znači \( b > 0 \)". Zapravo, \( x_T = -\frac{b}{2a} \), pa za \( a > 0 \) i \( x_T > 0 \) mora biti \( b < 0 \). Minus u formuli „preokreće" znak!

✗

Zaboravljanje da je \( c = f(0) \): Neki učenici pokušavaju da odrede \( c \) iz diskriminante ili drugih formula. Najjednostavniji pristup: \( c \) je bukvalno vrednost funkcije za \( x = 0 \), tj. tačka preseka grafika sa \( y \)-osom.

Dodatni izazov

Ako bi parabola imala isto teme (u IV kvadrantu) i isti presek sa \( y \)-osom (ispod nule), ali bila okrenuta nadole, koji bi se znakovi koeficijenata promenili?

Pogledaj nagoveštaj

Ako je parabola okrenuta nadole, onda je \( a < 0 \). Teme i dalje desno od \( y \)-ose znači \( x_T = -\frac{b}{2a} > 0 \), ali sada je \( a < 0 \), pa množenje sa \( 2a < 0 \) menja znak nejednakosti: \( -b < 0 \Rightarrow b > 0 \). Presek sa \( y \)-osom i dalje ispod nule: \( c < 0 \). Dakle odgovor bi bio: \( a < 0, b > 0, c < 0 \).
