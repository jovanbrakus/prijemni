Minimum funkcije cos⁴x + sin⁴x

Trigonometrijski identiteti i ekstremne vrednosti

📜 Zadatak

Najmanji realan broj \(a\) za koji jednačina \(\cos^4 x + \sin^4 x = a\) ima rešenje jednak je:

(A)
\(2\)
(B)
\(\sqrt{2}\)
(C)
\(1\)
(D)
\(\dfrac{1}{\sqrt{2}}\)
(E)
\(\dfrac{1}{2}\)
📌 Plan rešavanja

Transformisaćemo izraz \(\cos^4 x + \sin^4 x\) koristeći identitet za zbir kvadrata i formulu dvostrukog ugla, čime ćemo dobiti izraz čiji je opseg lako odrediti. Zatim nalazimo minimum.

📐 Podsetnik: Ključne formule i identiteti

Pitagorin trigonometrijski identitet:

\[\sin^2 x + \cos^2 x = 1\]

Zbir kvadrata kao kvadrat zbira:

\[a^2 + b^2 = (a + b)^2 - 2ab\]

Formula dvostrukog ugla za sinus:

\[\sin 2x = 2\sin x \cos x \quad \Rightarrow \quad \sin^2 2x = 4\sin^2 x \cos^2 x\]

Opseg funkcije \(\sin^2\):

\[0 \leq \sin^2(2x) \leq 1 \quad \text{za svako } x \in \mathbb{R}\]
📊 Grafički prikaz

Grafik funkcije \(f(x) = \cos^4 x + \sin^4 x\) na intervalu \([0,\, 2\pi]\). Isprekidane linije prikazuju minimum i maksimum.

Minimum se dostiže za \(x = \frac{\pi}{4} + k\frac{\pi}{2}\)

✏️ Rešenje korak po korak
Korak 1: Primena identiteta za zbir kvadrata

Polazimo od izraza \(\cos^4 x + \sin^4 x\). Primetimo da je ovo zbir kvadrata, gde su \(a = \cos^2 x\) i \(b = \sin^2 x\).

Koristimo formulu \(a^2 + b^2 = (a+b)^2 - 2ab\):

\[\cos^4 x + \sin^4 x = (\cos^2 x + \sin^2 x)^2 - 2\cos^2 x \cdot \sin^2 x\]
Korak 2: Primena Pitagorinog identiteta

Znamo da je \(\cos^2 x + \sin^2 x = 1\), pa je:

\[\cos^4 x + \sin^4 x = 1^2 - 2\cos^2 x \sin^2 x = 1 - 2\cos^2 x \sin^2 x\]
Korak 3: Uvođenje dvostrukog ugla

Iz formule \(\sin 2x = 2\sin x \cos x\) sledi:

\[\sin^2 2x = 4\sin^2 x \cos^2 x \quad \Rightarrow \quad \sin^2 x \cos^2 x = \frac{\sin^2 2x}{4}\]

Zamenjujemo u naš izraz:

\[\cos^4 x + \sin^4 x = 1 - 2 \cdot \frac{\sin^2 2x}{4} = 1 - \frac{\sin^2 2x}{2}\]
✅ Proveri se: Kolika je vrednost izraza za \(x = 0\)?

Za \(x = 0\): \(\sin 0 = 0\), pa \(\sin^2(2 \cdot 0) = 0\).

Izraz: \(1 - \frac{0}{2} = 1\). Direktno: \(\cos^4 0 + \sin^4 0 = 1 + 0 = 1\). Tačno!

Korak 4: Određivanje opsega funkcije

Funkcija \(\sin^2 2x\) uzima vrednosti iz intervala \([0,\, 1]\).

Kada je \(\sin^2 2x = 0\) (tj. \(x = 0,\, \frac{\pi}{2},\, \pi, \ldots\)):

\[f(x) = 1 - \frac{0}{2} = 1 \quad \text{(maksimum)}\]

Kada je \(\sin^2 2x = 1\) (tj. \(x = \frac{\pi}{4},\, \frac{3\pi}{4}, \ldots\)):

\[f(x) = 1 - \frac{1}{2} = \frac{1}{2} \quad \text{(minimum)}\]
✅ Proveri se: Kolika je vrednost za \(x = \frac{\pi}{4}\)?

\(\cos\frac{\pi}{4} = \sin\frac{\pi}{4} = \frac{\sqrt{2}}{2}\)

\(\cos^4\frac{\pi}{4} + \sin^4\frac{\pi}{4} = \left(\frac{\sqrt{2}}{2}\right)^4 + \left(\frac{\sqrt{2}}{2}\right)^4 = \frac{1}{4} + \frac{1}{4} = \frac{1}{2}\)

Potvrđeno — minimum je zaista \(\frac{1}{2}\).

Korak 5: Zaključak

Funkcija \(f(x) = \cos^4 x + \sin^4 x\) uzima vrednosti iz intervala \(\left[\frac{1}{2},\, 1\right]\).

Dakle, jednačina \(\cos^4 x + \sin^4 x = a\) ima rešenje ako i samo ako:

\[\frac{1}{2} \leq a \leq 1\]

Najmanji takav broj \(a\) je \(\frac{1}{2}\).

💡 Ključni uvid

Trik je u transformaciji zbira četvrtih stepeni u izraz koji zavisi samo od \(\sin^2 2x\):

\[\cos^4 x + \sin^4 x = 1 - \frac{\sin^2 2x}{2}\]

Pošto \(\sin^2 2x\) osciluje između 0 i 1, ceo izraz osciluje između \(\frac{1}{2}\) i \(1\). Ova transformacija pretvara naizgled složen problem u jednostavnu analizu opsega.

Konačan odgovor
(E) \(\; a_{\min} = \dfrac{1}{2}\)
(A) \(2\) (B) \(\sqrt{2}\) (C) \(1\) (D) \(\frac{1}{\sqrt{2}}\) (E) \(\frac{1}{2}\)

Verifikacija: Za \(x = \frac{\pi}{4}\) direktno dobijamo \(\frac{1}{4} + \frac{1}{4} = \frac{1}{2}\), a vrednost \(a < \frac{1}{2}\) je nemoguća jer bi \(\sin^2 2x\) moralo biti veće od 1.

⚠️ Česte greške
Biranje odgovora \(a = 1\): Vrednost 1 je maksimum funkcije \(\cos^4 x + \sin^4 x\), ne minimum. Zadatak traži najmanji broj \(a\) za koji jednačina ima rešenje, a to je upravo minimum funkcije.
Zaborav formule dvostrukog ugla: Neki učenici pokušavaju da direktno analiziraju \(\cos^4 x + \sin^4 x\) bez transformacije, što značajno otežava određivanje opsega. Ključna je veza \(2\sin^2 x \cos^2 x = \frac{\sin^2 2x}{2}\).
🏆 Dodatni izazov

Odredi opseg funkcije \(f(x) = \cos^6 x + \sin^6 x\). Za koje vrednosti \(a\) jednačina \(\cos^6 x + \sin^6 x = a\) ima rešenje?

Savet

Koristi identitet za zbir kubova: \(a^3 + b^3 = (a + b)(a^2 - ab + b^2)\) sa \(a = \cos^2 x\), \(b = \sin^2 x\). Dobićeš \(1 - \frac{3}{4}\sin^2 2x\), pa je opseg \(\left[\frac{1}{4},\, 1\right]\).
