Racionalna funkcija sa tri uslova

Racionalne funkcije — sistem jednacina sa parametrima

Tekst zadatka

Funkcija \(f\) je zadata sa \(f(x) = \dfrac{ax + b}{cx + d}\), gde su \(a\), \(b\), \(c\) i \(d\) realni brojevi. Ako je \(f(0) = 1\), \(f(1) = 0\) i \(f(2) = 3\), koliko je \(f(3)\)?

A) −1
B) 3/2
C) 5
D) 2
E) 3
Plan resavanja

Iskoristicemo tri data uslova \(f(0)=1\), \(f(1)=0\) i \(f(2)=3\) da postavimo sistem jednacina sa cetiri nepoznate \(a, b, c, d\). Posto je funkcija razlomak, bitan nam je samo odnos koeficijenata, pa mozemo izraziti sve preko jednog parametra, a zatim izracunati \(f(3)\).

Podsetnik iz teorije
Racionalne (razlomljene) funkcije

Funkcija oblika \(f(x) = \dfrac{ax + b}{cx + d}\) naziva se Mebijusova (linearna razlomljena) transformacija. Ima vertikalnu asimptotu u \(x = -\dfrac{d}{c}\) i horizontalnu asimptotu \(y = \dfrac{a}{c}\).

Kljucno: ako pomnozimo brojilac i imenilac istim brojem razlicitim od nule, funkcija se ne menja. Zato od cetiri parametra \(a, b, c, d\) efektivno imamo tri stepena slobode — dovoljno za tri uslova.

Sistem jednacina sa parametrima

Kada imamo vise nepoznatih, ali i mogucnost da jednu izaberemo slobodno (jer nas zanima samo odnos), tada korak po korak eliminisemo nepoznate kroz zamenu (supstituciju).

Vizuelni prikaz
Resenje korak po korak
1 Uslov \(f(0) = 1\)

Uvrstimo \(x = 0\) u \(f(x) = \dfrac{ax+b}{cx+d}\):

\[f(0) = \frac{a \cdot 0 + b}{c \cdot 0 + d} = \frac{b}{d} = 1\]

Odavde zakljucujemo:

\[b = d\]
2 Uslov \(f(1) = 0\)

Uvrstimo \(x = 1\):

\[f(1) = \frac{a + b}{c + d} = 0\]

Razlomak je nula ako i samo ako je brojilac nula (a imenilac razlicit od nule):

\[a + b = 0 \quad \Longrightarrow \quad a = -b\]
Proveri sebe: Zasto je dovoljno da brojilac bude nula?

Razlomak \(\dfrac{P}{Q} = 0\) vazi ako i samo ako je \(P = 0\) i \(Q \neq 0\). Posto je \(c + d\) imenilac i pretpostavljamo da funkcija ima smisla u tacki \(x = 1\), vazi \(c + d \neq 0\).

3 Uslov \(f(2) = 3\) — nalazenje \(c\)

Vec znamo \(a = -b\) i \(d = b\). Uvrstimo \(x = 2\):

\[f(2) = \frac{2a + b}{2c + d} = \frac{2(-b) + b}{2c + b} = \frac{-b}{2c + b} = 3\]

Resimo po \(c\):

\[-b = 3(2c + b)\] \[-b = 6c + 3b\] \[-4b = 6c\] \[c = -\frac{2b}{3}\]
4 Formiranje konacnog oblika funkcije

Sada znamo sve koeficijente izrazene preko \(b\):

\[a = -b, \quad b = b, \quad c = -\frac{2b}{3}, \quad d = b\]

Uvrstimo u \(f(x)\):

\[f(x) = \frac{-bx + b}{-\dfrac{2b}{3}\,x + b} = \frac{b(-x + 1)}{b\!\left(-\dfrac{2x}{3} + 1\right)} = \frac{1 - x}{1 - \dfrac{2x}{3}}\]

Pomnozimo brojilac i imenilac sa \(3\) da bismo uklonili razlomak u imeniocu:

\[f(x) = \frac{3(1 - x)}{3 - 2x}\]
Proveri sebe: Sta se desilo sa \(b\)?

Parametar \(b\) se skratio jer se pojavljuje i u brojiocu i u imeniocu kao zajednicki faktor. To je ocekivano: kod razlomljene linearne funkcije bitan je samo odnos koeficijenata, ne njihove apsolutne vrednosti.

5 Racunanje \(f(3)\)
\[f(3) = \frac{3(1 - 3)}{3 - 2 \cdot 3} = \frac{3 \cdot (-2)}{3 - 6} = \frac{-6}{-3} = 2\]
6 Verifikacija svih uslova

Proverimo da nasa funkcija \(f(x) = \dfrac{3(1-x)}{3-2x}\) zaista zadovoljava sve tri polazne vrednosti:

\[f(0) = \frac{3(1-0)}{3-0} = \frac{3}{3} = 1 \;\checkmark\] \[f(1) = \frac{3(1-1)}{3-2} = \frac{0}{1} = 0 \;\checkmark\] \[f(2) = \frac{3(1-2)}{3-4} = \frac{-3}{-1} = 3 \;\checkmark\]
Kljucni uvid

Kod razlomljene linearne funkcije \(\dfrac{ax+b}{cx+d}\) postoje cetiri parametra, ali mnozenje svih sa istom konstantom ne menja funkciju. Zato imamo tri efektivna stepena slobode — i tri uslova su dovoljni da potpuno odrede funkciju. Nakon toga, vrednost u bilo kojoj tacki je jedinstveno odredjena.

Konacan odgovor
\(f(3) = 2\)
A) −1
B) 3/2
C) 5
D) 2
E) 3

Verifikacija: \(f(x) = \dfrac{3(1-x)}{3-2x}\), pa \(f(3) = \dfrac{3 \cdot (-2)}{-3} = 2\).

Ceste greske
!

Zaboravljanje da se \(b\) skracuje: Mnogi ucenici pokusavaju da nadju konkretnu vrednost za \(b\), sto je nemoguce jer je slobodan parametar. Kljuc je uociti da se skracuje u razlomku.

!

Greska pri radu sa dvema negativnim vrednostima: U koraku \(f(3) = \dfrac{-6}{-3}\), greska sa predznakom vodi do \(-2\) umesto \(2\). Dvostruki minus daje plus!

!

Pogresno postavljanje uslova \(f(1) = 0\): Uslov \(\dfrac{a+b}{c+d} = 0\) znaci da je brojilac nula, tj. \(a + b = 0\). Nije ispravno izjednacavati imenilac sa nulom.

Dodatni izazov

Za istu vrstu funkcije \(g(x) = \dfrac{ax+b}{cx+d}\), ako je \(g(0) = 2\), \(g(1) = 1\) i \(g(3) = -1\), koliko je \(g(2)\)?

Pomoc

Iz \(g(0) = 2\): \(b = 2d\). Iz \(g(1) = 1\): \(a + b = c + d\), tj. \(a + 2d = c + d\), pa \(a = c - d\).

Iz \(g(3) = -1\): \(\dfrac{3a + b}{3c + d} = -1\), tj. \(3a + b = -(3c + d)\). Uvrstavanjem \(a = c - d\) i \(b = 2d\): \(3(c-d) + 2d = -3c - d\), pa \(3c - 3d + 2d = -3c - d\), tj. \(3c - d = -3c - d\), odakle \(6c = 0\), pa \(c = 0\).

Tada: \(a = -d\), \(b = 2d\), \(c = 0\), i \(g(x) = \dfrac{-dx + 2d}{d} = -x + 2\). Provera: \(g(2) = 0\).
