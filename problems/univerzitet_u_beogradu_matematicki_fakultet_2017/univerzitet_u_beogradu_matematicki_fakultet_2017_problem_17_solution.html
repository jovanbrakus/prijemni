Kompleksni brojevi — jednačina sa modulom

Algebarski pristup rešavanju jednačine sa kompleksnim brojem i modulom

Postavka zadatka

Neka je \(z\) kompleksan broj za koji važi:

\[\frac{\sqrt{2}}{z + i} = -|z|\]

Tada je \(2z - \bar{z}\) jednako:

(A)
\(-1 - 3i\)
(B)
\(6 - i\)
(C)
\(2 - 3i\)
(D)
\(1 - i\)
(E)
nije jednoznačno određeno

Plan rešavanja

Zapišemo \(z = a + bi\) i iskoristimo činjenicu da je desna strana jednačine realan broj, pa i leva strana mora biti realna. Iz tog uslova ćemo odrediti imaginarni deo broja \(z\), a zatim i realni deo kvadriranjem jednačine.

Ključne formule i teoreme
  • Algebarski oblik kompleksnog broja: \(z = a + bi\), gde su \(a, b \in \mathbb{R}\), \(\bar{z} = a - bi\)
  • Modul kompleksnog broja: \(|z| = \sqrt{a^2 + b^2}\), uvek je \(|z| \geq 0\)
  • Količnik realnog i kompleksnog broja: \(\frac{r}{z}\) je realan ako i samo ako je \(z\) realan broj (za \(r \in \mathbb{R}\))
  • Linearna kombinacija: \(2z - \bar{z} = 2(a+bi) - (a-bi) = a + 3bi\)

Vizuelni prikaz

Kompleksna ravan sa tačkom \(z = -1 - i\)

Rešenje korak po korak

1

Zapišemo \(z\) u algebarskom obliku

Neka je \(z = a + bi\), gde su \(a, b \in \mathbb{R}\). Tada:

\[|z| = \sqrt{a^2 + b^2}, \quad z + i = a + (b+1)i, \quad \bar{z} = a - bi\]
2

Analiziramo realnost obe strane jednačine

Desna strana jednačine, \(-|z|\), je realan broj (jer je \(|z|\) uvek realan).

Dakle, i leva strana \(\frac{\sqrt{2}}{z+i}\) mora biti realan broj.

Pošto je \(\sqrt{2}\) realan, to zahteva da \(z + i\) bude realan broj.

\[z + i = a + (b+1)i \in \mathbb{R} \implies b + 1 = 0 \implies b = -1\]
Proveri se: Ako je \(b = -1\), koliko iznosi \(z + i\)?
\(z + i = a + (-1+1)i = a\). Dakle, \(z + i\) je čist realan broj!
3

Unesemo \(b = -1\) u jednačinu

Sada je \(z = a - i\), pa \(|z| = \sqrt{a^2 + 1}\) i \(z + i = a\).

Jednačina postaje:

\[\frac{\sqrt{2}}{a} = -\sqrt{a^2 + 1}\]

Desna strana je uvek negativna (ili nula), pa mora biti \(a < 0\) da bi i leva strana bila negativna.

4

Kvadriramo obe strane

Obe strane su negativne, pa je kvadriranje ekvivalentna transformacija:

\[\frac{2}{a^2} = a^2 + 1\]

Pomnožimo sa \(a^2\):

\[2 = a^4 + a^2\]
5

Uvedemo smenu \(u = a^2\)

Dobijamo kvadratnu jednačinu po \(u\):

\[u^2 + u - 2 = 0\]

Faktorizujemo:

\[(u + 2)(u - 1) = 0\]

Rešenja su \(u = -2\) i \(u = 1\).

Pošto je \(u = a^2 \geq 0\), odbacujemo \(u = -2\). Dakle \(a^2 = 1\).

Proveri se: Zašto biramo \(a = -1\) a ne \(a = 1\)?
Iz koraka 3, zaključili smo da mora biti \(a < 0\) (jer je leva strana \(\frac{\sqrt{2}}{a}\) negativna). Zato je jedino rešenje \(a = -1\).
6

Odredimo \(z\)

Iz \(a^2 = 1\) i uslova \(a < 0\) sledi \(a = -1\).

\[z = -1 - i\]
7

Izračunamo \(2z - \bar{z}\)

Konjugovano od \(z = -1 - i\) je \(\bar{z} = -1 + i\). Sada:

\[2z - \bar{z} = 2(-1 - i) - (-1 + i)\]
\[= -2 - 2i + 1 - i = -1 - 3i\]
Proveri se: Da li početna jednačina važi za \(z = -1 - i\)?
\(|z| = \sqrt{1 + 1} = \sqrt{2}\), \(z + i = -1\). Leva strana: \(\frac{\sqrt{2}}{-1} = -\sqrt{2}\). Desna strana: \(-|z| = -\sqrt{2}\). Jednačina važi!

Ključni uvid

Trik je u tome što realnost desne strane odmah daje jak uslov: imaginarni deo od \(z + i\) mora biti nula. To eliminiše jednu nepoznatu i svodi problem na jednačinu sa jednom realnom promenljivom.

\[z = -1 - i \implies 2z - \bar{z} = -1 - 3i\]

Konačan odgovor

\(2z - \bar{z} = -1 - 3i\)
(A) \(-1 - 3i\)
(B) \(6 - i\)
(C) \(2 - 3i\)
(D) \(1 - i\)
(E) nije jednoznačno određeno

Verifikacija: Za \(z = -1 - i\) važi \(\frac{\sqrt{2}}{z+i} = \frac{\sqrt{2}}{-1} = -\sqrt{2} = -|z|\). Jednačina je zadovoljena, a rešenje je jedinstveno, pa opcija (E) nije tačna.

Česte greške

  • Zaboraviti uslov \(a < 0\): Ako se ne proveri znak od \(a\), dobija se i lažno rešenje \(a = 1\), tj. \(z = 1 - i\), koje ne zadovoljava originalnu jednačinu (leva strana bi bila pozitivna, a desna negativna).
  • Ne proveriti realnost: Mnogi studenti odmah kvadriraju jednačinu bez analize realnosti, što uvodi nepotrebne komplikacije sa imaginarnim delom.

Dodatni izazov

Šta bi se desilo ako bi jednačina glasila \(\frac{\sqrt{2}}{z + i} = |z|\) (bez minusa)? Da li bi postojalo rešenje?

Nagovještaj

Isti postupak: \(b = -1\), ali sada je \(\frac{\sqrt{2}}{a} = \sqrt{a^2+1} > 0\), pa mora biti \(a > 0\). To daje \(a = 1\) i \(z = 1 - i\).
