Trigonometrijska jednacina - broj resenja

Kompozicija trigonometrijskih funkcija, analiza nula sinusne funkcije i odredjivanje broja resenja na datom intervalu

📋 Tekst zadatka

Broj resenja jednacine \(\sin(2\cos x + 2) = 0\) u intervalu \([0, 2\pi]\) je:

(A)
veci od 4
(B)
\(1\)
(C)
\(2\)
(D)
\(4\)
(E)
\(3\)

🧭 Plan resavanja

Iskoristicemo cinjenicu da je \(\sin(\theta) = 0\) tacno kada je \(\theta = n\pi\) za neko celo \(n\). Postavicemo uslov \(2\cos x + 2 = n\pi\), odrediti koje celobrojne vrednosti \(n\) daju \(\cos x \in [-1, 1]\), i za svako takvo \(n\) prebrojati resenja na intervalu \([0, 2\pi]\).

📐 Kljucne formule i teoreme
  • Nule sinusne funkcije: \(\sin(\theta) = 0 \iff \theta = n\pi, \; n \in \mathbb{Z}\)
  • Opseg kosinusne funkcije: Za svako realno \(x\) vazi \(-1 \leq \cos x \leq 1\)
  • Broj resenja jednacine \(\cos x = c\) na \([0, 2\pi]\):
    • Ako je \(-1 < c < 1\): tacno 2 resenja (\(x = \arccos c\) i \(x = 2\pi - \arccos c\))
    • Ako je \(c = 1\): tacno 1 resenje (\(x = 0\), ali posto je interval zatvoren, i \(x = 2\pi\) daje istu vrednost)
    • Ako je \(c = -1\): tacno 1 resenje (\(x = \pi\))
    • Ako je \(|c| > 1\): nema resenja

📝 Resenje korak po korak

1

Postavljamo uslov za nule sinusa

Jednacina glasi:

\[\sin(2\cos x + 2) = 0\]

Sinusna funkcija je jednaka nuli kada je njen argument celobrojni umnozak broja \(\pi\):

\[\sin(\theta) = 0 \iff \theta = n\pi, \quad n \in \mathbb{Z}\]

Stoga nas uslov postaje:

\[2\cos x + 2 = n\pi\]
2

Izrazavamo \(\cos x\) preko \(n\)

Iz jednacine \(2\cos x + 2 = n\pi\) izrazavamo kosinus:

\[\cos x = \frac{n\pi - 2}{2}\]
3

Odredjujemo dozvoljene vrednosti \(n\)

Posto \(\cos x\) uvek pripada intervalu \([-1, 1]\), mora da vazi:

\[-1 \leq \frac{n\pi - 2}{2} \leq 1\]

Mnozenjem celom nejednakoscu sa \(2\):

\[-2 \leq n\pi - 2 \leq 2\]

Dodavanjem broja \(2\) svuda:

\[0 \leq n\pi \leq 4\]

Deljenjem sa \(\pi \approx 3{,}14159\):

\[0 \leq n \leq \frac{4}{\pi} \approx 1{,}273\]

Posto je \(n\) ceo broj, jedine moguce vrednosti su:

\[n = 0 \quad \text{ili} \quad n = 1\]
✅ Proveri sebe: Zasto \(n = -1\) i \(n = 2\) ne zadovoljavaju uslov?

Za \(n = -1\): \(\cos x = \frac{-\pi - 2}{2} \approx -2{,}57\). Posto je \(-2{,}57 < -1\), ovo je van opsega kosinusa.
Za \(n = 2\): \(\cos x = \frac{2\pi - 2}{2} \approx 2{,}14\). Posto je \(2{,}14 > 1\), i ovo je van opsega kosinusa.

4

Slucaj \(n = 0\): koliko resenja?

Za \(n = 0\) dobijamo:

\[\cos x = \frac{0 \cdot \pi - 2}{2} = \frac{-2}{2} = -1\]

Jednacina \(\cos x = -1\) na intervalu \([0, 2\pi]\) ima tacno jedno resenje:

\[x = \pi\]
5

Slucaj \(n = 1\): koliko resenja?

Za \(n = 1\) dobijamo:

\[\cos x = \frac{\pi - 2}{2} \approx \frac{1{,}1416}{2} \approx 0{,}5708\]

Proveravamo: \(-1 < 0{,}5708 < 1\), dakle ova vrednost jeste u opsegu kosinusa.

Jednacina \(\cos x = c\) za \(-1 < c < 1\) na intervalu \([0, 2\pi]\) ima tacno dva resenja:

\[x_1 = \arccos\!\left(\frac{\pi - 2}{2}\right) \approx 0{,}963\]
\[x_2 = 2\pi - \arccos\!\left(\frac{\pi - 2}{2}\right) \approx 5{,}320\]

Oba resenja su u intervalu \([0, 2\pi]\).

✅ Proveri sebe: Koliko ukupno resenja imamo?

Od \(n = 0\): \(1\) resenje. Od \(n = 1\): \(2\) resenja. Ukupno: \(1 + 2 = 3\) resenja.

6

Sabiramo sva resenja

Prebrojimo sva resenja jednacine \(\sin(2\cos x + 2) = 0\) na intervalu \([0, 2\pi]\):

Vrednost \(n\) Vrednost \(\cos x\) Broj resenja za \(x\)
\(n = 0\) \(\cos x = -1\) 1 (tacno \(x = \pi\))
\(n = 1\) \(\cos x = \frac{\pi - 2}{2} \approx 0{,}57\) 2
\[\text{Ukupan broj resenja} = 1 + 2 = \boxed{3}\]
7

Verifikacija svih resenja

Resenje \(x = \pi\) (od \(n = 0\)):

\[\sin(2\cos\pi + 2) = \sin(2 \cdot (-1) + 2) = \sin(0) = 0 \; \checkmark\]

Resenja \(x_1\) i \(x_2\) (od \(n = 1\)), gde je \(\cos x = \frac{\pi - 2}{2}\):

\[\sin\!\left(2 \cdot \frac{\pi - 2}{2} + 2\right) = \sin(\pi - 2 + 2) = \sin(\pi) = 0 \; \checkmark\]

Sva tri resenja su potvrdjena.

✅ Proveri sebe: Sta bi bilo da je umesto \(\sin\) bila funkcija \(\cos\)?

Tada bi uslov bio \(2\cos x + 2 = \frac{\pi}{2} + n\pi\), sto vodi na razlicite vrednosti za \(\cos x\) i potencijalno drugaciji broj resenja. Pokusajte sami!

💡 Kljucni uvid

Ovaj zadatak je u sustini zadatak o prebrojavanju. Umesto da trazimo tacne vrednosti \(x\), dovoljno je:

  1. Svesti jednacinu na oblik \(\cos x = c_n\) za razne celobrojne \(n\),
  2. Proveriti koji \(c_n\) pripadaju intervalu \([-1, 1]\),
  3. Prebrojati resenja koristeci pravilo: \(\cos x = c\) ima 2 resenja ako \(|c| < 1\), a 1 resenje ako \(|c| = 1\).
\[n = 0 \Rightarrow 1 \text{ resenje}, \quad n = 1 \Rightarrow 2 \text{ resenja} \quad \Longrightarrow \quad \text{Ukupno: } 3\]

✅ Konacan odgovor

\(\text{E) } 3\)

(A) veci od 4
(B) 1
(C) 2
(D) 4
(E) 3
Provera: Sve tri tacke smo direktno uvrstili u jednacinu i potvrdili da daju \(\sin(\ldots) = 0\). Takodje, grafik funkcije na dijagramu iznad jasno pokazuje tacno 3 preseka sa x-osom na intervalu \([0, 2\pi]\).

⚠ Ceste greske

  • Zaboravljanje da \(\cos x = -1\) daje samo jedno resenje: Ucenici cesto automatski broje po 2 resenja za svako \(n\), ali za \(\cos x = -1\) postoji samo jedno resenje \(x = \pi\).
  • Ispitivanje samo pozitivnih \(n\): Iako u ovom zadatku negativne vrednosti \(n\) ne daju validna resenja, uvek treba proveriti i negativne vrednosti. Neispitivanje moze dovesti do propustanja resenja u slicnim zadacima.
  • Brkanje argumenta: Jednacina nije \(\sin(2\cos x) + 2 = 0\) (sto bi bilo \(\sin(2\cos x) = -2\), nemoguce), vec \(\sin(2\cos x + 2) = 0\). Pazljivo citajte zagrade!

🎯 Dodatni izazov

Koliko resenja ima jednacina \(\sin(3\cos x + 3) = 0\) na intervalu \([0, 2\pi]\)?

Pogledaj nagovestaj

Postupak je isti: \(3\cos x + 3 = n\pi\), pa \(\cos x = \frac{n\pi - 3}{3}\). Uslov \(-1 \leq \frac{n\pi - 3}{3} \leq 1\) daje \(\frac{3 - 3}{\pi} \leq n \leq \frac{3 + 3}{\pi}\), tj. \(0 \leq n \leq \frac{6}{\pi} \approx 1{,}91\). Dakle opet \(n \in \{0, 1\}\). Za \(n=0\): \(\cos x = -1\) (1 resenje). Za \(n=1\): \(\cos x = \frac{\pi - 3}{3} \approx 0{,}047\) (2 resenja). Ukupno opet 3!
