Znaci koeficijenata kvadratne funkcije na osnovu grafika

Algebra — kvadratna funkcija, graficka analiza koeficijenata, svojstva parabole

Tekst zadatka

Na slici je prikazan grafik funkcije

\[ f(x) = ax^2 + bx + c \]

Parabola se otvara navise, teme joj se nalazi desno od \(y\)-ose, a grafik sece \(y\)-osu ispod \(x\)-ose. Tacan je iskaz:

(A)
\( a > 0,\; b > 0,\; c < 0 \)
(B)
\( a > 0,\; b > 0,\; c > 0 \)
(C)
\( a > 0,\; b < 0,\; c < 0 \)
(D)
\( a > 0,\; b < 0,\; c > 0 \)
(E)
\( a < 0,\; b < 0,\; c < 0 \)

Plan resavanja

Analiziracemo grafik korak po korak: iz smera otvaranja parabole odredimo znak koeficijenta \( a \), iz polozaja temena u odnosu na \( y \)-osu odredimo znak koeficijenta \( b \), a iz preseka grafika sa \( y \)-osom odredimo znak slobodnog clana \( c \). Na kraju eliminisemo netacne opcije.

Podsetnik iz teorije

Kljucne formule i svojstva
  • Smer otvaranja parabole: Ako je \( a > 0 \), parabola se otvara navise. Ako je \( a < 0 \), otvara se nadole.
  • \(x\)-koordinata temena: \[ x_T = -\frac{b}{2a} \] Ako je \( x_T > 0 \), teme je desno od \( y \)-ose. Ako je \( x_T < 0 \), teme je levo.
  • Presek sa \( y \)-osom: Tacka preseka grafika sa \( y \)-osom je \( (0,\, c) \), jer: \[ f(0) = a \cdot 0^2 + b \cdot 0 + c = c \] Ako je \( c < 0 \), presek je ispod \( x \)-ose. Ako je \( c > 0 \), presek je iznad.
  • Veza znaka \( b \) i polozaja temena: Iz formule \( x_T = -\frac{b}{2a} \) i poznatog znaka \( a \) moze se odrediti znak \( b \). Na primer, ako \( a > 0 \) i \( x_T > 0 \), onda \( b < 0 \).

Vizuelni prikaz

-3,0

Pomeri klizac da vidis kako promena koeficijenta \( b \) pomera teme parabole levo-desno (koeficijenti \( a = 1 \), \( c = -2 \) su fiksirani)

Resenje korak po korak

1

Odredjujemo znak koeficijenta \( a \)

Posmatramo smer otvaranja parabole. Prema opisu (i slici), parabola se otvara navise.

Kod kvadratne funkcije \( f(x) = ax^2 + bx + c \), parabola se otvara navise ako i samo ako je vodeci koeficijent pozitivan:

\[ a > 0 \]

Ovim odmah eliminisemo opciju E, gde je navedeno \( a < 0 \). Preostale opcije (A, B, C, D) sve imaju \( a > 0 \).

2

Odredjujemo znak koeficijenta \( b \) iz polozaja temena

\(x\)-koordinata temena parabole data je formulom:

\[ x_T = -\frac{b}{2a} \]

Sa slike vidimo da se teme nalazi desno od \( y \)-ose, sto znaci:

\[ x_T > 0 \quad \Longrightarrow \quad -\frac{b}{2a} > 0 \]
3

Odredjujemo znak razlomka

Posto smo utvrdili da je \( a > 0 \), imenilac \( 2a \) je pozitivan. Nejednacina postaje:

\[ -\frac{b}{2a} > 0 \]

Pomnozimo obe strane sa \( 2a > 0 \) (znak nejednakosti se ne menja):

\[ -b > 0 \]

Odavde direktno sledi:

\[ b < 0 \]

Ovim eliminisemo opcije A i B, gde je navedeno \( b > 0 \). Preostaju opcije C i D.

Proveri sebe: Ako bi teme bilo levo od \( y \)-ose (uz \( a > 0 \)), kakav bi bio znak \( b \)?

Ako je \( x_T < 0 \) i \( a > 0 \), onda bi \( -\frac{b}{2a} < 0 \), pa \( -b < 0 \), odnosno \( b > 0 \). Dakle, teme levo od \( y \)-ose pri \( a > 0 \) znaci \( b > 0 \).

4

Odredjujemo znak koeficijenta \( c \) iz preseka sa \( y \)-osom

Presek grafika sa \( y \)-osom dobijamo za \( x = 0 \):

\[ f(0) = a \cdot 0^2 + b \cdot 0 + c = c \]

Sa slike vidimo da parabola sece \( y \)-osu ispod \( x \)-ose, sto znaci da je:

\[ f(0) < 0 \quad \Longrightarrow \quad c < 0 \]

Ovim eliminisemo opciju D (gde je \( c > 0 \)). Ostaje jedino opcija C.

Proveri sebe: Sta nam govori presek parabole sa \( y \)-osom iznad \( x \)-ose?

Ako bi parabola sekla \( y \)-osu iznad \( x \)-ose, to bi znacilo \( f(0) = c > 0 \). U tom slucaju slobodan clan \( c \) bio bi pozitivan.

5

Zakljucak

Kombinujuci sva tri uslova:

\[ a > 0 \qquad (\text{parabola se otvara navise}) \]
\[ b < 0 \qquad (\text{teme je desno od } y\text{-ose}) \]
\[ c < 0 \qquad (\text{presek sa } y\text{-osom je ispod } x\text{-ose}) \]

Jedina opcija koja odgovara svim uslovima je:

\[ \boxed{\text{C)}\quad a > 0,\; b < 0,\; c < 0} \]
Proveri sebe: Zasto opcija A (\( a > 0,\, b > 0,\, c < 0 \)) nije tacna?

Opcija A ima \( b > 0 \). Uz \( a > 0 \) i \( b > 0 \), teme bi bilo na \( x_T = -\frac{b}{2a} < 0 \), dakle levo od \( y \)-ose. Ali sa slike vidimo da je teme desno od \( y \)-ose, pa opcija A ne odgovara grafiku.

Kljucni uvid

Svaki od tri koeficijenta kvadratne funkcije \( f(x) = ax^2 + bx + c \) ima jasno geometrijsko znacenje koje se moze procitati direktno sa grafika:

\[ a \;\longleftrightarrow\; \text{smer otvaranja} \qquad b \;\longleftrightarrow\; \text{polozaj temena} \qquad c \;\longleftrightarrow\; \text{presek sa } y\text{-osom} \]

Kljuc za koeficijent \( b \) je formula za \(x\)-koordinatu temena: \( x_T = -\frac{b}{2a} \). Znakovi \( a \) i polozaj temena zajedno jednoznacno odredjuju znak \( b \).

Konacan odgovor
\( a > 0,\; b < 0,\; c < 0 \)
A) \( a > 0,\, b > 0,\, c < 0 \)
B) \( a > 0,\, b > 0,\, c > 0 \)
C) \( a > 0,\, b < 0,\, c < 0 \)
D) \( a > 0,\, b < 0,\, c > 0 \)
E) \( a < 0,\, b < 0,\, c < 0 \)

Verifikacija: Uzmimo konkretan primer \( f(x) = x^2 - 3x - 2 \) (gde je \( a = 1 > 0 \), \( b = -3 < 0 \), \( c = -2 < 0 \)). Teme je u \( x_T = \frac{3}{2} = 1{,}5 > 0 \) (desno od \( y \)-ose), a \( f(0) = -2 < 0 \) (ispod \( x \)-ose). Parabola se otvara navise. Sve se slaze sa opisom grafika.

Ceste greske

✗

Zamena znaka \( b \) i polozaja temena: Cesta greska je zakljuciti da "teme desno" znaci \( b > 0 \). Medjutim, iz formule \( x_T = -\frac{b}{2a} \) vidimo da postoji minus ispred \( b \). Ako je \( a > 0 \) i teme desno (\( x_T > 0 \)), onda je \( b < 0 \), ne \( b > 0 \)! Taj minus u formuli je cest izvor greske.

✗

Zaboraviti da je \( c = f(0) \): Koeficijent \( c \) nije "neka apstraktna konstanta" — on je tacno vrednost funkcije u nuli, tj. \( y \)-koordinata preseka sa \( y \)-osom. Kad vidite gde grafik sece \( y \)-osu, odmah znate znak od \( c \).

Dodatni izazov

Ako za istu parabolu (\( a > 0 \), \( b < 0 \), \( c < 0 \)) znate da je diskriminanta \( D = b^2 - 4ac > 0 \), koliko realnih nula ima funkcija \( f \)? Da li obe nule mogu biti pozitivne?

Pogledaj hint

Koristite Vijetove formule: \( x_1 + x_2 = -\frac{b}{a} \) i \( x_1 \cdot x_2 = \frac{c}{a} \). Posto je \( b < 0 \) i \( a > 0 \), zbir nula je pozitivan. Posto je \( c < 0 \) i \( a > 0 \), proizvod nula je negativan. Sta to govori o znacima pojedinacnih nula?
