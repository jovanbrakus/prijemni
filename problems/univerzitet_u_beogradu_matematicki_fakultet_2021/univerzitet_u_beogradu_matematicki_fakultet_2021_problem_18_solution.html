Deljenje polinoma sa dva delioca

Polinomsko deljenje, sistem jednacina i odredjivanje koeficijenata

📋 Tekst zadatka

Neka su \(a\), \(b\), \(c\), \(d\) realni brojevi. Ako polinom \(x^4 - x^3 + ax^2 + bx + c\) pri deljenju sa \(x^2 + d^2\) daje ostatak \(x\), dok pri deljenju sa \(x^2 - d^2\) daje ostatak \(-x\), onda je \(\dfrac{a + b + c}{d^2}\) jednako:

Polinom
\(P(x) = x^4 - x^3 + ax^2 + bx + c\)
Delilac 1
\(x^2 + d^2\), ostatak \(x\)
Delilac 2
\(x^2 - d^2\), ostatak \(-x\)
(A)
\(-1\)
(B)
\(0\)
(C)
\(1\)
(D)
\(2\)
(E)
\(\dfrac{1}{4}\)

🗺️ Plan resavanja

Zapisacemo jednakost deljenja za oba delioca i porediti koeficijente uz iste stepene od \(x\). Iz svakog deljenja dobijamo po dve jednacine, sto daje sistem od cetiri jednacine sa cetiri nepoznate. Resavanjem tog sistema nalazimo \(a\), \(b\), \(c\) i \(d^2\), a zatim racunamo trazeni izraz.

📐 Podsetnik iz teorije

Kljucne formule i pojmovi za ovaj zadatak:

\[P(x) = D(x) \cdot q(x) + r(x), \quad \deg(r) < \deg(D)\]

Teorema o deljenju polinoma: Za svaka dva polinoma \(P(x)\) i \(D(x)\) (gde \(D \neq 0\)) postoje jedinstveni polinomi \(q(x)\) (kolicnik) i \(r(x)\) (ostatak) takvi da vazi gornja jednakost, pri cemu je stepen od \(r(x)\) strogo manji od stepena od \(D(x)\).

\[\text{Ako } P(x) = D(x) \cdot q(x) + r(x), \text{ onda oba polinoma imaju iste koeficijente uz svaki stepen od } x.\]

Poredjenje koeficijenata: Ako dva polinoma imaju jednake vrednosti za svako \(x\), onda su im koeficijenti uz iste stepene jednaki. Ovo nam omogucava da iz jedne polinomske jednakosti izvucemo vise numerickih jednacina.

\[\text{Za } P(x) = x^4 - x^3 + ax^2 + bx + c \text{ i delilac stepena 2: } q(x) = x^2 + \alpha x + \beta\]

Stepen kolicnika: Ako polinom stepena 4 delimo polinomom stepena 2, kolicnik je stepena 2 (razlika stepena), a ostatak je stepena najvise 1 (manji od stepena delioca).

Resenja sistema

📝 Resenje korak po korak

1 Zapisimo jednakost prvog deljenja

Polinom \(P(x) = x^4 - x^3 + ax^2 + bx + c\) pri deljenju sa \(x^2 + d^2\) daje ostatak \(x\). Prema teoremi o deljenju:

\[x^4 - x^3 + ax^2 + bx + c = (x^2 + d^2) \cdot q_1(x) + x\]

Posto je \(P(x)\) stepena 4, a delilac stepena 2, kolicnik \(q_1(x)\) mora biti stepena 2. Neka je:

\[q_1(x) = x^2 + \alpha x + \beta\]
2 Raspisimo proizvod i poredimo koeficijente

Raspisemo desnu stranu:

\[(x^2 + d^2)(x^2 + \alpha x + \beta) + x\]

Mnozimo zagrade:

\[= x^4 + \alpha x^3 + \beta x^2 + d^2 x^2 + \alpha d^2 x + \beta d^2 + x\]

Grupismo po stepenima:

\[= x^4 + \alpha x^3 + (\beta + d^2) x^2 + (\alpha d^2 + 1) x + \beta d^2\]
3 Izvucimo jednacine iz prvog deljenja

Poredimo koeficijente sa \(x^4 - x^3 + ax^2 + bx + c\):

\[\begin{align} x^3 &: \quad \alpha = -1 \\[4pt] x^2 &: \quad \beta + d^2 = a \\[4pt] x^1 &: \quad \alpha d^2 + 1 = b \\[4pt] x^0 &: \quad \beta d^2 = c \end{align}\]

Posto je \(\alpha = -1\), iz jednacine za \(x^1\) dobijamo:

\[-d^2 + 1 = b \quad \Longrightarrow \quad \boxed{b + d^2 = 1} \quad \text{...(I)}\]

A iz jednacina za \(x^2\) i \(x^0\): \(\beta = a - d^2\) i \(c = (a - d^2)d^2\), tj.:

\[\boxed{c - ad^2 + d^4 = 0} \quad \text{...(II)}\]
Proveri sebe: Zasto je kolicnik oblika \(x^2 + \alpha x + \beta\)?

Zato sto delimo polinom stepena 4 sa polinomom stepena 2, pa je stepen kolicnika \(4 - 2 = 2\). Takodje, vodecu koeficijent kolicnika mora biti 1 (jer je vodeci koeficijent i polinoma i delioca jednak 1, a \(1 \cdot 1 = 1\)).

4 Zapisimo jednakost drugog deljenja

Sada koristimo drugi uslov: deljenje sa \(x^2 - d^2\) daje ostatak \(-x\):

\[x^4 - x^3 + ax^2 + bx + c = (x^2 - d^2) \cdot q_2(x) + (-x)\]

Neka je \(q_2(x) = x^2 + \gamma x + \delta\). Raspisemo desnu stranu:

\[(x^2 - d^2)(x^2 + \gamma x + \delta) - x\]
\[= x^4 + \gamma x^3 + (\delta - d^2) x^2 + (-\gamma d^2 - 1) x - \delta d^2\]
5 Izvucimo jednacine iz drugog deljenja

Poredimo koeficijente:

\[\begin{align} x^3 &: \quad \gamma = -1 \\[4pt] x^2 &: \quad \delta - d^2 = a \\[4pt] x^1 &: \quad -\gamma d^2 - 1 = b \\[4pt] x^0 &: \quad -\delta d^2 = c \end{align}\]

Posto je \(\gamma = -1\), iz jednacine za \(x^1\):

\[d^2 - 1 = b \quad \Longrightarrow \quad \boxed{b - d^2 = -1} \quad \text{...(III)}\]

A iz jednacina za \(x^2\) i \(x^0\): \(\delta = a + d^2\) i \(c = -(a + d^2)d^2\), tj.:

\[\boxed{c + ad^2 + d^4 = 0} \quad \text{...(IV)}\]
6 Resimo sistem: odredimo \(b\) i \(d^2\)

Imamo dve jednacine sa \(b\) i \(d^2\):

I \(\quad b + d^2 = 1\)
III \(\quad b - d^2 = -1\)

Saberemo (I) i (III):

\[2b = 0 \quad \Longrightarrow \quad \boxed{b = 0}\]

Oduzmemo (III) od (I):

\[2d^2 = 2 \quad \Longrightarrow \quad \boxed{d^2 = 1}\]
Proveri sebe: Koliko je \(b + d^2\) i \(b - d^2\) za nase vrednosti?

Za \(b = 0\) i \(d^2 = 1\): \(b + d^2 = 0 + 1 = 1\) i \(b - d^2 = 0 - 1 = -1\). Tacno se poklapa sa jednacinama (I) i (III)!

7 Odredimo \(a\) i \(c\)

Koristimo jednacine (II) i (IV) sa poznatim \(d^2 = 1\):

II \(\quad c - a \cdot 1 + 1 = 0 \quad \Longrightarrow \quad c = a - 1\)
IV \(\quad c + a \cdot 1 + 1 = 0 \quad \Longrightarrow \quad c = -a - 1\)

Izjednacimo dva izraza za \(c\):

\[a - 1 = -a - 1\]
\[2a = 0 \quad \Longrightarrow \quad \boxed{a = 0}\]

Vratimo u bilo koji izraz za \(c\):

\[c = 0 - 1 = -1 \quad \Longrightarrow \quad \boxed{c = -1}\]
8 Izracunajmo trazenu vrednost

Sada imamo sve sto nam treba: \(a = 0\), \(b = 0\), \(c = -1\), \(d^2 = 1\).

\[\frac{a + b + c}{d^2} = \frac{0 + 0 + (-1)}{1} = \frac{-1}{1} = -1\]
Proveri sebe: Da li polinom \(x^4 - x^3 - 1\) zaista zadovoljava oba uslova?

Sa \(a = 0\), \(b = 0\), \(c = -1\), \(d^2 = 1\), polinom je \(P(x) = x^4 - x^3 - 1\).

Deljenje sa \(x^2 + 1\):
\((x^2 + 1)(x^2 - x - 1) = x^4 - x^3 - x^2 + x^2 - x - 1 = x^4 - x^3 - x - 1\).
Dodajmo ostatak \(x\): \(x^4 - x^3 - x - 1 + x = x^4 - x^3 - 1 = P(x)\). Tacno!

Deljenje sa \(x^2 - 1\):
\((x^2 - 1)(x^2 - x + 1) = x^4 - x^3 + x^2 - x^2 + x - 1 = x^4 - x^3 + x - 1\).
Dodajmo ostatak \(-x\): \(x^4 - x^3 + x - 1 - x = x^4 - x^3 - 1 = P(x)\). Tacno!

💡 Kljucni uvid

Dva uslova deljenja (sa \(x^2 + d^2\) i \(x^2 - d^2\)) daju dva para jednacina koji se medjusobno lepo dopunjuju. Jednacine za \(b\) i \(d^2\) se dobijaju sabiranjem i oduzimanjem, a jednacine za \(a\) i \(c\) se takodje resavaju izjednacavanjem. Simetrija izmedju delioca \(x^2 + d^2\) i \(x^2 - d^2\) je kljuc za elegantno resenje.

\[\frac{a + b + c}{d^2} = \frac{0 + 0 + (-1)}{1} = -1\]

Konacan odgovor

\[\frac{a + b + c}{d^2} = -1\]
(A) \(-1\)
(B) \(0\)
(C) \(1\)
(D) \(2\)
(E) \(\dfrac{1}{4}\)

Odgovor je (A)

Verifikacija: Sa \(a = 0\), \(b = 0\), \(c = -1\), \(d^2 = 1\), polinom je \(P(x) = x^4 - x^3 - 1\). Deljenje sa \(x^2 + 1\): \(P(x) = (x^2 + 1)(x^2 - x - 1) + x\). Deljenje sa \(x^2 - 1\): \(P(x) = (x^2 - 1)(x^2 - x + 1) + (-x)\). Oba uslova su zadovoljena. Trazena vrednost: \((0 + 0 - 1)/1 = -1\).

⚠️ Ceste greske

  • Zaboravljanje znaka ostatka: Ostatak pri deljenju sa \(x^2 - d^2\) je \(-x\), ne \(x\). Ako se ovo previdi i koristi \(x\) za oba deljenja, dobija se pogresan sistem jednacina.
  • Pogresna pretpostavka o obliku kolicnika: Kolicnik pri deljenju polinoma stepena 4 sa polinomom stepena 2 mora biti stepena 2 sa vodecim koeficijentom 1. Neki studenti postave kolicnik kao polinom stepena 3, sto dovodi do kontradikcija.

🚀 Izazov za tebe

Neka su \(a\), \(b\), \(c\), \(d\) realni brojevi. Ako polinom \(x^4 + ax^3 + bx^2 + cx + d\) pri deljenju sa \(x^2 + 1\) daje ostatak \(2x + 3\), a pri deljenju sa \(x^2 - 1\) daje ostatak \(x - 1\), koliko je \(a + b + c + d\)?

Nagovestaj

Postupak je potpuno analogan: zapisi kolicnike kao \(x^2 + \alpha x + \beta\) i \(x^2 + \gamma x + \delta\), raspisite proizvode i poredite koeficijente. Dobicete sistem od 4 jednacine sa 4 nepoznate. Obratite paznju na vodeci koeficijent kolicnika!
