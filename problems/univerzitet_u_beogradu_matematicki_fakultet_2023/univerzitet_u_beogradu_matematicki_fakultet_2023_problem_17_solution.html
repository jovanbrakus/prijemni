Ostatak pri deljenju polinoma visokog stepena

Deljenje polinoma, kompleksne nule i evaluacija ostatka

📋 Tekst zadatka

Ostatak pri deljenju polinoma

\[x^{2024} + x^{2023} + 1\]

sa

\[x^3 - x^2 + x - 1\]

jednak je:

(A)
\(-x^2 - x + 5\)
(B)
\(x^2 - x + 3\)
(C)
\(-x^2 + x + 3\)
(D)
\(-x^2 + 4\)
(E)
\(x^2 + 2\)

🗺️ Plan rešavanja

Prvo ćemo faktorizovati delilac i pronaći njegove nule (realne i kompleksne). Zatim ćemo koristiti činjenicu da je ostatak stepena manjeg od 3, pa ga možemo odrediti evaluacijom polaznog polinoma u nulama delitelja.

📖 Podsetnik iz teorije

Ključni pojmovi za ovaj zadatak:

  • Deljenje polinoma sa ostatkom: Za polinome \(p(x)\) i \(q(x)\) važi \[p(x) = k(x) \cdot q(x) + r(x)\] gde je \(\deg r < \deg q\).
  • Evaluacija u nulama: Ako je \(q(\alpha) = 0\), tada \(p(\alpha) = r(\alpha)\). Ovo nam omogućava da odredimo koeficijente ostatka bez stvarnog deljenja.
  • Stepeni imaginarne jedinice: \[i^0 = 1, \quad i^1 = i, \quad i^2 = -1, \quad i^3 = -i, \quad i^4 = 1, \quad \ldots\] Ciklus se ponavlja sa periodom 4.
  • Grupisanje pri faktorizaciji: \[x^3 - x^2 + x - 1 = x^2(x-1) + 1(x-1) = (x-1)(x^2+1)\]

📊 Vizuelni prikaz metode

📝 Rešenje korak po korak

1 Faktorizacija delitelja

Rastavimo \(q(x) = x^3 - x^2 + x - 1\) grupisanjem:

\[x^3 - x^2 + x - 1 = x^2(x - 1) + 1 \cdot (x - 1)\]

Iznosimo zajednički faktor \((x-1)\):

\[q(x) = (x - 1)(x^2 + 1)\]

Nule delitelja su: \(x = 1\) i \(x = \pm i\) (gde je \(i\) imaginarna jedinica).

2 Oblik ostatka

Pošto je delilac stepena 3, ostatak je oblika:

\[r(x) = ax^2 + bx + c\]

Važi jednakost:

\[p(x) = x^{2024} + x^{2023} + 1 = k(x) \cdot q(x) + ax^2 + bx + c\]

U nulama delitelja \(q(\alpha) = 0\), pa je \(p(\alpha) = r(\alpha)\). To nam daje dovoljno jednačina za nalaženje \(a, b, c\).

Proveri sebe: Koliko nepoznatih imamo i koliko nula delitelja?

Imamo 3 nepoznate (\(a, b, c\)) i 3 nule delitelja (\(1, i, -i\)). Sistem je određen!

3 Evaluacija za \(x = 1\)

Uvrštavamo \(x = 1\) u polinom \(p(x)\):

\[p(1) = 1^{2024} + 1^{2023} + 1 = 1 + 1 + 1 = 3\]

Takođe, \(r(1) = a + b + c\). Dakle:

\[a + b + c = 3 \quad \text{...(I)}\]
4 Evaluacija za \(x = i\)

Uvrštavamo \(x = i\) u polinom. Prvo odredimo stepene broja \(i\), koristeći periodičnost sa periodom 4:

\[i^{2024} = (i^4)^{506} = 1^{506} = 1\]

Za \(i^{2023}\): pošto je \(2023 = 4 \cdot 505 + 3\), imamo:

\[i^{2023} = i^3 = -i\]

Prema tome:

\[p(i) = 1 + (-i) + 1 = 2 - i\]
5 Jednačina za \(r(i)\)

Uvrštavamo \(x = i\) u ostatak:

\[r(i) = a \cdot i^2 + b \cdot i + c = -a + bi + c\]

Izjednačavamo sa \(p(i) = 2 - i\):

\[-a + bi + c = 2 - i\]

Razdvajamo na realni i imaginarni deo:

\[\text{Re:} \quad -a + c = 2 \quad \text{...(II)}\]
\[\text{Im:} \quad b = -1 \quad \text{...(III)}\]
Proveri sebe: Zašto poređenje realnog i imaginarnog dela daje dve jednačine?

Dva kompleksna broja su jednaka ako i samo ako su im jednaki i realni i imaginarni delovi. Zato iz jedne kompleksne jednačine dobijamo dve realne jednačine.

6 Rešavanje sistema

Imamo sistem od tri jednačine:

\[\begin{align} a + b + c &= 3 \quad \text{(I)} \\ -a + c &= 2 \quad \text{(II)} \\ b &= -1 \quad \text{(III)} \end{align}\]

Iz (III): \(b = -1\).

Iz (II): \(c = a + 2\).

Zamenom u (I):

\[a + (-1) + (a + 2) = 3\]
\[2a + 1 = 3 \implies a = 1\]

Odavde: \(c = 1 + 2 = 3\).

7 Formiranje ostatka

Sa \(a = 1\), \(b = -1\), \(c = 3\), ostatak je:

\[r(x) = x^2 - x + 3\]
Verifikacija: Proverimo za \(x = -i\)

\(p(-i) = (-i)^{2024} + (-i)^{2023} + 1\). Pošto je \((-i)^{2024} = ((-i)^4)^{506} = 1\) i \((-i)^{2023} = (-i)^3 = i\), dobijamo \(p(-i) = 1 + i + 1 = 2 + i\).
Sa druge strane: \(r(-i) = (-i)^2 - (-i) + 3 = -1 + i + 3 = 2 + i\). Slaže se!

Ključni uvid

Umesto da stvarno delimo polinom ogromnog stepena (2024!), koristimo činjenicu da se ostatak u potpunosti određuje svojim vrednostima u nulama delitelja. Faktorizacija \(q(x) = (x-1)(x^2+1)\) nam daje nule \(1, i, -i\), a ciklična priroda stepena imaginarne jedinice (\(i^4 = 1\)) omogućava lako računanje \(i^{2024}\) i \(i^{2023}\).

Konačan odgovor

\[\boxed{x^2 - x + 3}\]

Odgovor je (B)

Verifikacija: Za \(x = 1\) ostatak daje \(1 - 1 + 3 = 3 = p(1)\). Za \(x = i\) ostatak daje \(-1 - i + 3 = 2 - i = p(i)\). Sve se slaže.

⚠️ Česte greške

  • Pogrešno određivanje \(i^{2023}\): Ostatak pri deljenju eksponenta sa 4 je ključan. \(2023 = 4 \cdot 505 + 3\), pa je \(i^{2023} = i^3 = -i\), ne \(i\).
  • Zaboravljanje da \(i^2 = -1\) pri računanju \(r(i) = ai^2 + bi + c\). Treba pisati \(r(i) = -a + bi + c\), a ne \(r(i) = a + bi + c\).

Izazov za tebe

Odredi ostatak pri deljenju polinoma \(x^{100} + x^{50} + 1\) sa \(x^2 - 1\).

Savet

Delilac \(x^2 - 1 = (x-1)(x+1)\) ima nule \(x = 1\) i \(x = -1\). Ostatak je oblika \(ax + b\). Izračunaj \(p(1)\) i \(p(-1)\) da dobiješ sistem od dve jednačine sa dve nepoznate.
