Симетрала дужи кроз две тачке

Аналитичка геометрија у равни — Симетрала дужи

Задатак

Симетрала дужи која спаја тачке \(M(2,\, 5)\) и \(N(4,\, 1)\) је права:

А) \(x - 2y + 3 = 0\)
Б) \(2y - x + 3 = 0\)
В) \(x - 2y - 3 = 0\)
Г) \(2x - y + 3 = 0\)

План решавања

  1. Одредити средиште \(S\) дужи \(MN\) — симетрала пролази кроз ту тачку.
  2. Израчунати нагиб дужи \(MN\).
  3. Одредити нагиб симетрале као негативан реципрочан нагиб (нормала на \(MN\)).
  4. Написати једначину праве кроз \(S\) са добијеним нагибом и средити у општи облик.
Теоријски подсетник

Симетрала дужи је права која је нормална на дуж и пролази кроз њено средиште. Свака тачка симетрале је једнако удаљена од оба краја дужи.

Средиште дужи са крајевима \(A(x_1, y_1)\) и \(B(x_2, y_2)\):

\[S = \left(\frac{x_1 + x_2}{2},\; \frac{y_1 + y_2}{2}\right)\]

Нагиб праве кроз две тачке:

\[k = \frac{y_2 - y_1}{x_2 - x_1}\]

Ако је нагиб праве \(k\), онда нормала на ту праву има нагиб:

\[k_\perp = -\frac{1}{k}\]

Једначина праве кроз тачку \((x_0, y_0)\) са нагибом \(k\):

\[y - y_0 = k(x - x_0)\]

Два правца су узајамно нормална ако и само ако је производ њихових нагиба једнак \(-1\), тј. \(k_1 \cdot k_2 = -1\).

Визуелни приказ

Дијаграм приказује тачке \(M\) и \(N\), средиште \(S\), дуж \(MN\) и симетралу.

Решавање корак по корак

1
Средиште дужи \(MN\)

Симетрала увек пролази кроз средиште дужи. Средиште \(S\) налазимо као аритметичку средину координата:

\[S = \left(\frac{x_M + x_N}{2},\; \frac{y_M + y_N}{2}\right) = \left(\frac{2 + 4}{2},\; \frac{5 + 1}{2}\right) = (3,\, 3)\]
Контролна тачка
Средиште дужи: \(S(3,\, 3)\)
2
Нагиб дужи \(MN\)

Нагиб праве кроз тачке \(M(2, 5)\) и \(N(4, 1)\):

\[k_{MN} = \frac{y_N - y_M}{x_N - x_M} = \frac{1 - 5}{4 - 2} = \frac{-4}{2} = -2\]
Контролна тачка
Нагиб дужи \(MN\): \(k_{MN} = -2\)
3
Нагиб симетрале (нормале на \(MN\))

Симетрала је нормална на дуж \(MN\), па јој је нагиб негативан реципрочан:

\[k_s = -\frac{1}{k_{MN}} = -\frac{1}{-2} = \frac{1}{2}\]

Провера: \(k_{MN} \cdot k_s = (-2) \cdot \dfrac{1}{2} = -1\) — услов нормалности је испуњен.

Контролна тачка
Нагиб симетрале: \(k_s = \dfrac{1}{2}\)
4
Једначина симетрале

Симетрала пролази кроз \(S(3, 3)\) и има нагиб \(\dfrac{1}{2}\). Користимо тачка-нагиб облик:

\[y - 3 = \frac{1}{2}(x - 3)\]

Множимо обе стране са 2:

\[2(y - 3) = x - 3\] \[2y - 6 = x - 3\]

Преносимо све на једну страну:

\[0 = x - 3 - 2y + 6\] \[x - 2y + 3 = 0\]
Контролна тачка
Једначина симетрале: \(x - 2y + 3 = 0\)

Кључни увид

Симетрала дужи се одређује помоћу два кључна податка: тачке кроз коју пролази (средиште дужи) и правца (нормала на дуж). Ово је чест образац у аналитичкој геометрији — права је потпуно одређена тачком и правцем.

Обратите пажњу: одговори Б) и А) се разликују само у знаку испред 3. Довољна је провера да средиште \(S(3, 3)\) задовољава једначину да се елиминише Б).

Коначан одговор

Симетрала дужи \(MN\) је права: \(\quad \mathbf{x - 2y + 3 = 0}\)

Верификација:

1. Средиште \(S(3, 3)\) лежи на правој:

\(3 - 2 \cdot 3 + 3 = 3 - 6 + 3 = 0\)   ✓

2. Тачке \(M\) и \(N\) су једнако удаљене од праве:

\(d(M) = \dfrac{|2 - 2 \cdot 5 + 3|}{\sqrt{1^2 + (-2)^2}} = \dfrac{|2 - 10 + 3|}{\sqrt{5}} = \dfrac{5}{\sqrt{5}} = \sqrt{5}\)

\(d(N) = \dfrac{|4 - 2 \cdot 1 + 3|}{\sqrt{1^2 + (-2)^2}} = \dfrac{|4 - 2 + 3|}{\sqrt{5}} = \dfrac{5}{\sqrt{5}} = \sqrt{5}\)

\(d(M) = d(N) = \sqrt{5}\)   ✓

3. Нормалност на \(MN\):

Нагиб симетрале: \(k_s = \dfrac{1}{2}\), нагиб дужи: \(k_{MN} = -2\)

\(k_s \cdot k_{MN} = \dfrac{1}{2} \cdot (-2) = -1\)   ✓

А) \(x - 2y + 3 = 0\)
Б) \(2y - x + 3 = 0\)
В) \(x - 2y - 3 = 0\)
Г) \(2x - y + 3 = 0\)

Честе грешке

Мешање нагиба дужи и симетрале: Нагиб симетрале није исти као нагиб дужи! Симетрала је нормална на дуж, па јој је нагиб негативан реципрочан: \(k_s = -1/k_{MN}\). Ако се користи исти нагиб, добија се паралелна права (одговор Б не задовољава).
Грешка у знаку слободног члана: Одговори А) и В) имају исту структуру, али различит знак: \(x - 2y + 3 = 0\) наспрам \(x - 2y - 3 = 0\). Увек проверите уврштавањем средишта \(S\) у једначину.
Замена одговора А) и Б): Приметите да \(2y - x + 3 = 0\) преуређен даје \(-x + 2y + 3 = 0\), тј. \(x - 2y - 3 = 0\) — што је заправо исто као одговор В). Зато пажљиво проверите знакове при преуређивању.

Додатни изазов

Одредите једначину симетрале дужи која спаја тачке \(A(-1, 3)\) и \(B(5, -1)\). Затим проверите да ли тачка \(P(4, 5)\) лежи на тој симетрали.

Савет: следите исту процедуру — средиште, нагиб, нормала, једначина. За проверу припадности, уврстите координате тачке \(P\) у једначину.
