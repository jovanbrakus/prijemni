Тангенте круга паралелне са правом

Рударско-геолошки факултет, Београд 2009 — Задатак 16

Задатак

Тангенте круга \(x^2 + y^2 = 5\) паралелне са правом \(4x - 2y + 5 = 0\) су праве:

А) \(2x - y \pm 5 = 0\)
Б) \(x - 2y \pm 5 = 0\)
В) \(2x - y \pm 7 = 0\)
Г) \(x - 2y \pm 7 = 0\)

План решавања

  1. Одредити центар и полупречник датог круга.
  2. Из једначине дате праве очитати коефицијент правца.
  3. Написати општи облик тангенте паралелне са том правом.
  4. Применити услов тангентности: растојање од центра до праве = полупречник.
  5. Решити једначину и добити обе тангенте.
Теоријски подсетник

Круг са центром у координатном почетку: Једначина \(x^2 + y^2 = r^2\) представља круг са центром \(O(0,0)\) и полупречником \(r\).

\[x^2 + y^2 = r^2 \implies C(0,\,0),\quad r = \sqrt{r^2}\]

Коефицијент правца праве: За праву \(ax + by + c = 0\), коефицијент правца је \(k = -\dfrac{a}{b}\).

\[ax + by + c = 0 \implies y = -\frac{a}{b}\,x - \frac{c}{b} \implies k = -\frac{a}{b}\]

Паралелне праве имају исти коефицијент правца. Ако је дата права \(y = kx + m\), свака паралелна права има облик \(y = kx + n\), где се \(n\) разликује од \(m\).

\[\text{Паралелне праве:}\quad y = kx + n_1 \;\|\; y = kx + n_2\]

Растојање тачке од праве: Растојање тачке \((x_0, y_0)\) од праве \(ax + by + c = 0\) је:

\[d = \frac{|ax_0 + by_0 + c|}{\sqrt{a^2 + b^2}}\]

Услов тангентности: Права је тангента круга ако и само ако је растојање од центра круга до праве једнако полупречнику:

\[d(C,\, \text{права}) = r\]

Визуелни приказ

Круг \(x^2 + y^2 = 5\) са две тангенте паралелне датој правој.

Решавање корак по корак

1
Одредити центар и полупречник круга

Дата је једначина круга:

\[x^2 + y^2 = 5\]

Ово је круг са центром у координатном почетку и полупречником:

\[C(0,\,0), \quad r = \sqrt{5}\]
Контролна тачка
Центар \(O(0,\,0)\), полупречник \(r = \sqrt{5}\).
2
Одредити коефицијент правца дате праве

Дата права је \(4x - 2y + 5 = 0\). Решавамо по \(y\):

\[4x - 2y + 5 = 0 \implies 2y = 4x + 5 \implies y = 2x + \frac{5}{2}\]

Коефицијент правца ове праве је:

\[k = 2\]
Контролна тачка
Коефицијент правца дате праве: \(k = 2\).
3
Написати општи облик тражених тангенти

Тражене тангенте су паралелне са датом правом, па имају исти коефицијент правца \(k = 2\):

\[y = 2x + n\]

Или у општем облику:

\[2x - y + n = 0\]

Потребно је одредити вредност параметра \(n\).

Контролна тачка
Облик тангенти: \(2x - y + n = 0\), непознато \(n\).
4
Применити услов тангентности

Права \(2x - y + n = 0\) је тангента круга ако је растојање од центра \(O(0,\,0)\) до праве једнако полупречнику \(r = \sqrt{5}\):

\[d(O, \text{права}) = \frac{|2 \cdot 0 + (-1) \cdot 0 + n|}{\sqrt{2^2 + (-1)^2}} = \frac{|n|}{\sqrt{4 + 1}} = \frac{|n|}{\sqrt{5}}\]

Постављамо услов:

\[\frac{|n|}{\sqrt{5}} = \sqrt{5}\]
Контролна тачка
Услов: \(\dfrac{|n|}{\sqrt{5}} = \sqrt{5}\).
5
Решити једначину по \(n\)

Множимо обе стране са \(\sqrt{5}\):

\[|n| = \sqrt{5} \cdot \sqrt{5} = 5\]

Одавде добијамо два решења:

\[n = 5 \quad \text{или} \quad n = -5\]
Контролна тачка
\(n = \pm 5\).
6
Записати једначине тангенти

Враћамо вредности \(n\) у општи облик \(2x - y + n = 0\):

Прва тангента (\(n = 5\)):

\[2x - y + 5 = 0 \quad \Leftrightarrow \quad y = 2x + 5\]

Друга тангента (\(n = -5\)):

\[2x - y - 5 = 0 \quad \Leftrightarrow \quad y = 2x - 5\]

Компактан запис:

\[\boxed{2x - y \pm 5 = 0}\]
Контролна тачка
Тангенте: \(2x - y + 5 = 0\) и \(2x - y - 5 = 0\).

Кључни увид

Круг има тачно две тангенте паралелне са сваком правом која не пролази кроз центар. Те две тангенте су симетричне у односу на центар круга — једна „додирује" круг са горње, друга са доње стране. Зато се у решењу увек појављују два решења: \(n = +c\) и \(n = -c\).

Приметимо и корисну везу: \(|n| = r \cdot \sqrt{a^2 + b^2}\) за тангенту облика \(ax + by + n = 0\) на круг \(x^2 + y^2 = r^2\). У нашем случају: \(|n| = \sqrt{5} \cdot \sqrt{5} = 5\).

Коначан одговор

\[2x - y \pm 5 = 0\]

Верификација:

Проверимо да обе праве заиста додирују круг. Тачке додира налазимо заменом \(y = 2x \pm 5\) у \(x^2 + y^2 = 5\):

За \(y = 2x + 5\):

\(x^2 + (2x+5)^2 = 5 \implies x^2 + 4x^2 + 20x + 25 = 5 \implies 5x^2 + 20x + 20 = 0 \implies x^2 + 4x + 4 = 0\)

\((x+2)^2 = 0 \implies x = -2,\ y = 2(-2)+5 = 1\). Тачка додира: \((-2,\,1)\). ✓

За \(y = 2x - 5\):

\(x^2 + (2x-5)^2 = 5 \implies x^2 + 4x^2 - 20x + 25 = 5 \implies 5x^2 - 20x + 20 = 0 \implies x^2 - 4x + 4 = 0\)

\((x-2)^2 = 0 \implies x = 2,\ y = 2(2)-5 = -1\). Тачка додира: \((2,\,-1)\). ✓

Обе квадратне једначине имају двоструки корен (дискриминанта = 0), што потврђује да су праве тангенте. ✓

А) \(2x - y \pm 5 = 0\)
Б) \(x - 2y \pm 5 = 0\)
В) \(2x - y \pm 7 = 0\)
Г) \(x - 2y \pm 7 = 0\)

Честе грешке

Замена коефицијената \(a\) и \(b\): Из праве \(4x - 2y + 5 = 0\) коефицијент правца је \(k = -\dfrac{4}{-2} = 2\), а не \(k = \dfrac{1}{2}\). Грешка „обрнутог разломка" води до одговора Б) или Г) уместо тачног.
Погрешна формула за растојање: У имениоцу формуле за растојање стоји \(\sqrt{a^2 + b^2}\), а не \(a^2 + b^2\). Ако заборавите корен, добићете \(|n| = 5\sqrt{5}\) уместо \(|n| = 5\).
Заборављање \(\pm\) знака: Из \(|n| = 5\) следе два решења: \(n = 5\) и \(n = -5\). Круг увек има две тангенте паралелне датој правој (осим ако та права пролази кроз центар, што овде није случај).
Скраћивање једначине праве пре постављања облика тангенте: Из \(4x - 2y + 5 = 0\) можемо скратити са 2 и добити \(2x - y + \frac{5}{2} = 0\). Тангента мора имати исте коефицијенте уз \(x\) и \(y\), али слободан члан \(n\) је непознат — не преносимо \(\frac{5}{2}\) из дате праве!

Додатни изазов

Нађи тангенте круга \(x^2 + y^2 = 13\) које су нормалне (управне) на праву \(3x + 2y - 7 = 0\).

Савет: прво одреди коефицијент правца управне праве (користи услов \(k_1 \cdot k_2 = -1\)), а затим примени исти поступак — услов тангентности преко растојања тачке од праве.
