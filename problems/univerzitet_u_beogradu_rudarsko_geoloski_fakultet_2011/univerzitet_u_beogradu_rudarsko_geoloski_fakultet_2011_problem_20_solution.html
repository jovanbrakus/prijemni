Stepen kompleksnog broja \(z^4\)

Zadatak

Ako je \(z = 1 + i\), tada je \(z^4\):

A) \(1 - i\)
Б) \(-2 + 2i\)
В) \(4i\)
Г) \(-4\)
Plan rešavanja

Umesto da direktno računamo \((1+i)^4\), iskoristićemo strategiju stepenovanja u dva koraka: prvo ćemo naći \(z^2\), a zatim rezultat kvadrirati da dobijemo \(z^4 = (z^2)^2\). Kao alternativnu potvrdu, koristićemo i trigonometrijski oblik kompleksnog broja.

Podsetnik iz teorije
Kompleksni brojevi — osnovna pravila
  • Imaginarna jedinica: \(i^2 = -1\)
  • Algebarski oblik: \(z = a + bi\), gde su \(a, b \in \mathbb{R}\)
  • Kvadriranje: \((a + bi)^2 = a^2 + 2abi + b^2 i^2 = (a^2 - b^2) + 2abi\)
  • Moduo: \(|z| = \sqrt{a^2 + b^2}\)
  • Trigonometrijski oblik: \(z = r(\cos\varphi + i\sin\varphi)\), gde je \(r = |z|\)
  • De Moavr-ova formula: \(z^n = r^n(\cos(n\varphi) + i\sin(n\varphi))\)
Vizuelni prikaz — stepeni broja \(z = 1 + i\) na kompleksnoj ravni
Rešenje korak po korak
1
Izračunajmo \(z^2 = (1 + i)^2\)

Koristimo formulu za kvadrat binoma \((a + b)^2 = a^2 + 2ab + b^2\):

\[ z^2 = (1 + i)^2 = 1^2 + 2 \cdot 1 \cdot i + i^2 = 1 + 2i + (-1) = 2i \]

Ključna stvar: \(i^2 = -1\), pa se realni delovi \(1\) i \(-1\) poništavaju.

Proveri sebe: Da li je \(|z^2| = |z|^2\)?

\(|z| = \sqrt{1^2 + 1^2} = \sqrt{2}\), pa je \(|z|^2 = 2\). Takođe, \(|z^2| = |2i| = 2\). Tačno, moduo se poklapa!

2
Izračunajmo \(z^4 = (z^2)^2 = (2i)^2\)

Sada kvadriramo rezultat iz prethodnog koraka:

\[ z^4 = (2i)^2 = 4 \cdot i^2 = 4 \cdot (-1) = -4 \]

Dakle, \(z^4 = -4\).

3
Verifikacija trigonometrijskim oblikom

Zapišimo \(z = 1 + i\) u trigonometrijskom obliku. Moduo:

\[ r = |z| = \sqrt{1^2 + 1^2} = \sqrt{2} \]

Argument (oba dela su pozitivna, prvi kvadrant):

\[ \varphi = \arctan\frac{1}{1} = \frac{\pi}{4} \]

Dakle: \(z = \sqrt{2}\left(\cos\frac{\pi}{4} + i\sin\frac{\pi}{4}\right)\).

Po De Moavr-ovoj formuli:

\[ z^4 = \left(\sqrt{2}\right)^4 \cdot \left(\cos\frac{4\pi}{4} + i\sin\frac{4\pi}{4}\right) = 4 \cdot (\cos\pi + i\sin\pi) \] \[ = 4 \cdot (-1 + 0 \cdot i) = -4 \]
Proveri sebe: Šta je \((\sqrt{2})^4\)?

\((\sqrt{2})^4 = ((\sqrt{2})^2)^2 = 2^2 = 4\). Možemo i ovako: \((\sqrt{2})^4 = 2^{4/2} = 2^2 = 4\).

Ključni uvid

Umesto da množimo \((1+i)\) četiri puta uzastopno, razbijamo stepenovanje na dva kvadriranja: \(z^4 = (z^2)^2\). Ovo drastično smanjuje količinu računa. Posebno je korisno zapamtiti da je \((1+i)^2 = 2i\) — realni delovi se poništavaju jer je \(i^2 = -1\), a preostaje čisto imaginaran broj čije je dalje kvadriranje trivijalno.

Konačan odgovor
\(z^4 = \boxed{-4}\)
A) \(1 - i\)
Б) \(-2 + 2i\)
В) \(4i\)
Г) \(-4\)

Verifikacija: \((1+i)^2 = 2i\), pa \((2i)^2 = 4i^2 = -4\). Oba pristupa (algebarski i trigonometrijski) daju isti rezultat. Tačan odgovor je Г).

Česte greške
  • Zaboravljanje da je \(i^2 = -1\) — ako se ovo previdi, kvadrat \((1+i)^2\) pogrešno postaje \(1 + 2i + 1 = 2 + 2i\) umesto tačnog \(2i\).
  • Pogrešna primena formule za kvadrat binoma — izostavljanje srednjeg člana \(2ab\), što daje \(1 + i^2 = 0\) umesto \(1 + 2i - 1 = 2i\).
  • Računanje \((2i)^2 = 2i^2 = -2\) umesto \((2i)^2 = 4i^2 = -4\). Pazite: kvadrira se i koeficijent i \(i\)!
  • Mešanje stepena imaginarne jedinice: \(i^1 = i\), \(i^2 = -1\), \(i^3 = -i\), \(i^4 = 1\) — cikličan niz sa periodom 4.
Dodatni izazov

Koristeći isti pristup, pokušaj da izračunaš:

\[ (1 + i)^8 \]
Savet

Već znamo da je \((1+i)^4 = -4\). Dakle, \((1+i)^8 = ((1+i)^4)^2 = (-4)^2 = 16\). Primetite kako se stepenima od dva brzo dolazi do visokih eksponenata!
