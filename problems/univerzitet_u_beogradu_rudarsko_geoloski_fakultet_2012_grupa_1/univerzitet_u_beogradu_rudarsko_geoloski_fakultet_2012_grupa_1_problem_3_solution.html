Identičnost složenog razlomačkog izraza

Uprošćavanje višestrukih verižnih (lančanih) razlomaka pomoću supstitucije

📝 Postavka zadatka
Izraz:
\[ \frac{1}{a + \dfrac{1}{b + \dfrac{1}{a}}} \cdot \frac{1}{b + \dfrac{1}{a}} \;-\; \frac{1}{b + \dfrac{1}{a + \dfrac{1}{b}}} \cdot \frac{1}{a + \dfrac{1}{b}} \]
za one vrednosti promenljivih \(a\) i \(b\) za koje je definisan, identički je jednak izrazu:
(A)
\(ab + 1\)
(Б)
\(a - b\)
(В)
\(\dfrac{ab + 1}{ab}\)
(Г)
\(0\)
🎯 Plan rešavanja

1. Uvedemo smene za unutrašnje razlomke: \(P = b + \frac{1}{a}\) i \(Q = a + \frac{1}{b}\), čime se izraz značajno uprošćava.
2. Izrazimo sve složene razlomke preko \(P\) i \(Q\), izračunamo svaki sabirak posebno, pa ih oduzmemo.

📐 Ključne formule i pravila
  • Sabiranje razlomaka:
    \[ a + \frac{1}{b} = \frac{ab + 1}{b} \]
  • Recipročna vrednost razlomka:
    \[ \frac{1}{\;\dfrac{p}{q}\;} = \frac{q}{p} \]
  • Množenje razlomaka:
    \[ \frac{a}{b} \cdot \frac{c}{d} = \frac{ac}{bd} \]
  • Verižni (lančani) razlomci: Složeni razlomci oblika \(a + \frac{1}{b + \frac{1}{c}}\) se uprošćavaju „iznutra ka spolja" — prvo se sredi najdublji razlomak, pa sledeći nivo.
🎨 Vizuelni prikaz strukture izraza
🛠 Rešenje korak po korak
1 Uvedemo smene za unutrašnje razlomke

Definišemo dva pomoćna izraza koji se ponavljaju u zadatku:

\[ P = b + \frac{1}{a} = \frac{ab + 1}{a} \]
\[ Q = a + \frac{1}{b} = \frac{ab + 1}{b} \]

Primetimo da \(P\) i \(Q\) imaju isti brojilac \(ab + 1\), samo sa različitim imeniocima. Ovo je ključ zadatka.

2 Izrazimo spoljašnje razlomke

Sada izračunamo \(a + \frac{1}{P}\), odnosno spoljašnji nivo prvog lančanog razlomka:

\[ a + \frac{1}{P} = a + \frac{1}{\;\frac{ab+1}{a}\;} = a + \frac{a}{ab + 1} \]

Svedemo na zajednički imenilac:

\[ a + \frac{a}{ab + 1} = \frac{a(ab + 1) + a}{ab + 1} = \frac{a^2b + a + a}{ab + 1} = \frac{a^2 b + 2a}{ab + 1} = \frac{a(ab + 2)}{ab + 1} \]

Potpuno analogno za \(b + \frac{1}{Q}\):

\[ b + \frac{1}{Q} = b + \frac{b}{ab + 1} = \frac{b(ab + 1) + b}{ab + 1} = \frac{ab^2 + 2b}{ab + 1} = \frac{b(ab + 2)}{ab + 1} \]
Proveri se: Šta je zajednički faktor u oba rezultata?

Oba izraza imaju imenilac \(ab + 1\) i faktor \(ab + 2\) u brojiocu. Razlika je samo u tome da prvi ima \(a\), a drugi \(b\) ispred zagrade.

3 Izračunamo prvi sabirak

Prvi sabirak u polaznom izrazu je:

\[ \frac{1}{a + \frac{1}{P}} \cdot \frac{1}{P} \]

Zamenimo ono što smo izračunali:

\[ \frac{1}{\;\dfrac{a(ab+2)}{ab+1}\;} \cdot \frac{1}{\;\dfrac{ab+1}{a}\;} = \frac{ab+1}{a(ab+2)} \cdot \frac{a}{ab+1} \]

Skratimo \(ab + 1\) i \(a\):

\[ = \frac{\cancel{(ab+1)}}{\cancel{a}\,(ab+2)} \cdot \frac{\cancel{a}}{\cancel{(ab+1)}} = \frac{1}{ab + 2} \]
4 Izračunamo drugi sabirak

Drugi sabirak je:

\[ \frac{1}{b + \frac{1}{Q}} \cdot \frac{1}{Q} \]

Primenimo isti postupak:

\[ \frac{1}{\;\dfrac{b(ab+2)}{ab+1}\;} \cdot \frac{1}{\;\dfrac{ab+1}{b}\;} = \frac{ab+1}{b(ab+2)} \cdot \frac{b}{ab+1} \]

Skratimo \(ab + 1\) i \(b\):

\[ = \frac{\cancel{(ab+1)}}{\cancel{b}\,(ab+2)} \cdot \frac{\cancel{b}}{\cancel{(ab+1)}} = \frac{1}{ab + 2} \]
Proveri se: Da li su prvi i drugi sabirak jednaki?

Da! Oba sabirka su jednaka \(\frac{1}{ab + 2}\). To je posledica simetrije izraza — zamena \(a \leftrightarrow b\) u formuli daje isti rezultat.

5 Oduzmemo sabirke

Konačno, oduzimamo drugi sabirak od prvog:

\[ \frac{1}{ab + 2} - \frac{1}{ab + 2} = 0 \]

Dva identična razlomka se međusobno ponište i rezultat je nula!

Proveri se: Numerička verifikacija sa \(a = 2, \; b = 3\)

Za \(a = 2, b = 3\):

\(P = 3 + \frac{1}{2} = \frac{7}{2}\),   \(Q = 2 + \frac{1}{3} = \frac{7}{3}\)

Prvi sabirak: \(\frac{1}{ab + 2} = \frac{1}{8}\)

Drugi sabirak: \(\frac{1}{ab + 2} = \frac{1}{8}\)

Razlika: \(\frac{1}{8} - \frac{1}{8} = 0\)   ✔

Ključni uvid

Oba sabirka u izrazu imaju istu strukturu, samo sa zamenjenim ulogama \(a\) i \(b\). Pošto izraz \(b + \frac{1}{a}\) i \(a + \frac{1}{b}\) dele zajednički brojilac \(ab + 1\), oba sabirka se pojedinačno svode na isti rezultat \(\frac{1}{ab+2}\). Zato je njihova razlika uvek jednaka nuli, bez obzira na konkretne vrednosti \(a\) i \(b\).

Konačan odgovor
\(0\)
(A) \(ab + 1\)
(Б) \(a - b\)
(В) \(\dfrac{ab+1}{ab}\)
(Г) \(0\)

Verifikacija: za \(a = 2,\; b = 3\): oba sabirka jednaka su \(\frac{1}{8}\), razlika \(= 0\) ✔

⚠️ Česte greške
Pogrešan redosled sređivanja lančanog razlomka

Verižni razlomci se moraju sređivati „iznutra ka spolja". Ako počnete od spoljašnjeg nivoa, dobićete pogrešan oblik. Uvek prvo sredite \(b + \frac{1}{a}\), pa tek onda računajte \(a + \frac{1}{b + \frac{1}{a}}\).

Previđanje simetrije izraza

Mnogi studenti pokušavaju da raspisuju ceo izraz odjednom, što vodi do veoma komplikovanih razlomaka. Uvođenje smena \(P\) i \(Q\) i uočavanje simetrije \(a \leftrightarrow b\) drastično pojednostavljuje račun.

🏆 Dodatni izazov

Šta bi se desilo da umesto razlike imamo zbir istih sabiraka:

\[ \frac{1}{a + \dfrac{1}{b + \dfrac{1}{a}}} \cdot \frac{1}{b + \dfrac{1}{a}} \;+\; \frac{1}{b + \dfrac{1}{a + \dfrac{1}{b}}} \cdot \frac{1}{a + \dfrac{1}{b}} \]
Nagovještaj

Već znamo da je svaki sabirak jednak \(\frac{1}{ab+2}\). Šta dobijamo kada ih saberemo?

Rešenje

\(\frac{1}{ab+2} + \frac{1}{ab+2} = \frac{2}{ab+2}\).
