Plan rešavanja

Jednačina sadrži dva izraza sa apsolutnim vrednostima: \(|x + 4|\) i \(|x - 3|\). Kritične tačke su \(x = -4\) i \(x = 3\). Podelićemo realnu osu na tri intervala, ukloniti apsolutne vrednosti u svakom, rešiti linearne jednačine, proveriti pripadnost rešenja intervalu, a zatim izračunati zbir kvadrata svih validnih rešenja.

Ključne formule i teoreme
Definicija apsolutne vrednosti: \[ |a| = \begin{cases} a, & \text{ako } a \geq 0 \\ -a, & \text{ako } a < 0 \end{cases} \]
Metoda intervala: Kod jednačina sa apsolutnim vrednostima, pronalazimo kritične tačke (gde su izrazi pod apsolutnom vrednošću jednaki nuli) i rešavamo jednačinu posebno na svakom intervalu.
Provera rešenja: Svako rešenje mora pripadati intervalu na kom je dobijeno — inače ga odbacujemo.
Zbir kvadrata: Za rešenja \(x_1, x_2, \ldots, x_n\), zbir kvadrata je \(x_1^2 + x_2^2 + \cdots + x_n^2\).

Grafički prikaz

Grafik funkcija \(f(x) = |x+4| - |x-3|\) (plava) i \(g(x) = x\) (zelena). Presečne tačke su rešenja jednačine.

Rešenje korak po korak

1

Određivanje kritičnih tačaka

Izrazi unutar apsolutnih vrednosti menjaju znak u tačkama gde su jednaki nuli:

\[ x + 4 = 0 \implies x = -4 \] \[ x - 3 = 0 \implies x = 3 \]

Ove tačke dele brojevnu pravu na tri intervala: \((-\infty, -4)\), \([-4, 3)\) i \([3, +\infty)\).

2

Slučaj 1 Interval \(x < -4\)

Na ovom intervalu oba izraza su negativna:

\[ x + 4 < 0 \implies |x + 4| = -(x + 4) \] \[ x - 3 < 0 \implies |x - 3| = -(x - 3) \]

Zamenom u jednačinu:

\[ -(x + 4) - (-(x - 3)) = x \] \[ -x - 4 + x - 3 = x \] \[ -7 = x \]

Provera pripadnosti: \(x = -7 < -4\) — pripada intervalu.

Provera u jednačini: \(|-7 + 4| - |-7 - 3| = 3 - 10 = -7 = x\) ✓

Proveri se: Koliko je \((-7)^2\)?

\((-7)^2 = 49\). Zapamti ovu vrednost!

3

Slučaj 2 Interval \(-4 \leq x < 3\)

Na ovom intervalu prvi izraz je nenegativan, a drugi je negativan:

\[ x + 4 \geq 0 \implies |x + 4| = x + 4 \] \[ x - 3 < 0 \implies |x - 3| = -(x - 3) \]

Zamenom u jednačinu:

\[ (x + 4) - (-(x - 3)) = x \] \[ x + 4 + x - 3 = x \] \[ 2x + 1 = x \] \[ x = -1 \]

Provera pripadnosti: \(-4 \leq -1 < 3\) — pripada intervalu.

Provera u jednačini: \(|-1 + 4| - |-1 - 3| = 3 - 4 = -1 = x\) ✓

4

Slučaj 3 Interval \(x \geq 3\)

Na ovom intervalu oba izraza su nenegativna:

\[ x + 4 > 0 \implies |x + 4| = x + 4 \] \[ x - 3 \geq 0 \implies |x - 3| = x - 3 \]

Zamenom u jednačinu:

\[ (x + 4) - (x - 3) = x \] \[ x + 4 - x + 3 = x \] \[ 7 = x \]

Provera pripadnosti: \(x = 7 \geq 3\) — pripada intervalu.

Provera u jednačini: \(|7 + 4| - |7 - 3| = 11 - 4 = 7 = x\) ✓

Proveri se: Koja su sva tri rešenja?

\(x_1 = -7\), \(x_2 = -1\), \(x_3 = 7\). Sva tri zadovoljavaju jednačinu i pripadaju odgovarajućim intervalima.

5

Izračunavanje zbira kvadrata

Sada računamo zbir kvadrata svih rešenja:

\[ x_1^2 + x_2^2 + x_3^2 = (-7)^2 + (-1)^2 + 7^2 \] \[ = 49 + 1 + 49 \] \[ = 99 \]
Proveri se: Zašto je \((-7)^2 = 7^2\)?

Jer kvadriranje „poništava" znak: \((-a)^2 = (-a) \cdot (-a) = a^2\). Zato \((-7)^2 = 49 = 7^2\).

Ključni uvid

Jednačina ima tri rešenja — po jedno u svakom intervalu. Interesantno je da su dva od tri rešenja suprotna po znaku (\(-7\) i \(7\)), tako da su im kvadrati jednaki!

\[ (-7)^2 + (-1)^2 + 7^2 = 49 + 1 + 49 = \color{#4ade80}{99} \]

Česte greške

Zaboravljanje provere pripadnosti intervalu: Rešenje linearne jednačine u datom slučaju ne mora nužno pripadati intervalu na kom je dobijeno. Uvek je neophodno proveriti da li rešenje zadovoljava uslov intervala — inače se može desiti da uračunamo lažno rešenje.
Pogrešan znak pri uklanjanju apsolutne vrednosti: Česta greška je pisati \(|x - 3| = x - 3\) na intervalu gde je \(x < 3\). Zapamtite: ako je izraz negativan, apsolutna vrednost menja znak celog izraza, tj. \(|x - 3| = -(x - 3) = -x + 3\).

Izazov za razmišljanje

Koliki je zbir (ne zbir kvadrata!) svih rešenja jednačine \(|x + 4| - |x - 3| = x\)?

Pogledaj nagoveštaj

Rešenja su \(-7\), \(-1\) i \(7\). Saberi ih direktno: \(-7 + (-1) + 7 = -1\). Primetite kako se \(-7\) i \(7\) ponište, a ostaje samo \(-1\).
