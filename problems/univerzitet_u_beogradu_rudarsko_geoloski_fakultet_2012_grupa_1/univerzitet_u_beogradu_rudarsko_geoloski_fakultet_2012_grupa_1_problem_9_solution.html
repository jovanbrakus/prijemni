Plan rešavanja

Najpre ćemo odrediti uslove pod kojima jednačina ima smisla (domen), zatim kvadrirati obe strane da eliminišemo koren, rešiti dobijenu kvadratnu jednačinu, i na kraju proveriti svako rešenje u polaznoj jednačini. Zbir validnih rešenja daje traženi odgovor.

Ključne formule i teoreme
  • Iracionalna jednačina oblika \(\sqrt{f(x)} = g(x)\): Uslov za rešavanje je \(f(x) \geq 0\) i \(g(x) \geq 0\). Kvadriranjem dobijamo \(f(x) = [g(x)]^2\).
  • Kvadratna jednačina \(ax^2 + bx + c = 0\): Rešenja su \(x_{1,2} = \dfrac{-b \pm \sqrt{b^2 - 4ac}}{2a}\). Zbir rešenja: \(x_1 + x_2 = -\dfrac{b}{a}\). Proizvod rešenja: \(x_1 \cdot x_2 = \dfrac{c}{a}\).
  • Lažna (strana) rešenja: Kvadriranje jednačine može uneti lažna rešenja. Svako dobijeno rešenje se mora proveriti u polaznoj jednačini.
  • Faktorizacija: \(x^2 - 3x + 2 = (x-1)(x-2)\)

Grafički prikaz

Grafici funkcija \(y = \sqrt{2x^2 - x + 3}\) (plava kriva) i \(y = x + 1\) (zelena prava). Preseci su rešenja jednačine.

Tačke preseka su rešenja jednačine

Rešenje korak po korak

1

Odredimo uslove (domen)

Da bi jednačina \(\sqrt{2x^2 - x + 3} = x + 1\) imala smisla, moraju biti ispunjena dva uslova:

\[ \text{1) } 2x^2 - x + 3 \geq 0 \qquad \text{2) } x + 1 \geq 0 \]

Uslov 1: Izračunajmo diskriminantu trinoma \(2x^2 - x + 3\):

\[ D = (-1)^2 - 4 \cdot 2 \cdot 3 = 1 - 24 = -23 < 0 \]

Pošto je \(D < 0\) i vodeći koeficijent je pozitivan (\(a = 2 > 0\)), izraz \(2x^2 - x + 3\) je uvek pozitivan za svako realno \(x\). Ovaj uslov je automatski ispunjen.

Uslov 2: Desna strana mora biti nenegativna:

\[ x + 1 \geq 0 \implies x \geq -1 \]

Dakle, rešenja moramo tražiti samo za \(x \geq -1\).

2

Kvadrirajmo obe strane

Pošto su obe strane nenegativne (za \(x \geq -1\)), možemo kvadrirati:

\[ \left(\sqrt{2x^2 - x + 3}\right)^2 = (x + 1)^2 \]
\[ 2x^2 - x + 3 = x^2 + 2x + 1 \]
3

Sredimo jednačinu

Prenesimo sve članove na levu stranu:

\[ 2x^2 - x + 3 - x^2 - 2x - 1 = 0 \]
\[ x^2 - 3x + 2 = 0 \]

Dobili smo standardnu kvadratnu jednačinu.

Proveri se: Koliki je zbir koeficijenata ove kvadratne jednačine?

Zbir koeficijenata: \(1 + (-3) + 2 = 0\). Kada je zbir koeficijenata nula, jedno rešenje je uvek \(x = 1\)!

4

Rešimo kvadratnu jednačinu

Faktorizujemo trinom \(x^2 - 3x + 2\):

\[ x^2 - 3x + 2 = (x - 1)(x - 2) = 0 \]

Odavde su kandidati za rešenja:

\[ x_1 = 1 \qquad \text{i} \qquad x_2 = 2 \]
5

Proverimo rešenja u polaznoj jednačini

Ovo je obavezan korak kod iracionalnih jednačina, jer kvadriranje može uvesti lažna rešenja.

Za \(x_1 = 1\):

\[ \text{Leva strana: } \sqrt{2 \cdot 1^2 - 1 + 3} = \sqrt{2 - 1 + 3} = \sqrt{4} = 2 \] \[ \text{Desna strana: } 1 + 1 = 2 \] \[ 2 = 2 \quad \checkmark \]

Za \(x_2 = 2\):

\[ \text{Leva strana: } \sqrt{2 \cdot 2^2 - 2 + 3} = \sqrt{8 - 2 + 3} = \sqrt{9} = 3 \] \[ \text{Desna strana: } 2 + 1 = 3 \] \[ 3 = 3 \quad \checkmark \]

Oba rešenja zadovoljavaju polaznu jednačinu i uslov \(x \geq -1\). Nema lažnih rešenja!

Proveri se: Zašto je provera neophodna?

Kada kvadriramo jednačinu, dodajemo rešenja jednačine \(\sqrt{2x^2 - x + 3} = -(x+1)\). Ta dodatna rešenja zadovoljavaju kvadriranu jednačinu, ali ne i polaznu. U ovom zadatku smo imali sreće — oba rešenja su validna. Ali u opštem slučaju, provera je neophodna.

6

Izračunajmo zbir rešenja

Oba rešenja su validna, pa je traženi zbir:

\[ x_1 + x_2 = 1 + 2 = 3 \]
Proveri se: Može li se zbir odrediti bez nalaženja pojedinačnih rešenja?

Da! Po Vijetovim formulama, zbir korena jednačine \(x^2 - 3x + 2 = 0\) je \(-\frac{b}{a} = -\frac{-3}{1} = 3\). Ali pažnja — ovo važi samo ako su oba korena validna rešenja polazne iracionalne jednačine. Zato je provera i dalje neophodna!

Ključni uvid

U ovom zadatku, oba korena kvadratne jednačine dobijene kvadriranjem su se pokazala kao validna rešenja polazne jednačine. To se desilo zato što su oba korena (\(x = 1\) i \(x = 2\)) veća od \(-1\), pa je desna strana \(x + 1\) pozitivna u oba slučaja. Zbir je upravo jednak koeficijentu iz Vijetove formule:

\[ x_1 + x_2 = 1 + 2 = 3 \]

Odgovor je (В) \(3\)

Zbir svih rešenja jednačine \(\sqrt{2x^2 - x + 3} = x + 1\) iznosi \(3\).

Verifikacija: Rešenja su \(x_1 = 1\) i \(x_2 = 2\). Provera: \(\sqrt{4} = 2 = 1 + 1\) i \(\sqrt{9} = 3 = 2 + 1\). Zbir: \(1 + 2 = 3\).

Česte greške

  • Izostavljanje provere rešenja: Kod iracionalnih jednačina, kvadriranje može uvesti lažna rešenja. Uvek se moraju proveriti svi kandidati u polaznoj jednačini. U ovom zadatku su oba rešenja validna, ali učenici koji preskoče proveru ne mogu to znati sa sigurnošću.
  • Zaboravljanje uslova \(x + 1 \geq 0\): Desna strana jednačine je \(x + 1\), a koren je po definiciji nenegativan. Ako se dobije kandidat \(x < -1\), on se mora odbaciti čak i ako zadovoljava kvadriranu jednačinu.

Bonus izazov

Rešite jednačinu \(\sqrt{2x^2 - x + 3} = -(x + 1)\). Koliko rešenja ima ova jednačina?

Pogledaj nagoveštaj

Leva strana (koren) je uvek nenegativna, a desna strana \(-(x+1)\) je nenegativna samo za \(x \leq -1\). Kvadriranjem dobijate istu jednačinu \(x^2 - 3x + 2 = 0\) sa rešenjima \(x = 1\) i \(x = 2\). Ali nijedno od ovih rešenja ne zadovoljava uslov \(x \leq -1\), pa ova jednačina nema rešenja.
