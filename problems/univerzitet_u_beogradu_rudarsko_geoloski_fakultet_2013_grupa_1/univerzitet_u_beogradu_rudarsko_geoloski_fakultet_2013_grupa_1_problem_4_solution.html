Број решења једначине са апсолутном вредношћу

Задатак

Број решења једначине \(x^2 + |x - 1| = 1\) је:

(А) 1
(Б) 2
(В) 3
(Г) више од 3

План решавања

  1. Размотрити случајеве за апсолутну вредност: \(x \geq 1\) и \(x < 1\)
  2. У сваком случају решити квадратну једначину
  3. Проверити да ли решења припадају одговарајућем домену
  4. Пребројати укупан број решења

Теоријски подсетник

Апсолутна вредност и раздвајање на случајеве

За функцију \(|f(x)|\) важи:

\[ |f(x)| = \begin{cases} f(x), & \text{ако } f(x) \geq 0 \\ -f(x), & \text{ако } f(x) < 0 \end{cases} \]

За \(|x - 1|\) гранична тачка је \(x = 1\):

\[ |x - 1| = \begin{cases} x - 1, & \text{ако } x \geq 1 \\ 1 - x, & \text{ако } x < 1 \end{cases} \]

Кључно је да после решавања сваког случаја проверимо да ли добијена решења заиста припадају домену тог случаја.

Визуелни приказ

Графици функција \(y = x^2 + |x - 1|\) (плава) и \(y = 1\) (зелена). Пресечне тачке су решења.

Решење корак по корак

1Поставка: раздвајање на случајеве

Једначина гласи:

\[ x^2 + |x - 1| = 1 \]

Апсолутна вредност \(|x - 1|\) мења израз у тачки \(x = 1\), па разматрамо два случаја.

2Случај 1: \(x \geq 1\)

За \(x \geq 1\) важи \(|x - 1| = x - 1\)

Једначина постаје:

\[ x^2 + x - 1 = 1 \] \[ x^2 + x - 2 = 0 \]

Факторишемо:

\[ (x + 2)(x - 1) = 0 \]

Решења су:

\[ x_1 = -2 \quad \text{или} \quad x_2 = 1 \]

Проверавамо домен \(x \geq 1\):

  • \(x = -2\) — не припада домену \(x \geq 1\), одбацујемо
  • \(x = 1\) — припада домену, задржавамо
Проверa: \(1^2 + |1 - 1| = 1 + 0 = 1\) ✓

3Случај 2: \(x < 1\)

За \(x < 1\) важи \(|x - 1| = 1 - x\)

Једначина постаје:

\[ x^2 + 1 - x = 1 \] \[ x^2 - x = 0 \]

Факторишемо:

\[ x(x - 1) = 0 \]

Решења су:

\[ x_1 = 0 \quad \text{или} \quad x_2 = 1 \]

Проверавамо домен \(x < 1\):

  • \(x = 0\) — припада домену, задржавамо
  • \(x = 1\) — не припада домену \(x < 1\) (строга неједнакост), одбацујемо
Проверa: \(0^2 + |0 - 1| = 0 + 1 = 1\) ✓

4Збир решења

Из оба случаја добијамо укупно два решења:

\[ x = 0 \quad \text{и} \quad x = 1 \]

Напомена: \(x = 1\) се појавило у оба случаја, али је прихваћено само у Случају 1 (где је домен \(x \geq 1\)). Нема дуплирања.

Кључни увид

Када решавамо једначине са апсолутном вредношћу, свако добијено решење мора припадати домену случаја из којег је произашло. Од четири кандидата (\(-2, 0, 1, 1\)), само два задовољавају и једначину и услов домена.

Коначан одговор

\(\boxed{\text{Б) } 2}\)

Једначина \(x^2 + |x - 1| = 1\) има тачно два решења: \(x = 0\) и \(x = 1\).

(А) 1
(Б) 2
(В) 3
(Г) више од 3

Честе грешке

Заборављање провере домена: Ако не проверимо да ли решења припадају одговарајућем интервалу, можемо погрешно урачунати \(x = -2\) (из Случаја 1) или дупло рачунати \(x = 1\).
Строга vs. нестрога неједнакост: У Случају 2 (\(x < 1\)), решење \(x = 1\) не припадa домену. Међутим, оно је већ покривено Случајем 1 (\(x \geq 1\)), па се укупан број не мења. Ипак, важно је бити прецизан.
Погрешна факторизација: У Случају 1, ученици понекад погрешно факторишу \(x^2 + x - 2\). Дискриминанта је \(D = 1 + 8 = 9\), решења \(x = \frac{-1 \pm 3}{2}\), односно \(x = 1\) и \(x = -2\).

Додатни изазов

Покушај да решиш сличну једначину: \(x^2 + |x + 2| = 4\). Колико решења има ова једначина? Примени исту технику раздвајања на случајеве.
