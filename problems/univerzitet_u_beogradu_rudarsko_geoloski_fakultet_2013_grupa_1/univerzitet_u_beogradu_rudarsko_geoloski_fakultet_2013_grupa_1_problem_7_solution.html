Троструки композитни логаритам

Решавање једначине са три уgnježdena логаритма различитих основа

Поставка задатка

Решење једначине

\[ \log_3\!\big(\log_2(\log_5 x)\big) = 0 \]

припада интервалу:

(А)
\((0,\; 8)\)
(Б)
\((8,\; 16)\)
(В)
\((16,\; 24)\)
(Г)
\((24,\; 32)\)

План решавања

Имамо три уgnježdena логаритма различитих основа (3, 2 и 5). Решаваћемо једначину „од споља ка унутра" — скидамо слој по слој применом дефиниције логаритма: \(\log_b A = C \iff A = b^C\). Тако ћемо свести проблем на три узастопне просте операције. На крају проверавамо домен и одређујемо ком интервалу припада решење.

Кључне формуле и теореме
  • Дефиниција логаритма: \(\log_b A = C \iff A = b^C\), при чему је \(b > 0\), \(b \neq 1\), \(A > 0\).
  • Логаритам од 1: \(\log_b 1 = 0\) за свако допуштено \(b\), јер је \(b^0 = 1\).
  • Домен логаритма: Аргумент мора бити строго позитиван. Код вишеструке композиције, сваки унутрашњи израз мора бити позитиван.
  • Степен \(b^0 = 1\): Када је логаритам једнак нули, његов аргумент мора бити \(1\).
  • Степен \(b^1 = b\): Када је логаритам једнак \(1\), његов аргумент мора бити једнак основи.

Визуелни приказ

Дијаграм „љуштења слојева" — од спољашњег логаритма ка решењу \(x = 25\)

Решење — корак по корак

1

Скинемо спољашњи слој: \(\log_3\)

Полазна једначина гласи:

\[ \log_3\!\big(\log_2(\log_5 x)\big) = 0 \]

Применимо дефиницију логаритма — ако је \(\log_3(\ldots) = 0\), тада је аргумент једнак \(3^0 = 1\):

\[ \log_2(\log_5 x) = 3^0 = 1 \]

Сваки број степенован нулом даје 1.

Провери себе: Зашто је \(3^0 = 1\)?

По дефиницији, за свако \(b > 0\) важи \(b^0 = 1\). Ово директно следи из правила \(\frac{b^n}{b^n} = b^{n-n} = b^0 = 1\).

2

Скинемо средишњи слој: \(\log_2\)

Сада имамо:

\[ \log_2(\log_5 x) = 1 \]

Применимо дефиницију логаритма — ако је \(\log_2(\ldots) = 1\), тада је аргумент једнак \(2^1 = 2\):

\[ \log_5 x = 2^1 = 2 \]
Провери себе: Колико је \(\log_2(2)\)?

\(\log_2(2) = 1\), јер је \(2^1 = 2\). Одговара нашем резултату из Корака 1.

3

Скинемо унутрашњи слој: \(\log_5\)

Преостаје:

\[ \log_5 x = 2 \]

Применимо дефиницију логаритма по трећи пут:

\[ x = 5^2 = 25 \]
4

Проверимо услове домена

За све три логаритамске функције, аргумент мора бити строго позитиван. Проверимо за \(x = 25\):

\[ \begin{align} &\text{Услов 1 (аргумент } \log_5\text{): } x > 0 \implies 25 > 0 \quad \checkmark \\[6pt] &\text{Услов 2 (аргумент } \log_2\text{): } \log_5 x > 0 \implies \log_5(25) = 2 > 0 \quad \checkmark \\[6pt] &\text{Услов 3 (аргумент } \log_3\text{): } \log_2(\log_5 x) > 0 \implies \log_2(2) = 1 > 0 \quad \checkmark \end{align} \]

Сви услови су задовољени — решење \(x = 25\) припада домену функције.

5

Верификација — уврстимо \(x = 25\) у полазну једначину

Рачунамо од унутра ка споља:

\[ \log_5(25) = \log_5(5^2) = 2 \]
\[ \log_2(2) = 1 \]
\[ \log_3(1) = 0 \quad \checkmark \]

Добили смо тачно 0 — решење је потврђено!

6

Одредимо интервал

Решење је \(x = 25\). Проверимо ком интервалу припада:

\[ 24 < 25 < 32 \implies x \in (24,\; 32) \]

Дакле, тачан одговор је (Г).

Провери себе: Зашто \(x = 25\) не припада интервалу \((16, 24)\)?

Интервал \((16, 24)\) садржи бројеве строго између 16 и 24. Пошто је \(25 > 24\), вредност 25 није у том интервалу. Тачно је \(25 \in (24, 32)\).

Кључни увид

Троструки композитни логаритам решавамо у три узастопна корака — сваки пут скидамо један слој применом дефиниције \(\log_b A = C \iff A = b^C\). Различите основе (3, 2, 5) не мењају поступак!

\[ \underbrace{\log_3}_{3^0=1}\!\Big(\underbrace{\log_2}_{2^1=2}\!\big(\underbrace{\log_5 x}_{5^2=25}\big)\Big) = 0 \implies x = 25 \]

Коначан одговор

\( x = 25 \in (24,\; 32) \)
(А) \((0,\; 8)\)
(Б) \((8,\; 16)\)
(В) \((16,\; 24)\)
(Г) \((24,\; 32)\)

Провера: \(\log_3\!\big(\log_2(\log_5 25)\big) = \log_3\!\big(\log_2(2)\big) = \log_3(1) = 0\) — тачно!

Честе грешке

! Мешање основа логаритама: У овом задатку имамо три различите основе (3, 2, 5). Пазите да при сваком скидању слоја користите тачну основу — грешка у основи води до погрешног резултата.
! Погрешан редослед скидања: Увек крећемо од спољашњег логаритма. Покушај да се прво реши унутрашњи (\(\log_5 x\)) не даје конкретну вредност јер немамо за шта да га изједначимо.
! Заборављање интервала: Задатак не тражи само вредност \(x\), већ интервал коме та вредност припада. Након израчунавања \(x = 25\), неопходно је проверити у који од понуђених интервала упада.
! Непровера домена: Код троструке композиције потребно је проверити три услова позитивности. Ако било који услов не важи, решење отпада.

Додатни изазов

Реши једначину: \(\log_2\!\big(\log_3(\log_7 x)\big) = 1\)

Наговештај

Примени исти поступак: прво \(\log_2(\ldots) = 1 \implies \ldots = 2^1 = 2\), затим \(\log_3(\log_7 x) = 2 \implies \log_7 x = 3^2 = 9\), и на крају \(x = 7^9\). Не заборави проверу домена!
