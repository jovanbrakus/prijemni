🎯 План решавања

Пронаћи ћемо критичне тачке (нуле израза под модулима), поделити бројевну праву на интервале, и на сваком интервалу решити линеарну једначину без модула. Свако решење проверавамо да ли припада свом интервалу.

📐 Кључне формуле и теореме
  • Дефиниција апсолутне вредности:
    \[ |a| = \begin{cases} a, & \text{ако } a \geq 0 \\ -a, & \text{ако } a < 0 \end{cases} \]
  • Метод интервала: За једначину са модулима облика \(|f(x)| + |g(x)| = c\), налазимо нуле функција \(f(x)\) и \(g(x)\), делимо бројевну праву на интервале, и на сваком интервалу уклањамо модуле према знаку израза.
  • Провера решења: Свако добијено решење мора припадати интервалу на коме је једначина постављена — у супротном, решење одбацујемо.
🎨 Графички приказ

Превуците тачку по x-оси да видите вредност функције \(f(x) = |2x+1| + |x-4|\)

🛠 Решење корак по корак
1 Препишемо једначину

Пребацимо \(-6\) на десну страну да добијемо једноставнији облик:

\[ |2x + 1| + |x - 4| = 6 \]

Тражимо вредности \(x\) за које је збир две апсолутне вредности једнак \(6\).

2 Одредимо критичне тачке

Изрази унутар модула мењају знак у тачкама где су једнаки нули:

\[ 2x + 1 = 0 \quad \Rightarrow \quad x = -\frac{1}{2} \]
\[ x - 4 = 0 \quad \Rightarrow \quad x = 4 \]

Ове две тачке деле бројевну праву на три интервала:

\[ \text{I: } x < -\frac{1}{2}, \qquad \text{II: } -\frac{1}{2} \leq x < 4, \qquad \text{III: } x \geq 4 \]
✅ Провери се: Зашто баш ове тачке?

У тачки \(x = -\frac{1}{2}\), израз \(2x+1\) мења знак (прелази из негативног у позитиван). У тачки \(x = 4\), израз \(x-4\) мења знак. Зато морамо засебно анализирати свако „подручје".

3 Интервал I: \(x < -\frac{1}{2}\)

На овом интервалу оба израза су негативна:

\[ 2x + 1 < 0 \quad \Rightarrow \quad |2x+1| = -(2x+1) = -2x - 1 \]
\[ x - 4 < 0 \quad \Rightarrow \quad |x-4| = -(x-4) = -x + 4 \]

Уврштавамо у једначину:

\[ (-2x - 1) + (-x + 4) = 6 \]
\[ -3x + 3 = 6 \]
\[ -3x = 3 \quad \Rightarrow \quad x = -1 \]

Провера припадности: \(x = -1 < -\frac{1}{2}\)? Да, важи!

Верификација: \(|2 \cdot (-1) + 1| + |(-1) - 4| = |-1| + |-5| = 1 + 5 = 6\) ✔

4 Интервал II: \(-\frac{1}{2} \leq x < 4\)

На овом интервалу први израз је неnегативан, а други је негативан:

\[ 2x + 1 \geq 0 \quad \Rightarrow \quad |2x+1| = 2x + 1 \]
\[ x - 4 < 0 \quad \Rightarrow \quad |x-4| = -x + 4 \]

Уврштавамо у једначину:

\[ (2x + 1) + (-x + 4) = 6 \]
\[ x + 5 = 6 \]
\[ x = 1 \]

Провера припадности: \(-\frac{1}{2} \leq 1 < 4\)? Да, важи!

Верификација: \(|2 \cdot 1 + 1| + |1 - 4| = |3| + |-3| = 3 + 3 = 6\) ✔

✅ Провери се: Колико решења имамо до сад?

Из прва два интервала добили смо два решења: \(x = -1\) и \(x = 1\). Остаје још трећи интервал.

5 Интервал III: \(x \geq 4\)

На овом интервалу оба израза су ненегативна:

\[ 2x + 1 > 0 \quad \Rightarrow \quad |2x+1| = 2x + 1 \]
\[ x - 4 \geq 0 \quad \Rightarrow \quad |x-4| = x - 4 \]

Уврштавамо у једначину:

\[ (2x + 1) + (x - 4) = 6 \]
\[ 3x - 3 = 6 \]
\[ 3x = 9 \quad \Rightarrow \quad x = 3 \]

Провера припадности: \(x = 3 \geq 4\)? Не, не важи! Решење \(x = 3\) не припада интервалу \([4, +\infty)\), па га одбацујемо.

Контролна провера: \(|2 \cdot 3 + 1| + |3 - 4| = |7| + |-1| = 7 + 1 = 8 \neq 6\). Заиста, \(x = 3\) није решење оригиналне једначине.

6 Закључак

Укупан број решења: из интервала I добијамо \(x = -1\), из интервала II добијамо \(x = 1\), а из интервала III нема решења. Дакле, једначина има тачно два решења.

\[ x_1 = -1, \qquad x_2 = 1 \]
✅ Провери се: Зашто \(x = 3\) „не ради"?

Иако је \(x = 3\) алгебарско решење једначине \(3x - 3 = 6\), ова једначина важи само за \(x \geq 4\). За \(x = 3\), израз \(x - 4\) је негативан (\(-1\)), па \(|x-4| = 1\), а не \(x - 4 = -1\). Ако уврстимо \(x = 3\) у оригиналну једначину, добијамо \(8 \neq 6\).

Кључни увид

Функција \(f(x) = |2x+1| + |x-4|\) је делимично линеарна — састоји се од три линеарна сегмента. Хоризонтална права \(y = 6\) сече ову функцију у тачно две тачке. Трећи интервал даје „фантомско" решење \(x = 3\), које не припада свом интервалу — класичан пример зашто је провера припадности интервалу обавезна.

⚠️ Честе грешке
Прихватање решења без провере интервала

Многи ученици добију \(x = 3\) из трећег интервала и прихвате га без провере. Међутим, \(x = 3 \notin [4, +\infty)\), па то решење не важи. Увек проверите да ли решење припада интервалу на коме сте уклонили модуле!

Погрешно уклањање модула

На интервалу где је \(2x + 1 < 0\), ставља се \(|2x+1| = -(2x+1) = -2x - 1\), а не \(|2x+1| = 2x + 1\). Знак минус испред целе заграде значи да се мења знак свих чланова.
