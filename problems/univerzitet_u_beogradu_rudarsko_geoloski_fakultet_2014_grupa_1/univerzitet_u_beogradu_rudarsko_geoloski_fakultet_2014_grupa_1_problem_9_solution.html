Plan rešavanja

Sve logaritme sa različitim osnovama (16, 4 i 2) svešćemo na zajedničku osnovu 2 korišćenjem formule za promenu osnove. Tako ćemo jednačinu svesti na linearnu jednačinu po promenljivoj \(t = \log_2 x\), iz koje nalazimo \(x\).

Teorijska osnova
Formula za promenu osnove logaritma
  • Promena osnove: \(\log_a x = \dfrac{\log_b x}{\log_b a}\) za bilo koju osnovu \(b > 0, \; b \neq 1\)
  • Specijalan slučaj: Ako je osnova \(a\) stepen neke osnove \(b\), tj. \(a = b^k\), tada: \(\log_{b^k} x = \dfrac{\log_b x}{k}\)
  • Definicija logaritma: \(\log_a x = y \iff a^y = x\), pri čemu je \(a > 0, \; a \neq 1, \; x > 0\)

Ovo nam omogućava da sve logaritme sa različitim osnovama svedemo na jednu zajedničku osnovu.

Stepeni broja 2

Korisno je zapamtiti:

  • \(2^1 = 2\), \(2^2 = 4\), \(2^3 = 8\), \(2^4 = 16\)
  • \(2^5 = 32\), \(2^6 = 64\), \(2^7 = 128\), \(2^8 = 256\)
  • \(2^9 = 512\), \(2^{10} = 1024\)

Uočavamo da je \(4 = 2^2\) i \(16 = 2^4\), što znači da su sve tri osnove u zadatku stepeni broja 2.

Grafički prikaz

Pomeri klizač da istražiš kako se menja zbir \(\log_{16} x + \log_4 x + \log_2 x\) u zavisnosti od \(x\), i nađi gde taj zbir iznosi tačno 14.

Vrednost \(x\):
x = 256
Rešenje korak po korak
1 Uočavamo da su osnove 16, 4 i 2 stepeni broja 2

Sve tri osnove logaritama u zadatku možemo zapisati kao stepene broja 2:

\[16 = 2^4, \quad 4 = 2^2, \quad 2 = 2^1\]

Ovo nam sugeriše da sve logaritme svedemo na osnovu 2.

2 Primenjujemo formulu za promenu osnove

Koristimo pravilo \(\log_{b^k} x = \dfrac{\log_b x}{k}\) da svaki logaritam izrazimo pomoću \(\log_2 x\):

\[\log_{16} x = \log_{2^4} x = \frac{\log_2 x}{4}\]
\[\log_4 x = \log_{2^2} x = \frac{\log_2 x}{2}\]
\[\log_2 x = \log_2 x\]
3 Uvodimo smenu \(t = \log_2 x\)

Neka je \(t = \log_2 x\). Tada naša jednačina postaje:

\[\frac{t}{4} + \frac{t}{2} + t = 14\]

Umesto tri logaritma sa različitim osnovama, sada imamo jednostavnu linearnu jednačinu po \(t\).

Proveri se: Zašto smemo uvesti smenu \(t = \log_2 x\)?

Zato što su sva tri logaritma izražena pomoću \(\log_2 x\). Smena samo skraćuje zapis i čini jednačinu preglednijom. Uslov je \(x > 0\), pa je \(t\) dobro definisano.

4 Svodimo na zajednički imenilac

Zajednički imenilac razlomaka \(\frac{t}{4}\), \(\frac{t}{2}\) i \(t\) je 4:

\[\frac{t}{4} + \frac{2t}{4} + \frac{4t}{4} = 14\]

Sabiramo brojioce:

\[\frac{t + 2t + 4t}{4} = 14\] \[\frac{7t}{4} = 14\]
5 Rešavamo linearnu jednačinu po \(t\)

Množimo obe strane sa 4:

\[7t = 56\]

Delimo sa 7:

\[t = 8\]
Proveri se: Koliko je \(\frac{8}{4} + \frac{8}{2} + 8\)?

\(\frac{8}{4} + \frac{8}{2} + 8 = 2 + 4 + 8 = 14\). Tačno, naše \(t = 8\) zadovoljava jednačinu!

6 Nalazimo \(x\) iz smene

Vraćamo se na smenu \(t = \log_2 x\):

\[\log_2 x = 8\]

Po definiciji logaritma (\(\log_a x = y \iff x = a^y\)):

\[x = 2^8 = 256\]
7 Određujemo interval kome pripada \(x = 256\)

Proveravamo svaki ponuđeni interval:

  • \((50,\; 100)\) — Pošto je \(256 > 100\), rešenje nije u ovom intervalu.
  • \((100,\; 200)\) — Pošto je \(256 > 200\), rešenje nije u ovom intervalu.
  • \((200,\; 300)\) — Važi \(200 < 256 < 300\), rešenje jeste u ovom intervalu.
  • \((300,\; 400)\) — Pošto je \(256 < 300\), rešenje nije u ovom intervalu.
Proveri se: Zameni \(x = 256\) u polaznu jednačinu

\(\log_{16} 256 = 2\) jer \(16^2 = 256\).

\(\log_4 256 = 4\) jer \(4^4 = 256\).

\(\log_2 256 = 8\) jer \(2^8 = 256\).

Zbir: \(2 + 4 + 8 = 14\). Jednačina važi!

Ključni uvid

Kada se u jednačini pojave logaritmi sa različitim osnovama koje su stepeni istog broja (ovde: \(2, 4 = 2^2, 16 = 2^4\)), sve se može svesti na jednu osnovu. Smenom \(t = \log_2 x\) logaritamska jednačina prelazi u linearnu:

\[\frac{t}{4} + \frac{t}{2} + t = 14 \;\;\Longrightarrow\;\; t = 8 \;\;\Longrightarrow\;\; x = 2^8 = 256\]
Konačan odgovor
\(x = 256 \in (200,\; 300)\)
(A) \((50,\; 100)\)
(B) \((100,\; 200)\)
(V) \((200,\; 300)\)
(G) \((300,\; 400)\)

Tačan odgovor je (V)

Verifikacija: Zamenjujemo \(x = 256\) u polaznu jednačinu: \(\log_{16} 256 + \log_4 256 + \log_2 256 = 2 + 4 + 8 = 14\). Jednakost važi. Broj 256 pripada otvorenom intervalu \((200,\; 300)\) jer važi \(200 < 256 < 300\).
Česte greške
  • Pogrešna primena formule za promenu osnove: pisati \(\log_{16} x = 4 \cdot \log_2 x\) umesto \(\log_{16} x = \frac{\log_2 x}{4}\). Zapamtite: veća osnova daje manji logaritam, pa delimo (ne množimo) sa eksponentom.
  • Zaboraviti da se vrati na promenljivu \(x\) nakon nalaženja \(t\). Rešenje nije \(t = 8\) nego \(x = 2^8 = 256\). Uvek proveriti šta se tačno traži!
Dodatni izazov
Reši jednačinu: \(\log_9 x + \log_3 x + \log_{\sqrt{3}} x = 13\)

Savet: Uoči da je \(9 = 3^2\) i \(\sqrt{3} = 3^{1/2}\). Sve svedi na osnovu 3 koristeći smenu \(t = \log_3 x\).

Dobićeš: \(\frac{t}{2} + t + 2t = 13\), tj. \(\frac{7t}{2} = 13\), odakle \(t = \frac{26}{7}\) i \(x = 3^{26/7}\).
