Nejednačina sa apsolutnom vrednošću

Algebra — Rešavanje nejednačina sa apsolutnom vrednošću raščlanjivanjem na slučajeve

📋 Tekst zadatka

Skup svih rešenja nejednačine \( x + |x + 2| < 0 \) u skupu realnih brojeva je:

(A)
\( (-\infty, -2) \)
(B)
\( (-\infty, +\infty) \)
(C)
\( (-\infty, -1) \)
(D)
\( (-\infty, 1) \)

🧭 Plan rešavanja

Raščlanićemo apsolutnu vrednost \( |x + 2| \) na dva slučaja prema znaku izraza \( x + 2 \) — kada je \( x \geq -2 \) i kada je \( x < -2 \). U svakom slučaju rešavamo linearnu nejednačinu i tražimo presek sa uslovom. Na kraju ujedinjujemo rešenja.

📐 Ključne formule i pojmovi
  • Definicija apsolutne vrednosti: \( |a| = \begin{cases} a, & \text{ako } a \geq 0 \\ -a, & \text{ako } a < 0 \end{cases} \)
  • Metod slučajeva: Za izraz \( |f(x)| \) posmatramo kritičnu tačku gde je \( f(x) = 0 \). Realna osa se deli na intervale u kojima \( f(x) \) ne menja znak.
  • Presek i unija intervala: U svakom slučaju nalazimo presek rešenja nejednačine sa uslovom za taj slučaj. Konačno rešenje je unija svih slučajeva.
  • Linearna nejednačina: Nejednačina oblika \( ax + b < 0 \) ima rešenje \( x < -\frac{b}{a} \) za \( a > 0 \), odnosno \( x > -\frac{b}{a} \) za \( a < 0 \).

📊 Interaktivni brojevni pravac

Prevucite klizač da proverite vrednost nejednačine za različite vrednosti \( x \)

📝 Rešenje korak po korak

1

Identifikacija kritične tačke

Apsolutna vrednost \( |x + 2| \) menja ponašanje tamo gde je izraz unutar nje jednak nuli:

\[ x + 2 = 0 \quad \Longrightarrow \quad x = -2 \]

Ova tačka deli brojevni pravac na dva intervala: \( x < -2 \) i \( x \geq -2 \).

2

Slučaj 1: \( x \geq -2 \)

Slučaj 1

Kada je \( x \geq -2 \), izraz \( x + 2 \geq 0 \), pa je \( |x + 2| = x + 2 \).

Nejednačina postaje:

\[ x + (x + 2) < 0 \] \[ 2x + 2 < 0 \] \[ 2x < -2 \] \[ x < -1 \]

Presek sa uslovom \( x \geq -2 \):

\[ x \geq -2 \quad \text{i} \quad x < -1 \quad \Longrightarrow \quad x \in [-2,\; -1) \]

Iz prvog slučaja dobijamo interval \( [-2, -1) \).

🔍 Proveri se: Koliko iznosi \( f(-1{,}5) = (-1{,}5) + |(-1{,}5) + 2| \)?
\( f(-1{,}5) = -1{,}5 + |0{,}5| = -1{,}5 + 0{,}5 = -1 < 0 \) — tačka \( x = -1{,}5 \) jeste rešenje.
3

Slučaj 2: \( x < -2 \)

Slučaj 2

Kada je \( x < -2 \), izraz \( x + 2 < 0 \), pa je \( |x + 2| = -(x + 2) = -x - 2 \).

Nejednačina postaje:

\[ x + (-x - 2) < 0 \] \[ x - x - 2 < 0 \] \[ -2 < 0 \]

Ovo je uvek tačna tvrdnja! Nejednačina je ispunjena za svako \( x \) iz ovog slučaja.

\[ x \in (-\infty,\; -2) \]

Iz drugog slučaja dobijamo ceo interval \( (-\infty, -2) \).

🔍 Proveri se: Zašto je nejednačina uvek tačna za \( x < -2 \)?
Zato što se \( x \) i \( -x \) poništavaju, pa ostaje samo \( -2 < 0 \), što ne zavisi od \( x \) i uvek važi.
4

Unija rešenja

Konačan skup rešenja je unija oba slučaja:

\[ (-\infty,\; -2) \;\cup\; [-2,\; -1) \;=\; (-\infty,\; -1) \]

Intervali se „nadovezuju" (tačka \( x = -2 \) pripada drugom intervalu), pa se spajaju u jedan.

5

Provera graničnih tačaka

Proverimo da li granična tačka \( x = -1 \) pripada skupu rešenja:

\[ f(-1) = (-1) + |(-1) + 2| = -1 + |1| = -1 + 1 = 0 \]

Pošto je \( 0 \not< 0 \), tačka \( x = -1 \) nije rešenje. Interval je otvoren sa desne strane.

🔍 Proveri se: Koliko iznosi \( f(-2) \)?
\( f(-2) = (-2) + |(-2) + 2| = -2 + |0| = -2 + 0 = -2 < 0 \). Tačka \( x = -2 \) jeste rešenje.

💡 Ključni uvid

Za \( x < -2 \), apsolutna vrednost „poništava" \( x \), ostavljajući konstantu \( -2 < 0 \) — nejednačina važi automatski. Pravi rad je samo u intervalu \( [-2, +\infty) \)!

\[ x + |x + 2| < 0 \quad \Longleftrightarrow \quad x \in (-\infty,\; -1) \]

✅ Konačan odgovor

\( x \in (-\infty,\; -1) \)
(A) \( (-\infty, -2) \)
(B) \( (-\infty, +\infty) \)
(C) \( (-\infty, -1) \)
(D) \( (-\infty, 1) \)
Verifikacija:
  • \( x = -3 \): \( -3 + |-1| = -3 + 1 = -2 < 0 \) — rešenje ✓
  • \( x = -2 \): \( -2 + |0| = -2 + 0 = -2 < 0 \) — rešenje ✓
  • \( x = -1{,}5 \): \( -1{,}5 + |0{,}5| = -1{,}5 + 0{,}5 = -1 < 0 \) — rešenje ✓
  • \( x = -1 \): \( -1 + |1| = -1 + 1 = 0 \not< 0 \) — nije rešenje ✓
  • \( x = 0 \): \( 0 + |2| = 0 + 2 = 2 \not< 0 \) — nije rešenje ✓

Odgovor \( (-\infty, -1) \) obuhvata tačno ona \( x \) za koja je nejednačina ispunjena. Odgovor A bi propustio interval \( [-2, -1) \).

⚠ Česte greške

  • Zaboravljen slučaj: Ako se razmatra samo \( x \geq -2 \), dobija se \( [-2, -1) \) umesto punog rešenja \( (-\infty, -1) \). Obavezno treba obraditi oba slučaja!
  • Pogrešna unija: Odgovor (A) daje samo \( (-\infty, -2) \) — to je samo drugi slučaj, bez prvog. Rešenja iz oba slučaja se moraju ujediniti.
  • Uključivanje granice: Tačka \( x = -1 \) daje vrednost 0, a nejednačina je stroga (\( < \)), pa \( -1 \) ne pripada rešenju.

🏆 Dodatni izazov

Reši nejednačinu \( x + |x + 2| \leq -1 \). Kako se rešenje menja kada umesto strogog \( < 0 \) imamo \( \leq -1 \)?

💡 Nagoveštaj

Primeni isti metod dva slučaja. U prvom slučaju (\( x \geq -2 \)) dobićeš \( 2x + 2 \leq -1 \), tj. \( x \leq -1{,}5 \). U drugom slučaju (\( x < -2 \)), dobićeš \( -2 \leq -1 \), što je uvek tačno. Razmisli o uniji!
