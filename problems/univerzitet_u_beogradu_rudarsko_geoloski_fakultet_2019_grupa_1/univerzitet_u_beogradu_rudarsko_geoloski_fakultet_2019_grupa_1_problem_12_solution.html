Vrednost sin 2x ako je sin x - cos x = a

Trigonometrijski identiteti i formula za dupli ugao

Zadatak Tekst zadatka

Ako je \(\sin x - \cos x = a\), onda je \(\sin 2x\) jednako:

(A) \(1 - a^2\)
(B) \(1 + a^2\)
(C) \(\sqrt{1 - a^2}\)
(D) \(1 + 2a^2\)

Plan Kako cemo resiti

  1. Kvadrirati obe strane jednakosti \(\sin x - \cos x = a\).
  2. Razviti kvadrat binoma na levoj strani.
  3. Primeniti Pitagorin identitet \(\sin^2 x + \cos^2 x = 1\).
  4. Prepoznati formulu za dupli ugao \(2\sin x \cos x = \sin 2x\).
  5. Izraziti \(\sin 2x\) preko \(a\).

Teorija Podsetnik

Pitagorin trigonometrijski identitet

Za svako \(x\) vazi fundamentalni identitet:

\[\sin^2 x + \cos^2 x = 1\]

Ovo je posledica Pitagorine teoreme na jedinicnom krugu. Ovaj identitet je kljucan za eliminisanje \(\sin^2 x + \cos^2 x\) u racunu.

Formula za dupli ugao (sin 2x)

Sinus dvostrukog ugla je:

\[\sin 2x = 2 \sin x \cos x\]

Ovo sledi iz adicione formule \(\sin(x + x) = \sin x \cos x + \cos x \sin x = 2\sin x \cos x\).

Proizvod \(\sin x \cdot \cos x\) se pojavljuje kad kvadriramo \(\sin x \pm \cos x\), sto je osnova cele strategije.

Kvadrat binoma

Podsetimo se formule za kvadrat razlike:

\[(A - B)^2 = A^2 - 2AB + B^2\]

Kada stavimo \(A = \sin x\) i \(B = \cos x\):

\[(\sin x - \cos x)^2 = \sin^2 x - 2\sin x \cos x + \cos^2 x\]

Vizuelni prikaz Algebarski tok resavanja

Resenje Korak po korak

Korak 1 — Kvadriranje obe strane

Polazimo od date jednakosti i kvadriramo obe strane:

\[\sin x - \cos x = a\] \[(\sin x - \cos x)^2 = a^2\]

Korak 2 — Razvijanje kvadrata binoma

Primenjujemo formulu \((A - B)^2 = A^2 - 2AB + B^2\) sa \(A = \sin x\) i \(B = \cos x\):

\[\sin^2 x - 2\sin x \cos x + \cos^2 x = a^2\]
Kontrolna tacka 1: Na levoj strani prepoznajemo \(\sin^2 x + \cos^2 x\) i proizvod \(2\sin x \cos x\).

Korak 3 — Primena Pitagorinog identiteta

Preuredimo levu stranu grupisanjem:

\[(\sin^2 x + \cos^2 x) - 2\sin x \cos x = a^2\]

Posto je \(\sin^2 x + \cos^2 x = 1\), dobijamo:

\[1 - 2\sin x \cos x = a^2\]

Korak 4 — Prepoznavanje formule za dupli ugao

Znamo da je \(2\sin x \cos x = \sin 2x\), pa zamenjujemo:

\[1 - \sin 2x = a^2\]
Kontrolna tacka 2: Jednacina \(1 - \sin 2x = a^2\) sadrzi tacno ono sto trazimo — \(\sin 2x\).

Korak 5 — Izrazavanje sin 2x

Prebacimo \(\sin 2x\) na jednu stranu:

\[\sin 2x = 1 - a^2\]
Kontrolna tacka 3: Rezultat \(\sin 2x = 1 - a^2\) odgovara ponudenom odgovoru (A).

Kljucni uvid

Celokupna strategija pociva na jednoj ideji: kvadriranjem izraza \(\sin x \pm \cos x\) prirodno se pojavljuju oba kljucna identiteta — Pitagorin identitet i formula za dupli ugao.

Kad god u zadatku vidite \(\sin x \pm \cos x\) i traze vas \(\sin 2x\) (ili obrnuto), kvadriranje je standardni pristup. To je zato sto:

\[(\sin x \pm \cos x)^2 = \underbrace{\sin^2 x + \cos^2 x}_{= 1} \pm \underbrace{2\sin x \cos x}_{= \sin 2x}\]

Dakle, uvek vazi: \((\sin x \pm \cos x)^2 = 1 \pm \sin 2x\).

Rezultat Konacan odgovor

sin 2x =
\(1 - a^2\)
(A) 1 - a²
(B) 1 + a²
(C) √(1 - a²)
(D) 1 + 2a²

Zamke Ceste greske

Pogresan znak pri razvijanju kvadrata: Najcesca greska je pisanje \((\sin x - \cos x)^2 = \sin^2 x + \cos^2 x + 2\sin x \cos x\) umesto \(\sin^2 x + \cos^2 x - 2\sin x \cos x\). Kvadrat razlike ima minus ispred dvostrukog proizvoda!
Brkanje sa korenom: Neki ucenici pokusavaju da vade koren iz \(1 - a^2\) i biraju odgovor (C). Medjutim, kvadriranjem smo dobili tacan algebarski identitet — koren nije potreban.
Zaboravljanje da je sin 2x = 2 sin x cos x: Bez ove formule za dupli ugao, zadatak se ne moze resiti ovim pristupom. Ova formula je jedna od najvaznijih u trigonometriji i treba je znati napamet.
Zamena + i - u konacanom rezultatu: Da je u zadatku stajalo \(\sin x + \cos x = a\), dobili bismo \(\sin 2x = a^2 - 1\). Znak zavisi od toga da li je u polaznome izrazu plus ili minus.

Bonus Dodatni izazov

Ako je \(\sin x + \cos x = b\), odredi \(\sin 2x\) i \(\sin^4 x + \cos^4 x\) u funkciji od \(b\).

Pogledaj resenje

Deo 1: Kvadriramo \(\sin x + \cos x = b\):

\[(\sin x + \cos x)^2 = b^2\] \[\sin^2 x + 2\sin x \cos x + \cos^2 x = b^2\] \[1 + \sin 2x = b^2\] \[\sin 2x = b^2 - 1\]

Deo 2: Koristimo identitet \(\sin^4 x + \cos^4 x = (\sin^2 x + \cos^2 x)^2 - 2\sin^2 x \cos^2 x\):

\[\sin^4 x + \cos^4 x = 1 - 2\sin^2 x \cos^2 x = 1 - \frac{\sin^2 2x}{2} = 1 - \frac{(b^2 - 1)^2}{2}\]
