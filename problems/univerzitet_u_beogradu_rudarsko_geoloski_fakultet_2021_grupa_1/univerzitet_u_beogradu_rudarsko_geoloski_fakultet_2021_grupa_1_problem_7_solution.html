Nejednačina sa apsolutnom vrednošću

Zadatak 7 Tekst zadatka

Skup svih rešenja nejednačine \( x + |x + 2| \leq -2 \) u skupu realnih brojeva je:

\(\text{A) } (-\infty,\, -2]\)
\(\text{B) } \{-2\}\)
\(\text{C) } (-\infty,\, -1)\)
\(\text{D) } \varnothing\)

Plan rešavanja

  1. Odrediti kritičnu tačku za apsolutnu vrednost: \(x + 2 = 0 \Rightarrow x = -2\).
  2. Slučaj 1: Za \(x \geq -2\), ukloniti apsolutnu vrednost kao \(|x+2| = x+2\) i rešiti nejednačinu.
  3. Slučaj 2: Za \(x < -2\), ukloniti apsolutnu vrednost kao \(|x+2| = -(x+2)\) i rešiti nejednačinu.
  4. Spojiti rešenja oba slučaja.

Teorija Osvrt na pojmove

Apsolutna vrednost — definicija i osobine

Apsolutna vrednost broja \(a\) definiše se kao:

\[ |a| = \begin{cases} a, & \text{ako } a \geq 0 \\ -a, & \text{ako } a < 0 \end{cases} \]

Kada rešavamo nejednačine sa apsolutnom vrednošću, ključni korak je podela na slučajeve prema znaku izraza unutar apsolutnih zagrada. Kritična tačka je vrednost za koju je izraz jednak nuli.

Metod rešavanja po slučajevima

Za nejednačinu oblika \(f(x) + |g(x)| \leq c\):

  1. Nađemo kritičnu tačku: \(g(x) = 0\).
  2. Podelimo realnu osu na intervale oko kritične tačke.
  3. Na svakom intervalu, \(|g(x)|\) zamenjujemo sa \(g(x)\) ili \(-g(x)\).
  4. Rešimo dobijenu nejednačinu i presečemo sa intervalom tog slučaja.
  5. Konačno rešenje je unija rešenja svih slučajeva.

Vizuelizacija Grafički prikaz

Grafici funkcija \(y = x + |x+2|\) (plava) i \(y = -2\) (zelena). Rešenje su sve vrednosti \(x\) gde je plava kriva ispod ili na zelenoj pravoj.

Rešenje Korak po korak

Kritična tačka

Izraz unutar apsolutne vrednosti je \(x + 2\). Jednak je nuli kada:

\[ x + 2 = 0 \quad \Longrightarrow \quad x = -2 \]

Ovo deli realnu osu na dva intervala: \(x \geq -2\) i \(x < -2\).

Slučaj 1: \(x \geq -2\)

Kada je \(x \geq -2\), važi \(x + 2 \geq 0\), pa je \(|x+2| = x + 2\). Nejednačina postaje:

\[ x + (x + 2) \leq -2 \] \[ 2x + 2 \leq -2 \] \[ 2x \leq -4 \] \[ x \leq -2 \]

Presek sa uslovom slučaja \(x \geq -2\):

\[ x \leq -2 \;\;\text{i}\;\; x \geq -2 \quad \Longrightarrow \quad x = -2 \]

Provera: \(x = -2\): \(-2 + |-2+2| = -2 + 0 = -2 \leq -2\)   Tačno!

Kontrolna tačka: Iz prvog slučaja dobijamo samo jednu tačku: \(x = -2\).

Slučaj 2: \(x < -2\)

Kada je \(x < -2\), važi \(x + 2 < 0\), pa je \(|x+2| = -(x+2) = -x - 2\). Nejednačina postaje:

\[ x + (-x - 2) \leq -2 \] \[ x - x - 2 \leq -2 \] \[ -2 \leq -2 \]

Ovo je uvek tačno — važi za svaki realan broj! Dakle, uslov nejednačine je ispunjen za sve vrednosti \(x\) iz ovog slučaja.

Presek sa uslovom slučaja \(x < -2\):

\[ x \in (-\infty,\, -2) \]
Kontrolna tačka: Iz drugog slučaja svi \(x < -2\) su rešenja. Proverimo npr. \(x = -5\): \(-5 + |-5+2| = -5 + 3 = -2 \leq -2\). Tačno!

Unija rešenja

Ukupno rešenje je unija rešenja iz oba slučaja:

\[ \{-2\} \cup (-\infty,\, -2) = (-\infty,\, -2] \]
Završna provera: \(x = -10\): \(-10 + |-10+2| = -10 + 8 = -2 \leq -2\). Tačno! \(x = 0\): \(0 + |0+2| = 0 + 2 = 2 \not\leq -2\). Nije rešenje. Sve se slaže.

Ključni uvid

U drugom slučaju (\(x < -2\)), članovi \(x\) i \(-x\) se potiru, i nejednačina se svodi na identitet \(-2 \leq -2\), koji je uvek tačan. To znači da je ceo interval \((-\infty, -2)\) skup rešenja. Sa granicom \(x = -2\) iz prvog slučaja, dobijamo zatvoren interval \((-\infty, -2]\).

Konačan odgovor

\(\text{A) } (-\infty,\, -2]\)

Greške Česte zamke

  • Zaboraviti granični slučaj: Ako se u slučaju 1 ne proveri tačka \(x = -2\), može se pogrešno zaključiti da je rešenje samo \((-\infty, -2)\) bez granice, ili se zaokruži odgovor D (prazan skup) misleći da \(x \leq -2\) i \(x \geq -2\) nemaju presek.
  • Pogrešan znak: Pri uklanjanju apsolutne vrednosti za \(x < -2\), česta greška je pisanje \(|x+2| = x + 2\) umesto \(-(x+2)\). To dovodi do potpuno pogrešnog rešenja.
  • Odgovor B) \(\{-2\}\): Ovo je zamka za one koji reše samo prvi slučaj i zaborave drugi. Uvek treba proveriti oba slučaja!
  • Odgovor D) \(\varnothing\): Neki učenici pomisle da \(x + |x+2|\) ne može biti negativno. Ali za dovoljno negativne \(x\), funkcija upravo dostiže vrednost \(-2\).

Izazov Za razmišljanje

Reši sličnu nejednačinu: \( x + |x + 2| \leq k \) za proizvoljno \(k \in \mathbb{R}\). Za koje vrednosti \(k\) skup rešenja nije prazan? Kako izgleda rešenje u zavisnosti od \(k\)?

Savet: Razmotri šta se dešava kada je \(k < -2\), \(k = -2\) i \(k > -2\).
