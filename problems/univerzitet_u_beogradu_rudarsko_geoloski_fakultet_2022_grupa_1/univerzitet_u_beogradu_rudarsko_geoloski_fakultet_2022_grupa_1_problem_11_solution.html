Kosinus razlike uglova

Trigonometrija — primena adicione formule i Pitagorine trigonometrijske teoreme

Tekst zadatka

Ako je \((\sin\alpha + \sin\beta)^2 = 1\) i \((\cos\alpha + \cos\beta)^2 = 2\), tada je \(\cos(\alpha - \beta)\) jednako:

Uslov 1
\((\sin\alpha + \sin\beta)^2 = 1\)
Uslov 2
\((\cos\alpha + \cos\beta)^2 = 2\)
Trazi se
\(\cos(\alpha - \beta)\)
(A)
\(\dfrac{\sqrt{3}}{2}\)
(B)
\(1\)
(C)
\(\dfrac{1}{2}\)
(D)
\(\dfrac{1}{3}\)

Plan resavanja

Razvicemo oba kvadrata binoma, zatim ih sabrati. Koristicemo cinjenicu da je \(\sin^2 x + \cos^2 x = 1\) i prepoznacemo adicionu formulu za kosinus razlike uglova \(\cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta\).

Kljucne formule i teoreme
  • Kvadrat binoma: \((a + b)^2 = a^2 + 2ab + b^2\)
  • Osnovna trigonometrijska jednakost (Pitagorina teorema): \(\sin^2 x + \cos^2 x = 1\) za svaki ugao \(x\)
  • Kosinus razlike uglova (adiciona formula): \(\cos(\alpha - \beta) = \cos\alpha \cdot \cos\beta + \sin\alpha \cdot \sin\beta\)
  • Veza sa zadatkom: Izraz \(\sin\alpha\sin\beta + \cos\alpha\cos\beta\) se direktno pojavljuje kad saberemo razvijene kvadrate — to je upravo \(\cos(\alpha - \beta)\)

Vizuelni prikaz

Pomeranjem klizaca mozete menjati uglove alfa i beta i pratiti kako se menja cos(alfa - beta)



Resenje korak po korak

1

Razvijamo prvi kvadrat binoma

Primenimo formulu \((a+b)^2 = a^2 + 2ab + b^2\) na izraz \((\sin\alpha + \sin\beta)^2\):

\[ (\sin\alpha + \sin\beta)^2 = \sin^2\alpha + 2\sin\alpha\sin\beta + \sin^2\beta = 1 \quad \ldots (I) \]

Ovde \(a = \sin\alpha\), \(b = \sin\beta\). Dvostruki proizvod daje \(2\sin\alpha\sin\beta\).

2

Razvijamo drugi kvadrat binoma

Na isti nacin razvijemo i drugi dati izraz:

\[ (\cos\alpha + \cos\beta)^2 = \cos^2\alpha + 2\cos\alpha\cos\beta + \cos^2\beta = 2 \quad \ldots (II) \]

Potpuno analogan postupak — sada sa kosinusima.

Proveri sebe: Sta dobijamo sabiranjem levih strana jednacina (I) i (II)?
Dobijamo: \(\sin^2\alpha + \sin^2\beta + \cos^2\alpha + \cos^2\beta + 2\sin\alpha\sin\beta + 2\cos\alpha\cos\beta\). Desna strana je \(1 + 2 = 3\).
3

Sabiramo jednacine (I) i (II)

Saberemo leve i desne strane dve jednacine:

\[ \sin^2\alpha + 2\sin\alpha\sin\beta + \sin^2\beta + \cos^2\alpha + 2\cos\alpha\cos\beta + \cos^2\beta = 1 + 2 \]

Desna strana je jednostavno:

\[ 1 + 2 = 3 \]
4

Grupišemo clanove po uglovima

Preuredimo levu stranu tako da grupišemo \(\sin^2\) i \(\cos^2\) istog ugla zajedno:

\[ \underbrace{(\sin^2\alpha + \cos^2\alpha)}_{= 1} + \underbrace{(\sin^2\beta + \cos^2\beta)}_{= 1} + 2(\sin\alpha\sin\beta + \cos\alpha\cos\beta) = 3 \]

Koristimo osnovnu trigonometrijsku jednakost: \(\sin^2 x + \cos^2 x = 1\) za svaki ugao \(x\).

5

Primenjujemo Pitagorinu trigonometrijsku teoremu

Zamenom \(\sin^2\alpha + \cos^2\alpha = 1\) i \(\sin^2\beta + \cos^2\beta = 1\) dobijamo:

\[ 1 + 1 + 2(\sin\alpha\sin\beta + \cos\alpha\cos\beta) = 3 \]

Pojednostavljeno:

\[ 2 + 2(\sin\alpha\sin\beta + \cos\alpha\cos\beta) = 3 \]
Proveri sebe: Koju poznatu formulu prepoznajes u izrazu \(\sin\alpha\sin\beta + \cos\alpha\cos\beta\)?
To je adiciona formula za kosinus razlike: \(\cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta\). Redosled sabiraka nije bitan jer je sabiranje komutativno!
6

Prepoznajemo adicionu formulu

Izraz \(\sin\alpha\sin\beta + \cos\alpha\cos\beta\) je upravo \(\cos(\alpha - \beta)\):

\[ 2 + 2\cos(\alpha - \beta) = 3 \]
7

Resavamo po \(\cos(\alpha - \beta)\)

Oduzimamo 2 sa obe strane:

\[ 2\cos(\alpha - \beta) = 3 - 2 = 1 \]

Delimo sa 2:

\[ \cos(\alpha - \beta) = \frac{1}{2} \]
Proveri sebe: Za koji poznati ugao vazi \(\cos\theta = \frac{1}{2}\)?
\(\cos 60° = \frac{1}{2}\) (odnosno \(\cos\frac{\pi}{3} = \frac{1}{2}\)). Dakle \(\alpha - \beta = \pm 60° + k \cdot 360°\), sto potvrdjuje da je rezultat realan i smislen.

Kljucni uvid

Trik je u tome sto sabiranjem razvijenih kvadratnih izraza prirodno nastaje kombinacija \(\sin^2 + \cos^2 = 1\) i adiciona formula. Umesto da trazimo pojedinacne uglove, direktno dobijamo trazeni kosinus razlike!

\[ \underbrace{(\sin\alpha + \sin\beta)^2}_{=1} + \underbrace{(\cos\alpha + \cos\beta)^2}_{=2} = 2 + 2\cos(\alpha - \beta) = 3 \]

Konacan odgovor

\(\cos(\alpha - \beta) = \dfrac{1}{2}\)
(A)
\(\dfrac{\sqrt{3}}{2}\)
(B)
\(1\)
(C)
\(\dfrac{1}{2}\)
(D)
\(\dfrac{1}{3}\)
Verifikacija: Rezultat \(\cos(\alpha - \beta) = \frac{1}{2}\) odgovara razlici uglova od \(60°\). Mozemo proveriti konzistentnost: ako stavimo \(\alpha - \beta = 60°\), uz pomocu formule zbira dobijamo \(\sin\alpha + \sin\beta = 2\sin\frac{\alpha+\beta}{2}\cos 30° = \sqrt{3}\sin\frac{\alpha+\beta}{2}\) i \(\cos\alpha + \cos\beta = \sqrt{3}\cos\frac{\alpha+\beta}{2}\). Uslov 1 daje \(3\sin^2\frac{\alpha+\beta}{2} = 1\), uslov 2 daje \(3\cos^2\frac{\alpha+\beta}{2} = 2\), a zbir je \(\frac{1}{3} + \frac{2}{3} = 1\). Sve se poklapa!

Ceste greske

  • Pokusaj nalazenja pojedinacnih uglova: Nije potrebno (i nije moguce iz datih podataka) odrediti \(\alpha\) i \(\beta\) pojedinacno. Trazeni izraz \(\cos(\alpha - \beta)\) se dobija direktno sabiranjem jednacina.
  • Zamena adicione formule: Pazite da ne pomesate \(\cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta\) sa \(\cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta\). Razlika ima plus, zbir ima minus!
  • Zaboravljen faktor 2: Pri razvoju kvadrata binoma, dvostruki proizvod \(2ab\) je kljucan. Bez faktora 2 dolazite do pogresnog rezultata.

Dodatni izazov

Ako je \((\sin\alpha + \sin\beta)^2 = 3\) i \((\cos\alpha + \cos\beta)^2 = 1\), koliko je \(\cos(\alpha - \beta)\)?

Pogledaj nagoveštaj

Primeni potpuno isti postupak: razvij, saberi, prepoznaj formule. Zbir desnih strana je sada \(3 + 1 = 4\), pa je \(2 + 2\cos(\alpha - \beta) = 4\), odakle je \(\cos(\alpha - \beta) = 1\). To znaci da je \(\alpha = \beta\) (uglovi su jednaki)!
