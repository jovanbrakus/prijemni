Logaritamska nejednačina sa složenom bazom

📝 Tekst zadatka
Neka je \(S\) skup svih rešenja nejednačine \[ \log_{\frac{1}{3}}\!\left(\log_{x^2 - 2x + 1}\, x^2\right) \geq 0 \] Tada za neke realne brojeve \(a, b, c\) i \(d\), \(a < b < c < d\), skup \(S\) je oblika:
(A)
\((-\infty,\, a)\)
(B)
\((-\infty,\, a] \cup (b,\, c)\)
(C)
\((-\infty,\, a) \cup [b,\, c) \cup (c,\, d)\)
(D)
\((-\infty,\, a) \cup [b,\, c)\)
(E)
\((-\infty,\, a]\)
🎯 Plan rešavanja

1. Prepoznajemo da je \(x^2 - 2x + 1 = (x-1)^2\) i određujemo uslove domena.
2. Koristimo svojstvo logaritma sa bazom manjom od 1 da prevedemo nejednačinu na ekvivalentan oblik.
3. Uprostimo unutrašnji logaritam na odnos \(\frac{\ln|x|}{\ln|x-1|}\).
4. Razdvajamo na slučajeve prema znaku \(\ln|x-1|\) i rešavamo svaki posebno.
5. Ujedinjujemo rešenja i identifikujemo oblik skupa.

📚 Podsetnik iz teorije
Logaritam sa bazom manjom od 1

Za bazu \(0 < a < 1\) važi:

\[ \log_a y \geq 0 \iff 0 < y \leq 1 \]

Logaritamska funkcija sa bazom manjom od 1 je opadajuća: veće vrednosti argumenta daju manje vrednosti logaritma. Vrednost 0 se dostiže za \(y = 1\), a za \(0 < y < 1\) logaritam je pozitivan.

Promena baze logaritma

Za bilo koji logaritam važi formula promene baze:

\[ \log_b c = \frac{\ln c}{\ln b} \]

Ovo nam omogućava da unutrašnji logaritam zapišemo kao razlomak prirodnih logaritama, što značajno olakšava analizu.

Uslovi definisanosti logaritamske funkcije

Da bi \(\log_b c\) bio definisan, moraju biti ispunjeni uslovi:

\[ b > 0, \quad b \neq 1, \quad c > 0 \]
🎨 Grafik funkcije \(g(x) = \frac{\ln|x|}{\ln|x-1|}\)

Zeleno osjenčene oblasti prikazuju skup rešenja \(S = (-\infty,\, -1) \cup \left[\tfrac{1}{2},\, 1\right)\)

🛠 Rešenje korak po korak
1 Određujemo domen

Prepoznajemo da je baza unutrašnjeg logaritma potpun kvadrat:

\[ x^2 - 2x + 1 = (x-1)^2 \]

Da bi logaritam \(\log_{(x-1)^2}(x^2)\) bio definisan, moraju biti ispunjeni sledeći uslovi:

\[ \begin{align} &(x-1)^2 > 0 \quad &\Rightarrow \quad x \neq 1 \\[4pt] &(x-1)^2 \neq 1 \quad &\Rightarrow \quad x \neq 0 \;\text{ i }\; x \neq 2 \\[4pt] &x^2 > 0 \quad &\Rightarrow \quad x \neq 0 \end{align} \]

Domen je:

\[ \mathcal{D} = \mathbb{R} \setminus \{0,\, 1,\, 2\} \]
Zašto \((x-1)^2 \neq 1\)?

Baza logaritma mora biti pozitivna i različita od 1. Jednačina \((x-1)^2 = 1\) daje \(x - 1 = \pm 1\), odakle \(x = 0\) ili \(x = 2\).

2 Prevodimo spoljašnju nejednačinu

Baza spoljašnjeg logaritma je \(\frac{1}{3} < 1\). Za opadajuću logaritamsku funkciju važi:

\[ \log_{\frac{1}{3}} y \geq 0 \iff 0 < y \leq 1 \]

Označimo \(y = \log_{(x-1)^2}(x^2)\). Nejednačina postaje:

\[ 0 < \log_{(x-1)^2}(x^2) \leq 1 \]
3 Uprostimo unutrašnji logaritam

Primenom formule promene baze i svojstava logaritma:

\[ \log_{(x-1)^2}(x^2) = \frac{\ln(x^2)}{\ln\!\big((x-1)^2\big)} = \frac{2\ln|x|}{2\ln|x-1|} = \frac{\ln|x|}{\ln|x-1|} \]

Nejednačina se svodi na:

\[ 0 < \frac{\ln|x|}{\ln|x-1|} \leq 1 \]

Ovo uprošćavanje je ključno jer se parni eksponenti u brojiocu i imeniocu skraćuju, ostavljajući razlomak apsolutnih vrednosti.

Kontrolna tačka 1

Nejednačinu smo sveli na: odrediti gde je količnik \(\frac{\ln|x|}{\ln|x-1|}\) strogo pozitivan i ne veći od 1. Dalje razlažemo na slučajeve prema znaku imenioca.

4 Slučaj 1: \(\ln|x-1| > 0\)

Uslov \(\ln|x-1| > 0\) znači \(|x-1| > 1\), tj.:

\[ x - 1 < -1 \;\;\text{ ili }\;\; x - 1 > 1 \quad\Longrightarrow\quad x < 0 \;\;\text{ ili }\;\; x > 2 \]
Uslov \(0 < \frac{\ln|x|}{\ln|x-1|}\) (imenilac pozitivan)

Kada je imenilac pozitivan, razlomak je pozitivan akko je i brojilac pozitivan:

\[ \ln|x| > 0 \;\;\Longrightarrow\;\; |x| > 1 \;\;\Longrightarrow\;\; x < -1 \;\text{ ili }\; x > 1 \]

Presek sa \(x < 0\) ili \(x > 2\):

\[ x < -1 \;\;\text{ ili }\;\; x > 2 \]
Uslov \(\frac{\ln|x|}{\ln|x-1|} \leq 1\) (imenilac pozitivan)

Pošto je imenilac pozitivan, množimo nejednačinu imeniocem bez promene znaka:

\[ \ln|x| \leq \ln|x-1| \;\;\Longrightarrow\;\; |x| \leq |x-1| \]

Kvadriramo obe strane (obe su nenegativne):

\[ x^2 \leq x^2 - 2x + 1 \;\;\Longrightarrow\;\; 0 \leq -2x + 1 \;\;\Longrightarrow\;\; x \leq \frac{1}{2} \]

Presek svih uslova iz Slučaja 1:

\[ (x < -1 \;\text{ ili }\; x > 2) \;\;\cap\;\; \left(x \leq \tfrac{1}{2}\right) = \boxed{x < -1} \]

Oblast \(x > 2\) otpada jer protivreči uslovu \(x \leq \frac{1}{2}\).

Provera: \(x = -2\)

Baza: \((x-1)^2 = (-3)^2 = 9\). Argument: \(x^2 = 4\).
Unutrašnji logaritam: \(\log_9 4 = \frac{\ln 4}{\ln 9} = \frac{2\ln 2}{2\ln 3} \approx 0{,}631\).
Spoljašnji: \(\log_{1/3}(0{,}631) = \frac{\ln 0{,}631}{\ln(1/3)} = \frac{-0{,}461}{-1{,}099} \approx 0{,}42 \geq 0\). Tačno!

5 Slučaj 2: \(\ln|x-1| < 0\)

Uslov \(\ln|x-1| < 0\) znači \(|x-1| < 1\), tj.:

\[ -1 < x - 1 < 1 \quad\Longrightarrow\quad 0 < x < 2 \quad (x \neq 1) \]
Uslov \(0 < \frac{\ln|x|}{\ln|x-1|}\) (imenilac negativan)

Kada je imenilac negativan, razlomak je pozitivan akko je i brojilac negativan:

\[ \ln|x| < 0 \;\;\Longrightarrow\;\; |x| < 1 \;\;\Longrightarrow\;\; -1 < x < 1 \]

Presek sa \(0 < x < 2\), \(x \neq 0\), \(x \neq 1\):

\[ 0 < x < 1 \]
Uslov \(\frac{\ln|x|}{\ln|x-1|} \leq 1\) (imenilac negativan)

Pošto je imenilac negativan, množenje imeniocem menja smer nejednačine:

\[ \ln|x| \geq \ln|x-1| \;\;\Longrightarrow\;\; |x| \geq |x-1| \]

Kvadriramo:

\[ x^2 \geq x^2 - 2x + 1 \;\;\Longrightarrow\;\; 2x \geq 1 \;\;\Longrightarrow\;\; x \geq \frac{1}{2} \]

Presek svih uslova iz Slučaja 2:

\[ (0 < x < 1) \;\;\cap\;\; \left(x \geq \tfrac{1}{2}\right) = \boxed{\frac{1}{2} \leq x < 1} \]
Provera graničnih tačaka

\(x = \frac{1}{2}\): Baza: \((x-1)^2 = \frac{1}{4}\). Argument: \(x^2 = \frac{1}{4}\).
\(\log_{1/4}\!\left(\frac{1}{4}\right) = 1\). Zatim \(\log_{1/3}(1) = 0 \geq 0\). Tačno! (Granična tačka pripada skupu jer imamo \(\geq\).)

\(x = 1\): Baza \((x-1)^2 = 0\) nije definisana. Isključeno iz domena.

Kontrolna tačka 2

Slučaj 1 daje \((-\infty, -1)\), slučaj 2 daje \([\frac{1}{2}, 1)\). Ostaje da proverimo da nema rešenja u preostalim oblastima.

6 Provera preostalih oblasti

Proverimo tačke iz oblasti koje nisu pokrivene prethodnim slučajevima:

\(-1 < x < 0\) (npr. \(x = -\frac{1}{2}\))

\(|x| = \frac{1}{2}\), \(|x-1| = \frac{3}{2}\). Dakle \(\ln|x| < 0\) i \(\ln|x-1| > 0\).
Količnik: \(\frac{\ln(1/2)}{\ln(3/2)} = \frac{-0{,}693}{0{,}405} \approx -1{,}71 < 0\). Nije zadovoljen uslov \(> 0\).

\(0 < x < \frac{1}{2}\) (npr. \(x = 0{,}3\))

\(|x| = 0{,}3\), \(|x-1| = 0{,}7\). Oba logaritma negativna.
Količnik: \(\frac{\ln 0{,}3}{\ln 0{,}7} = \frac{-1{,}204}{-0{,}357} \approx 3{,}37 > 1\). Nije zadovoljen uslov \(\leq 1\).

\(1 < x < 2\) (npr. \(x = 1{,}5\))

\(|x| = 1{,}5\), \(|x-1| = 0{,}5\). Dakle \(\ln|x| > 0\) i \(\ln|x-1| < 0\).
Količnik: \(\frac{\ln 1{,}5}{\ln 0{,}5} = \frac{0{,}405}{-0{,}693} \approx -0{,}585 < 0\). Nije zadovoljen uslov \(> 0\).

\(x > 2\) (npr. \(x = 3\))

\(|x| = 3\), \(|x-1| = 2\). Oba logaritma pozitivna.
Količnik: \(\frac{\ln 3}{\ln 2} \approx 1{,}585 > 1\). Nije zadovoljen uslov \(\leq 1\).

Nijedna od preostalih oblasti ne sadrži rešenja. Naš skup je potpun.

7 Formiramo konačan skup rešenja

Ujedinjujemo rezultate oba slučaja:

\[ S = (-\infty,\, -1) \cup \left[\frac{1}{2},\, 1\right) \]

Ovo je oblika \((-\infty,\, a) \cup [b,\, c)\) gde:

\[ a = -1, \quad b = \frac{1}{2}, \quad c = 1, \quad d = 2 \quad (a < b < c < d) \]

Vrednost \(d = 2\) služi samo da zadovolji uslov \(a < b < c < d\) iz teksta zadatka (sve četiri tačke su granične tačke domena). Odgovor je (D).

Zašto nije (C)?

Opcija (C) uključuje dodatni interval \((c, d) = (1, 2)\). Međutim, za \(1 < x < 2\) smo pokazali da je \(\frac{\ln|x|}{\ln|x-1|} < 0\), pa taj interval ne pripada skupu rešenja. Opcija (C) je pogrešna.

Ključni uvid

Cela nejednačina se svodi na analizu količnika \(\frac{\ln|x|}{\ln|x-1|}\). Uočite da se parni eksponenti iz \(x^2\) i \((x-1)^2\) skraćuju pri promeni baze, pa se složena logaritamska nejednačina pretvara u elementarnu analizu znakova i veličina dva prirodna logaritma. Ključni tehnika je razlaganje na slučajeve prema znaku imenioca jer množenje negativnim brojem obrće nejednačinu.

\[ \log_{(x-1)^2}(x^2) = \frac{\ln|x|}{\ln|x-1|} \quad\longrightarrow\quad \text{analiza po znaku }\ln|x-1| \]
Konačni odgovor
\(S = (-\infty,\, -1) \cup \left[\tfrac{1}{2},\, 1\right)\)
(A) \((-\infty,\, a)\) (B) \((-\infty,\, a] \cup (b,\, c)\) (C) \((-\infty,\, a) \cup [b,\, c) \cup (c,\, d)\) (D) \((-\infty,\, a) \cup [b,\, c)\) (E) \((-\infty,\, a]\)

Verifikacija: Za \(x = -2\): \(\log_9 4 \approx 0{,}63\), pa \(\log_{1/3}(0{,}63) \approx 0{,}42 \geq 0\). Za \(x = \frac{1}{2}\): \(\log_{1/4}\frac{1}{4} = 1\), pa \(\log_{1/3}(1) = 0 \geq 0\). Za \(x = 3\): \(\frac{\ln 3}{\ln 2} \approx 1{,}59 > 1\), pa \(\log_{1/3}(1{,}59) < 0\). ✔

⚠️ Česte greške
Zaboravljanje uslova domena

Tačke \(x = 0\), \(x = 1\) i \(x = 2\) moraju biti isključene. Naročito je česta greška da se ne uoči da \((x-1)^2 = 1\) daje \(x = 0\) ili \(x = 2\), pa da se baza 1 ne isključi.

Pogrešan smer nejednačine pri množenju negativnim

U Slučaju 2, imenilac \(\ln|x-1|\) je negativan. Kada množimo obe strane nejednačine negativnim brojem, nejednačina \(\leq\) postaje \(\geq\). Propuštanje ove promene smera vodi na pogrešno rešenje \(0 < x \leq \frac{1}{2}\) umesto \(\frac{1}{2} \leq x < 1\).

Tretiranje \((x-1)^2\) kao uvek veće baze

Baza \((x-1)^2\) može biti veća ili manja od 1 u zavisnosti od \(x\). Za \(0 < x < 2\) je \((x-1)^2 < 1\) (baza manja od 1), a za \(x < 0\) ili \(x > 2\) je \((x-1)^2 > 1\). Ova razlika fundamentalno menja ponašanje logaritma.

🏆 Dodatni izazov

Odredite skup svih rešenja nejednačine \[ \log_{\frac{1}{2}}\!\left(\log_{x^2 + 4x + 4}\, x^2\right) > 0 \]

Nagoveštaj

Uočite da je \(x^2 + 4x + 4 = (x+2)^2\). Primenite identičan postupak: uprostite logaritam na \(\frac{\ln|x|}{\ln|x+2|}\) i analizirajte nejednačinu \(0 < \frac{\ln|x|}{\ln|x+2|} < 1\) (ovde je stroga nejednačina jer je spoljašnji uslov \(> 0\) umesto \(\geq 0\)). Domen: \(x \neq 0, x \neq -2, x \neq -1, x \neq -3\).
