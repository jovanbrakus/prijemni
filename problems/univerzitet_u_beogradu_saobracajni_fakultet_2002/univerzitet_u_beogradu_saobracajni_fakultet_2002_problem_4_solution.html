🎯 Plan rešavanja

Prvo ćemo izračunati vrednost \(a\) koristeći svojstva stepena i logaritama (pretvaranje baze 4 u bazu 2). Zatim ćemo izračunati \(b\) pažljivim izračunavanjem kvadratnog i kubnog korena. Na kraju ćemo stepenovanje \(c = a^b\) svesti na razlomak.

📚 Podsetnik iz teorije
Svojstva stepena

Stepen stepena — eksponenti se množe:

\[ (a^m)^n = a^{m \cdot n} \]

Stepen količnika:

\[ a^{m - n} = \frac{a^m}{a^n} \]
Osnovno svojstvo logaritma

Po definiciji logaritma važi:

\[ a^{\log_a x} = x \]

Takođe, za množenje eksponenta logaritmom:

\[ n \cdot \log_a x = \log_a x^n \]
Kvadratni i kubni koren

Kvadratni koren je uvek nenegativan:

\[ \sqrt{x^2} = |x| \quad \text{(za svako realno } x\text{)} \]

Kubni koren realnog broja zadržava znak:

\[ \sqrt[3]{-1} = -1 \quad \text{jer } (-1)^3 = -1 \]

🛠 Rešenje korak po korak
1 Pretvaramo bazu broja \(a\)

Osnova stepena je \(4\). Pošto se u eksponentu pojavljuje \(\log_2 5\), zgodno je bazu pretvoriti u dvojku:

\[ 4 = 2^2 \]

Pa zamenom dobijamo:

\[ a = 4^{1 - \log_2 5} = \left(2^2\right)^{1 - \log_2 5} \]
2 Primenjujemo pravilo stepena stepena

Po pravilu \((a^m)^n = a^{m \cdot n}\), eksponente množimo:

\[ a = 2^{2 \cdot (1 - \log_2 5)} = 2^{2 - 2\log_2 5} \]
3 Sređujemo eksponent pomoću svojstava logaritma

Koristimo pravilo \(n \cdot \log_a x = \log_a x^n\):

\[ 2\log_2 5 = \log_2 5^2 = \log_2 25 \]

Dakle:

\[ a = 2^{2 - \log_2 25} \]
Proveri se: Koliko iznosi \(\log_2 25\) približno?

\(\log_2 25 \approx 4{,}64\), jer \(2^4 = 16\) i \(2^5 = 32\), a \(25\) je između. Dakle eksponent \(2 - 4{,}64 \approx -2{,}64\), što daje broj manji od 1.

4 Razdvajamo stepen na razlomak

Koristimo pravilo \(a^{m-n} = \frac{a^m}{a^n}\):

\[ a = \frac{2^2}{2^{\log_2 25}} \]

Brojilac je jednostavan: \(2^2 = 4\).

Za imenilac koristimo osnovno svojstvo logaritma \(a^{\log_a x} = x\):

\[ 2^{\log_2 25} = 25 \]
5 Dobijamo vrednost \(a\)

Zamenom u razlomak:

\[ a = \frac{4}{25} \]
Proveri se: Da li \(\frac{4}{25}\) odgovara približnoj vrednosti?

\(\frac{4}{25} = 0{,}16\). Ranije smo procenili da je eksponent oko \(-2{,}64\), pa \(2^{-2{,}64} \approx 0{,}16\). Sve se slaže!

6 Računamo \(\sqrt{(-1)^2}\)

Prvo izračunavamo izraz unutar korena:

\[ (-1)^2 = 1 \]

Sada vadimo kvadratni koren. Kvadratni koren je uvek nenegativan broj:

\[ \sqrt{(-1)^2} = \sqrt{1} = 1 \]

Napomena: Važi opšte pravilo \(\sqrt{x^2} = |x|\), pa je \(\sqrt{(-1)^2} = |-1| = 1\).

7 Računamo \(\sqrt[3]{-1}\)

Kubni koren negativnog broja je negativan, jer je kubna funkcija neparna:

\[ \sqrt[3]{-1} = -1 \quad \text{jer } (-1)^3 = -1 \]
8 Dobijamo vrednost \(b\)

Zamenjujemo u izraz za \(b\):

\[ b = \sqrt{(-1)^2} - \sqrt[3]{-1} = 1 - (-1) = 1 + 1 = 2 \]
Proveri se: Zašto se oduzima negativan broj?

Izraz kaže \(b = \sqrt{(-1)^2} - \sqrt[3]{-1}\). Kubni koren od \(-1\) je \(-1\), pa oduzimanje daje \(1 - (-1) = 1 + 1 = 2\). Minus i minus daju plus!

9 Računamo \(c = a^b\)

Sada imamo sve što nam treba:

\[ a = \frac{4}{25}, \quad b = 2 \]

Stepenujemo razlomak:

\[ c = a^b = \left(\frac{4}{25}\right)^2 = \frac{4^2}{25^2} = \frac{16}{625} \]
Ključni uvid

Celokupan zadatak se svodi na tri odvojene veštine: pretvaranje baze stepena da odgovara bazi logaritma, pažljivo razlikovanje parnog i neparnog korena (kvadratni koren je nenegativan, kubni zadržava znak), i stepenovanje razlomka. Ove tri veštine zajedno daju elegantan rezultat.

\[ 4^{1-\log_2 5} = \frac{4}{25}, \quad 1 - (-1) = 2, \quad \left(\frac{4}{25}\right)^{\!2} = \frac{16}{625} \]
Konačni odgovor
\( c = \dfrac{16}{625} \)
(A) \( c = \dfrac{16}{625} \) (B) \( c = 1 \) (C) \( c = \dfrac{625}{16} \) (D) \( c = \dfrac{5}{2} \) (E) \( c = 0{,}25 \)

Verifikacija: \(\left(\frac{4}{25}\right)^2 = \frac{16}{625} = 0{,}0256\). Numerička provera: \(4^{1 - \log_2 5} \approx 4^{1 - 2{,}322} = 4^{-1{,}322} \approx 0{,}16\), pa \(0{,}16^2 \approx 0{,}0256\). ✔

⚠️ Česte greške
Pogrešno tumačenje \(\sqrt{(-1)^2}\)

Česta greška je pisati \(\sqrt{(-1)^2} = -1\). Kvadratni koren uvek daje nenegativan rezultat: \(\sqrt{x^2} = |x|\), pa je rezultat \(1\), ne \(-1\). Ako se napravi ova greška, dobija se \(b = 0\) i \(c = 1\) (opcija B).

Zamena brojioca i imenioca

Ako se pogrešno izračuna \(a = \frac{25}{4}\) umesto \(\frac{4}{25}\) (npr. zamenom reda u razlomku), dobija se \(c = \frac{625}{16}\) (opcija C) — recipročna vrednost tačnog odgovora.

🏆 Dodatni izazov

Odredite vrednost izraza \(d = 9^{1 - \log_3 7} + \sqrt{(-2)^2} - \sqrt[3]{-8}\).

Nagoveštaj

Koristite isti princip: \(9 = 3^2\), pa \(9^{1 - \log_3 7} = 3^{2(1 - \log_3 7)} = \frac{9}{49}\). Zatim \(\sqrt{(-2)^2} = 2\) i \(\sqrt[3]{-8} = -2\). Saberite sve tri vrednosti.
