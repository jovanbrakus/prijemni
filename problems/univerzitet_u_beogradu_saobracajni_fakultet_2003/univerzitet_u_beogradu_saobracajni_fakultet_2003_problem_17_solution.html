Trigonometrijska jednačina sa sin²

Zbir rešenja trigonometrijske jednačine na datom intervalu — identitet razlike kvadrata sinusa

Postavka zadatka

Zbir svih rešenja jednačine

\[\sin^2 x + \sin^2\!\left(\frac{2\pi}{3} + x\right) - \sin^2\!\left(\frac{2\pi}{3} - x\right) = 0\]

na intervalu \(\left[-\dfrac{\pi}{4},\, \dfrac{5\pi}{4}\right]\) je:

(A)
\(\pi\)
(C)
\(\dfrac{\pi}{3}\)
(E)
\(\dfrac{5\pi}{3}\)
(G)
\(\dfrac{4\pi}{3}\)
(I)
\(2\pi\)

Plan rešavanja

Primenićemo identitet \(\sin^2 A - \sin^2 B = \sin(A+B)\cdot\sin(A-B)\) na poslednja dva člana jednačine, čime ćemo je svesti na proizvod elementarnih trigonometrijskih funkcija. Potom nalazimo sva rešenja na datom intervalu i sabiramo ih.

Ključne formule i teoreme
  • Razlika kvadrata sinusa: \(\sin^2 A - \sin^2 B = \sin(A+B)\cdot\sin(A-B)\)
  • Dupli ugao: \(\sin 2x = 2\sin x \cos x\)
  • Osnovne vrednosti: \(\sin\!\left(\dfrac{4\pi}{3}\right) = -\dfrac{\sqrt{3}}{2}\)
  • Rešenja sin x = 0: \(x = k\pi,\; k \in \mathbb{Z}\)
  • Rešenja tan x = \sqrt{3}: \(x = \dfrac{\pi}{3} + k\pi,\; k \in \mathbb{Z}\)

Vizuelni prikaz rešenja na jediničnoj kružnici

Rešenja su označena na jediničnoj kružnici: x = 0, x = π/3 i x = π

Rešenje korak po korak

1

Identifikacija strukture jednačine

Jednačina ima oblik:

\[\sin^2 x + \sin^2\!\left(\frac{2\pi}{3} + x\right) - \sin^2\!\left(\frac{2\pi}{3} - x\right) = 0\]

Poslednja dva člana su razlika kvadrata sinusa, pa možemo primeniti identitet \(\sin^2 A - \sin^2 B = \sin(A+B)\sin(A-B)\).

2

Primena identiteta na poslednja dva člana

Neka je \(A = \dfrac{2\pi}{3} + x\) i \(B = \dfrac{2\pi}{3} - x\). Tada:

\[A + B = \frac{2\pi}{3} + x + \frac{2\pi}{3} - x = \frac{4\pi}{3}\]
\[A - B = \left(\frac{2\pi}{3} + x\right) - \left(\frac{2\pi}{3} - x\right) = 2x\]

Prema identitetu:

\[\sin^2\!\left(\frac{2\pi}{3} + x\right) - \sin^2\!\left(\frac{2\pi}{3} - x\right) = \sin\frac{4\pi}{3} \cdot \sin 2x\]
3

Izračunavanje sin(4π/3)

Ugao \(\dfrac{4\pi}{3} = \pi + \dfrac{\pi}{3}\) se nalazi u trećem kvadrantu, gde je sinus negativan:

\[\sin\frac{4\pi}{3} = -\sin\frac{\pi}{3} = -\frac{\sqrt{3}}{2}\]
Proveri se: Koliko iznosi sin(4π/3)?

Tačan odgovor: \(\sin\dfrac{4\pi}{3} = -\dfrac{\sqrt{3}}{2} \approx -0{,}866\)

4

Zamena u jednačinu

Jednačina postaje:

\[\sin^2 x + \left(-\frac{\sqrt{3}}{2}\right)\cdot \sin 2x = 0\]

Koristimo \(\sin 2x = 2\sin x\cos x\):

\[\sin^2 x - \frac{\sqrt{3}}{2} \cdot 2\sin x\cos x = 0\]
\[\sin^2 x - \sqrt{3}\,\sin x\cos x = 0\]
5

Izvlačenje zajedničkog faktora

Oba člana sadrže \(\sin x\) kao zajednički faktor:

\[\sin x\,\bigl(\sin x - \sqrt{3}\,\cos x\bigr) = 0\]

Proizvod je nula ako je barem jedan činilac jednak nuli.

6

Slučaj 1: sin x = 0

Opšte rešenje:

\[x = k\pi, \quad k \in \mathbb{Z}\]

Na intervalu \(\left[-\dfrac{\pi}{4},\, \dfrac{5\pi}{4}\right]\):

  • \(k = 0\): \(x = 0\) — važi, jer \(0 \in \left[-\dfrac{\pi}{4},\, \dfrac{5\pi}{4}\right]\) ✓
  • \(k = 1\): \(x = \pi \approx 3{,}14\) — važi, jer \(\pi < \dfrac{5\pi}{4} \approx 3{,}93\) ✓
  • \(k = -1\): \(x = -\pi \approx -3{,}14 < -\dfrac{\pi}{4}\) ✗

Rešenja iz ovog slučaja: x = 0 i x = π

Proveri se: Koje je rešenje jednačine sin x = 0 najbliže broju 5?

To je \(x = 2\pi \approx 6{,}28\) ili \(x = \pi \approx 3{,}14\). Najbliže broju 5 je \(x = 2\pi\), ali ono nije u našem intervalu.

7

Slučaj 2: sin x − √3 cos x = 0

Iz jednačine \(\sin x = \sqrt{3}\,\cos x\) delimo obe strane sa \(\cos x\) (za \(\cos x \neq 0\)):

\[\tan x = \sqrt{3}\]

Opšte rešenje:

\[x = \frac{\pi}{3} + k\pi, \quad k \in \mathbb{Z}\]

Na intervalu \(\left[-\dfrac{\pi}{4},\, \dfrac{5\pi}{4}\right]\):

  • \(k = 0\): \(x = \dfrac{\pi}{3} \approx 1{,}047\) ✓
  • \(k = 1\): \(x = \dfrac{\pi}{3} + \pi = \dfrac{4\pi}{3} \approx 4{,}189\), ali \(\dfrac{5\pi}{4} \approx 3{,}927\), pa \(\dfrac{4\pi}{3} > \dfrac{5\pi}{4}\) ✗
  • \(k = -1\): \(x = \dfrac{\pi}{3} - \pi = -\dfrac{2\pi}{3} \approx -2{,}094 < -\dfrac{\pi}{4}\) ✗

Rešenje iz ovog slučaja: x = π/3

Napomena: za rešenja sin x = 0, imamo cos x = ±1 ≠ 0, tako da deljenje sa cos x ne gubi rešenja.

8

Sva rešenja i njihov zbir

Sva rešenja na intervalu \(\left[-\dfrac{\pi}{4},\, \dfrac{5\pi}{4}\right]\):

\[x_1 = 0, \quad x_2 = \frac{\pi}{3}, \quad x_3 = \pi\]

Zbir:

\[0 + \frac{\pi}{3} + \pi = \frac{\pi}{3} + \frac{3\pi}{3} = \frac{4\pi}{3}\]
Proveri se: Da li x = 4π/3 pripada intervalu [-π/4, 5π/4]?

Ne! Jer \(\dfrac{4\pi}{3} \approx 4{,}189 > \dfrac{5\pi}{4} \approx 3{,}927\). Upravo zato je važno pažljivo proveriti granice intervala.

Ključni uvid

Identitet \(\sin^2 A - \sin^2 B = \sin(A+B)\sin(A-B)\) pretvara komplikovanu jednačinu u proizvod elementarnih funkcija:

\[\sin x\,(\sin x - \sqrt{3}\,\cos x) = 0\]

Konačan odgovor

\[\frac{4\pi}{3}\]
(A) \(\pi\) (C) \(\dfrac{\pi}{3}\) (E) \(\dfrac{5\pi}{3}\) (G) \(\dfrac{4\pi}{3}\) (I) \(2\pi\)
Verifikacija: Proveravamo svako rešenje u polaznoj jednačini:
Za \(x = 0\): \(\sin^2 0 + \sin^2\!\tfrac{2\pi}{3} - \sin^2\!\tfrac{2\pi}{3} = 0\) ✓
Za \(x = \tfrac{\pi}{3}\): \(\sin^2\!\tfrac{\pi}{3} + \sin^2\!\pi - \sin^2\!\tfrac{\pi}{3} = \tfrac{3}{4} + 0 - \tfrac{3}{4} = 0\) ✓
Za \(x = \pi\): \(\sin^2\!\pi + \sin^2\!\tfrac{5\pi}{3} - \sin^2\!\bigl(-\tfrac{\pi}{3}\bigr) = 0 + \tfrac{3}{4} - \tfrac{3}{4} = 0\) ✓
Zbir: \(0 + \dfrac{\pi}{3} + \pi = \dfrac{4\pi}{3}\) — odgovara opciji (G).

Česte greške

  • Pogrešna provera granica intervala: Mnogi učenici uključe \(x = \dfrac{4\pi}{3}\) misleći da pripada intervalu, ali \(\dfrac{4\pi}{3} \approx 4{,}189 > \dfrac{5\pi}{4} \approx 3{,}927\). Ovo bi dovelo do zbira \(\dfrac{4\pi}{3} + \dfrac{4\pi}{3} = \dfrac{8\pi}{3}\), što nije među ponuđenim odgovorima — ili do pogrešnog odgovora \(2\pi\) ako se zabunom uključi \(\dfrac{4\pi}{3}\) a izostavi neko drugo rešenje.
  • Zaboravljanje rešenja sin x = 0: Prilikom deljenja sa sin x (ili cos x) lako se izgubi rešenje. Uvek najpre izdvojiti zajednički faktor, pa tek onda rešavati svaki činilac posebno.

Dodatni izazov

Kako bi se promenio zbir rešenja da je interval bio \(\left[-\dfrac{\pi}{4},\, \dfrac{3\pi}{2}\right]\)?

Pomoć

U proširenom intervalu, rešenje \(x = \dfrac{4\pi}{3}\) bi sada pripadalo intervalu jer \(\dfrac{4\pi}{3} < \dfrac{3\pi}{2}\). Zbir bi tada bio \(0 + \dfrac{\pi}{3} + \pi + \dfrac{4\pi}{3} = \dfrac{8\pi}{3}\).
