🎯 Plan rešavanja

Prvo ćemo uprostiti proizvod na levoj strani koristeći razliku kvadrata i poznati trigonometrijski identitet \(\operatorname{ctg}^2 x + 1 = \dfrac{1}{\sin^2 x}\). Zatim ćemo odrediti oblast definisanosti jednačine, pa naći sve vrednosti \(x\) iz dobijene jednačine koje pripadaju intervalu i ispunjavaju uslove domena.

📚 Podsetnik iz teorije
Razlika kvadrata

Za bilo koje izraze \(A\) i \(B\) važi:

\[ (A + B)(A - B) = A^2 - B^2 \]
Pitagorin trigonometrijski identitet za kotangens

Za svako \(x\) gde je \(\sin x \neq 0\) važi:

\[ \operatorname{ctg}^2 x + 1 = \frac{1}{\sin^2 x} \quad \Leftrightarrow \quad \frac{\cos^2 x}{\sin^2 x} + 1 = \frac{1}{\sin^2 x} \]

Ovo direktno sledi iz osnovnog identiteta \(\sin^2 x + \cos^2 x = 1\) deljenjem sa \(\sin^2 x\).

Rešenja jednačine \(\operatorname{ctg}\, x = 1\)

Jednačina \(\operatorname{ctg}\, x = 1\) važi kada je \(\cos x = \sin x\), tj.:

\[ x = \frac{\pi}{4} + n\pi, \quad n \in \mathbb{Z} \]
Oblast definisanosti korena \(\sqrt{x - 2\pi}\)

Izraz \(\sqrt{x - 2\pi}\) je definisan i različit od nule kada je:

\[ x - 2\pi > 0 \quad \Rightarrow \quad x > 2\pi \]
🎨 Rešenja na brojevnoj pravoj

Dijagram prikazuje interval \([\pi/2,\; 8\pi]\), oblast definisanosti \((2\pi,\; 8\pi)\) i svih 6 rešenja jednačine \(\operatorname{ctg}\, x = 1\).

🛠 Rešenje korak po korak
1 Uprošćavamo proizvod na levoj strani

Označimo \(A = 1 + \operatorname{ctg}\, x\) i \(B = \dfrac{1}{\sin x}\). Tada proizvod na levoj strani ima oblik \((A + B)(A - B)\), što je razlika kvadrata:

\[ (A + B)(A - B) = A^2 - B^2 \]

Uvrštavamo:

\[ \left(1 + \operatorname{ctg}\, x\right)^2 - \frac{1}{\sin^2 x} \]
2 Razvijamo kvadrat binoma

Razvijamo \(\left(1 + \operatorname{ctg}\, x\right)^2\):

\[ \left(1 + \operatorname{ctg}\, x\right)^2 = 1 + 2\operatorname{ctg}\, x + \operatorname{ctg}^2 x \]

Pa ceo izraz postaje:

\[ 1 + 2\operatorname{ctg}\, x + \operatorname{ctg}^2 x - \frac{1}{\sin^2 x} \]
3 Primenjujemo trigonometrijski identitet

Koristimo identitet \(\operatorname{ctg}^2 x + 1 = \dfrac{1}{\sin^2 x}\). Zamenjujemo \(\dfrac{1}{\sin^2 x}\) sa \(\operatorname{ctg}^2 x + 1\):

\[ 1 + 2\operatorname{ctg}\, x + \operatorname{ctg}^2 x - \left(\operatorname{ctg}^2 x + 1\right) \]

Otvaramo zagradu i skraćujemo:

\[ 1 + 2\operatorname{ctg}\, x + \operatorname{ctg}^2 x - \operatorname{ctg}^2 x - 1 = 2\operatorname{ctg}\, x \]

Oba \(\operatorname{ctg}^2 x\) se poništavaju, kao i konstante \(+1\) i \(-1\). Ostaje samo \(2\operatorname{ctg}\, x\).

Proveri se: Koliko je \((1 + \operatorname{ctg}\, x)^2 - \csc^2 x\) za \(x = \pi/4\)?

Za \(x = \pi/4\): \(\operatorname{ctg}(\pi/4) = 1\), \(\sin(\pi/4) = \frac{\sqrt{2}}{2}\), pa \(\csc^2(\pi/4) = 2\).

\((1 + 1)^2 - 2 = 4 - 2 = 2\). A \(2\operatorname{ctg}(\pi/4) = 2 \cdot 1 = 2\). Poklapa se!

4 Zapisujemo uprošćenu jednačinu

Posle uprošćavanja, polazna jednačina postaje:

\[ \frac{2\operatorname{ctg}\, x}{\sqrt{x - 2\pi}} = \frac{2}{\sqrt{x - 2\pi}} \]
5 Određujemo oblast definisanosti

Pre nego što skratimo razlomke, moramo utvrditi za koje vrednosti \(x\) je jednačina definisana. Potrebno je:

\[ \begin{align} &1.\quad x - 2\pi > 0 \;\Rightarrow\; x > 2\pi \quad \text{(koren mora biti definisan i nenula)} \\[6pt] &2.\quad \sin x \neq 0 \;\Rightarrow\; x \neq n\pi,\; n \in \mathbb{Z} \quad \text{(ctg i 1/sin moraju postojati)} \\[6pt] &3.\quad x \in \left[\frac{\pi}{2},\; 8\pi\right] \quad \text{(zadati interval)} \end{align} \]

Kombinovanjem ovih uslova, domen je:

\[ x \in (2\pi,\; 8\pi), \quad x \neq 3\pi,\; 4\pi,\; 5\pi,\; 6\pi,\; 7\pi \]

Tačka \(x = 8\pi\) je isključena jer je \(\sin(8\pi) = 0\).

6 Skraćujemo jednačinu

Pošto je u domenu \(\sqrt{x - 2\pi} > 0\), možemo podeliti obe strane sa \(\dfrac{2}{\sqrt{x - 2\pi}}\):

\[ \operatorname{ctg}\, x = 1 \]
Proveri se: Šta znači \(\operatorname{ctg}\, x = 1\)?

To znači \(\dfrac{\cos x}{\sin x} = 1\), odnosno \(\cos x = \sin x\), tj. \(\tan x = 1\). Opšte rešenje je \(x = \dfrac{\pi}{4} + n\pi\), \(n \in \mathbb{Z}\).

7 Nalazimo opšte rešenje

Jednačina \(\operatorname{ctg}\, x = 1\) ima rešenja:

\[ x = \frac{\pi}{4} + n\pi, \quad n \in \mathbb{Z} \]
8 Biramo rešenja iz domena \((2\pi,\; 8\pi)\)

Tražimo celobrojne vrednosti \(n\) takve da važi:

\[ 2\pi < \frac{\pi}{4} + n\pi < 8\pi \]

Iz leve nejednakosti:

\[ 2\pi < \frac{\pi}{4} + n\pi \;\Rightarrow\; n\pi > \frac{7\pi}{4} \;\Rightarrow\; n > \frac{7}{4} = 1{,}75 \;\Rightarrow\; n \geq 2 \]

Iz desne nejednakosti:

\[ \frac{\pi}{4} + n\pi < 8\pi \;\Rightarrow\; n\pi < \frac{31\pi}{4} \;\Rightarrow\; n < \frac{31}{4} = 7{,}75 \;\Rightarrow\; n \leq 7 \]

Dakle, \(n \in \{2,\; 3,\; 4,\; 5,\; 6,\; 7\}\).

9 Eksplicitno ispisujemo sva rešenja

Za svaku vrednost \(n\) dobijamo jedno rešenje:

\[ \begin{align} n = 2: \quad x &= \frac{\pi}{4} + 2\pi = \frac{9\pi}{4} \approx 7{,}07 \\[6pt] n = 3: \quad x &= \frac{\pi}{4} + 3\pi = \frac{13\pi}{4} \approx 10{,}21 \\[6pt] n = 4: \quad x &= \frac{\pi}{4} + 4\pi = \frac{17\pi}{4} \approx 13{,}35 \\[6pt] n = 5: \quad x &= \frac{\pi}{4} + 5\pi = \frac{21\pi}{4} \approx 16{,}49 \\[6pt] n = 6: \quad x &= \frac{\pi}{4} + 6\pi = \frac{25\pi}{4} \approx 19{,}63 \\[6pt] n = 7: \quad x &= \frac{\pi}{4} + 7\pi = \frac{29\pi}{4} \approx 22{,}78 \end{align} \]

Proverimo da nijedno rešenje nije umnožak broja \(\pi\): svako ima oblik \(\frac{(4n+1)\pi}{4}\), pa \(\sin x \neq 0\) za svako od njih.

Proveri se: Da li je \(\frac{29\pi}{4} < 8\pi\)?

\(\frac{29\pi}{4} = 7{,}25\pi < 8\pi\). Da, jeste u intervalu!

Ključni uvid

Proizvod \(\left(1 + \operatorname{ctg}\, x + \dfrac{1}{\sin x}\right)\left(1 + \operatorname{ctg}\, x - \dfrac{1}{\sin x}\right)\) ima oblik razlike kvadrata \(A^2 - B^2\), a zahvaljujući identitetu \(\operatorname{ctg}^2 x + 1 = \csc^2 x\), drastično se uprošćava u \(2\operatorname{ctg}\, x\). Ključna stvar je da domen ograničava interval sa \([\frac{\pi}{2},\; 8\pi]\) na \((2\pi,\; 8\pi)\) zbog uslova \(x > 2\pi\).

\[ (1 + \operatorname{ctg}\, x)^2 - \csc^2 x = 2\operatorname{ctg}\, x \]
Konačni odgovor
Broj rešenja: \(6\)
(A) \(13\) (B) \(14\) (C) \(6\) (D) \(7\) (E) \(12\)

Verifikacija: Rešenja su oblika \(x = \frac{\pi}{4} + n\pi\) za \(n \in \{2, 3, 4, 5, 6, 7\}\), što daje tačno 6 vrednosti. Sve su u intervalu \((2\pi,\; 8\pi)\) i nijedna nije umnožak \(\pi\), pa su svi uslovi domena ispunjeni ✔.

⚠️ Česte greške
Zaboravljanje uslova \(x > 2\pi\)

Ako se ne uzme u obzir da koren \(\sqrt{x - 2\pi}\) zahteva \(x > 2\pi\), rešenja sa \(n = 0\) (\(x = \pi/4\)) i \(n = 1\) (\(x = 5\pi/4\)) bi se pogrešno uključila, dajući više od 6 rešenja.

Uključivanje tačke \(x = 8\pi\)

Mada interval uključuje \(8\pi\), u toj tački je \(\sin(8\pi) = 0\), pa kotangens i \(1/\sin x\) nisu definisani. Srećom, \(8\pi\) ionako nije oblika \(\pi/4 + n\pi\), ali treba biti oprezan sa krajnjim tačkama.
