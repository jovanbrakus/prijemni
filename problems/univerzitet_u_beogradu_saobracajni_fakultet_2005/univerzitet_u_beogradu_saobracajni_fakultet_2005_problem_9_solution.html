Ostatak pri deljenju polinoma

Određivanje nepoznatih koeficijenata polinoma na osnovu poznatog ostatka pri deljenju

Tekst zadatka

Ako je ostatak pri deljenju polinoma \(x^3 + 2x^2 + ax + b\) polinomom \(x^2 - x - 2\) jednak \(7x + 7\), onda je \(a + 2b\) jednako:

(A)
\(4\)
(B)
\(0\)
(C)
\(3\)
(D)
\(2\)
(E)
\(6\)

Plan rešavanja

Primenićemo teorem o deljenju polinoma: polinom trećeg stepena možemo zapisati kao proizvod delioca (stepen 2) i količnika (stepen 1), plus ostatak. Izjednačavanjem koeficijenata odredićemo nepoznate \(a\) i \(b\).

Ključne formule i teoreme
  • Teorem o deljenju polinoma: Za polinome \(P(x)\) i \(D(x)\) (gde je \(\deg D \geq 1\)), postoje jedinstveni polinomi \(Q(x)\) (količnik) i \(R(x)\) (ostatak) takvi da je: \[ P(x) = D(x) \cdot Q(x) + R(x), \quad \deg R < \deg D \]
  • Stepen količnika: Ako je \(\deg P = n\) i \(\deg D = m\), onda je \(\deg Q = n - m\). U našem slučaju: \(\deg P = 3\), \(\deg D = 2\), pa je \(\deg Q = 1\), tj. količnik je oblika \(x + c\).
  • Metod neodređenih koeficijenata: Izjednačavanjem koeficijenata uz iste stepene \(x\) na levoj i desnoj strani jednakosti dobijamo sistem jednačina za nepoznate koeficijente.

Rešenje korak po korak

1

Postavimo jednakost deljenja

Prema teoremu o deljenju polinoma, ako je ostatak \(7x + 7\), možemo zapisati:

\[ x^3 + 2x^2 + ax + b = (x^2 - x - 2) \cdot Q(x) + 7x + 7 \]

Pošto je deljenik stepena 3, a delilac stepena 2, količnik \(Q(x)\) mora biti stepena 1. Zato pišemo \(Q(x) = x + c\), gde je \(c\) nepoznata konstanta.

\[ x^3 + 2x^2 + ax + b = (x^2 - x - 2)(x + c) + 7x + 7 \]
2

Razvijemo proizvod \((x^2 - x - 2)(x + c)\)

Množimo svaki član prvog polinoma sa svakim članom drugog:

\[ (x^2 - x - 2)(x + c) = x^3 + cx^2 - x^2 - cx - 2x - 2c \]

Grupišemo po stepenima \(x\):

\[ = x^3 + (c - 1)x^2 + (-c - 2)x + (-2c) \]
Proveri se: Koliki je koeficijent uz \(x^2\) u razvijenom proizvodu ako je \(c = 3\)?
Odgovor: \(c - 1 = 3 - 1 = 2\). Koeficijent uz \(x^2\) je \(2\).
3

Dodajemo ostatak

Desnu stranu formiramo dodavanjem ostatka \(7x + 7\) na razvijeni proizvod:

\[ x^3 + (c-1)x^2 + (-c-2)x + (-2c) + 7x + 7 \]

Sređujemo:

\[ = x^3 + (c - 1)x^2 + (5 - c)x + (7 - 2c) \]
4

Izjednačimo koeficijente

Leva strana je \(x^3 + 2x^2 + ax + b\), a desna \(x^3 + (c-1)x^2 + (5-c)x + (7-2c)\). Izjednačavamo koeficijente uz iste stepene:

\[ \begin{align} x^2: &\quad c - 1 = 2 \quad \Rightarrow \quad c = 3 \\ x^1: &\quad 5 - c = a \quad \Rightarrow \quad a = 5 - 3 = 2 \\ x^0: &\quad 7 - 2c = b \quad \Rightarrow \quad b = 7 - 6 = 1 \end{align} \]

Koeficijent uz \(x^3\) je \(1 = 1\) na obe strane, što je automatski zadovoljeno.

Proveri se: Koliki je polinom \(x^3 + 2x^2 + ax + b\) za \(a = 2\) i \(b = 1\)?
Odgovor: \(x^3 + 2x^2 + 2x + 1\).
5

Izračunamo \(a + 2b\)

Sada znamo da je \(a = 2\) i \(b = 1\). Tražena vrednost:

\[ a + 2b = 2 + 2 \cdot 1 = 2 + 2 = 4 \]
Proveri se: Potvrdimo deljenje — da li je \((x^2 - x - 2)(x + 3) + 7x + 7 = x^3 + 2x^2 + 2x + 1\)?
Odgovor: \((x^2 - x - 2)(x+3) = x^3 + 3x^2 - x^2 - 3x - 2x - 6 = x^3 + 2x^2 - 5x - 6\). Dodamo \(7x + 7\): \(x^3 + 2x^2 - 5x - 6 + 7x + 7 = x^3 + 2x^2 + 2x + 1\). Tačno!

Ključni uvid

Teorem o deljenju polinoma nam omogućava da zapišemo jednakost i izjednačavanjem koeficijenata odredimo sve nepoznate:

\[ a = 2, \quad b = 1 \quad \Rightarrow \quad a + 2b = 4 \]

Količnik deljenja je \(x + 3\), a ostatak \(7x + 7\) je upravo onaj koji je dat u zadatku.

Konačan odgovor

\[ a + 2b = 4 \]
(A) \(4\) (B) \(0\) (C) \(3\) (D) \(2\) (E) \(6\)

Verifikacija: Za \(a = 2\), \(b = 1\) važi \(x^3 + 2x^2 + 2x + 1 = (x^2 - x - 2)(x + 3) + 7x + 7\). Razvojem proizvoda i sabiranjem sa ostatkom potvrđujemo jednakost. Traženi izraz \(a + 2b = 2 + 2 = 4\) odgovara opciji (A).

Česte greške

  • Pogrešno pretpostavljanje oblika količnika — količnik mora biti stepena \(3 - 2 = 1\), tj. oblika \(x + c\), a ne konstanta ili polinom višeg stepena.
  • Greška pri množenju polinoma, posebno sa negativnim koeficijentima kod \(-x\) i \(-2\). Pažljivo rasporedite svaki član pri razvijanju proizvoda.

Dodatni izazov

Odredite vrednosti \(a\) i \(b\) ako je ostatak pri deljenju polinoma \(x^3 + ax^2 + bx + 6\) polinomom \(x^2 + x - 2\) jednak \(3x + 8\).

Nagovještaj

Primenite isti postupak: zapišite \(x^3 + ax^2 + bx + 6 = (x^2 + x - 2)(x + c) + 3x + 8\), razvijte desnu stranu i izjednačite koeficijente uz svaki stepen od \(x\).
