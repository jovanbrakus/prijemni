Trigonometrijska jednačina — broj rešenja na intervalu

Rešavanje trigonometrijske jednačine faktorizacijom i određivanje broja rešenja na datom intervalu \([-\pi,\,\pi]\)

📝 Tekst zadatka
Broj realnih rešenja jednačine \((\sin x + \cos x)^2 = 4\sin x \cos^2 x + 1\) na intervalu \([-\pi,\,\pi]\) je:

\(5\)

🎯 Plan rešavanja

Razvićemo levu stranu koristeći Pitagorin trigonometrijski identitet \(\sin^2 x + \cos^2 x = 1\) i formulu za dupli ugao \(\sin 2x = 2\sin x \cos x\). Zatim ćemo jednačinu svesti na faktorizovani oblik i odrediti sve tri familije rešenja. Na kraju ćemo prebrojati koliko rešenja upada u interval \([-\pi,\,\pi]\).

📚 Podsetnik iz teorije
Pitagorin trigonometrijski identitet
\[ \sin^2 x + \cos^2 x = 1 \]

Ovo je osnovni identitet koji važi za svako realno \(x\). Koristimo ga da uprostimo zbir \(\sin^2 x + \cos^2 x\) u razvoju kvadrata.

Formula za dupli ugao (sinus)
\[ \sin 2x = 2\sin x \cos x \]

Ova formula nam omogućava da prepoznamo proizvod \(2\sin x \cos x\) kao \(\sin 2x\), ili obrnuto.

Kvadrat zbira
\[ (a + b)^2 = a^2 + 2ab + b^2 \]

Primenjujemo ovu formulu na \((\sin x + \cos x)^2\) da razvijemo levu stranu jednačine.

Rešenja elementarnih trig. jednačina na \([-\pi, \pi]\)

\(\sin x = 0\) ima rešenja \(x \in \{-\pi,\, 0,\, \pi\}\) — tri rešenja.

\(\cos x = 0\) ima rešenja \(x \in \left\{-\dfrac{\pi}{2},\, \dfrac{\pi}{2}\right\}\) — dva rešenja.

\(\cos x = \dfrac{1}{2}\) ima rešenja \(x \in \left\{-\dfrac{\pi}{3},\, \dfrac{\pi}{3}\right\}\) — dva rešenja.

🎨 Vizuelni prikaz: rešenja na trigonometrijskom krugu

Svih 7 rešenja označeno je na trigonometrijskom krugu sa pripadajućim vrednostima

🛠 Rešenje korak po korak
1 Razvijamo levu stranu jednačine

Primenjujemo formulu za kvadrat zbira na \((\sin x + \cos x)^2\):

\[ (\sin x + \cos x)^2 = \sin^2 x + 2\sin x \cos x + \cos^2 x \]

Kako je \(\sin^2 x + \cos^2 x = 1\), dobijamo:

\[ (\sin x + \cos x)^2 = 1 + 2\sin x \cos x = 1 + \sin 2x \]
2 Zamenjujemo u polaznu jednačinu

Polazna jednačina glasi:

\[ (\sin x + \cos x)^2 = 4\sin x \cos^2 x + 1 \]

Uvrštavajući razvijenu levu stranu:

\[ 1 + \sin 2x = 4\sin x \cos^2 x + 1 \]
3 Pojednostavljujemo

Oduzimamo 1 sa obe strane:

\[ \sin 2x = 4\sin x \cos^2 x \]

Sada koristimo \(\sin 2x = 2\sin x \cos x\):

\[ 2\sin x \cos x = 4\sin x \cos^2 x \]
Proveri se: Zašto smemo da koristimo \(\sin 2x = 2\sin x\cos x\)?

To je standardna formula za dupli ugao sinusa, koja važi za svako realno \(x\). Nije potreban nikakav dodatni uslov.

4 Prebacujemo sve na jednu stranu i faktorizujemo

Prebacujemo desnu stranu na levu:

\[ 2\sin x \cos x - 4\sin x \cos^2 x = 0 \]

Izvlačimo zajednički činilac \(2\sin x \cos x\):

\[ 2\sin x \cos x \,(1 - 2\cos x) = 0 \]

Proizvod tri faktora je nula ako i samo ako je barem jedan od njih jednak nuli.

5 Slučaj 1: \(\sin x = 0\)

Jednačina \(\sin x = 0\) ima rešenja oblika \(x = k\pi\), gde je \(k \in \mathbb{Z}\).

Na intervalu \([-\pi,\,\pi]\) to su:

\[ x \in \{-\pi,\; 0,\; \pi\} \]

Iz ovog slučaja imamo 3 rešenja.

6 Slučaj 2: \(\cos x = 0\)

Jednačina \(\cos x = 0\) ima rešenja oblika \(x = \frac{\pi}{2} + k\pi\), gde je \(k \in \mathbb{Z}\).

Na intervalu \([-\pi,\,\pi]\) to su:

\[ x \in \left\{-\frac{\pi}{2},\; \frac{\pi}{2}\right\} \]

Iz ovog slučaja imamo 2 rešenja.

Proveri se: Da li se rešenja iz slučaja 1 i 2 preklapaju?

Ne! U tačkama gde je \(\sin x = 0\) (tj. \(x = -\pi, 0, \pi\)), vrednosti kosinusa su \(\cos(-\pi) = -1\), \(\cos 0 = 1\), \(\cos \pi = -1\) — nijedna nije nula. Dakle, skupovi su disjunktni.

7 Slučaj 3: \(1 - 2\cos x = 0\)

Rešavamo jednačinu po \(\cos x\):

\[ 1 - 2\cos x = 0 \quad \Longrightarrow \quad \cos x = \frac{1}{2} \]

Na intervalu \([-\pi,\,\pi]\) rešenja su:

\[ x \in \left\{-\frac{\pi}{3},\; \frac{\pi}{3}\right\} \]

Iz ovog slučaja imamo 2 rešenja.

8 Proverimo da li su sva rešenja različita

Skupovi rešenja iz tri slučaja su:

  • \(\sin x = 0\): \(\{-\pi,\; 0,\; \pi\}\)
  • \(\cos x = 0\): \(\left\{-\dfrac{\pi}{2},\; \dfrac{\pi}{2}\right\}\)
  • \(\cos x = \dfrac{1}{2}\): \(\left\{-\dfrac{\pi}{3},\; \dfrac{\pi}{3}\right\}\)

Nijedna tačka se ne ponavlja u dva različita skupa (niti \(\sin x\) i \(\cos x\) mogu istovremeno biti nula, niti je \(\cos x = \frac{1}{2}\) moguće kada je \(\sin x = 0\) ili \(\cos x = 0\)).

Ukupan broj rešenja: \(3 + 2 + 2 = 7\).

Proveri se: Verifikacija za \(x = \frac{\pi}{3}\)

Proveravamo u originalnoj jednačini. Za \(x = \frac{\pi}{3}\):

Leva strana: \(\left(\sin\frac{\pi}{3} + \cos\frac{\pi}{3}\right)^2 = \left(\frac{\sqrt{3}}{2} + \frac{1}{2}\right)^2 = \frac{3}{4} + \frac{\sqrt{3}}{2} + \frac{1}{4} = 1 + \frac{\sqrt{3}}{2}\)

Desna strana: \(4 \cdot \frac{\sqrt{3}}{2} \cdot \frac{1}{4} + 1 = \frac{\sqrt{3}}{2} + 1\)

Leva = desna. Potvrđeno!

Ključni uvid

Razvijanjem leve strane i primenom identiteta \(\sin 2x = 2\sin x \cos x\), cela jednačina se svodi na jednostavan faktorizovani oblik:

\[ 2\sin x \cos x\,(1 - 2\cos x) = 0 \]

Svaki faktor daje zasebnu familiju rešenja, a na intervalu \([-\pi,\,\pi]\) ukupno dobijamo \(3 + 2 + 2 = 7\) rešenja.

Konačan odgovor
\(7\) rešenja
(A) \(7\) (B) \(5\) (C) \(6\) (D) \(8\) (E) veći od 8

Verifikacija: Svih 7 rešenja su \(x \in \left\{-\pi,\, -\dfrac{\pi}{2},\, -\dfrac{\pi}{3},\, 0,\, \dfrac{\pi}{3},\, \dfrac{\pi}{2},\, \pi\right\}\). Svako od njih zadovoljava polaznu jednačinu, što je potvrđeno direktnom proverom.

⚠ Česte greške
Deljenje jednačine sa \(\sin x\) ili \(\cos x\)
Ako jednačinu delimo sa \(\sin x\) (ili \(\cos x\)), gubimo rešenja za koja je \(\sin x = 0\) (ili \(\cos x = 0\)). Uvek je bolje prebaciti sve na jednu stranu i faktorizovati.
Zaboravljanje krajnjih tačaka intervala
Na intervalu \([-\pi,\,\pi]\) krajnje tačke \(x = -\pi\) i \(x = \pi\) su uključene. Jednačina \(\sin x = 0\) daje 3 rešenja (ne samo \(x = 0\)), jer je \(\sin(-\pi) = \sin(\pi) = 0\). Propuštanje ovih tačaka vodi na pogrešan odgovor 5.
🌟 Izazov za napredne

Koliko realnih rešenja ima jednačina \((\sin x + \cos x)^2 = 4\sin x \cos^2 x + 1\) na intervalu \([0,\, 4\pi]\)?

Nagoveštaj

Period funkcija \(\sin x\) i \(\cos x\) je \(2\pi\). Interval \([0,\, 4\pi]\) pokriva tačno dva puna perioda. Prebrojte rešenja na \([0,\, 2\pi]\) pazeći na krajnje tačke, pa udvostručite (uz korekciju za zajedničke tačke).
