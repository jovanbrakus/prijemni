🎯 Plan rešavanja

Koristićemo činjenicu da se polinom \(P(x)\) može zapisati kao \(P(x) = D(x) \cdot Q(x) + R(x)\), gde su \(D(x)\) delilac, \(Q(x)\) količnik i \(R(x)\) ostatak. Razvićemo desnu stranu i uporediti koeficijente sa levom stranom da bismo odredili \(a\) i \(b\). Kao alternativu, iskoristićemo nule delioca za brzu proveru.

📚 Podsetnik iz teorije
Deljenje polinoma sa ostatkom

Za svaka dva polinoma \(P(x)\) i \(D(x)\), gde je \(D(x) \neq 0\), postoje jedinstveni polinomi \(Q(x)\) (količnik) i \(R(x)\) (ostatak) takvi da:

\[ P(x) = D(x) \cdot Q(x) + R(x), \quad \deg(R) < \deg(D) \]

Stepen ostatka je uvek strogo manji od stepena delioca.

Poređenje koeficijenata

Ako su dva polinoma identički jednaka (tj. jednaka za svako \(x\)), tada su im svi odgovarajući koeficijenti jednaki:

\[ a_n x^n + \ldots + a_1 x + a_0 = b_n x^n + \ldots + b_1 x + b_0 \implies a_k = b_k \text{ za svako } k \]
Metoda nula delioca (Bezuova teorema)

Ako je \(x_0\) nula delioca \(D(x)\), tj. \(D(x_0) = 0\), tada iz jednakosti \(P(x) = D(x) \cdot Q(x) + R(x)\) sledi:

\[ P(x_0) = R(x_0) \]

Ovo nam daje jednačinu koja povezuje koeficijente polinoma \(P\) sa vrednostima ostatka \(R\) u nulama delioca.

🎨 Vizuelizacija polinoma i ostatka

🛠 Rešenje korak po korak
1 Zapišimo jednakost deljenja polinoma

Prema teoremi o deljenju polinoma, polinom \(P(x) = x^3 + ax^2 + x + b\) se može izraziti kao:

\[ P(x) = D(x) \cdot Q(x) + R(x) \]

gde je \(D(x) = x^2 + x - 2\) delilac, \(Q(x)\) količnik, i \(R(x) = 3x + 2\) ostatak.

2 Odredimo oblik količnika

Pošto je \(\deg(P) = 3\) i \(\deg(D) = 2\), količnik \(Q(x)\) mora biti polinomom stepena \(3 - 2 = 1\), tj. linearna funkcija:

\[ Q(x) = x + c \]

Koeficijent uz \(x\) mora biti 1, jer je vodeći koeficijent polinoma \(P(x)\) jednak 1, a vodeći koeficijent delioca \(D(x)\) takođe 1.

3 Razvijemo desnu stranu

Pomnožimo \(D(x) \cdot Q(x)\):

\[ \begin{align} (x^2 + x - 2)(x + c) &= x^3 + cx^2 + x^2 + cx - 2x - 2c \\ &= x^3 + (c + 1)x^2 + (c - 2)x - 2c \end{align} \]

Sada dodajemo ostatak \(R(x) = 3x + 2\):

\[ x^3 + (c + 1)x^2 + (c - 2 + 3)x + (-2c + 2) \]

Što se pojednostavljuje u:

\[ x^3 + (c + 1)x^2 + (c + 1)x + (2 - 2c) \]
Proveri se: Koliko je \((x^2 + x - 2)(x + 0)\)?

\(x^3 + x^2 - 2x\). Dodamo ostatak \(3x + 2\): dobijamo \(x^3 + x^2 + x + 2\). Koeficijenti: \(a = 1\), \(b = 2\), pa \(a + b = 3\). Brzo potvrđeno!

4 Uporedimo koeficijente

Leva strana je \(x^3 + ax^2 + x + b\), a desna strana je \(x^3 + (c+1)x^2 + (c+1)x + (2-2c)\). Poredimo koeficijente uz iste stepene od \(x\):

\[ \begin{align} x^2: &\quad a = c + 1 \\ x^1: &\quad 1 = c + 1 \\ x^0: &\quad b = 2 - 2c \end{align} \]
5 Rešimo sistem jednačina

Iz jednačine za koeficijent uz \(x^1\):

\[ 1 = c + 1 \implies c = 0 \]

Uvrstimo \(c = 0\) u preostale jednačine:

\[ a = 0 + 1 = 1 \]
\[ b = 2 - 2 \cdot 0 = 2 \]
6 Izračunamo \(a + b\)

Konačno:

\[ a + b = 1 + 2 = 3 \]
Proveri se: Alternativna metoda koristeći nule delioca

Delilac \(x^2 + x - 2 = (x + 2)(x - 1)\) ima nule \(x = 1\) i \(x = -2\).

Za \(x = 1\): \(P(1) = R(1)\), tj. \(1 + a + 1 + b = 3 \cdot 1 + 2\), pa \(2 + a + b = 5\), odakle \(a + b = 3\).

Za \(x = -2\): \(P(-2) = R(-2)\), tj. \(-8 + 4a - 2 + b = 3 \cdot (-2) + 2\), pa \(-10 + 4a + b = -4\), odakle \(4a + b = 6\).

Iz \(a + b = 3\) i \(4a + b = 6\) oduzimanjem: \(3a = 3\), pa \(a = 1\), \(b = 2\). Potvrđeno!

Ključni uvid

Umesto kompletnog poređenja koeficijenata, traženi zbir \(a + b\) se može dobiti u jednom koraku koristeći nulu delioca \(x = 1\). Pošto je \(D(1) = 0\), važi \(P(1) = R(1)\), a to direktno daje \(a + b\) bez potrebe za pojedinačnim određivanjem \(a\) i \(b\).

\[ P(1) = R(1) \implies 2 + a + b = 5 \implies a + b = 3 \]
Konačni odgovor
\(a + b = 3\)
(A) \(4\) (B) \(5\) (C) \(3\) (D) \(2\) (E) \(6\)

Verifikacija: Za \(a = 1\), \(b = 2\) polinom je \(x^3 + x^2 + x + 2\). Deljenje sa \(x^2 + x - 2\) daje količnik \(x\) i ostatak \(3x + 2\). Potvrđeno! ✔

⚠️ Česte greške
Pogrešan oblik količnika

Učenici ponekad pretpostave da je količnik samo konstanta (tj. \(Q(x) = c\)), zaboravljajući da je količnik linearna funkcija oblika \(x + c\). Stepen količnika je uvek \(\deg(P) - \deg(D)\).

Greška pri množenju polinoma

Pri razvijanju proizvoda \((x^2 + x - 2)(x + c)\) lako je zaboraviti negativan predznak ili pogrešno sastaviti koeficijente. Preporučuje se sistematično množenje svakog člana sa svakim i pažljivo skupljanje sličnih sabiraka.

🏆 Dodatni izazov

Nađi vrednosti \(a\) i \(b\) tako da ostatak pri deljenju polinoma \(x^4 + ax^3 + bx + 1\) polinomom \(x^2 - 1\) bude jednak \(2x + 3\).

Nagoveštaj

Nule delioca \(x^2 - 1 = (x-1)(x+1)\) su \(x = 1\) i \(x = -1\). Uvrsti ih u jednakost \(P(x) = R(x)\) da dobiješ sistem od dve jednačine sa dve nepoznate \(a\) i \(b\).
