🎯 Plan rešavanja

Prenosimo \(-1\) na levu stranu, svodimo nejednačinu na oblik jednog razlomka \(\leq 0\), zatim analiziramo znak brojioca i imenioca. Ključ je u tome što brojilac nema realnih nula, pa se problem svodi na analizu znaka imenioca.

📚 Podsetnik iz teorije
Rešavanje racionalnih nejednačina

Racionalna nejednačina oblika \(\frac{P(x)}{Q(x)} \leq 0\) rešava se analizom znaka brojioca \(P(x)\) i imenioca \(Q(x)\). Nikada ne množimo obe strane imeniocem jer ne znamo njegov znak!

\[ \frac{P(x)}{Q(x)} \leq 0 \quad \Leftrightarrow \quad P(x) \cdot Q(x) \leq 0, \; Q(x) \neq 0 \]
Diskriminanta kvadratnog trinoma

Za kvadratni trinom \(ax^2 + bx + c\) sa \(a > 0\):

\[ D = b^2 - 4ac \]

Ako je \(D < 0\), trinom nema realnih nula i važi \(ax^2 + bx + c > 0\) za svako \(x \in \mathbb{R}\) (kada je \(a > 0\)).

Znak kvadratnog trinoma sa poznatim korenima

Ako je \(ax^2 + bx + c = a(x - x_1)(x - x_2)\) sa \(x_1 < x_2\) i \(a > 0\), tada:

\[ a(x - x_1)(x - x_2) < 0 \quad \Leftrightarrow \quad x_1 < x < x_2 \]

Trinom je negativan između korena, a pozitivan izvan korena.

🎨 Grafik funkcije i oblast rešenja

Grafik prikazuje funkciju \(f(x) = \frac{x^2 + 8x - 6}{x^2 - 6x + 8}\) i pravu \(y = -1\). Oblast gde je \(f(x) \leq -1\) je osenčena zelenom bojom.

🛠 Rešenje korak po korak
1 Prenosimo \(-1\) na levu stranu

Polazna nejednačina je:

\[ \frac{x^2 + 8x - 6}{x^2 - 6x + 8} \leq -1 \]

Dodajemo \(1\) na obe strane (odnosno, prenosimo \(-1\) ulevo):

\[ \frac{x^2 + 8x - 6}{x^2 - 6x + 8} + 1 \leq 0 \]

Nikada ne množimo obe strane imeniocem jer imenilac može biti pozitivan ili negativan — to bi promenilo smer nejednakosti.

2 Svodimo na jedan razlomak

Zapisujemo \(1\) kao razlomak sa istim imeniocem:

\[ \frac{x^2 + 8x - 6}{x^2 - 6x + 8} + \frac{x^2 - 6x + 8}{x^2 - 6x + 8} \leq 0 \]

Sabiramo brojioce:

\[ \frac{(x^2 + 8x - 6) + (x^2 - 6x + 8)}{x^2 - 6x + 8} \leq 0 \]
3 Sređujemo brojilac

Sabiramo slične članove u brojiocu:

\[ x^2 + 8x - 6 + x^2 - 6x + 8 = 2x^2 + 2x + 2 \]

Izdvajamo faktor \(2\):

\[ 2x^2 + 2x + 2 = 2(x^2 + x + 1) \]

Nejednačina sada glasi:

\[ \frac{2(x^2 + x + 1)}{x^2 - 6x + 8} \leq 0 \]
Proveri se: Da li je \(x^2 + 8x - 6 + x^2 - 6x + 8 = 2x^2 + 2x + 2\)?

\(x^2 + x^2 = 2x^2\), zatim \(8x - 6x = 2x\), i na kraju \(-6 + 8 = 2\). Dakle: \(2x^2 + 2x + 2\). Tačno!

4 Ispitujemo znak brojioca

Brojilac je \(2(x^2 + x + 1)\). Izračunavamo diskriminantu trinoma \(x^2 + x + 1\):

\[ D = b^2 - 4ac = 1^2 - 4 \cdot 1 \cdot 1 = 1 - 4 = -3 \]

Pošto je \(D = -3 < 0\) i vodeći koeficijent je pozitivan (\(a = 1 > 0\)), trinom \(x^2 + x + 1\) nema realnih nula i važi:

\[ x^2 + x + 1 > 0 \quad \text{za svako } x \in \mathbb{R} \]

Samim tim:

\[ 2(x^2 + x + 1) > 0 \quad \text{za svako } x \in \mathbb{R} \]

Ovo je ključni zaključak: brojilac je uvek strogo pozitivan, pa nikada ne može biti nula. To znači da jednakost u \(\leq\) nije moguća.

5 Faktorizujemo imenilac

Imenilac je \(x^2 - 6x + 8\). Tražimo dva broja čiji je zbir \(-6\), a proizvod \(8\). To su \(-2\) i \(-4\):

\[ x^2 - 6x + 8 = (x - 2)(x - 4) \]

Imenilac je jednak nuli za \(x = 2\) i \(x = 4\). Te vrednosti nisu u domenu!

Proveri se: Da li je \((x-2)(x-4) = x^2 - 6x + 8\)?

\((x-2)(x-4) = x^2 - 4x - 2x + 8 = x^2 - 6x + 8\). Tačno!

6 Određujemo znak razlomka

Nejednačina je:

\[ \frac{2(x^2 + x + 1)}{(x - 2)(x - 4)} \leq 0 \]

Pošto je brojilac \(2(x^2 + x + 1)\) uvek strogo pozitivan, znak razlomka zavisi samo od znaka imenioca.

Da bi razlomak bio \(\leq 0\), potrebno je da imenilac bude strogo negativan (ne može biti nula jer bi to značilo deljenje nulom):

\[ (x - 2)(x - 4) < 0 \]
7 Rešavamo \((x - 2)(x - 4) < 0\)

Proizvod \((x - 2)(x - 4)\) je kvadratna funkcija sa korenima \(x_1 = 2\) i \(x_2 = 4\) i pozitivnim vodećim koeficijentom.

Takva parabola je ispod \(x\)-ose (negativna) između korena:

\[ (x - 2)(x - 4) < 0 \quad \Leftrightarrow \quad 2 < x < 4 \]

Krajnje tačke \(x = 2\) i \(x = 4\) ne pripadaju rešenju jer u njima imenilac iznosi nula.

Otvoreni interval \((2, 4)\) znači da su krajnje tačke isključene — upravo zato što u njima polazni razlomak nije definisan.

Proveri se: Koliko je \(f(3)\)?

\(f(3) = \frac{9 + 24 - 6}{9 - 18 + 8} = \frac{27}{-1} = -27\). Važi \(-27 \leq -1\). Tačno, \(x = 3\) je rešenje!

Ključni uvid

Cela nejednačina se svodi na pitanje znaka imenioca, jer se brojilac uvek ponaša isto (strogo pozitivan). Kada trinom u brojiocu nema realnih korena, problem se drastično pojednostavljuje. Ovo je čest obrazac kod racionalnih nejednačina na prijemnim ispitima.

\[ \underbrace{2(x^2 + x + 1)}_{\text{uvek } > 0} \Big/ \underbrace{(x-2)(x-4)}_{\text{< 0 za } x \in (2,4)} \leq 0 \quad \Rightarrow \quad x \in (2, 4) \]
Konačni odgovor
\(x \in (2, 4)\)
(A) \((-\infty, 2] \cup (4, +\infty)\) (B) \((-\infty, 2) \cup (4, +\infty)\) (C) \((2, 4)\) (D) \((-\infty, 2] \cup (2, 4) \cup [6, +\infty)\) (E) \([2, 4]\)

Verifikacija: za \(x = 3\) dobijamo \(\frac{27}{-1} = -27 \leq -1\) ✔. Za \(x = 0\) dobijamo \(\frac{-6}{8} = -0{,}75 \not\leq -1\) ✔. Za \(x = 5\) dobijamo \(\frac{59}{3} \approx 19{,}67 \not\leq -1\) ✔.

⚠️ Česte greške
Množenje obe strane imeniocem

Ako pomnožimo obe strane sa \(x^2 - 6x + 8\), moramo razlikovati slučaj kada je ovaj izraz pozitivan i kada je negativan — jer množenje negativnim brojem okreće smer nejednakosti. Sigurniji pristup je prebacivanje svega na jednu stranu.

Uključivanje tačaka \(x = 2\) i \(x = 4\)

U tačkama \(x = 2\) i \(x = 4\) imenilac je nula, pa polazni razlomak nije definisan. Odgovor mora biti otvoreni interval \((2, 4)\), a ne zatvoreni \([2, 4]\). Ovo razlikuje odgovor (C) od odgovora (E).

🏆 Dodatni izazov

Rešiti nejednačinu \(\displaystyle\frac{x^2 + 8x - 6}{x^2 - 6x + 8} \geq -1\) (sa znakom \(\geq\) umesto \(\leq\)).

Nagoveštaj

Sada tražimo \(\frac{2(x^2 + x + 1)}{(x-2)(x-4)} \geq 0\). Pošto je brojilac uvek pozitivan, treba nam imenilac \(> 0\), tj. \((x-2)(x-4) > 0\). To važi za \(x \in (-\infty, 2) \cup (4, +\infty)\).
