Kvadrat zbira trigonometrijskih funkcija

Trigonometrija -- trigonometrijski identiteti, formula dvostrukog ugla

📋 Zadatak

Izraz \((\cos x + \sin x)^2\) identički je jednak izrazu:

(A)
\(1\)
(B)
\(\sin 2x - 1\)
(C)
\(1 + \sin 2x\)
(D)
\(\cos 2x - 1\)
(E)
\(1 + \cos 2x\)

🗺️ Plan rešavanja

  1. Razviti kvadrat binoma \((\cos x + \sin x)^2\) koristeći formulu \((a+b)^2 = a^2 + 2ab + b^2\).
  2. Primeniti osnovni trigonometrijski identitet i formulu dvostrukog ugla za sinusnu funkciju da bismo pojednostavili izraz.

📖 Teorija

📐 Ključne formule i teoreme
  • Kvadrat binoma: \((a + b)^2 = a^2 + 2ab + b^2\)
  • Osnovni trigonometrijski identitet (Pitagorin): \(\cos^2 x + \sin^2 x = 1\)
  • Formula dvostrukog ugla za sinus: \(\sin 2x = 2 \sin x \cos x\)
  • Formula dvostrukog ugla za kosinus: \(\cos 2x = \cos^2 x - \sin^2 x\) (ova formula nije potrebna ovde, ali služi za eliminaciju pogrešnih odgovora)

🎨 Vizuelni prikaz -- poređenje funkcija

\(f(x) = (\cos x + \sin x)^2\) \(g(x) = 1 + \sin 2x\)

Prevlačite klizač da vidite vrednosti funkcija u datoj tački. Grafici se savršeno poklapaju!

🔢 Rešenje korak po korak

Korak 1

Zapišimo polazni izraz

Potrebno je da pojednostavimo izraz:

\[ (\cos x + \sin x)^2 \]

Ovo je kvadrat zbira dve trigonometrijske funkcije. Koristićemo algebarsku formulu za kvadrat binoma.

Korak 2

Razvijemo kvadrat binoma

Primenjujemo formulu \((a+b)^2 = a^2 + 2ab + b^2\), gde je \(a = \cos x\) i \(b = \sin x\):

\[ (\cos x + \sin x)^2 = \cos^2 x + 2 \cos x \sin x + \sin^2 x \]
✓ Proveri sebe: Koliko članova dobijamo razvijanjem kvadrata binoma?
Tri člana: \(a^2\), \(2ab\) i \(b^2\). U našem slučaju: \(\cos^2 x\), \(2\cos x \sin x\) i \(\sin^2 x\).
Korak 3

Pregrupisumo članove

Primetimo da se u razvijenom izrazu pojavljuju \(\cos^2 x\) i \(\sin^2 x\). Pregrupisumo:

\[ \cos^2 x + \sin^2 x + 2 \cos x \sin x \]

Sada možemo primeniti poznati trigonometrijski identitet na prva dva člana.

Korak 4

Primenimo Pitagorin identitet

Na osnovu osnovnog trigonometrijskog identiteta važi:

\[ \cos^2 x + \sin^2 x = 1 \]

Zamenjujemo u naš izraz:

\[ 1 + 2 \cos x \sin x \]
✓ Proveri sebe: Šta je vrednost izraza \(2 \sin x \cos x\) prema formuli dvostrukog ugla?
Prema formuli dvostrukog ugla za sinus: \(2 \sin x \cos x = \sin 2x\). Ovo je jedna od najvažnijih trigonometrijskih formula!
Korak 5

Primenimo formulu dvostrukog ugla

Preostali član \(2 \cos x \sin x\) prepoznajemo kao formulu dvostrukog ugla za sinus:

\[ 2 \sin x \cos x = \sin 2x \]

Dakle, konačno dobijamo:

\[ (\cos x + \sin x)^2 = 1 + \sin 2x \]
✓ Proveri sebe: Proverite za \(x = \frac{\pi}{4}\). Da li se obe strane poklapaju?
Leva strana: \(\left(\cos\frac{\pi}{4} + \sin\frac{\pi}{4}\right)^2 = \left(\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}\right)^2 = (\sqrt{2})^2 = 2\).
Desna strana: \(1 + \sin\frac{\pi}{2} = 1 + 1 = 2\). Poklapaju se!

💡 Ključni uvid

Ceo postupak se svodi na dve ključne formule: Pitagorin identitet \(\cos^2 x + \sin^2 x = 1\) i formula dvostrukog ugla \(2\sin x \cos x = \sin 2x\). Kad god vidite kvadrat zbira ili razlike trigonometrijskih funkcija, uvek prvo razvijte pomoću algebarskih formula, a zatim primenite trigonometrijske identitete.

Ovaj tip zadatka proverava da li znate da povežete algebarske i trigonometrijske transformacije u jedinstven postupak.

Izraz \((\cos x + \sin x)^2\) identički je jednak:

\(1 + \sin 2x\)
(A) \(1\) (B) \(\sin 2x - 1\) (C) \(1 + \sin 2x\) (D) \(\cos 2x - 1\) (E) \(1 + \cos 2x\)

Verifikacija: Za \(x = 0\): leva strana \((\cos 0 + \sin 0)^2 = 1^2 = 1\), desna strana \(1 + \sin 0 = 1\). Za \(x = \frac{\pi}{4}\): leva strana \((\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2})^2 = 2\), desna strana \(1 + \sin\frac{\pi}{2} = 2\). Identitet važi za sve vrednosti \(x\).

⚠️ Česte greške

  • Zaboraviti dupli proizvod \(2\cos x \sin x\) pri razvijanju kvadrata binoma i pogrešno dobiti samo \(\cos^2 x + \sin^2 x = 1\) (odgovor A). Ovo je najčešća greška!
  • Zameniti formulu dvostrukog ugla za sinus (\(\sin 2x = 2\sin x\cos x\)) sa formulom za kosinus (\(\cos 2x = \cos^2 x - \sin^2 x\)), što vodi ka pogrešnom odgovoru (E).

🚀 Izazov

Dopunski zadatak: Pojednostavite izraz \((\cos x - \sin x)^2\). Koji od sledećih izraza dobijate?

(A) \(1\)   (B) \(1 - \sin 2x\)   (C) \(1 + \sin 2x\)   (D) \(\cos 2x - 1\)

Savet

Postupak je identičan, samo pazite na znak ispred dvostrukog proizvoda. Kod \((a - b)^2 = a^2 - 2ab + b^2\), dupli proizvod ima negativan znak, pa ćete dobiti \(1 - 2\sin x\cos x = 1 - \sin 2x\). Tačan odgovor je (B).
