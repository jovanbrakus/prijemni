Trigonometrijski identitet sa četvrtim stepenima

Trigonometrija — transformacija izraza pomoću osnovnih identiteta

📝 Tekst zadatka
Izraz \(\cos^4 x + \sin^4 x\) identički je jednak izrazu:
(A)
\(1 - \dfrac{\sin^2 2x}{2}\)
(B)
\(\sin^2 2x + 1\)
(C)
\(\sin^2 2x\)
(D)
\(\cos^2 2x\)
(E)

🎯 Plan rešavanja

Izraz \(\cos^4 x + \sin^4 x\) ćemo transformisati koristeći algebarski identitet za zbir kvadrata: \(a^2 + b^2 = (a+b)^2 - 2ab\), gde su \(a = \cos^2 x\) i \(b = \sin^2 x\). Zatim ćemo primeniti osnovni Pitagorin identitet i formulu za dupli ugao.

📚 Podsetnik iz teorije
Pitagorin trigonometrijski identitet

Za svako \(x\) važi:

\[ \sin^2 x + \cos^2 x = 1 \]

Ovo je najvažniji trigonometrijski identitet i koristimo ga kada želimo da pojednostavimo izraze koji sadrže zbir kvadrata sinusa i kosinusa.

Formula za dupli ugao sinusa

Sinus dvostrukog ugla:

\[ \sin 2x = 2 \sin x \cos x \]

Odavde sledi da je \(\sin x \cos x = \dfrac{\sin 2x}{2}\).

Algebarski identitet za zbir kvadrata

Za bilo koje \(a\) i \(b\):

\[ a^2 + b^2 = (a + b)^2 - 2ab \]

Ovo je posledica razvijanja \((a+b)^2 = a^2 + 2ab + b^2\), odakle se premešta \(2ab\) na levu stranu.

🎨 Grafički prikaz identiteta

Pomeri klizač da vidiš kako se vrednosti menjaju za različite uglove x

🛠 Rešenje korak po korak
1 Zapišimo izraz pomoću smene

Neka je \(a = \cos^2 x\) i \(b = \sin^2 x\). Tada traženi izraz postaje:

\[ \cos^4 x + \sin^4 x = a^2 + b^2 \]

Četvrti stepen je kvadrat kvadrata, pa je \(\cos^4 x = (\cos^2 x)^2 = a^2\).

2 Primenimo identitet za zbir kvadrata

Koristimo algebarski identitet \(a^2 + b^2 = (a + b)^2 - 2ab\):

\[ a^2 + b^2 = (a + b)^2 - 2ab \]

Zamenom \(a = \cos^2 x\) i \(b = \sin^2 x\) dobijamo:

\[ \cos^4 x + \sin^4 x = (\cos^2 x + \sin^2 x)^2 - 2\cos^2 x \sin^2 x \]
3 Uprostimo zbir \(\cos^2 x + \sin^2 x\)

Prema Pitagorinom identitetu, \(\cos^2 x + \sin^2 x = 1\). Zato:

\[ (\cos^2 x + \sin^2 x)^2 = 1^2 = 1 \]

Dosadašnji rezultat:

\[ \cos^4 x + \sin^4 x = 1 - 2\cos^2 x \sin^2 x \]
Proveri se: Koliko je \((\cos^2 x + \sin^2 x)^2\) za \(x = 30°\)?

\(\cos^2 30° + \sin^2 30° = \frac{3}{4} + \frac{1}{4} = 1\), pa je \(1^2 = 1\). Pitagorin identitet važi za svaki ugao!

4 Transformišemo proizvod \(\cos^2 x \sin^2 x\)

Iz formule za dupli ugao \(\sin 2x = 2\sin x \cos x\) sledi:

\[ \sin x \cos x = \frac{\sin 2x}{2} \]

Kvadriranjem obe strane:

\[ \sin^2 x \cos^2 x = \frac{\sin^2 2x}{4} \]
5 Uvrstimo nazad u izraz

Zamenom \(\cos^2 x \sin^2 x = \dfrac{\sin^2 2x}{4}\) u naš izraz:

\[ \cos^4 x + \sin^4 x = 1 - 2 \cdot \frac{\sin^2 2x}{4} \]

Skratimo dvojku:

\[ \cos^4 x + \sin^4 x = 1 - \frac{\sin^2 2x}{2} \]
Proveri se: Koliko je \(\cos^4 45° + \sin^4 45°\)?

\(\cos 45° = \sin 45° = \frac{\sqrt{2}}{2}\), pa je \(\cos^4 45° + \sin^4 45° = \frac{1}{4} + \frac{1}{4} = \frac{1}{2}\).

Proverom: \(1 - \frac{\sin^2 90°}{2} = 1 - \frac{1}{2} = \frac{1}{2}\). Poklapa se!

6 Numerička verifikacija

Proverimo za \(x = 0\):

\[ \cos^4 0 + \sin^4 0 = 1^4 + 0^4 = 1 \]

Naš rezultat za \(x = 0\):

\[ 1 - \frac{\sin^2 0}{2} = 1 - 0 = 1 \; \checkmark \]

Proverimo za \(x = \frac{\pi}{6}\) (tj. 30°):

\[ \cos^4 30° + \sin^4 30° = \left(\frac{\sqrt{3}}{2}\right)^4 + \left(\frac{1}{2}\right)^4 = \frac{9}{16} + \frac{1}{16} = \frac{10}{16} = \frac{5}{8} \]
\[ 1 - \frac{\sin^2 60°}{2} = 1 - \frac{(\frac{\sqrt{3}}{2})^2}{2} = 1 - \frac{3/4}{2} = 1 - \frac{3}{8} = \frac{5}{8} \; \checkmark \]
Proveri se: Eliminišimo ostale opcije za \(x = 45°\)

Za \(x = 45°\) traženi izraz ima vrednost \(\frac{1}{2}\). Proverimo ostale opcije:

(B) \(\sin^2 90° + 1 = 1 + 1 = 2 \neq \frac{1}{2}\)

(C) \(\sin^2 90° = 1 \neq \frac{1}{2}\)

(D) \(\cos^2 90° = 0 \neq \frac{1}{2}\)

(E) \(1 \neq \frac{1}{2}\)

Sve se eliminišu. Samo opcija (A) daje tačan rezultat.

Ključni uvid

Suština rešavanja leži u prepoznavanju algebarskog obrasca: zbir četvrtih stepeni je zapravo zbir kvadrata, koji se može izraziti pomoću kvadrata zbira. Kombinacijom identiteta \(a^2 + b^2 = (a+b)^2 - 2ab\), Pitagorinog identiteta i formule za dupli ugao, složen trigonometrijski izraz se svodi na elegantnu formulu:

\[ \cos^4 x + \sin^4 x = 1 - \frac{\sin^2 2x}{2} \]
Konačni odgovor
\(\cos^4 x + \sin^4 x = 1 - \dfrac{\sin^2 2x}{2}\)
(A) \(1 - \dfrac{\sin^2 2x}{2}\) (B) \(\sin^2 2x + 1\) (C) \(\sin^2 2x\) (D) \(\cos^2 2x\) (E) \(1\)

Verifikacija: Za \(x = 0\) dobijamo \(1 = 1\), za \(x = 45°\) dobijamo \(\frac{1}{2} = \frac{1}{2}\), za \(x = 30°\) dobijamo \(\frac{5}{8} = \frac{5}{8}\). Identitet važi za sve testirane vrednosti. ✔

⚠️ Česte greške
Pretpostavka da je \(\cos^4 x + \sin^4 x = (\cos^2 x + \sin^2 x)^2 = 1\)

Ovo je najčešća greška! Kvadrat zbira nije zbir kvadrata: \((a+b)^2 = a^2 + 2ab + b^2 \neq a^2 + b^2\). Mešani član \(2ab = 2\cos^2 x \sin^2 x\) nije nula (osim u specijalnim slučajevima).

Pogrešno računanje sa duplim uglom

Česta greška je pisati \(\sin^2 x \cos^2 x = \frac{\sin^2 2x}{2}\) umesto \(\frac{\sin^2 2x}{4}\). Pažljivo: \(\sin 2x = 2\sin x \cos x\), pa je \(\sin x \cos x = \frac{\sin 2x}{2}\), i tek nakon kvadriranja dobijamo \(\frac{\sin^2 2x}{4}\).

🏆 Dodatni izazov

Pojednostavi izraz \(\cos^6 x + \sin^6 x\) koristeći sličan pristup.

Nagoveštaj

Koristi identitet za zbir kubova: \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\), gde su \(a = \cos^2 x\) i \(b = \sin^2 x\). Pošto je \(a + b = 1\), izraz se svodi na \(a^2 - ab + b^2 = (a^2 + b^2) - ab\). Već znamo da je \(a^2 + b^2 = 1 - \frac{\sin^2 2x}{2}\), a \(ab = \frac{\sin^2 2x}{4}\). Konačni rezultat: \(1 - \frac{3\sin^2 2x}{4}\).
