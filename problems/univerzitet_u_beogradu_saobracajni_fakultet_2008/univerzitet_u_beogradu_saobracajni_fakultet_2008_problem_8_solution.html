Ostatak pri deljenju polinoma

Algebra -- deljenje polinoma, sistem linearnih jednačina, teorema o ostatku

📋 Zadatak

Ako je ostatak pri deljenju polinoma \(x^4 + 2x^3 + ax + b\) polinomom \(x^2 + x - 2\) jednak \(x + 1\), onda je \(a + 2b\) jednako:

(A)
\(-1\)
(B)
\(-2\)
(C)
\(2\)
(D)
\(1\)
(E)
\(0\)

🗺️ Plan rešavanja

  1. Faktorizovati delilac \(x^2 + x - 2\) i iskoristiti teoremu o ostatku: za svaki koren delioca, vrednost polinoma \(P(x)\) mora biti jednaka vrednosti ostatka \(R(x)\) u toj tački.
  2. Postaviti sistem od dve jednačine sa dve nepoznate (\(a\) i \(b\)), rešiti ga i izračunati \(a + 2b\).

📖 Teorija

📐 Ključne formule i teoreme
  • Algoritam deljenja polinoma: Za polinome \(P(x)\) i \(D(x)\), postoje jedinstveni \(Q(x)\) i \(R(x)\) takvi da: \(P(x) = D(x) \cdot Q(x) + R(x)\), gde je \(\deg(R) < \deg(D)\).
  • Teorema o ostatku (generalizovana): Ako je \(x = c\) koren delioca \(D(x)\), tada važi \(P(c) = R(c)\), jer je \(D(c) = 0\).
  • Faktorizacija kvadratnog trinoma: \(x^2 + x - 2 = (x + 2)(x - 1)\), pa su koreni \(x = 1\) i \(x = -2\).
  • Rešavanje sistema linearnih jednačina: Metoda supstitucije ili eliminacije za nalaženje vrednosti \(a\) i \(b\).

🎨 Vizuelni prikaz -- deljenje polinoma

\(P(x) = x^4 + 2x^3 - 1\) \(D(x) \cdot Q(x) = (x^2+x-2)(x^2+x+1)\) Ostatak \(R(x) = x + 1\) Koreni delioca

Grafici polinoma P(x), proizvoda D(x)·Q(x) i ostatka R(x). Uočite da se P(x) i D(x)·Q(x) razlikuju tačno za R(x).

🔢 Rešenje korak po korak

Korak 1

Zapišimo uslov deljenja

Prema algoritmu deljenja polinoma, važi:

\[ P(x) = D(x) \cdot Q(x) + R(x) \]

gde je \(P(x) = x^4 + 2x^3 + ax + b\), \(D(x) = x^2 + x - 2\) i \(R(x) = x + 1\).

Dakle:

\[ x^4 + 2x^3 + ax + b = (x^2 + x - 2) \cdot Q(x) + (x + 1) \]
Korak 2

Faktorizujmo delilac

Faktorizujemo kvadratni trinom \(x^2 + x - 2\):

\[ x^2 + x - 2 = (x + 2)(x - 1) \]

Koreni delioca su \(x_1 = 1\) i \(x_2 = -2\).

Korak 3

Primenimo generalizovanu teoremu o ostatku

Za svaki koren delioca \(D(x)\) važi \(D(x_i) = 0\), pa iz jednakosti \(P(x_i) = D(x_i) \cdot Q(x_i) + R(x_i)\) dobijamo:

\[ P(x_i) = R(x_i) \]

Ovo nam daje dva uslova -- po jedan za svaki koren delioca.

✓ Proveri sebe: Zašto je \(P(x_i) = R(x_i)\) za korene delioca?
Zato što je \(D(x_i) = 0\), pa član \(D(x_i) \cdot Q(x_i)\) nestaje. Ostaje samo \(P(x_i) = 0 \cdot Q(x_i) + R(x_i) = R(x_i)\).
Korak 4

Uslov za \(x = 1\)

Uvrštavamo \(x = 1\) u \(P(1) = R(1)\):

\[ P(1) = 1^4 + 2 \cdot 1^3 + a \cdot 1 + b = 1 + 2 + a + b = 3 + a + b \] \[ R(1) = 1 + 1 = 2 \]

Izjednačavamo:

\[ 3 + a + b = 2 \] \[ a + b = -1 \quad \ldots \text{(jednačina 1)} \]
Korak 5

Uslov za \(x = -2\)

Uvrštavamo \(x = -2\) u \(P(-2) = R(-2)\):

\[ P(-2) = (-2)^4 + 2(-2)^3 + a(-2) + b = 16 - 16 - 2a + b = -2a + b \] \[ R(-2) = -2 + 1 = -1 \]

Izjednačavamo:

\[ -2a + b = -1 \quad \ldots \text{(jednačina 2)} \]
✓ Proveri sebe: Koliko je \((-2)^4 + 2 \cdot (-2)^3\)?
\((-2)^4 = 16\) i \(2 \cdot (-2)^3 = 2 \cdot (-8) = -16\), pa je zbir \(16 - 16 = 0\). Zato ostaje samo \(-2a + b\).
Korak 6

Rešimo sistem jednačina

Imamo sistem:

\[ \begin{cases} a + b = -1 \\ -2a + b = -1 \end{cases} \]

Oduzmimo drugu jednačinu od prve:

\[ (a + b) - (-2a + b) = -1 - (-1) \] \[ a + b + 2a - b = 0 \] \[ 3a = 0 \implies a = 0 \]

Iz jednačine 1:

\[ 0 + b = -1 \implies b = -1 \]
Korak 7

Izračunajmo \(a + 2b\)

Sada kada znamo da je \(a = 0\) i \(b = -1\), izračunavamo traženu vrednost:

\[ a + 2b = 0 + 2 \cdot (-1) = -2 \]
✓ Proveri sebe: Kako da verifikujemo rezultat?
Podelimo \(x^4 + 2x^3 + 0 \cdot x + (-1) = x^4 + 2x^3 - 1\) sa \(x^2 + x - 2\) dugim deljenjem. Količnik je \(Q(x) = x^2 + x + 1\) i ostatak je \(x + 1\), što se poklapa sa zadatim uslovom!

💡 Ključni uvid

Umesto da izvršimo kompletno dugo deljenje polinoma (što bi bilo mnogo složenije), iskoristili smo činjenicu da koreni delioca poništavaju delilac. To nam omogućava da postavimo sistem jednačina direktno iz uslova \(P(x_i) = R(x_i)\) za svaki koren \(x_i\) delioca.

Ova tehnika je izuzetno korisna kad god delilac ima poznate korene -- za linearne delioce dobijamo jednu jednačinu (klasična teorema o ostatku), a za kvadratne dobijamo sistem od dve jednačine.

Vrednost izraza \(a + 2b\) je:

\(a + 2b = -2\)
(A) \(-1\) (B) \(-2\) (C) \(2\) (D) \(1\) (E) \(0\)

Verifikacija: Za \(a = 0\) i \(b = -1\), polinom \(P(x) = x^4 + 2x^3 - 1\) podeljen sa \(x^2 + x - 2\) daje količnik \(x^2 + x + 1\) i ostatak \(x + 1\). Zaista: \((x^2+x-2)(x^2+x+1) = x^4+2x^3-x-2\), pa \(P(x) - (x^4+2x^3-x-2) = -1-(-x-2) = x+1\). ✓

⚠️ Česte greške

  • Pokušati dugo deljenje polinoma sa nepoznatim koeficijentima \(a\) i \(b\) -- to je znatno komplikovanije. Bolje je iskoristiti korene delioca i teoremu o ostatku.
  • Pogrešno izračunati \((-2)^4\) ili \(2 \cdot (-2)^3\): čest propust je napisati \((-2)^4 = -16\) umesto \(16\), ili zaboraviti da je \((-2)^3 = -8\), pa pogrešno dobiti \(2 \cdot (-2)^3 = -4\) umesto \(-16\).

🚀 Izazov

Dopunski zadatak: Neka je ostatak pri deljenju polinoma \(x^5 + 3x^4 + cx^2 + dx + e\) polinomom \(x^3 - x\) jednak \(2x^2 - x + 3\). Odredite \(c + d + e\).

Savet

Delilac \(x^3 - x = x(x-1)(x+1)\) ima tri korena: \(0\), \(1\) i \(-1\). Upotrebite isti pristup -- za svaki koren izjednačite \(P(x_i) = R(x_i)\) i dobijte sistem od tri jednačine sa tri nepoznate.
