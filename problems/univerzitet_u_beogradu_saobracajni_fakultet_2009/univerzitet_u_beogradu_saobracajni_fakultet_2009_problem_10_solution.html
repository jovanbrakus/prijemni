Transformacija trigonometrijskog izraza

Trigonometrijski identiteti -- pojednostavljivanje zbira četvrtih stepena sinusa i kosinusa

Postavka zadatka

Izraz \(\cos^4 x + \sin^4 x\) identički je jednak izrazu:

(A)
\(1\)
(B)
\(1 + \dfrac{1}{2}\sin^2 2x\)
(C)
\(1 - \dfrac{1}{2}\sin^2 2x\)
(D)
\(1 - \dfrac{1}{2}\cos^2 2x\)
(E)
\(1 + \dfrac{1}{2}\cos^2 2x\)

Plan rešavanja

Iskoristićemo algebarski identitet \(a^2 + b^2 = (a+b)^2 - 2ab\) za \(a = \cos^2 x\) i \(b = \sin^2 x\), a zatim primeniti Pitagorin identitet i formulu dvostrukog ugla da bismo dobili konačan oblik.

Ključne formule i teoreme
  • Osnovni Pitagorin identitet: \(\sin^2 x + \cos^2 x = 1\)
  • Algebarski identitet: \(a^2 + b^2 = (a+b)^2 - 2ab\) — omogućava nam da izrazimo zbir kvadrata preko zbira i proizvoda.
  • Formula dvostrukog ugla za sinus: \(\sin 2x = 2\sin x \cos x\), odakle sledi \(\sin^2 2x = 4\sin^2 x \cos^2 x\).
  • Ekvivalentni oblik: \(\sin^2 x \cos^2 x = \dfrac{\sin^2 2x}{4}\)

Vizuelni prikaz

Grafik prikazuje funkciju \(\cos^4 x + \sin^4 x\) (plavo) i njenu transformaciju \(1 - \frac{1}{2}\sin^2 2x\) (zeleno) -- potpuno se poklapaju

Rešenje korak po korak

1

Prepoznajmo strukturu izraza

Imamo zbir četvrtih stepena:

\[\cos^4 x + \sin^4 x = (\cos^2 x)^2 + (\sin^2 x)^2\]

Ovo je oblik \(a^2 + b^2\) gde je \(a = \cos^2 x\) i \(b = \sin^2 x\).

2

Primenimo algebarski identitet

Koristimo \(a^2 + b^2 = (a+b)^2 - 2ab\):

\[(\cos^2 x)^2 + (\sin^2 x)^2 = (\cos^2 x + \sin^2 x)^2 - 2\cos^2 x \cdot \sin^2 x\]
3

Primenimo Pitagorin identitet

Pošto je \(\cos^2 x + \sin^2 x = 1\), sledi:

\[= 1^2 - 2\cos^2 x \cdot \sin^2 x = 1 - 2\sin^2 x \cos^2 x\]
Proveri se: Koliko iznosi \(\cos^4\frac{\pi}{4} + \sin^4\frac{\pi}{4}\)?

\(\cos\frac{\pi}{4} = \sin\frac{\pi}{4} = \frac{\sqrt{2}}{2}\), pa je \(\left(\frac{\sqrt{2}}{2}\right)^4 + \left(\frac{\sqrt{2}}{2}\right)^4 = \frac{1}{4} + \frac{1}{4} = \frac{1}{2}\).

Formula daje: \(1 - 2 \cdot \frac{1}{4} \cdot \frac{1}{4} \cdot 4 = 1 - \frac{1}{2} = \frac{1}{2}\). Poklapa se!

4

Koristimo formulu dvostrukog ugla

Iz \(\sin 2x = 2\sin x \cos x\) kvadriranjem dobijamo:

\[\sin^2 2x = 4\sin^2 x \cos^2 x\]

Odavde izrazimo proizvod:

\[\sin^2 x \cos^2 x = \frac{\sin^2 2x}{4}\]
5

Zamenimo u izraz

Uvrštavamo u rezultat iz koraka 3:

\[1 - 2\sin^2 x \cos^2 x = 1 - 2 \cdot \frac{\sin^2 2x}{4}\]
6

Pojednostavimo

Skraćujemo dvojku:

\[= 1 - \frac{2}{4}\sin^2 2x = 1 - \frac{1}{2}\sin^2 2x\]

Ovo je konačan pojednostavljeni oblik.

Proveri se: Za \(x = \frac{\pi}{6}\), koliko iznosi \(\cos^4 x + \sin^4 x\)?

Direktno: \(\cos^4\frac{\pi}{6} + \sin^4\frac{\pi}{6} = \left(\frac{\sqrt{3}}{2}\right)^4 + \left(\frac{1}{2}\right)^4 = \frac{9}{16} + \frac{1}{16} = \frac{10}{16} = \frac{5}{8}\).

Formulom: \(1 - \frac{1}{2}\sin^2\frac{\pi}{3} = 1 - \frac{1}{2} \cdot \frac{3}{4} = 1 - \frac{3}{8} = \frac{5}{8}\). Tačno!

Ključni uvid

Zbir četvrtih stepena sinusa i kosinusa se elegantno svodi na jednostavan izraz kombinacijom algebarskog identiteta za zbir kvadrata i formule dvostrukog ugla:

\[\cos^4 x + \sin^4 x = 1 - \frac{1}{2}\sin^2 2x\]

Konačan odgovor

\(1 - \dfrac{1}{2}\sin^2 2x\)
(A) \(1\) (B) \(1 + \frac{1}{2}\sin^2 2x\) (C) \(1 - \frac{1}{2}\sin^2 2x\) (D) \(1 - \frac{1}{2}\cos^2 2x\) (E) \(1 + \frac{1}{2}\cos^2 2x\)
Provera: Za \(x = 0\): \(\cos^4 0 + \sin^4 0 = 1\), a formula daje \(1 - \frac{1}{2}\sin^2 0 = 1\). Za \(x = \frac{\pi}{4}\): obe strane daju \(\frac{1}{2}\). Za \(x = \frac{\pi}{6}\): obe strane daju \(\frac{5}{8}\). Identitet važi za sve vrednosti.

Česte greške

Brkanje sa Pitagorinim identitetom: Mnogi učenici pomisle da je \(\cos^4 x + \sin^4 x = 1\), po analogiji sa \(\cos^2 x + \sin^2 x = 1\). Međutim, Pitagorin identitet važi samo za druge stepene, ne za četvrte.

Pogrešan znak ispred \(\frac{1}{2}\sin^2 2x\): Kako oduzimamo \(2\sin^2 x\cos^2 x\), ispred člana sa \(\sin^2 2x\) stoji minus. Odgovor \(1 + \frac{1}{2}\sin^2 2x\) je pogrešan -- taj izraz bi bio veći od 1, što je nemoguće jer je za \(x = \frac{\pi}{4}\) vrednost jednaka \(\frac{1}{2}\).

Dodatni izazov

Pojednostavite izraz \(\cos^6 x + \sin^6 x\).

Nagoveštaj

Koristite formulu za zbir kubova: \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\) sa \(a = \cos^2 x\) i \(b = \sin^2 x\). Pošto je \(a + b = 1\), dobijate \(a^2 - ab + b^2 = (a^2 + b^2) - ab\). Već znamo koliko je \(a^2 + b^2\) iz ovog zadatka!
