🎯 Plan rešavanja

Razdvojićemo jednačinu na dva slučaja u zavisnosti od znaka izraza pod apsolutnom vrednošću: kada je \(3x + 3 \geq 0\) i kada je \(3x + 3 < 0\). Za svaki slučaj rešavamo linearnu jednačinu, proveravamo da li rešenje pripada odgovarajućem intervalu, a zatim računamo zbir kvadrata svih validnih rešenja.

📚 Podsetnik iz teorije
Definicija apsolutne vrednosti
\[ |a| = \begin{cases} a, & \text{ako } a \geq 0 \\ -a, & \text{ako } a < 0 \end{cases} \]

Apsolutna vrednost broja je njegovo rastojanje od nule na brojevnoj pravoj. Uvek je nenegativan broj.

Rešavanje jednačina sa apsolutnom vrednošću
\[ |f(x)| = \begin{cases} f(x), & \text{ako } f(x) \geq 0 \\ -f(x), & \text{ako } f(x) < 0 \end{cases} \]

Jednačinu sa apsolutnom vrednošću rešavamo razdvajanjem na slučajeve. Ključno je proveriti da svako dobijeno rešenje zadovoljava uslov pod kojim smo ga dobili.

Kritična tačka
\[ 3x + 3 = 0 \implies x = -1 \]

Kritična tačka je vrednost \(x\) u kojoj izraz pod apsolutnom vrednošću menja znak. Ona deli brojevnu pravu na dva intervala koje razmatramo odvojeno.

🎨 Grafički prikaz

Grafički prikaz funkcija \(y = |3x + 3| + x\) i \(y = 3\) sa označenim presecima

🛠 Rešenje korak po korak
1 Odredimo kritičnu tačku

Izraz pod apsolutnom vrednošću je \(3x + 3\). Izjednačavamo ga sa nulom da bismo našli gde menja znak:

\[ 3x + 3 = 0 \implies 3x = -3 \implies x = -1 \]

Ovo znači da posmatramo dva intervala: \(x \geq -1\) (gde je \(3x + 3 \geq 0\)) i \(x < -1\) (gde je \(3x + 3 < 0\)).

2 Slučaj 1: \(x \geq -1\)

Kada je \(x \geq -1\), izraz \(3x + 3\) je nenegativan, pa važi \(|3x + 3| = 3x + 3\). Jednačina postaje:

\[ 3x + 3 + x = 3 \]

Saberemo članove sa \(x\):

\[ 4x + 3 = 3 \]

Oduzimamo 3 sa obe strane:

\[ 4x = 0 \]

Delimo sa 4:

\[ x = 0 \]

Proveravamo uslov: \(0 \geq -1\) — tačno! Rešenje \(x_1 = 0\) je validno.

Proveri se: Koliko je \(|3 \cdot 0 + 3| + 0\)?

\(|0 + 3| + 0 = |3| + 0 = 3\). Jednako je desnoj strani, pa je \(x = 0\) zaista rešenje.

3 Slučaj 2: \(x < -1\)

Kada je \(x < -1\), izraz \(3x + 3\) je negativan, pa važi \(|3x + 3| = -(3x + 3) = -3x - 3\). Jednačina postaje:

\[ -3x - 3 + x = 3 \]

Saberemo članove sa \(x\):

\[ -2x - 3 = 3 \]

Dodamo 3 sa obe strane:

\[ -2x = 6 \]

Delimo sa \(-2\):

\[ x = -3 \]

Proveravamo uslov: \(-3 < -1\) — tačno! Rešenje \(x_2 = -3\) je validno.

Proveri se: Koliko je \(|3 \cdot (-3) + 3| + (-3)\)?

\(|-9 + 3| + (-3) = |-6| - 3 = 6 - 3 = 3\). Jednako je desnoj strani, pa je i \(x = -3\) rešenje.

4 Zbir kvadrata rešenja

Imamo dva rešenja: \(x_1 = 0\) i \(x_2 = -3\). Zbir njihovih kvadrata je:

\[ x_1^2 + x_2^2 = 0^2 + (-3)^2 \]
\[ = 0 + 9 = 9 \]
Proveri se: Da li smo pronašli sva rešenja?

Da. Razmotrili smo oba moguća slučaja (\(x \geq -1\) i \(x < -1\)) koji zajedno pokrivaju celu realnu pravu. Svaki slučaj dao je po jedno rešenje, i oba su zadovoljila odgovarajuće uslove.

Ključni uvid

Jednačina sa apsolutnom vrednošću \(|3x + 3| + x = 3\) ima tačno dva rešenja — jedno u svakom intervalu koji definiše kritična tačka \(x = -1\). Jedno rešenje je trivijalno (\(x = 0\), sa kvadratom 0), a kompletni zbir kvadrata potiče od drugog rešenja:

\[ 0^2 + (-3)^2 = 9 \]
Konačni odgovor
\(x_1^2 + x_2^2 = 0^2 + (-3)^2 = 9\)
(A) \(10\) (B) \(4\) (C) \(9\) (D) \(5\) (E) \(2\)

Verifikacija: Za \(x = 0\): \(|3 \cdot 0 + 3| + 0 = 3\) ✔. Za \(x = -3\): \(|3 \cdot (-3) + 3| + (-3) = 6 - 3 = 3\) ✔. Zbir kvadrata: \(0 + 9 = 9\).

⚠️ Česte greške
Zaboravljanje provere uslova

Svako rešenje dobijeno u jednom slučaju mora da zadovolji uslov tog slučaja (npr. \(x \geq -1\) ili \(x < -1\)). Ako se ovo preskoči, može se prihvatiti lažno rešenje koje ne pripada odgovarajućem intervalu.

Pogrešan znak pri uklanjanju apsolutne vrednosti

Kada je izraz pod apsolutnom vrednošću negativan, \(|3x + 3| = -(3x + 3) = -3x - 3\). Česta greška je da se zapiše samo \(-3x + 3\) ili da se zaboravi promeniti znak oba člana u zagradi.

🏆 Dodatni izazov

Reši jednačinu \(|2x - 4| + 3x = 10\) i odredi zbir svih rešenja (ne kvadrata, nego samih rešenja).

Nagoveštaj

Kritična tačka je \(x = 2\). Razdvoji na slučajeve \(x \geq 2\) i \(x < 2\). Ne zaboravi da proveriš da li svako rešenje pripada odgovarajućem intervalu!
