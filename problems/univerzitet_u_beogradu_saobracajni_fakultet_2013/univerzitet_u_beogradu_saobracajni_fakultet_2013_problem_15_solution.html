Trigonometrijska jednačina na datom intervalu

Trigonometrija: rešavanje jednačina pomoću identiteta dvostrukog ugla, faktorizacije i određivanje broja rešenja na intervalu \([0, 3\pi]\)

📝 Tekst zadatka
Broj različitih rešenja jednačine \(1 + \sin 2x - 2\sin x = \cos 2x\) na intervalu \([0, 3\pi]\) je:
(A)
\(6\)
(B)
\(3\)
(C)
\(4\)
(D)
\(2\)
(E)
\(5\)
🎯 Plan rešavanja

Primenom identiteta dvostrukog ugla (\(\sin 2x = 2\sin x \cos x\), \(\cos 2x = 1 - 2\sin^2 x\)) svešćemo jednačinu na oblik koji se može faktorizovati po \(\sin x\). Zatim ćemo odvojeno rešiti svaki faktor i prebrojati sva rešenja na intervalu \([0, 3\pi]\).

📚 Podsetnik iz teorije
Identiteti dvostrukog ugla

Dvostruki ugao za sinus i kosinus:

\[\sin 2x = 2\sin x \cos x\]
\[\cos 2x = 1 - 2\sin^2 x = 2\cos^2 x - 1 = \cos^2 x - \sin^2 x\]
Pomoćni ugao: \(\sin x + \cos x\)

Zbir \(\sin x + \cos x\) može se izraziti kao:

\[\sin x + \cos x = \sqrt{2} \sin\!\left(x + \frac{\pi}{4}\right)\]

jer je \(\sin x + \cos x = \sqrt{2}\left(\frac{1}{\sqrt{2}}\sin x + \frac{1}{\sqrt{2}}\cos x\right) = \sqrt{2}\sin\!\left(x + \frac{\pi}{4}\right)\).

Osnovna rešenja za \(\sin x = 0\) i \(\sin \alpha = \frac{1}{\sqrt{2}}\)

Jednačina \(\sin x = 0\) ima rešenja:

\[x = k\pi, \quad k \in \mathbb{Z}\]

Jednačina \(\sin \alpha = \frac{\sqrt{2}}{2}\) ima rešenja:

\[\alpha = \frac{\pi}{4} + 2k\pi \quad \text{ili} \quad \alpha = \frac{3\pi}{4} + 2k\pi, \quad k \in \mathbb{Z}\]
🎨 Grafički prikaz

Grafik funkcija \(f(x) = 1 + \sin 2x - 2\sin x\) (plava) i \(g(x) = \cos 2x\) (ljubičasta) na intervalu \([0, 3\pi]\). Presečne tačke su rešenja jednačine.

🛠 Rešenje korak po korak
1 Zapišemo polaznu jednačinu

Polazna jednačina glasi:

\[1 + \sin 2x - 2\sin x = \cos 2x\]

Cilj nam je da primenimo identitete dvostrukog ugla i sve izrazimo preko \(\sin x\) i \(\cos x\).

2 Primenimo identitete dvostrukog ugla

Zamenimo \(\sin 2x = 2\sin x \cos x\) i \(\cos 2x = 1 - 2\sin^2 x\):

\[1 + 2\sin x \cos x - 2\sin x = 1 - 2\sin^2 x\]
3 Pojednostavimo jednačinu

Jedinice se pokrate (\(1 - 1 = 0\)):

\[2\sin x \cos x - 2\sin x + 2\sin^2 x = 0\]

Iznesemo zajednički faktor \(2\sin x\):

\[2\sin x (\cos x - 1 + \sin x) = 0\]

Preglednije:

\[2\sin x (\sin x + \cos x - 1) = 0\]
Proveri sebe: Koji su mogući slučajevi kad je proizvod jednak nuli?

Proizvod dva faktora jednak je nuli ako i samo ako je bar jedan od njih jednak nuli. Dakle imamo dva slučaja: \(\sin x = 0\) ili \(\sin x + \cos x - 1 = 0\).

4 Slučaj 1: \(\sin x = 0\)

Opšte rešenje jednačine \(\sin x = 0\) je:

\[x = k\pi, \quad k \in \mathbb{Z}\]

Tražimo rešenja na intervalu \([0, 3\pi]\):

\[0 \leq k\pi \leq 3\pi \quad \Rightarrow \quad 0 \leq k \leq 3\]

Za \(k = 0, 1, 2, 3\) dobijamo:

\[x_1 = 0, \quad x_2 = \pi, \quad x_3 = 2\pi, \quad x_4 = 3\pi\]

Iz prvog faktora dobijamo 4 rešenja na datom intervalu.

5 Slučaj 2: \(\sin x + \cos x = 1\)

Drugi faktor daje jednačinu:

\[\sin x + \cos x = 1\]

Koristimo pomoćni ugao. Znamo da važi:

\[\sin x + \cos x = \sqrt{2}\sin\!\left(x + \frac{\pi}{4}\right)\]

Dakle:

\[\sqrt{2}\sin\!\left(x + \frac{\pi}{4}\right) = 1\]
\[\sin\!\left(x + \frac{\pi}{4}\right) = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}\]
6 Rešimo jednačinu za pomoćni ugao

Jednačina \(\sin\alpha = \frac{\sqrt{2}}{2}\) ima dva niza rešenja:

\[\alpha = \frac{\pi}{4} + 2k\pi \quad \text{ili} \quad \alpha = \pi - \frac{\pi}{4} + 2k\pi = \frac{3\pi}{4} + 2k\pi\]

Pošto je \(\alpha = x + \frac{\pi}{4}\), dobijamo:

\[x + \frac{\pi}{4} = \frac{\pi}{4} + 2k\pi \quad \Rightarrow \quad x = 2k\pi\]
\[x + \frac{\pi}{4} = \frac{3\pi}{4} + 2k\pi \quad \Rightarrow \quad x = \frac{\pi}{2} + 2k\pi\]
Proveri sebe: Koliko rešenja oblika \(x = 2k\pi\) pripada intervalu \([0, 3\pi]\)?

Za \(k = 0\): \(x = 0\). Za \(k = 1\): \(x = 2\pi\). Za \(k = 2\): \(x = 4\pi > 3\pi\). Dakle, \(x = 0\) i \(x = 2\pi\), ali oba su već pronađena u Slučaju 1.

7 Nađemo rešenja iz Slučaja 2 na \([0, 3\pi]\)

Iz \(x = 2k\pi\) na \([0, 3\pi]\):

\[x = 0 \quad (k=0), \quad x = 2\pi \quad (k=1)\]

Oba su već pronađena u Slučaju 1 (jer za \(x = 0\) i \(x = 2\pi\) važi \(\sin x = 0\)).

Iz \(x = \frac{\pi}{2} + 2k\pi\) na \([0, 3\pi]\):

\[x = \frac{\pi}{2} \quad (k=0), \quad x = \frac{5\pi}{2} \quad (k=1)\]

Ovo su dva nova rešenja koja nisu iz Slučaja 1.

8 Prebrojimo sva različita rešenja

Objedinimo sva rešenja iz oba slučaja u rastućem redosledu:

  • 1. \(x = 0\) (Slučaj 1 i Slučaj 2)
  • 2. \(x = \dfrac{\pi}{2}\) (Slučaj 2)
  • 3. \(x = \pi\) (Slučaj 1)
  • 4. \(x = 2\pi\) (Slučaj 1 i Slučaj 2)
  • 5. \(x = \dfrac{5\pi}{2}\) (Slučaj 2)
  • 6. \(x = 3\pi\) (Slučaj 1)

Ukupno 6 različitih rešenja.

Proveri sebe: Verifikacija za \(x = \frac{5\pi}{2}\)

Uvrštavamo \(x = \frac{5\pi}{2}\):

\[\sin\frac{5\pi}{2} = 1, \quad \cos\frac{5\pi}{2} = 0, \quad \sin 5\pi = 0, \quad \cos 5\pi = -1\]

Leva strana: \(1 + 0 - 2 \cdot 1 = -1\). Desna strana: \(-1\). Jednako! ✔

Ključni uvid

Identiteti dvostrukog ugla omogućuju da se jednačina svede na faktorizovani oblik \(2\sin x(\sin x + \cos x - 1) = 0\). Ključ je prepoznati da se drugi faktor elegantno rešava pomoćnim uglom. Dva rešenja (\(x = 0\) i \(x = 2\pi\)) pojavljuju se u oba slučaja, pa moramo pažljivo ukloniti duplikate pre konačnog prebrojavanja.

Konačni odgovor
Broj rešenja: \(6\)
(A) \(6\) (B) \(3\) (C) \(4\) (D) \(2\) (E) \(5\)

Verifikacija: Svih 6 rešenja (\(0,\; \frac{\pi}{2},\; \pi,\; 2\pi,\; \frac{5\pi}{2},\; 3\pi\)) možemo uvrstiti u polaznu jednačinu i potvrditi da je zadovoljavaju. Na grafiku iznad jasno se vide tačno 6 presečnih tačaka krivih \(f(x)\) i \(g(x)\) na intervalu \([0, 3\pi]\). ✔

⚠️ Česte greške
Zaboravljena rešenja na krajevima intervala

Interval je zatvoren: \([0, 3\pi]\). To znači da su \(x = 0\) i \(x = 3\pi\) validna rešenja. Ako se uzme otvoreni interval, gube se dva rešenja i dobija se pogrešan odgovor 4.

Duplo brojanje zajedničkih rešenja

Rešenja \(x = 0\) i \(x = 2\pi\) zadovoljavaju oba faktora (\(\sin x = 0\) i \(\sin x + \cos x = 1\)). Moraju se brojati samo jednom, inače se dobija 8 umesto 6.

🏆 Dodatni izazov

Koliko rešenja ima jednačina \(1 + \sin 2x - 2\sin x = \cos 2x\) na intervalu \([0, 5\pi]\)?

Nagoveštaj

Rešenja iz \(\sin x = 0\) su \(x = 0, \pi, 2\pi, 3\pi, 4\pi, 5\pi\) (6 rešenja). Rešenja iz \(\sin x + \cos x = 1\) na \([0, 5\pi]\) su \(x = 0, \frac{\pi}{2}, 2\pi, \frac{5\pi}{2}, 4\pi, \frac{9\pi}{2}\) (6 rešenja, od kojih su 3 nova). Prebrojte jedinstvena rešenja!
