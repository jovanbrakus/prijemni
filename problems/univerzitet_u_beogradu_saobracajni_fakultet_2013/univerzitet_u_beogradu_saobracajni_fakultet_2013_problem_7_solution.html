Deljenje polinoma i ostatak

Algebra — deljenje polinoma sa ostatkom i nalaženje vrednosti izraza

📝 Postavka zadatka

Dati su polinomi \(P(x) = x^4 + 2x^3 + 5x^2 - x + 7\) i \(Q(x) = x^2 + x + 2\). Ako je \(R(x) = ax + b\) ostatak deljenja polinoma \(P(x)\) sa polinomom \(Q(x)\), tada je \(2b - a\) jednako:

🎯 Plan rešavanja

Primenićemo algoritam deljenja polinoma (dugačko deljenje) da podelimo \(P(x)\) sa \(Q(x)\). Pošto je \(\deg P = 4\) i \(\deg Q = 2\), količnik će biti polinom stepena 2, a ostatak \(R(x) = ax + b\) stepena najviše 1. Kada nađemo \(a\) i \(b\), izračunamo \(2b - a\).

📚 Podsetnik iz teorije
📐 Ključne formule i teoreme
  • Teorema o deljenju polinoma: Za polinome \(P(x)\) i \(Q(x)\), gde je \(Q(x) \neq 0\), postoje jedinstveni polinomi \(D(x)\) (količnik) i \(R(x)\) (ostatak) takvi da je: \[ P(x) = Q(x) \cdot D(x) + R(x), \quad \deg R < \deg Q \]
  • Algoritam dugačkog deljenja: U svakom koraku delimo vodeći član trenutnog ostatka vodećim članom delioca, množimo ceo delilac dobijenim rezultatom, pa oduzimamo od trenutnog ostatka.
  • Stepen ostatka: Ako je \(\deg Q = 2\), tada je \(\deg R \leq 1\), tj. ostatak je oblika \(R(x) = ax + b\).
  • Provera: Rezultat deljenja se proverava tako što se pokaže da je \(P(x) = Q(x) \cdot D(x) + R(x)\) za sve vrednosti \(x\).

🛠 Rešenje korak po korak
1 Postavljanje deljenja

Delimo polinom \(P(x) = x^4 + 2x^3 + 5x^2 - x + 7\) polinomom \(Q(x) = x^2 + x + 2\).

Pošto je stepen deljenika 4, a stepen delioca 2, količnik \(D(x)\) će biti polinom stepena \(4 - 2 = 2\), tj. oblika \(D(x) = cx^2 + dx + e\).

Ostatak \(R(x) = ax + b\) je stepena strogo manjeg od stepena delioca, dakle stepena najviše 1.

2 Prvi korak deljenja: delimo \(x^4\) sa \(x^2\)

Delimo vodeći član deljenika vodećim članom delioca:

\[ \frac{x^4}{x^2} = x^2 \]

Množimo ceo delilac sa \(x^2\):

\[ x^2 \cdot (x^2 + x + 2) = x^4 + x^3 + 2x^2 \]

Oduzimamo od deljenika:

\[ \begin{align} &(x^4 + 2x^3 + 5x^2 - x + 7) - (x^4 + x^3 + 2x^2) \\[6pt] &= (2-1)x^3 + (5-2)x^2 - x + 7 \\[6pt] &= x^3 + 3x^2 - x + 7 \end{align} \]
Proveri se: Koliki je stepen novog ostatka nakon prvog koraka?

Novi ostatak je \(x^3 + 3x^2 - x + 7\), koji je stepena 3. Pošto je \(\deg Q = 2\), nastavljamo deljenje jer je \(3 \geq 2\).

3 Drugi korak deljenja: delimo \(x^3\) sa \(x^2\)

Delimo vodeći član novog ostatka vodećim članom delioca:

\[ \frac{x^3}{x^2} = x \]

Množimo ceo delilac sa \(x\):

\[ x \cdot (x^2 + x + 2) = x^3 + x^2 + 2x \]

Oduzimamo od prethodnog ostatka:

\[ \begin{align} &(x^3 + 3x^2 - x + 7) - (x^3 + x^2 + 2x) \\[6pt] &= (3-1)x^2 + (-1-2)x + 7 \\[6pt] &= 2x^2 - 3x + 7 \end{align} \]
4 Treći korak deljenja: delimo \(2x^2\) sa \(x^2\)

Delimo vodeći član novog ostatka vodećim članom delioca:

\[ \frac{2x^2}{x^2} = 2 \]

Množimo ceo delilac sa \(2\):

\[ 2 \cdot (x^2 + x + 2) = 2x^2 + 2x + 4 \]

Oduzimamo od prethodnog ostatka:

\[ \begin{align} &(2x^2 - 3x + 7) - (2x^2 + 2x + 4) \\[6pt] &= (-3-2)x + (7-4) \\[6pt] &= -5x + 3 \end{align} \]

Stepen ostatka \(-5x + 3\) je 1, što je manje od stepena delioca (\(2\)). Deljenje je završeno!

Proveri se: Koliko iznosi količnik \(D(x)\)?

Sabiranjem svih članova koje smo dobili u koracima deljenja: \(D(x) = x^2 + x + 2\). Zanimljivo, količnik je identičan deliocu!

5 Određivanje koeficijenata \(a\) i \(b\)

Ostatak deljenja je:

\[ R(x) = -5x + 3 \]

Poređenjem sa oblikom \(R(x) = ax + b\), dobijamo:

\[ a = -5, \qquad b = 3 \]
6 Izračunavanje \(2b - a\)

Zamenjujemo vrednosti \(a = -5\) i \(b = 3\) u traženi izraz:

\[ 2b - a = 2 \cdot 3 - (-5) = 6 + 5 = 11 \]
Proveri se: Da li se rezultat može verifikovati?

Da! Proverimo da važi \(P(x) = Q(x) \cdot D(x) + R(x)\): \[ (x^2 + x + 2)(x^2 + x + 2) + (-5x + 3) = (x^2 + x + 2)^2 - 5x + 3 \] Razvijamo: \((x^2 + x + 2)^2 = x^4 + 2x^3 + 5x^2 + 4x + 4\). Dodajemo ostatak: \(x^4 + 2x^3 + 5x^2 + 4x + 4 - 5x + 3 = x^4 + 2x^3 + 5x^2 - x + 7 = P(x)\). Tačno!

Ključni uvid

U ovom zadatku, količnik deljenja je identičan deliocu: \(D(x) = Q(x) = x^2 + x + 2\). To znači da se polinom \(P(x)\) može zapisati kao:

\[ P(x) = (x^2 + x + 2)^2 - 5x + 3 \]

Ovo je lep strukturni rezultat koji bi se mogao iskoristiti i za alternativno rešavanje: razvijemo \((x^2 + x + 2)^2\) i poredimo sa \(P(x)\) da nađemo ostatak.

Konačni odgovor
\(2b - a = 2 \cdot 3 - (-5) = 11\)
(A) \(11\) (B) \(5\) (C) \(1\) (D) \(9\) (E) \(7\)

Verifikacija: \(P(x) = Q(x) \cdot (x^2 + x + 2) + (-5x + 3)\). Razvijanje potvrđuje \(P(x) = x^4 + 2x^3 + 5x^2 - x + 7\). Ostatak \(R(x) = -5x + 3\) daje \(a = -5\), \(b = 3\), pa je \(2b - a = 6 + 5 = 11\).

⚠️ Česte greške
Greška u znaku prilikom oduzimanja

Najčešća greška je pogrešan znak pri oduzimanju. Na primer, u trećem koraku: \((-3x) - (2x) = -5x\), a ne \(-x\). Treba pažljivo oduzimati svaki član ponaosob.

Pogrešno izračunavanje \(2b - a\)

Pošto je \(a = -5\), izraz \(2b - a = 2b - (-5) = 2b + 5\). Lako je zaboraviti da se minus ispred negativnog broja pretvara u plus. Ako se računa \(2 \cdot 3 - 5 = 1\) umesto \(2 \cdot 3 - (-5) = 11\), dobija se pogrešan odgovor (C).

🏆 Dodatni izazov

Nađite ostatak deljenja polinoma \(P(x) = x^5 + 3x^4 + 2x^3 - x^2 + x - 1\) sa polinomom \(Q(x) = x^2 + x + 1\).

Nagoveštaj

Postupak je isti kao u ovom zadatku, samo sa jednim korakom više jer je \(\deg P = 5\). Količnik će biti stepena 3. Pazite na znakove prilikom oduzimanja! Alternativno, možete iskoristiti činjenicu da su koreni polinoma \(x^2 + x + 1\) kompleksni: \(\omega = e^{i2\pi/3}\). Uvrštavanjem \(\omega\) u \(P(x)\) dobijate vrednost ostatka u toj tački.
