Logaritamska nejednačina sa složenim logaritmima

Rešavanje nejednačine sa kompozicijom logaritama različitih osnova i određivanje oblika skupa rešenja

📝 Tekst zadatka
Neka je skup \(S\) rešenje nejednačine
\[ \log_{\frac{1}{2}}\!\left(\log_2\!\left(x^2 - x + \frac{19}{16}\right)\right) \geq -1 \]
Tačan je iskaz:
(A)
\(S = [a, b) \cup (c, d]\) i \(a + b + c + d = 2\)
(B)
\(S = [a, b]\) i \(a + b = 1\)
(C)
\(S = (-\infty, a] \cup [b, +\infty)\) i \(a + b = 1\)
(D)
\(S = [a, b) \cup (c, d]\) i \(a + b + c + d = 4\)
(E)
\(S = (-\infty, a] \cup (b, c)\) i \(a + b + c = 13\)
🎯 Plan rešavanja

1. Postavimo uslove definisanosti za oba logaritma (argument unutrašnjeg logaritma mora biti pozitivan, a vrednost unutrašnjeg logaritma mora biti pozitivna za spoljašnji).
2. Rešimo glavnu nejednačinu koristeći svojstvo da je \(\log_{1/2}\) opadajuća funkcija, pa se smer nejednačine menja.
3. Presečemo sve dobijene uslove da bismo odredili konačan skup rešenja \(S\).
4. Identifikujemo oblik skupa i izračunamo zbir graničnih vrednosti.

📚 Podsetnik iz teorije
Svojstva logaritamske funkcije

Logaritamska funkcija \(\log_a(x)\) je definisana za \(x > 0\) i \(a > 0,\; a \neq 1\).

\[ \log_a(x) = y \quad \Longleftrightarrow \quad a^y = x \]

Za osnovu \(a > 1\): funkcija je rastuća (veći argument, veća vrednost).
Za osnovu \(0 < a < 1\): funkcija je opadajuća (veći argument, manja vrednost).

Rešavanje nejednačina sa opadajućom logaritamskom funkcijom

Za osnovu \(0 < a < 1\), nejednačina \(\log_a(t) \geq c\) ekvivalentna je sa:

\[ \log_a(t) \geq c \quad \Longleftrightarrow \quad 0 < t \leq a^c \]

Smer nejednačine se menja zato što je funkcija opadajuća. Uslov \(t > 0\) je domen logaritma.

Rešavanje nejednačina sa rastućom logaritamskom funkcijom

Za osnovu \(a > 1\), nejednačina \(\log_a(u) \leq c\) ekvivalentna je sa:

\[ \log_a(u) \leq c \quad \Longleftrightarrow \quad 0 < u \leq a^c \]

Smer nejednačine se čuva jer je funkcija rastuća. Uslov \(u > 0\) je domen logaritma.

🎨 Vizuelni prikaz: brojevna prava sa skupom rešenja
Skup rešenja \(S\)
Uslov \(\log_2(\ldots) \leq 2\): interval \(\left[-\frac{5}{4},\; \frac{9}{4}\right]\)
Isključene tačke \(x = \frac{1}{4}\) i \(x = \frac{3}{4}\)
🛠 Rešenje korak po korak
1 Uslov definisanosti unutrašnjeg logaritma

Unutrašnji logaritam je \(\log_2\!\left(x^2 - x + \frac{19}{16}\right)\). Da bi bio definisan, argument mora biti strogo pozitivan:

\[ x^2 - x + \frac{19}{16} > 0 \]

Izračunajmo diskriminantu ovog kvadratnog trinoma:

\[ D = (-1)^2 - 4 \cdot 1 \cdot \frac{19}{16} = 1 - \frac{19}{4} = -\frac{15}{4} < 0 \]

Pošto je \(D < 0\) i vodeći koeficijent je pozitivan (\(a = 1 > 0\)), ovaj trinom je uvek pozitivan za svako \(x \in \mathbb{R}\). Dakle, unutrašnji logaritam je uvek definisan.

2 Uslov definisanosti spoljašnjeg logaritma

Spoljašnji logaritam je \(\log_{1/2}(t)\) gde je \(t = \log_2\!\left(x^2 - x + \frac{19}{16}\right)\). Da bi bio definisan:

\[ t > 0 \quad \Longleftrightarrow \quad \log_2\!\left(x^2 - x + \frac{19}{16}\right) > 0 \]

Pošto je osnova \(2 > 1\), logaritam je rastuća funkcija, pa je \(\log_2(u) > 0\) ekvivalentno sa \(u > 2^0 = 1\):

\[ x^2 - x + \frac{19}{16} > 1 \]

Sredimo:

\[ x^2 - x + \frac{19}{16} - 1 > 0 \] \[ x^2 - x + \frac{3}{16} > 0 \]

Nađimo nule ovog trinoma:

\[ D = 1 - 4 \cdot \frac{3}{16} = 1 - \frac{3}{4} = \frac{1}{4} \] \[ x_{1,2} = \frac{1 \pm \frac{1}{2}}{2} \] \[ x_1 = \frac{1 - \frac{1}{2}}{2} = \frac{1}{4}, \quad x_2 = \frac{1 + \frac{1}{2}}{2} = \frac{3}{4} \]

Dakle, \(x^2 - x + \frac{3}{16} = \left(x - \frac{1}{4}\right)\!\left(x - \frac{3}{4}\right)\). Pošto je vodeći koeficijent pozitivan, trinom je pozitivan van korena:

\[ x^2 - x + \frac{3}{16} > 0 \quad \Longleftrightarrow \quad x < \frac{1}{4} \;\;\text{ili}\;\; x > \frac{3}{4} \]

Ovo je uslov pod kojim je spoljašnji logaritam definisan. Tačke \(x = \frac{1}{4}\) i \(x = \frac{3}{4}\) su isključene iz domena.

Proveri sebe: Šta se dešava u tačkama \(x = \frac{1}{4}\) i \(x = \frac{3}{4}\)?

U tim tačkama važi \(x^2 - x + \frac{19}{16} = 1\), pa je \(\log_2(1) = 0\). Tada bi trebalo izračunati \(\log_{1/2}(0)\), a to ne postoji jer je logaritam definisan samo za strogo pozitivne argumente.

3 Rešavanje glavne nejednačine

Sada rešavamo samu nejednačinu. Osnova spoljašnjeg logaritma je \(\frac{1}{2} < 1\), pa je logaritamska funkcija opadajuća. To znači:

\[ \log_{\frac{1}{2}}(t) \geq -1 \quad \Longleftrightarrow \quad 0 < t \leq \left(\frac{1}{2}\right)^{-1} = 2 \]

Uslov \(t > 0\) smo već obradili u Koraku 2. Preostaje gornja granica:

\[ \log_2\!\left(x^2 - x + \frac{19}{16}\right) \leq 2 \]

Pošto je osnova \(2 > 1\), logaritam je rastuća funkcija i smer nejednačine se čuva:

\[ x^2 - x + \frac{19}{16} \leq 2^2 = 4 \]
4 Rešavanje kvadratne nejednačine

Prenesemo sve na levu stranu:

\[ x^2 - x + \frac{19}{16} - 4 \leq 0 \] \[ x^2 - x - \frac{45}{16} \leq 0 \]

Pomnožimo sa \(16\) da eliminišemo razlomke:

\[ 16x^2 - 16x - 45 \leq 0 \]

Izračunajmo diskriminantu:

\[ D = (-16)^2 - 4 \cdot 16 \cdot (-45) = 256 + 2880 = 3136 \] \[ \sqrt{D} = \sqrt{3136} = 56 \]

Nule trinoma:

\[ x = \frac{16 \pm 56}{2 \cdot 16} = \frac{16 \pm 56}{32} \] \[ x_1 = \frac{16 - 56}{32} = \frac{-40}{32} = -\frac{5}{4}, \quad x_2 = \frac{16 + 56}{32} = \frac{72}{32} = \frac{9}{4} \]

Pošto je vodeći koeficijent pozitivan (\(16 > 0\)), trinom je negativan ili nula između korena:

\[ 16x^2 - 16x - 45 \leq 0 \quad \Longleftrightarrow \quad -\frac{5}{4} \leq x \leq \frac{9}{4} \]
Proveri sebe: Koliko je \(x^2 - x + \frac{19}{16}\) za \(x = -\frac{5}{4}\) i \(x = \frac{9}{4}\)?

Za \(x = -\frac{5}{4}\): \(\frac{25}{16} + \frac{5}{4} + \frac{19}{16} = \frac{25}{16} + \frac{20}{16} + \frac{19}{16} = \frac{64}{16} = 4\).
Za \(x = \frac{9}{4}\): \(\frac{81}{16} - \frac{9}{4} + \frac{19}{16} = \frac{81}{16} - \frac{36}{16} + \frac{19}{16} = \frac{64}{16} = 4\).
U obe tačke: \(\log_2(4) = 2\) i \(\log_{1/2}(2) = -1 \geq -1\). Jednakost važi, pa su obe tačke uključene.

5 Presek uslova — konačan skup rešenja

Kombinujemo dva uslova:

Uslov 1 (domen spoljašnjeg logaritma): \(x < \frac{1}{4}\) ili \(x > \frac{3}{4}\)

Uslov 2 (glavna nejednačina): \(-\frac{5}{4} \leq x \leq \frac{9}{4}\)

Presečemo ova dva skupa:

\[ \left[-\frac{5}{4},\; \frac{9}{4}\right] \;\cap\; \left(\left(-\infty,\; \frac{1}{4}\right) \cup \left(\frac{3}{4},\; +\infty\right)\right) \]

Presek po delovima:

\[ \left[-\frac{5}{4},\; \frac{9}{4}\right] \cap \left(-\infty,\; \frac{1}{4}\right) = \left[-\frac{5}{4},\; \frac{1}{4}\right) \] \[ \left[-\frac{5}{4},\; \frac{9}{4}\right] \cap \left(\frac{3}{4},\; +\infty\right) = \left(\frac{3}{4},\; \frac{9}{4}\right] \]

Dakle, konačan skup rešenja je:

\[ S = \left[-\frac{5}{4},\; \frac{1}{4}\right) \cup \left(\frac{3}{4},\; \frac{9}{4}\right] \]
6 Identifikacija oblika i izračunavanje zbira

Skup \(S\) ima oblik \([a, b) \cup (c, d]\) gde je:

\[ a = -\frac{5}{4}, \quad b = \frac{1}{4}, \quad c = \frac{3}{4}, \quad d = \frac{9}{4} \]

Izračunamo zbir:

\[ a + b + c + d = -\frac{5}{4} + \frac{1}{4} + \frac{3}{4} + \frac{9}{4} = \frac{-5 + 1 + 3 + 9}{4} = \frac{8}{4} = 2 \]

Ovo odgovara iskazu (A): \(S = [a, b) \cup (c, d]\) i \(a + b + c + d = 2\).

Proveri sebe: Zašto su zagrade baš takve — \([\) i \()\) za prvi deo, \((\) i \(]\) za drugi?

Tačka \(x = -\frac{5}{4}\) je uključena jer tu važi jednakost u \(\leq\) (iz Koraka 4). Tačka \(x = \frac{1}{4}\) je isključena jer tu \(\log_2(\ldots) = 0\), pa je argument spoljašnjeg logaritma nula (nije definisan). Analogno, \(x = \frac{3}{4}\) je isključena, a \(x = \frac{9}{4}\) je uključena.

Ključni uvid

Kod složenih logaritamskih nejednačina sa kompozicijom logaritama, ključno je pažljivo razlikovati uslove definisanosti (domen) od uslova nejednačine. U ovom zadatku, upravo uslov definisanosti spoljašnjeg logaritma (\(\log_{1/2}(t)\) zahteva \(t > 0\)) stvara „rupu" u intervalu rešenja i deli ga na dva dela. Opadajuća osnova \(\frac{1}{2}\) okreće smer nejednačine, pretvarajući \(\geq -1\) u gornju granicu \(\leq 2\) za unutrašnji logaritam.

\[ \log_{\frac{1}{2}}(t) \geq -1 \;\;\xRightarrow{\text{opadajuća}}\;\; 0 < t \leq 2 \;\;\Longrightarrow\;\; S = \left[-\frac{5}{4},\; \frac{1}{4}\right) \cup \left(\frac{3}{4},\; \frac{9}{4}\right] \]
Konačni odgovor
\( S = \left[-\frac{5}{4},\; \frac{1}{4}\right) \cup \left(\frac{3}{4},\; \frac{9}{4}\right] \)
(A) \(S = [a, b) \cup (c, d]\), \(a+b+c+d = 2\) (B) \(S = [a, b]\), \(a+b = 1\) (C) \(S = (-\infty, a] \cup [b, +\infty)\), \(a+b = 1\) (D) \(S = [a, b) \cup (c, d]\), \(a+b+c+d = 4\) (E) \(S = (-\infty, a] \cup (b, c)\), \(a+b+c = 13\)

Verifikacija: Za \(x = -\frac{5}{4}\): \(x^2 - x + \frac{19}{16} = 4\), \(\log_2(4) = 2\), \(\log_{1/2}(2) = -1 \geq -1\) ✔. Za \(x = \frac{9}{4}\): isti račun daje istu vrednost ✔. Za \(x = 0\) (iz prvog intervala): \(x^2 - x + \frac{19}{16} = \frac{19}{16}\), \(\log_2\!\left(\frac{19}{16}\right) \approx 0{,}248\), \(\log_{1/2}(0{,}248) \approx 2{,}01 \geq -1\) ✔. Zbir granica: \(-\frac{5}{4} + \frac{1}{4} + \frac{3}{4} + \frac{9}{4} = 2\) ✔.

⚠️ Česte greške
Zaboravljanje uslova definisanosti spoljašnjeg logaritma

Mnogi učenici rešavaju samo glavnu nejednačinu \(\log_2(\ldots) \leq 2\) i dobijaju interval \(\left[-\frac{5}{4},\; \frac{9}{4}\right]\), zaboravljajući da argument spoljašnjeg logaritma mora biti strogo pozitivan. Bez tog uslova, tačke \(x = \frac{1}{4}\) i \(x = \frac{3}{4}\) (gde je \(\log_2(\ldots) = 0\)) bi bile pogrešno uključene, dajući pogrešan oblik skupa.

Pogrešan smer nejednačine pri opadajućoj osnovi

Logaritam sa osnovom \(\frac{1}{2} < 1\) je opadajuća funkcija. Nejednačina \(\log_{1/2}(t) \geq -1\) se prevodi u \(t \leq 2\) (a ne \(t \geq 2\)!). Greška u smeru dovodi do potpuno pogrešnog skupa rešenja.

🏆 Dodatni izazov

Kako bi se promenio skup rešenja kada bi nejednačina bila stroga: \(\log_{1/2}\!\left(\log_2\!\left(x^2 - x + \frac{19}{16}\right)\right) > -1\)?

Nagoveštaj

Stroga nejednačina \(\log_{1/2}(t) > -1\) daje \(0 < t < 2\) (umesto \(\leq 2\)). To znači da bi krajnje tačke \(x = -\frac{5}{4}\) i \(x = \frac{9}{4}\) bile isključene (umesto uključene), jer u njima važi jednakost \(\log_2(\ldots) = 2\). Skup bi bio \(\left(-\frac{5}{4},\; \frac{1}{4}\right) \cup \left(\frac{3}{4},\; \frac{9}{4}\right)\) — otvoren interval sa obe strane.
