🎯 Plan rešavanja

Kritične tačke apsolutnih vrednosti su \(x = 0\) i \(x = 1\). Analiziraćemo jednačinu u tri intervala: \(x < 0\), \(0 \leq x < 1\) i \(x \geq 1\). U svakom intervalu uklanjamo apsolutne vrednosti prema znaku izraza, rešavamo linearnu jednačinu i proveravamo da li rešenje pripada datom intervalu. Na kraju množimo sva validna rešenja.

📚 Podsetnik iz teorije
📐 Ključne formule i definicije
  • Definicija apsolutne vrednosti: \(|a| = \begin{cases} a, & a \geq 0 \\ -a, & a < 0 \end{cases}\)
  • Metoda analize slučajeva: Nulte tačke izraza unutar apsolutnih vrednosti dele brojevnu pravu na intervale. U svakom intervalu, apsolutne vrednosti se zamenjuju odgovarajućim izrazima (sa ili bez minusa).
  • Provera rešenja: Svako dobijeno rešenje mora pripadati intervalu u kome je izvedeno. Ako ne pripada — rešenje se odbacuje.
  • Kritične tačke: Za \(|x|\) kritična tačka je \(x = 0\); za \(|x - 1|\) kritična tačka je \(x = 1\).

🛠 Rešenje korak po korak
1 Određivanje kritičnih tačaka

Jednačina sadrži dva izraza sa apsolutnom vrednošću: \(|x|\) i \(|x - 1|\).

Izraz \(|x|\) menja znak u tački \(x = 0\), a izraz \(|x - 1|\) menja znak u tački \(x = 1\).

Ove dve kritične tačke dele brojevnu pravu na tri intervala:

\[ x < 0, \qquad 0 \leq x < 1, \qquad x \geq 1 \]
2 Slučaj 1: \(x < 0\)

Kada je \(x < 0\), tada je \(x\) negativno, pa \(|x| = -x\). Takođe, \(x - 1 < -1 < 0\), pa \(|x - 1| = -(x - 1) = 1 - x\).

Zamenjujemo u jednačinu:

\[ -x + (1 - x) = x + \frac{1}{2} \]

Sređujemo levu stranu:

\[ 1 - 2x = x + \frac{1}{2} \]

Prebacujemo nepoznate na jednu, a poznate na drugu stranu:

\[ 1 - \frac{1}{2} = x + 2x \] \[ \frac{1}{2} = 3x \] \[ x = \frac{1}{6} \]
Odbačeno Rešenje \(x = \dfrac{1}{6}\) ne pripada intervalu \(x < 0\) jer je \(\dfrac{1}{6} > 0\).
3 Slučaj 2: \(0 \leq x < 1\)

Kada je \(0 \leq x < 1\), tada je \(x \geq 0\), pa \(|x| = x\). Takođe, \(x - 1 < 0\), pa \(|x - 1| = 1 - x\).

Zamenjujemo u jednačinu:

\[ x + (1 - x) = x + \frac{1}{2} \]

Sređujemo levu stranu:

\[ 1 = x + \frac{1}{2} \]

Rešavamo po \(x\):

\[ x = 1 - \frac{1}{2} = \frac{1}{2} \]
Prihvaćeno Rešenje \(x = \dfrac{1}{2}\) pripada intervalu \([0, 1)\) jer je \(0 \leq \dfrac{1}{2} < 1\). ✔
Proveri se: Da li \(x = \frac{1}{2}\) zaista zadovoljava jednačinu?

Uvrštavamo: \(\left|\dfrac{1}{2}\right| + \left|\dfrac{1}{2} - 1\right| = \dfrac{1}{2} + \dfrac{1}{2} = 1\). Desna strana: \(\dfrac{1}{2} + \dfrac{1}{2} = 1\). Jednake su! ✔

4 Slučaj 3: \(x \geq 1\)

Kada je \(x \geq 1\), tada je \(x > 0\), pa \(|x| = x\). Takođe, \(x - 1 \geq 0\), pa \(|x - 1| = x - 1\).

Zamenjujemo u jednačinu:

\[ x + (x - 1) = x + \frac{1}{2} \]

Sređujemo levu stranu:

\[ 2x - 1 = x + \frac{1}{2} \]

Rešavamo po \(x\):

\[ 2x - x = \frac{1}{2} + 1 \] \[ x = \frac{3}{2} \]
Prihvaćeno Rešenje \(x = \dfrac{3}{2}\) pripada intervalu \([1, +\infty)\) jer je \(\dfrac{3}{2} \geq 1\). ✔
Proveri se: Da li \(x = \frac{3}{2}\) zaista zadovoljava jednačinu?

Uvrštavamo: \(\left|\dfrac{3}{2}\right| + \left|\dfrac{3}{2} - 1\right| = \dfrac{3}{2} + \dfrac{1}{2} = 2\). Desna strana: \(\dfrac{3}{2} + \dfrac{1}{2} = 2\). Jednake su! ✔

5 Računanje proizvoda rešenja

Jednačina ima tačno dva realna rešenja: \(x_1 = \dfrac{1}{2}\) i \(x_2 = \dfrac{3}{2}\).

Njihov proizvod je:

\[ x_1 \cdot x_2 = \frac{1}{2} \cdot \frac{3}{2} = \frac{3}{4} \]
Proveri se: Koji od ponuđenih odgovora je \(\frac{3}{4}\)?

To je odgovor (C). ✔

Ključni uvid

Na intervalu \([0, 1)\), leva strana jednačine se pojednostavljuje u konstantu \(1\), jer se \(x\) i \(-x\) poništavaju: \(|x| + |x - 1| = x + (1-x) = 1\). Ovo je posledica toga što za \(0 \leq x \leq 1\) zbir udaljenosti od tačaka 0 i 1 na brojevnoj pravoj uvek iznosi tačno 1 — tačka je „između" njih. Ovaj geometrijski uvid značajno ubrzava rešavanje.

\[ \text{Za } 0 \leq x \leq 1: \quad |x| + |x - 1| = |x - 0| + |x - 1| = 1 \]
Konačni odgovor
\(\dfrac{1}{2} \cdot \dfrac{3}{2} = \dfrac{3}{4}\)
(A) \(\dfrac{1}{8}\) (B) \(\dfrac{1}{2}\) (C) \(\dfrac{3}{4}\) (D) \(\dfrac{5}{6}\) (E) \(\dfrac{3}{2}\)

Verifikacija: \(x_1 = \frac{1}{2}\) daje \(|\frac{1}{2}| + |\frac{1}{2} - 1| = 1 = \frac{1}{2} + \frac{1}{2}\) ✔   \(x_2 = \frac{3}{2}\) daje \(|\frac{3}{2}| + |\frac{3}{2} - 1| = 2 = \frac{3}{2} + \frac{1}{2}\) ✔   Proizvod: \(\frac{1}{2} \cdot \frac{3}{2} = \frac{3}{4}\) ✔

⚠️ Česte greške
Prihvatanje rešenja koje ne pripada intervalu

U prvom slučaju (\(x < 0\)) dobija se \(x = \frac{1}{6}\), ali ovo rešenje je pozitivno i ne pripada intervalu \((-\infty, 0)\). Ako se zaboravi provera pripadnosti intervalu, može se pogrešno uračunati i ovo rešenje u proizvod.

Zaboravljena analiza svih slučajeva

Ako se razmatraju samo dva od tri intervala (na primer, preskoči se \(x \geq 1\)), pronalazi se samo jedno rešenje \(x = \frac{1}{2}\) i proizvod se pogrešno smatra jednakim \(\frac{1}{2}\), što je odgovor (B).

🏆 Dodatni izazov

Kolika je suma svih realnih rešenja jednačine \(|x + 2| + |x - 3| = x + 4\)?

Nagoveštaj

Kritične tačke su \(x = -2\) i \(x = 3\). Analizirajte tri intervala: \(x < -2\), \(-2 \leq x < 3\) i \(x \geq 3\). U središnjem intervalu leva strana je konstantna i jednaka rastojanju između tačaka \(-2\) i \(3\), tj. iznosi \(5\). Obratite pažnju i na desnu stranu — za koje vrednosti \(x\) je ona veća od ili jednaka \(5\)?
