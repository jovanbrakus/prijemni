Ostatak deljenja polinoma

Određivanje ostatka pri deljenju polinoma petog stepena polinomom drugog stepena pomoću teoreme o ostatku

📝 Postavka zadatka

Dati su polinomi \(P(x) = x^5 - 3x^4 + 2x^2 + x + 7\) i \(Q(x) = x^2 - x - 2\). Ako je \(R(x) = ax + b\) ostatak deljenja polinoma \(P(x)\) sa polinomom \(Q(x)\), tada je \(2a + b\) jednako:

(A)
\(11\)
(B)
\(5\)
(C)
\(1\)
(D)
\(9\)
(E)
\(7\)
🎯 Plan rešavanja

Faktorizovaćemo polinom \(Q(x)\) da nađemo njegove nule, a zatim ćemo iskoristiti činjenicu da u nulama delioca važi \(P(x_0) = R(x_0)\). Ubacivanjem \(x = 2\) direktno dobijamo traženu vrednost \(2a + b\).

📚 Podsetnik iz teorije
📐 Ključne formule i teoreme
  • Deljenje polinoma s ostatkom: Za polinome \(P(x)\) i \(Q(x)\) (\(\deg Q \geq 1\)) postoje jedinstveni polinomi \(S(x)\) i \(R(x)\) takvi da \(P(x) = Q(x) \cdot S(x) + R(x)\), gde je \(\deg R < \deg Q\).
  • Teorema o ostatku: Ako je \(Q(x_0) = 0\), tada iz \(P(x_0) = Q(x_0)\cdot S(x_0) + R(x_0)\) sledi \(P(x_0) = R(x_0)\).
  • Faktorizacija kvadratnog trinoma: \(x^2 - x - 2 = (x - 2)(x + 1)\), jer su nule \(x = 2\) i \(x = -1\).
  • Hornerova šema: Brz način izračunavanja vrednosti polinoma u datoj tački zamenom koeficijenata.
🎨 Grafik polinoma i ostatak

Grafički prikaz polinoma \(P(x)\) i ostatka \(R(x) = -x + 3\) sa označenim nulama delioca

\(P(x)\)
\(R(x) = -x + 3\)
Nule delioca: \(x = -1\), \(x = 2\)
🛠 Rešenje korak po korak
1

Faktorizujemo delilac \(Q(x)\)

Polinom \(Q(x) = x^2 - x - 2\) je kvadratni trinom. Tražimo njegove nule pomoću formule ili rastavljanja:

\[x^2 - x - 2 = (x - 2)(x + 1)\]

Nule delioca su \(x_1 = 2\) i \(x_2 = -1\).

2

Postavljamo jednakost deljenja s ostatkom

Pošto je stepen delioca \(Q(x)\) jednak 2, ostatak \(R(x)\) ima stepen najviše 1, pa je oblika \(R(x) = ax + b\). Važi:

\[P(x) = Q(x) \cdot S(x) + R(x)\]

Za svaku nulu delioca \(x_0\) važi \(Q(x_0) = 0\), pa se gornja jednakost svodi na:

\[P(x_0) = R(x_0)\]
3

Ubacujemo \(x = 2\) u \(P(x)\)

Izračunavamo vrednost polinoma \(P(x)\) u tački \(x = 2\):

\[P(2) = 2^5 - 3 \cdot 2^4 + 2 \cdot 2^2 + 2 + 7\]
\[= 32 - 48 + 8 + 2 + 7\]
\[= \boldsymbol{1}\]
4

Izražavamo \(R(2)\) i odmah dobijamo odgovor

Pošto je \(R(x) = ax + b\), za \(x = 2\) imamo:

\[R(2) = 2a + b\]

Iz uslova \(P(2) = R(2)\) sledi:

\[2a + b = P(2) = \boldsymbol{1}\]

Tražena vrednost \(2a + b = 1\) je direktno dobijena! Primetimo da nam je dovoljno bilo samo jedno uvrštavanje.

✅ Proveri sebe: Zašto nam nije bilo potrebno ubacivanje \(x = -1\)?

Zato što je u zadatku tražena konkretno vrednost \(2a + b\), a to je upravo \(R(2) = P(2)\). Ubacivanje \(x = -1\) bi bilo potrebno da tražimo \(a\) i \(b\) pojedinačno.

5

Potpuno određivanje ostatka (provera)

Radi provere, odredimo i \(a\) i \(b\) pojedinačno. Ubacujemo drugu nulu \(x = -1\) u \(P(x)\):

\[P(-1) = (-1)^5 - 3 \cdot (-1)^4 + 2 \cdot (-1)^2 + (-1) + 7\]
\[= -1 - 3 + 2 - 1 + 7 = 4\]

Iz \(R(-1) = -a + b = 4\) dobijamo drugi uslov. Sada imamo sistem:

\[\begin{cases} 2a + b = 1 \\ -a + b = 4 \end{cases}\]

Oduzimanjem druge od prve jednačine:

\[3a = -3 \implies a = -1\]

Zamenom u drugu jednačinu:

\[-(-1) + b = 4 \implies 1 + b = 4 \implies b = 3\]

Dakle, ostatak je \(R(x) = -x + 3\).

✅ Proveri sebe: Koliko je \(2a + b\) za \(a = -1\) i \(b = 3\)?

\(2 \cdot (-1) + 3 = -2 + 3 = \) \(1\). Potvrđeno!

6

Verifikacija ostatka

Proverimo još jednom: ako je \(R(x) = -x + 3\), tada:

\[R(2) = -2 + 3 = 1 = P(2) \quad \checkmark\]
\[R(-1) = -(-1) + 3 = 1 + 3 = 4 = P(-1) \quad \checkmark\]

Obe vrednosti se poklapaju, što potvrđuje da smo ispravno odredili ostatak.

💡 Ključni uvid

Kada zadatak traži tačno određenu kombinaciju koeficijenata ostatka (kao \(2a + b\)), nije uvek potrebno rešavati ceo sistem jednačina. Dovoljno je pronaći nulu delioca koja daje upravo traženi izraz pri uvrštavanju u ostatak:

\[R(x_0) = P(x_0) \quad \text{za svaku nulu } x_0 \text{ delioca } Q(x)\]

U ovom slučaju, \(x_0 = 2\) daje \(R(2) = 2a + b\), što je upravo tražena vrednost.

🏆 Konačan odgovor
\(2a + b = 1\)
(A) \(11\) (B) \(5\) (C) \(1\) (D) \(9\) (E) \(7\)

Provera: Direktnim uvrštavanjem \(x = 2\) u \(P(x)\) dobijamo \(32 - 48 + 8 + 2 + 7 = 1\). Pošto je \(R(2) = 2a + b\), odmah sledi \(2a + b = 1\). Takođe, potpuno određivanje ostatka daje \(R(x) = -x + 3\), odakle je \(2 \cdot (-1) + 3 = 1\). ✅

⚠️ Česte greške
Računska greška pri stepenima negativnih brojeva
Pri izračunavanju \(P(-1)\), učenici često pogreše sa predznacima. Zapamtite: \((-1)^5 = -1\) (neparan stepen), a \((-1)^4 = 1\) (paran stepen). Svaki sabirak treba pažljivo izračunati.
Pokušaj dugog deljenja umesto korišćenja nula delioca
Deljenje polinoma petog stepena polinomom drugog stepena je dugačak postupak sa velikim rizikom od greške. Mnogo elegantnije je faktorisati delilac i iskoristiti teoremu o ostatku.
🌟 Dodatni izazov

Neka je \(T(x)\) ostatak deljenja polinoma \(P(x) = x^5 - 3x^4 + 2x^2 + x + 7\) sa polinomom \((x-1)(x+2)\). Koliko je \(T(1) + T(-2)\)?

💡 Hint

Primenite isti pristup: izračunajte \(P(1)\) i \(P(-2)\). Pošto je \(T(x)\) ostatak stepena najviše 1, važi \(T(1) = P(1)\) i \(T(-2) = P(-2)\). Zbir tih vrednosti je traženi odgovor.
