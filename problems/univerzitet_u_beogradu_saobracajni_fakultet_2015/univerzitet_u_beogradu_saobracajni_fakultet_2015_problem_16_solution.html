Iracionalna nejednakost

Određivanje broja celih rešenja nejednačine sa kvadratnim korenom — analiza po slučajevima

📝 Tekst zadatka
Celih brojeva \(x\) za koje važi nejednakost \(1 - x < \sqrt{3 - x}\) ima:
(A)
\(3\)
(B)
\(7\)
(C)
\(4\)
(D)
\(2\)
(E)
\(5\)
🎯 Plan rešavanja

Prvo ćemo odrediti domen nejednakosti (izraz pod korenom mora biti nenegativan). Zatim ćemo razmotriti dva slučaja: kada je leva strana negativna (nejednakost automatski važi), i kada je leva strana nenegativna (kvadriramo obe strane). Na kraju prebrojavamo cele brojeve koji zadovoljavaju uslove.

📚 Podsetnik iz teorije
Iracionalne nejednakosti — pravila kvadriranja

Za nejednakost oblika \(f(x) < \sqrt{g(x)}\), rešavamo je u dva slučaja:

  • Slučaj 1: Ako je \(f(x) < 0\), nejednakost automatski važi jer je \(\sqrt{g(x)} \geq 0\). Uslov: \(g(x) \geq 0\) i \(f(x) < 0\).
  • Slučaj 2: Ako je \(f(x) \geq 0\), obe strane su nenegativne i možemo kvadrirati bez promene smera nejednakosti: \[ f(x)^2 < g(x) \]

Konačno rešenje je unija rešenja iz oba slučaja.

Kvadratna nejednakost — metod faktorizacije

Nejednakost \(ax^2 + bx + c < 0\) (sa \(a > 0\)) važi za \(x\) između nula polinoma:

\[ (x - x_1)(x - x_2) < 0 \implies x_1 < x < x_2 \quad (\text{za } x_1 < x_2) \]
🎨 Vizuelni prikaz — grafici funkcija

Grafik prikazuje \(y = 1 - x\) (plava linija) i \(y = \sqrt{3 - x}\) (zelena kriva). Rešenja su tamo gde je plava ispod zelene.

🛠 Rešenje korak po korak
1 Odredimo domen

Da bi koren \(\sqrt{3 - x}\) bio definisan, izraz pod korenom mora biti nenegativan:

\[ 3 - x \geq 0 \implies x \leq 3 \]

Dakle, domen nejednakosti je \(x \leq 3\), odnosno \(x \in (-\infty, 3]\).

2 Slučaj 1: Leva strana je negativna (\(1 - x < 0\))

Kada je \(1 - x < 0\), tj. \(x > 1\), leva strana nejednakosti je negativna.

Desna strana \(\sqrt{3 - x}\) je uvek nenegativna (jer je koren), pa je nejednakost automatski zadovoljena za svako \(x > 1\) iz domena.

\[ x > 1 \quad \text{i} \quad x \leq 3 \implies x \in (1, 3] \]

Celi brojevi iz ovog intervala: \(x = 2\) i \(x = 3\).

Proveri sebe: Zašto je nejednakost automatski tačna kada je \(x > 1\)?

Kada je \(x > 1\), leva strana \(1 - x\) je negativan broj. Desna strana \(\sqrt{3 - x}\) je koren, koji je uvek \(\geq 0\). Negativan broj je uvek manji od nenegativnog, pa nejednakost važi bez dodatnog računanja.

3 Slučaj 2: Leva strana je nenegativna (\(1 - x \geq 0\))

Kada je \(1 - x \geq 0\), tj. \(x \leq 1\), obe strane nejednakosti su nenegativne. U tom slučaju smemo da kvadriramo obe strane bez promene smera nejednakosti:

\[ (1 - x)^2 < 3 - x \]
4 Razvijemo i sredimo kvadratnu nejednakost

Razvijemo levu stranu:

\[ 1 - 2x + x^2 < 3 - x \]

Prebacimo sve na levu stranu:

\[ x^2 - 2x + x + 1 - 3 < 0 \] \[ x^2 - x - 2 < 0 \]
5 Faktorizujemo i rešimo kvadratnu nejednakost

Faktorizujemo kvadratni trinom \(x^2 - x - 2\). Tražimo dva broja čiji je zbir \(-1\) a proizvod \(-2\). To su \(-2\) i \(1\):

\[ x^2 - x - 2 = (x - 2)(x + 1) < 0 \]

Proizvod dva faktora je negativan kada je jedan pozitivan a drugi negativan. Pošto su nule \(x = -1\) i \(x = 2\):

\[ (x - 2)(x + 1) < 0 \implies -1 < x < 2 \]
6 Presečemo sa uslovom \(x \leq 1\)

Iz Slučaja 2 imamo uslov \(x \leq 1\), a iz nejednakosti \(-1 < x < 2\). Presek ova dva uslova je:

\[ -1 < x < 2 \quad \cap \quad x \leq 1 \implies -1 < x \leq 1 \]

Celi brojevi iz ovog intervala: \(x = 0\) i \(x = 1\).

Proveri sebe: Zašto \(x = -1\) nije rešenje?

Za \(x = -1\): leva strana je \(1 - (-1) = 2\), a desna strana je \(\sqrt{3 - (-1)} = \sqrt{4} = 2\). Nejednakost glasi \(2 < 2\), što nije tačno (potrebna je stroga nejednakost). Zato \(x = -1\) nije rešenje.

7 Unija rešenja — prebrojavanje celih brojeva

Konačno rešenje je unija oba slučaja:

\[ x \in (-1, 3] \]

Celi brojevi koji pripadaju intervalu \((-1, 3]\) su:

\[ x \in \{0, 1, 2, 3\} \]

Ima ih ukupno 4.

Proveri sebe: Verifikuj sva četiri rešenja

\(x = 0\): \(1 - 0 = 1 < \sqrt{3} \approx 1{,}73\) ✔
\(x = 1\): \(1 - 1 = 0 < \sqrt{2} \approx 1{,}41\) ✔
\(x = 2\): \(1 - 2 = -1 < \sqrt{1} = 1\) ✔
\(x = 3\): \(1 - 3 = -2 < \sqrt{0} = 0\) ✔
Sva četiri broja zadovoljavaju nejednakost.

Ključni uvid

Kod iracionalnih nejednakosti oblika \(f(x) < \sqrt{g(x)}\), ključno je razmotriti znak leve strane. Kada je leva strana negativna, nejednakost automatski važi jer je koren uvek nenegativan. Tek kada je leva strana nenegativna, potrebno je kvadrirati. Ovaj razdvojeni pristup po slučajevima sprečava greške i čini rešenje preglednim.

Konačni odgovor
Celih rešenja ima \(4\): \(\{0, 1, 2, 3\}\)
(A) \(3\) (B) \(7\) (C) \(4\) (D) \(2\) (E) \(5\)

Verifikacija: Za \(x \in \{0, 1, 2, 3\}\) nejednakost \(1 - x < \sqrt{3 - x}\) važi. Za \(x = -1\) ne važi (jednakost, ne stroga nejednakost), a za \(x = 4\) koren nije definisan.

⚠️ Česte greške
Zaboravljanje da je koren uvek nenegativan

Mnogi učenici odmah kvadriraju obe strane bez provere znaka leve strane. Ako je leva strana negativna, kvadriranje može dovesti do gubitka rešenja. Uvek prvo razmotriti slučaj \(f(x) < 0\).

Uključivanje \(x = -1\) u skup rešenja

Za \(x = -1\) važi \(1 - (-1) = 2\) i \(\sqrt{3 - (-1)} = 2\), pa je \(2 < 2\) netačno. Nejednakost je stroga (\( < \), ne \( \leq \)), pa granična tačka \(x = -1\) nije rešenje.

🏆 Dodatni izazov

Koliko celih brojeva \(x\) zadovoljava nejednakost \(1 - x \leq \sqrt{3 - x}\) (sa \(\leq\) umesto \(<\))?

Nagoveštaj

Jedina razlika je granična tačka \(x = -1\). Za nju važi \(2 \leq 2\), što je tačno. Dakle, sa neoštrom nejednakošću dobijamo 5 celih rešenja: \(\{-1, 0, 1, 2, 3\}\).
