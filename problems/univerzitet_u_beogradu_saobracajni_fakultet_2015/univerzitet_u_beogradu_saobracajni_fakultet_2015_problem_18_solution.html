🎯 Plan rešavanja

Razvićemo levu stranu jednačine koristeći formulu za kvadrat zbira, zatim ćemo uprostiti jednačinu i faktorizovati je. Iz faktorizovanog oblika odredićemo sve slučajeve koji daju rešenja, a zatim ćemo prebrojati sva rešenja na intervalu \([0, 2\pi]\).

📚 Podsetnik iz teorije
Ključne formule i identiteti
  • Pitagorin trigonometrijski identitet: \(\sin^2 x + \cos^2 x = 1\)
  • Dupli ugao (sinus): \(\sin 2x = 2\sin x \cos x\)
  • Kvadrat zbira: \((a + b)^2 = a^2 + 2ab + b^2\)
  • Nule sinusa: \(\sin x = 0 \implies x = k\pi,\; k \in \mathbb{Z}\)
  • Nule kosinusa: \(\cos x = 0 \implies x = \frac{\pi}{2} + k\pi,\; k \in \mathbb{Z}\)
  • Specijalna vrednost: \(\cos x = \frac{\sqrt{2}}{2} \implies x = \frac{\pi}{4} + 2k\pi\) ili \(x = \frac{7\pi}{4} + 2k\pi,\; k \in \mathbb{Z}\)
🎨 Vizuelni prikaz

Grafički prikaz leve (plava) i desne (zelena) strane jednačine — preseci su rešenja

🛠 Rešenje korak po korak
1 Razvijanje leve strane

Primenjujemo formulu za kvadrat zbira na levu stranu jednačine:

\[ (\sin x + \cos x)^2 = \sin^2 x + 2\sin x \cos x + \cos^2 x \]

Koristeći Pitagorin identitet \(\sin^2 x + \cos^2 x = 1\):

\[ (\sin x + \cos x)^2 = 1 + 2\sin x \cos x \]

Prepoznajemo da je \(2\sin x \cos x = \sin 2x\), pa je leva strana:

\[ (\sin x + \cos x)^2 = 1 + \sin 2x \]
2 Postavljanje uprošćene jednačine

Sada jednačina postaje:

\[ 1 + \sin 2x = 2\sqrt{2} \cdot \sin x \cdot \cos^2 x + 1 \]

Oduzimamo \(1\) sa obe strane:

\[ \sin 2x = 2\sqrt{2} \cdot \sin x \cdot \cos^2 x \]
Proveri sebe: Koliko je \(\sin 2x\) izraženo preko \(\sin x\) i \(\cos x\)?

\(\sin 2x = 2\sin x \cos x\). Ovo je formula za dupli ugao sinusa.

3 Zamena formule za dupli ugao

Zamenjujemo \(\sin 2x = 2\sin x \cos x\) na levoj strani:

\[ 2\sin x \cos x = 2\sqrt{2} \cdot \sin x \cdot \cos^2 x \]
4 Prebacivanje na jednu stranu i faktorizacija

Prebacujemo desnu stranu na levu:

\[ 2\sin x \cos x - 2\sqrt{2} \cdot \sin x \cdot \cos^2 x = 0 \]

Izvlačimo zajednički faktor \(2\sin x \cos x\):

\[ 2\sin x \cos x \left(1 - \sqrt{2}\cos x\right) = 0 \]

Proizvod je jednak nuli ako je bar jedan faktor jednak nuli.

5 Slučaj 1: \(\sin x = 0\)

Rešavamo \(\sin x = 0\) na intervalu \([0, 2\pi]\):

\[ \sin x = 0 \implies x = 0, \;\; x = \pi, \;\; x = 2\pi \]

To su 3 rešenja iz ovog slučaja.

6 Slučaj 2: \(\cos x = 0\)

Rešavamo \(\cos x = 0\) na intervalu \([0, 2\pi]\):

\[ \cos x = 0 \implies x = \frac{\pi}{2}, \;\; x = \frac{3\pi}{2} \]

To su 2 rešenja iz ovog slučaja.

Proveri sebe: Da li se rešenja iz slučaja 1 i 2 preklapaju?

Ne! U tačkama gde je \(\sin x = 0\) (to su \(0, \pi, 2\pi\)), kosinus je \(\pm 1 \neq 0\). Obrnuto, gde je \(\cos x = 0\) (to su \(\frac{\pi}{2}, \frac{3\pi}{2}\)), sinus je \(\pm 1 \neq 0\). Dakle, nema preklapanja.

7 Slučaj 3: \(1 - \sqrt{2}\cos x = 0\)

Rešavamo treći faktor:

\[ 1 - \sqrt{2}\cos x = 0 \implies \cos x = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2} \]

Na intervalu \([0, 2\pi]\), kosinus ima vrednost \(\frac{\sqrt{2}}{2}\) u dvema tačkama:

\[ x = \frac{\pi}{4}, \;\; x = \frac{7\pi}{4} \]

To su 2 rešenja iz ovog slučaja.

8 Prebrojavanje svih rešenja

Proverimo da nema preklapanja među slučajevima. Za slučaj 3 (\(x = \frac{\pi}{4}\) i \(x = \frac{7\pi}{4}\)), ni sinus ni kosinus nisu nula, pa se ova rešenja ne pojavljuju ni u slučaju 1 ni u slučaju 2.

Spisak svih 7 rešenja na \([0, 2\pi]\):

\[ x \in \left\{ 0,\;\; \frac{\pi}{4},\;\; \frac{\pi}{2},\;\; \pi,\;\; \frac{3\pi}{2},\;\; \frac{7\pi}{4},\;\; 2\pi \right\} \]

Ukupan broj rešenja: \(3 + 2 + 2 = \mathbf{7}\).

9 Verifikacija probom

Proverimo nekoliko rešenja uvrštavanjem u polaznu jednačinu \((\sin x + \cos x)^2 = 2\sqrt{2}\sin x\cos^2 x + 1\):

Za \(x = 0\): Leva strana: \((0 + 1)^2 = 1\). Desna strana: \(2\sqrt{2} \cdot 0 \cdot 1 + 1 = 1\). Jednako!

Za \(x = \frac{\pi}{4}\): Leva: \(\left(\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}\right)^2 = (\sqrt{2})^2 = 2\). Desna: \(2\sqrt{2} \cdot \frac{\sqrt{2}}{2} \cdot \frac{1}{2} + 1 = 1 + 1 = 2\). Jednako!

Za \(x = \frac{7\pi}{4}\): Leva: \(\left(-\frac{\sqrt{2}}{2} + \frac{\sqrt{2}}{2}\right)^2 = 0\). Desna: \(2\sqrt{2} \cdot \left(-\frac{\sqrt{2}}{2}\right) \cdot \frac{1}{2} + 1 = -1 + 1 = 0\). Jednako!

Proveri sebe: Koliko rešenja daje slučaj \(\sin x = 0\) na \([0, 2\pi]\)?

Tri rešenja: \(x = 0\), \(x = \pi\), \(x = 2\pi\). Pažnja: \(2\pi\) pripada zatvorenom intervalu \([0, 2\pi]\), pa je i to rešenje.

Ključni uvid

Ključ ovog zadatka je u faktorizaciji: nakon razvijanja i uprošćavanja, jednačina poprima oblik \(2\sin x \cos x(1 - \sqrt{2}\cos x) = 0\). Proizvod tri faktora je nula ako je bar jedan od njih nula. Svaki faktor daje zasebnu grupu rešenja, a pažljivo prebrojavanje (bez preskakanja \(x = 0\) i \(x = 2\pi\)) daje ukupno 7 rešenja.

Konačni odgovor
Broj rešenja: \(7\)
(A) \(5\) (B) \(3\) (C) \(4\) (D) \(7\) (E) \(8\)

Verifikacija: Svih 7 rešenja — \(0, \frac{\pi}{4}, \frac{\pi}{2}, \pi, \frac{3\pi}{2}, \frac{7\pi}{4}, 2\pi\) — potvrđeno uvrštavanjem u polaznu jednačinu. Tri slučaja (\(\sin x = 0\), \(\cos x = 0\), \(\cos x = \frac{\sqrt{2}}{2}\)) nemaju zajednička rešenja, pa je zbir \(3 + 2 + 2 = 7\) korektan. ✔

⚠️ Česte greške
Zaboravljanje rešenja \(x = 0\) i \(x = 2\pi\)

Interval je zatvoren: \([0, 2\pi]\). I \(x = 0\) i \(x = 2\pi\) pripadaju intervalu, a oba zadovoljavaju \(\sin x = 0\). Ako se izostave ova dva rešenja, dobija se 5 umesto 7.

Deljenje jednačine sa \(\sin x\) ili \(\cos x\)

Ako se obe strane jednačine podele sa \(\sin x\) ili \(\cos x\), gube se rešenja za koja su ti izrazi jednaki nuli. Uvek je bezbednije prebaciti sve na jednu stranu i faktorizovati.

🏆 Dodatni izazov

Koliki je broj rešenja jednačine \((\sin x + \cos x)^2 = 2\sqrt{2} \cdot \sin x \cdot \cos^2 x + 1\) na intervalu \((0, 2\pi)\) (otvorenom intervalu)?

Nagoveštaj

Na otvorenom intervalu \((0, 2\pi)\) ne pripadaju krajnje tačke \(x = 0\) i \(x = 2\pi\). Od 7 rešenja, izbacujemo ta dva, pa ostaje \(7 - 2 = 5\) rešenja. To je upravo odgovor (A) iz ponuđenih opcija za polazni zadatak, ali za drugačiji interval!
