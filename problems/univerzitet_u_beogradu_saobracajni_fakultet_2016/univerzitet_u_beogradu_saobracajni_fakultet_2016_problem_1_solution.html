Uprošćavanje algebarskog izraza sa kubovima

Faktorizacija zbira kubova i algebarsko skraćivanje razlomka sa decimalnim brojevima

📝 Tekst zadatka
Ako je \(a = 2{,}3584\) i \(b = 1{,}6416\), onda izraz \[\frac{a^3 + 8b^3}{(a - b)^2 + 3b^2} - b\] ima vrednost:
(A)
\(2\)
(B)
\(16\)
(C)
\(42{,}56\)
(D)
\(4\)
(E)
\(0{,}7168\)
🎯 Plan rešavanja

Umesto da računamo sa komplikovanim decimalnim brojevima, primenićemo algebarsku faktorizaciju. Prepoznaćemo da je \(8b^3 = (2b)^3\), pa brojilac predstavlja zbir kubova. Imenilac ćemo razviti i uporediti sa delom brojioca. Posle skraćivanja, izraz će se svesti na jednostavan zbir \(a + b\).

📚 Podsetnik iz teorije
Zbir kubova
\[ x^3 + y^3 = (x + y)(x^2 - xy + y^2) \]

Ovo je jedna od ključnih formula za faktorizaciju. Zbir kubova se rastavlja na proizvod binoma \((x + y)\) i trinoma \((x^2 - xy + y^2)\).

Kvadrat binoma
\[ (a - b)^2 = a^2 - 2ab + b^2 \]

Kvadrat razlike dva broja jednak je kvadratu prvog, minus dvostruki proizvod, plus kvadrat drugog.

🎨 Vizuelni prikaz transformacije izraza

Prikaz korak-po-korak algebarskog uprošćavanja izraza

🛠 Rešenje korak po korak
1 Prepoznajmo strukturu brojioca

Primetimo da je \(8b^3 = (2b)^3\). Dakle, brojilac \(a^3 + 8b^3\) je zapravo zbir kubova:

\[ a^3 + 8b^3 = a^3 + (2b)^3 \]

Ovo je ključna opservacija koja nam omogućava da primenimo formulu za zbir kubova.

2 Primenimo formulu za zbir kubova

Koristeći formulu \(x^3 + y^3 = (x + y)(x^2 - xy + y^2)\) sa \(x = a\) i \(y = 2b\):

\[ a^3 + (2b)^3 = (a + 2b)\bigl(a^2 - a \cdot 2b + (2b)^2\bigr) \]

Sredimo izraz u zagradi:

\[ a^3 + (2b)^3 = (a + 2b)(a^2 - 2ab + 4b^2) \]
Proveri sebe: Koliko je \((2b)^2\)?

\((2b)^2 = 4b^2\). Kvadriramo i koeficijent i promenljivu: \(2^2 \cdot b^2 = 4b^2\).

3 Razvijmo imenilac

Imenilac je \((a - b)^2 + 3b^2\). Razvijmo kvadrat binoma:

\[ (a - b)^2 = a^2 - 2ab + b^2 \]

Dodajmo \(3b^2\):

\[ (a - b)^2 + 3b^2 = a^2 - 2ab + b^2 + 3b^2 = a^2 - 2ab + 4b^2 \]
4 Uočimo poklapanje

Uporedimo rezultate iz koraka 2 i 3. Imenilac je upravo jednak drugom faktoru brojioca:

\[ \underbrace{a^2 - 2ab + 4b^2}_{\text{imenilac}} = \underbrace{a^2 - 2ab + 4b^2}_{\text{drugi faktor brojioca}} \]

Ovo znači da se razlomak može skratiti!

Proveri sebe: Šta preostaje posle skraćivanja?

Posle skraćivanja \((a^2 - 2ab + 4b^2)\) u brojiocu i imeniocu, preostaje samo prvi faktor brojioca: \(a + 2b\).

5 Skratimo razlomak

Zamenom faktorisanih oblika u originalni izraz:

\[ \frac{a^3 + 8b^3}{(a - b)^2 + 3b^2} = \frac{(a + 2b)\cancel{(a^2 - 2ab + 4b^2)}}{\cancel{a^2 - 2ab + 4b^2}} = a + 2b \]
6 Oduzimamo \(b\)

Ceo izraz glasi:

\[ \frac{a^3 + 8b^3}{(a - b)^2 + 3b^2} - b = (a + 2b) - b = a + b \]

Izraz se svodi na najjednostavniji mogući oblik: \(a + b\).

7 Izračunajmo brojčanu vrednost

Sada zamenjujemo konkretne vrednosti \(a = 2{,}3584\) i \(b = 1{,}6416\):

\[ a + b = 2{,}3584 + 1{,}6416 = 4{,}0000 = 4 \]

Lepo se vidi da su decimale komplementarne: 0,3584 + 0,6416 = 1,0000, pa zajedno sa celim delovima 2 + 1 = 3 dobijamo ukupno 4.

Proveri sebe: Da li je 4 među ponuđenim odgovorima?

Da! Vrednost \(4\) odgovara opciji (D). Rešenje je tačno.

Ključni uvid

Zadatak se čini komplikovanim zbog „ružnih" decimalnih brojeva, ali upravo to je zamka. Suština je da se ceo izraz algebarski uprosti pre zamene brojeva. Kada primenimo formulu za zbir kubova na brojilac i razvijemo imenilac, dolazimo do identičnog trinoma \(a^2 - 2ab + 4b^2\) koji se skraćuje. Konačni rezultat je jednostavno \(a + b\), a decimale su namerno izabrane tako da im je zbir okrugao broj.

Konačni odgovor
\(\dfrac{a^3 + 8b^3}{(a-b)^2 + 3b^2} - b = a + b = 4\)
(A) \(2\) (B) \(16\) (C) \(42{,}56\) (D) \(4\) (E) \(0{,}7168\)

Verifikacija: Proverimo direktnim računom. \(a + 2b = 2{,}3584 + 3{,}2832 = 5{,}6416\). \(a - b = 0{,}7168\). Razlomak daje \(a + 2b = 5{,}6416\). Oduzimanjem \(b = 1{,}6416\) dobijamo \(5{,}6416 - 1{,}6416 = 4\). Primetimo i da je \(0{,}7168 = a - b\), što je ponuđeni odgovor (E) — ovo je distraktror koji testira da li ste se zaustavili na \(a - b\) umesto da nastavite sa punom faktorizacijom. ✔

⚠️ Česte greške
Pokušaj direktnog izračunavanja

Najveća zamka je pokušaj da se stepenuju i množe decimalni brojevi direktno. Ovo vodi do velikih numeričkih grešaka i ogromnog gubitka vremena. Uvek prvo pokušajte algebarsko uprošćavanje!

Zaboravljena supstitucija \(8b^3 = (2b)^3\)

Ako ne prepoznate da je \(8b^3\) kub od \(2b\), ne možete primeniti formulu za zbir kubova. Ključ je primetiti da je \(8 = 2^3\).

Zamena \(a - b\) kao konačnog odgovora

Vrednost \(a - b = 0{,}7168\) je ponuđena kao odgovor (E). To je distraktror za studente koji pogrešno izračunaju ili se zaustave pre finalnog koraka.

🏆 Dodatni izazov

Uprosti izraz \(\dfrac{a^3 - 27b^3}{a^2 - 6ab + 9b^2} + \dfrac{3ab}{a - 3b}\) za \(a \neq 3b\).

Nagoveštaj

Primeni formulu za razliku kubova: \(a^3 - 27b^3 = a^3 - (3b)^3 = (a - 3b)(a^2 + 3ab + 9b^2)\). Imenilac prvog razlomka je \((a - 3b)^2\). Posle skraćivanja dobija se \(\frac{a^2 + 3ab + 9b^2}{a - 3b} + \frac{3ab}{a - 3b}\), što daje \(\frac{a^2 + 6ab + 9b^2}{a - 3b} = \frac{(a + 3b)^2}{a - 3b}\).
