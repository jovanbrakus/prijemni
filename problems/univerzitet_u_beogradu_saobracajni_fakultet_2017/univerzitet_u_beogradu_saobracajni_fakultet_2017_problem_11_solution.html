🎯 Plan rešavanja

Funkcija \(f\) je zadata pravilom \(f(x + 2017) = 3x + 2017\). Da bismo našli \(f(2016)\), potrebno je da argument funkcije \(x + 2017\) bude jednak \(2016\). To nam daje konkretnu vrednost \(x\) koju zamenjujemo u desnu stranu. Alternativno, možemo izvesti eksplicitnu formulu za \(f(t)\) smenom \(t = x + 2017\).

📚 Podsetnik iz teorije
Funkcija zadata implicitnim pravilom

Kada je funkcija zadata u obliku \(f(\text{izraz od } x) = \text{izraz od } x\), ključna tehnika je supstitucija: izjednačimo argument funkcije sa vrednošću za koju tražimo \(f\), i rešimo po \(x\).

\[ f(g(x)) = h(x) \quad \Rightarrow \quad f(a) = h(x_0) \text{ gde je } g(x_0) = a \]

Alternativno, možemo uvesti smenu \(t = g(x)\), izraziti \(x\) preko \(t\), i dobiti eksplicitnu formulu \(f(t)\).

Smena promenljive za nalaženje eksplicitne formule

Ako je \(f(x + a) = h(x)\), uvedemo smenu \(t = x + a\), pa je \(x = t - a\), i dobijamo:

\[ f(t) = h(t - a) \]

Ovo nam daje eksplicitnu formulu funkcije za svako \(t\), što omogućava izračunavanje \(f\) u bilo kojoj tački.

1 Postavi uslov za argument funkcije

Dato nam je pravilo:

\[ f(x + 2017) = 3x + 2017 \]

Tražimo \(f(2016)\). Da bismo to izračunali, potrebno je da argument funkcije bude jednak \(2016\):

\[ x + 2017 = 2016 \]
2 Reši jednačinu po \(x\)

Rešavamo linearnu jednačinu:

\[ x = 2016 - 2017 = -1 \]

Dakle, za \(x = -1\) je argument funkcije \(f\) jednak \(2016\).

Proveri sebe: Da li važi \(x + 2017 = 2016\) za \(x = -1\)?

\(-1 + 2017 = 2016\). ✔ Tačno!

3 Zameni \(x = -1\) u desnu stranu

Sada zamenjujemo \(x = -1\) u izraz \(3x + 2017\):

\[ f(2016) = 3 \cdot (-1) + 2017 = -3 + 2017 = 2014 \]

Vrednost funkcije u tački \(2016\) je \(2014\).

4 Verifikacija: eksplicitna formula za \(f(t)\)

Kao dodatnu proveru, izvedimo opštu formulu za \(f\). Uvedimo smenu \(t = x + 2017\), odakle je \(x = t - 2017\):

\[ f(t) = 3(t - 2017) + 2017 = 3t - 6051 + 2017 = 3t - 4034 \]

Sada izračunajmo \(f(2016)\) pomoću ove formule:

\[ f(2016) = 3 \cdot 2016 - 4034 = 6048 - 4034 = 2014 \]
Proveri sebe: Da li se oba metoda poklapaju?

Direktna supstitucija: \(f(2016) = 3 \cdot (-1) + 2017 = 2014\).
Eksplicitna formula: \(f(2016) = 3 \cdot 2016 - 4034 = 2014\).
Oba metoda daju isti rezultat. ✔

Proveri sebe: Da li formula daje tačan rezultat za \(x = 0\)?

Po definiciji: \(f(0 + 2017) = f(2017) = 3 \cdot 0 + 2017 = 2017\).
Po formuli: \(f(2017) = 3 \cdot 2017 - 4034 = 6051 - 4034 = 2017\). ✔ Poklapaju se!

Ključni uvid

Kada je funkcija zadata pravilom oblika \(f(x + a) = h(x)\), za izračunavanje \(f(c)\) dovoljno je naći onu vrednost \(x_0\) za koju je \(x_0 + a = c\), tj. \(x_0 = c - a\), i zatim izračunati \(h(x_0)\). U ovom zadatku: \(x_0 = 2016 - 2017 = -1\), pa je \(f(2016) = 3 \cdot (-1) + 2017 = 2014\).

\[ x + 2017 = 2016 \;\Rightarrow\; x = -1 \;\Rightarrow\; f(2016) = 2014 \]

⚠️ Česte greške
Zamena \(x = 2016\) direktno u formulu

Najčešća greška je da se stavi \(x = 2016\) u \(3x + 2017\), što daje \(3 \cdot 2016 + 2017 = 8065\). To bi bio odgovor (D), ali je pogrešno jer \(x = 2016\) daje \(f(2016 + 2017) = f(4033)\), a ne \(f(2016)\).

Brkanje \(f(2016)\) i \(f(2017)\)

Za \(x = 0\) dobijamo \(f(2017) = 2017\), što je odgovor (C). Ovo je tačna vrednost, ali u pogrešnoj tački! Traženo je \(f(2016)\), ne \(f(2017)\).

Greška u aritmetici sa negativnim brojem

Kada se izračunava \(3 \cdot (-1) + 2017\), neki učenici dobiju \(3 + 2017 = 2020\) jer zaborave minus. Pravilno: \(3 \cdot (-1) = -3\), pa je \(-3 + 2017 = 2014\).

🏆 Dodatni izazov

Koristeći istu funkciju \(f(x + 2017) = 3x + 2017\), odredi za koje \(a\) važi \(f(a) = a\). Drugim rečima, nađi fiksnu tačku funkcije \(f\).

Nagoveštaj

Koristi eksplicitnu formulu \(f(t) = 3t - 4034\). Postavi uslov \(f(a) = a\), tj. \(3a - 4034 = a\). Reši po \(a\): \(2a = 4034\), pa \(a = 2017\). Proveri: \(f(2017) = 3 \cdot 2017 - 4034 = 6051 - 4034 = 2017\). ✔
