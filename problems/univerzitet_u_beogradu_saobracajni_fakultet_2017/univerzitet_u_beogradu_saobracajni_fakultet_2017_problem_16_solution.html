🎯 Plan rešavanja

Uvešćemo smenu \(t = \sin x - \cos x\), jer se iz \(t^2\) može izraziti proizvod \(\sin x \cos x\). Ovo će svesti jednačinu na algebarsku jednačinu po \(t\), čije rešenje daje prostiju trigonometrijsku jednačinu. Na kraju ćemo prebrojati rešenja na datom intervalu.

📚 Podsetnik iz teorije
Veza sin x - cos x i sin x cos x

Ako stavimo \(t = \sin x - \cos x\), tada:

\[ t^2 = \sin^2 x - 2\sin x\cos x + \cos^2 x = 1 - 2\sin x\cos x \]

Odavde: \(\sin x\cos x = \dfrac{1 - t^2}{2}\). Ova identična veza omogućava eliminaciju proizvoda \(\sin x\cos x\).

Svođenje a sin x + b cos x na jedan sinus

Za \(a\sin x + b\cos x\) važi:

\[ a\sin x + b\cos x = \sqrt{a^2 + b^2}\;\sin(x + \varphi), \quad \tan\varphi = \frac{b}{a} \]

Posebno, \(\sin x - \cos x = \sqrt{2}\;\sin\!\left(x - \dfrac{\pi}{4}\right)\).

Opšte rešenje jednačine sin u = sin α

Jednačina \(\sin u = \sin\alpha\) ima opšte rešenje:

\[ u = \alpha + 2k\pi \quad \text{ili} \quad u = \pi - \alpha + 2k\pi, \quad k \in \mathbb{Z} \]

Rešenja (preseci)
🛠 Rešenje korak po korak
1 Uvedi smenu t = sin x - cos x

Definišemo novu promenljivu:

\[ t = \sin x - \cos x \]

Kvadriramo obe strane:

\[ t^2 = \sin^2 x - 2\sin x\cos x + \cos^2 x = 1 - 2\sin x\cos x \]

Odavde izražavamo proizvod:

\[ \sin x\cos x = \frac{1 - t^2}{2} \]
2 Zameni u polaznu jednačinu

Polazna jednačina \(\sin x - \cos x + \sin x\cos x = 1\) postaje:

\[ t + \frac{1 - t^2}{2} = 1 \]

Pomnožimo celu jednačinu sa 2:

\[ 2t + 1 - t^2 = 2 \]

Sredimo:

\[ -t^2 + 2t - 1 = 0 \quad \Rightarrow \quad t^2 - 2t + 1 = 0 \quad \Rightarrow \quad (t - 1)^2 = 0 \]

Dakle, jedino rešenje je:

\[ t = 1 \]
Proveri sebe: Zašto je supstitucija t = sin x - cos x korisna?

Zato što se \(t^2\) direktno izražava pomoću \(\sin x\cos x\), a u jednačini figurišu upravo \(\sin x - \cos x\) i \(\sin x\cos x\). Ova smena pretvara trigonometrijsku jednačinu u algebarsku (kvadratnu) po \(t\), koja se lako rešava.

3 Reši jednačinu sin x - cos x = 1

Iz \(t = 1\) sledi:

\[ \sin x - \cos x = 1 \]

Korišćenjem identiteta \(\sin x - \cos x = \sqrt{2}\;\sin\!\left(x - \dfrac{\pi}{4}\right)\), dobijamo:

\[ \sqrt{2}\;\sin\!\left(x - \frac{\pi}{4}\right) = 1 \]
\[ \sin\!\left(x - \frac{\pi}{4}\right) = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2} \]
4 Nađi opšta rešenja

Jednačina \(\sin\!\left(x - \dfrac{\pi}{4}\right) = \dfrac{\sqrt{2}}{2}\) ima rešenja:

\[ x - \frac{\pi}{4} = \frac{\pi}{4} + 2k\pi \quad \Rightarrow \quad x = \frac{\pi}{2} + 2k\pi \]
\[ x - \frac{\pi}{4} = \pi - \frac{\pi}{4} + 2k\pi = \frac{3\pi}{4} + 2k\pi \quad \Rightarrow \quad x = \pi + 2k\pi \]

gde je \(k \in \mathbb{Z}\).

Proveri sebe: Kako smo dobili sin(x - π/4)?

Koristimo formulu: \(\sin x - \cos x = \sqrt{2}\left(\frac{1}{\sqrt{2}}\sin x - \frac{1}{\sqrt{2}}\cos x\right) = \sqrt{2}\;\sin\!\left(x - \frac{\pi}{4}\right)\).
To je primena identiteta za linearne kombinacije sinusa i kosinusa: \(a\sin x + b\cos x = \sqrt{a^2+b^2}\;\sin(x + \varphi)\), gde je ovde \(a = 1\), \(b = -1\), pa \(\sqrt{a^2+b^2} = \sqrt{2}\) i \(\varphi = -\frac{\pi}{4}\).

5 Izaberi rešenja iz intervala [0, 5π/2]

Proveravamo koja rešenja pripadaju intervalu \(\left[0,\, \dfrac{5\pi}{2}\right]\):

\(k\) \(x\) Pripada \(\left[0, \frac{5\pi}{2}\right]\)?

Ukupan broj rešenja: 3 — to su \(x = \dfrac{\pi}{2}\), \(x = \pi\) i \(x = \dfrac{5\pi}{2}\).

6 Verifikacija svakog rešenja

Proverimo da svako od tri rešenja zaista zadovoljava polaznu jednačinu \(\sin x - \cos x + \sin x\cos x = 1\):

\(x = \dfrac{\pi}{2}\):
\(\sin\dfrac{\pi}{2} - \cos\dfrac{\pi}{2} + \sin\dfrac{\pi}{2}\cdot\cos\dfrac{\pi}{2} = 1 - 0 + 1\cdot 0 = 1\) ✔
\(x = \pi\):
\(\sin\pi - \cos\pi + \sin\pi\cdot\cos\pi = 0 - (-1) + 0\cdot(-1) = 0 + 1 + 0 = 1\) ✔
\(x = \dfrac{5\pi}{2}\):
\(\sin\dfrac{5\pi}{2} - \cos\dfrac{5\pi}{2} + \sin\dfrac{5\pi}{2}\cdot\cos\dfrac{5\pi}{2} = 1 - 0 + 1\cdot 0 = 1\) ✔
Proveri sebe: Zašto je x = 5π/2 isto što i x = π/2?

Jer je \(\dfrac{5\pi}{2} = \dfrac{\pi}{2} + 2\pi\), pa imaju iste vrednosti trigonometrijskih funkcija. Ali, to su različite tačke na brojevnoj pravoj, pa se broje kao dva različita rešenja. Zadatak traži broj rešenja na intervalu, a ne broj suštinski različitih uglova.

Ključni uvid

Smena \(t = \sin x - \cos x\) je standardna tehnika kada jednačina sadrži i razliku \(\sin x - \cos x\) i proizvod \(\sin x\cos x\). Ključno je uočiti da kvadrat smene daje \(t^2 = 1 - 2\sin x\cos x\), čime se proizvod izražava preko \(t\). U ovom zadatku, algebarska jednačina daje dvostruki koren \(t = 1\), pa se broj rešenja određuje samo iz jedne trigonometrijske jednačine \(\sin x - \cos x = 1\).

⚠️ Česte greške
Zaboravljanje rešenja x = 5π/2

Interval je \(\left[0, \frac{5\pi}{2}\right]\), što je zatvoren. Vrednost \(x = \frac{5\pi}{2}\) pripada intervalu i mora se uračunati. Ako se previdi ova granična tačka, dobija se pogrešan odgovor 2 umesto 3.

Uvođenje lažnih rešenja pri kvadriranju

Kada se koristi smena \(t = \sin x - \cos x\), moramo proveriti da svako rešenje dobijeno iz algebarske jednačine zaista zadovoljava polaznu jednačinu. Ovde nema lažnih rešenja jer je (t-1)² = 0 jedini koren i jednačina je dobro uslovljena, ali u sličnim zadacima provera je neophodna.

Brkanje 3π i 5π/2

Gornja granica intervala je \(\frac{5\pi}{2} \approx 7{,}85\), a ne \(3\pi \approx 9{,}42\). Vrednost \(x = 3\pi\) je van intervala, pa se ne računa. Pažljivo čitajte granice!

Nagovještaj

Koristi sličnu smenu: \(u = \sin x + \cos x\), pa \(u^2 = 1 + 2\sin x\cos x\). Jednačina postaje \(u + \frac{u^2 - 1}{2} = 1\), tj. \(u^2 + 2u - 3 = 0\), što daje \(u = 1\) ili \(u = -3\). Pošto je \(|u| \leq \sqrt{2}\), samo \(u = 1\) je moguće. Reši \(\sin x + \cos x = 1\) na \([0, 4\pi]\).
