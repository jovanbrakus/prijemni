Deljivost polinoma — određivanje koeficijenata

Polinomi visokog stepena — korišćenje nula delitelja za formiranje sistema jednačina

📝 Tekst zadatka
Ako je polinom \(x^{2018} + x^{2017} + ax + b\) deljiv polinomom \(x^2 - 1\), onda je \(a + 3b\) jednako:
(A)
\(0\)
(B)
\(2\)
(C)
\(-2\)
(D)
\(4\)
(E)
\(-4\)
🎯 Plan rešavanja

Polinom \(x^2 - 1\) se faktoriše kao \((x-1)(x+1)\). Ako je \(P(x)\) deljiv sa \(x^2 - 1\), onda su \(x = 1\) i \(x = -1\) nule polinoma \(P(x)\). Uvrstićemo obe vrednosti u \(P(x) = 0\), čime dobijamo sistem od dve jednačine sa dve nepoznate (\(a\) i \(b\)). Rešavanjem sistema nalazimo \(a\) i \(b\), pa računamo \(a + 3b\).

📚 Podsetnik iz teorije
Teorema o nulama polinoma i deljivost

Ključni alat u ovom zadatku je sledeća činjenica:

  • Teorema o ostatku: Ostatak pri deljenju polinoma \(P(x)\) sa \((x - c)\) jednak je \(P(c)\).
  • Posledica: Ako je \(P(x)\) deljiv sa \((x - c)\), onda je ostatak nula, tj. \(P(c) = 0\).
  • Proširenje: Ako je \(P(x)\) deljiv sa \((x - c_1)(x - c_2)\), onda važi \(P(c_1) = 0\) i \(P(c_2) = 0\).
  • Parni i neparni stepeni: \((-1)^{2018} = 1\) (paran stepen), \((-1)^{2017} = -1\) (neparan stepen).

🛠 Rešenje korak po korak
1 Faktoriši delilac

Polinom \(x^2 - 1\) je razlika kvadrata i faktoriše se kao:

\[ x^2 - 1 = (x - 1)(x + 1) \]

Ako je \(P(x) = x^{2018} + x^{2017} + ax + b\) deljiv sa \(x^2 - 1\), onda mora biti deljiv i sa \((x - 1)\) i sa \((x + 1)\). Po teoremi o ostatku, to znači:

\[ P(1) = 0 \quad \text{i} \quad P(-1) = 0 \]
2 Izračunaj P(1) = 0

Uvrstimo \(x = 1\) u polinom. Pošto je \(1^n = 1\) za svako \(n\):

\[ P(1) = 1^{2018} + 1^{2017} + a \cdot 1 + b = 1 + 1 + a + b = 2 + a + b \]

Iz uslova \(P(1) = 0\) sledi:

\[ 2 + a + b = 0 \quad \Longrightarrow \quad \boxed{a + b = -2} \quad \ldots (1) \]
Proveri sebe: Zašto je 1 na bilo koji stepen jednako 1?

Broj 1 pomnožen sam sa sobom koliko god puta uvek daje 1. Formalno: \(1^n = \underbrace{1 \cdot 1 \cdot \ldots \cdot 1}_{n} = 1\) za svako \(n \in \mathbb{N}\).

3 Izračunaj P(−1) = 0

Uvrstimo \(x = -1\). Ključno je odrediti stepene od \(-1\):

\[ (-1)^{2018} = 1 \quad \text{(paran stepen)} \qquad (-1)^{2017} = -1 \quad \text{(neparan stepen)} \]

Dakle:

\[ P(-1) = (-1)^{2018} + (-1)^{2017} + a \cdot (-1) + b = 1 + (-1) + (-a) + b = -a + b \]

Iz uslova \(P(-1) = 0\) sledi:

\[ -a + b = 0 \quad \Longrightarrow \quad \boxed{b = a} \quad \ldots (2) \]
Proveri sebe: Šta znači −a + b = 0?

Jednačina \(-a + b = 0\) se može napisati kao \(b = a\). To znači da su koeficijenti \(a\) i \(b\) jednaki. Ovo značajno pojednostavljuje dalji račun.

4 Reši sistem jednačina

Imamo sistem:

\[ \begin{cases} a + b = -2 & \ldots (1) \\ b = a & \ldots (2) \end{cases} \]

Uvrstimo \(b = a\) iz jednačine (2) u jednačinu (1):

\[ a + a = -2 \quad \Longrightarrow \quad 2a = -2 \quad \Longrightarrow \quad a = -1 \]

Iz \(b = a\) sledi:

\[ b = -1 \]
Proveri sebe: Verifikuj P(1) i P(−1) sa a = −1, b = −1

\(P(1) = 1 + 1 + (-1) + (-1) = 0\) ✔
\(P(-1) = 1 - 1 - (-1) + (-1) = 1 - 1 + 1 - 1 = 0\) ✔
Obe nule su zadovoljene!

5 Izračunaj a + 3b

Sada uvrstimo dobijene vrednosti \(a = -1\) i \(b = -1\):

\[ a + 3b = (-1) + 3 \cdot (-1) = -1 - 3 = \boxed{-4} \]
6 Verifikacija faktorizacijom

Da bismo se uverili u rezultat, proverimo da li se polinom sa \(a = b = -1\) zaista može podeliti sa \(x^2 - 1\):

\[ P(x) = x^{2018} + x^{2017} - x - 1 \]

Grupišemo članove:

\[ P(x) = x^{2017}(x + 1) - 1 \cdot (x + 1) = (x + 1)(x^{2017} - 1) \]

Polinom \(x^{2017} - 1\) je uvek deljiv sa \((x - 1)\) (jer je \(1^{2017} - 1 = 0\)), pa:

\[ P(x) = (x + 1)(x - 1) \cdot Q(x) = (x^2 - 1) \cdot Q(x) \]

gde je \(Q(x) = x^{2016} + x^{2015} + \ldots + x + 1\). Deljivost je potvrđena.

Ključni uvid

Kada polinom \(P(x)\) mora biti deljiv sa \(x^2 - 1 = (x-1)(x+1)\), dovoljno je uvrstiti nule delitelja (\(x = 1\) i \(x = -1\)) u \(P(x) = 0\). To daje sistem od dve linearne jednačine sa nepoznatim koeficijentima. Stepeni polinoma, koliko god visoki bili (\(2018\), \(2017\)), sami se uprošćavaju jer je \(1^n = 1\) i \((-1)^n = \pm 1\).

Konačni odgovor
\(a + 3b = -4\)
(A) \(0\) (B) \(2\) (C) \(-2\) (D) \(4\) (E) \(-4\)

Verifikacija: \(a = -1,\; b = -1\). Polinom \(x^{2018} + x^{2017} - x - 1 = (x+1)(x-1)(x^{2016} + x^{2015} + \ldots + 1)\) ✔

⚠️ Česte greške
Pogrešno računanje (−1) na visoke stepene

Najčešća greška je pogrešno odrediti \((-1)^{2018}\) ili \((-1)^{2017}\). Pravilo je jednostavno: paran stepen daje \(+1\), neparan stepen daje \(-1\). Broj 2018 je paran, 2017 je neparan.

Zaboravljanje jednog uslova

Deljivost sa \(x^2 - 1\) zahteva oba uslova: \(P(1) = 0\) i \(P(-1) = 0\). Ako se koristi samo jedan uslov, dobija se jedna jednačina sa dve nepoznate, što nije dovoljno za jednoznačno rešenje.

Računanje a + b umesto a + 3b

Pažljivo pročitajte šta se traži! Sistem daje \(a + b = -2\), ali zadatak traži \(a + 3b\). Greška je odgovoriti \(-2\) (opcija C) jer se to poklapa sa \(a + b\), ne sa traženim izrazom.

🏆 Dodatni izazov

Ako je polinom \(x^{100} - x^{99} + cx + d\) deljiv sa \(x^2 - 1\), koliko je \(c^2 + d^2\)?

Nagoveštaj

Isti pristup: \(P(1) = 0\) daje \(1 - 1 + c + d = 0\), tj. \(c + d = 0\). Zatim \(P(-1) = 0\) daje \(1 + 1 - c + d = 0\), tj. \(-c + d = -2\). Iz sistema: \(c = 1\), \(d = -1\), pa \(c^2 + d^2 = 1 + 1 = 2\).
