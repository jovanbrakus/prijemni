Deljenje polinoma i vrednost izraza

Algoritam dugog deljenja polinoma petog stepena sa ostatkom

Postavka zadatka

Neka je \(ax^2 + bx + c\) ostatak pri deljenju polinoma \(x^5 + x^4 + 2x^3 + 3x^2 + 3x + 2\) polinomom \(x^3 + x + 1\). Tada izraz \(a - 2b + 3c\) ima vrednost:

(A)
\(2\)
(B)
\(0\)
(C)
\(-2\)
(D)
\(3\)
(E)
\(-3\)

Plan rešavanja

Primenićemo algoritam dugog deljenja polinoma: delimo \(x^5 + x^4 + 2x^3 + 3x^2 + 3x + 2\) sa \(x^3 + x + 1\), korak po korak pronalazimo količnik i ostatak oblika \(ax^2 + bx + c\), a zatim izračunavamo traženi izraz \(a - 2b + 3c\).

Ključne formule i teoreme
  • Deljenje polinoma sa ostatkom: Za polinome \(P(x)\) i \(D(x)\) (gde je \(D(x) \neq 0\)), postoje jedinstveni polinomi \(Q(x)\) (količnik) i \(R(x)\) (ostatak) takvi da: \[ P(x) = Q(x) \cdot D(x) + R(x), \quad \deg(R) < \deg(D) \]
  • Stepeni pri deljenju: Ako je \(\deg(P) = n\) i \(\deg(D) = m\), onda je \(\deg(Q) = n - m\), a \(\deg(R) \leq m - 1\).
  • Algoritam dugog deljenja: U svakom koraku delimo vodeći član trenutnog ostatka vodećim članom delioca, množimo čitav delilac dobijenim rezultatom i oduzimamo od trenutnog ostatka. Postupak se ponavlja dok stepen ostatka ne padne ispod stepena delioca.
  • Provera: Rezultat je tačan ako i samo ako važi \(Q(x) \cdot D(x) + R(x) = P(x)\) identički (za svako \(x\)).

Vizuelni prikaz deljenja

Koristite klizač da pomerite vrednost \(x\) i vidite kako se \(P(x)\), \(Q(x) \cdot D(x)\) i ostatak \(R(x)\) ponašaju. Primetite da je razlika \(P(x) - Q(x) \cdot D(x)\) uvek jednaka \(R(x) = x^2 + x + 1\).

Rešenje korak po korak

1

Postavimo deljenje

Označimo polinome:

\[ P(x) = x^5 + x^4 + 2x^3 + 3x^2 + 3x + 2 \quad (\text{deljenik, stepen } 5) \]
\[ D(x) = x^3 + x + 1 \quad (\text{delilac, stepen } 3) \]

Tražimo količnik \(Q(x)\) i ostatak \(R(x) = ax^2 + bx + c\) tako da važi:

\[ P(x) = Q(x) \cdot D(x) + R(x), \quad \deg(R) < 3 \]

Pošto je stepen deljenika 5, a stepen delioca 3, količnik \(Q(x)\) će biti stepena \(5 - 3 = 2\), a ostatak \(R(x)\) stepena najviše 2 (oblika \(ax^2 + bx + c\)).

2

Prvi korak deljenja: delimo \(x^5\) sa \(x^3\)

Delimo vodeći član deljenika vodećim članom delioca:

\[ \frac{x^5}{x^3} = x^2 \]

Množimo čitav delilac sa \(x^2\):

\[ x^2 \cdot (x^3 + x + 1) = x^5 + x^3 + x^2 \]

Oduzimamo od deljenika:

\[ (x^5 + x^4 + 2x^3 + 3x^2 + 3x + 2) - (x^5 + x^3 + x^2) \]
\[ = x^4 + x^3 + 2x^2 + 3x + 2 \]

Član \(x^5\) se poništio. Primetimo: \(2x^3 - x^3 = x^3\) i \(3x^2 - x^2 = 2x^2\). Ostaje nam polinom stepena 4 za sledeći korak.

Proveri se: Koliko članova ima međurezultat?

Međurezultat \(x^4 + x^3 + 2x^2 + 3x + 2\) ima 5 članova (uz stepene \(x^4, x^3, x^2, x, 1\)). Njegov vodeći stepen je 4, što je i dalje veće od stepena delioca (3), pa nastavljamo deljenje.

3

Drugi korak deljenja: delimo \(x^4\) sa \(x^3\)

Nastavljamo sa ostatkom iz prethodnog koraka. Delimo vodeći član:

\[ \frac{x^4}{x^3} = x \]

Množimo čitav delilac sa \(x\):

\[ x \cdot (x^3 + x + 1) = x^4 + x^2 + x \]

Oduzimamo:

\[ (x^4 + x^3 + 2x^2 + 3x + 2) - (x^4 + x^2 + x) \]
\[ = x^3 + x^2 + 2x + 2 \]

Ponovo se \(x^4\) poništio. Ostaje: \(x^3 + (2-1)x^2 + (3-1)x + 2 = x^3 + x^2 + 2x + 2\). Stepen je još uvek 3, pa nastavljamo.

4

Treći korak deljenja: delimo \(x^3\) sa \(x^3\)

Delimo vodeći član:

\[ \frac{x^3}{x^3} = 1 \]

Množimo čitav delilac sa \(1\):

\[ 1 \cdot (x^3 + x + 1) = x^3 + x + 1 \]

Oduzimamo:

\[ (x^3 + x^2 + 2x + 2) - (x^3 + x + 1) \]
\[ = x^2 + x + 1 \]

Stepen ostatka \(x^2 + x + 1\) je 2, što je manje od stepena delioca (3). Deljenje je završeno!

Proveri se: Da li je deljenje tačno za \(x = 1\)?

Za \(x = 1\):

Leva strana: \(P(1) = 1 + 1 + 2 + 3 + 3 + 2 = 12\)

Količnik: \(Q(1) = 1 + 1 + 1 = 3\), Delilac: \(D(1) = 1 + 1 + 1 = 3\)

Desna strana: \(Q(1) \cdot D(1) + R(1) = 3 \cdot 3 + (1 + 1 + 1) = 9 + 3 = 12\)

Jednako! Naš rezultat prolazi numeričku proveru.

5

Zapišemo rezultat deljenja

Količnik je zbir članova koje smo dobili u koracima deljenja:

\[ Q(x) = x^2 + x + 1 \]

Ostatak je:

\[ R(x) = x^2 + x + 1 \]

Kompletna jednakost glasi:

\[ x^5 + x^4 + 2x^3 + 3x^2 + 3x + 2 = (x^2 + x + 1)(x^3 + x + 1) + (x^2 + x + 1) \]

Zanimljivo: količnik i ostatak su isti polinom! To je lepa osobina ovog konkretnog primera.

6

Očitavamo koeficijente \(a\), \(b\), \(c\)

Ostatak je \(R(x) = ax^2 + bx + c = x^2 + x + 1\), pa čitamo:

\[ a = 1, \quad b = 1, \quad c = 1 \]
7

Izračunavamo \(a - 2b + 3c\)

Uvršćujemo dobijene vrednosti:

\[ a - 2b + 3c = 1 - 2 \cdot 1 + 3 \cdot 1 \]
\[ = 1 - 2 + 3 \]
\[ = 2 \]
8

Verifikacija množenjem

Proverimo: \(Q(x) \cdot D(x) + R(x)\) treba da bude jednako \(P(x)\).

\[ (x^2 + x + 1)(x^3 + x + 1) \]

Raspišemo po članovima:

\[ = x^2 \cdot x^3 + x^2 \cdot x + x^2 \cdot 1 + x \cdot x^3 + x \cdot x + x \cdot 1 + 1 \cdot x^3 + 1 \cdot x + 1 \cdot 1 \]
\[ = x^5 + x^3 + x^2 + x^4 + x^2 + x + x^3 + x + 1 \]
\[ = x^5 + x^4 + 2x^3 + 2x^2 + 2x + 1 \]

Dodamo ostatak \(R(x) = x^2 + x + 1\):

\[ x^5 + x^4 + 2x^3 + 2x^2 + 2x + 1 + x^2 + x + 1 = x^5 + x^4 + 2x^3 + 3x^2 + 3x + 2 \]

Dobili smo polazni polinom \(P(x)\). Rezultat je potvrđen!

Proveri se: Šta bi izraz \(a - 2b + 3c\) bio da je ostatak \(x^2 - x + 1\)?

Tada bi bilo \(a = 1\), \(b = -1\), \(c = 1\), pa:

\(a - 2b + 3c = 1 - 2(-1) + 3(1) = 1 + 2 + 3 = 6\).

Ali to nije naš slučaj -- naš ostatak je \(x^2 + x + 1\), gde su svi koeficijenti jednaki 1.

Ključni uvid

U ovom zadatku količnik i ostatak su isti polinom \(x^2 + x + 1\). To znači da možemo faktorizovati deljenik na elegantan način:

\[ P(x) = (x^2 + x + 1)(x^3 + x + 1) + (x^2 + x + 1) = (x^2 + x + 1)(x^3 + x + 2) \]

Polinom \(x^2 + x + 1\) je treći ciklotomički polinom -- njegovi koreni su primitivni treći koreni jedinice \(\omega = e^{2\pi i/3}\).

Konačan odgovor

\(a - 2b + 3c = 2\)
(A) \(2\) (B) \(0\) (C) \(-2\) (D) \(3\) (E) \(-3\)
Verifikacija: Ostatak je \(R(x) = x^2 + x + 1\), dakle \(a = 1\), \(b = 1\), \(c = 1\). Izraz \(a - 2b + 3c = 1 - 2 + 3 = 2\). Kompletna provera: za \(x = 0\), \(P(0) = 2\) i \(Q(0) \cdot D(0) + R(0) = 1 \cdot 1 + 1 = 2\). Za \(x = -1\), \(P(-1) = -1 + 1 - 2 + 3 - 3 + 2 = 0\) i \(Q(-1) \cdot D(-1) + R(-1) = 1 \cdot (-1) + 1 = 0\).

Česte greške

  • Greška u znaku pri oduzimanju: Delilac \(x^3 + x + 1\) nema član uz \(x^2\) (koeficijent je 0). Pri množenju sa \(x^2\), \(x\) ili \(1\), lako je zaboraviti da taj „nulti član" postoji. Preporučuje se pisanje u obliku \(x^3 + 0 \cdot x^2 + x + 1\) kao čuvar mesta.
  • Zamena koeficijenata: Kada ostatak ima oblik \(ax^2 + bx + c\), pažljivo očitajte \(a\), \(b\) i \(c\). Česta greška je zamena redosleda ili pogrešan predznak pri uvrštavanju u izraz \(a - 2b + 3c\).

Bonus izazov

Pokažite da se polinom \(P(x) = x^5 + x^4 + 2x^3 + 3x^2 + 3x + 2\) može faktorizovati kao \((x^2 + x + 1)(x^3 + x + 2)\). Zatim pokažite da \(x^3 + x + 2 = (x + 1)(x^2 - x + 2)\) i nađite sve (realne) nule polinoma \(P(x)\).

Pogledaj nagoveštaj

Koristite činjenicu da je \(P(x) = (x^2 + x + 1)(x^3 + x + 1) + (x^2 + x + 1) = (x^2 + x + 1)(x^3 + x + 2)\).

Za \(x^3 + x + 2\), proverite da je \(x = -1\) nula: \((-1)^3 + (-1) + 2 = -1 - 1 + 2 = 0\). Dakle \(x^3 + x + 2 = (x + 1)(x^2 - x + 2)\).

Polinom \(x^2 + x + 1\) ima diskriminantu \(1 - 4 = -3 < 0\), a \(x^2 - x + 2\) ima diskriminantu \(1 - 8 = -7 < 0\). Jedina realna nula je \(x = -1\).
