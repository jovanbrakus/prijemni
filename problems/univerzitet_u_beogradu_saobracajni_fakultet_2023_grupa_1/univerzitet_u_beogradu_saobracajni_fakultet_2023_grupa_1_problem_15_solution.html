Jednacina sa kubnim korenima

Algebra — resavanje jednacine sa zbirom kubnih korena koriscenjem identiteta za zbir kubova

Tekst zadatka

Najveci ceo broj \(x\) za koji je

\[ \sqrt[3]{x+1} + \sqrt[3]{x+3} + \sqrt[3]{x+5} = 0 \]

je:

(A)
\(-9\)
(B)
\(-5\)
(C)
\(-2\)
(D)
\(-4\)
(E)
\(-3\)

Plan resavanja

Oznacicemo svaki kubni koren kao \(a\), \(b\), \(c\) i iskoristicemo algebarski identitet: ako je \(a + b + c = 0\), tada \(a^3 + b^3 + c^3 = 3abc\). Time cemo jednacinu svesti na algebarski oblik koji se lako resava razdvajanjem na slucajeve.

Podsetnik iz teorije

Kljucne formule i teoreme

Identitet za zbir kubova (kada je zbir nula):

\[ a + b + c = 0 \;\;\Longrightarrow\;\; a^3 + b^3 + c^3 = 3abc \]

Ovo sledi iz opsteg identiteta:

\[ a^3 + b^3 + c^3 - 3abc = (a + b + c)(a^2 + b^2 + c^2 - ab - bc - ca) \]

Kada je \(a + b + c = 0\), desna strana je nula, pa sledi \(a^3 + b^3 + c^3 = 3abc\).

Kvadrat binoma:

\[ (x + 3)^2 = x^2 + 6x + 9 \]

Proizvod zbira i razlike:

\[ (x+1)(x+5) = x^2 + 6x + 5 \]

Graficka vizuelizacija

Grafik funkcije \(f(x) = \sqrt[3]{x+1} + \sqrt[3]{x+3} + \sqrt[3]{x+5}\). Nula funkcije odgovara resenju jednacine. Pomeri klizac da istrazis vrednosti funkcije.

Resenje korak po korak

1

Uvodjenje oznaka

Oznacimo svaki kubni koren posebnom promenljivom da bismo lakse radili sa jednacinom:

\[ a = \sqrt[3]{x+1}, \quad b = \sqrt[3]{x+3}, \quad c = \sqrt[3]{x+5} \]

Tada uslov iz zadatka postaje:

\[ a + b + c = 0 \]
2

Primena identiteta za zbir kubova

Koristimo kljucni algebarski identitet: ako je zbir tri broja jednak nuli, onda je zbir njihovih kubova jednak trostrukom proizvodu:

\[ a + b + c = 0 \;\;\Longrightarrow\;\; a^3 + b^3 + c^3 = 3abc \]

Posto je \(a^3 = x+1\), \(b^3 = x+3\), \(c^3 = x+5\), leva strana postaje:

\[ a^3 + b^3 + c^3 = (x+1) + (x+3) + (x+5) = 3x + 9 \]
3

Formiranje jednacine

Desna strana identiteta je:

\[ 3abc = 3\sqrt[3]{(x+1)(x+3)(x+5)} \]

Dakle imamo:

\[ 3x + 9 = 3\sqrt[3]{(x+1)(x+3)(x+5)} \]

Delimo obe strane sa 3:

\[ x + 3 = \sqrt[3]{(x+1)(x+3)(x+5)} \]
Proveri sebe: Sta dobijamo kubiranjem obe strane?

Kubiranjem eliminisemo kubni koren i dobijamo: \((x+3)^3 = (x+1)(x+3)(x+5)\).

4

Kubiranje obe strane

Kubiramo obe strane jednacine da eliminisemo kubni koren:

\[ (x+3)^3 = (x+1)(x+3)(x+5) \]

Sada razmatramo dva slucaja.

5

Slucaj 1: \(x + 3 = 0\)

Ako je \(x + 3 = 0\), tj. \(x = -3\), proveravamo:

\[ \text{Leva strana: } (x+3)^3 = 0^3 = 0 \]
\[ \text{Desna strana: } (x+1)(x+3)(x+5) = (-2)(0)(2) = 0 \]

Obe strane su jednake nuli. Dakle, \(x = -3\) je resenje.

6

Slucaj 2: \(x + 3 \neq 0\)

Ako \(x + 3 \neq 0\), mozemo podeliti obe strane jednacine sa \((x+3)\):

\[ (x+3)^2 = (x+1)(x+5) \]

Raspisemo levu stranu:

\[ x^2 + 6x + 9 = (x+1)(x+5) \]

Raspisemo desnu stranu:

\[ x^2 + 6x + 9 = x^2 + 6x + 5 \]

Oduzimanjem \(x^2 + 6x\) sa obe strane dobijamo:

\[ 9 = 5 \]

Ovo je kontradikcija — nemoguc iskaz! Dakle, u ovom slucaju nema resenja.

Proveri sebe: Zasto kontradikcija 9 = 5 znaci da nema drugih resenja?

Kontradikcija znaci da ne postoji nijedna vrednost \(x\) (osim \(x = -3\) koju smo vec obradili) za koju bi jednacina bila zadovoljena. Svi koraci su bili ekvivalentne transformacije (pod uslovom \(x + 3 \neq 0\)), pa zakljucujemo da jedino resenje ostaje \(x = -3\).

7

Verifikacija resenja

Proverimo \(x = -3\) u polaznoj jednacini:

\[ \sqrt[3]{-3+1} + \sqrt[3]{-3+3} + \sqrt[3]{-3+5} \]
\[ = \sqrt[3]{-2} + \sqrt[3]{0} + \sqrt[3]{2} \]
\[ = -\sqrt[3]{2} + 0 + \sqrt[3]{2} = 0 \;\; \checkmark \]

Resenje je potvrdeno. Primetimo lepu simetriju: vrednosti \(-2\), \(0\) i \(2\) su simetricne oko nule, pa se kubni koreni od \(-2\) i \(2\) poniste.

Kljucni uvid

Srednji clan pod korenom, \(x + 3\), je aritmeticka sredina spoljnih clanova \(x + 1\) i \(x + 5\). Kada je \(x = -3\), sva tri argumenta postaju \(-2\), \(0\), \(2\) — simetricni oko nule. Kubni koren cuva tu simetriju (jer je neparna funkcija), pa se vrednosti poniste. Identitet \(a + b + c = 0 \Rightarrow a^3 + b^3 + c^3 = 3abc\) nam omogucava da ovo elegantno dokazemo bez pogadjanja.

\[ \sqrt[3]{-2} + \sqrt[3]{0} + \sqrt[3]{2} = 0 \]
Konacan odgovor
\( x = -3 \)
(A) \(-9\) (B) \(-5\) (C) \(-2\) (D) \(-4\) (E) \(-3\)
Verifikacija: \(\sqrt[3]{-2} + \sqrt[3]{0} + \sqrt[3]{2} = -\sqrt[3]{2} + 0 + \sqrt[3]{2} = 0\). Takodje, matematicki smo dokazali da je \(x = -3\) jedino resenje jednacine (drugi slucaj vodi na kontradikciju \(9 = 5\)).

Ceste greske

✗

Zaboravljanje analize slucajeva: Posle kubiranja, mnogi ucenici odmah dele sa \((x+3)\) bez provere da li je \(x + 3 = 0\). Deljenje nulom nije dozvoljeno i moze dovesti do gubitka resenja. Uvek prvo razmotrite slucaj kada je delilac jednak nuli.

✗

Pogresna primena identiteta: Identitet \(a^3 + b^3 + c^3 = 3abc\) vazi samo kada je \(a + b + c = 0\). Ako se primeni bez ovog uslova, rezultat je pogresan. Opsti oblik ukljucuje dodatni faktor: \(a^3 + b^3 + c^3 - 3abc = (a+b+c)(a^2+b^2+c^2-ab-bc-ca)\).

Dodatni izazov

Resi jednacinu \(\sqrt[3]{x+2} + \sqrt[3]{x+4} + \sqrt[3]{x+6} = 0\). Da li uocavas obrazac?

Pogledaj nagovestaj

Primeni isti identitet. Srednji argument je \(x + 4\), a spoljni su \(x + 2\) i \(x + 6\). Koja vrednost \(x\) cini da srednji argument bude nula? Razmisli o simetriji tri argumenta oko srednjeg clana — svaki put kada su argumenti u aritmetickom nizu, resenje je ono \(x\) koje ponistava srednji clan.
