🎯 Plan rešavanja

Apsolutne vrednosti \(|x + 1|\) i \(|x|\) menjaju izraz u zavisnosti od znaka argumenta. Kritične tačke su \(x = -1\) i \(x = 0\). Razdvojićemo brojevnu pravu na tri intervala, rešiti nejednačinu na svakom, a zatim prebrojati celobrojne vrednosti iz skupa rešenja.

• Kritične tačke → Slučaj \(x \geq 0\) → Slučaj \(-1 \leq x < 0\) → Slučaj \(x < -1\) → Prebrojavanje celih brojeva

📚 Podsetnik iz teorije
Definicija apsolutne vrednosti
\[ |a| = \begin{cases} a, & a \geq 0 \\ -a, & a < 0 \end{cases} \]

Apsolutna vrednost uklanja znak broja. Ključna tehnika je analiza po slučajevima — određujemo gde je argument nenegativan, a gde negativan.

Metod analize po slučajevima

Kada nejednačina sadrži izraze \(|f(x)|\), nalazimo sve tačke gde je \(f(x) = 0\). Te tačke dele brojevnu pravu na intervale. Na svakom intervalu, apsolutna vrednost se zamenjuje sa \(f(x)\) ili \(-f(x)\), čime dobijamo običnu nejednačinu.

Presek skupova rešenja
\[ \text{Rešenje na intervalu} = \text{(rešenje nejednačine)} \cap \text{(interval)} \]

Na kraju, ukupno rešenje je unija rešenja sa svih intervala.

🛠 Rešenje korak po korak
1 Odredi kritične tačke

Izrazi unutar apsolutnih vrednosti su \(x + 1\) i \(x\). Postavljamo ih na nulu:

\[ x + 1 = 0 \;\Rightarrow\; x = -1 \] \[ x = 0 \;\Rightarrow\; x = 0 \]

Kritične tačke \(x = -1\) i \(x = 0\) dele brojevnu pravu na tri intervala: \((-\infty, -1)\), \([-1, 0)\) i \([0, +\infty)\).

2 Slučaj 1: \(x \geq 0\)

Slučaj 1

Za \(x \geq 0\) važi \(x + 1 > 0\) i \(x \geq 0\), pa:

\[ |x + 1| = x + 1, \quad |x| = x \]

Zamenom u nejednačinu:

\[ (x + 1) + x \leq x + \frac{3}{2} \]

Pojednostavimo levu stranu:

\[ 2x + 1 \leq x + \frac{3}{2} \]

Oduzimamo \(x\) sa obe strane:

\[ x + 1 \leq \frac{3}{2} \]

Oduzimamo 1:

\[ x \leq \frac{1}{2} \]

Presek sa uslovom \(x \geq 0\):

\[ 0 \leq x \leq \frac{1}{2} \]

Celi brojevi u ovom intervalu: \(x = 0\)

✅ Proveri sebe: Da li je \(x = 0\) zaista rešenje?

Uvrsti \(x = 0\): leva strana je \(|0 + 1| + |0| = 1 + 0 = 1\). Desna strana je \(0 + \frac{3}{2} = 1{,}5\). Pošto je \(1 \leq 1{,}5\), jeste rešenje. ✔

3 Slučaj 2: \(-1 \leq x < 0\)

Slučaj 2

Za \(-1 \leq x < 0\) važi \(x + 1 \geq 0\) i \(x < 0\), pa:

\[ |x + 1| = x + 1, \quad |x| = -x \]

Zamenom u nejednačinu:

\[ (x + 1) + (-x) \leq x + \frac{3}{2} \]

Leva strana se upršćava:

\[ 1 \leq x + \frac{3}{2} \]

Oduzimamo \(\frac{3}{2}\) sa obe strane:

\[ -\frac{1}{2} \leq x \]

Presek sa uslovom \(-1 \leq x < 0\):

\[ -\frac{1}{2} \leq x < 0 \]

Nema celih brojeva u intervalu \(\left[-\frac{1}{2},\; 0\right)\)

4 Slučaj 3: \(x < -1\)

Slučaj 3

Za \(x < -1\) važi \(x + 1 < 0\) i \(x < 0\), pa:

\[ |x + 1| = -(x + 1) = -x - 1, \quad |x| = -x \]

Zamenom u nejednačinu:

\[ (-x - 1) + (-x) \leq x + \frac{3}{2} \]

Uprošćavamo levu stranu:

\[ -2x - 1 \leq x + \frac{3}{2} \]

Dodajemo \(2x\) na obe strane:

\[ -1 \leq 3x + \frac{3}{2} \]

Oduzimamo \(\frac{3}{2}\):

\[ -\frac{5}{2} \leq 3x \]

Delimo sa 3:

\[ x \geq -\frac{5}{6} \]

Ali uslov ovog slučaja zahteva \(x < -1\), a \(-\frac{5}{6} \approx -0{,}833\). Pošto je \(-1 < -\frac{5}{6}\), ne postoji nijedan \(x\) koji je istovremeno manji od \(-1\) i veći ili jednak od \(-\frac{5}{6}\).

\[ x < -1 \;\;\text{i}\;\; x \geq -\frac{5}{6} \quad \Rightarrow \quad \text{prazan skup} \]

Nema rešenja u ovom slučaju

✅ Proveri sebe: Zašto \(x = -1\) nije rešenje?

Uvrsti \(x = -1\): leva strana je \(|{-1} + 1| + |{-1}| = 0 + 1 = 1\). Desna strana je \(-1 + \frac{3}{2} = \frac{1}{2}\). Pošto \(1 \leq \frac{1}{2}\) ne važi, \(x = -1\) nije rešenje. ✔

5 Objedinjavanje rezultata

Unija svih rešenja sa tri intervala daje ukupan skup rešenja:

\[ \left[-\frac{1}{2},\; \frac{1}{2}\right] \]

Celi brojevi u intervalu \(\left[-\frac{1}{2},\; \frac{1}{2}\right]\):

\[ \text{Jedini ceo broj je } x = 0 \]

Dakle, postoji tačno 1 celobrojno rešenje.

✅ Proveri sebe: Da li je \(x = 0{,}5\) granični slučaj?

Uvrsti \(x = \frac{1}{2}\): leva strana je \(|\frac{1}{2} + 1| + |\frac{1}{2}| = \frac{3}{2} + \frac{1}{2} = 2\). Desna strana je \(\frac{1}{2} + \frac{3}{2} = 2\). Pošto je \(2 \leq 2\), jeste rešenje (jednakost važi). Ali \(\frac{1}{2}\) nije ceo broj, pa ne utiče na naš odgovor. ✔

Ključni uvid

Analiza po slučajevima pokazuje da je skup rešenja interval \(\left[-\frac{1}{2},\; \frac{1}{2}\right]\) — veoma uzak deo brojevne prave koji sadrži samo jedan ceo broj. To nije slučajnost: leva strana \(|x+1| + |x|\) raste brže od desne strane \(x + \frac{3}{2}\) za velike \(|x|\), pa se nejednačina zadovoljava samo u uskoj okolini nule.

Konačni odgovor
\(1\) celobrojno rešenje \((x = 0)\)
(A) \(0\)
(B) \(1\)
(C) \(2\)
(D) \(3\)
(E) Beskonačno mnogo

Verifikacija: Za \(x = 0\): \(|0 + 1| + |0| = 1 \leq 0 + \frac{3}{2} = 1{,}5\) ✔
Za \(x = 1\): \(|2| + |1| = 3 \leq 1 + \frac{3}{2} = 2{,}5\)? Ne — \(3 > 2{,}5\) ✘
Za \(x = -1\): \(|0| + |{-1}| = 1 \leq -1 + \frac{3}{2} = 0{,}5\)? Ne — \(1 > 0{,}5\) ✘

⚠️ Česte greške
Zaboravljanje da se skup rešenja preseca sa intervalom slučaja

U svakom slučaju rešavamo nejednačinu, ali rezultat moramo presecati sa uslovom tog intervala. Na primer, u slučaju 3 dobijamo \(x \geq -\frac{5}{6}\), ali uz uslov \(x < -1\) — presek je prazan.

Pogrešno uklanjanje apsolutne vrednosti

Česta greška je pisanje \(|x| = x\) za sve vrednosti \(x\). Za \(x < 0\) važi \(|x| = -x\). Posebno pažljivo treba odrediti znak argumenta na svakom intervalu.

🏆 Dodatni izazov

Koliko celobrojnih rešenja ima nejednačina:

\[ |x + 2| + |x - 1| \leq x + 4 \]
Nagovještaj

Kritične tačke su \(x = -2\) i \(x = 1\). Analiziraj tri slučaja. U slučaju \(x \geq 1\) dobija se \(x \leq 3\), u slučaju \(-2 \leq x < 1\) dobija se \(x \geq -1\), a u slučaju \(x < -2\) dobija se \(x \geq -\frac{7}{3}\). Ukupno rešenje je \(\left[-\frac{7}{3},\; 3\right]\), što sadrži cele brojeve \(-2, -1, 0, 1, 2, 3\) — ukupno 6.
