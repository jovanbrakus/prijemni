Nejednakost sa apsolutnom vrednošću razlomka

Kvadratne nejednačine, diskriminanta i uslovi za pozitivnu definitnost — traženje najvećeg parametra

📝 Zadatak

Najveći prirodan broj \(p\) takav da nejednakost

\[\left|\frac{x^2 + px + 1}{x^2 + x + 1}\right| < 2\]

važi za svako \(x\), je:

(A)
\(2\)
(B)
\(6\)
(C)
\(8\)
(D)
\(3\)
(E)
\(4\)

🎯 Plan rešavanja

Najpre ćemo pokazati da je imenilac \(x^2 + x + 1\) uvek pozitivan, pa apsolutnu vrednost razlomka možemo rastaviti na dve nejednačine. Svaka od njih daje uslov da odgovarajući kvadratni trinom bude pozitivno definitan (diskriminanta manja od nule). Presekom oba uslova dobijamo opseg za \(p\) i biramo najveći prirodan broj.

📖 Teorijska osnova

Ključne formule i pojmovi
  • Pozitivna definitnost: Kvadratni trinom \(ax^2 + bx + c\) (\(a > 0\)) je pozitivan za svako realno \(x\) ako i samo ako je diskriminanta \(D = b^2 - 4ac < 0\).
  • Apsolutna vrednost i nejednakosti: \(|f(x)| < M \iff -M < f(x) < M\) za \(M > 0\).
  • Diskriminanta: Za \(ax^2 + bx + c = 0\) važi \(D = b^2 - 4ac\). Ako je \(D < 0\) i \(a > 0\), trinom nema realnih nula i uvek je pozitivan.
  • Nejednakost sa apsolutnom vrednošću: \(|a| < b \iff -b < a < b\) za \(b > 0\).

📊 Vizuelni prikaz — grafik razlomka za različite vrednosti \(p\)

Grafik funkcije \(f(x) = \frac{x^2 + px + 1}{x^2 + x + 1}\) za \(p = 3\) (zeleno) i \(p = 4\) (crveno). Isprekidane linije označavaju granice \(y = 2\) i \(y = -2\).

🔢 Rešenje korak po korak

1
Analiziramo imenilac

Imenilac je \(x^2 + x + 1\). Izračunajmo njegovu diskriminantu:

\[D = 1^2 - 4 \cdot 1 \cdot 1 = 1 - 4 = -3 < 0\]

Pošto je vodeći koeficijent pozitivan (\(a = 1 > 0\)) i \(D < 0\), zaključujemo:

\[x^2 + x + 1 > 0 \quad \text{za svako realno } x\]

Ovo je ključan podatak — imenilac je uvek pozitivan, pa možemo bezbedno množiti nejednačinu njime bez promene znaka.

2
Raščlanjujemo apsolutnu vrednost

Nejednakost \(\left|\frac{x^2 + px + 1}{x^2 + x + 1}\right| < 2\) je ekvivalentna:

\[-2 < \frac{x^2 + px + 1}{x^2 + x + 1} < 2\]

Množimo sve delove sa \(x^2 + x + 1 > 0\) (znak se ne menja):

\[-2(x^2 + x + 1) < x^2 + px + 1 < 2(x^2 + x + 1)\]

Ovo nam daje dve nezavisne nejednačine koje obe moraju da važe za svako \(x\).

Proveri se: Zašto smemo da množimo nejednakost imeniocem bez promene znaka?

Zato što smo u koraku 1 dokazali da je \(x^2 + x + 1 > 0\) za svako \(x\). Množenje pozitivnim brojem ne menja smer nejednakosti.

3
Desna nejednačina

Iz \(x^2 + px + 1 < 2(x^2 + x + 1)\) dobijamo:

\[x^2 + px + 1 < 2x^2 + 2x + 2\]

Prebacimo sve na desnu stranu:

\[0 < x^2 + (2 - p)x + 1\]

Dakle tražimo da je trinom \(x^2 + (2-p)x + 1\) pozitivan za svako \(x\).

Vodeći koeficijent je \(1 > 0\), pa je potreban i dovoljan uslov \(D_1 < 0\):

\[D_1 = (2-p)^2 - 4 \cdot 1 \cdot 1 < 0\]
\[(2-p)^2 < 4\]

Primenjujemo \(|a| < b \iff -b < a < b\):

\[|2 - p| < 2 \implies -2 < 2 - p < 2\]

Oduzimamo \(2\) od svih delova i množimo sa \(-1\) (menja se smer):

\[\boxed{0 < p < 4}\]
4
Leva nejednačina

Iz \(-2(x^2 + x + 1) < x^2 + px + 1\) dobijamo:

\[-2x^2 - 2x - 2 < x^2 + px + 1\]

Prebacimo sve na desnu stranu:

\[0 < 3x^2 + (p + 2)x + 3\]

Tražimo da je \(3x^2 + (p+2)x + 3\) pozitivan za svako \(x\).

Vodeći koeficijent je \(3 > 0\), pa uslov glasi \(D_2 < 0\):

\[D_2 = (p+2)^2 - 4 \cdot 3 \cdot 3 < 0\]
\[(p+2)^2 - 36 < 0 \implies (p+2)^2 < 36\]

Ponovo primenjujemo osobinu apsolutne vrednosti:

\[|p + 2| < 6 \implies -6 < p + 2 < 6 \implies -8 < p < 4\]

Dakle leva nejednačina daje:

\[\boxed{-8 < p < 4}\]
Proveri se: Šta bi se desilo da je \(D_1 = 0\) (granični slučaj)?

Ako je \(D_1 = 0\), trinom \(x^2 + (2-p)x + 1\) bi imao dvostruku nulu, tj. bio bi nula za neko \(x\). To znači da bi razlomak bio tačno \(2\) za to \(x\), a nama treba stroga nejednakost (manje od \(2\)), pa granični slučaj ne zadovoljava.

5
Presek uslova

Oba uslova moraju da važe istovremeno:

\[\text{Iz desne: } 0 < p < 4\]
\[\text{Iz leve: } -8 < p < 4\]

Presek ova dva intervala je:

\[0 < p < 4\]
6
Određujemo najveći prirodan broj

Pošto je \(p\) prirodan broj i mora da važi \(0 < p < 4\), mogući su:

\[p \in \{1,\; 2,\; 3\}\]

Najveći od ovih je:

\[\boxed{p = 3}\]
7
Verifikacija za \(p = 3\)

Proverimo oba uslova za \(p = 3\):

Desna nejednačina: Trinom je \(x^2 - x + 1\).

\[D_1 = (-1)^2 - 4 \cdot 1 \cdot 1 = 1 - 4 = -3 < 0 \quad \checkmark\]

Leva nejednačina: Trinom je \(3x^2 + 5x + 3\).

\[D_2 = 5^2 - 4 \cdot 3 \cdot 3 = 25 - 36 = -11 < 0 \quad \checkmark\]

Obe diskriminante su negativne, dakle oba trinoma su pozitivna za svako \(x\). Nejednakost važi.

Proveri se: Zašto \(p = 4\) ne zadovoljava uslov?

Za \(p = 4\), desna nejednačina daje trinom \(x^2 - 2x + 1 = (x-1)^2\). Ovaj trinom je \(0\) za \(x = 1\), dakle nije strogo pozitivan za svako \(x\). Za \(x = 1\) dobijamo \(f(1) = \frac{1 + 4 + 1}{1 + 1 + 1} = \frac{6}{3} = 2\), što nije manje od \(2\) — nejednakost nije zadovoljena.

💡 Ključni uvid

Celokupan zadatak se svodi na pitanje: za koje vrednosti parametra \(p\) su dva kvadratna trinoma pozitivno definitna? Ključni alat je diskriminanta — trinom sa pozitivnim vodećim koeficijentom je pozitivan za svako \(x\) ako i samo ako mu je diskriminanta negativna.

\[D < 0 \;\Longrightarrow\; ax^2 + bx + c > 0 \;\;\text{za svako } x \;\;(a > 0)\]

✅ Konačan odgovor

\[p = 3\]
(A) \(2\) (B) \(6\) (C) \(8\) (D) \(3\) (E) \(4\)

Verifikacija: Za \(p = 3\) obe diskriminante su negativne (\(D_1 = -3\), \(D_2 = -11\)), pa nejednakost važi za svako \(x\). Za \(p = 4\) razlomak dostiže vrednost \(2\) za \(x = 1\), pa stroga nejednakost ne važi.

⚠️ Najčešće greške

  • Zaboravljena leva nejednačina: Mnogi razmatraju samo \(f(x) < 2\) i zaborave uslov \(f(x) > -2\). U ovom zadatku oba uslova daju istu gornju granicu \(p < 4\), ali generalno leva nejednačina može dati strožiji uslov.
  • Uključivanje graničnih vrednosti: Za \(p = 4\) desni trinom postaje \((x-1)^2 \geq 0\), koji je nula za \(x = 1\). To daje \(f(1) = 2\), a nama treba stroga nejednakost \(< 2\), ne \(\leq 2\). Dakle \(p = 4\) ne zadovoljava uslov.

🏆 Dodatni izazov

Za koje realne vrednosti parametra \(a\) važi:

\[\left|\frac{x^2 + ax + 1}{x^2 + x + 1}\right| \leq 3 \quad \text{za svako } x \in \mathbb{R}\text{?}\]
Pogledaj nagoveštaj

Postupi isto: raspiši na dve nejednačine, zahtevaj \(D \leq 0\) (sada sa \(\leq\) jer je nejednakost neoštrog tipa). Iz desne nejednačine dobićeš trinom \(x^2 + (2-a)x + 2 \geq 0\) za svako \(x\), a iz leve \(4x^2 + (a+3)x + 4 \geq 0\). Nađi uslove na \(a\) pa preseci.
