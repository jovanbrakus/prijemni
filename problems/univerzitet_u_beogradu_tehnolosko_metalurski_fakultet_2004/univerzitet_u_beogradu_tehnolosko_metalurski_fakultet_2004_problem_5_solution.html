Koeficijenti kvadratnog polinoma

Sistemi linearnih jednačina — kvadratna funkcija

📜 Zadatak

Neka je \(P(x) = ax^2 + bx + c\). Ako je \(P(1) = 1\), \(P(0) = 2\), \(P(-1) = 7\), onda su koeficijenti \(a\), \(b\), \(c\) elementi skupa:

(A)
\(\{1,\, 2,\, 3\}\)
(B)
\(\{-1,\, 2,\, 3\}\)
(C)
\(\{1,\, 2,\, -3\}\)
(D)
\(\{-1,\, -2,\, 3\}\)
(E)
\(\{-1,\, -2,\, -3\}\)
📌 Plan rešavanja
  1. Uvrstiti zadate vrednosti \(x = 0\), \(x = 1\) i \(x = -1\) u polinom i dobiti sistem od tri jednačine sa tri nepoznate
  2. Rešiti sistem i odrediti koeficijente \(a\), \(b\), \(c\)
  3. Proveriti kom ponuđenom skupu pripadaju dobijeni koeficijenti
Podsetnik iz teorije

Opšti oblik kvadratnog polinoma:

\[P(x) = ax^2 + bx + c\]

Vrednost polinoma u tački: Zamenom \(x\) datom vrednošću izračunavamo \(P(x)\).

Sistem linearnih jednačina: Tri uslova (\(P(1)\), \(P(0)\), \(P(-1)\)) daju tri linearne jednačine po nepoznatim \(a\), \(b\), \(c\).

Metod sabiranja: Sabiranjem i oduzimanjem jednačina eliminišemo nepoznate jednu po jednu.

📊 Grafički prikaz

Graf polinoma \(P(x) = 2x^2 - 3x + 2\) sa označenim tačkama iz uslova zadatka.

✏️ Rešenje korak po korak
Korak 1: Postavljanje jednačina

Uvrstimo zadate vrednosti u \(P(x) = ax^2 + bx + c\):

Za \(x = 0\):

\[P(0) = a \cdot 0^2 + b \cdot 0 + c = c = 2\]

Za \(x = 1\):

\[P(1) = a \cdot 1^2 + b \cdot 1 + c = a + b + c = 1\]

Za \(x = -1\):

\[P(-1) = a \cdot (-1)^2 + b \cdot (-1) + c = a - b + c = 7\]
Korak 2: Određivanje \(c\)

Iz prvog uslova odmah dobijamo:

\[c = 2\]

Uvršćivanje \(x = 0\) direktno daje slobodan član polinoma.

Proveri se: Koliko iznosi \(c\)?

\(c = P(0) = 2\). Tačno!

Korak 3: Zamena \(c = 2\) u preostale jednačine

Iz \(P(1) = 1\):

\[a + b + 2 = 1 \quad \Rightarrow \quad a + b = -1 \qquad \text{...(I)}\]

Iz \(P(-1) = 7\):

\[a - b + 2 = 7 \quad \Rightarrow \quad a - b = 5 \qquad \text{...(II)}\]
Korak 4: Rešavanje sistema za \(a\) i \(b\)

Saberemo jednačine (I) i (II):

\[(a + b) + (a - b) = -1 + 5\] \[2a = 4\] \[a = 2\]

Zamenom \(a = 2\) u jednačinu (I):

\[2 + b = -1\] \[b = -3\]
Proveri se: Da li su \(a = 2\), \(b = -3\), \(c = 2\) tačni?

\(P(1) = 2 \cdot 1 + (-3) \cdot 1 + 2 = 2 - 3 + 2 = 1\) ✔

\(P(0) = 2\) ✔

\(P(-1) = 2 \cdot 1 - (-3) \cdot 1 + 2 = 2 + 3 + 2 = 7\) ✔

Korak 5: Određivanje odgovarajućeg skupa

Dobijeni koeficijenti su: \(a = 2\), \(b = -3\), \(c = 2\).

Proverimo kom skupu pripadaju sve tri vrednosti:

  • (A) \(\{1, 2, 3\}\): \(b = -3 \notin \{1, 2, 3\}\) ✘
  • (B) \(\{-1, 2, 3\}\): \(b = -3 \notin \{-1, 2, 3\}\) ✘
  • (C) \(\{1, 2, -3\}\): \(a = 2 \in\), \(b = -3 \in\), \(c = 2 \in\) ✔
  • (D) \(\{-1, -2, 3\}\): \(a = 2 \notin \{-1, -2, 3\}\) ✘
  • (E) \(\{-1, -2, -3\}\): \(a = 2 \notin \{-1, -2, -3\}\) ✘

Jedino skup \(\{1, 2, -3\}\) sadrži sve tri vrednosti koeficijenata.

💡 Ključni uvid

Uvršćivanje \(x = 0\) u polinom odmah daje vrednost slobodnog člana \(c\), čime se sistem svodi na dve jednačine sa dve nepoznate. Zatim, izbor tačaka \(x = 1\) i \(x = -1\) je posebno pogodan jer daje simetričan par jednačina — sabiranjem eliminišemo \(b\), a oduzimanjem eliminišemo \(a\).

Konačan odgovor
\(a = 2,\; b = -3,\; c = 2\)
(A) \(\{1, 2, 3\}\) (B) \(\{-1, 2, 3\}\) (C) \(\{1, 2, -3\}\) (D) \(\{-1, -2, 3\}\) (E) \(\{-1, -2, -3\}\)

Provera: \(P(x) = 2x^2 - 3x + 2\) zadovoljava sva tri uslova: \(P(1) = 1\), \(P(0) = 2\), \(P(-1) = 7\).

⚠️ Česte greške
Mešanje redosleda koeficijenata sa elementima skupa: Zadatak pita kom skupu pripadaju koeficijenti, ne u kom redosledu se pojavljuju. Vrednost 1 je u skupu (C), ali nije nijedan od koeficijenata — to je u redu, jer skup može imati i element koji se ne koristi.
Greška pri računanju \(P(-1)\): Obratite pažnju da je \((-1)^2 = 1\) (pozitivno!), pa je \(P(-1) = a - b + c\), ne \(-a - b + c\).
🏆 Dodatni izazov

Neka je \(Q(x) = ax^3 + bx^2 + cx + d\). Ako je \(Q(0) = 1\), \(Q(1) = 0\), \(Q(-1) = 4\), \(Q(2) = -3\), odredite koeficijente \(a\), \(b\), \(c\), \(d\).

Savet: Isti pristup — \(Q(0)\) odmah daje \(d\), a zatim rešavate sistem od tri jednačine sa tri nepoznate.
