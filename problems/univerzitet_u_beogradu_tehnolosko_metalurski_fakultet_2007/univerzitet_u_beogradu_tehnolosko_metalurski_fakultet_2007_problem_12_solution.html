🧭 Plan rešavanja

Prebacićemo jednačinu u oblik \(\sin x + \cos x = -1\), zatim primeniti metod pomoćnog ugla da izraz \(\sin x + \cos x\) zapišemo kao \(\sqrt{2}\,\sin\!\left(x + \dfrac{\pi}{4}\right)\). Time dobijamo elementarnu sinusnu jednačinu čija rešenja analiziramo na datom otvorenom intervalu.

📐 Ključne formule i pojmovi
  • Zbir sinusa i kosinusa: \[\sin x + \cos x = \sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right)\] Ovo sledi iz adicione formule: \(\sin\alpha\cos\beta + \cos\alpha\sin\beta = \sin(\alpha + \beta)\).
  • Pomoćni ugao: Za izraz \(a\sin x + b\cos x\) važi: \[a\sin x + b\cos x = \sqrt{a^2 + b^2}\,\sin(x + \varphi), \quad \tan\varphi = \frac{b}{a}\]
  • Rešenja \(\sin t = c\): Za \(|c| \le 1\), opšta rešenja su: \[t = \arcsin c + 2k\pi \quad \text{ili} \quad t = \pi - \arcsin c + 2k\pi, \quad k \in \mathbb{Z}\]
  • Specijalne vrednosti: \(\sin\!\left(-\dfrac{\pi}{4}\right) = -\dfrac{\sqrt{2}}{2}\), \quad \(\sin\!\left(-\dfrac{3\pi}{4}\right) = -\dfrac{\sqrt{2}}{2}\)
  • Otvoren interval: \((-\pi, \pi)\) ne uključuje krajnje tačke \(-\pi\) i \(\pi\).

📝 Rešenje korak po korak

1

Preuređujemo jednačinu

Polazna jednačina je:

\[1 + \sin x + \cos x = 0\]

Prebacujemo 1 na desnu stranu:

\[\sin x + \cos x = -1\]
2

Primenjujemo metod pomoćnog ugla

Koristimo identitet za zbir sinusa i kosinusa sa koeficijentima \(a = 1\), \(b = 1\):

\[\sin x + \cos x = \sqrt{1^2 + 1^2}\,\sin\!\left(x + \varphi\right) = \sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right)\]

gde je \(\tan\varphi = \dfrac{b}{a} = \dfrac{1}{1} = 1\), pa \(\varphi = \dfrac{\pi}{4}\).

Provera: \(\sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right) = \sqrt{2}\left(\sin x \cdot \frac{\sqrt{2}}{2} + \cos x \cdot \frac{\sqrt{2}}{2}\right) = \sin x + \cos x\) ✓

🔍 Proveri sebe: Zašto je \(\varphi = \pi/4\)?

Traži se ugao \(\varphi\) takav da važi \(\sin x + \cos x = \sqrt{2}\,\sin(x + \varphi)\).

Razvijamo desnu stranu: \(\sqrt{2}\,(\sin x \cos\varphi + \cos x \sin\varphi)\).

Poredimo koeficijente: \(\sqrt{2}\cos\varphi = 1\) i \(\sqrt{2}\sin\varphi = 1\), dakle \(\cos\varphi = \sin\varphi = \dfrac{1}{\sqrt{2}}\), što daje \(\varphi = \dfrac{\pi}{4}\).

3

Dobijamo elementarnu sinusnu jednačinu

Zamenom u jednačinu \(\sin x + \cos x = -1\) dobijamo:

\[\sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right) = -1\]

Delimo sa \(\sqrt{2}\):

\[\sin\!\left(x + \frac{\pi}{4}\right) = -\frac{1}{\sqrt{2}} = -\frac{\sqrt{2}}{2}\]
4

Nalazimo opšta rešenja

Uvodimo smenu \(t = x + \dfrac{\pi}{4}\). Jednačina postaje:

\[\sin t = -\frac{\sqrt{2}}{2}\]

Opšta rešenja ove jednačine su:

\[t_1 = -\frac{\pi}{4} + 2k\pi, \quad k \in \mathbb{Z}\] \[t_2 = \pi + \frac{\pi}{4} + 2k\pi = \frac{5\pi}{4} + 2k\pi, \quad k \in \mathbb{Z}\]

Koristimo činjenicu da \(\arcsin\!\left(-\frac{\sqrt{2}}{2}\right) = -\frac{\pi}{4}\), a drugo rešenje je \(\pi - \left(-\frac{\pi}{4}\right) = \frac{5\pi}{4}\).

🔍 Proveri sebe: Koji interval za \(t\) odgovara intervalu \((-\pi, \pi)\) za \(x\)?

Pošto je \(t = x + \dfrac{\pi}{4}\) i \(x \in (-\pi, \pi)\), dobijamo:

\[t \in \left(-\pi + \frac{\pi}{4},\; \pi + \frac{\pi}{4}\right) = \left(-\frac{3\pi}{4},\; \frac{5\pi}{4}\right)\]

Ovo je otvoren interval, pa krajnje tačke \(-\dfrac{3\pi}{4}\) i \(\dfrac{5\pi}{4}\) ne pripadaju intervalu.

5

Biramo rešenja iz intervala

Pošto je \(t \in \left(-\dfrac{3\pi}{4},\; \dfrac{5\pi}{4}\right)\), proveravamo kandidate:

Kandidat \(t_1 = -\dfrac{\pi}{4}\) (za \(k = 0\)): \[-\frac{3\pi}{4} < -\frac{\pi}{4} < \frac{5\pi}{4} \quad \checkmark\] \[x_1 = t_1 - \frac{\pi}{4} = -\frac{\pi}{4} - \frac{\pi}{4} = -\frac{\pi}{2}\]
Kandidat \(t_2 = -\dfrac{3\pi}{4}\) (alternativni zapis za \(k = -1\) u drugoj grani): \[t_2 = \frac{5\pi}{4} - 2\pi = -\frac{3\pi}{4}\] Ali \(-\dfrac{3\pi}{4}\) je granica intervala, a interval je otvoren! \[-\frac{3\pi}{4} \notin \left(-\frac{3\pi}{4},\; \frac{5\pi}{4}\right) \quad \times\]
Kandidat \(t_3 = \dfrac{5\pi}{4}\) (za \(k = 0\) u drugoj grani): \[\frac{5\pi}{4}\] je granica intervala, opet ne pripada otvorenom intervalu. \[\frac{5\pi}{4} \notin \left(-\frac{3\pi}{4},\; \frac{5\pi}{4}\right) \quad \times\]

Sva ostala rešenja (za druge vrednosti \(k\)) padaju van intervala.

Oba granična kandidata odgovaraju istom \(x = -\pi\), koji ne pripada otvorenom intervalu \((-\pi, \pi)\).

6

Verifikacija rešenja

Jedino rešenje je \(x = -\dfrac{\pi}{2}\). Proveravamo u polaznoj jednačini:

\[\begin{aligned} 1 + \sin\!\left(-\frac{\pi}{2}\right) + \cos\!\left(-\frac{\pi}{2}\right) &= 1 + (-1) + 0 \\ &= 0 \quad \checkmark \end{aligned}\]

Proveravamo i granični slučaj \(x = -\pi\):

\[1 + \sin(-\pi) + \cos(-\pi) = 1 + 0 + (-1) = 0 \quad \checkmark\]

Vrednost \(x = -\pi\) jeste rešenje jednačine, ali ne pripada otvorenom intervalu \((-\pi, \pi)\), pa se ne računa.

🔍 Proveri sebe: Šta bi bilo da je interval zatvoren \([-\pi, \pi]\)?

Na zatvorenom intervalu \([-\pi, \pi]\) dobili bismo dva rešenja:

\[x_1 = -\frac{\pi}{2}, \quad x_2 = -\pi\]

Upravo zato je pitanje pažljivo formulisano sa otvorenim intervalom \((-\pi, \pi)\) — da bi odgovor bio jednoznačno 1.

💡 Ključni uvid

Metod pomoćnog ugla transformiše zbir \(\sin x + \cos x\) u jedan sinusni izraz, čime se jednačina sa dva trigonometrijska izraza svodi na elementarnu:

\[\sin x + \cos x = \sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right)\]

Ključan detalj je da je interval otvoren — tačka \(x = -\pi\) zadovoljava jednačinu, ali ne pripada intervalu \((-\pi, \pi)\). Da je interval bio zatvoren, odgovor bi bio 2 umesto 1.

✅ Konačan odgovor

Broj rešenja: \(1\)

Jedino rešenje je \(x = -\dfrac{\pi}{2}\).

(A) 4 (B) 2 (C) 1 (D) 0 (E) 3
Verifikacija: Za \(x = -\frac{\pi}{2}\): \(1 + \sin\!\left(-\frac{\pi}{2}\right) + \cos\!\left(-\frac{\pi}{2}\right) = 1 + (-1) + 0 = 0\) ✓. Granična tačka \(x = -\pi\) takođe zadovoljava jednačinu (\(1 + 0 + (-1) = 0\)), ali ne pripada otvorenom intervalu \((-\pi, \pi)\). Dakle, tačno jedno rešenje.

⚠️ Česte greške

Ignorisanje da je interval otvoren. Vrednost \(x = -\pi\) zadovoljava jednačinu, ali \(-\pi \notin (-\pi, \pi)\). Razlika između otvorenog i zatvorenog intervala ovde menja konačan odgovor.
Pogrešna primena pomoćnog ugla. Ako se umesto \(\sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right)\) pogrešno napiše \(\sqrt{2}\,\cos\!\left(x - \frac{\pi}{4}\right)\), rezultat može biti isti, ali su fazni pomaci različiti i lako je pogrešiti pri određivanju rešenja.
Kvadriranje jednačine. Mnogi učenici pokušavaju da reše jednačinu kvadriranjem (\(\sin^2 x + 2\sin x\cos x + \cos^2 x = 1\)), što uvodi lažna rešenja i komplikuje problem. Metod pomoćnog ugla je daleko efikasniji.
Propuštanje provere kandidata u intervalu. Sinusna jednačina daje beskonačno mnogo rešenja; neophodno je pažljivo izdvojiti samo ona koja pripadaju traženom intervalu.
