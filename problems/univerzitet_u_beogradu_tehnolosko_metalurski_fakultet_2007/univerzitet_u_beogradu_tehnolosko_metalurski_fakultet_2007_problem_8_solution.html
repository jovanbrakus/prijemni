Jednačina sa apsolutnim vrednostima

Apsolutne vrednosti -- metoda razmatranja slučajeva

Zadatak

Zbir svih rešenja jednačine

\[|x| + |x + 2| = |x + 3| + 3\]

jednak je:

A)
\(\dfrac{17}{3}\)
B)
\(\dfrac{4}{3}\)
C)
\(\dfrac{15}{3}\)
D)
\(\dfrac{18}{3}\)
E)
\(\dfrac{11}{3}\)

Plan rešavanja

Jednačina sadrži tri apsolutne vrednosti čije su kritične tačke \(x = -3\), \(x = -2\) i \(x = 0\). Podelićemo brojevnu pravu na četiri intervala prema ovim tačkama, ukloniti apsolutne vrednosti u svakom intervalu, rešiti dobijenu linearnu jednačinu i proveriti da li rešenje pripada datom intervalu. Na kraju sabiramo sva validna rešenja.

Ključne formule i pojmovi
  • Apsolutna vrednost: \(|a| = \begin{cases} a, & a \geq 0 \\ -a, & a < 0 \end{cases}\)
  • Metoda slučajeva: Kritične tačke (nule izraza pod apsolutnim vrednostima) dele brojevnu pravu na intervale. U svakom intervalu znamo znak svakog izraza, pa uklanjamo apsolutne vrednosti.
  • Provera pripadnosti: Rešenje linearne jednačine je validno samo ako pripada intervalu u kome smo ga dobili.
  • Kritične tačke: Za \(|x|\): \(x=0\); za \(|x+2|\): \(x=-2\); za \(|x+3|\): \(x=-3\).

Rešenje korak po korak

1Određivanje kritičnih tačaka

Pronalazimo nule izraza unutar apsolutnih vrednosti:

\[|x| \Rightarrow x = 0, \qquad |x+2| \Rightarrow x = -2, \qquad |x+3| \Rightarrow x = -3\]

Ove tri tačke dele brojevnu pravu na četiri intervala:

\[(-\infty,\, -3), \quad [-3,\, -2), \quad [-2,\, 0), \quad [0,\, +\infty)\]
2Slučaj 1: \(x \geq 0\)

U ovom intervalu su svi izrazi nenegativni: \(x \geq 0\), \(x+2 \geq 2 > 0\), \(x+3 \geq 3 > 0\).

\[x + (x + 2) = (x + 3) + 3\] \[2x + 2 = x + 6\] \[x = 4\]

Validno: \(x = 4 \geq 0\) -- pripada intervalu.

Proveri se: Verifikacija za \(x = 4\)

\(|4| + |4 + 2| = 4 + 6 = 10\)

\(|4 + 3| + 3 = 7 + 3 = 10\)

\(10 = 10\) -- tačno.

3Slučaj 2: \(-2 \leq x < 0\)

Ovde: \(x < 0\), ali \(x+2 \geq 0\) i \(x+3 \geq 1 > 0\).

\[(-x) + (x + 2) = (x + 3) + 3\] \[2 = x + 6\] \[x = -4\]

Nevalidno: \(x = -4 \notin [-2,\, 0)\) -- ne pripada intervalu.

4Slučaj 3: \(-3 \leq x < -2\)

Ovde: \(x < 0\), \(x+2 < 0\), ali \(x+3 \geq 0\).

\[(-x) + (-(x + 2)) = (x + 3) + 3\] \[-x - x - 2 = x + 6\] \[-2x - 2 = x + 6\] \[-3x = 8\] \[x = -\frac{8}{3}\]

Validno: \(-3 \leq -\dfrac{8}{3} \approx -2{,}667 < -2\) -- pripada intervalu.

Proveri se: Verifikacija za \(x = -\dfrac{8}{3}\)

\(\left|-\dfrac{8}{3}\right| + \left|-\dfrac{8}{3} + 2\right| = \dfrac{8}{3} + \left|-\dfrac{2}{3}\right| = \dfrac{8}{3} + \dfrac{2}{3} = \dfrac{10}{3}\)

\(\left|-\dfrac{8}{3} + 3\right| + 3 = \left|\dfrac{1}{3}\right| + 3 = \dfrac{1}{3} + 3 = \dfrac{10}{3}\)

\(\dfrac{10}{3} = \dfrac{10}{3}\) -- tačno.

5Slučaj 4: \(x < -3\)

Ovde su svi izrazi negativni: \(x < 0\), \(x+2 < -1 < 0\), \(x+3 < 0\).

\[(-x) + (-(x + 2)) = (-(x + 3)) + 3\] \[-2x - 2 = -x - 3 + 3\] \[-2x - 2 = -x\] \[-x = 2\] \[x = -2\]

Nevalidno: \(x = -2 \not< -3\) -- ne pripada intervalu.

6Zbir svih rešenja

Validna rešenja su \(x_1 = 4\) i \(x_2 = -\dfrac{8}{3}\). Računamo njihov zbir:

\[x_1 + x_2 = 4 + \left(-\frac{8}{3}\right) = \frac{12}{3} - \frac{8}{3} = \boxed{\frac{4}{3}}\]
Proveri se: Da li smo pronašli sva rešenja?

Razmotrili smo sva četiri intervala određena kritičnim tačkama. U svakom intervalu jednačina se svodi na linearnu, koja ima tačno jedno rešenje. Od četiri kandidata, dva su validna (pripadaju svom intervalu), a dva su nevalidna. Dakle, jednačina ima tačno dva rešenja.

Ključni uvid

Kada jednačina sadrži više apsolutnih vrednosti, metoda razmatranja slučajeva je sistematična i sigurna. Ključno je tačno identifikovati kritične tačke i proveriti da li svaki kandidat pripada svom intervalu. Zanimljivo je da u intervalu \([-2, 0)\) leva strana postaje konstanta \(2\), što odmah daje jedinstven (ali nevalidan) kandidat. Ovo se dešava jer se \(-x\) i \(x+2\) na tom intervalu "poništavaju" pri sabiranju.

Konačan odgovor
\(\displaystyle x_1 + x_2 = 4 + \left(-\frac{8}{3}\right) = \frac{4}{3}\)
(A) \(\dfrac{17}{3}\)
(B) \(\dfrac{4}{3}\)
(C) \(\dfrac{15}{3}\)
(D) \(\dfrac{18}{3}\)
(E) \(\dfrac{11}{3}\)

Česte greške

  • Propuštanje intervala: Ako se ne razmotre svi intervali, može se propustiti rešenje \(x = -\dfrac{8}{3}\), a dobiti samo \(x = 4\). U tom slučaju bi se pogrešno zaključilo da je "zbir" jednak 4.
  • Pogrešni znaci pri uklanjanju apsolutne vrednosti: Na intervalu \([-3, -2)\) važi \(|x+2| = -(x+2)\), ne \(x+2\). Pogrešan znak menja celokupno rešenje.
  • Neprovera pripadnosti intervalu: Kandidat \(x = -4\) iz drugog slučaja i \(x = -2\) iz četvrtog slučaja ne pripadaju odgovarajućim intervalima -- moraju se odbaciti.
  • Greška u aritmetici razlomaka: Zbir \(4 + \left(-\dfrac{8}{3}\right) = \dfrac{12-8}{3} = \dfrac{4}{3}\), ne \(\dfrac{12+8}{3}\).

Dodatni izazov

Rešiti sličnu jednačinu:

\[|x - 1| + |x + 3| = |x - 2| + 5\]
Prikaži nagoveštaj

Kritične tačke su \(x = -3\), \(x = 1\) i \(x = 2\). Razmatraj četiri intervala i u svakom rešavaj linearnu jednačinu.

Prikaži rešenje

Za \(x \geq 2\): \((x-1) + (x+3) = (x-2) + 5\) daje \(x = 1\), ali \(1 \not\geq 2\) -- nevalidno.

Za \(1 \leq x < 2\): \((x-1) + (x+3) = -(x-2) + 5\) daje \(3x = 5\), tj. \(x = \frac{5}{3}\). Pošto \(1 \leq \frac{5}{3} < 2\) -- validno.

Za \(-3 \leq x < 1\): \(-(x-1) + (x+3) = -(x-2) + 5\) daje \(4 = -x+7\), tj. \(x = 3\), ali \(3 \not\in [-3, 1)\) -- nevalidno.

Za \(x < -3\): \(-(x-1) + (-(x+3)) = -(x-2) + 5\) daje \(-x-2 = -x+7\), tj. \(-2 = 7\) -- nemoguće.

Jedino rešenje je \(x = \frac{5}{3}\).
