Trigonometrijska jednačina na datom intervalu

Trigonometrija — rešavanje jednačine oblika \(a\sin x + b\cos x = c\) pomoću pomoćnog ugla

📋 Tekst zadatka

Broj rešenja jednačine \(1 + \sin x + \cos x = 0\) na intervalu \((-\pi, \pi)\) je:

(A)
\(4\)
(B)
\(1\)
(C)
\(2\)
(D)
\(3\)
(E)
\(0\)
🎯 Plan rešavanja

Prebacićemo jednačinu u oblik \(\sin x + \cos x = -1\), zatim pomoću metode pomoćnog ugla svesti na jednačinu \(\sin(x + \frac{\pi}{4}) = -\frac{\sqrt{2}}{2}\). Potom ćemo naći sva rešenja i proveriti koja pripadaju otvorenom intervalu \((-\pi, \pi)\).

📚 Podsetnik iz teorije
Ključne formule i teoreme (klikni da otvoriš)

Metoda pomoćnog ugla:

\[ a\sin x + b\cos x = \sqrt{a^2 + b^2}\,\sin\!\left(x + \varphi\right) \]

gde je \(\varphi\) ugao za koji važi \(\cos\varphi = \frac{a}{\sqrt{a^2+b^2}}\) i \(\sin\varphi = \frac{b}{\sqrt{a^2+b^2}}\).

Specijalan slučaj za \(a = b = 1\):

\[ \sin x + \cos x = \sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right) \]

Opšte rešenje jednačine \(\sin\alpha = c\):

\[ \alpha = \alpha_0 + 2k\pi \quad \text{ili} \quad \alpha = \pi - \alpha_0 + 2k\pi, \quad k \in \mathbb{Z} \]

gde je \(\alpha_0\) partikularno rešenje iz intervala \(\left[-\frac{\pi}{2}, \frac{\pi}{2}\right]\).

Poznate vrednosti: \(\sin\!\left(-\frac{\pi}{4}\right) = -\frac{\sqrt{2}}{2}\)

📊 Grafički prikaz rešenja

Grafik funkcije \(f(x) = 1 + \sin x + \cos x\) na intervalu \((-\pi, \pi)\) — nule funkcije su rešenja jednačine

🔢 Rešenje korak po korak
1
Prebacimo jednačinu u pogodniji oblik

Polazna jednačina je:

\[ 1 + \sin x + \cos x = 0 \]

Prebacimo jedinicu na desnu stranu:

\[ \sin x + \cos x = -1 \]
2
Primenimo metodu pomoćnog ugla

Za zbir \(\sin x + \cos x\) koristimo identitet sa pomoćnim uglom \(\frac{\pi}{4}\):

\[ \sin x + \cos x = \sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right) \]

Ovo važi jer:

\[ \sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right) = \sqrt{2}\left(\sin x \cdot \cos\frac{\pi}{4} + \cos x \cdot \sin\frac{\pi}{4}\right) = \sqrt{2}\left(\frac{\sqrt{2}}{2}\sin x + \frac{\sqrt{2}}{2}\cos x\right) = \sin x + \cos x \]
3
Svedemo na osnovnu sinusnu jednačinu

Zamenom u jednačinu \(\sin x + \cos x = -1\) dobijamo:

\[ \sqrt{2}\,\sin\!\left(x + \frac{\pi}{4}\right) = -1 \]

Podelimo obe strane sa \(\sqrt{2}\):

\[ \sin\!\left(x + \frac{\pi}{4}\right) = -\frac{1}{\sqrt{2}} = -\frac{\sqrt{2}}{2} \]
Proveri sebe: Koliko iznosi \(\sin\!\left(-\frac{\pi}{4}\right)\)?
\(\sin\!\left(-\frac{\pi}{4}\right) = -\frac{\sqrt{2}}{2}\). Tačno, to je upravo vrednost koju tražimo!
4
Nađemo opšta rešenja

Uvedemo smenu \(\alpha = x + \frac{\pi}{4}\). Jednačina postaje:

\[ \sin\alpha = -\frac{\sqrt{2}}{2} \]

Partikularno rešenje je \(\alpha_0 = -\frac{\pi}{4}\). Opšta rešenja su:

\[ \alpha = -\frac{\pi}{4} + 2k\pi \quad \text{ili} \quad \alpha = \pi - \left(-\frac{\pi}{4}\right) + 2k\pi = \frac{5\pi}{4} + 2k\pi \]

gde je \(k \in \mathbb{Z}\).

5
Vratimo se na promenljivu \(x\)

Pošto je \(\alpha = x + \frac{\pi}{4}\), imamo \(x = \alpha - \frac{\pi}{4}\):

\[ x = -\frac{\pi}{4} - \frac{\pi}{4} + 2k\pi = -\frac{\pi}{2} + 2k\pi \]
\[ x = \frac{5\pi}{4} - \frac{\pi}{4} + 2k\pi = \pi + 2k\pi \]
6
Proverimo koja rešenja pripadaju intervalu \((-\pi, \pi)\)

Interval je otvoren, dakle tačke \(-\pi\) i \(\pi\) ne pripadaju intervalu.

Prva familija: \(x = -\frac{\pi}{2} + 2k\pi\)

\[ k = 0: \quad x = -\frac{\pi}{2} \approx -1{,}571 \quad \Rightarrow \quad -\pi < -\frac{\pi}{2} < \pi \quad \checkmark \]
\[ k = 1: \quad x = -\frac{\pi}{2} + 2\pi = \frac{3\pi}{2} \approx 4{,}712 > \pi \quad \times \]
\[ k = -1: \quad x = -\frac{\pi}{2} - 2\pi = -\frac{5\pi}{2} \approx -7{,}854 < -\pi \quad \times \]

Druga familija: \(x = \pi + 2k\pi\)

\[ k = 0: \quad x = \pi \quad \Rightarrow \quad \pi \notin (-\pi, \pi) \quad \times \]
\[ k = -1: \quad x = \pi - 2\pi = -\pi \quad \Rightarrow \quad -\pi \notin (-\pi, \pi) \quad \times \]
Proveri sebe: Zašto \(x = \pi\) nije rešenje na datom intervalu?
Zato što je interval otvoren: \((-\pi, \pi)\). To znači da krajnje tačke \(-\pi\) i \(\pi\) nisu uključene. Vrednost \(x = \pi\) jeste rešenje jednačine (jer \(1 + \sin\pi + \cos\pi = 1 + 0 - 1 = 0\)), ali ne pripada otvorenom intervalu.
7
Verifikacija rešenja \(x = -\frac{\pi}{2}\)

Uvrstimo \(x = -\frac{\pi}{2}\) u polaznu jednačinu:

\[ 1 + \sin\!\left(-\frac{\pi}{2}\right) + \cos\!\left(-\frac{\pi}{2}\right) = 1 + (-1) + 0 = 0 \quad \checkmark \]

Rešenje zadovoljava jednačinu.

💡 Ključni uvid

Druga familija rešenja \(x = \pi + 2k\pi\) daje samo krajnje tačke intervala (\(\pi\) i \(-\pi\)), koje su isključene jer je interval otvoren. Da je interval bio zatvoren \([-\pi, \pi]\), imali bismo dva rešenja. Ovako, jedino rešenje na \((-\pi, \pi)\) je:

\[ x = -\frac{\pi}{2} \]

Ovo je lep primer gde tip intervala (otvoren ili zatvoren) suštinski utiče na broj rešenja!

Konačan odgovor
Jednačina ima \(1\) rešenje na \((-\pi, \pi)\)
(A) \(4\) (B) \(1\) (C) \(2\) (D) \(3\) (E) \(0\)

Verifikacija: \(1 + \sin\!\left(-\frac{\pi}{2}\right) + \cos\!\left(-\frac{\pi}{2}\right) = 1 - 1 + 0 = 0\). Jedino rešenje \(x = -\frac{\pi}{2}\) zadovoljava jednačinu. Tačke \(x = \pm\pi\) zadovoljavaju jednačinu ali ne pripadaju otvorenom intervalu.

⚠️ Česte greške
✗

Zanemarivanje tipa intervala: Mnogi učenici ne obrate pažnju da je interval otvoren \((-\pi, \pi)\) i ubroje i \(x = \pi\) (eventualno i \(x = -\pi\)) kao rešenja, dobijajući odgovor 2 ili 3 umesto 1. Uvek proverite da li krajnje tačke pripadaju intervalu!

✗

Greška pri deljenju sa \(\sqrt{2}\): Neki učenici zaborave da podele desnu stranu sa \(\sqrt{2}\) i pokušavaju da reše \(\sin\alpha = -1\) umesto \(\sin\alpha = -\frac{\sqrt{2}}{2}\), čime gube rešenja ili ih pogrešno nalaze.

🏆 Dodatni izazov

Koliko rešenja ima jednačina \(1 + \sin x + \cos x = 0\) na zatvorenom intervalu \([-\pi, \pi]\)?

Pogledaj nagoveštaj

Na zatvorenom intervalu, krajnje tačke \(-\pi\) i \(\pi\) su uključene. Proverite: \(1 + \sin(\pi) + \cos(\pi) = 1 + 0 - 1 = 0\) i \(1 + \sin(-\pi) + \cos(-\pi) = 1 + 0 - 1 = 0\). Obe krajnje tačke su rešenja! Ali pažnja — da li je \(-\pi\) i \(\pi\) zapravo ista tačka na trigonometrijskom krugu?
