Nejednakost sa parametrom

Algebra - nejednakosti sa parametrom, analiza kvadratne funkcije

📋 Tekst zadatka

Nejednakost

\[ \frac{x + a}{x^2 + x + 1} < \frac{x}{x^2 + 2x + 3} \]

je tačna za svako realno \( x \) ako i samo ako je:

(A)
\( a < -2 \)
(B)
\( -1 < a < -\dfrac{1}{2} \)
(C)
\( a \leq -1 \)
(D)
\( -\dfrac{1}{2} < a < \dfrac{1}{2} \)
(E)
\( -\infty < a < +\infty \)
🎯 Plan rešavanja

Proverićemo da su oba imenioca uvek pozitivna, pa ćemo pomnožiti nejednakost unakrsno. Nakon razvijanja i sređivanja, dobićemo kvadratnu nejednakost po \( x \) sa parametrom \( a \). Analiziraćemo kada ta nejednakost važi za svako realno \( x \).

📚 Podsetnik iz teorije
Ključne formule i teoreme (klikni da otvoriš)

Znak kvadratnog trinoma: Za \( f(x) = ax^2 + bx + c \) sa \( a > 0 \), važi \( f(x) > 0 \) za svako \( x \) ako i samo ako je diskriminanta \( D = b^2 - 4ac < 0 \).

\[ ax^2 + bx + c > 0, \;\forall x \in \mathbb{R} \iff a > 0 \text{ i } D < 0 \]

Kvadratna nejednakost: \( ax^2 + bx + c < 0 \) važi za svako \( x \) ako i samo ako je \( a < 0 \) i \( D < 0 \), ili je \( a = 0 \) i \( b = 0 \) i \( c < 0 \).

\[ ax^2 + bx + c < 0, \;\forall x \in \mathbb{R} \iff a < 0 \text{ i } D < 0, \text{ ili } a = 0, b = 0, c < 0 \]

Dopuna do potpunog kvadrata:

\[ ax^2 + 2abx = a(x + b)^2 - ab^2 \]
📊 Interaktivni dijagram
a = -1,0

Pomeranjem klizača menjate vrednost parametra \( a \) i posmatrate grafik funkcije \( f(x) = (1+a)x^2 + 2(1+a)x + 3a \). Nejednakost važi za svako \( x \) kad je ceo grafik ispod \( x \)-ose.

🔢 Rešenje korak po korak
1
Proverimo znakove imenilaca

Pre bilo kakve manipulacije, moramo utvrditi da su oba imenioca pozitivna za svako realno \( x \).

Prvi imenilac: \( x^2 + x + 1 \)

\[ D_1 = 1^2 - 4 \cdot 1 \cdot 1 = 1 - 4 = -3 < 0 \]

Pošto je vodeći koeficijent pozitivan i \( D_1 < 0 \), važi \( x^2 + x + 1 > 0 \) za svako \( x \).

Drugi imenilac: \( x^2 + 2x + 3 \)

\[ D_2 = 2^2 - 4 \cdot 1 \cdot 3 = 4 - 12 = -8 < 0 \]

Isto tako, \( x^2 + 2x + 3 > 0 \) za svako \( x \).

2
Pomnožimo nejednakost unakrsno

Pošto su oba imenioca strogo pozitivna, možemo bezbedno pomnožiti obe strane njima bez promene znaka nejednakosti:

\[ (x + a)(x^2 + 2x + 3) < x(x^2 + x + 1) \]
Proveri sebe: Zašto smemo pomnožiti unakrsno?
Kada množimo nejednakost pozitivnim brojem, znak nejednakosti se ne menja. Pošto su oba imenioca \( x^2 + x + 1 > 0 \) i \( x^2 + 2x + 3 > 0 \) za svako \( x \), proizvod je pozitivan i množenje je ekvivalentna transformacija.
3
Razvijemo levu stranu

Množimo \( (x + a) \) sa \( (x^2 + 2x + 3) \):

\[ \begin{align} (x + a)(x^2 + 2x + 3) &= x \cdot x^2 + x \cdot 2x + x \cdot 3 + a \cdot x^2 + a \cdot 2x + a \cdot 3 \\ &= x^3 + 2x^2 + 3x + ax^2 + 2ax + 3a \end{align} \]

Grupišemo po stepenima \( x \):

\[ x^3 + (2 + a)x^2 + (3 + 2a)x + 3a \]
4
Razvijemo desnu stranu
\[ x(x^2 + x + 1) = x^3 + x^2 + x \]
5
Prebacimo sve na jednu stranu

Nejednakost je ekvivalentna sa: leva - desna \( < 0 \):

\[ x^3 + (2+a)x^2 + (3+2a)x + 3a - x^3 - x^2 - x < 0 \]

Skratimo \( x^3 \) i grupišemo:

\[ (1+a)x^2 + (2+2a)x + 3a < 0 \]

Uočimo da možemo izlučiti faktor 2 iz srednjeg člana:

\[ (1+a)x^2 + 2(1+a)x + 3a < 0 \]
6
Dopunimo do potpunog kvadrata

Označimo \( f(x) = (1+a)x^2 + 2(1+a)x + 3a \). Dopunimo prva dva člana do potpunog kvadrata:

\[ f(x) = (1+a)\bigl(x^2 + 2x\bigr) + 3a \]

Dodajemo i oduzimamo 1 unutar zagrade:

\[ f(x) = (1+a)\bigl(x^2 + 2x + 1 - 1\bigr) + 3a = (1+a)(x+1)^2 - (1+a) + 3a \]

Sredimo slobodni član:

\[ -(1+a) + 3a = -1 - a + 3a = 2a - 1 \]

Dakle:

\[ f(x) = (1+a)(x+1)^2 + (2a - 1) \]
Proveri sebe: Koliko iznosi \( f(-1) \)?
\( f(-1) = (1+a) \cdot 0 + (2a - 1) = 2a - 1 \). Ovo je vrednost funkcije u temenu (ako je \( 1 + a \neq 0 \)), tj. maksimum ili minimum parabole.
7
Analiziramo uslove po slučajevima

Tražimo da \( f(x) = (1+a)(x+1)^2 + (2a-1) < 0 \) važi za svako realno \( x \). Kako je \( (x+1)^2 \geq 0 \), razlikujemo tri slučaja:

Slučaj 1: \( 1 + a > 0 \) (tj. \( a > -1 \))

Tada \( (1+a)(x+1)^2 \) raste neograničeno kad \( x \to \pm\infty \), pa \( f(x) \to +\infty \). Nejednakost ne može da važi za svako \( x \).

Slučaj 2: \( 1 + a = 0 \) (tj. \( a = -1 \))

Tada je \( f(x) = 0 \cdot (x+1)^2 + (2 \cdot (-1) - 1) = -3 \).

\[ f(x) = -3 < 0, \quad \forall x \in \mathbb{R} \quad \checkmark \]

Slučaj 3: \( 1 + a < 0 \) (tj. \( a < -1 \))

Tada \( (1+a) < 0 \) i \( (x+1)^2 \geq 0 \), pa \( (1+a)(x+1)^2 \leq 0 \).

Maksimum se dostiže za \( x = -1 \), gde je \( (x+1)^2 = 0 \), pa:

\[ \max_{x} f(x) = f(-1) = 0 + (2a - 1) = 2a - 1 \]

Za \( a < -1 \): \( 2a - 1 < 2(-1) - 1 = -3 < 0 \).

Dakle \( f(x) \leq 2a - 1 < 0 \) za svako \( x \). Nejednakost važi!

8
Zaključak

Spajanjem slučajeva 2 i 3, nejednakost važi za svako realno \( x \) ako i samo ako:

\[ \boxed{a \leq -1} \]
Proveri sebe: Šta se dešava za granični slučaj \( a = -1 \)?
Za \( a = -1 \), polazna nejednakost postaje \( \frac{x - 1}{x^2 + x + 1} < \frac{x}{x^2 + 2x + 3} \), što je ekvivalentno sa \( -3 < 0 \). To je uvek tačno, pa \( a = -1 \) zadovoljava uslov.
💡 Ključni uvid

Trik ovog zadatka leži u tome da oba imenioca \( x^2 + x + 1 \) i \( x^2 + 2x + 3 \) imaju negativne diskriminante, pa su uvek pozitivni. To nam omogućava bezuslovno unakrsno množenje. Nakon toga, problem se svodi na pitanje: kada je kvadratni (ili linearni) izraz strogo negativan za svako realno \( x \)? Dopuna do potpunog kvadrata razotkriva da funkcija dostiže maksimum u tački \( x = -1 \), a taj maksimum je \( 2a - 1 \), koji mora biti negativan.

Konačan odgovor
\( a \leq -1 \)
(A) \( a < -2 \) (B) \( -1 < a < -\dfrac{1}{2} \) (C) \( a \leq -1 \) (D) \( -\dfrac{1}{2} < a < \dfrac{1}{2} \) (E) \( -\infty < a < +\infty \)

Verifikacija: Za \( a = -1 \), nejednakost se svodi na \( -3 < 0 \) (tačno). Za \( a = -2 \), dobijamo \( -(x+1)^2 - 5 < 0 \) (tačno). Za \( a = 0 \), dobijamo \( (x+1)^2 - 1 \), što za \( x = 0 \) daje 0, pa nejednakost ne važi. Samo opcija (C) obuhvata sve ispravne vrednosti.

⚠️ Česte greške
✗

Zaboravljanje provere imenilaca: Unakrsno množenje nejednakosti razlomaka zahteva poznavanje znaka imenilaca. Da su imenioci mogli biti negativni, nejednakost bi mogla promeniti smer, i čitav pristup bi bio pogrešan.

✗

Isključivanje graničnog slučaja \( a = -1 \): Opcija (A) daje \( a < -2 \), koja je podskup tačnog rešenja ali propušta vrednosti \( -2 \leq a \leq -1 \). Posebno, za \( a = -1 \) funkcija je konstantno \( -3 \), što je strogo negativno.

🏆 Dodatni izazov

Za koje vrednosti parametra \( a \) nejednakost \( \dfrac{x + a}{x^2 + x + 1} \leq \dfrac{x}{x^2 + 2x + 3} \) (sa znakom \( \leq \) umesto \( < \)) važi za svako realno \( x \)?

Pogledaj nagoveštaj

Sada tražimo \( f(x) = (1+a)(x+1)^2 + (2a-1) \leq 0 \) za svako \( x \). Za \( a < -1 \) i \( a = -1 \), ovo važi kao i pre. Ali, razmislite: da li postoji neka vrednost \( a > -1 \) za koju bi \( f(x) \leq 0 \) moglo da važi za svako \( x \)? Odgovor: ne, jer za \( a > -1 \) funkcija \( f \) raste bez ograničenja. Dakle, uslov ostaje isti: \( a \leq -1 \).
