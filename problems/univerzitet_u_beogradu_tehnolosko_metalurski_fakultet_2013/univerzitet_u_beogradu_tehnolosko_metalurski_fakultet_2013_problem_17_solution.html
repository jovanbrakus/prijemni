Logaritamska jednacina sa kompozicijom logaritama

Resavanje ugnjezdenih logaritama razlicitih osnova

Postavka zadatka

Resenje jednacine

\[ \log_4\!\left(\log_2\!\left(\log_{\frac{1}{3}} x\right)\right) = 0 \]

nalazi se u intervalu:

(A) \(\left(0,\;\dfrac{1}{8}\right)\)
(B) \(\left(\dfrac{1}{8},\;\dfrac{1}{6}\right)\)
(C) \(\left(\dfrac{1}{6},\;\dfrac{1}{2}\right)\)
(D) \(\left(\dfrac{1}{2},\;1\right)\)
(E) \((1,\;5)\)

Plan resavanja

Jednacinu resavamo „spolja ka unutra“: polazeci od najspoljasnjeg logaritma, koristimo definiciju logaritma da bismo u svakom koraku eliminisali po jedan sloj. Zatim odredjujemo u koji interval pada dobijeno resenje.

Podsetnik iz teorije

Korisne cinjenice za ovaj zadatak:

\[\log_a b = c \;\;\Longleftrightarrow\;\; a^c = b \quad (a > 0,\; a \neq 1,\; b > 0)\]
\[\log_a 1 = 0 \quad \text{za svako } a > 0,\; a \neq 1\]
\[\log_a\!\left(a^k\right) = k\]
\[\text{Ako je } 0 < a < 1, \text{ onda je } \log_a x \text{ opadajuca funkcija.}\]

Kljucna ideja: svaki logaritam „odmotavamo“ pomocu definicije \(\log_a b = c \Leftrightarrow b = a^c\), radeci od spoljasnjeg ka unutrasnjem.

Vizuelni prikaz — odmotavanje slojeva

Resenje korak po korak

1

Spoljasnji logaritam: \(\log_4(\ldots) = 0\)

Polazimo od jednacine:

\[\log_4\!\left(\log_2\!\left(\log_{\frac{1}{3}} x\right)\right) = 0\]

Po definiciji logaritma, \(\log_4 y = 0\) znaci \(y = 4^0 = 1\). Dakle:

\[\log_2\!\left(\log_{\frac{1}{3}} x\right) = 1\]
Proveri sebe: Zasto je \(4^0 = 1\)?
Svaki broj razlicit od nule stepenovan nulom daje 1. To je osnovno pravilo stepenovanja: \(a^0 = 1\) za \(a \neq 0\).
2

Srednji logaritam: \(\log_2(\ldots) = 1\)

Sada imamo:

\[\log_2\!\left(\log_{\frac{1}{3}} x\right) = 1\]

Po definiciji, \(\log_2 y = 1\) znaci \(y = 2^1 = 2\). Dakle:

\[\log_{\frac{1}{3}} x = 2\]
Proveri sebe: Zasto argument logaritma mora biti pozitivan?
Da bi \(\log_2\!\left(\log_{\frac{1}{3}} x\right)\) imao smisla, mora vaziti \(\log_{\frac{1}{3}} x > 0\). Videcemo da je \(\log_{\frac{1}{3}} x = 2 > 0\), pa je uslov zadovoljen.
3

Unutrasnji logaritam: \(\log_{\frac{1}{3}} x = 2\)

Preostaje nam:

\[\log_{\frac{1}{3}} x = 2\]

Po definiciji, \(\log_{\frac{1}{3}} x = 2\) znaci \(x = \left(\frac{1}{3}\right)^2\). Racunamo:

\[x = \left(\frac{1}{3}\right)^2 = \frac{1}{9}\]
4

Odredjivanje intervala

Dobili smo \(x = \dfrac{1}{9}\). Sada treba utvrditi u koji od ponudjenih intervala spada ova vrednost.

Poredimo \(\dfrac{1}{9}\) sa granicama intervala iz odgovora (A):

\[0 < \frac{1}{9} < \frac{1}{8}\]

Ovo vazi jer je \(\dfrac{1}{9} \approx 0{,}111\), a \(\dfrac{1}{8} = 0{,}125\). Dakle:

\[\frac{1}{9} \in \left(0,\;\frac{1}{8}\right)\]
Proveri sebe: Kako poredimo razlomke \(\frac{1}{9}\) i \(\frac{1}{8}\)?
Za pozitivne razlomke oblika \(\frac{1}{a}\) i \(\frac{1}{b}\), veci imenilac daje manji razlomak. Posto je \(9 > 8\), sledi \(\frac{1}{9} < \frac{1}{8}\).
5

Verifikacija

Proverimo resenje ubacivanjem \(x = \frac{1}{9}\) u polaznu jednacinu:

\[\log_{\frac{1}{3}} \frac{1}{9} = \log_{\frac{1}{3}} \left(\frac{1}{3}\right)^2 = 2\]
\[\log_2 2 = 1\]
\[\log_4 1 = 0 \;\;\checkmark\]

Kljucni uvid

Kada imamo kompoziciju vise logaritama, jednacinu resavamo od spoljasnjeg ka unutrasnjem sloju. Svaki put primenjujemo definiciju logaritma: \(\log_a b = c \Leftrightarrow b = a^c\). Ovim postupkom se u svakom koraku uklanja jedan logaritam, dok ne ostane prosta jednacina.

Konacan odgovor

\(x = \dfrac{1}{9} \in \left(0,\;\dfrac{1}{8}\right)\)
(A) \(\left(0,\;\frac{1}{8}\right)\)
(B) \(\left(\frac{1}{8},\;\frac{1}{6}\right)\)
(C) \(\left(\frac{1}{6},\;\frac{1}{2}\right)\)
(D) \(\left(\frac{1}{2},\;1\right)\)
(E) \((1,\;5)\)

Verifikacija: \(\log_4\!\left(\log_2\!\left(\log_{\frac{1}{3}} \frac{1}{9}\right)\right) = \log_4(\log_2 2) = \log_4 1 = 0\;\;\checkmark\)

Ceste greske

  • Zameniti redosled odmotavanja — krenuti od unutrasnjeg umesto od spoljasnjeg logaritma, sto dovodi do mnogo komplikovanijeg postupka.
  • Zaboraviti da je osnova \(\frac{1}{3}\) manja od 1, pa pomisliti da \(\log_{\frac{1}{3}} x = 2\) nema resenje. Zapravo, \(\left(\frac{1}{3}\right)^2 = \frac{1}{9}\) je sasvim validan rezultat.
  • Pogresan racun pri poredjenju \(\frac{1}{9}\) i \(\frac{1}{8}\) — veci imenilac daje manji razlomak, pa je \(\frac{1}{9} < \frac{1}{8}\), a ne obrnuto.
  • Ne proveriti uslove definisanosti: argumenti svih logaritama moraju biti pozitivni. U ovom slucaju su svi uslovi zadovoljeni.

Izazov za razmisljanje

Sta ako se jednacina promeni u \(\log_4\!\left(\log_2\!\left(\log_{\frac{1}{3}} x\right)\right) = \frac{1}{2}\)? U kom intervalu bi se tada nalazilo resenje?

Pogledaj nagovest
Sada bi \(\log_2(\log_{1/3} x) = 4^{1/2} = 2\), pa \(\log_{1/3} x = 2^2 = 4\), i konacno \(x = (1/3)^4 = \frac{1}{81}\). Posto je \(\frac{1}{81} \approx 0{,}0123 < \frac{1}{8} = 0{,}125\), resenje bi i dalje bilo u intervalu \(\left(0,\;\frac{1}{8}\right)\).
