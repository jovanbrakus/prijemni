Postavka zadatka

Ako je \( \text{tg}\, x = 2 \), onda je vrednost izraza \( \sin^4 x + \cos^4 x \) jednaka:

Dato
\( \text{tg}\, x = 2 \)
Trazi se
\( \sin^4 x + \cos^4 x \)

Ponudjeni odgovori

(A)
\( \dfrac{4}{25} \)
(B)
\( \dfrac{7}{25} \)
(C)
\( \dfrac{4}{5} \)
(D)
\( \dfrac{17}{25} \)
(E)
\( 1 \)

Plan resavanja

Iz uslova \( \text{tg}\, x = 2 \) izrazicemo \( \sin x \) preko \( \cos x \), zatim pomocu osnovnog trigonometrijskog identiteta odrediti \( \sin^2 x \) i \( \cos^2 x \). Na kraju cemo izracunati \( \sin^4 x + \cos^4 x \) kvadriranjem tih vrednosti.

Kljucne formule i teoreme
  • Osnovni trigonometrijski identitet: \[ \sin^2 x + \cos^2 x = 1 \]
  • Definicija tangensa: \[ \text{tg}\, x = \frac{\sin x}{\cos x} \] Odatle sledi: \( \sin x = \text{tg}\, x \cdot \cos x \).
  • Algebarski identitet za zbir kvadrata: \[ a^2 + b^2 = (a + b)^2 - 2ab \] Primenjen na \( \sin^2 x \) i \( \cos^2 x \): \[ \sin^4 x + \cos^4 x = (\sin^2 x + \cos^2 x)^2 - 2\sin^2 x \cos^2 x = 1 - 2\sin^2 x \cos^2 x \]
  • Veza sa tangensom: Iz \( 1 + \text{tg}^2 x = \frac{1}{\cos^2 x} \) sledi: \[ \cos^2 x = \frac{1}{1 + \text{tg}^2 x} \]

Graficki prikaz

Pravougli trougao sa tangensom 2. Pomerajte slider da vidite kako se menja ugao \( x \) i vrednost izraza.

Resenje korak po korak

1
Izrazavamo \( \sin x \) pomocu \( \cos x \)

Iz definicije tangensa:

\[ \text{tg}\, x = \frac{\sin x}{\cos x} = 2 \]

sledi:

\[ \sin x = 2\cos x \]
2
Odredjujemo \( \cos^2 x \) pomocu osnovnog identiteta

Zamenjujemo \( \sin x = 2\cos x \) u osnovni trigonometrijski identitet:

\[ \sin^2 x + \cos^2 x = 1 \]
\[ (2\cos x)^2 + \cos^2 x = 1 \]
\[ 4\cos^2 x + \cos^2 x = 1 \]
\[ 5\cos^2 x = 1 \]

Delimo obe strane sa 5:

\[ \cos^2 x = \frac{1}{5} \]
3
Odredjujemo \( \sin^2 x \)

Iz osnovnog identiteta:

\[ \sin^2 x = 1 - \cos^2 x = 1 - \frac{1}{5} = \frac{4}{5} \]

Provera: \( \sin^2 x = (2\cos x)^2 = 4\cos^2 x = 4 \cdot \frac{1}{5} = \frac{4}{5} \). Slaze se.

Proveri se: Koliko je \( \sin^2 x + \cos^2 x \)?

\( \frac{4}{5} + \frac{1}{5} = \frac{5}{5} = 1 \). Tacno, osnovni identitet je zadovoljen.

4
Racunamo \( \sin^4 x \) i \( \cos^4 x \)

Kvadriramo obe vrednosti:

\[ \sin^4 x = \left(\sin^2 x\right)^2 = \left(\frac{4}{5}\right)^2 = \frac{16}{25} \]
\[ \cos^4 x = \left(\cos^2 x\right)^2 = \left(\frac{1}{5}\right)^2 = \frac{1}{25} \]
5
Sabiramo i dobijamo konacnu vrednost

Sabiramo dobijene vrednosti:

\[ \sin^4 x + \cos^4 x = \frac{16}{25} + \frac{1}{25} = \frac{17}{25} \]
Proveri se: Alternativni nacin preko algebarskog identiteta

Mozemo koristiti i identitet \( \sin^4 x + \cos^4 x = 1 - 2\sin^2 x \cos^2 x \).

\( \sin^2 x \cdot \cos^2 x = \frac{4}{5} \cdot \frac{1}{5} = \frac{4}{25} \)

\( 1 - 2 \cdot \frac{4}{25} = 1 - \frac{8}{25} = \frac{25 - 8}{25} = \frac{17}{25} \). Isti rezultat!

Kljucni uvid

Kada znamo tangens ugla, mozemo odrediti \( \sin^2 x \) i \( \cos^2 x \) bez poznavanja samog ugla \( x \). Kljuc je zamena \( \sin x = \text{tg}\, x \cdot \cos x \) u Pitagorin identitet, cime dobijamo jednacinu samo po \( \cos^2 x \).

\[ \text{tg}\, x = 2 \;\;\Longrightarrow\;\; \cos^2 x = \frac{1}{5}, \;\; \sin^2 x = \frac{4}{5} \;\;\Longrightarrow\;\; \sin^4 x + \cos^4 x = \frac{17}{25} \]

Opcenito, za \( \text{tg}\, x = t \) vazi: \( \cos^2 x = \frac{1}{1+t^2} \) i \( \sin^2 x = \frac{t^2}{1+t^2} \). Ovo je korisna formula za mnoge zadatke ovog tipa.

Konacan odgovor

\( \dfrac{17}{25} \)

Vrednost izraza \( \sin^4 x + \cos^4 x \) za \( \text{tg}\, x = 2 \) je \( \dfrac{17}{25} \).

(A) 4/25
(B) 7/25
(C) 4/5
(D) 17/25
(E) 1

Verifikacija: \( \sin^2 x = \frac{4}{5} \) i \( \cos^2 x = \frac{1}{5} \) zadovoljavaju \( \sin^2 x + \cos^2 x = 1 \), a \( \frac{\sin^2 x}{\cos^2 x} = \frac{4/5}{1/5} = 4 = \text{tg}^2 x \). Takodje, \( \frac{16}{25} + \frac{1}{25} = \frac{17}{25} \). Sve se slaze.

Ceste greske

  • Brkanje \( \sin^4 x + \cos^4 x \) sa \( (\sin^2 x + \cos^2 x)^2 \): Izraz \( \sin^4 x + \cos^4 x \) nije isto sto i \( (\sin^2 x + \cos^2 x)^2 = 1 \). Razlika je u clanu \( -2\sin^2 x \cos^2 x \). Ako zaboravite ovu razliku, dobicete odgovor E) 1, sto nije tacno.
  • Racunanje \( \sin^4 x \) kao \( (\sin x)^4 \) umesto \( (\sin^2 x)^2 \): Neophodno je prvo odrediti \( \sin^2 x \) i \( \cos^2 x \), pa ih tek onda kvadrirati. Pokusaj da se direktno izracuna \( \sin x = \frac{2}{\sqrt{5}} \) pa podigne na cetvrti stepen vodi ka komplikovanijim racunima i mogucnosti greske sa korenom.

Dodatni izazov

Ako je \( \text{tg}\, x = 3 \), kolika je vrednost izraza \( \sin^6 x + \cos^6 x \)?

Hint

Koristite identitet \( a^3 + b^3 = (a + b)(a^2 - ab + b^2) \) za \( a = \sin^2 x \) i \( b = \cos^2 x \). Dakle: \( \sin^6 x + \cos^6 x = (\sin^2 x + \cos^2 x)(\sin^4 x - \sin^2 x \cos^2 x + \cos^4 x) = 1 \cdot (\sin^4 x + \cos^4 x - \sin^2 x \cos^2 x) \). Za \( \text{tg}\, x = 3 \): \( \cos^2 x = \frac{1}{10} \), \( \sin^2 x = \frac{9}{10} \).
