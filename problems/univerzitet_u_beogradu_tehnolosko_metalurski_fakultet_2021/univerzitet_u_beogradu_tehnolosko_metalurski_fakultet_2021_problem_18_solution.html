Ekstremne vrednosti trigonometrijske funkcije

Trigonometrija i kvadratna funkcija — nalaženje razlike najveće i najmanje vrednosti

Tekst zadatka

Razlika najveće i najmanje vrednosti funkcije

\[ y = \sin x + \cos^2 x + 1 \]

jednaka je:

(A)
\(\dfrac{9}{4}\)
(B)
\(1\)
(C)
\(\dfrac{1}{2}\)
(D)
\(2\)
(E)
\(\dfrac{1}{3}\)

Plan rešavanja

Iskoristićemo identitet \(\cos^2 x = 1 - \sin^2 x\) da prevedemo funkciju na kvadratnu po \(\sin x\). Zatim ćemo naći maksimum i minimum te kvadratne funkcije na intervalu \([-1, 1]\) i izračunati njihovu razliku.

📐 Ključne formule i teoreme
  • Osnovni trigonometrijski identitet: \(\sin^2 x + \cos^2 x = 1\), odakle je \(\cos^2 x = 1 - \sin^2 x\).
  • Opseg sinusa: \(\sin x \in [-1, 1]\) za svako \(x \in \mathbb{R}\).
  • Kvadratna funkcija \(f(t) = at^2 + bt + c\) ima teme u \(t_0 = -\dfrac{b}{2a}\). Ako je \(a < 0\), teme je maksimum.
  • Kanonski oblik: \(f(t) = a(t - t_0)^2 + f(t_0)\), gde je \(f(t_0)\) vrednost u temenu.

Grafički prikaz

Pomerajte klizač ispod da vidite kako se menja vrednost funkcije za različite uglove

Rešenje korak po korak

1

Uvodimo smenu

Neka je \(t = \sin x\). Pošto sinus uzima vrednosti od \(-1\) do \(1\), važi:

\[ t \in [-1,\; 1] \]
2

Primenjujemo trigonometrijski identitet

Zamenjujemo \(\cos^2 x\) koristeći identitet \(\cos^2 x = 1 - \sin^2 x = 1 - t^2\):

\[ y = t + (1 - t^2) + 1 \]
3

Sređujemo izraz

Sređujemo i dobijamo kvadratnu funkciju po \(t\):

\[ y = -t^2 + t + 2 \]

Koeficijent uz \(t^2\) je negativan (\(a = -1\)), pa je parabola okrenuta nadole.

🔍 Proveri se: Koliko iznosi \(y\) za \(t = 0\)?

\(y = -(0)^2 + 0 + 2 = 2\)

4

Nalazimo teme parabole

Za funkciju \(f(t) = -t^2 + t + 2\), teme se nalazi u:

\[ t_0 = -\frac{b}{2a} = -\frac{1}{2 \cdot (-1)} = \frac{1}{2} \]

Vrednost \(t_0 = \dfrac{1}{2}\) pripada intervalu \([-1, 1]\), pa je teme dostupno.

5

Zapisujemo kanonski oblik

Dopunjavanjem do potpunog kvadrata:

\[ \begin{align} y &= -(t^2 - t) + 2 \\ &= -\!\left(t^2 - t + \frac{1}{4} - \frac{1}{4}\right) + 2 \\ &= -\!\left(t - \frac{1}{2}\right)^2 + \frac{1}{4} + 2 \\ &= -\!\left(t - \frac{1}{2}\right)^2 + \frac{9}{4} \end{align} \]
6

Nalazimo najveću vrednost (maksimum)

Pošto je \(-\!\left(t - \frac{1}{2}\right)^2 \leq 0\), maksimalna vrednost se dostiže kada je ovaj izraz jednak nuli, tj. za \(t = \dfrac{1}{2}\):

\[ y_{\max} = \frac{9}{4} \]

To odgovara \(\sin x = \dfrac{1}{2}\), tj. \(x = \dfrac{\pi}{6} + 2k\pi\) ili \(x = \dfrac{5\pi}{6} + 2k\pi\).

🔍 Proveri se: Koliko je \(y\) za \(x = \dfrac{\pi}{6}\)?

\(y = \sin\dfrac{\pi}{6} + \cos^2\dfrac{\pi}{6} + 1 = \dfrac{1}{2} + \dfrac{3}{4} + 1 = \dfrac{9}{4}\) ✓

7

Nalazimo najmanju vrednost (minimum)

Proveravamo vrednosti na krajevima intervala \([-1, 1]\):

\[ \begin{align} f(-1) &= -(-1)^2 + (-1) + 2 = -1 - 1 + 2 = 0 \\ f(1) &= -(1)^2 + 1 + 2 = -1 + 1 + 2 = 2 \end{align} \]

Najmanja vrednost je:

\[ y_{\min} = 0 \]

Dostiže se za \(\sin x = -1\), tj. \(x = -\dfrac{\pi}{2} + 2k\pi\).

8

Izračunavamo razliku

Razlika najveće i najmanje vrednosti funkcije:

\[ y_{\max} - y_{\min} = \frac{9}{4} - 0 = \frac{9}{4} \]
🔍 Provera: Da li se \(y = 0\) zaista dostiže?

Za \(x = -\dfrac{\pi}{2}\): \(y = \sin\!\left(-\dfrac{\pi}{2}\right) + \cos^2\!\left(-\dfrac{\pi}{2}\right) + 1 = -1 + 0 + 1 = 0\) ✓

Ključni uvid

Smenom \(t = \sin x\) i primenom identiteta \(\cos^2 x = 1 - t^2\), trigonometrijska funkcija se svodi na kvadratnu funkciju čije ekstreme nalazimo standardnim postupkom:

\[ y = -\!\left(t - \frac{1}{2}\right)^2 + \frac{9}{4}, \quad t \in [-1, 1] \]

Konačan odgovor

\[ y_{\max} - y_{\min} = \frac{9}{4} - 0 = \frac{9}{4} \]
(A) \(\dfrac{9}{4}\)
(B) \(1\)
(C) \(\dfrac{1}{2}\)
(D) \(2\)
(E) \(\dfrac{1}{3}\)
Verifikacija: Za \(x = \frac{\pi}{6}\): \(y = \frac{1}{2} + \frac{3}{4} + 1 = \frac{9}{4}\). Za \(x = -\frac{\pi}{2}\): \(y = -1 + 0 + 1 = 0\). Razlika: \(\frac{9}{4} - 0 = \frac{9}{4}\). Odgovor je potvrđen.

Česte greške

  • Zaboraviti opseg promenljive: Smena \(t = \sin x\) zahteva da \(t \in [-1, 1]\). Bez ovog ograničenja, mogli bismo pogrešno proceniti minimum (npr. uzeti teme parabole kao minimum za parabolu okrennutu nagore).
  • Pogrešna primena identiteta: Pisati \(\cos^2 x = \sin^2 x - 1\) umesto ispravnog \(\cos^2 x = 1 - \sin^2 x\) dovodi do potpuno pogrešne funkcije.

Dodatni izazov

Nađite razliku najveće i najmanje vrednosti funkcije \(y = \cos x - \sin^2 x + 3\).

💡 Pomoć

Primenite sličnu smenu \(t = \cos x\), pa koristite \(\sin^2 x = 1 - t^2\). Dobijate kvadratnu funkciju po \(t\) na intervalu \([-1, 1]\). Postupak je analogan!
