Razlika ekstrema funkcije sin x + cos x

Postavka zadatka

Razlika najveće i najmanje vrednosti funkcije \(y = \sin x + \cos x\) je:

(A)
\(2\)
(B)
\(1\)
(C)
\(0\)
(D)
\(\dfrac{\sqrt{2}}{2}\)
(E)
\(2\sqrt{2}\)

Plan rešavanja

Funkciju \(y = \sin x + \cos x\) transformisaćemo u oblik \(R\sin(x + \varphi)\) korišćenjem formule za pomoćni ugao. Iz tog oblika odmah očitavamo maksimalnu i minimalnu vrednost, a zatim računamo njihovu razliku.

Ključne formule i pojmovi
  • Pomoćni ugao (auxiliary angle): Za \(a\sin x + b\cos x\), važi: \[a\sin x + b\cos x = \sqrt{a^2 + b^2}\;\sin(x + \varphi)\] gde je \(\tan\varphi = \dfrac{b}{a}\).
  • Oblast vrednosti sinusa: Za svaki realni argument važi \(-1 \le \sin\theta \le 1\), pa je oblast vrednosti funkcije \(R\sin(x + \varphi)\) interval \([-R,\; R]\).
  • Razlika ekstrema: Za funkciju sa oblašću vrednosti \([-R, R]\), razlika najveće i najmanje vrednosti je \(R - (-R) = 2R\).

Vizuelni prikaz

Graf funkcije \(y = \sin x + \cos x = \sqrt{2}\sin\!\left(x + \frac{\pi}{4}\right)\)

Rešenje korak po korak

1

Prepoznajemo oblik za pomoćni ugao

Data funkcija je:

\[y = \sin x + \cos x = 1 \cdot \sin x + 1 \cdot \cos x\]

Ovo je oblik \(a\sin x + b\cos x\) sa koeficijentima \(a = 1\) i \(b = 1\).

2

Računamo amplitudu \(R\)

Koristimo formulu za pomoćni ugao. Amplituda je:

\[R = \sqrt{a^2 + b^2} = \sqrt{1^2 + 1^2} = \sqrt{2}\]
3

Određujemo fazni pomeraj \(\varphi\)

Iz uslova \(\tan\varphi = \dfrac{b}{a} = \dfrac{1}{1} = 1\) dobijamo:

\[\varphi = \frac{\pi}{4}\]

Biramo \(\varphi = \frac{\pi}{4}\) jer su oba koeficijenta pozitivna (ugao u prvom kvadrantu).

Proveri se: Koji je konačan oblik funkcije nakon transformacije?
\(y = \sqrt{2}\;\sin\!\left(x + \dfrac{\pi}{4}\right)\)
4

Zapisujemo transformisani oblik

Funkcija se može zapisati kao:

\[y = \sqrt{2}\;\sin\!\left(x + \frac{\pi}{4}\right)\]

Provera: Razvojimo koristeći adicionu formulu:

\[\sqrt{2}\;\sin\!\left(x + \frac{\pi}{4}\right) = \sqrt{2}\left(\sin x \cos\frac{\pi}{4} + \cos x \sin\frac{\pi}{4}\right) = \sqrt{2}\left(\frac{\sqrt{2}}{2}\sin x + \frac{\sqrt{2}}{2}\cos x\right)\]
\[= \sqrt{2} \cdot \frac{\sqrt{2}}{2}(\sin x + \cos x) = \frac{2}{2}(\sin x + \cos x) = \sin x + \cos x \;\;\checkmark\]
5

Određujemo najveću i najmanju vrednost

Pošto je \(-1 \le \sin\theta \le 1\) za svako \(\theta\), množeći sa \(\sqrt{2} > 0\) dobijamo:

\[-\sqrt{2} \le \sqrt{2}\;\sin\!\left(x + \frac{\pi}{4}\right) \le \sqrt{2}\]

Dakle:

\[y_{\max} = \sqrt{2}, \qquad y_{\min} = -\sqrt{2}\]
Proveri se: Za koje vrednosti \(x\) se dostižu ekstremi?
Maksimum \(\sqrt{2}\) se dostiže kada je \(\sin\!\left(x + \frac{\pi}{4}\right) = 1\), tj. za \(x = \frac{\pi}{4}\) (i periodično \(x = \frac{\pi}{4} + 2k\pi\)).
Minimum \(-\sqrt{2}\) se dostiže kada je \(\sin\!\left(x + \frac{\pi}{4}\right) = -1\), tj. za \(x = -\frac{3\pi}{4}\) (i periodično \(x = -\frac{3\pi}{4} + 2k\pi\)).
6

Računamo razliku ekstrema

Tražena razlika najveće i najmanje vrednosti je:

\[y_{\max} - y_{\min} = \sqrt{2} - (-\sqrt{2}) = \sqrt{2} + \sqrt{2} = 2\sqrt{2}\]

Ključni uvid

Svaka funkcija oblika \(a\sin x + b\cos x\) se transformiše u \(\sqrt{a^2+b^2}\;\sin(x + \varphi)\). Oblast vrednosti je \([-\sqrt{a^2+b^2},\;\sqrt{a^2+b^2}]\), pa je razlika ekstrema uvek \(2\sqrt{a^2+b^2}\).

\[2\sqrt{1^2 + 1^2} = 2\sqrt{2}\]

Konačan odgovor

\(\sqrt{2} - (-\sqrt{2}) = 2\sqrt{2}\)
(A)
\(2\)
(B)
\(1\)
(C)
\(0\)
(D)
\(\dfrac{\sqrt{2}}{2}\)
(E)
\(2\sqrt{2}\)
Provera: \(y = \sin x + \cos x = \sqrt{2}\sin\!\left(x + \frac{\pi}{4}\right)\). Maksimum: \(\sqrt{2}\), minimum: \(-\sqrt{2}\). Razlika: \(\sqrt{2} - (-\sqrt{2}) = 2\sqrt{2}\). Tačan odgovor je (E) \(2\sqrt{2}\).

Česte greške

  • Misliti da je maksimum 2: Intuitivno, ako su maksimumi \(\sin x\) i \(\cos x\) oba jednaki 1, zbir bi bio 2. Ali \(\sin x\) i \(\cos x\) ne dostižu maksimum istovremeno (osim nikada), pa je stvarni maksimum \(\sqrt{2} \approx 1{,}414\).
  • Zaboraviti minus ispred \(\sqrt{2}\): Najmanja vrednost je \(-\sqrt{2}\), ne \(0\). Obe funkcije \(\sin x\) i \(\cos x\) mogu biti negativne istovremeno.
  • Razlika umesto zbira: Razlika ekstrema je \(\sqrt{2} - (-\sqrt{2}) = 2\sqrt{2}\), ne \(\sqrt{2} + 0 = \sqrt{2}\). Pazite na dvostruki minus.

Dodatni izazov

Kolika je razlika najveće i najmanje vrednosti funkcije \(y = 3\sin x + 4\cos x\)?

Prikaži pomoć

Koristite isti metod: \(R = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5\). Oblast vrednosti je \([-5, 5]\), pa je razlika ekstrema \(5 - (-5) = 10\).
