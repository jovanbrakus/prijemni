Plan rešavanja

Transformisaćemo izraz \(\sin^4 x + \cos^4 x\) koristeći algebarski identitet za zbir kvadrata i formulu dvostrukog ugla. Svešćemo jednačinu na oblik \(\sin(2x) = 0\), a zatim prebrojati rešenja na datom intervalu.

Ključne formule i pojmovi
  • Pitagorin trigonometrijski identitet: \(\sin^2 x + \cos^2 x = 1\) — osnovni identitet koji važi za svako \(x\).
  • Zbir kvadrata preko zbira i proizvoda: \(a^2 + b^2 = (a + b)^2 - 2ab\). Primenom na \(a = \sin^2 x\), \(b = \cos^2 x\): \[\sin^4 x + \cos^4 x = (\sin^2 x + \cos^2 x)^2 - 2\sin^2 x \cos^2 x\]
  • Formula dvostrukog ugla: \(\sin(2x) = 2\sin x \cos x\), odakle sledi \(\sin^2 x \cos^2 x = \dfrac{\sin^2(2x)}{4}\).
  • Rešenja sin(t) = 0: \(t = k\pi\), za \(k \in \mathbb{Z}\).

Vizuelni prikaz

Grafik funkcije f(x) = sin⁴x + cos⁴x na intervalu [0, π]. Rešenja su preseci sa pravom y = 1.

Rešenje korak po korak

1

Transformišemo zbir četvrtih stepena

Koristimo algebarski identitet \(a^2 + b^2 = (a+b)^2 - 2ab\), gde su \(a = \sin^2 x\) i \(b = \cos^2 x\):

\[\sin^4 x + \cos^4 x = (\sin^2 x + \cos^2 x)^2 - 2\sin^2 x \cos^2 x\]
2

Primenjujemo Pitagorin identitet

Pošto je \(\sin^2 x + \cos^2 x = 1\), dobijamo:

\[\sin^4 x + \cos^4 x = 1^2 - 2\sin^2 x \cos^2 x = 1 - 2\sin^2 x \cos^2 x\]
3

Uvodimo formulu dvostrukog ugla

Iz \(\sin(2x) = 2\sin x \cos x\) sledi da je \(\sin x \cos x = \dfrac{\sin(2x)}{2}\), pa:

\[\sin^2 x \cos^2 x = \frac{\sin^2(2x)}{4}\]

Zamenom u prethodni izraz:

\[\sin^4 x + \cos^4 x = 1 - 2 \cdot \frac{\sin^2(2x)}{4} = 1 - \frac{\sin^2(2x)}{2}\]
Proveri se: Koliko iznosi \(\sin^4 x + \cos^4 x\) za \(x = \pi/4\)?
\(\sin^4(\pi/4) + \cos^4(\pi/4) = \left(\frac{\sqrt{2}}{2}\right)^4 + \left(\frac{\sqrt{2}}{2}\right)^4 = \frac{1}{4} + \frac{1}{4} = \frac{1}{2}\). Proveravamo formulom: \(1 - \frac{\sin^2(\pi/2)}{2} = 1 - \frac{1}{2} = \frac{1}{2}\). Poklapa se!
4

Postavljamo jednačinu

Jednačina \(\sin^4 x + \cos^4 x = 1\) sada postaje:

\[1 - \frac{\sin^2(2x)}{2} = 1\]

Oduzimamo 1 sa obe strane:

\[-\frac{\sin^2(2x)}{2} = 0 \quad \Longrightarrow \quad \sin^2(2x) = 0 \quad \Longrightarrow \quad \sin(2x) = 0\]
5

Rešavamo \(\sin(2x) = 0\)

Opšte rešenje jednačine \(\sin(2x) = 0\) je:

\[2x = k\pi, \quad k \in \mathbb{Z} \quad \Longrightarrow \quad x = \frac{k\pi}{2}\]
6

Biramo rešenja iz intervala \([0, \pi]\)

Tražimo sve celobrojne vrednosti \(k\) takve da \(x = \dfrac{k\pi}{2} \in [0, \pi]\), tj. \(0 \le \dfrac{k\pi}{2} \le \pi\), odakle \(0 \le k \le 2\):

\[k = 0: \quad x = 0\] \[k = 1: \quad x = \frac{\pi}{2}\] \[k = 2: \quad x = \pi\]

Vrednosti \(k = -1\) i \(k = 3\) daju \(x = -\pi/2\) i \(x = 3\pi/2\) koji su van intervala.

Proveri se: Da li sva tri rešenja zadovoljavaju polaznu jednačinu?

\(x = 0\!: \;\sin^4(0) + \cos^4(0) = 0 + 1 = 1\) ✓

\(x = \pi/2\!: \;\sin^4(\pi/2) + \cos^4(\pi/2) = 1 + 0 = 1\) ✓

\(x = \pi\!: \;\sin^4(\pi) + \cos^4(\pi) = 0 + 1 = 1\) ✓

Sva tri rešenja su validna.

7

Zaključak

Jednačina \(\sin^4 x + \cos^4 x = 1\) ima tačno 3 rešenja na intervalu \([0, \pi]\):

\[x \in \left\{0, \;\frac{\pi}{2}, \;\pi\right\}\]

Ključni uvid

Jednačina \(\sin^4 x + \cos^4 x = 1\) se elegantno svodi na \(\sin(2x) = 0\) primenom identiteta za zbir kvadrata i formule dvostrukog ugla. Suština je u tome da je \(\sin^4 x + \cos^4 x = 1\) samo kada je \(\sin^2 x \cos^2 x = 0\), tj. kada je bar jedan od sinusa ili kosinusa jednak nuli.

\[\sin^4 x + \cos^4 x = 1 - \frac{\sin^2(2x)}{2}\]

Česte greške

  • Zaboravljanje graničnih tačaka: Rešenja \(x = 0\) i \(x = \pi\) pripadaju intervalu \([0, \pi]\) jer je interval zatvoren (uključuje krajeve). Da je interval bio otvoren \((0, \pi)\), odgovor bi bio samo 1.
  • Pogrešna transformacija: Učenici ponekad pišu \(\sin^4 x + \cos^4 x = (\sin^2 x + \cos^2 x)^2 = 1\), zaboravljajući na član \(-2\sin^2 x \cos^2 x\). Važi \(a^2 + b^2 \neq (a+b)^2\) osim ako je \(ab = 0\).
  • Mešanje intervala za \(x\) i \(2x\): Ako je \(x \in [0, \pi]\), onda je \(2x \in [0, 2\pi]\). Moramo naći nule sinusa na intervalu \([0, 2\pi]\), a ne na \([0, \pi]\).

Dodatni izazov

Koliko rešenja ima jednačina \(\sin^6 x + \cos^6 x = 1\) na intervalu \([0, 2\pi]\)?

Prikaži pomoć

Koristite identitet \(a^3 + b^3 = (a+b)(a^2 - ab + b^2)\) sa \(a = \sin^2 x\), \(b = \cos^2 x\). Dobija se \(\sin^6 x + \cos^6 x = 1 - 3\sin^2 x \cos^2 x = 1 - \frac{3\sin^2(2x)}{4}\). Jednačina ponovo svodi na \(\sin(2x) = 0\), ali sada na intervalu \([0, 2\pi]\) — prebrojte rešenja!
