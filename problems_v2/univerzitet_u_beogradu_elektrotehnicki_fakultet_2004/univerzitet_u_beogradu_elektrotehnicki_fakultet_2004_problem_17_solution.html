Jednakost trigonometrijskih izraza

Trigonometrija — formule za poluugao i adicione formule

📋 Tekst zadatka

Dati su izrazi:

\[ E_1 = \sin^2\!\frac{x+y}{2} + \cos x \cos y \] \[ E_2 = \cos^2\!\frac{x-y}{2} - \sin x \sin y \] \[ E_3 = \cos^2\!\frac{x+y}{2} + \sin x \sin y \] \[ E_4 = \sin^2\!\frac{x-y}{2} + \cos x \cos y \]

Tačan je iskaz:

(A) \( E_1 \neq E_2,\; E_3 = E_4 \)
(B) \( E_1 = E_2,\; E_3 \neq E_4 \)
(C) Među datim izrazima nema međusobno jednakih
(D) \( E_1 = E_2,\; E_3 = E_4 \)
(E) \( E_1 = E_3,\; E_2 = E_4 \)

🎯 Plan rešavanja

Primenićemo formule za \(\sin^2\alpha\) i \(\cos^2\alpha\) preko dvostrukog ugla na svaki izraz, zatim ćemo razviti \(\cos(x \pm y)\) adicionim formulama i uporediti uprošćene oblike.

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Formule za poluugao (preko dvostrukog ugla): \[ \sin^2\alpha = \frac{1 - \cos 2\alpha}{2}, \qquad \cos^2\alpha = \frac{1 + \cos 2\alpha}{2} \]
  • Adicione formule za kosinus: \[ \cos(x+y) = \cos x \cos y - \sin x \sin y \] \[ \cos(x-y) = \cos x \cos y + \sin x \sin y \]
  • Korisne posledice: \[ \cos x \cos y = \frac{\cos(x-y) + \cos(x+y)}{2}, \quad \sin x \sin y = \frac{\cos(x-y) - \cos(x+y)}{2} \]

📐 Vizuelni prikaz

Struktura izraza: \( E_1 = E_3 \) (Grupa I) i \( E_2 = E_4 \) (Grupa II)

✏️ Rešenje korak po korak

1

Ključne formule

Zapisujemo formule koje ćemo primeniti na svaki izraz. Za \(\alpha = \dfrac{x+y}{2}\) imamo \(2\alpha = x+y\), a za \(\alpha = \dfrac{x-y}{2}\) imamo \(2\alpha = x-y\):

\[ \sin^2\!\frac{x+y}{2} = \frac{1 - \cos(x+y)}{2}, \qquad \cos^2\!\frac{x+y}{2} = \frac{1 + \cos(x+y)}{2} \] \[ \sin^2\!\frac{x-y}{2} = \frac{1 - \cos(x-y)}{2}, \qquad \cos^2\!\frac{x-y}{2} = \frac{1 + \cos(x-y)}{2} \]
2

Uprostimo \( E_1 \)

Zamenjujemo \(\sin^2\!\frac{x+y}{2}\) i razvijamo \(\cos(x+y)\) pomoću adicione formule:

\[ E_1 = \frac{1 - \cos(x+y)}{2} + \cos x \cos y \] \[ = \frac{1}{2} - \frac{\cos x \cos y - \sin x \sin y}{2} + \cos x \cos y \] \[ = \frac{1}{2} + \frac{\cos x \cos y}{2} + \frac{\sin x \sin y}{2} = \frac{1}{2} + \frac{\cos x \cos y + \sin x \sin y}{2} \] \[ = \boxed{\frac{1 + \cos(x-y)}{2}} \]
Proveri sebe: Zašto je \(\cos x \cos y + \sin x \sin y = \cos(x-y)\)?

To je adiciona formula za kosinus razlike dva ugla. Direktno iz definicije: \(\cos(x - y) = \cos x \cos y + \sin x \sin y\).

3

Uprostimo \( E_2 \)

Zamenjujemo \(\cos^2\!\frac{x-y}{2}\) i razvijamo \(\cos(x-y)\):

\[ E_2 = \frac{1 + \cos(x-y)}{2} - \sin x \sin y \] \[ = \frac{1}{2} + \frac{\cos x \cos y + \sin x \sin y}{2} - \sin x \sin y \] \[ = \frac{1}{2} + \frac{\cos x \cos y}{2} - \frac{\sin x \sin y}{2} = \frac{1}{2} + \frac{\cos x \cos y - \sin x \sin y}{2} \] \[ = \boxed{\frac{1 + \cos(x+y)}{2}} \]
4

Uprostimo \( E_3 \)

Zamenjujemo \(\cos^2\!\frac{x+y}{2}\) i razvijamo \(\cos(x+y)\):

\[ E_3 = \frac{1 + \cos(x+y)}{2} + \sin x \sin y \] \[ = \frac{1}{2} + \frac{\cos x \cos y - \sin x \sin y}{2} + \sin x \sin y \] \[ = \frac{1}{2} + \frac{\cos x \cos y}{2} + \frac{\sin x \sin y}{2} = \frac{1}{2} + \frac{\cos x \cos y + \sin x \sin y}{2} \] \[ = \boxed{\frac{1 + \cos(x-y)}{2}} \]

Isti rezultat kao \( E_1 \)! Dakle \( E_1 = E_3 \).

5

Uprostimo \( E_4 \)

Zamenjujemo \(\sin^2\!\frac{x-y}{2}\) i razvijamo \(\cos(x-y)\):

\[ E_4 = \frac{1 - \cos(x-y)}{2} + \cos x \cos y \] \[ = \frac{1}{2} - \frac{\cos x \cos y + \sin x \sin y}{2} + \cos x \cos y \] \[ = \frac{1}{2} + \frac{\cos x \cos y}{2} - \frac{\sin x \sin y}{2} = \frac{1}{2} + \frac{\cos x \cos y - \sin x \sin y}{2} \] \[ = \boxed{\frac{1 + \cos(x+y)}{2}} \]

Isti rezultat kao \( E_2 \)! Dakle \( E_2 = E_4 \).

6

Poređenje rezultata

Sada uporedimo sve dobijene uprošćene oblike:

Izraz Uprošćeni oblik Grupa
\( E_1 \) \( \dfrac{1 + \cos(x-y)}{2} \) I
\( E_2 \) \( \dfrac{1 + \cos(x+y)}{2} \) II
\( E_3 \) \( \dfrac{1 + \cos(x-y)}{2} \) I
\( E_4 \) \( \dfrac{1 + \cos(x+y)}{2} \) II
\[ E_1 = E_3 = \frac{1 + \cos(x-y)}{2}, \qquad E_2 = E_4 = \frac{1 + \cos(x+y)}{2} \]

💡 Ključni uvid

Izrazi sa \(\sin^2\) i \(\cos^2\) istog argumenta \(\dfrac{x+y}{2}\), ali različitim “partnerima” (\(\cos x \cos y\) i \(\sin x \sin y\)), daju isti rezultat! Razlog je što formule \(\sin^2\alpha = \dfrac{1-\cos 2\alpha}{2}\) i \(\cos^2\alpha = \dfrac{1+\cos 2\alpha}{2}\) uvode suprotan znak ispred \(\cos(x+y)\), a kada se ti članovi kombinuju sa “partnerima”, razlike se tačno kompenzuju.

✅ Konačan odgovor

\[ E_1 = E_3, \quad E_2 = E_4 \]

(A) \( E_1 \neq E_2,\; E_3 = E_4 \) (B) \( E_1 = E_2,\; E_3 \neq E_4 \) (C) Nema međusobno jednakih (D) \( E_1 = E_2,\; E_3 = E_4 \) (E) \( E_1 = E_3,\; E_2 = E_4 \)

Provera: za \( x = \frac{\pi}{3},\; y = \frac{\pi}{6} \): \( E_1 = \sin^2\!\frac{\pi}{4} + \cos\frac{\pi}{3}\cos\frac{\pi}{6} = \frac{1}{2} + \frac{\sqrt{3}}{4} \), \( E_3 = \cos^2\!\frac{\pi}{4} + \sin\frac{\pi}{3}\sin\frac{\pi}{6} = \frac{1}{2} + \frac{\sqrt{3}}{4} \). Jednako. ✓

⚠️ Česte greške

  • Mešanje adicionih formula: Česta greška je zameniti znakove u formulama \(\cos(x \pm y)\). Zapamtite: \(\cos(x+y) = \cos x \cos y \mathbf{-} \sin x \sin y\), a \(\cos(x-y) = \cos x \cos y \mathbf{+} \sin x \sin y\). Minus je kod zbira, plus je kod razlike!
  • Pogrešna primena formule za poluugao: Ako stavimo \(\alpha = \frac{x+y}{2}\), onda je \(2\alpha = x+y\), ne \(\frac{x+y}{4}\). Dvostruki ugao poluugla je upravo ceo ugao.
  • Prenagljeno zaključivanje: Mnogi učenici pokušaju da “pogode” odgovor na osnovu simetrije izraza, bez razvijanja. Razlike su suptilne — uvek proverite računom.

🏆 Dodatni izazov

Pokušaj da dokažeš sledeći identitet bez korišćenja adicionih formula, samo primenom formule za razliku kvadrata:

\[ \cos^2\!\frac{x+y}{2} + \sin x \sin y = \sin^2\!\frac{x+y}{2} + \cos x \cos y \]
Pogledaj nagoveštaj

Prebaci sve na jednu stranu i koristi \(\cos^2\alpha - \sin^2\alpha = \cos 2\alpha\) i \(\cos x \cos y - \sin x \sin y = \cos(x+y)\).
