Osobina jednakosti razlomaka

Proporcije i linearne kombinacije

📋 Tekst zadatka

Ako je \(\dfrac{a}{A} = \dfrac{b}{B} = \dfrac{c}{C} = k\), tada je \(\dfrac{4a - 3b + c}{4A - 3B + C}\) jednako:

(A) \(2k\)
(B) \(k\)
(C) \(\dfrac{2k}{3}\)
(D) \(-2k\)
(E) \(\dfrac{1}{2}\)

🎯 Plan rešavanja

Iz datih jednakosti izrazićemo \(a\), \(b\) i \(c\) preko \(k\), zameniti u brojilac, izvući zajednički faktor i skratiti sa imeniocem.

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Osobina proporcije: Ako je \(\dfrac{a_1}{b_1} = \dfrac{a_2}{b_2} = \cdots = \dfrac{a_n}{b_n} = k\), tada za proizvoljne koeficijente \(\alpha_1, \alpha_2, \ldots, \alpha_n\) važi: \[\frac{\alpha_1 a_1 + \alpha_2 a_2 + \cdots + \alpha_n a_n}{\alpha_1 b_1 + \alpha_2 b_2 + \cdots + \alpha_n b_n} = k\] pod uslovom da je imenilac različit od nule.
  • Zašto ovo važi? Iz \(\dfrac{a_i}{b_i} = k\) sledi \(a_i = k \cdot b_i\). Kada zamenimo u brojilac, svaki član sadrži faktor \(k\), koji se zatim izvlači ispred zagrade i skraćuje sa imeniocem.
  • Specijalan slučaj: Za koeficijente \(\alpha_i = 1\) dobijamo: \[\frac{a_1 + a_2 + \cdots + a_n}{b_1 + b_2 + \cdots + b_n} = k\]

✏️ Rešenje korak po korak

1

Izražavanje veličina preko \(k\)

Iz uslova zadatka znamo da su sva tri razlomka jednaka konstanti \(k\):

\[\frac{a}{A} = k \;\Rightarrow\; a = kA\]
\[\frac{b}{B} = k \;\Rightarrow\; b = kB\]
\[\frac{c}{C} = k \;\Rightarrow\; c = kC\]
Proveri sebe: Svako malo slovo je jednako \(k\) puta odgovarajuće veliko slovo. Zašto?

Iz \(\frac{a}{A} = k\) množimo obe strane sa \(A\) i dobijamo \(a = kA\). Isto važi za \(b\) i \(c\). Ovo je ključna zamena koja omogućava izvlačenje faktora \(k\).

2

Zamena u brojilac

Zamenimo \(a = kA\), \(b = kB\) i \(c = kC\) u izraz \(4a - 3b + c\):

\[4a - 3b + c = 4(kA) - 3(kB) + kC = 4kA - 3kB + kC\]
3

Izvlačenje zajedničkog faktora

Iz svakog sabirka možemo izvući zajednički faktor \(k\):

\[4kA - 3kB + kC = k(4A - 3B + C)\]
4

Skraćivanje razlomka

Formiramo traženi razlomak:

\[\frac{4a - 3b + c}{4A - 3B + C} = \frac{k(4A - 3B + C)}{4A - 3B + C} = k\]

Pod pretpostavkom da je \(4A - 3B + C \neq 0\), skraćujemo i dobijamo rezultat \(k\).

Proveri sebe: Zašto brojilac i imenilac moraju biti različiti od nule?

Ako bi bilo \(4A - 3B + C = 0\), tada bi i brojilac bio \(k \cdot 0 = 0\), pa bismo dobili neodređen oblik \(\frac{0}{0}\). Zato rezultat važi samo uz uslov da imenilac nije nula.

5

Verifikacija na konkretnom primeru

Proverimo sa \(k = 2\), \(A = 1\), \(B = 1\), \(C = 1\):

Tada je \(a = 2\), \(b = 2\), \(c = 2\).

\[\frac{4 \cdot 2 - 3 \cdot 2 + 2}{4 \cdot 1 - 3 \cdot 1 + 1} = \frac{8 - 6 + 2}{4 - 3 + 1} = \frac{4}{2} = 2 = k \;\; \checkmark\]

💡 Ključni uvid

Kada je više razlomaka jednako istoj konstanti \(k\), svaka linearna kombinacija brojioca, podeljena istom linearnom kombinacijom imenioca (sa istim koeficijentima), takođe daje \(k\). To je zato što se \(k\) ponaša kao zajednički faktor koji se izvlači iz brojioca i skraćuje sa imeniocem. Koeficijenti linearne kombinacije (u ovom slučaju 4, -3, 1) su potpuno nebitni za rezultat!

✅ Konačan odgovor

\[\dfrac{4a - 3b + c}{4A - 3B + C} = k\]

(A) \(2k\) (B) \(k\) (C) \(\dfrac{2k}{3}\) (D) \(-2k\) (E) \(\dfrac{1}{2}\)

Provera: za \(k = 2\), \(A = B = C = 1\): \(\frac{8 - 6 + 2}{4 - 3 + 1} = \frac{4}{2} = 2 = k\) ✓

⚠️ Česte greške

  • Računanje sa koeficijentima: Neki učenici pokušavaju da "uračunaju" koeficijente 4, -3 i 1 u rezultat (npr. dobiju \(2k\) ili \(\frac{2k}{3}\)). Koeficijenti se pojavljuju i u brojiocu i u imeniocu, pa se skrate!
  • Zaborav uslova: Rezultat važi samo ako je \(4A - 3B + C \neq 0\). U suprotnom, razlomak nije definisan. Na testu, autori biraju vrednosti tako da imenilac nije nula.
  • Mešanje brojioca i imenioca: Osobina važi samo ako koeficijenti u brojiocu i imeniocu odgovaraju istim parovima \((a, A)\), \((b, B)\), \((c, C)\). Ako se pomešaju parovi, rezultat nije \(k\).

🏆 Dodatni izazov

Ako je \(\dfrac{a}{A} = \dfrac{b}{B} = \dfrac{c}{C} = k\), koliko je:

\[\frac{a^2 + b^2 + c^2}{A^2 + B^2 + C^2} \;?\]
Pogledaj nagoveštaj

Zameni \(a = kA\), \(b = kB\), \(c = kC\) i izvuci \(k^2\) ispred zagrade. Rezultat je \(k^2\).
