Zbir rešenja trigonometrijske jednačine

Trigonometrija — harmonijsko sabiranje i elementarne jednačine

📋 Tekst zadatka

Zbir rešenja jednačine \(\sqrt{3}\sin x + \cos x = \sqrt{3}\), koja pripadaju intervalu \((0, 2\pi)\), je:

(A) \(\dfrac{\pi}{2}\)
(B) \(0\)
(C) \(\dfrac{\pi}{3}\)
(D) \(\dfrac{2\pi}{3}\)
(E) \(\dfrac{\pi}{6}\)

🎯 Plan rešavanja

Levi deo jednačine svesti na oblik \(R\sin(x + \varphi)\) pomoću formule za harmonijsko sabiranje, zatim rešiti elementarnu trigonometrijsku jednačinu i odabrati rešenja iz datog intervala.

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Harmonijsko sabiranje: \(a\sin x + b\cos x = R\sin(x + \varphi)\), gde je \(R = \sqrt{a^2+b^2}\), \(\cos\varphi = \frac{a}{R}\), \(\sin\varphi = \frac{b}{R}\)
  • Opšte rešenje: \(\sin\theta = c\) daje \(\theta = \arcsin c + 2k\pi\) ili \(\theta = \pi - \arcsin c + 2k\pi\)
  • Poznate vrednosti: \(\sin\frac{\pi}{3} = \frac{\sqrt{3}}{2}\)

📐 Vizuelni prikaz

Graf funkcije f(x) = sqrt(3)sin(x) + cos(x) sa rešenjima jednačine

✏️ Rešenje korak po korak

1

Harmonijsko sabiranje

Imamo \(a = \sqrt{3}\), \(b = 1\). Računamo:

\[ R = \sqrt{(\sqrt{3})^2 + 1^2} = \sqrt{4} = 2 \]

Fazni ugao: \(\cos\varphi = \frac{\sqrt{3}}{2}\), \(\sin\varphi = \frac{1}{2}\), dakle \(\varphi = \frac{\pi}{6}\).

\[ \sqrt{3}\sin x + \cos x = 2\sin\!\left(x + \frac{\pi}{6}\right) \]
Proveri sebe: Razvij \(2\sin(x + \frac{\pi}{6})\)

\(2[\sin x \cdot \frac{\sqrt{3}}{2} + \cos x \cdot \frac{1}{2}] = \sqrt{3}\sin x + \cos x\) ✓

2

Svođenje na elementarnu jednačinu

\[ 2\sin\!\left(x + \frac{\pi}{6}\right) = \sqrt{3} \quad \Longrightarrow \quad \sin\!\left(x + \frac{\pi}{6}\right) = \frac{\sqrt{3}}{2} \]
3

Rešavanje jednačine

Opšta rešenja \(\sin\theta = \frac{\sqrt{3}}{2}\):

\[ x + \frac{\pi}{6} = \frac{\pi}{3} + 2k\pi \quad \text{ili} \quad x + \frac{\pi}{6} = \frac{2\pi}{3} + 2k\pi \]

Rešavamo za \(x\):

\[ x = \frac{\pi}{6} + 2k\pi \quad \text{ili} \quad x = \frac{\pi}{2} + 2k\pi \]
4

Izbor rešenja iz \((0, 2\pi)\)

Za \(k = 0\): \(x_1 = \frac{\pi}{6} \in (0, 2\pi)\) ✓ i \(x_2 = \frac{\pi}{2} \in (0, 2\pi)\) ✓

Za \(k = 1\): oba prelaze \(2\pi\). Za \(k = -1\): oba su negativna.

5

Zbir rešenja

\[ x_1 + x_2 = \frac{\pi}{6} + \frac{\pi}{2} = \frac{\pi}{6} + \frac{3\pi}{6} = \frac{4\pi}{6} = \frac{2\pi}{3} \]

💡 Ključni uvid

Formula za harmonijsko sabiranje pretvara zbir \(a\sin x + b\cos x\) u jednu sinusnu funkciju sa pomerenom fazom. Ovo je standardna tehnika za rešavanje jednačina oblika \(a\sin x + b\cos x = c\).

✅ Konačan odgovor

\[ x_1 + x_2 = \frac{2\pi}{3} \]

(A) \(\frac{\pi}{2}\) (B) \(0\) (C) \(\frac{\pi}{3}\) (D) \(\frac{2\pi}{3}\) (E) \(\frac{\pi}{6}\)

Provera: \(x=\frac{\pi}{6}\): \(\sqrt{3} \cdot \frac{1}{2} + \frac{\sqrt{3}}{2} = \sqrt{3}\) ✓; \(x=\frac{\pi}{2}\): \(\sqrt{3} \cdot 1 + 0 = \sqrt{3}\) ✓

⚠️ Česte greške

  • Pogrešan fazni ugao: Važno je ispravno odrediti \(\varphi\) iz koeficijenata — ne mešati \(\cos\varphi\) i \(\sin\varphi\).
  • Propust druge grane: Jednačina \(\sin\theta = c\) uvek ima dve grane rešenja — ne zaboraviti \(\theta = \pi - \arcsin c + 2k\pi\).

🏆 Dodatni izazov

Rešiti jednačinu \(\sin x - \sqrt{3}\cos x = 1\) na intervalu \((0, 2\pi)\).

Pogledaj nagoveštaj

Primeni isti postupak: \(R = 2\), \(\varphi = -\frac{\pi}{3}\). Jednačina postaje \(2\sin(x - \frac{\pi}{3}) = 1\), tj. \(\sin(x - \frac{\pi}{3}) = \frac{1}{2}\).
