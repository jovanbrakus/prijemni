Zbir rešenja logaritamske jednačine

Logaritmi — promena baze, apsolutna vrednost i smena

📋 Tekst zadatka

Zbir svih rešenja jednačine

\[ 2\,\bigl(\log_4 |x+1|\bigr)^{2} + \log_4 |x^2-1| + \log_{\frac{1}{4}} |x-1| = 0 \]

je:

(A) \(1\)
(B) \(-2\)
(C) \(-4\)
(D) \(4\)
(E) \(-\tfrac{1}{2}\)

🎯 Plan rešavanja

Primeniti pravila za logaritme (promena baze, logaritam proizvoda), uvesti smenu \(u = \log_4|x+1|\), svesti na kvadratnu po \(u\), rešiti i proveriti u domenu.

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Promena baze: \(\log_{\frac{1}{a}} b = -\log_a b\)
  • Logaritam proizvoda: \(\log_a(bc) = \log_a b + \log_a c\) za \(b, c > 0\)
  • Razlika kvadrata: \(|x^2-1| = |x+1| \cdot |x-1|\)

✏️ Rešenje korak po korak

1

Određivanje domena

Argumenti svih logaritama moraju biti strogo pozitivni:

\[ |x+1| > 0 \;\Rightarrow\; x \neq -1, \qquad |x-1| > 0 \;\Rightarrow\; x \neq 1 \]

Domen: \(x \in \mathbb{R} \setminus \{-1, 1\}\).

2

Transformacija logaritama

Treći član: \(\log_{\frac{1}{4}} |x-1| = -\log_4 |x-1|\)

Drugi član: \(\log_4|x^2-1| = \log_4|x+1| + \log_4|x-1|\)

Jednačina postaje:

\[ 2(\log_4|x+1|)^2 + \log_4|x+1| + \log_4|x-1| - \log_4|x-1| = 0 \]
3

Smena i pojednostavljenje

Neka je \(u = \log_4|x+1|\). Članovi sa \(\log_4|x-1|\) se poništavaju:

\[ 2u^2 + u = 0 \quad \Longrightarrow \quad u(2u + 1) = 0 \]
Proveri sebe: Zašto se \(v = \log_4|x-1|\) poništava?

Iz drugog člana dobijamo \(+v\), a iz trećeg \(-v\). Zbir: \(v - v = 0\).

4

Vraćanje na \(x\)

Slučaj 1: \(u = 0 \Rightarrow |x+1| = 1\):

\[ x + 1 = 1 \;\Rightarrow\; x = 0 \qquad \text{ili} \qquad x + 1 = -1 \;\Rightarrow\; x = -2 \]

Slučaj 2: \(u = -\frac{1}{2} \Rightarrow |x+1| = 4^{-1/2} = \frac{1}{2}\):

\[ x + 1 = \frac{1}{2} \;\Rightarrow\; x = -\frac{1}{2} \qquad \text{ili} \qquad x + 1 = -\frac{1}{2} \;\Rightarrow\; x = -\frac{3}{2} \]
5

Provera i zbir

Sva četiri rešenja (\(0, -2, -\frac{1}{2}, -\frac{3}{2}\)) zadovoljavaju \(x \neq \pm 1\).

\[ S = 0 + (-2) + \left(-\frac{1}{2}\right) + \left(-\frac{3}{2}\right) = -2 - 2 = -4 \]

💡 Ključni uvid

Izvanredna elegancija ovog zadatka: nakon transformacije, članovi sa \(\log_4|x-1|\) se potpuno poništavaju, pa jednačina zavisi samo od \(\log_4|x+1|\). Ovo drastično pojednostavljuje postupak.

✅ Konačan odgovor

\[ S = -4 \]

(A) \(1\) (B) \(-2\) (C) \(-4\) (D) \(4\) (E) \(-\frac{1}{2}\)

Provera za \(x = 0\): \(2 \cdot 0 + 0 + 0 = 0\) ✓; Zbir: \(0 + (-2) + (-\frac{1}{2}) + (-\frac{3}{2}) = -4\) ✓

⚠️ Česte greške

  • Zaborav apsolutne vrednosti: Jednačina \(|x+1| = c\) ima dva rešenja: \(x+1 = c\) i \(x+1 = -c\). Bez toga dobijamo samo dva od četiri rešenja.
  • Pogrešna promena baze: \(\log_{\frac{1}{4}} = -\log_4\), ne \(\frac{1}{\log_4}\).

🏆 Dodatni izazov

Šta ako umesto \(\log_{\frac{1}{4}}|x-1|\) u jednačini stoji \(+\log_4|x-1|\)?

Pogledaj nagoveštaj

Tada se članovi sa \(v\) ne poništavaju: \(2u^2 + u + 2v = 0\). Jednačina zavisi od obe veličine i postaje znatno teža.
