Apsolutna vrednost ugnježdenog izraza

Algebra — osobine apsolutne vrednosti i algebarsko preuređivanje

📋 Tekst zadatka

Ako je \(|x + a| = a\) \((a \geq 0)\), tada je \(||x| - a|\) jednako:

(A) \(0\)
(B) \(a\)
(C) \(2a\)
(D) \(3a\)
(E) \(\dfrac{a}{2}\)

🎯 Plan rešavanja

Iz uslova \(|x + a| = a\) odredimo moguće vrednosti za \(x\), uočimo da je \(x \leq 0\), pa izraz \(||x| - a|\) svedemo nazad na polazni uslov algebarskom transformacijom.

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Jednačina \(|t| = c\) za \(c \geq 0\): rešenja su \(t = c\) ili \(t = -c\)
  • Za \(x \leq 0\): važi \(|x| = -x\)
  • Osobina: \(|-t| = |t|\) za svako realno \(t\)

📐 Vizuelni prikaz

Brojevna prava prikazuje rešenja jednačine \(|x + a| = a\) za \(a = 3\) i geometrijsko tumačenje rastojanja

✏️ Rešenje korak po korak

1

Rešimo jednačinu \(|x + a| = a\)

Primenimo \(|t| = c \iff t = c\) ili \(t = -c\) na \(t = x + a\), \(c = a\):

\[ x + a = a \;\Rightarrow\; x = 0 \qquad \text{ili} \qquad x + a = -a \;\Rightarrow\; x = -2a \]
Proveri sebe: Zašto ne čitav interval \([-2a, 0]\)?

Nejednačina \(|x + a| \leq a\) bi dala interval, ali jednačina \(|x + a| = a\) ima tačno dva rešenja: \(x = 0\) i \(x = -2a\).

2

Uočimo da je \(x \leq 0\)

Oba rešenja zadovoljavaju \(x \leq 0\) (jer je \(a \geq 0\)): \(x = 0 \leq 0\) i \(x = -2a \leq 0\).

Pošto je \(x \leq 0\):

\[ |x| = -x \]
3

Transformišemo izraz

Zamenom \(|x| = -x\):

\[ |x| - a = -x - a = -(x + a) \]

Primenimo spoljašnju apsolutnu vrednost:

\[ ||x| - a| = |-(x + a)| = |x + a| \]
4

Primenimo polazni uslov

Iz zadatka znamo \(|x + a| = a\), pa:

\[ ||x| - a| = |x + a| = a \]
Proveri sebe: Numerička verifikacija za \(a = 5\)

Za \(x = 0\): \(||0| - 5| = |-5| = 5 = a\) ✓

Za \(x = -10\): \(||-10| - 5| = |10 - 5| = 5 = a\) ✓

💡 Ključni uvid

Za \(x \leq 0\) važi \(|x| = -x\), pa se \(|x| - a\) svodi na \(-(x + a)\). Spoljašnja apsolutna vrednost \(|-(x + a)|\) daje \(|x + a|\), što je upravo ono što nam je dato u zadatku! Ovaj elegantni krug nas vraća na polazni uslov u tri reda.

✅ Konačan odgovor

\[ ||x| - a| = a \]

(A) \(0\) (B) \(a\) (C) \(2a\) (D) \(3a\) (E) \(\dfrac{a}{2}\)

Provera: za \(a = 5\), rešenja su \(x = 0\) i \(x = -10\). U oba slučaja \(||x| - a| = 5 = a\). Algebarski: \(||x| - a| = |-(x+a)| = |x+a| = a\).

⚠️ Česte greške

  • Zaboravljanje da je \(x \leq 0\): Bez ovog zaključka ne može se primeniti \(|x| = -x\), čime se gubi ključna transformacija.
  • Mešanje jednačine i nejednačine: Jednačina \(|x+a| = a\) ima tačno dva rešenja, a ne čitav interval. Ipak, algebarski pristup funkcioniše jer se oslanja samo na \(x \leq 0\).
  • Odgovor 0 ili 2a usled parcijalne analize: Razmatranje samo jednog slučaja ili pogrešan račun može dati netačan rezultat.

🏆 Dodatni izazov

Ako je \(|x - a| = 2a\) \((a \geq 0)\), koliko je \(||x| + a|\)?

Pogledaj nagoveštaj

Reši jednačinu: \(x = 3a\) ili \(x = -a\). Pošto je \(|x| + a \geq 0\) uvek, spoljašnja apsolutna vrednost je suvišna. Za \(x = 3a\): \(3a + a = 4a\). Za \(x = -a\): \(a + a = 2a\). Rezultat zavisi od \(x\)!
