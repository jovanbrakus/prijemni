Ostatak deljenja polinoma visokog stepena

Algebra — deljenje polinoma, kompleksni koreni, cikličnost stepena broja \(i\)

📋 Tekst zadatka

Ako se polinom \(x^{2008} + x^{1007} + 1\) podeli sa \(x^2 + 1\), ostatak je:

Polinom \(P(x) = x^{2008} + x^{1007} + 1\)
Delilac \(D(x) = x^2 + 1\)
Traži se Ostatak \(R(x) = ax + b\)
(A) \(2x + 1\)
(B) \(-x + 2\)
(C) \(0\)
(D) \(x + 1\)
(E) \(x - 2\)

🎯 Plan rešavanja

Umesto direktnog deljenja, koristićemo kompleksne korene delioca \(x = \pm i\) i cikličnost stepena broja \(i\) da odredimo koeficijente ostatka.

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Deljenje polinoma: \(P(x) = Q(x) \cdot D(x) + R(x)\), gde je \(\deg R < \deg D\). Za \(\deg D = 2\), ostatak je \(R(x) = ax + b\).
  • Generalizovana teorema o ostatku: Ako je \(D(\alpha) = 0\), tada \(P(\alpha) = R(\alpha)\).
  • Cikličnost stepena broja \(i\): \(i^0 = 1\), \(i^1 = i\), \(i^2 = -1\), \(i^3 = -i\), \(i^4 = 1\), ... Pravilo: \(i^n = i^{n \bmod 4}\).

📐 Vizuelni prikaz

Stepeni broja \(i\) u kompleksnoj ravni i postupak izračunavanja \(P(i)\)

✏️ Rešenje korak po korak

1

Postavka — oblik ostatka

Po teoremi o deljenju polinoma: \(P(x) = Q(x) \cdot (x^2 + 1) + R(x)\), gde je \(R(x) = ax + b\).

Koreni delioca \(x^2 + 1 = 0\) su \(x = i\) i \(x = -i\). Za koren \(\alpha\) važi \(P(\alpha) = R(\alpha)\).

2

Izračunavanje \(i^{2008}\)

Odredimo ostatak eksponenta pri deljenju sa 4:

\[2008 \div 4 = 502 \quad (\text{ostatak } 0) \quad \Rightarrow \quad i^{2008} = (i^4)^{502} = 1\]
3

Izračunavanje \(i^{1007}\)

\[1007 = 4 \cdot 251 + 3 \quad \Rightarrow \quad i^{1007} = i^3 = -i\]
Proveri sebe: Da li je zaista \(1007 = 4 \cdot 251 + 3\)?

\(4 \cdot 251 = 1004\), a \(1007 - 1004 = 3\). Tačno!

4

Izračunavanje \(P(i)\)

\[P(i) = i^{2008} + i^{1007} + 1 = 1 + (-i) + 1 = 2 - i\]
5

Određivanje koeficijenata \(a\) i \(b\)

Iz uslova \(R(i) = P(i)\):

\[ai + b = 2 - i\]

Izjednačavamo realne i imaginarne delove:

\[\text{Realni deo: } b = 2 \qquad \text{Imaginarni deo: } a = -1\]
6

Verifikacija sa \(x = -i\)

\[(-i)^{2008} = 1, \quad (-i)^{1007} = (-1)^{1007} \cdot i^{1007} = (-1)(-i) = i\]
\[P(-i) = 1 + i + 1 = 2 + i\]

Provera: \(R(-i) = -(-1)(-i) + 2 = i + 2 = 2 + i\) ✓

Proveri sebe: Zašto je \((-1)^{2008} = 1\) a \((-1)^{1007} = -1\)?

\((-1)^n = 1\) za paran \(n\) i \(-1\) za neparan \(n\). Broj 2008 je paran, a 1007 je neparan.

💡 Ključni uvid

Umesto direktnog deljenja polinoma stepena 2008, koriste se kompleksni koreni delioca \(\pm i\). Uvrštavanje korena „poništava“ količnik i ostavlja samo ostatak. Ceo postupak se svodi na izračunavanje stepena broja \(i\) pomoću cikličnosti perioda 4.

✅ Konačan odgovor

\[ R(x) = -x + 2 \]

(A) \(2x + 1\) (B) \(-x + 2\) (C) \(0\) (D) \(x + 1\) (E) \(x - 2\)

Provera: \(P(i) = 2 - i\) i \(R(i) = -i + 2 = 2 - i\) ✓. Takođe \(P(-i) = 2 + i\) i \(R(-i) = i + 2 = 2 + i\) ✓.

⚠️ Česte greške

  • Pogrešan ostatak deljenja eksponenta sa 4: Greška u \(1007 \bmod 4\) vodi na pogrešan odgovor. Zapamtite: \(1007 = 4 \cdot 251 + 3\), ostatak je 3.
  • Mešanje \(i^3 = -i\) sa \(i^3 = i\): Važi \(i^3 = i^2 \cdot i = (-1) \cdot i = -i\). Predznak minus se ne sme izgubiti.
  • Brkanje odgovora (B) i (E): Opcija (E) je \(x - 2\), a tačan je \(-x + 2\). Razlika u predznacima!

🏆 Dodatni izazov

Odredi ostatak pri deljenju polinoma \(x^{2025} + x^{2024} + 1\) sa \(x^2 + 1\).

Pogledaj nagoveštaj

\(2025 \bmod 4 = 1\), pa \(i^{2025} = i\). \(2024 \bmod 4 = 0\), pa \(i^{2024} = 1\). Dakle \(P(i) = i + 1 + 1 = 2 + i\), što daje \(a = 1\), \(b = 2\), tj. \(R(x) = x + 2\).
