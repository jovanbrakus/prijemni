Zbir rešenja trigonometrijske jednačine

Trigonometrija — jednačine sa dvostrukim uglom, faktorisanje, pomoćni ugao

📋 Tekst zadatka

Zbir svih rešenja jednačine

\[\sin 2x = 1 + \sqrt{2}\,\cos x + \cos 2x\]

na intervalu \((0,\, 2\pi)\) je:

(A) \(\dfrac{7\pi}{2}\)
(B) \(\dfrac{3\pi}{2}\)
(C) \(\dfrac{5\pi}{2}\)
(D) \(\pi\)
(E) \(\dfrac{\pi}{2}\)

🎯 Plan rešavanja

Primenićemo formule za dvostruki ugao, izlučiti zajednički faktor \(\cos x\), rastaviti jednačinu na dva slučaja i rešiti svaki pomoću metoda pomoćnog ugla.

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Dvostruki ugao: \(\sin 2x = 2\sin x \cos x\), \(\cos 2x = 2\cos^2 x - 1\)
  • Pomoćni ugao: \(\sin x - \cos x = \sqrt{2}\,\sin\!\left(x - \frac{\pi}{4}\right)\)
  • \(\sin t = \frac{1}{2}\): \(t = \frac{\pi}{6} + 2k\pi\) ili \(t = \frac{5\pi}{6} + 2k\pi\)

📐 Vizuelni prikaz

Četiri rešenja jednačine na jediničnoj kružnici

✏️ Rešenje korak po korak

1

Primena formula za dvostruki ugao

Zamenjujemo \(\sin 2x = 2\sin x\cos x\) i \(\cos 2x = 2\cos^2 x - 1\):

\[2\sin x\cos x = 1 + \sqrt{2}\,\cos x + 2\cos^2 x - 1\]

Jedinicom se pokrate:

\[2\sin x\cos x = \sqrt{2}\,\cos x + 2\cos^2 x\]
2

Faktorisanje

Prebacimo sve na levu stranu i izlučimo \(\cos x\):

\[\cos x\,\bigl(2\sin x - \sqrt{2} - 2\cos x\bigr) = 0\]
3

Slučaj 1: \(\cos x = 0\)

Na intervalu \((0,\, 2\pi)\):

\[x_1 = \frac{\pi}{2}, \qquad x_2 = \frac{3\pi}{2}\]
Proveri sebe: da li \(x = \pi/2\) zadovoljava polaznu jednačinu?

Leva strana: \(\sin\pi = 0\). Desna strana: \(1 + 0 + \cos\pi = 1 - 1 = 0\). Važi \(0 = 0\) ✓

4

Slučaj 2: \(2\sin x - \sqrt{2} - 2\cos x = 0\)

Preuredimo i podelimo sa 2:

\[\sin x - \cos x = \frac{\sqrt{2}}{2}\]

Primenimo identitet \(\sin x - \cos x = \sqrt{2}\,\sin\!\left(x - \frac{\pi}{4}\right)\):

\[\sqrt{2}\,\sin\!\left(x - \frac{\pi}{4}\right) = \frac{\sqrt{2}}{2} \quad \Rightarrow \quad \sin\!\left(x - \frac{\pi}{4}\right) = \frac{1}{2}\]
5

Rešavanje sinusne jednačine

Jednačina \(\sin t = \frac{1}{2}\) daje \(t = \frac{\pi}{6}\) ili \(t = \frac{5\pi}{6}\) (za \(k = 0\)).

Vraćamo smenu \(x = t + \frac{\pi}{4}\):

\[x_3 = \frac{\pi}{6} + \frac{\pi}{4} = \frac{5\pi}{12}, \qquad x_4 = \frac{5\pi}{6} + \frac{\pi}{4} = \frac{13\pi}{12}\]
6

Zbir svih rešenja

RešenjeVrednostZajednički imenilac
\(x_1\)\(\frac{\pi}{2}\)\(\frac{6\pi}{12}\)
\(x_2\)\(\frac{3\pi}{2}\)\(\frac{18\pi}{12}\)
\(x_3\)\(\frac{5\pi}{12}\)\(\frac{5\pi}{12}\)
\(x_4\)\(\frac{13\pi}{12}\)\(\frac{13\pi}{12}\)
\[\frac{6\pi + 18\pi + 5\pi + 13\pi}{12} = \frac{42\pi}{12} = \frac{7\pi}{2}\]

💡 Ključni uvid

Ključ je u prepoznavanju zajedničkog faktora \(\cos x\) nakon primene formula za dvostruki ugao. Ovo razbija jednačinu na dva prostija problema. Drugi ključni korak je metod pomoćnog ugla: \(\sin x - \cos x = \sqrt{2}\sin(x - \pi/4)\).

✅ Konačan odgovor

\[ \frac{\pi}{2} + \frac{3\pi}{2} + \frac{5\pi}{12} + \frac{13\pi}{12} = \frac{7\pi}{2} \]

(A) \(\dfrac{7\pi}{2}\) (B) \(\dfrac{3\pi}{2}\) (C) \(\dfrac{5\pi}{2}\) (D) \(\pi\) (E) \(\dfrac{\pi}{2}\)

Provera: \(x = \pi/2\): \(\sin\pi = 0 = 1 + 0 + \cos\pi\) ✓. Sva četiri rešenja potvrđena.

⚠️ Česte greške

  • Deljenje sa \(\cos x\) umesto faktorisanja: Gubimo rešenja \(x = \pi/2\) i \(x = 3\pi/2\).
  • Greška u pomoćnom uglu: Identitet glasi \(\sin x - \cos x = \sqrt{2}\sin(x - \pi/4)\), ne \(\sin(x + \pi/4)\).
  • Propuštena rešenja: Jednačina \(\sin t = 1/2\) ima dva rešenja u svakom periodu: \(\pi/6\) i \(5\pi/6\).
