Poređenje vrednosti izraza sa brojem 0,08

Određivanje odnosa veličina \(x\), \(y\) i \(z\) definisanih pomoću kvadriranja i recipročne vrednosti

📋 Tekst zadatka

Ako je \(x = (0{,}08)^2\), \(y = \dfrac{1}{(0{,}08)^2}\) i \(z = (1 - 0{,}08)^2 - 1\), koji od sledećih iskaza je tačan?

\(x\) \((0{,}08)^2\)
\(y\) \(\dfrac{1}{(0{,}08)^2}\)
\(z\) \((1 - 0{,}08)^2 - 1\)
(A) \( x = y = z \)
(B) \( y < z < x \)
(C) \( z < x < y \)
(D) \( y < x = z \)
(E) \( x + z = y \)

🎯 Plan rešavanja

Označićemo \(a = 0{,}08\) i izračunati numeričke vrednosti sva tri izraza \(x\), \(y\) i \(z\). Zatim ćemo ih uporediti i odrediti koji iskaz je tačan. Dodatno ćemo proveriti i preko razlomaka (\(a = \frac{2}{25}\)) za potpunu preciznost.

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Recipročna vrednost: ako je \(0 < a < 1\), tada je \(\frac{1}{a^2} > 1\)
  • Kvadrat razlike: \((1 - a)^2 = 1 - 2a + a^2\), pa je \((1-a)^2 - 1 = a^2 - 2a\)
  • Znak izraza: ako je \(0 < a < 1\), tada \(a^2 < 2a\), pa je \(a^2 - 2a < 0\)
  • Razlomak: \(0{,}08 = \frac{8}{100} = \frac{2}{25}\)

📐 Vizuelni prikaz

Brojevna prava sa pozicijama \(z\), \(x\) i \(y\)

✏️ Rešenje korak po korak

1

Uvedimo oznaku i zapišimo \(a\) kao razlomak

Neka je \(a = 0{,}08\). Pretvorimo u razlomak:

\[ a = 0{,}08 = \frac{8}{100} = \frac{2}{25} \]
2

Izračunajmo \(x = a^2\)

\[ x = a^2 = \left(\frac{2}{25}\right)^2 = \frac{4}{625} = 0{,}0064 \]

Dakle, \(x\) je veoma mali pozitivan broj, blizu nule.

Proveri sebe: Zašto je \(0{,}08^2 = 0{,}0064\)?

Računamo \(8 \times 8 = 64\). Svaki faktor ima 2 decimale, pa proizvod ima \(2 + 2 = 4\) decimale: \(0{,}0064\).

3

Izračunajmo \(y = \dfrac{1}{a^2}\)

\[ y = \frac{1}{a^2} = \frac{1}{\frac{4}{625}} = \frac{625}{4} = 156{,}25 \]

Dakle, \(y\) je veoma veliki broj, veći od 156!

4

Izračunajmo \(z = (1-a)^2 - 1\)

Prvo izračunamo \(1 - a = 0{,}92\), zatim kvadriramo:

\[ (0{,}92)^2 = 0{,}8464 \]

Oduzimamo 1:

\[ z = 0{,}8464 - 1 = -0{,}1536 \]

Dakle, \(z\) je negativan broj!

Proveri sebe: Algebarski pristup za \(z\)

Koristeći formulu za kvadrat razlike:

\[ z = (1 - a)^2 - 1 = 1 - 2a + a^2 - 1 = a^2 - 2a = a(a - 2) \]

Pošto je \(a = 0{,}08 > 0\) i \(a - 2 = -1{,}92 < 0\), dobijamo \(z < 0\).

5

Uporedimo \(x\), \(y\) i \(z\)

\[ z = -0{,}1536, \qquad x = 0{,}0064, \qquad y = 156{,}25 \]

Jasno je da važi:

\[ z < x < y \]

Ovaj poredak odgovara opciji (C).

💡 Ključni uvid

Kada je \(0 < a < 1\), kvadriranje ga smanjuje (približava nuli), dok recipročna vrednost ga povećava. Sa druge strane, \(z = a(a-2)\) je negativno za svako \(a \in (0, 2)\). Samo iz znakova već znamo: \(z < 0 < x < y\).

✅ Konačan odgovor

\[ z < x < y \]

(A) \( x = y = z \) (B) \( y < z < x \) (C) \( z < x < y \) (D) \( y < x = z \) (E) \( x + z = y \)

Provera: \(z = -0{,}1536\), \(x = 0{,}0064\), \(y = 156{,}25\). Zaista važi \(-0{,}1536 < 0{,}0064 < 156{,}25\). ✓

⚠️ Česte greške

  • Pogrešan znak za \(z\): Mnogi zaborave da je \((0{,}92)^2 = 0{,}8464 < 1\), pa kada se oduzme 1, rezultat je negativan.
  • Zamena \(x\) i \(y\): Važno je razlikovati \(a^2\) (kvadrat malog broja — još manji) od \(1/a^2\) (recipročna vrednost — veoma veliki). Za \(a < 1\) uvek važi \(a^2 < 1 < 1/a^2\).
  • Greška pri računanju \((0{,}92)^2\): Treba pažljivo pomnožiti: \(92 \times 92 = 8464\), pa sa četiri decimale: \(0{,}8464\).

🏆 Dodatni izazov

Za koje vrednosti \(a \in (0, 1)\) važi \(x + z = 0\), tj. \(a^2 + (1-a)^2 - 1 = 0\)?

Pogledaj nagoveštaj

Razvijemo: \(a^2 + 1 - 2a + a^2 - 1 = 2a^2 - 2a = 2a(a-1) = 0\). Rešenja su \(a = 0\) ili \(a = 1\), ali nijedno nije u \((0, 1)\). Za svako \(a \in (0, 1)\) važi \(x + z < 0\).
