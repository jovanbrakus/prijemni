Nejednačina sa apsolutnim vrednostima

Algebra — analiza po slučajevima, apsolutna vrednost, racionalne nejednačine

📋 Tekst zadatka

Skup svih realnih rešenja nejednačine

\[ \frac{|1 - x|}{1 - |x|} < \frac{1 + |x|}{|1 + x|} \]

je oblika (za neke realne brojeve \(a\) i \(b\) takve da je \(0 < a < b < +\infty\)):

(A) \((-\infty, -a)\)
(B) \((a, +\infty)\)
(C) \((-\infty, -a) \cup (a, +\infty)\)
(D) \((-b, -a) \cup (a, b)\)
(E) \((-\infty, -a) \cup (-a, a) \cup (a, +\infty)\)

🎯 Plan rešavanja

Najpre ćemo odrediti domen nejednačine (gde su imenioci definisani i različiti od nule). Zatim ćemo razložiti realnu osu na intervale prema znaku izraza unutar apsolutnih vrednosti i u svakom intervalu ukloniti apsolutne vrednosti. Na kraju spajamo intervale na kojima nejednačina važi.

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Definicija apsolutne vrednosti: \( |a| = \begin{cases} a, & a \geq 0 \\ -a, & a < 0 \end{cases} \)
  • Kritične tačke: Izrazi \(|1-x|\), \(|x|\), \(|1+x|\) menjaju oblik u tačkama \(x = -1\), \(x = 0\) i \(x = 1\). Ove tačke dele realnu osu na intervale u kojima svaki izraz ima konstantan znak.
  • Domen razlomka: Imenilac mora biti različit od nule. Za \(1 - |x|\) to znači \(|x| \neq 1\), tj. \(x \neq \pm 1\). Za \(|1+x|\) to znači \(x \neq -1\).

📐 Vizuelni prikaz

Grafik leve strane (narandžasto) i desne strane (zeleno) nejednačine. Rešenja su oblasti gde je leva kriva ispod desne.

✏️ Rešenje korak po korak

1

Određujemo domen nejednačine

Nejednačina sadrži dva razlomka. Imenioci moraju biti različiti od nule:

\[ 1 - |x| \neq 0 \implies |x| \neq 1 \implies x \neq \pm 1 \] \[ |1 + x| \neq 0 \implies x \neq -1 \]

Uslovi se kombinuju: \(x \neq -1\) i \(x \neq 1\).

Domen je: \(\mathbb{R} \setminus \{-1, \, 1\}\).

2

Identifikujemo kritične tačke i intervale

Izrazi unutar apsolutnih vrednosti menjaju znak u tačkama \(x = -1\), \(x = 0\) i \(x = 1\). Uz isključene tačke \(x = \pm 1\) iz domena, dobijamo četiri intervala za analizu:

\[ x < -1, \quad -1 < x < 0, \quad 0 \leq x < 1, \quad x > 1 \]
Proveri sebe: Koliko je \(|1 - x|\) za \(x = 3\)?

\(|1 - 3| = |-2| = 2 = x - 1\). Za \(x > 1\) važi \(|1 - x| = x - 1\).

3

Slučaj 1: \(x > 1\)

U ovom intervalu važi: \(1 - x < 0\), \(x > 0\), \(1 + x > 0\). Dakle:

\[ |1 - x| = x - 1, \quad |x| = x, \quad |1 + x| = 1 + x \]

Zamenom u nejednačinu:

\[ \text{Leva strana} = \frac{x - 1}{1 - x} = \frac{-(1-x)}{1 - x} = -1 \]
\[ \text{Desna strana} = \frac{1 + x}{1 + x} = 1 \]

Nejednačina: \(-1 < 1\) — TAČNO za sve \(x > 1\).

4

Slučaj 2: \(0 \leq x < 1\)

U ovom intervalu važi: \(1 - x > 0\), \(x \geq 0\), \(1 + x > 0\). Dakle:

\[ |1 - x| = 1 - x, \quad |x| = x, \quad |1 + x| = 1 + x \]
\[ \text{Leva strana} = \frac{1 - x}{1 - x} = 1, \qquad \text{Desna strana} = \frac{1 + x}{1 + x} = 1 \]

Nejednačina: \(1 < 1\) — NETAČNO — nema rešenja.

5

Slučaj 3: \(-1 < x < 0\)

U ovom intervalu važi: \(1 - x > 0\), \(x < 0\), \(1 + x > 0\). Dakle:

\[ |1 - x| = 1 - x, \quad |x| = -x, \quad |1 + x| = 1 + x \]
\[ \text{Leva strana} = \frac{1 - x}{1 - (-x)} = \frac{1 - x}{1 + x} \]
\[ \text{Desna strana} = \frac{1 + (-x)}{1 + x} = \frac{1 - x}{1 + x} \]

Leva i desna strana su identične, pa stroga nejednakost ne može da važi — NETAČNO.

Proveri sebe: Zašto su leva i desna strana jednake za \(-1 < x < 0\)?

Zato što u tom intervalu \(|1 - x| = 1 - x\), \(1 - |x| = 1 + x\), \(1 + |x| = 1 - x\), i \(|1 + x| = 1 + x\). I levi i desni razlomak daju \(\frac{1-x}{1+x}\).

6

Slučaj 4: \(x < -1\)

U ovom intervalu važi: \(1 - x > 0\), \(x < 0\), \(1 + x < 0\). Dakle:

\[ |1 - x| = 1 - x, \quad |x| = -x, \quad |1 + x| = -(1 + x) \]
\[ \text{Leva strana} = \frac{1 - x}{1 + x} \]

Pošto je \(x < -1\): \(1 + x < 0\) i \(1 - x > 0\), pa je leva strana negativna.

\[ \text{Desna strana} = \frac{1 - x}{-(1 + x)} = -\frac{1 - x}{1 + x} \]

Pošto je \(-(1 + x) > 0\) i \(1 - x > 0\), desna strana je pozitivna.

Nejednačina: negativan broj \(<\) pozitivan broj — TAČNO za sve \(x < -1\).

7

Objedinjavanje rešenja

Rezimirajmo rezultate iz sva četiri slučaja:

\[ \begin{array}{lcl} x > 1 & : & \text{nejednačina važi} \; \checkmark \\[4pt] 0 \leq x < 1 & : & \text{nejednačina ne važi} \; \times \\[4pt] -1 < x < 0 & : & \text{nejednačina ne važi} \; \times \\[4pt] x < -1 & : & \text{nejednačina važi} \; \checkmark \end{array} \]

Skup rešenja je:

\[ (-\infty, -1) \cup (1, +\infty) \]

Ovo je oblika \((-\infty, -a) \cup (a, +\infty)\) za \(a = 1\).

Proveri sebe: Numerička provera za \(x = -2\)

LHS \(= \frac{|1-(-2)|}{1-|-2|} = \frac{3}{1-2} = \frac{3}{-1} = -3\). RHS \(= \frac{1+|-2|}{|1+(-2)|} = \frac{3}{1} = 3\). Nejednačina: \(-3 < 3\) — tačno!

💡 Ključni uvid

U intervalima \(0 \leq x < 1\) i \(-1 < x < 0\) obe strane nejednačine se uprošćavaju do identičnog izraza, pa stroga nejednakost nikada ne važi. U intervalima \(x > 1\) i \(x < -1\), leva strana postaje konstantna (i to negativna ili \(-1\)), dok desna strana ostaje pozitivna (konstantna ili \(1\)), pa nejednačina automatski važi.

\[ \text{Rešenje:} \quad (-\infty, -1) \cup (1, +\infty) \]

✅ Konačan odgovor

\[ (-\infty, -1) \cup (1, +\infty) \]

(A) \((-\infty, -a)\) (B) \((a, +\infty)\) (C) \((-\infty, -a) \cup (a, +\infty)\) (D) \((-b, -a) \cup (a, b)\) (E) \((-\infty, -a) \cup (-a, a) \cup (a, +\infty)\)

Provera: Za \(x = 2\): LHS \(= \frac{1}{-1} = -1\), RHS \(= \frac{3}{3} = 1\), pa \(-1 < 1\) važi. Za \(x = 0{,}5\): LHS \(= \frac{0{,}5}{0{,}5} = 1\), RHS \(= \frac{1{,}5}{1{,}5} = 1\), pa \(1 < 1\) ne važi. Za \(x = -2\): LHS \(= \frac{3}{-1} = -3\), RHS \(= \frac{3}{1} = 3\), pa \(-3 < 3\) važi.

⚠️ Česte greške

  • Zaboraviti da se \(x = \pm 1\) mora isključiti iz domena. Imenilac \(1 - |x|\) se poništava za \(x = \pm 1\), što čini razlomak nedefinisanim.
  • Ne razdvojiti slučaj \(x < -1\) od slučaja \(-1 < x < 0\). Znak izraza \(1 + x\) se menja pri prelasku preko \(x = -1\), što bitno utiče na oblik apsolutne vrednosti \(|1 + x|\).

🏆 Dodatni izazov

Šta ako umesto stroge nejednakosti (\(<\)) imamo \(\leq\)? Kako se menja skup rešenja?

Pogledaj nagoveštaj

Kada imamo \(\leq\), jednakost važi u intervalima gde su obe strane bile identične: \(-1 < x < 0\) i \(0 \leq x < 1\). Skup rešenja bi onda bio \((-\infty, -1) \cup (-1, 1) \cup (1, +\infty) = \mathbb{R} \setminus \{-1, 1\}\), što je ceo domen.
