Određivanje kvadratne jednačine za \(y_1\) i \(y_2\)

Vijetove formule i transformacija korena

📋 Tekst zadatka

Ako su \(x_1\) i \(x_2\) rešenja kvadratne jednačine \(x^2 + x + 1 = 0\), tada su \(y_1 = a \cdot x_1 + x_2\) i \(y_2 = x_1 + a \cdot x_2\), \((a \in \mathbb{R})\), rešenja kvadratne jednačine:

(A) \(y^2 + (a+1)\,y - a^2 + a + 1 = 0\)
(B) \(y^2 + (a^2+1)\,y + 1 = 0\)
(C) \(y^2 + (a+1)\,y + a^2 - a + 1 = 0\)
(D) \(y^2 + (a^2+1)\,y + a^2 - a + 1 = 0\)
(E) nijedan od ponuđenih odgovora

🎯 Plan rešavanja

Iz jednačine \(x^2 + x + 1 = 0\) pomoću Vijetovih formula odredićemo \(x_1 + x_2\) i \(x_1 \cdot x_2\), zatim ćemo izračunati zbir \(y_1 + y_2\) i proizvod \(y_1 \cdot y_2\), i na kraju formirati jednačinu \(y^2 - (y_1+y_2)\,y + y_1 \cdot y_2 = 0\).

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Vijetove formule: Za kvadratnu jednačinu \(ax^2 + bx + c = 0\) sa rešenjima \(x_1\) i \(x_2\): \[ x_1 + x_2 = -\frac{b}{a}, \qquad x_1 \cdot x_2 = \frac{c}{a} \]
  • Obrnute Vijetove formule: Ako znamo zbir \(S = y_1 + y_2\) i proizvod \(P = y_1 \cdot y_2\), onda su \(y_1\) i \(y_2\) rešenja jednačine: \[ y^2 - S \cdot y + P = 0 \]
  • Zbir kvadrata korena: \(x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2\,x_1 \cdot x_2\)

✏️ Rešenje korak po korak

1

Vijetove formule za polaznu jednačinu

Jednačina \(x^2 + x + 1 = 0\) ima koeficijente \(a=1\), \(b=1\), \(c=1\).

Primenom Vijetovih formula:

\[ x_1 + x_2 = -\frac{b}{a} = -\frac{1}{1} = -1 \]
\[ x_1 \cdot x_2 = \frac{c}{a} = \frac{1}{1} = 1 \]
Proveri sebe: Koliko iznosi \(x_1^2 + x_2^2\)?

\(x_1^2 + x_2^2 = (x_1+x_2)^2 - 2\,x_1 x_2 = (-1)^2 - 2 \cdot 1 = 1 - 2 = -1\)

2

Zbir \(y_1 + y_2\)

Sabiramo izraze za \(y_1\) i \(y_2\):

\[ y_1 + y_2 = (a \cdot x_1 + x_2) + (x_1 + a \cdot x_2) \]

Grupišemo po \(x_1\) i \(x_2\):

\[ y_1 + y_2 = (a+1)\,x_1 + (a+1)\,x_2 = (a+1)(x_1 + x_2) \]

Zamenom \(x_1 + x_2 = -1\):

\[ \boxed{y_1 + y_2 = -(a+1)} \]
3

Proizvod \(y_1 \cdot y_2\)

Množimo izraze za \(y_1\) i \(y_2\):

\[ y_1 \cdot y_2 = (a\,x_1 + x_2)(x_1 + a\,x_2) \]

Raspisujemo proizvod:

\[ = a\,x_1^2 + a^2\,x_1 x_2 + x_1 x_2 + a\,x_2^2 \]

Grupišemo:

\[ = a\underbrace{(x_1^2 + x_2^2)}_{= -1} + (a^2 + 1)\underbrace{x_1 x_2}_{= 1} \]
\[ = a \cdot (-1) + (a^2 + 1) \cdot 1 = -a + a^2 + 1 \]
\[ \boxed{y_1 \cdot y_2 = a^2 - a + 1} \]
Proveri sebe: Za \(a = 0\), koliko iznosi \(y_1 \cdot y_2\)?

\(y_1 \cdot y_2 = 0^2 - 0 + 1 = 1\). Logično, jer za \(a=0\) imamo \(y_1 = x_2\) i \(y_2 = x_1\), pa je \(y_1 y_2 = x_1 x_2 = 1\). ✓

4

Formiranje jednačine

Primenom obrnutih Vijetovih formula, jednačina sa rešenjima \(y_1\) i \(y_2\) glasi:

\[ y^2 - (y_1 + y_2)\,y + y_1 \cdot y_2 = 0 \]

Zamenom izračunatih vrednosti:

\[ y^2 - (-(a+1))\,y + (a^2 - a + 1) = 0 \]
\[ \boxed{y^2 + (a+1)\,y + a^2 - a + 1 = 0} \]

Ovo se poklapa sa odgovorom (C).

5

Provera rezultata

Proverimo za konkretnu vrednost \(a = 1\):

\[ y_1 = x_1 + x_2 = -1, \qquad y_2 = x_1 + x_2 = -1 \]

Naša jednačina za \(a=1\): \(y^2 + 2y + 1 = 0\), tj. \((y+1)^2 = 0\), jedino rešenje \(y = -1\). ✓

Proverimo za \(a = 0\):

\[ y_1 = x_2, \quad y_2 = x_1 \implies \text{ista jednačina kao polazna: } y^2 + y + 1 = 0 \]

Naša jednačina za \(a=0\): \(y^2 + y + 1 = 0\). ✓

💡 Ključni uvid

Nikada nismo morali da izračunamo same korene \(x_1\) i \(x_2\)! Vijetove formule nam daju zbir i proizvod korena, što je dovoljno da izrazimo sve simetrične funkcije korena. Trik kod proizvoda \(y_1 \cdot y_2\) je prepoznavanje da se pojavljuje izraz \(x_1^2 + x_2^2\), koji se lako izračunava pomoću identiteta \((x_1+x_2)^2 - 2x_1 x_2\).

✅ Konačan odgovor

\[ y^2 + (a+1)\,y + a^2 - a + 1 = 0 \]

(A) \(y^2 + (a+1)\,y - a^2 + a + 1 = 0\) (B) \(y^2 + (a^2+1)\,y + 1 = 0\) (C) \(y^2 + (a+1)\,y + a^2 - a + 1 = 0\) (D) \(y^2 + (a^2+1)\,y + a^2 - a + 1 = 0\) (E) nijedan od ponuđenih odgovora

Provera: za \(a=1\), jednačina je \(y^2 + 2y + 1 = 0\), tj. \((y+1)^2 = 0\), rešenje \(y=-1\), što je \(x_1+x_2=-1\) ✓. Za \(a=0\), dobijamo \(y^2+y+1=0\), istu polaznu jednačinu ✓

⚠️ Česte greške

  • Zaboraviti minus u Vijetovoj formuli: Zbir korena je \(-b/a\), ne \(b/a\). Za \(x^2 + x + 1 = 0\) zbir je \(-1\), ne \(+1\).
  • Pogrešno raspisati proizvod: Mnogi pogreše kod \((ax_1+x_2)(x_1+ax_2)\), zaboravljajući da srednji članovi daju \(a^2 x_1 x_2 + x_1 x_2\), a ne \(ax_1 x_2 + ax_1 x_2\).
  • Mešanje predznaka u konačnoj jednačini: Pošto je zbir \(y_1+y_2 = -(a+1)\), u jednačini \(y^2 - S\,y + P = 0\) ispred \((a+1)\) ostaje \(+\), ne \(-\).

🏆 Dodatni izazov

Za koje vrednosti parametra \(a \in \mathbb{R}\) jednačina \(y^2 + (a+1)y + a^2 - a + 1 = 0\) ima realna rešenja?

Pogledaj nagoveštaj

Ispitaj diskriminantu \(D = (a+1)^2 - 4(a^2 - a + 1)\) i odredi za koja \(a\) važi \(D \geq 0\).
