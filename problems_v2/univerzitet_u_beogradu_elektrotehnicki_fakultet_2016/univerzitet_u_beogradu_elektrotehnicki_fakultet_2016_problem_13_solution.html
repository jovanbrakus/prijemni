Deljivost polinoma visokog stepena

Algebra — faktor teorema, sistemi jednačina

📋 Tekst zadatka

Ako je polinom \(P(x) = x^{2016} + x^{2015} - x^{2014} + ax^{2013} - bx^2 + c\) (\(a, b, c \in \mathbb{R}\)) deljiv sa \(x^3 - x\), tada je \(4a^2 + 3b^2 + 8c^2\) jednako:

(A)\( 4 \)
(B)\( 3 \)
(C)\( 12 \)
(D)\( 15 \)
(E)\( 18 \)

🎯 Plan rešavanja

Pošto je \(x^3 - x = x(x-1)(x+1)\), koreni su \(0, 1, -1\). Ako je \(P(x)\) deljiv ovim polinomom, tada \(P(0) = P(1) = P(-1) = 0\). Iz ova tri uslova dobijamo sistem jednačina za \(a, b, c\).

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Faktor teorema: Ako je \(P(x)\) deljiv sa \((x - r)\), tada \(P(r) = 0\).
  • Parnost stepena: \((-1)^{2k} = 1\), \((-1)^{2k+1} = -1\)

✏️ Rešenje korak po korak

1

Faktorizujemo delilac

\[ x^3 - x = x(x-1)(x+1) \]

Koreni: \(x = 0\), \(x = 1\), \(x = -1\).

2

\(P(0) = 0\)

\[ P(0) = c = 0 \quad \Rightarrow \quad c = 0 \]
3

\(P(1) = 0\)

\[ P(1) = 1 + 1 - 1 + a - b + 0 = 1 + a - b = 0 \quad \Rightarrow \quad a - b = -1 \]
4

\(P(-1) = 0\)

\[ P(-1) = 1 - 1 - 1 - a - b = -1 - a - b = 0 \quad \Rightarrow \quad a + b = -1 \]
Proveri sebe: Kako smo odredili parnost stepena?

\((-1)^{2016} = 1\), \((-1)^{2015} = -1\), \((-1)^{2014} = 1\), \((-1)^{2013} = -1\), \((-1)^2 = 1\).

5

Rešavamo sistem

Iz \(a - b = -1\) i \(a + b = -1\): sabiranjem \(2a = -2 \Rightarrow a = -1\), oduzimanjem \(2b = 0 \Rightarrow b = 0\).

\[ 4a^2 + 3b^2 + 8c^2 = 4 \cdot 1 + 0 + 0 = 4 \]

💡 Ključni uvid

Uprkos visokom stepenu polinoma (2016), zadatak se svodi na jednostavan sistem linearnih jednačina zahvaljujući faktor teoremi. Ključno zapažanje: za \(x = \pm 1\), svi stepeni se svode na \(\pm 1\).

✅ Konačan odgovor

\[ 4a^2 + 3b^2 + 8c^2 = 4 \]

(A) \( 4 \) (B) \( 3 \) (C) \( 12 \) (D) \( 15 \) (E) \( 18 \)

Provera: \(P(x) = x^{2013}(x^3+x^2-x-1) = x^{2013}(x-1)(x+1)^2\), deljivo sa \(x(x-1)(x+1)\). ✓

⚠️ Česte greške

  • Pogrešna parnost stepena: Paziti da su 2016 i 2014 parni, a 2015 i 2013 neparni pri računanju \(P(-1)\).

🏆 Dodatni izazov

Pogledaj nagoveštaj
