Zbir kosinusa uglova trougla

Kosinusna teorema — odnos stranica i uglova u trouglu

📋 Tekst zadatka

U trouglu \(ABC\) je \(|BC| = 3 \cdot |AB|\) i \(\angle ABC = 60°\). Tada je zbir \(\cos \angle BAC + \cos \angle ACB\) jednak:

(A) \(\dfrac{1}{\sqrt{7}}\)
(B) \(\dfrac{2}{\sqrt{7}}\)
(C) \(\dfrac{3}{\sqrt{7}}\)
(D) \(\dfrac{4}{\sqrt{7}}\)
(E) \(\dfrac{5}{\sqrt{7}}\)

🎯 Plan rešavanja

Uvedemo oznake za stranice, kosinusnom teoremom odredimo nepoznatu stranicu \(|AC|\), a zatim ponovo kosinusnom teoremom izračunamo \(\cos A\) i \(\cos C\) pojedinačno. Na kraju saberemo rezultate.

📚 Podsetnik iz teorije

Ključne formule i teoreme
  • Kosinusna teorema: \(b^2 = a^2 + c^2 - 2ac \cos B\) — povezuje stranicu trougla sa naspramnim uglom i preostale dve stranice.
  • Kosinusna teorema (oblik za ugao): \(\cos A = \dfrac{b^2 + c^2 - a^2}{2bc}\) — određuje kosinus ugla kada su poznate sve tri stranice.
  • Zbir uglova u trouglu: \(A + B + C = 180°\) — korisno za proveru rezultata.

✏️ Rešenje korak po korak

1

Uvedimo oznake

Neka je \(c = |AB|\) i \(a = |BC| = 3c\). Ugao kod temena \(B\) je \(\angle ABC = 60°\). Nepoznata stranica je \(b = |AC|\).

\[ c = |AB|, \quad a = |BC| = 3c, \quad \angle B = 60° \]
2

Izračunajmo \(b^2\) kosinusnom teoremom

Stranica \(b = |AC|\) je naspram ugla \(B\). Kosinusna teorema daje:

\[ b^2 = a^2 + c^2 - 2ac \cos B = (3c)^2 + c^2 - 2 \cdot 3c \cdot c \cdot \cos 60° \]

Pošto je \(\cos 60° = \frac{1}{2}\):

\[ b^2 = 9c^2 + c^2 - 6c^2 \cdot \frac{1}{2} = 10c^2 - 3c^2 = 7c^2 \]

Dakle: \( b = c\sqrt{7} \)

3

Odredimo \(\cos A\)

Kosinusna teorema za ugao \(A\) (naspram stranice \(a = 3c\)):

\[ \cos A = \frac{b^2 + c^2 - a^2}{2bc} = \frac{7c^2 + c^2 - 9c^2}{2 \cdot c\sqrt{7} \cdot c} = \frac{-c^2}{2c^2\sqrt{7}} \]
\[ \cos A = -\frac{1}{2\sqrt{7}} \]

Negativan kosinus znači da je ugao \(A\) tup (veći od \(90°\)).

Proveri sebe: Koliko je \(b^2\) ako je \(c = 1\)?

\(b^2 = 7 \cdot 1^2 = 7\), pa je \(b = \sqrt{7} \approx 2{,}646\). Ovo ima smisla jer je \(b\) između \(a - c = 2\) i \(a + c = 4\) (nejednakost trougla).

4

Odredimo \(\cos C\)

Kosinusna teorema za ugao \(C\) (naspram stranice \(c\)):

\[ \cos C = \frac{a^2 + b^2 - c^2}{2ab} = \frac{9c^2 + 7c^2 - c^2}{2 \cdot 3c \cdot c\sqrt{7}} = \frac{15c^2}{6c^2\sqrt{7}} \]
\[ \cos C = \frac{5}{2\sqrt{7}} \]
5

Izračunajmo traženi zbir

Sada saberemo oba kosinusa:

\[ \cos A + \cos C = -\frac{1}{2\sqrt{7}} + \frac{5}{2\sqrt{7}} = \frac{-1 + 5}{2\sqrt{7}} = \frac{4}{2\sqrt{7}} \]
\[ \cos A + \cos C = \frac{2}{\sqrt{7}} \]
Proveri sebe: Koliko je \(\frac{2}{\sqrt{7}}\) približno?

\(\sqrt{7} \approx 2{,}646\), pa je \(\frac{2}{\sqrt{7}} \approx 0{,}756\). Proverimo: \(\cos A \approx -0{,}189\), \(\cos C \approx 0{,}945\), zbir \(\approx 0{,}756\). Slaže se!

💡 Ključni uvid

Zahvaljujući tome što su oba kosinusa izražena preko istog imenioca \(2\sqrt{7}\), sabiranje postaje trivijalno. Ključna ideja je da kosinusnom teoremom možemo izračunati kosinus bilo kog ugla kada poznajemo sve tri stranice — a treća stranica se upravo dobija iz iste kosinusne teoreme!

✅ Konačan odgovor

\[ \cos A + \cos C = \frac{2}{\sqrt{7}} \]

(A) \(\dfrac{1}{\sqrt{7}}\) (B) \(\dfrac{2}{\sqrt{7}}\) (C) \(\dfrac{3}{\sqrt{7}}\) (D) \(\dfrac{4}{\sqrt{7}}\) (E) \(\dfrac{5}{\sqrt{7}}\)

Provera: Za \(c = 1\): stranice su \(a = 3\), \(b = \sqrt{7}\), \(c = 1\). Zbir uglova: \(A \approx 100{,}9° + 60° + 19{,}1° = 180°\) ✓. Numerička provera: \(\cos(100{,}9°) + \cos(19{,}1°) \approx -0{,}189 + 0{,}945 \approx 0{,}756 = \frac{2}{\sqrt{7}}\) ✓

⚠️ Česte greške

  • Zamena stranica u kosinusnoj teoremi: Česta greška je pogrešno identifikovati koja stranica je naspram kog ugla. Stranica \(a = |BC|\) je naspram ugla \(A\) (ne ugla \(B\) ili \(C\)!).
  • Zaboravljen minus kod \(\cos A\): Pošto je \(b^2 + c^2 - a^2 = -c^2 < 0\), kosinus ugla \(A\) je negativan. Ako se previdi minus, dobija se pogrešan zbir \(\frac{3}{\sqrt{7}}\) (odgovor C), što je zamka!

🏆 Dodatni izazov

U istom trouglu (\(|BC| = 3|AB|\), \(\angle B = 60°\)), izračunajte proizvod \(\sin \angle BAC \cdot \sin \angle ACB\).

Pogledaj nagoveštaj

Koristite sinusnu teoremu: \(\frac{a}{\sin A} = \frac{c}{\sin C}\), pa je \(\frac{\sin A}{\sin C} = \frac{a}{c} = 3\). Takođe, iz \(A + C = 120°\) možete dobiti sistem jednačina.
